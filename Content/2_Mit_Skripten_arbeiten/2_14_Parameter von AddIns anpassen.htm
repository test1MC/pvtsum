Parameter von AddIns anpassen

Wählen Sie im Menü Skripte den Eintrag Skriptmenü bearbeiten.

Das Fenster Skriptmenü bearbeiten öffnet sich. Die Einträge und Untermenüeinträge des Menüs Skripte werden angezeigt.

Wählen Sie den Untermenüeintrag oder das Add In aus, dessen Parameter Sie bearbeiten möchten.

Klicken Sie auf die Schaltfläche Bearbeiten.

Alternativ können Sie auf den zu bearbeitenden Untermenüeintrag oder Eintrag doppelklicken.

Das Fenster Skriptmenü-Eintrag öffnet sich.

Wählen Sie die Option Add In.

Klicken Sie im Bereich Add In auf die Schaltfläche Parameter.

Das Fenster Parameter von Add In-Name öffnet sich.

Nehmen Sie die gewünschten Änderungen vor.

Bestätigen Sie im Fenster Skriptmenü-Eintrag mit OK.

Bestätigen Sie im Fenster Skriptmenü bearbeiten mit OK.

Ihre Änderungen werden übernommen.

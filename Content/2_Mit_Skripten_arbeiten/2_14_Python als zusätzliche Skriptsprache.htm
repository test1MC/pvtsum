Python als zusätzliche Skriptsprache

Scripting und COM erlauben Ihnen, neue Funktionen hinzuzufügen und wiederkehrende Aufgaben zu automatisieren.

Seit dem Release 9.4 wird Python zusätzlich neben VBA/VBS unterstützt. Python zeichnet sich aus durch sehr komfortable Funktionen für die Matrixbearbeitung, die Erstellung von Diagrammen und Funktionen für Benutzeroberflächen. Mit Python kann eine auf ein spezielles Projekt zugeschnittene Benutzerführung aufbaut werden.

Python und alle Zusatzbibliotheken sind Open Source-Programme ohne Nutzungsbeschränkung.

Setup

Voraussetzung ist die vorherige Installation des Python-Interpreters und der Erweiterung PythonWin, die Python mit COM-Funktionalität ausstattet.

Das -Setup installiert auf Wunsch ein Paket aus dem Python-Basis-System und nützlichen Zusatzbibliotheken.

Dokumentation und Beispieldateien

Das Verzeichnis ...Benutzer\Öffentlich\Öffentliche Dokumente\PTV Vision\\COM\Examples_ComDocu\Python Ihrer -Installation enthält Python-Beispieldateien.

Die Datei ...Benutzer\Öffentlich\Öffentliche Dokumente\PTV Vision\\COM\Introduction to the PTV Visum COM-API.pdf) enthält die Dokumentation der Python-Bibliothek VisumPy.

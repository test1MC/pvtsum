Ein Analysezeitintervall einfügen

Wählen Sie im Menü Rechnen den Eintrag Allgemeine Verfahrenseinstellungen.

Das Fenster Allgemeine Verfahrenseinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag Analysezeiträume.

Klicken Sie auf die Schaltfläche Einfügen.

In der Liste der Analysezeitintervalle wird eine neue Zeile eingefügt.

 

Sie können die Werte ändern, indem Sie in die Zellen doppelklicken.

 

Spalte

Beschreibung

Code

Kurzbezeichnung für das Analysezeitintervall

Hinweise

Der Code muss eindeutig sein.

Beim Einfügen werden automatisch die ersten Ziffern der von Zeit verwendet (Anzeige: hhmm). Wenn Sie mit einem Kalender arbeiten, wird vor der Uhrzeit der Wochentag oder das Datum angezeigt.

von Zeit

Intervallbeginn (Eingabeformat: hh, hh:mm oder hh:mm:ss)

bis Zeit

Intervallende (Eingabeformat: hh, hh:mm oder hh:mm:ss)

Tag

Über die Auswahlliste legen Sie bei Verwendung eines Kalenders den Tag fest.

  • Wochenkalender: Wochentag, in dem das Intervall liegt
  • Jahreskalender: Datum des Tages, in dem das Intervall liegt

Name

Name des Analysezeitintervalls (optional)

Hinweis

Beim Einfügen wird der Name automatisch aus von Zeit und bis Zeit gebildet. Wenn Sie mit einem Kalender arbeiten, wird vor dem Zeitraum der Wochentag oder das Datum angezeigt.

 

Die Analysezeitintervalle können Sie chronologisch sortieren (Analysezeitintervalle chronologisch sortieren).

Die Analysezeitintervalle werden gespeichert.

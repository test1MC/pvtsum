Mehrere aufeinanderfolgende Analysezeitintervalle einfügen

Wählen Sie im Menü Rechnen den Eintrag Allgemeine Verfahrenseinstellungen.

Das Fenster Allgemeine Verfahrenseinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag Analysezeiträume.

Klicken Sie auf die Schaltfläche Mehrere.

Das Fenster Mehrere Analysezeitintervalle einfügen öffnet sich.

 

Element

Beschreibung

Anzahl

Anzahl der Analysezeitintervalle, die Sie anlegen möchten

Intervall-Länge

Dauer der einzelnen Analysezeitintervalle in Minuten

Hinweis

Sie können die Dauer mit Einheit eingeben (h, min, s).

Erste Von-Zeit

Beginn des ersten Analysezeitintervalls

Hinweis

Wenn Sie mit einem Kalender arbeiten, wird vor der Uhrzeit der Wochentag oder das Datum angezeigt.

Klicken Sie auf die Schaltfläche Einfügen.

Die Analysezeitintervalle werden angelegt.

 

Code und Name für das Analysezeitintervall werden automatisch vergeben. Abhängig vom eingestellten Kalender, setzt sich der Eintrag in den Feldern Code und Name unterschiedlich zusammen. Sie können die automatisch vergebenen Einträge ändern, wenn Sie in das gewünschte Feld klicken.

Die Analysezeitintervalle werden gespeichert.

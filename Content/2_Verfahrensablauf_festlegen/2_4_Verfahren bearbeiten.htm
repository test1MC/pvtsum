Verfahren bearbeiten

Sie können ein bereits ausgewähltes Verfahren durch ein anderes Verfahren ersetzen und die Parameter eines Verfahrens bearbeiten.

Ein Verfahren durch ein anderes ersetzen

Klicken Sie in der Symbolleiste auf das Symbol Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Klicken Sie in der Zeile des Verfahrens oder der Gruppe, die ersetzt werden soll, in die Spalte Verfahren.

Das Fenster Verfahren öffnet sich.

Wählen Sie das gewünschte Verfahren aus der entsprechenden Kategorie aus.

Klicken Sie auf die Schaltfläche OK.

Das Verfahren wird ersetzt.

Legen Sie bei Bedarf die nötigen Einstellungen für das Verfahren fest.

Das Verfahren ist geändert.

Parameter eines Verfahrens bearbeiten

Markieren Sie in der Liste das Verfahren, dessen Parameter Sie bearbeiten möchten.

Klicken Sie im Bereich Aktionen auf die Schaltfläche Bearbeiten.

Das Fenster Parameter <Verfahren> öffnet sich.

Bearbeiten Sie die Parameter.

Die Funktion Bearbeiten steht nur bei den Verfahren zur Verfügung, bei denen Sie Parameter bearbeiten können.

Die Parameter sind bei den jeweiligen Verfahren beschrieben (Verfahren in Verfahrensablauf einfügen).
Bezugsobjekte oder Variante/Datei bearbeiten

Klicken Sie in der Zeile des Verfahrens, das Sie bearbeiten möchten, in die Spalte Bezugsobjekt(e) oder Variante/Datei.

Legen Sie die gewünschten Einstellungen fest.

Kommentar oder Code zu einem Verfahren eingeben

Klicken Sie in der Zeile des Verfahrens oder der Gruppe, für die Sie einen Kommentar oder einen Code einfügen möchten, in die Spalte Kommentar oder Code.

Geben Sie den gewünschten Kommentar oder Code ein.

Der Kommentar oder Code wird eingefügt.

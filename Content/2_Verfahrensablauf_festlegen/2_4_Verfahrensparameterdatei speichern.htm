Verfahrensparameterdatei speichern

Sie können Verfahrensparameter entweder im Fenster Allgemeine Verfahrenseinstellungen oder im Verfahrensablauf speichern.

Öffnen Sie den Verfahrensablauf oder das Fenster Allgemeine Verfahrenseinstellungen.

Nehmen Sie bei Bedarf die gewünschten Einstellungen vor.

Im Verfahrensablauf klicken Sie auf die Schaltfläche Verfahrensparameter auf Datei speichern. Im Fenster Allgemeine Verfahrenseinstellungen klicken Sie auf die Schaltfläche Speichern.

Das Fenster Verfahrensparameter speichern öffnet sich.

Wählen Sie die gewünschte Datei aus oder geben Sie den Pfad und Dateinamen ein.

Wählen Sie in der Auswahlliste Dateityp den gewünschten Eintrag.

 

Sie können die Verfahrensparameter im Format *.par oder *.xml speichern. Im Format *.xml sind noch nicht alle Verfahren enthalten (Übersicht über die XML-Elemente und XML-Attribute).

Klicken Sie auf die Schaltfläche Speichern.

 

Falls Sie das Format *.xml gewählt haben und der Verfahrensablauf Verfahren enthält, die noch nicht in diesem Format implementiert sind, wird eine entsprechende Meldung angezeigt. Sie können diese Meldung mit der Schaltfläche Keine Warnungen bestätigen, dann wird die Datei ohne weitere Meldung gespeichert. Sie können sie auch mit der Schaltfläche OK bestätigen, dann wird für jedes nicht implementierte Verfahren eine Meldung ausgegeben, bevor die Datei gespeichert wird.

Der Verfahrensablauf und die Verfahrenseinstellungen werden in der festgelegten Parameterdatei gespeichert.

Sie können Verfahrensparameter auch über das Menü Datei > Speichern unter > Eintrag Verfahrensparameter speichern.

Umlegungsergebnisse pro Analysezeitintervall speichern

Sie können festlegen, dass die Umlegungsergebnisse pro Analysezeitintervall gespeichert werden.

Wählen Sie im Menü Rechnen den Eintrag Allgemeine Verfahrenseinstellungen.

Das Fenster Allgemeine Verfahrenseinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag Analysezeiträume.

Legen Sie die gewünschten Analysezeitintervalle fest (Analysezeitintervalle festlegen).

Aktivieren Sie unter Für Berechnungsergebnisse verwenden die gewünschte Option.

 

Element

Beschreibung

IV

Die IV-Umlegungsergebnisse werden pro Analysezeitintervall gespeichert.

ÖV

Die ÖV-Umlegungsergebnisse werden pro Analysezeitintervall gespeichert.

Die Änderungen werden übernommen. Wenn Sie eine Umlegung rechnen, werden die Ergebnisse pro Analysezeitintervall gespeichert.

Nachfragematrix auf Wege verteilen

Sie können die Werte einer Matrix auf vorhandene IV-Wege verteilen. Dabei wird der Matrixwert von Bezirk A nach Bezirk B zur Belastung des Weges von Bezirk A nach Bezirk B. Wenn es von einem Bezirk A zu einem Bezirk B mehrere Wege gibt, wird der entsprechende Matrixwert entsprechend dem Wege-Attribut AnteilAnWegemengeSoll zwischen den Wegen aller oder aller aktiven Wegemengen aufgeteilt.

Wählen Sie im Menü Rechnen den Eintrag Verfahrensablauf.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Nachfragematrix auf Wege verteilen aus der Kategorie Matrizen ein (siehe „Verfahrensablauf festlegen und starten“ auf Seite 1).

Klicken Sie in die Spalte Bezugsobjekt(e).

Das Fenster Matrix auswählen öffnet sich.

Wählen Sie die gewünschte Matrix. Dafür stehen Ihnen zwei Methoden zur Verfügung (siehe „Fenster Matrizen auswählen“ auf Seite 1).

Element Beschreibung

Matrix auswählen nach Nummer

Wählen Sie aus der Liste aller Matrizen anhand der Nummer die gewünschte Matrix aus.

Matrix auswählen nach Eigenschaften

Wählen Sie anhand bestimmter Attribute und deren Werte die Matrix aus.

Klicken Sie auf die Schaltfläche Bearbeiten.

Das Fenster Matrix auf Wege verteilen öffnet sich.

Wählen Sie die gewünschten Optionen.

Element

Beschreibung

Nur aktive Wegemengen

Ist die Option ausgewählt, wird jeder Matrixwert nur auf die Wege der aktiven Wegemengen verteilt. Die Belastungen der anderen Wege bleiben unverändert.

Beziehungsfilter berücksichtigen

Ist die Option ausgewählt, werden nur die aktiven Beziehungen verwendet (siehe „Sonderfall: Beziehungsfilter“ auf Seite 1).

Wenn Sie über die Optionen Wegemengen aus der Verteilung ausschließen, wird das Wege-Attribut AnteilAnWegemengeSoll der ausgeschlossenen Wegemengen bei der Verteilung der Matrixwerte nicht berücksichtigt.

Beispiel: Sie haben die Wege d, e und f von Bezirk A nach Bezirk B. Weg d hat einen AnteilAnWegemengeSoll von 3, Weg e den Wert 1 und Weg f den Wert 1. Der zu verteilende Matrixwert beträgt 100. Weg f ist über Wegefilter inaktiv gesetzt und über die ausgewählte Option Nur aktive Wegemengen von der Verteilung ausgeschlossen. Der Matrixwert wird folgendermaßen auf die Wege d und e verteilt.

Weg d erhält 100 : 4 x 3 = 75.

Weg e erhält 100 : 4 x 1 = 25.

Führen Sie das Verfahren aus (siehe „Verfahren im Verfahrensablauf ausführen“ auf Seite 1).

Die Matrixwerte werden während des Verfahrensablaufs zum festgelegten Zeitpunkt auf die Wege verteilt.

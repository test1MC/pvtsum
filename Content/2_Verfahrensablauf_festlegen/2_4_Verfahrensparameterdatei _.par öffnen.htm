Verfahrensparameterdatei *.par öffnen

Sie können eine Verfahrensparameterdatei im Format *.par öffnen. Dabei legen Sie fest, ob Sie den Verfahrensablauf, die allgemeinen Einstellungen oder beides einlesen möchten. Sie können eine Verfahrensparameterdatei entweder im Fenster Allgemeine Verfahrenseinstellungen oder im Verfahrensablauf öffnen.

Öffnen Sie den Verfahrensablauf oder das Fenster Allgemeine Verfahrenseinstellungen.

Im Verfahrensablauf klicken Sie in der Symbolleiste auf das Symbol Verfahrensparameter von Datei laden. Im Fenster Allgemeine Verfahrenseinstellungen klicken Sie auf die Schaltfläche Öffnen.

Das Fenster Verfahrensparameter öffnen öffnet sich.

Wählen Sie die Datei aus, die Sie öffnen möchten.

Klicken Sie auf die Schaltfläche Öffnen.

Das Fenster Verfahrensparameter lesen öffnet sich.

Wählen Sie die gewünschten Optionen.

 

Element

Beschreibung

Verfahrensablauf

Der Verfahrensablauf wird eingelesen. Der bestehende Verfahrensablauf wird dabei ersetzt.

Allgemeine Verfahrenseinstellungen

Die allgemeinen Verfahrenseinstellungen werden eingelesen. Die bestehenden Verfahrenseinstellungen werden dabei ersetzt.

Die Verfahrensparameter werden entsprechend Ihren Einstellungen eingelesen.

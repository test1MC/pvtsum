Verkehrszusammensetzungen aus HBEFA übernehmen

Sie können in HBEFA vordefinierte Verkehrszusammensetzungen nach übernehmen.

Wählen Sie im Menü Netz den Eintrag Verkehrszusammensetzungen.

Das Fenster Verkehrszusammensetzungen öffnet sich.

Klicken Sie auf die Schaltfläche Aus HBEFA übernehmen.

Ein Assistent öffnet sich.

Wählen Sie die gewünschten Verkehrszusammensetzungen in der Spalte Active aus.

Wählen Sie bei Bedarf ein Land aus.

Wählen Sie bei Bedarf ein Bezugsjahr aus.

Klicken Sie auf die Schaltfläche Weiter.

Die Seite Bitte wählen Sie die gewünschten Verkehrszusammensetzungen aus öffnet sich.

Die Seite enthält nur Verkehrszusammensetzungen, die dem eingestellten Land und dem Bezugsjahr entsprechen. Sie können auf die Schaltfläche Zurück klicken, wenn Sie Änderungen auf den vorigen Seiten vornehmen möchten.

Wählen Sie eine oder mehrere Verkehrszusammensetzungen in der Liste aus.

Halten Sie die Taste Strg gedrückt, wenn Sie mehrere Zeilen auswählen möchten.

Klicken Sie auf die Schaltfläche Weiter.

Die Seite Folgende Verkehrszusammensetzungen werden erzeugt öffnet sich.

Löschen Sie bei Bedarf Verkehrszusammensetzungen aus der Liste, indem Sie eine oder mehrere Zeilen markieren und danach auf die Schaltfläche Löschen klicken.

Klicken Sie die Schaltfläche Beenden.

Für jede aus HBEFA übernommene Verkehrszusammensetzung wird in eine Verkehrszusammensetzung eingefügt. Die Anteile der Fahrzeugschichten werden aus HBEFA übernommen.

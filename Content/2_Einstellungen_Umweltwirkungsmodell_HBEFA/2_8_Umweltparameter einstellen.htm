Umweltparameter einstellen

Sie können schon bestehende Umweltparameter aus der Umweltparameterdatei (*.upa) einlesen (Datei öffnen).

Wählen Sie im Menü Rechnen den Eintrag Umweltparameter.

Das Fenster Umweltparameter öffnet sich.

Wählen Sie das Register Basis.

Element

Beschreibung

Berechnungsgrundlage

Die Berechnungsgrundlage für alle Umweltberechnungsverfahren basiert auf verkehrlichen Belastungen.

Belastung-IV [Fzg]

Die Belastungswerte werden aus der Umlegung übernommen, das heißt, v_akt pro Strecke pro VSys ergibt sich aus t_akt über die streckentypspezifischen CR-Funktionen (Widerstandsfunktionen definieren und verwenden)

ZWert1 - ZWert3

Die Belastungswerte werden aus den ZWerten (Zählwerten) übernommen, das heißt, v_akt ergibt sich über die streckentypspezifischen CR-Funktionen (Widerstandsfunktionen definieren und verwenden).

Gültige Geschwindigkeiten

Wenn der Berechnung v_akt zugrunde liegt, können Sie für alle Umweltberechnungsverfahren die gültige Geschwindigkeit für PKW und LKW festlegen.

Hinweis

Wenn der Berechnung v_akt zugrunde liegt, gilt:

v_min wird berücksichtigt, wenn v_akt < v_min

v_max wird berücksichtigt, wenn v_akt > v_max

Lärm

In der Auswahlliste können Sie den Parameter für die Berechnung der Lärmbelastung auswählen (Lärmbelastungen berechnen).

mDTV

Für alle Umweltberechnungsverfahren kann ein Faktor angegeben werden, der je Streckenobertyp zur Umrechnung des durchschnittlichen täglichen Verkehrs DTV auf die maßgebende stündliche Verkehrsstärke M dient.

Wählen Sie das Register Schadstoffe.

Element

Beschreibung

Schadstoffe

In der Auswahlliste können Sie den Parameter für die Berechnung der Luftschadstoffemissionen auswählen (Luftschadstoffemissionen berechnen).

Die Änderungen werden übernommen.

Sie können Ihre Einstellungen in der Umweltparameterdatei (*.upa) speichern (Datei speichern).

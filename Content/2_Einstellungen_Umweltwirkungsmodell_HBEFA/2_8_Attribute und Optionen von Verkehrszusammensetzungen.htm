Attribute und Optionen von Verkehrszusammensetzungen

Element

Beschreibung

Nummer

Eindeutige Nummer der Verkehrszusammensetzung

Hinweis

Verkehrszusammensetzungen werden automatisch fortlaufend nummeriert. Sie können die vorgegebene Nummer mit einer Nummer überschreiben, die im Netz noch nicht vergeben ist.

Code

Kurzbezeichnung der Verkehrszusammensetzung

Name

Langbezeichnung der Verkehrszusammensetzung

Bezugsjahr

Gibt das Jahr an, dessen Kraftstoffqualität bei den Emissionsfaktoren zugrunde gelegt wird.

Kategorien

Ausgewählte Kategorien aus den sechs HBEFA-Verkehrszusammensetzungen, die der Verkehrszusammensetzung zugeordnet werden.

Hinweis

Mindestens eine Kategorie muss ausgewählt sein, eine Mehrfachauswahl ist möglich.

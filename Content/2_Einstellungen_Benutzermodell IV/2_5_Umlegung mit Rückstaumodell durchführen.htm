Umlegung mit Rückstaumodell durchführen

Sie können festlegen, dass gleichzeitig mit einer IV-Umlegung ein Rückstaumodell berechnet wird.

Legen Sie einen Verfahrensablauf mit dem gewünschten IV-Umlegungsverfahren fest (IV-Umlegung parametrisieren und rechnen).

Wählen Sie im Menü Rechnen den Eintrag Allgemeine Verfahrenseinstellungen.

Das Fenster Allgemeine Verfahrenseinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag IV-Einstellungen > Rückstaumodell.

Aktivieren Sie die Option Stauberechnung aktiv.

Die anderen Einstellungen in diesem Fenster können jetzt verändert werden.

Nehmen Sie die gewünschte Einstellung vor.

Element

Beschreibung

Stauberechnung auch während der Umlegung ausführen

Ist die Option ausgewählt, wird die Stauberechnung in der äußeren Iteration der IV-Umlegung ausgeführt, wobei die Ergebnisse in den Streckenwiderstand und damit in die Routenwahl eingehen.

Hinweis

Diese Betriebsart wird nicht empfohlen, weil sie die Konvergenz des jeweiligen Umlegungsverfahrens deutlich verschlechtert Siehe "Allgemeines zum Rückstaumodell".

Ist die Option nicht ausgewählt, wird das Rückstaumodell als Post-Processing im Anschluss an eine Umlegung gerechnet und beeinflusst somit die Routenwahl nicht.

Nehmen Sie ansonsten die gewünschten Einstellungen für das Rückstaumodell vor (Rückstaumodell im Anschluss an eine Umlegung durchführen).

Starten Sie den Verfahrensablauf.

Das gewählte Umlegungsverfahren wird in Kombination mit dem Rückstaumodell berechnet. Besteht bereits eine Umlegung für mindestens ein Nachfragesegment, welche nicht neu gerechnet wird, so wird außerdem vor Ausführung der Umlegung für die bereits umgelegten Nachfragesegmente das Rückstaumodell gerechnet.

Wenn alle bereits umgelegten IV-Nachfragesegmente Ihres Netzes umgelegt werden sollen, kann das Rückstaumodell mit einer Umlegung mit MPA gerechnet werden (Prozentuale Aufteilung der Nachfrage).

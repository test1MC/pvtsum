Beschreibung

Verteilungsfunktion des Value of Time

In der Tabelle legen Sie für jedes Nachfragesegment Parameter für die Berechnung des Value of Time fest (Grundlagen: Log-Normalverteilung der Zufallsvariable VT)

bezirksbeziehungsabhängig

Ist die Option ausgewählt, legen Sie bezirksbeziehungsabhängige Parameter für die Berechnung des Value of Time fest. Über die Schaltfläche, die in den Spalten Median, Streuungsparameter und Stdabw. der Lognormalvert. angezeigt wird, öffnet sich das Fenster Stützstellen für Value of Time für weitere Einstellungen (Bezirksbeziehungsabhängige Parameter für die Berechnung des Value of Time festlegen).

Ist die Option nicht ausgewählt, legen Sie für alle Bezirksbeziehungen die gleichen Parameter für die Verteilungsfunktion des Value of Time fest. Die Parameter geben Sie in die Spalten Median und Streuungsparameter ein.

Median

Median der Logarithmischen Normalverteilung der Zufallsvariable Value of Time (VT).

Streuungsparameter

Standardabweichung der assoziierten Standard-Normalverteilung

Aus diesem Wert berechnet automatisch die Standardabweichung der Logarithmischen Normalverteilung der Zufallsvariable Value of Time (VT) (Stdabw. der Lognormalvert.).

Hinweis

Diesen berechneten Wert müssen Sie als Standardabweichung eingeben, wenn Sie die Funktion beispielsweise in einem Tabellenkalkulationsprogramm darstellen möchten.

Faktor für Bezirkstyp verwenden

Ist die Option ausgewählt, können Sie über die Schaltfläche in der Spalte Faktoren pro Bezirkstyp einen Gewichtungsfaktor für den Value of Time festlegen.

Faktoren

Über die Schaltfläche öffnet sich ein Fenster für die Einstellung der Gewichtungsfaktoren (Gewichtungsfaktoren pro Bezirkstyp festlegen)

Element

Beschreibung

Abbruchbedingungen – global

maximale Anzahl Iterationen

Das Verfahren bricht nach der eingestellten Anzahl äußerer Iterationen ab.

maximale relative Abweichung von Widerstand und Maut

Das Verfahren bricht ab, wenn die relative Abweichung des gesamten Widerstands und der gesamten Mautkosten zwischen zwei aufeinander folgenden Iterationsschritten unterschritten wird.

Abbruchbedingungen – für den Ausgleich zwischen unterschiedlichen Mautlevels

maximale Abweichung in PKW-Einheiten

Der Ausgleich der Belastungen zwischen zwei Mautlevels bricht ab, wenn die Abweichung von der Verteilungsfunktion diesen Wert unterschreitet.

Abbruchbedingungen – für den Ausgleich von alternativen Routen mit gleicher Maut

zulässige Abweichung der Widerstände alternativer Routen

absolut

Der Ausgleich zwischen zwei Alternativrouten eines Mautlevels bricht ab, wenn die Differenz der Widerstände den hier festgelegten Wert unterschreitet.

relativ

Der Ausgleich zwischen zwei Alternativrouten eines Mautlevels bricht ab, wenn die relative Abweichung der Widerstände den festgelegten Wert unterschreitet.
Bezirksbeziehungsabhängige Parameter für die Berechnung des Value of Time festlegen

Im Fenster Stützstellen für Value of Time legen Sie bezirksbeziehungsabhängige Parameter für die Berechnung des Value of Time fest. Für bestimmte Attributwerte eines gewählten Bezirksbeziehungsattributs legen Sie die Parameter fest. Ausgehend von diesen Stützstellen werden für die dazwischen liegenden Attributwerte die Parameter automatisch linear interpoliert.

Element

Beschreibung

Bezirksbeziehungsattribut

Über die Schaltfläche wählen Sie das Attribut aus, dessen Werte Sie als Stützstellen verwenden möchten.

Hinweis

In der Regel werden Sie das Attribut Matrix-Wert verwenden. Die Werte können beispielsweise aus Infrastrukturdaten der Bezirke berechnet worden sein.

Einfügen

Über die Schaltfläche fügen Sie eine neue Zeile für eine neue Stützstelle ein.

Löschen

Über die Schaltfläche löschen Sie die aktuell markierten Zeilen.

Attributwert

Geben Sie hier den Attributwert ein, für den Sie die Parameter festlegen möchten.

Median

Median der Logarithmischen Normalverteilung der Zufallsvariable Value of Time (VT).

Streuungsparameter

Standardabweichung der assoziierten Standard-Normalverteilung

Aus diesem Wert berechnet automatisch die Standardabweichung der Logarithmischen Normalverteilung der Zufallsvariable Value of Time (VT) (Stdabw. der Lognormalvert.).

Hinweis

Diesen berechneten Wert müssen Sie als Standardabweichung eingeben, wenn Sie die Funktion beispielsweise in einem Tabellenkalkulationsprogramm darstellen möchten.

Auswerten

Über diese Schaltfläche öffnet sich ein Fenster, in dem Sie für eine bestimmte Bezirksbeziehung den Wert des gewählten Bezirksbeziehungsattributs und die sich daraus ergebenden Parameter Median und Streuungsparameter anzeigen können.
Gewichtungsfaktoren pro Bezirkstyp festlegen

Element

Beschreibung

Gewichtungsfaktor des Quellbezirks

Gewichtungsfaktor des Value of Time für das Quellverkehrsaufkommen eines Bezirks. Der Zielfaktor ergibt sich daraus automatisch als 1 - Quellfaktor. Standardmäßig sind Quellfaktor und Zielfaktor mit einem Wert von 0.5 gleich gewichtet.

Gewichtungsfaktoren pro Bezirkstyp

Über die Eingabefelder können Sie pro Bezirkstyp einen Gewichtungsfaktor für den Value of Time festlegen. Typischerweise liegen die Faktoren um den Wert 1. Ein Faktor größer 1 bedeutet beispielsweise, dass Verkehr in diesen Bezirk oder aus diesem Bezirk einen höheren Value of Time besitzt.

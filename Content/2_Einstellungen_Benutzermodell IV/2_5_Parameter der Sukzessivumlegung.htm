Parameter der Sukzessivumlegung

Element

Beschreibung

1 bis 12

Pro Iterationsschritt werden die hier angegebenen Anteile der Nachfrage umgelegt.

Hinweise

Sie können maximal zwölf Iterationsschritte durchführen.

Die Summe der Prozente muss 100 % betragen. Wird für einen Schritt 0 % gesetzt, werden weitere Anteile ignoriert.

Tipp

Wenn Sie weniger als 100 % der Matrix umlegen möchten, können Sie vorher mittels Matrix-Multiplikation mit einem Faktor < 1 eine Teilmatrix erzeugen (Matrixwerte multiplizieren).

CR-Funktionen für Streckentypen auswählen

Sie können eine Liste von CR-Funktionen mit benutzerdefinierten Parametern zusammenstellen und diese anschließend Streckentypen zuordnen.

Wählen Sie im Menü Rechnen den Eintrag Allgemeine Verfahrenseinstellungen.

Das Fenster Allgemeine Verfahrenseinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag IV-Einstellungen >CR-Kurven.

Wählen Sie im Bereich CR-Kurven die gewünschten CR-Funktionen aus (CR-Funktionen auswählen und Parameter einstellen ).

Wählen Sie im Bereich Streckentypen die gewünschten Streckentypen aus.

Die Tabelle zeigt die Zuordnung der CR-Funktionen zu den 100 Streckentypen. Die Zeilennummer repräsentiert dabei die Zehnerstelle der Streckentypnummer, die Spaltennummer die Einerstelle. Beispielsweise repräsentiert die Zelle in der zweiten Zeile und dritten Spalte den Streckentyp 12. Die Zellen enthalten die Nummer der aktuell zugeordneten CR-Funktion.

Sie können mehrere Zellen auswählen.

  • Wählen Sie mehrere einzelne Zellen aus, indem Sie die Taste Strg drücken und auf jede Zelle einzeln klicken.
  • Alternativ können Sie einen Bereich von mehreren zusammenhängenden Zellen auswählen, indem Sie den Mauszeiger mit gedrückter linker Maustaste über den gewünschten Bereich ziehen. Anschließend können Sie die grobe Auswahl nachbearbeiten. Drücken Sie die Taste Strg und klicken Sie auf eine beliebige Zelle, um diese Zelle der Auswahl hinzuzufügen oder daraus zu entfernen.

Geben Sie in das Eingabefeld Kurve für selektierte Felder die Nummer der gewünschten CR-Funktion ein.

Der Wert wird für die ausgewählten Zellen übernommen.

Aktivieren Sie, falls gewünscht, die Option vMin berücksichtigen.

Ist die Option ausgewählt, wird die minimale Geschwindigkeit einer Strecke auf die minimale Geschwindigkeit des Streckentyps (Staugeschwindigkeit) begrenzt (Streckentypen festlegen).

Die Änderungen werden übernommen.

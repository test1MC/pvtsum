Einstellungen für die Berechnung von IV-Kenngrößenmatrizen

Betrachtete Relationen

Element

Beschreibung

Nur Beziehungen mit Nachfrage > 0 berechnen

Ist die Option ausgewählt, werden nur die Quelle-Ziel-Beziehungen berücksichtigt, deren Verkehrsnachfrage größer null ist.

Hinweis

Quelle-Ziel-Beziehungen mit Verkehrsnachfrage = 0 erhalten den Kenngrößenwert null.

Filter-Auswahlliste

Über die Auswahlliste können Sie die betrachteten Bezirksbeziehungen weiter einschränken, die der aktuellen Einstellung der Option Nur Beziehungen mit Nachfrage > 0 berechnen entsprechen.

Hinweis

Für alle Bezirksbeziehungen, die aufgrund der gewählten Option nicht ausgewertet werden, enthält die Kenngrößenmatrix den Wert, der für Ignorierte Beziehungen gesetzt ist über Menü Rechnen > Allgemeine Verfahrenseinstellungen > IV-Einstellungen > Kenngrößen (Metadaten für Kenngrößenmatrizen festlegen).

alle

Ist die Option ausgewählt, werden alle Bezirksbeziehungen ausgewertet.

Nur aktive Beziehungen

Ist die Option ausgewählt, werden nur die aktiven Beziehungen ausgewertet.

Nur aktive Quell-Bezirke

Ist die Option ausgewählt, werden nur Beziehungen mit einem aktiven Quellbezirk ausgewertet.

Nur Paare aktiver Bezirke

Ist die Option ausgewählt, werden nur Beziehungen zwischen aktiven Bezirken ausgewertet.

Hinweise

Die Einstellung der betrachteten Bezirksbeziehungen für die Kenngrößenberechnung kann von derjenigen für die Umlegung abweichen.

Ist für die Umlegung ein einschränkendes Bezirksintervall definiert, schränkt dies die betrachteten Bezirksbeziehungen noch weiter ein.

Beim Einlesen von Parametern aus älteren Versionen von wird die damalige Auswahl dieser Option entweder als alle oder als Nur Paare aktiver Bezirke interpretiert.

Kriterium für die Wegesuche

Kriterium für die Auswahl des Wegs bzw. der Wege, aus denen pro Beziehung der Kenngrößenwert ermittelt wird.

tAkt

Widerstand

Abhängig von der Option Wege aus der Umlegung verwenden

  • Ist die Option nicht ausgewählt, wird der Weg mit dem niedrigsten Widerstand für das IV-VSys entsprechend der aktuellen Widerstandsfunktion verwendet.
  • Ist die Option ausgewählt, wird der aggregierte Wert aus allen Wegen verwendet.

Wege aus der Umlegung verwenden

Ist die Option ausgewählt, wird der Kenngrößenwert aus allen Wegen einer Beziehung ermittelt, gemäß der Option Gewichtung der Wege.

Ist die Option nicht ausgewählt, wird der Weg mit minimalem Widerstand für das IV-VSys entsprechend der aktuellen Widerstandsfunktion verwendet.

Hinweis

Die Option steht nur zur Verfügung, wenn als Kriterium für die Wegesuche die Option Widerstand gewählt ist.

Methode zur Ermittlung des Kenngrößenwerts aus den Werten aller Wege

Maximum je Kenngröße

Wert des Wegs mit dem höchsten Wert

Mittel über die Wege

Mittelwert über alle Wege

Mittel über Wegebelastung

Mittelwert über alle Wege, gewichtet mit der jeweiligen Belastung des Weges

Hinweise

Die Option steht nur zur Verfügung, wenn die Option Wege aus der Umlegung verwenden ausgewählt ist.

Für Beziehungen, für die in der Umlegung keine Wege berechnet wurden, wird der Weg mit dem geringsten Widerstand ermittelt.

Element

Beschreibung

Liste der Kenngrößen

Berechnen

Ist die Option ausgewählt, wird die entsprechende Kenngröße berechnet.

Auf Datei schreiben

Ist die Option ausgewählt, wird die entsprechende Kenngröße als Kenngrößenmatrixdatei ausgegeben. Weitere Einstellungen zur Ausgabe treffen Sie im Bereich Ausgabedatei (siehe „Ausgabedatei einer IV-Kenngrößenmatrix“ auf Seite 1126). Für jede Kenngröße, die Sie in eine Datei ausgeben, erzeugt eine Datei mit einer spezifischen Dateierweiterung, die auf einer englischen Abkürzung beruht.

Hinweise

Kenngrößenmatrixdateien, die mit älteren -Versionen erzeugt wurden, enthalten sprachabhängige Dateierweiterungen, die sogar benutzerdefiniert überschrieben worden sein können.

Diese Option ist ausgegraut, wenn die Option Berechnen nicht ausgewählt ist.

Öffnen

Ist die Option ausgewählt, wird die jeweilige Kenngrößenmatrix nach der Berechnung geöffnet.

Hinweis

Diese Option ist ausgegraut, wenn die Option Berechnen nicht ausgewählt ist.

Tipp

Sie können jede der zuvor genannten Spalten insgesamt aktivieren oder deaktivieren, indem Sie mit der rechten Maustaste auf den Spaltenkopf klicken und im Kontextmenü die Option Spalte aktivieren bzw. Spalte deaktivieren auswählen.

Format

Über die Auswahlliste wählen Sie das Format, in dem die Matrizen gespeichert werden sollen (Datenstrukturen von Matrizen in -spezifischen Textformaten).

  • Binär
  • V-Format (Textformat)
  • O-Format (Textformat)
  • E-Format (Textformat)
  • S-Format (Textformat)
  • CSV-Format (Textformat)

Trennzeichen

Über die Auswahlliste wählen Sie das Zeichen, mit dem die Einträge in der Textdatei getrennt werden.

Hinweise

Diese Auswahlliste ist nur für das Format CSV-Format verfügbar.

Für alle anderen Formate wird das Leerzeichen als Trennzeichen verwendet, damit die Matrix im Matrixeditor editiert werden kann.

Matrizen im Format *.csv mit Komma als Trennzeichnen können per Doppelklick als Excel-Datei geöffnet werden. Matrizen im Format *.csv mit einem anderen Trennzeichen müssen mit dem Excel-Assistent eingelesen werden.

Überschreiben bestätigen

Ist die Option ausgewählt, wird eine Meldung angezeigt, falls schon Matrizen mit dem gleichen Namen vorhanden sind. Sie können die Meldung dann bestätigen oder die Ausgabe abbrechen.

Ist die Option nicht ausgewählt, werden vorhandene Matrizen mit dem gleichen Namen ohne weitere Nachfrage überschrieben.

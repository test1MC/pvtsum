CR-Funktionen aus Umlegung mit ICA für andere Umlegung verwenden

Über eine Umlegung mit ICA berechnen Sie CR-Funktionsparameter für abbiegerfeine CR-Funktionen. Diese Parameter können Sie anschließend für eine andere Umlegung verwenden.

Rechnen Sie eine Umlegung mit ICA (Umlegung mit ICA).

Stellen Sie für die gewünschten Knoten die Methode für Knotenwiderstandsberechnung auf Aus letzter Umlegung mit ICA (Methode für Knotenwiderstandsberechnung festlegen).

Berechnen Sie die gewünschte Umlegung, beispielsweise eine Gleichgewichtsumlegung.

Grundbelastung berücksichtigen

Bei einer Umlegung können Sie eine vorliegende Grundbelastung berücksichtigen, die aus vorliegenden Attributdaten eingelesen und wie eine IV-Belastung q behandelt wird. Sie können zwischen einfacher und detaillierter Berechnung der Grundbelastung wählen.

Wählen Sie im Menü Rechnen den Eintrag Allgemeine Verfahrenseinstellungen.

Das Fenster Allgemeine Verfahrenseinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag IV-Einstellungen > Umlegung.

Nehmen Sie im Bereich Grundbelastung die gewünschten Einstellungen vor.

Element

Beschreibung

fixieren

Ist die Option ausgewählt, können Sie eine vorliegende Grundbelastung aus Attributen der Netzobjekte auslesen und bei der Umlegung berücksichtigen.

einfache Berechnung

Ist die Option ausgewählt, müssen Sie einen einheitlichen ZWert auswählen, der die Grundbelastung für alle überfahrenen Netzobjekttypen enthält (beispielsweise ZWert2, also StreckenZWert2, OberabbiegerZWert2, AbbiegerZWert2 und AnbindungsZWert2). Die jeweiligen Attributwerte des gewählten ZWerts können Sie mit einem einheitlichen Faktor multiplizieren.

detaillierte Berechnung

Ist die Option ausgewählt, können Sie für jeden überfahrenen Netzobjekttyp ein spezifisches Attribut auswählen, das die Grundbelastung für die überfahrenen Netzobjekte des Typs enthält. Für jeden Netzobjekttyp können Sie die jeweiligen Attributwerte mit einem spezifischen Faktor multiplizieren.

Die Änderungen werden übernommen. Sobald Sie eine IV-Umlegung durchführen, werden die neuen Einstellungen verwendet.

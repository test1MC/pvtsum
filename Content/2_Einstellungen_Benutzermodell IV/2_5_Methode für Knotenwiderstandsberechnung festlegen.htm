Methode für Knotenwiderstandsberechnung festlegen

Sie wählen eine Standardmethode für die Knotenwiderstandsberechnung, die zunächst für alle Knoten gilt. Für einzelne Knoten können Sie eine andere Methode festlegen.

Standardmethode für Knotenwiderstandsberechnung einstellen

Sie können eine Standardmethode für die Berechnung von Knotenwiderständen einstellen. Diese wird für alle Knoten verwendet, deren Attribut Eingestellte Methode für Widerstand am Knoten verwenden Falsch ist.

Setzen Sie für die gewünschten Knoten den Attributwert des Attributs Eingestellte Methode für Widerstand am Knoten verwenden auf Falsch.

Wählen Sie im Menü Rechnen den Eintrag Allgemeine Verfahrenseinstellungen.

Das Fenster Allgemeine Verfahrenseinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag IV-Einstellungen > Widerstände am Knoten.

Wählen Sie in der Auswahlliste Standard-Methode den gewünschten Eintrag Siehe "Widerstände am Knoten".

Nehmen Sie für die gewählte Methode die gewünschten Einstellungen vor.

Element

Beschreibung

Abbieger CR-Funktion

( Knotenwiderstände nach der Methode Abbieger CR-Funktion berechnen )

Knoten CR-Funktion

( Knotenwiderstände nach der Methode Knoten CR-Funktion berechnen )

Knotenwiderstandsberechnung (ICA)

( Knotenwiderstände nach der Methode Knotenwiderstandsberechnung (ICA) berechnen )

Die Standardmethode ist eingestellt.

Methode für Knotenwiderstandsberechnung für einzelne Knoten festlegen

Sie können für bestimmte Knoten eine Methode für die Knotenwiderstandsberechnung festlegen, die von der Standardmethode abweicht.

Setzen Sie für die gewünschten Knoten den Attributwert des Attributs Eingestellte Methode für Widerstand am Knoten verwenden auf Wahr.

Wählen Sie für die gewünschten Knoten als Methode für Widerstand am Knoten die gewünschte Methode.

Nehmen Sie für die gewählte Methode die gewünschten Einstellungen vor.

Element

Beschreibung

Abbieger CR-Funktion

( Knotenwiderstände nach der Methode Abbieger CR-Funktion berechnen )

Knoten CR-Funktion

( Knotenwiderstände nach der Methode Knoten CR-Funktion berechnen )

Knotenwiderstandsberechnung (ICA)

( Knotenwiderstände nach der Methode Knotenwiderstandsberechnung (ICA) berechnen )

Für die gewählten Knoten ist die gewünschte Methode zur Knotenwiderstandsberechnung eingestellt.

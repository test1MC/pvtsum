ÖV-Kenngrößenmatrizen: Bereich Ausgabedatei

Dieser Bereich steht nur zur Verfügung, wenn Sie mindestens eine Kenngrößenmatrix als Datei ausgeben.

Element

Beschreibung

Dateiname

 

 

Name der ausgegebenen Kenngrößenmatrixdateien. Über die Schaltfläche geben Sie Pfad und Dateiname der Datei ohne Dateierweiterung ein.

Hinweise

Wenn Sie den Dateinamen ohne Pfad eingeben, wird die Datei in das Projektverzeichnis für Matrizen ausgegeben.

ergänzt die Dateierweiterung(en) gemäß dem eingestellten Format und den Einstellungen im Menü Rechnen > Allgemeine Verfahrenseinstellungen > ÖV-Einstellungen > Kenngrößen (Metadaten für Kenngrößenmatrizen festlegen). Für jede Kenngröße, die Sie auf Datei ausgeben, erzeugt eine Datei mit einer spezifischen Dateierweiterung, die auf einer englischen Abkürzung beruht. Werden Kenngrößenmatrizen mit einer Umlegung mehrerer Nachfragesegmenten für mehrere Nachfragesegmente erzeugt, bleibt die NSeg-Zuordnung jeder Ausgabematrix sowohl im Speicher als auch im Dateinamen erhalten. Der benutzerdefinierte Dateiname wird in jedem Fall um den NSeg-Code ergänzt.

Hinweis

Die Dateierweiterung wird als Beispiel unter dem Eingabefeld angezeigt.

Format

Über die Auswahlliste wählen Sie das Format, in dem die Matrizen gespeichert werden sollen (Datenstrukturen von Matrizen in -spezifischen Textformaten).

  • binär
  • O-Format (Textformat)
  • V-Format (Textformat)
  • Teil-V-Format (Textformat)
  • CSV-Format (Textformat)

Trennzeichen

Über die Auswahlliste wählen Sie das Zeichen, mit dem die Einträge in der Textdatei getrennt werden.

Hinweise

Diese Auswahlliste ist nur für das Format CSV-Format verfügbar.

Für alle anderen Formate wird das Leerzeichen als Trennzeichen verwendet, damit die Matrix im Matrixeditor editiert werden kann.

Matrizen im Format *.csv mit Komma als Trennzeichnen können per Doppelklick als Excel-Datei geöffnet werden. Matrizen im Format *.csv mit einem anderen Trennzeichen müssen mit dem Excel-Assistent eingelesen werden.

Verkehrsmittelnr (VISEM)

Verkehrsmittelnummer (für VISEM)

1 = Fuß

2 = PKW-Mitfahrer

3 = ÖV

4 = PKW-Selbstfahrer (einschließlich LKW) etc.

Überschreiben bestätigen

Ist die Option ausgewählt, wird eine Meldung angezeigt, falls schon Matrizen mit dem gleichen Namen vorhanden sind. Sie können die Meldung dann bestätigen oder die Ausgabe abbrechen.

Ist die Option nicht ausgewählt, werden vorhandene Matrizen mit dem gleichen Namen ohne weitere Nachfrage überschrieben.

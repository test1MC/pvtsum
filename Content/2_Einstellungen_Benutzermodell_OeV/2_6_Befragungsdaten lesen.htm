Befragungsdaten lesen

Stellen Sie sicher, dass Ihre Befragungsdaten einem der erforderlichen Formate entsprechen (Erhebungsdaten vorbereiten).

Klicken Sie in der Symbolleiste auf das Symbol  Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Befragungsdaten lesen aus der Kategorie ÖV-Fahrgasterhebungen ein (Verfahrensablauf festlegen und starten).

Klicken Sie in der Zeile des neu eingefügten Verfahrens auf die Schaltfläche in der Spalte Bezugsobjekt(e).

Das Fenster Nachfragesegment Auswahl öffnet sich.

Wählen Sie das gewünschte Nachfragesegment.

In der Spalte Bezugsobjekt(e) wird das gewählte Nachfragesegment angezeigt.

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist, und klicken Sie im Bereich Aktionen auf Bearbeiten.

Das Fenster Befragungsdaten lesen öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (Einstellungen zum Einlesen von Erhebungsdaten).

Klicken Sie in der Zeile des eingefügten Verfahrens in die Spalte Variante/Datei.

Das Symbol wird angezeigt.

Klicken Sie auf das Symbol .

Das Fenster Befragungsdaten öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Der Dateiname wird in die Spalte Variante/Datei übernommen.

Alternativ können Sie in das Feld in der Spalte Variante/Datei doppelklicken und Pfad und Dateinamen manuell eingeben.

Führen Sie das Verfahren aus (Verfahren im Verfahrensablauf ausführen).

Falls Datensätze fehlerhafte Daten enthalten, wird pro Datensatz eine entsprechende Fehlermeldung angezeigt. Die fehlerhaften Datensätze werden nicht eingelesen.

Die Datei mit den Befragungsdaten wird eingelesen. Die Daten werden als verbindungsfeine ÖV-Wege gespeichert, fehlende Bezirke und Anbindungen werden entsprechend Ihren Einstellungen erzeugt.

Sie können die eingelesenen Daten in den Listen ÖV-Wege und ÖV-Teilwege anzeigen (Liste öffnen).

Neu eingelesene Datensätze haben zunächst die Plausibilisierungsstatus 0, also noch nicht plausibilisiert. Im nächsten Schritt müssen Sie die Daten plausibilisieren, damit Sie sie anschließend umlegen können.

Wenn Sie eingestellt haben, dass Belastungen des gewählten Nachfragesegments bei einem anderen Ziel-Nachfragesegment gespeichert werden (Belastung je Nachfragesegment oder additiv speichern), wird die Einstellung beim Einlesen automatisch zurückgesetzt (Ziel-Nachfragesegment = gewähltes Nachfragesegment).

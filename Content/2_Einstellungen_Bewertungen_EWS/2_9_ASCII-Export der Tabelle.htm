ASCII-Export der Tabelle

Sie können die Berechnungsergebnisse als ASCII-Datei *.rwf speichern.

Stellen Sie sicher, dass Sie EWS aufgerufen haben (EWS aufrufen).

Wählen Sie im Menü EWS den Eintrag ASCII-Export.

Das Fenster RASW-Datei speichern öffnet sich.

Geben Sie den gewünschten Namen der Datei ein.

Bestätigen Sie mit OK.

Die Datei wird im Format *.rwf gespeichert.

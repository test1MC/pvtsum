Manuelle Nachbearbeitung des schematischen Liniennetzplans

Anschlussknoten, Kantenverläufe und Ansatzpunkte („Slots”) der Fahrplanfahrten können Sie im schematischen Liniennetzplan markieren und direkt manuell bearbeiten. Folgende Bearbeitungsmöglichkeiten bestehen:

  • Anschlussknoten als Zwischenhalte einfügen
  • Anschlussknoten verschieben
  • Anschlussknoten vergrößern
  • Anschlussknoten auf minimale Größe reduzieren
  • Anschlussknoten zu Nachbarknoten wie im Netz positionieren
  • Kantenverläufe und Slots an Knoten neu anordnen
  • Anschlussknoten löschen
  • Ansatzpunkte der Fahrplanfahrten neu positionieren
  • Kanten zusammenfassen
  • Verläufe der Kanten ändern
  • Kanten trennen
  • Fahrplanfahrten aus der Ansicht löschen

Sie können einen gewünschten Anschlussknoten oder eine gewünschte Fahrplanfahrt zentriert anzeigen, indem Sie die Funktionen Anschlussknoten suchen oder Fahrplanfahrt suchen im Menü Schematischer Liniennetzplan nutzen.
Anschlussknoten als Zwischenhalte einfügen

Sie können auf einem Kantenverlauf ausgeblendete Anschlussknoten als Zwischenhalte einfügen. Dabei werden zusammengefasste Kanten in der Regel nicht getrennt. Eine Trennung wäre beispielsweise dann notwendig, wenn ein Teil der zusammengefassten Fahrplanfahrten den neuen Anschlussknoten nicht anfährt.

Markieren Sie den gewünschten Kantenverlauf.

Klicken Sie mit der rechten Maustaste auf den markierten Kantenverlauf.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Zwischenhalte einfügen.

Der Eintrag Zwischenhalte einfügen ist nur aktiv, wenn sich auf dem ausgewählten Kantenverlauf ausgeblendete Anschlussknoten befinden.

Das Fenster Zwischenhalte einfügen öffnet sich.

Wählen Sie bei Bedarf die Option Nur Haltestellen anzeigen, an denen Ein- und Aussteigen erlaubt ist aus, um die Menge der angezeigten Anschlussknoten einzuschränken.

Wählen Sie in der Liste der möglichen Anschlussknoten in der Spalte Anschlussknoten einfügen die gewünschten Anschlussknoten aus.

Die Anschlussknoten werden als Zwischenhalte angezeigt.

Anschlussknoten verschieben

Sie können die Anschlussknoten in der Ansicht wie folgt verschieben.

Klicken Sie auf den gewünschten Anschlussknoten und halten Sie die Maustaste gedrückt.

Ziehen Sie den Anschlussknoten an die gewünschte Position.

Der Anschlussknoten wird verschoben. Die Kantenverläufe werden automatisch angepasst.

Sie können mehrere Anschlussknoten gleichzeitig verschieben, wenn Sie beim Markieren die Taste Strg gedrückt halten.

Wenn Sie beim Verschieben die Taste Umschalt gedrückt halten, werden je nach Verschieberichtung alle links bzw. rechts davon oder auf gleicher Koordinate liegenden Anschlussknoten mit verschoben. Auf diese Weise bleiben die Abstände der Anschlussknoten erhalten.
Anschlussknoten vergrößern

Sie können den Anschlussknoten wie folgt vergrößern.

Klicken Sie auf eine beliebige Ecke des gewünschten Anschlussknotens und halten Sie die Maustaste gedrückt.

Ziehen Sie an dem Eckpunkt bis der Anschlussknoten die gewünschte Größe erreicht hat.

Der Anschlussknoten wird vergrößert.

Anschlussknoten auf minimale Größe reduzieren

Sie können den Anschlussknoten wie folgt auf die minimale Größe reduzieren. Die Anzeige ist dann so groß, dass alle Ansatzpunkte noch angeordnet werden können.

Klicken Sie mit der rechten Maustaste auf den gewünschten Anschlussknoten.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Minimale Größe.

Der Anschlussknoten wird auf die minimale Größe reduziert.

Sie können mehrere Anschlussknoten gleichzeitig verkleinern, wenn Sie beim Markieren die Taste Strg gedrückt halten.
Anschlussknoten zu Nachbarknoten wie im Netz positionieren

Sie können einen Anschlussknoten relativ zu seinen Nachbarknoten im Netz positionieren, wenn der Anschlussknoten mindestens zwei Nachbarknoten hat.

Klicken Sie mit der rechten Maustaste auf den gewünschten Anschlussknoten.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Positioniere zu Nachbarknoten wie im Netz.

Der Anschlussknoten wird relativ zu seinen Nachbarknoten wie im Netz positioniert.

Kantenverläufe und Slots an Knoten neu anordnen

Sie können die Positionen der Kantenverläufe und Slots an den Anschlussknoten optimieren, beispielsweise nachdem Sie einen Anschlussknoten verschoben haben.

Klicken Sie mit der rechten Maustaste auf den gewünschten Anschlussknoten.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Kantenverläufe und Slots an Knoten neu anordnen.

Die Kantenverläufe und Slots werden am markierten Knoten neu angeordnet.

Sie können die Slots und Kantenverläufe mehrerer Anschlussknoten gleichzeitig neu anordnen, wenn Sie beim Markieren der Anschlussknoten die Taste Strg gedrückt halten.
Anschlussknoten löschen

Sie können die Anschlussknoten in der Ansicht wie folgt löschen.

Klicken Sie mit der rechten Maustaste auf den gewünschten Anschlussknoten.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Löschen.

Der Anschlussknoten wird gelöscht.

Sie können mehrere Anschlussknoten gleichzeitig löschen, wenn Sie beim Markieren die Taste Strg gedrückt halten.
Slots der Fahrplanfahrten neu positionieren

Sie können die Slots der Fahrplanfahrten wie folgt neu positionieren.

Klicken Sie auf den gewünschten Ansatzpunkt einer Fahrplanfahrt am Anschlussknoten.

Der Ansatzpunkt wird durch ein Quadrat in der Größe des getroffenen Rasterpunkts hervorgehoben.

Klicken Sie auf das Quadrat und halten Sie die Maustaste gedrückt.

Ziehen Sie das Quadrat an die gewünschte Position.

Sie können das Quadrat ausschließlich auf der Knotenumrandung des ausgewählten Anschlussknotens verschieben.

Die neue Position wird durch ein Rasterquadrat dargestellt.

Sie können mehrere Ansatzpunkte eines Anschlussknotens gleichzeitig verschieben, wenn Sie beim Markieren die Taste Strg gedrückt halten.

Der Ansatzpunkt wird an die neue Position verschoben.

Wird ein Ansatzpunkt auf einen anderen Ansatzpunkt verschoben, bleiben die Kanten getrennt und die Ansatzpunkte werden verschoben. Wenn Sie die Taste Umschalt gedrückt halten, werden die zugehörigen Kanten zusammengefasst.
Kanten zusammenfassen

Wird ein Ansatzpunkt auf einen anderen Ansatzpunkt verschoben, werden die zugehörigen Kanten zusammengefasst, wenn sie die gleichen Anfangs- und Endanschlussknoten haben.

Klicken Sie auf den gewünschten Ansatzpunkt einer Fahrplanfahrt am Anschlussknoten.

Der Ansatzpunkt wird durch ein Quadrat in der Größe des getroffenen Rasterpunktes hervorgehoben.

Ziehen Sie das Quadrat an die Position des Ansatzpunkts, dessen Kanten Sie mit den Kanten des ausgewählten Ansatzpunkts zusammenfassen möchten.

Die Kanten werden zusammengefasst.

Verläufe der Kanten ändern

Sie können den Verlauf der angezeigten Kantenverläufe wie folgt ändern.

Klicken Sie mit der rechten Maustaste auf den Kantenverlauf, den Sie digitalisieren möchten.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Digitalisieren.

Auf dem markierten Kantenverlauf werden kleine Quadrate angezeigt. Sie markieren die Punkte, an denen der Kantenverlauf verändert werden kann.

Klicken Sie im Verlauf auf den Punkt, den Sie neu positionieren möchten, und halten Sie die Maustaste gedrückt.

Ziehen Sie den Punkt an die gewünschte Position.

Ziehen Sie bei Bedarf weitere Punkte an andere Positionen.

Löschen Sie bei Bedarf vorhandene Punkte indem Sie die Taste Strg drücken und auf den Punkt klicken.

Fügen Sie bei Bedarf weitere Punkte ein, indem Sie die Taste Strg drücken und auf den Verlauf klicken.

Drücken Sie die Taste Enter.

Der Kantenverlauf wird geändert.

Die Kantenverläufe richten sich immer an den Gitterpunkten aus, deshalb ergänzt , falls nötig, zusätzliche Punkte und passt den Kantenverlauf entsprechend an (Hilfsgitter anzeigen).

Wenn Sie einen Kantenverlauf innerhalb eines Anschlussknotens verändern, beachten Sie, dass Sie Digitalisierungspunkte nicht außerhalb des Anschlussknotens .verschieben können. Richten Sie die Kantenverläufe an einem Anschlussknoten neu aus (Kantenverläufe und Slots an Knoten neu anordnen) werden auch alle Kantenverläufe innerhalb des Anschlussknotens neu ausgerichtet und die Digitalisierungspunkte innerhalb des Anschlussknotens entfallen.

Verschieben Sie den Anschlussknoten, bleiben die Digitalisierungspunkte im Knoten erhalten.

Ändern Sie die Größe des Anschlussknoten, werden alle Digitalisierungspunkte, die danach außerhalb des Knotens liegen, gelöscht.
Kanten trennen

Liegen unter einem Kantenverlauf mehrere Kanten, können diese getrennt werden.

Klicken Sie mit der rechten Maustaste auf den gewünschten Kantenverlauf in der Darstellung.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Kanten trennen.

Die Kanten des Kantenverlaufs werden getrennt.

Alternativ können Sie alle Kanten der Kantenverläufe trennen (Alle Kanten trennen).
Fahrplanfahrten aus der Ansicht löschen

Sie können Fahrplanfahrten folgendermaßen aus der Anzeige löschen.

Klicken Sie mit der rechten Maustaste auf die gewünschte Kante.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Fahrplanfahrten aus Fahrplanfahrten-Auswahl löschen.

Alle Fahrplanfahrten, die mit der Kante assoziiert sind, werden aus der Fahrplanfahrten-Auswahl gelöscht.

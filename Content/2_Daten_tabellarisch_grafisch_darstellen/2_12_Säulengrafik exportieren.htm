Säulengrafik exportieren

Wenn Sie eine Säulengrafik erstellt haben, können Sie die einzelnen Diagramme anschließend in verschiedenen Grafikformaten exportieren oder direkt über die Zwischenablage in Berichte einfügen.

Klicken Sie im Fenster Säulengrafik für <Netzobjekt, Nummer> mit der rechten Maustaste auf das Diagramm, das Sie exportieren möchten.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Exportieren.

Das Fenster Diagramm Exportieren öffnet sich.

Element

Beschreibung

Diagramm in die Zwischenablage kopieren

Unkomprimiertes Bitmap

Ist die Option ausgewählt, wird das Diagramm als unkomprimiertes Bitmap exportiert.

Vektorgrafik

Ist die Option ausgewählt, wird das Diagramm als Vektorgrafik exportiert.

Kopieren

Sie können das ausgewählte Diagramm in die Zwischenablage kopieren, indem Sie auf die Schaltfläche klicken.

Tipp

Anschließend können Sie das Diagramm mit der Einfügefunktion in das Fremdprogramm einfügen.

Diagramm als Datei exportieren

Speichern

Sie können das Diagramm in eine Grafikdatei exportieren, indem Sie auf die Schaltfläche klicken.

Hinweis

Sie können im Fenster Bilddatei speichern die Dateierweiterungen *.emf, *.jpg oder *.bmp festlegen.

Speichern Sie die Datei am gewünschten Ort ab.

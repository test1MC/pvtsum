Zeilen einer Liste aggregieren

Mit dieser Funktion können Sie Zeilen mit gleichem Attributwert in einer Zeile zusammenfassen. Dabei müssen Sie für alle anderen Attribute (also Spalten) angeben, wie deren Attributwerte aggregiert werden sollen.

Eine Gruppierung können Sie für eine oder mehrere Spalten festlegen.

Beispiel: Sie möchten die durchschnittliche Länge der Strecken des Typs 20 ermitteln. Dazu gruppieren Sie die Liste der Strecken nach dem Attribut TypNr. Für das Attribut Länge legen Sie die Aggregationsfunktion Durchschnitt fest.

Sie können die Zahlen durch Kopieren und Einfügen ausgeben (Ausgewählte Attributwerte kopieren und einfügen).

Diese Funktion steht nicht für Wegelisten und Statistiken zur Verfügung.

Stellen Sie sicher, dass das Listenfenster der gewünschten Liste geöffnet ist.

Wählen Sie die Spalten aus, nach denen Sie gruppieren möchten (Listenbereiche auswählen).

Klicken Sie mit der rechten Maustaste auf eine der gewählten Spalten.

Wählen Sie den Eintrag Spalte gruppieren.

Alternativ können Sie die Gruppierung im Attributauswahlfenster festlegen (Anzeige-Eigenschaften von Attributen ändern).

Die Liste wird gruppiert dargestellt. Die Spaltenüberschriften der für die Gruppierung gewählten Spalten ändern sich in Grp(<Spaltenname>). aggregiert die anderen Attributwerte der zusammengefassten Zeilen. Die dabei jeweils verwendete Aggregationsfunktion wird in der Spaltenüberschrift angezeigt. Sie können anschließend für jede Spalte die Aggregationsfunktion ändern.

Attributwerte können Sie in gruppierten Listen nur eingeschränkt bearbeiten.

Die automatische Synchronisation einer gruppierten Liste mit dem Netzfenster ist ausgeschaltet. Sie können in diesem Fall mit der rechten Maustaste das Kontextmenü öffnen und über den Eintrag Im Netz markieren die Synchronisation manuell aufrufen.
Aggregationsfunktion für eine Spalte einer gruppierten Liste auswählen

Stellen Sie sicher, dass ein Listenfenster mit einer gruppierten Liste geöffnet ist.

Klicken Sie mit der rechten Maustaste in den Spaltenkopf der gewünschten Spalte.

In der Spaltenüberschrift wird die aktuell verwendete Aggregationsfunktion angezeigt.

Wählen Sie den Eintrag Aggregationsfunktion und den gewünschten Untereintrag.

Gewichtetes Mittel

Mittelwert der zusammengefassten Attributwerte, nach einem Attribut gewichtet. Wenn Sie diese Aggregationsfunktion wählen, ist zunächst die Gewichtung 1.0 eingestellt. Das gewünschte Attribut für die Gewichtung wählen Sie anschließend über das Symbol Attribute auswählen in der Spalte Gewicht aus (Anzeige-Eigenschaften von Attributen ändern).

Beispiel: Sie möchten die durchschnittliche Belastung eines Streckentyps ermitteln. Dabei möchten Sie lange Strecken entsprechend stärker gewichten als kurze. Dafür aggregieren Sie die Liste der Strecken nach Streckentyp, wählen bei der Belastung die Aggregationsfunktion gewichtetes Mittel und wählen als Gewichtungsattribut die Länge.

Hinweis

Diese Funktion ist für Attribute vom Typ Zeitpunkt verfügbar. Wenn die Zeitpunkte einem Taktmuster des Verfahrens Taktfahrtengruppen berechnen entsprechen, werden die Taktzeiten als String ausgegeben (siehe Taktfahrten bearbeiten auf Seite 1).

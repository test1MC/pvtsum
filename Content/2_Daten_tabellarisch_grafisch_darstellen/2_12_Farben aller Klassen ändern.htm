Farben aller Klassen ändern

Sie können die Farbgebung aller Klassen gleichzeitig bearbeiten.

Farben einer Klassifizierung eines Punktobjekts ändern

Nachfolgend wird am Beispiel des Netzobjekts Bezirke beschrieben, wie Sie die Farben der Klassen ändern können.

Wählen Sie im Navigator den Eintrag Bezirke > Darstellung.

Klicken Sie auf die Schaltfläche Farben für alle Klassen ändern.

Das Fenster Klassifizierung bearbeiten: Farben für alle Klassen ändern öffnet sich.

Nehmen Sie im Bereich Farbzuordnung Punktobjekt die gewünschte Einstellung vor.

Element

Beschreibung

Textfarbe/ Füllfarbe aus Farbrampe

Ist die Option ausgewählt, wird die Farbe aus dem Farbverlauf verwendet. Ist sie nicht ausgewählt, wird die Farbe der zuvor eingestellten Farbe verwendet.

Hinweis

Je nach gewähltem Punkttyp heißt die Farbe Textfarbe oder Füllfarbe.

Randfarbe aus Farbrampe / Rahmenfarbe aus Farbrampe

Wenn Sie Symbole oder Bilder klassifiziert darstellen, können Sie hier die Füllfarbe und die Randfarbe festlegen.

Wenn Sie Texte klassifiziert darstellen, können Sie hier die Textfarbe und die Rahmenfarbe der Texte festlegen.

Ist die Option ausgewählt, wird die Farbe aus dem Farbverlauf auch für den Rand bzw. den Rahmen verwendet.

Randfarbe konstant / Rahmenfarbe konstant

Ist die Option ausgewählt, können Sie über das Farbfeld eine Randfarbe bzw. Rahmenfarbe auswählen.

Randfarbe unverändert / Rahmenfarbe unverändert

Ist die Option ausgewählt, wird die bereits eingestellte Farbe für den Rand bzw. den Rahmen verwendet.

Nehmen Sie gegebenenfalls im Bereich Farbzuordnung Polygon die gewünschten Einstellungen vor.

Hier können Sie die Polygonfüllfarbe und die Randfarbe der Polygone festlegen. Der Bereich ist nur verfügbar, wenn das Netzobjekt über ein Polygon verfügt. Die Optionen entsprechen denen der Farbzuordnung der Punktobjekte.

Klicken Sie mit der rechten Maustaste in die Farbskala.

Über die Farbskala können Sie die Farben und Abstände der Klassen einstellen. Die linke Spalte der Darstellung zeigt die aktuellen Einstellungen an, die rechte Spalte eine Vorschau der neuen Einstellungen.

Element

Beschreibung

Farbe hier einfügen

Über den Eintrag fügen Sie an der gewählten Stelle der Farbskala eine weitere Farbe ein.

Tipp

Sie können eine neue Zwischenfarbe einfügen, indem Sie an die gewünschte Stelle in der Farbrampe klicken (der Mauszeiger wird dabei als Pluszeichen dargestellt) und anschließend die gewünschte Farbe einstellen.

Hinweis

Klicken Sie auf den Pfeil rechts neben einem Farbfeld, um eine neue Farbe zuzuordnen.

Farbverlauf für diesen Abschnitt setzen

Über den Eintrag legen Sie fest, dass die beiden Farben, zwischen denen Sie in die Farbskala geklickt haben, durch einen Farbverlauf verbunden werden.

Konstante Farbe für diesen Abschnitt setzen

Über den Eintrag legen Sie fest, dass der Bereich zwischen den beiden nächstgelegenen Farben die Farbe des oberhalb liegenden Farbfeldes erhält.

Auf Standardfarbrampe zurücksetzen

Über den Eintrag setzen Sie die Farbskala auf einen Standardwert zurück.

Farbrampe umkehren

Über den Eintrag drehen Sie die vorhandene Skala um 180 Grad.

Alle Zwischenfarben löschen

Über den Eintrag löschen Sie alle Zwischenfarben.

Qualitative Skala für momentane Anzahl Klassen erstellen

Über den Eintrag fügen Sie eine qualitative Skala für alle Klassen ein. So erhält jede Klasse eine eigene Farbe.

Transparenz für alle Farben setzen

Über den Schieberegler definieren Sie einen Wert in Prozent, der die Transparenz der Farben der Farbrampe festlegt.

Farbverlauf für alle Abschnitte setzen

Über den Eintrag legen Sie fest, dass alle Bereiche zwischen zwei Farben als Farbverläufe dargestellt werden.

Konstante Farbe für alle Abschnitte setzen

Über den Eintrag legen Sie fest, dass alle Bereiche zwischen zwei Farben in der Farbe des oberhalb liegenden Farbfeldes dargestellt werden.

Sie können eine Farbe in der Skala verschieben, indem Sie sie mit gedrückter Maustaste nach oben oder unten ziehen. Alternativ können Sie mit der rechten Maustaste auf eine Farbe klicken, den Eintrag Position bearbeiten wählen und einen Wert zwischen 0.01 und 0.99 eingeben.

Sie können eine Farbe durch eine bereits bestehende Farbe ersetzen, indem sie mit rechten Maustaste auf das Farbwahlfeld der bestehenden Farbe klicken und den Eintrag Kopieren wählen. Anschließend klicken Sie mit der rechten Maustaste auf das Farbfeld der Farbe, die sie ersetzen möchten, und wählen den Eintrag Einfügen.

Sie können eine Zwischenfarbe in der Skala duplizieren, indem Sie sie mit gedrückter Taste Strg an die gewünschte Stelle verschieben.

Sie können eine Zwischenfarbe löschen, indem Sie mit der rechten Maustaste auf eine Farbe klicken und den Eintrag Farbe löschen wählen.

Der Mauszeiger verändert seine Form entsprechend der Bearbeitungsmöglichkeiten:

  • Pluszeichen: eine Farbe wird an der Stelle eingefügt einfügen
  • Doppelpfeil: die Farbe kann verschoben werden
  • Doppelpfeil mit Pluszeichen: Farbe kann kopiert und verschoben werden

Die Einstellungen werden übernommen und in der Tabelle angezeigt.

Nachfolgend wird am Beispiel des Netzobjekts Strecken beschrieben, wie Sie die Eigenschaften einer Klassifizierung eines Linienobjekts ändern.

Gibt es mehrere Strichlayer, heißt die Farbe Füllfarbe. Bei einem Strichlayer heißt die Farbe Strichfarbe.

Die nachfolgenden Optionen sind nur verfügbar, wenn es mehrere Strichlayer gibt.

Randfarbe aus Farbrampe

Ist die Option ausgewählt, wird die Farbe aus dem Farbverlauf auch für den Rand verwendet.

Randfarbe konstant

Randfarbe unverändert

Ist die Option ausgewählt, wird die bereits eingestellte Farbe für den Rand verwendet.

Wählen Sie den gewünschten Eintrag.

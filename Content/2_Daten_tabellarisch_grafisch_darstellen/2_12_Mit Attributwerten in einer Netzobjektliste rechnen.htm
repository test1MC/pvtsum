Mit Attributwerten in einer Netzobjektliste rechnen

Sie können in Netzobjektlisten mit bearbeitbaren numerischen Werten Berechnungen durchführen.

Nicht bearbeitbare Attributwerte sind ausgegraut.

Alternativ können Sie mit Attributwerten folgendermaßen rechnen:

Sie können in der Liste mehrere Netzobjekte auswählen und über Rechtsklick den Eintrag Bearbeiten aufrufen. Im Fenster Mehrfach-Bearbeiten stehen Ihnen verschiedene Rechenoperationen zur Verfügung (Attributwerte von Netzobjekten bearbeiten).

Stellen Sie sicher, dass ein Listenfenster geöffnet ist.

Markieren Sie die gewünschten Zellen (Listenbereiche auswählen und Listenbereiche über die Tastatur auswählen).

Klicken Sie mit der rechten Maustaste auf einen markierten Wert.

Wählen Sie den Eintrag Rechenoperationen auf markiertem Bereich.

Element

Beschreibung

Wert zuweisen

Über diesen Eintrag weisen Sie den Attributwerten einen neuen Wert zu. In einem Fenster geben Sie den gewünschten neuen Wert ein.

Addieren

Über diesen Eintrag addieren Sie zu den Attributwerten einen Wert. In einem Fenster geben Sie den gewünschten Wert ein.

Subtrahieren

Über diesen Eintrag subtrahieren Sie von den Attributwerten einen Wert. In einem Fenster geben Sie den gewünschten Wert ein.

Multiplizieren

Über diesen Eintrag multiplizieren Sie die Attributwerte mit einem Wert. In einem Fenster geben Sie den gewünschten Wert ein.

Dividieren

Über diesen Eintrag teilen Sie die Attributwerte durch einen Wert. In einem Fenster geben Sie den gewünschten Wert ein.

Potenzieren

Über diesen Eintrag potenzieren Sie die Attributwerte mit einem Wert. In einem Fenster geben Sie den gewünschten Wert ein.

Minimum bilden

Über diesen Eintrag bilden Sie das Minimum zwischen jeweils dem Attributwert und einem Wert. In einem Fenster geben Sie den gewünschten Wert ein.

Maximum bilden

Über diesen Eintrag bilden Sie das Maximum zwischen jeweils dem Attributwert und einem Wert. In einem Fenster geben Sie den gewünschten Wert ein.

Exponentialfunktion

Über diesen Eintrag wenden Sie auf die ausgewählten Attributwerte die Exponentialfunktion an.

Natürlicher Logarithmus

Über diesen Eintrag bilden Sie den natürlichen Logarithmus der ausgewählten Attributwerte.

Kehrwert

Über diesen Eintrag bilden Sie den Kehrwert der ausgewählten Attributwerte.

Die Attributwerte werden berechnet. Falls einige der gewählten Werte nicht geändert werden können, öffnet sich eine Meldung. Sie können pro Spalte entscheiden, ob Sie die Änderung abbrechen oder nur für die Werte durchführen, bei denen die Änderung durchgeführt werden kann.

Sie können wie in der Schnellansicht und bei Matrizen die Rechenoperation auch direkt eingeben. Sie addieren beispielsweise den Wert 2, indem Sie die gewünschten Attributwerte markieren und dann den Ausdruck =+2 eingeben. Alle zur Verfügung stehenden Rechenoperationen und Ausdrücke sind mit der Schnellansicht beschrieben (Attributwerte in der Schnellansicht per Rechenoperation ändern).

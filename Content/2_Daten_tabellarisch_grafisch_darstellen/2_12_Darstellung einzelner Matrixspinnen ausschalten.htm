Darstellung einzelner Matrixspinnen ausschalten

Sie können für Bezirke die Darstellung einzelner Matrixspinnen getrennt je Richtung ausschalten. Für Oberbezirke ist das nicht möglich.

Wählen Sie im Menü Grafik den Eintrag Matrixspinne.

Das Fenster Grafikwerkzeuge (Matrixspinne) öffnet sich.

Klicken Sie auf den Bezirk, von dem die Matrixspinne ausgeht (= Von-Bezirk), die Sie ausschalten möchten.

Der Von-Bezirk wird im Fenster Grafikwerkzeuge (Matrixspinne) angezeigt.

Klicken Sie auf den Bezirk, an dem die Matrixspinne endet (= Nach-Bezirk), die Sie ausschalten möchten.

Der Nach-Bezirk wird im Fenster Grafikwerkzeuge (Matrixspinne) angezeigt.

Schalten Sie die Option Balken zeichnen aus.

Sie können über die Option Balkenbeschriftung zeichnen auch lediglich die Beschriftung der gewählten Matrixspinne ausschalten.

Über die Option Beschriftung verschieben können Sie die Beschriftung der gewählten Matrixspinne entlang der Matrixspinnenstrecke verschieben. Wenn Sie die Schaltfläche geklickt haben, können Sie mit der Maus anschließend die neue Position entlang der Matrixspinnenstrecke bestimmen.

Wenn Sie die Darstellung der Matrixspinnen ausschalten möchten, müssen Sie im Menü Grafik > Grafikparameter bearbeiten die entsprechende Option deaktivieren (Matrixspinnen anzeigen und Grundeinstellungen vornehmen).

Points of Interest anzeigen

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten: Netzeditor öffnet sich.

Wählen Sie im Navigator den Eintrag Points of Interest.

Element

Beschreibung

Layer POI-Punkte zeichnen

Ist die Option ausgewählt, werden die Symbole oder Bilder von POIs im Netzfenster dargestellt.

Layer POI-Linien zeichnen

Ist die Option ausgewählt, werden linienförmige POIs im Netzfenster dargestellt. Außerdem werden Zuordnungen von POIs als Verbindungen dargestellt.

Hinweis

Linienförmige POIs können Sie nicht selbst anlegen. Sie können durch den Import einer Netzdatei oder eines Shapefiles erzeugt werden.

Layer POI-Flächen zeichnen

Ist die Option ausgewählt, werden Polygone von POIs im Netzfenster dargestellt.

Legen Sie im Navigator der POI-Kategorien über die Kontrollkästchen fest, welche Kategorien angezeigt werden.

Über die Symbole können Sie diese Einstellungen für mehrere Kategorien gleichzeitig ändern:

  • Alle POI-Kategorien einschalten
  • Alle POI-Kategorien ausschalten
  • Auswahl der POI-Kategorien umkehren

Über die Symbole können Sie den Navigator der POI-Kategorien auf- und zuklappen:

  • Navigator aufklappen
  • Navigator einklappen

Hinter der POI-Kategorie wird entsprechend der aktuellen Einstellungen eine Vorschau der POIs angezeigt.

Legen Sie in der rechten Liste POI-Kategorie die Reihenfolge fest, in der die POIs gezeichnet werden.

Klicken Sie auf eine Kategorie und verwenden Sie eines der Symbole:

  • Kategorie nach unten verschieben: Layer wird früher gezeichnet
  • Kategorie nach oben verschieben: Layer wird später gezeichnet

Die aktuell nicht im Netzfenster angezeigten POI-Kategorien werden grau dargestellt.

Nehmen Sie die Einstellungen für die einzelnen POI-Kategorien vor (Anzeigeeinstellungen für eine POI-Kategorie ändern).

Die POIs werden entsprechend Ihren Einstellungen angezeigt oder ausgeblendet.

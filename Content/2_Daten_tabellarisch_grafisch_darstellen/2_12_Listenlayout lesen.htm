Listenlayout lesen

Sie haben zwei Möglichkeiten Listenlayouts einzulesen.

Erste Möglichkeit

Stellen Sie sicher, dass das Listenfenster der gewünschten Liste geöffnet ist.

Wählen Sie im Menü Liste (<Netzobjekttyp>) den Eintrag Layout öffnen.

Das Fenster Listen-Layout öffnen öffnet sich.

Öffnen Sie die gewünschte Datei.

Die gespeicherten Einstellungen werden für die geöffnete Liste übernommen.

Zweite Möglichkeit

Stellen Sie sicher, dass das Listenfenster der gewünschten Liste geöffnet ist.

Wählen Sie in der Symbolleiste über die Auswahlliste das gewünschte Layout, indem Sie auf den Eintrag klicken.

In der Auswahlliste können Sie alle Listen-Layoutdateien auswählen, die im eingestellten Projektverzeichnis gespeichert sind (Projektverzeichnisdatei öffnen und speichern).

Das Listenlayout wird geladen und die gespeicherten Einstellungen werden für die geöffnete Liste übernommen.

Das gespeicherte Listen-Layout einer POI-Kategorie kann auch für eine abweichende POI-Kategorie eingelesen werden.

Liste drucken

Sie können den Inhalt einer Liste drucken.

Sie können diverse Einstellungen für den Druck vornehmen, bevor Sie eine Liste drucken (Layout für Listendruck festlegen).

Klicken Sie in der Symbolleiste auf das Symbol  Drucken.

Das Fenster Drucken: Liste öffnet sich.

Wählen Sie unter Name eine Ausgabemöglichkeit aus.

Nehmen Sie bei Bedarf über die Schaltfläche Druckseite einrichten weitere Einstellungen vor (Netzdarstellung drucken).

Berücksichtigen Sie bei der Formatauswahl die Anzahl der Spalten und Zeilen, sowie die Spaltenbreite und die Schriftgröße.

Passt die Liste nicht auf eine Seite, wird sie auf mehrere Seiten gedruckt.

Das Fenster Druckinformation öffnet sich. Darin wird die Anzahl der Seiten und der Zeilen ausgegeben, die gedruckt werden.

Die Liste wird gedruckt.

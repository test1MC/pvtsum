Aktive Punktobjekte darstellen

Sie können festlegen, wie aktive Punktobjekte im Netzfenster dargestellt werden. Nachfolgend wird eine einheitliche Darstellung der Punktobjekte beschrieben. Sie können Objekte auch anhand von Attributwerten klassifiziert darstellen (Objekte anhand von Attributwerten klassifiziert darstellen).

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Alternativ können Sie das Fenster öffnen, indem Sie gleichzeitig die Tasten Strg und G drücken.

Das Fenster Grafikparameter bearbeiten: Netzeditor öffnet sich.

Wählen Sie im Navigator den Eintrag Bezirke.

Die Grafikparameter sind für alle Punktobjekte nahezu identisch. Diese Anleitung beschreibt beispielhaft die Einstellungen für das Punktobjekt Bezirk.

Element

Beschreibung

Layer Bezirke zeichnen

Ist die Option ausgewählt, wird der Layer in der Netzdarstellung gezeichnet.

Überlappungen beim Zeichnen vermeiden

Ist die Option ausgewählt, werden nur Punktobjekte eines Layers dargestellt, die sich nicht überschneiden.

Hinweis

Bei flächigen Punktobjekten gilt dies für die Schwerpunkte, nicht jedoch für die Umrandungen und Flächen. Tabellen und Diagramme werden nur dargestellt, wenn sie sich nicht überschneiden (Netzobjekte mit Tabelle oder Diagramm beschriften). Sie können jedoch Punktobjekte überschneiden.

Ist die Option nicht ausgewählt, werden alle Punktobjekte eines Layers dargestellt, auch wenn sie sich überschneiden.

Hinweis

Zoomen Sie bei Bedarf in den überlappenden Bereich hinein.

Wählen Sie im Navigator den Eintrag Bezirke > Darstellung > Register Aktiv.

Alternativ können Sie die Grafikparameterseite Darstellung eines Netzobjekttyps aufrufen, indem Sie im Fenster Netz mit der rechten Maustaste auf das Symbol des Netzobjekttyps klicken.

Element

Beschreibung

Einheitliche Darstellung

Ist die Option ausgewählt, werden alle aktiven Objekte einheitlich dargestellt.

Hinweis

Die unten beschriebenen Optionen werden angezeigt, wenn Sie eine einheitliche Darstellung wählen.

Klassifizierte Darstellung

Ist die Option ausgewählt, können die aktiven Objekte nach einem Attribut klassifiziert und dargestellt werden (Objekte anhand von Attributwerten klassifiziert darstellen).

Optionen bei einheitlicher Darstellung

Vorschau

Hier sehen Sie, wie ein Bezirk gemäß den aktuellen Einstellungen gezeichnet wird.

Objekt zeichnen

Ist die Option ausgewählt, werden aktive Punktobjekte gezeichnet.

Zeichnen bis Maßstab

Ist die Option ausgewählt, werden die Punktobjekte beim Rauszoomen bis zum eingestellten Maßstab gezeichnet (Netzmaßstab ändern).

Ist die Option nicht ausgewählt, werden die Punktobjekte immer dargestellt, unabhängig vom Netzmaßstab.

Tabelle zeichnen

Ist die Option ausgewählt, werden die Punktobjekte gemäß den Einstellungen auf der Seite Tabelle mit einer Tabelle beschriftet (Netzobjekte mit Tabellen beschriften).

Diagramm zeichnen

Ist die Option ausgewählt, werden die Punktobjekte gemäß den Einstellungen auf der Seite Diagramm beschriftet mit einem Diagramm (Netzobjekte mit Säulendiagrammen beschriften).

Polygon zeichnen

Ist die Option ausgewählt, werden vorhandene Umrandungen und Flächen von flächigen Punktobjekten gezeichnet. Sie können die Schraffur, Farbe und einen Strichtyp in einem separaten Fenster festlegen, indem Sie in das Feld klicken (Eigenschaften eines geschlossenen Polygons bearbeiten).

Hinweis

Die Option ist nur für die Netzobjekte Bezirke, Oberbezirke, Gebiete, POIs, Oberknoten und Mautsysteme verfügbar.

Punktobjekttyp

Hier wählen Sie aus, ob das Punktobjekt als Text, Symbol oder Bild dargestellt wird.

Text/Symbol/Bild

Je nach gewählter Option wird das Punktobjekt als Text, Symbol oder Bild dargestellt. Die Einstellungen ändern sich entsprechend.

Text

Ist die Option Text ausgewählt, wird das Punktobjekt durch einen Text dargestellt.

Inhalt

Über die Schaltfläche wählen Sie das Attribut aus, durch dessen Werte die Punktobjekte dargestellt werden.

Größe

Textgröße in mm

Font

Sie können die Schriftart und den Schriftschnitt in einem separaten Fenster festlegen, indem Sie in das Feld klicken.

Transparent

Ist die Option ausgewählt, wird der Text nur als Schrift ohne weißen Hintergrund dargestellt.

Farbe

Über die Auswahlliste können Sie eine Farbe für den Text auswählen.

Rahmen

Wenn Sie die Option auswählen, wird der Text umrahmt. Über die Auswahlliste können Sie eine Farbe für den Textrahmen auswählen.

Runden

Rundungsfaktor für die Attributwerte

Nachkommastellen

Angabe der Anzahl Dezimalstellen für die Darstellung der Attributwerte

Hinweis

Die Option ist nur für numerische Attribute vom Typ Fließkommazahl verfügbar.

Symbol

Ist die Option Symbol ausgewählt, wird das Punktobjekt durch das ausgewählte Symbol dargestellt. Über die Auswahlliste können Sie ein anderes Symbol auswählen.

Größe

Symbolgröße in mm

Farbe

Über die Auswahlliste können Sie eine Farbe für das Symbol auswählen (Eigenschaften eines geschlossenen Polygons bearbeiten).

Zeichen

Wenn Sie das Symbol Kreis oder Quadrat auswählen, können Sie über das Kontrollkästchen ein Eingabefeld aktivieren. In das Eingabefeld können Sie ein Zeichen eingeben, das innerhalb des Kreises oder Quadrates dargestellt wird. Über die Auswahlliste können Sie eine Farbe für das Zeichen auswählen.

Bild

Ist die Option Bild ausgewählt, wird das Punktobjekt durch eine Grafikdatei dargestellt.

Für die Darstellung der Punktobjekte können Sie über die Schaltfläche eine Bilddatei (*.bmp, *.jpg, etc.) auswählen.

Hinweise

Nachdem Sie eine Bilddatei ausgewählt haben, erscheint eine Abfrage. Wenn Sie die Abfrage bestätigen, wird die definierte Höhe durch die aktuelle Bildgröße ersetzt. Eine Vorschau der ausgewählten Bilddatei wird angezeigt.

Die Bilddatei wird als Verknüpfung eingefügt und nicht mit der Versionsdatei gespeichert. Wenn beim nächsten Öffnen der Version den Pfad zur Bilddatei nicht findet, erhalten Sie eine entsprechende Meldung und die Grafik wird nicht angezeigt. Wir empfehlen daher, alle Bilddateien in einem Verzeichnis zu speichern und dieses in der Projektverzeichnisdatei einzustellen (Speicherort von Dateien ändern).

Höhe

Höhe des Bildes in mm

Beschriftung zeichnen

Ist die Option ausgewählt, wird das Bild beschriftet. Über die Schaltfläche können Sie in einem separaten Fenster ein Attribut des jeweiligen Netzobjekttyps für die Beschriftung auswählen.

  • Höhe

Höhe der Beschriftung (prozentual zur Bildhöhe)

  • Runden

Rundungsfaktor für die Attributwerte

  • Farbe

Über die Auswahlliste können Sie eine Farbe für die Beschriftung auswählen.

  • Nachkommastellen

Angabe der Anzahl Dezimalstellen für die Darstellung der Attributwerte

Hinweis

Die Option ist nur für numerische Attribute vom Typ Fließkommazahl verfügbar.

Wenn Sie auf die Schaltfläche Vorschau klicken, werden Ihre Änderungen sofort angezeigt, ohne dass sich das Fenster Grafikparameter bearbeiten: Netzeditor schließt.

Die Punktobjekte werden gemäß den Einstellungen dargestellt.

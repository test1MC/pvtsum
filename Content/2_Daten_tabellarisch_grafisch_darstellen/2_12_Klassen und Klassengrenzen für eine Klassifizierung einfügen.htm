Klassen und Klassengrenzen für eine Klassifizierung einfügen

Sie können Klassen und Klassengrenzen für Punktobjekte und Linienobjekte einfügen. Sie können alle gewünschten Klassen gleichzeitig einfügen und die Darstellung neu eingefügter oder bestehender Klassen gleichzeitig bearbeiten.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den gewünschten Eintrag.

Die Option steht Ihnen auf verschiedenen Seiten zur Verfügung, beispielsweise unter Bezirke > Darstellung.

Wählen Sie die Option Klassifizierte Darstellung.

Wählen Sie über die mit Attributnamen beschriftete Schaltfläche das Attribut aus, nach dem Sie die Darstellung klassifizieren möchten.

Für das ausgewählte Attribut werden Klassen angezeigt.

Bei alphanumerischen, boolschen Attributen und Aufzählungstypen wird optional für jede Ausprägung des gewählten Attributs eine eigene Klasse angelegt. Die Klassen werden dann automatisch sinnvoll vorbelegt.

Wenn Sie die automatische Vorbelegung ablehnen, können Sie sie nachträglich über den Eintrag Klassifizierung für diskrete Werte erstellen im Kontextmenü des Klassifizierungsgrids vornehmen.

Klicken Sie auf die Schaltfläche Alle Klassen neu belegen.

Das Fenster Klassifizierung bearbeiten: Alle Klassen neu belegen öffnet sich.

Nehmen Sie im Bereich Klassengrenzen die gewünschten Einstellungen vor.

Element

Beschreibung

Verteilung der Klassengrenzen

Hier können Sie die gewünschte Verteilung der Klassengrenzen auswählen.

Tipp

Über den kleinen Pfeil oberhalb der Eingabefelder für die Klassengrenzen können Sie ebenfalls eine andere Verteilung einstellen.

Benutzerdefiniert

Ist diese Option ausgewählt, können Sie die gewünschten Klassengrenzen über die Eingabefelder eingeben.

Hinweis

Die Klassengrenzen müssen aufsteigend sortiert sein. Über den kleinen Pfeil oberhalb der Eingabefelder für die Klassengrenzen können Sie die Klassengrenzen automatisch sortieren. Sind die Klassengrenzen nicht aufsteigend sortiert, können sie beim Verlassen des Fensters automatisch sortiert werden.

Äquidistant

Ist diese Option ausgewählt, können Sie die Grenzen für die ersten beiden Klassen in die Eingabefelder neben der Skala eingeben. Alle weiteren Grenzen werden gemäß der Differenz der beiden ersten Werte automatisch berechnet. Jede neue Klassengrenze wird ermittelt, indem die Differenz zwischen den beiden ersten Klassen zur vorhergehenden Klassengrenze addiert wird. Wenn Sie einen neuen Wert eingeben, werden die Grenzen sofort neu berechnet.

Logarithmisch

Ist diese Option ausgewählt, können Sie die Grenzen für die ersten beiden Klassen in die Eingabefelder neben der Skala eingeben. Alle weiteren Grenzen werden gemäß der Differenz der beiden ersten Werte automatisch berechnet. Jede neue Klassengrenze wird ermittelt, indem die Differenz zwischen den beiden ersten Klassen mit der vorhergehenden Klassengrenze multipliziert wird. Wenn Sie einen neuen Wert eingeben, werden die Grenzen sofort neu berechnet.

Natural breaks

Ist diese Option ausgewählt, werden alle Klassengrenzen automatisch erzeugt. Basierend auf den im Netz vorhandenen Attributwerten und der aktuellen Anzahl Klassen wird eine sinnvolle Klasseneinteilung generiert.

Hinweis

Ziel der Methode ist es, die Grenzen so zu ermitteln, dass in jeder Klasse etwa gleich viele Elemente enthalten sind.

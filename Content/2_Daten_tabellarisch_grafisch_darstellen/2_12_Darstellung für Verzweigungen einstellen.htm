Darstellung für Verzweigungen einstellen

Für gekoppelte Fahrplanfahrtabschnitte und die damit verbundenen Flügelungen ist es möglich, Verzweigungen im schematischen Liniennetzplan anzuzeigen. Die Anzeige der Verzweigungen können Sie folgendermaßen einstellen.

Wählen Sie im Menü Schematischer Liniennetzplan den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator die Seite Verzweigungen.

Element

Beschreibung

Verzweigungen zeichnen

Ist die Option ausgewählt, werden Verzweigungen dargestellt.

Verzweigungen nur am Beginn oder Ende eines Koppelabschnitts zeichnen

Ist die Option zusätzlich zu der obigen Option ausgewählt, werden Verzweigungen lediglich gezeichnet, wenn die Verzweigung Beginn oder Ende eines Koppelabschnitts mindestens einer unterliegenden Fahrplanfahrt ist.

Symbol

In der Auswahlliste können Sie das Symbol auswählen, mit dem Sie die Verzweigungen darstellen wollen.

Größe

Symbolgröße in Rastereinheiten

Farbe

Über die Auswahlliste können Sie eine Farbe für das Symbol auswählen (Eigenschaften eines geschlossenen Polygons bearbeiten).

Zeichen

Wenn Sie das Symbol Kreis oder Quadrat auswählen, können Sie über das Kontrollkästchen ein Eingabefeld aktivieren. In das Eingabefeld können Sie ein Zeichen eingeben, das innerhalb des Kreises oder Quadrates dargestellt wird. Über die Auswahlliste können Sie eine Farbe für das Zeichen auswählen.

Die Verzweigungen werden gemäß den Einstellungen dargestellt.

Einzelne Klasse einfügen und löschen

Sie können in einer Klassifizierung Klassen einzeln einfügen und löschen. Die Vorgehensweise ist identisch für Punktobjekte und Linienobjekte.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den gewünschten Eintrag.

Wählen Sie die Option Klassifizierte Darstellung.

Klicken Sie in der Tabelle auf die Zeile, nach der Sie die neue Klasse einfügen möchten.

Klicken Sie auf die Schaltfläche  Einfügen.

Alternativ können Sie über das Symbol  Klasse duplizieren eine vorhandene Klasse mit den eingestellten Grafikparametern kopieren.

Die Klasse wird eingefügt. Die Klassengrenze der neuen Klasse liegt in der Mitte zwischen der gewählten und der nächsten Klasse.

Sie können die Klassengrenzen und andere Eigenschaften nachträglich ändern (Darstellung und Klassengrenzen einer Klasse bearbeiten).

Sie können eine Klasse löschen, indem Sie sie in der Tabelle markieren und auf die Schaltfläche  Löschen klicken. Die letzte Klasse können Sie nicht löschen.

Alternativ können Sie die Bearbeitungsfunktionen auch über das Kontextmenü der Tabelle aufrufen.

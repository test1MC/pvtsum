Position von Mittenbeschriftungen an Kanten einstellen

Sie können die Mittenbeschriftung von einer Kante oder mehreren Kanten gleichzeitig einstellen. Dabei legen Sie eine relative Position fest.

Klicken Sie mit der rechten Maustaste auf die Kante, deren Mittenbeschriftung Sie verschieben möchten.

Wählen Sie im Kontextmenü den Eintrag Mittenbeschriftungen verschieben.

Das Fenster Mittenbeschriftung positionieren öffnet sich.

Verschieben Sie mit Hilfe des Schiebereglers die Mittenbeschriftung an die gewünschte relative Position oder geben Sie die relative Position als Wert zwischen 0 und 1 in das Eingabefeld ein.

Die Position der Beschriftung wird Ihnen sofort im Netz angezeigt.

Die Mittenbeschriftung wird an der eingestellten Position angezeigt.

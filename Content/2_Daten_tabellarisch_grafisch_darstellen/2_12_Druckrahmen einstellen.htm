Druckrahmen einstellen

Wenn Sie die Netzdarstellung drucken möchten, können Sie für den Druckrahmen folgende Einstellungen vornehmen.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag Druckrahmen.

Element

Beschreibung

Zeichnen

Ist die Option ausgewählt, wird ein Druckrahmen gezeichnet.

Hinweis

Ist die Option ausgewählt, wird eine einfache Umrandung gezeichnet, wenn die Option Überschrift und Fußzeile zeichnen auf der jeweiligen Seite nicht ausgewählt wurde.

Umrandung

Sie können die Darstellung der Umrandung in einem separaten Fenster einstellen, indem Sie in das Feld klicken (Strichdarstellung einstellen).

Druckbereich auf dem Bildschirm anzeigen

Ist die Option ausgewählt, wird der Druckbereich als gestricheltes Rechteck auf dem Bildschirm angezeigt.

Hinweis

Der Druckbereich wird jedoch nur dargestellt, wenn im Fenster Druckseite einrichten die Option Druckbereich ausgewählt ist (Druckseite einrichten).

Wählen Sie die Seite Überschrift, wenn Sie eine Überschrift einfügen möchten.

Die Überschrift erstreckt sich über die gesamte Breite des Druckrahmens.

Element

Beschreibung

Zeichnen

Ist die Option ausgewählt, wird eine Überschrift angezeigt.

Schriftart

Sie können die Schriftart und den Schriftschnitt der Überschrift in einem separaten Fenster einstellen, indem Sie in das Feld klicken.

Hinweis

Sie können die Überschrift linksbündig, zentriert oder rechtsbündig positionieren, indem Sie auf das entsprechende Symbol auf der rechten Seite klicken.

Textgröße

Textgröße in mm

Farbe

Über die Auswahlliste können Sie eine Farbe für die Texte auswählen.

Mindestzeilenhöhe

Ist die Option ausgewählt, können Sie eine Mindesthöhe für die Zeile angeben. Geben Sie eine Höhe in mm ein.

Hinweis

Die Zeilenhöhe sollte größer als die Textgröße sein, da sonst als Zeilenhöhe die doppelte Textgröße verwendet wird.

Text

Über die Auswahllisten können Sie Texteinträge wählen, die in der Überschrift eingefügt werden.

  • benutzerdefiniert

Hier können Sie einen freien Text in das freie Feld eingeben.

  • leer
  • [VISUM-Version]
  • erstellt am: [Datum]
  • erstellt am: [Datum]/[Uhrzeit]
  • [Datum]/[Uhrzeit]
  • [Dateiname]
  • [Maßstab]
  • [Maßstabszahl]

Hilfe

Über die Schaltfläche öffnet sich ein Fenster mit sämtlichen verfügbaren Autowerten.

Hinweis

Die Einträge in eckigen Klammern werden automatisch mit vordefinierten Werten belegt (Autowerte). Sie werden beim Zeichnen des Druckrahmens mit den aktuellen Werten befüllt. Autowerte sind sprachabhängig und können auch in benutzerdefinierten Einträgen verwendet werden.

Wählen Sie die Seite Fußzeile, wenn Sie eine Fußzeile einfügen möchten.

Die Fußzeile kann aus bis zu sechs Feldern bestehen, die am unteren Rand des bedruckbaren Bereichs dargestellt werden.

Element

Beschreibung

Zeichnen

Ist die Option ausgewählt, wird eine Fußzeile gezeichnet.

Schriftart

Sie können die Schriftart und den Schriftschnitt der Fußzeile in einem separaten Fenster einstellen, indem Sie in das Feld klicken.

Textgröße

Textgröße in mm

Grafikhöhe

Größe eingefügter Grafiken in mm

Tipp

Grafiken können im Bereich Text über die Auswahlfelder eingefügt werden.

Farbe

Über die Auswahlliste können Sie eine Farbe für die Texte auswählen.

Mindestzeilenhöhe

Ist die Option ausgewählt, können Sie eine Mindesthöhe für die Zeile angeben. Geben Sie eine Höhe in mm ein.

Hinweis

Die Zeilenhöhe sollte größer als die Textgröße sein, da sonst als Zeilenhöhe die doppelte Textgröße verwendet wird.

Spaltenformatierung

Sie können die Breite jeder Spalte über einen Prozentwert regulieren.

Hinweise

Die Summe muss nicht 100 % ergeben. Die Breite wird anteilig berechnet.

Sie können die Spalteninhalte jeweils linksbündig, zentriert oder rechtsbündig positionieren, indem Sie auf das jeweilige Symbol unterhalb der Eingabefelder klicken.

Text

Über die Auswahllisten können Sie Texteinträge wählen, die in der Fußzeile eingefügt werden.

  • benutzerdefiniert

Sie können einen beliebigen Text in das freie Feld eingeben.

  • leer
  • Feld ausblenden

Ist dieser Eintrag ausgewählt, wird das jeweilige Feld nicht angezeigt.

  • [VISUM-Version]
  • erstellt am: [Datum]
  • erstellt am: [Datum]/[Uhrzeit]
  • [Datum]/[Uhrzeit]
  • [Dateiname]
  • [Maßstab]
  • [Maßstabszahl]
  • [Grafik]

Ist dieser Eintrag ausgewählt, kann unter dem Auswahlfeld der Pfad zu der gewünschten Grafik eingestellt werden, die angezeigt werden soll.

Hilfe

Über die Schaltfläche öffnet sich ein Fenster mit sämtlichen verfügbaren Autowerten.

Hinweise

Die Einträge in eckigen Klammern werden automatisch mit vordefinierten Werten belegt (Autowerte). Sie werden beim Zeichnen des Druckrahmens mit den aktuellen Werten befüllt. Autowerte sind sprachabhängig und können auch in benutzerdefinierten Einträgen verwendet werden.

Soll eine [Grafik] dargestellt werden, müssen Sie ein Komma und den Pfad zu der Grafik eingeben. Beispiel: [Grafik, „D:\logo.bmp”]. Die Größe wird im Bereich Darstellung unter Grafikhöhe festgelegt.

Wenn Sie auf die Schaltfläche Vorschau klicken, werden Ihre Änderungen sofort angezeigt, ohne dass sich das Fenster Grafikparameter bearbeiten schließt.

Die Einstellungen des Druckrahmens werden übernommen.

Zur ausführlichen Beschreibung einer -Netzdarstellung eignet sich eine Legende, die Sie an einer Ecke des Druckbereichs einfügen können (Legende erstellen).

Layout für Listendruck festlegen

Sie können festlegen, mit welchen zusätzlichen Bestandteilen und in welchem Layout eine Liste gedruckt werden sollen.

Seitenränder für Listendruck einstellen

Wählen Sie im Menü Liste (<Netzobjekttyp>) den Eintrag Druckseite einrichten.

Das Fenster Listen: Druckseite einrichten öffnet sich.

Alternativ zeigen Sie im Menü Datei auf Drucken und wählen den Eintrag Druckseite einrichten oder Sie klicken im Fenster Drucken: Liste auf die Schaltfläche Druckseite einrichten.

Wählen Sie im Fenster Listen: Druckseite einrichten das Register Seitenränder.

Element

Beschreibung

Papierrand [mm]

Hier können Sie die den gewünschten Abstand vom Papierrand festlegen.

Druckrahmen für Liste festlegen

Wählen Sie im Menü Liste (<Netzobjekttyp>) den Eintrag Druckseite einrichten.

Das Fenster Listen: Druckseite einrichten öffnet sich.

Wählen Sie im Fenster Listen: Druckseite einrichten das Register Druckrahmen.

Element

Beschreibung

Druckrahmen

Zeichnen

Ist die Option ausgewählt, wird ein Druckrahmen gedruckt.

Umrandung

Sie können die Darstellung des Rahmens festlegen, indem Sie in das Feld klicken (Strichdarstellung einstellen).

Überschrift für Listendruck festlegen

Wählen Sie im Menü Liste (<Netzobjekttyp>) den Eintrag Druckseite einrichten.

Das Fenster Listen: Druckseite einrichten öffnet sich.

Wählen Sie im Fenster Listen: Druckseite einrichten das Register Überschrift.

Element

Beschreibung

Darstellung

Zeichnen

Ist die Option ausgewählt, wird eine Überschrift gedruckt.

Font

Sie können in einem separaten Fenster eine Schriftart und einen Schriftschnitt auswählen, indem Sie in das Feld klicken.

Textgröße

Textgröße in mm

Farbe

Sie können eine Farbe für die Überschrift in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

Hinweis

Sie können die Überschrift linksbündig, zentriert oder rechtsbündig positionieren, indem Sie auf das entsprechende Symbol klicken.

Mindestzeilenhöhe

Ist die Option ausgewählt, können Sie eine Mindesthöhe für die Zeile angeben. Geben Sie eine Höhe in mm ein.

Hinweis

Die Zeilenhöhe sollte größer als die Textgröße sein, da sonst als Zeilenhöhe die doppelte Textgröße verwendet wird.

Text

Über die Auswahlliste können Sie einen Eintrag wählen, der in die Überschrift eingefügt wird.

  • benutzerdefiniert

Hier können Sie einen freien Text in das freie Feld eingeben.

  • leer
  • [VISUM-Version]
  • erstellt am: [Datum]
  • erstellt am: [Datum]/[Uhrzeit]
  • [Datum]/[Uhrzeit]
  • [Dateiname]
  • [NetzElementTyp]
  • [Seitenzahl]
  • [AnzahlSeiten]
  • Seite [Seitenzahl] / [AnzahlSeiten]
  • [Listenüberschrift]

Bei dieser Auswahl wird der Listenkopf als Überschrift gedruckt.

Hilfe

Über die Schaltfläche öffnet sich ein Fenster mit sämtlichen verfügbaren Autowerten.

Hinweis

Die Einträge in eckigen Klammern werden automatisch mit vordefinierten Werten belegt (Autowerte). Sie werden beim Drucken der Liste mit den aktuellen Werten befüllt. Autowerte sind sprachabhängig und können auch in benutzerdefinierten Einträgen verwendet werden.

Fußzeile für Listendruck festlegen

Wählen Sie im Menü Liste (<Netzobjekttyp>) den Eintrag Druckseite einrichten.

Das Fenster Listen: Druckseite einrichten öffnet sich.

Wählen Sie im Fenster Listen: Druckseite einrichten das Register Fußzeile.

Element

Beschreibung

Darstellung

Zeichnen

Ist die Option ausgewählt, wird eine Fußzeile gedruckt.

Font

Sie können in einem separaten Fenster eine Schriftart und einen Schriftschnitt auswählen, indem Sie in das Feld klicken.

Textgröße

Textgröße in mm

Grafikhöhe

Größe eingefügter Grafiken in mm

Tipp

Grafiken können im Bereich Text über die Auswahlfelder eingefügt werden.

Farbe

Sie können eine Farbe für die Fußzeile in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

Mindestzeilenhöhe

Ist die Option ausgewählt, können Sie eine Mindesthöhe für die Zeile angeben. Geben Sie eine Höhe in mm ein.

Hinweis

Die Zeilenhöhe sollte größer als die Textgröße sein, da sonst als Zeilenhöhe die doppelte Textgröße verwendet wird.

Spaltenformatierung

Sie können die Breite jeder Spalte über einen Prozentwert regulieren.

Hinweise

Die Summe muss nicht 100 % ergeben. Die Breite wird anteilig berechnet.

Sie können die Spalteninhalte jeweils linksbündig, zentriert oder rechtsbündig positionieren, indem Sie auf das jeweilige Symbol unterhalb der Eingabefelder klicken.

Text

Über die Auswahlliste können Sie einen Eintrag wählen, der in die Fußzeile eingefügt wird.

  • benutzerdefiniert

Hier können Sie einen Text in das freie Feld eingeben.

  • leer
  • [VISUM-Version]
  • erstellt am: [Datum]
  • erstellt am: [Datum]/[Uhrzeit]
  • [Datum]/[Uhrzeit]
  • [Dateiname]
  • [NetzElementTyp]
  • [Seitenzahl]
  • [AnzahlSeiten]
  • Seite [Seitenzahl] / [AnzahlSeiten]
  • [Listenüberschrift]

Bei dieser Auswahl wird der Listenkopf als Überschrift gedruckt.

  • [Grafik]

Ist dieser Eintrag ausgewählt, kann unter dem Auswahlfeld der Pfad zu der gewünschten Grafik eingestellt werden, die angezeigt werden soll.

Hilfe

Über die Schaltfläche öffnet sich ein Fenster mit sämtlichen verfügbaren Autowerten.

Hinweis

Die Einträge in eckigen Klammern werden automatisch mit vordefinierten Werten belegt (Autowerte). Sie werden beim Drucken der Liste mit den aktuellen Werten befüllt. Autowerte sind sprachabhängig und können auch in benutzerdefinierten Einträgen verwendet werden.

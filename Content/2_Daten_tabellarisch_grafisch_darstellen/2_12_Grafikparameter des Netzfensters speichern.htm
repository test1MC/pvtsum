Grafikparameter des Netzfensters speichern

Die Grafikparameter des Netzfensters werden als Datei *.gpa oder *.gpax gespeichert.
Sie haben mehrere Möglichkeiten Grafikeinstellungen zu speichern.
Alle oder ausgewählte Grafikparameter des Netzfensters speichern

Nehmen Sie die gewünschten Grafikeinstellungen vor.

Zeigen Sie im Menü Datei auf Speichern unter.

Wählen Sie den Eintrag Grafikparameter.

Das Fenster Grafikparameter speichern öffnet sich.

Wählen Sie den gewünschten Pfad und geben Sie den gewünschten Dateinamen ein.

Das Fenster Grafikparameter Schreiben Auswahl öffnet sich.

Element

Beschreibung

Alle Grafikparameter schreiben

Ist die Option ausgewählt, werden alle Grafikparameter gespeichert.

Ausgewählte Parameter schreiben

Ist die Option ausgewählt, werden die in der Liste ausgewählten Grafikparameter gespeichert.

GPar

Liste aller Grafikparameter

Schreiben

Ist die Option ausgewählt, werden die entsprechenden Grafikparameter gespeichert.

Alle aus

Über die Schaltfläche deaktivieren Sie alle Optionen.

Alle an

Über die Schaltfläche wählen Sie alle Optionen der Spalte Schreiben aus.
Nicht benutzte Inhalte auslassen

Ist die Option ausgewählt, werden nicht benötigte Daten (beispielsweise Klassifizierungen) nicht gespeichert.

Hinweis

In diesem Fall werden auch ausgeschaltete Layer und Layer von nicht aktiven Zusatzmodulen nicht gespeichert, selbst wenn die Option Inaktive Layer auslassen nicht ausgewählt ist.

Inaktive Layer auslassen

Ist die Option ausgewählt, werden ausgeschaltete Layer und darunter angeordnete Layer nicht gespeichert.

Die Grafikparameter des Netzfensters werden gespeichert.

Alternative Möglichkeit

Nehmen Sie die gewünschten Grafikeinstellungen vor.

Geben Sie in das Listenfeld Gpar auswählen einen Namen für die Datei ein.

Drücken Sie die Taste Enter.

Die Einstellungen werden unter dem eingegebenen Namen in einer Grafikparameterdatei *.gpa im für Grafikparameter-Dateien eingestellten Projektverzeichnis gespeichert (Projektverzeichnisdatei speichern).

Eine weitere Möglichkeit bietet das Fenster Grafikparameter bearbeiten. Hier können Sie über die Schaltfläche Speichern direkt eine Datei *.gpax im gewünschten Verzeichnis speichern.
Selektives Speichern von Grafikeinstellungen einzelner Elemente

Sie haben die Möglichkeit, einzelne Grafikeinstellungen des Netzes zu speichern.

Nehmen Sie die gewünschten Grafikeinstellungen vor.

Klicken Sie im Navigator des Fensters Grafikparameter bearbeiten mit der linken Maustaste auf das Element, für das Sie Grafikeinstellungen speichern möchten.

Wählen Sie den Eintrag Selektiv speichern.

Speichern Sie die Grafikdatei *.gpax im gewünschten Ordner ab.

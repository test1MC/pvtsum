Grafikparameter des Netzfensters lesen

Wenn Sie Ihre Einstellungen in einer Grafikparameterdatei *.gpa oder *.gpax gespeichert haben, können Sie die Grafikeinstellungen wie folgt einlesen. Sie haben mehrere Möglichkeiten Grafikeinstellungen einzulesen.

Alle oder ausgewählte Grafikparameter des Netzfensters lesen

Zeigen Sie im Menü Datei auf Öffnen.

Wählen Sie den Eintrag Grafikparameter.

Das Fenster Grafikparameter öffnen öffnet sich.

Wählen Sie die gewünschte Datei.

Das Fenster Grafikparameter Lesen Auswahl öffnet sich.

Element

Beschreibung

Alle Grafikparameter lesen

Ist die Option ausgewählt, werden alle Grafikparameter eingelesen.

Ausgewählte Parameter lesen

Ist die Option ausgewählt, werden die in der Liste ausgewählten Grafikparameter eingelesen.

GPar

Liste aller Grafikparameter

Einlesen

Ist die Option ausgewählt, werden die entsprechenden Grafikparameter eingelesen.

Hinweis

Die Option ist nur verfügbar, wenn die entsprechenden Parameter in der Datei enthalten sind.

Additiv

Ist die Option ausgewählt, wird der entsprechende Grafikparameter zu den vorhandenen Grafikparametern hinzugelesen.

Hinweis

Die Option steht nur für Hintergrundelemente zur Verfügung.

Alle aus

Über die Schaltfläche deaktivieren Sie alle Optionen.

Alle an

Über die Schaltfläche wählen Sie alle Optionen der Spalte Einlesen aus. Die Auswahl in der Spalte Additiv wird gelöscht.

Auf netzunabhängige Parameter einschränken

Ist die Option ausgewählt, werden folgende Parameter nicht eingelesen:

  • Bildausschnitt
  • Bing Maps Hintergründe
  • Hintergründe
  • Hintergrundpolygone
  • Hintergrundtexte
  • Knotenströme
  • Matrixspinne Bezirke Textposition
  • Säulendiagramme
  • Spinne

Hinweis

Diese Option ist nicht verfügbar, wenn Sie eine Datei *.gpax einlesen, sondern nur, wenn Sie eine Datei *.gpa einlesen.

Die Grafikparameter des Netzfensters werden eingelesen.

Alternative Möglichkeit
  • Wählen Sie in der Symbolleiste über die Auswahlliste die gewünschte Grafikparameterdatei, indem Sie auf den Eintrag klicken.

In der Auswahlliste können Sie alle Grafikparameter-Dateien auswählen, die im eingestellten Projektverzeichnis gespeichert sind (Projektverzeichnisdatei speichern).

Die Datei wird gelesen und die gespeicherten Einstellungen werden übernommen.

Eine weitere Möglichkeit bietet das Fenster Grafikparameter bearbeiten. Hier können Sie über die Schaltfläche Öffnendirekt eine Datei *.gpaxaus dem gewünschten Verzeichnis einlesen.

Sie können eine Grafikparameterdatei auch automatisch beim Programmstart öffnen ( mit anderen Einstellungen starten).
Selektives Lesen von Grafikeinstellungen einzelner Elemente

Sie haben die Möglichkeit, Grafikeinstellungen einzelner Elemente zu öffnen.

Klicken Sie im Navigator des Fensters Grafikparameter bearbeiten mit der linken Maustaste auf das Element, für das Sie Grafikeinsellungen öffnen möchten.

Wählen Sie den Eintrag Selektiv öffnen.

Öffnen Sie die gewünschte Grafikdatei *.gpax.

Druckausgabe starten

Wenn Sie alle gewünschten Einstellungen vorgenommen haben, können Sie die Druckausgabe starten.

Zeigen Sie im Menü Datei auf Drucken.

Wählen Sie den Eintrag Drucken.

Alternativ können Sie das Fenster öffnen, indem Sie gleichzeitig die Tasten Strg und P drücken.

Das Fenster Drucken: Netzeditor öffnet sich.

Element

Beschreibung

Drucker

Name

In der Auswahlliste können Sie den gewünschten Drucker auswählen.

Hinweis

Sie können die Netzdarstellung auf Ihrem Drucker oder im PDF-Format ausgeben (PostScript-PDF-Export).

Eigenschaften

Hier können Sie beispielsweise Papierquelle und -typ oder das Druckseitenformat festlegen.

Papier

Hier können Sie die Größe und Quelle des Papiers festlegen.

Orientierung

Hier können Sie festlegen, ob Sie im Hochformat oder Querformat drucken möchten.

Dateiausgabe

In Grafikdatei drucken

Ist die Option ausgewählt, exportieren Sie die Netzdarstellung in eine Grafikdatei im Format *.dxf oder *.svg (Grafikschnittstellen).

Druckseite einrichten

Über die Schaltfläche können Sie das Fenster Druckseite einrichten öffnen (Druckseite einrichten).

Ist der Ausgabebereich größer als die Ausgabeseite, öffnet sich das Fenster Druckmodus festlegen.

 

Element

Beschreibung

An Seitenrändern abschneiden

Ist die Option ausgewählt, wird die zu druckende Grafik an den Seitenrändern abgeschnitten.

Auf Seitengröße skalieren

Ist die Option ausgewählt, wird die Grafik auf die gesamte Seite ausgedehnt, ohne dass sich das Seitenverhältnis dabei ändert.

Hinweis

Ist diese Option bereits im Fenster Druckseite einrichten ausgewählt, öffnet sich das Fenster Druckmodus festlegen nicht.

Kacheln drucken

Ist die Option ausgewählt, erfolgt die Druckausgabe in Form von Kacheln.

Hinweise

Wir empfehlen diese Ausgabe, wenn kein großformatiger Drucker zu Verfügung steht, Sie aber dennoch ein großes Netz drucken möchten.

Im Fenster wird angezeigt, wie viele Kacheln mit den aktuellen Einstellungen gedruckt würden.

Ist die Option ausgewählt, stehen im Bereich Einstellungen folgende Optionen zur Verfügung:

Überlappungen

Überlappung der einzelnen Kacheln in mm

Trennlinien zeichnen

Ist die Option ausgewählt, werden an den Schnittkanten Trennlinien gezeichnet. Wir empfehlen Trennlinien zu zeichnen, wenn eine Überlappung ungleich null angegeben ist, um die einzelnen Kacheln einfacher und genauer zusammenzufügen.

Die Darstellung wird gemäß Ihren Einstellungen gedruckt.

Layout für den schematischen Liniennetzplan einstellen

Sie können spezielle Einstellungen für die Anzeige des schematischen Liniennetzplans vornehmen. Die Layoutparameter können Sie in einer separaten Layoutdatei *.xml speichern (Layout des schematischen Liniennetzplans speichern und lesen) und Ihre Einstellungen für weitere Fahrpläne verwenden Siehe "Speichern und Übertragen der Layoutinformationen auf Angebotsvarianten". Außerdem können Sie einstellen, an welche Haltestellen Kanten angebunden sind und Anzeige der Kanten auf ein Zeitfenster begrenzen.

Stellen Sie sicher, dass das Fenster Schematischer Liniennetzplan geöffnet ist (Schematischen Liniennetzplan öffnen).

Klicken Sie in der Symbolleiste auf das Symbol .

Alternativ können Sie im Menü Schematischer Liniennetzplan den Eintrag Layoutparameter wählen.

Das Fenster Layoutparameter für schematischen Liniennetzplan öffnet sich.

Wählen Sie das Register Allgemeines.

Element

Beschreibung

Name

Name für das Layout eingeben

Aggregationsebene

Hier können Sie die gewünschte Aggregationsebene auswählen.

Hinweis

Eine Kante gruppiert alle räumlichen Abschnitte von Fahrplanfahrten zwischen aufeinander folgenden Anschlussknoten oder innerhalb eines Anschlussknotens, die zum gleichen Verkehrssystem, zum gleichen Betreiber, zur gleichen Linie etc. gehören. Die Menge der Kanten ist durch die Menge der Anschlussknoten, die Fahrtenauswahl und die ausgewählte Aggregationsebene festgelegt Siehe "Darstellung des Angebots im schematischen Liniennetzplan".

Folgende Ebenen stehen zur Auswahl:

  • Verkehrssystem gerichtet / ungerichtet
  • Betreiber gerichtet / ungerichtet
  • Oberlinie gerichtet / ungerichtet
  • Linie gerichtet / ungerichtet
  • Linienroute
  • Fahrzeitprofil
  • Fahrplanfahrt
  • Taktgruppen-Nummer

Identifikationsattribut bei nicht benutzten Kantenverläufen

Hier können Sie ein Identifikationsattribut auswählen, das die Fahrplanfahrten den Kantenverläufen zuordnet.

Hinweise

Die Auswahl hängt von der gewählten Aggregationsebene ab. Das ausgewählte Attribut wird in der Layoutdatei *.xml gespeichert.

Bei Änderungen der Fahrplandaten oder der Layoutparameter werden die Fahrplanfahrten anhand dieses Attributs den Kantenverläufen neu zugeordnet. Die Layoutparameter für ausgeblendete Kantenverläufe gehen dabei nicht verloren Siehe "Speichern und Übertragen der Layoutinformationen auf Angebotsvarianten". Sie können so das Layout eines Fahrplans für weitere Fahrpläne verwenden.

Fahrplanfahrt-Identifikationsattribut für Fahrplanfahrt-Auswahl

Hier können Sie ein Identifikationsattribut für Fahrplanfahrten auswählen. Die für die Anzeige ausgewählten Fahrplanfahrten werden mit diesem Attribut in der Layoutdatei *.xml gespeichert. Folgende Attribute stehen zur Auswahl:

  • Fahrplanfahrt: Nummer
  • Fahrplanfahrt: Name

Fahrplanfahrten virtuell auf komplette Fahrzeitprofile ausdehnen

Ist die Option ausgewählt, werden Fahrplanfahrten virtuell über das ganze Fahrzeitprofil ausgedehnt.

Hinweis

Fahrplanfahrten werden dann so behandelt, als würden sie vom ersten bis zum letzten Fahrzeitprofilelement reichen, ihre tatsächliche räumliche Ausdehnung wird also nicht beachtet.

Wählen Sie das Register Anbindung von Kanten.

Element

Beschreibung

Kanten einer Fahrt anbinden an...

Sie können einstellen, an welche Haltestellen Kanten angebunden sind. Unter den Optionen finden Sie ein erläuterndes Beispiel.

Grundsätzlich sind Kanten an alle Haltestellen angebunden, an denen das Ein- oder Aussteigen erlaubt ist.

alle Haltestellen, bei denen Ein- oder Aussteigen erlaubt ist

Ist die Option ausgewählt, sind Kanten nur an den Haltestellen angebunden, an denen das Ein- oder Aussteigen erlaubt ist.

alle Haltestellen mit Profilpunkten des Fahrzeitprofils

Ist die Option ausgewählt, sind Kanten zusätzlich an Haltestellen angebunden, die Profilpunkte des Fahrzeitprofils sind.

alle Haltestellen mit Routenpunkten der Linienroute

Ist die Option ausgewählt, sind Kanten zusätzlich an Haltestellen angebunden, die Routenpunkte der Linienroute sind.

alle Haltestellen der Linienroute

Ist die Option ausgewählt, sind Kanten zusätzlich an alle Haltestellen der Linienroute angebunden.

Beispiel

Eine Linienroute verläuft von A über B nach C. A und C sind Profilpunkte des Fahrzeitprofils, an denen Ein- oder Aussteigen erlaubt ist, B ist nur ein Routenpunkt der Linienroute. Ist eine der ersten zwei Optionen ausgewählt, sind die Kanten nur an die Anschlussknoten für A und C angebunden, an B laufen sie vorbei. Ist eine der letzten zwei Optionen ausgewählt, verlaufen Kanten zwischen den Anschlussknoten für A und B, sowie zwischen den Anschlussknoten für B und C.

Kanten einer Fahrt an einer Haltestelle...

Diese Option stellt eine weitere Filtermöglichkeit dar.

Ist die Option ausgewählt, können Sie ein Linienrouten-Verlaufsattribut auswählen. Kanten sind dann nur an jene Haltestellen angebunden, an denen dieses Attribut ungleich null oder nicht leer ist.

Hinweis

Kanten sind trotzdem immer an Haltestellen angebunden, an denen das Ein- oder Aussteigen erlaubt ist.

Wählen Sie das Register Zeitfenster.

Element

Beschreibung

Nur Kanten in Zeitfenster darstellen

Ist die Option ausgewählt, werden nur die Kanten einer Fahrplanfahrt angezeigt, die den nachfolgenden Filterbedingungen entsprechen.

Hinweis

Anders als im Fenster zur Auswahl der angezeigten Fahrplanfahrten, in dem Sie ein Zeitfenster einstellen können, in dem Fahrplanfahrten angezeigt werden, können Sie hier einstellen, welche Kanten einer Fahrplanfahrt in einem bestimmten Zeitfenster angezeigt werden.

Es werden nur die Kanten einer Fahrt dargestellt, auf denen die Fahrt innerhalb des unten angegebenen Zeitfensters ...

Hier können Sie festlegen, welche Kanten einer Fahrplanfahrt innerhalb des unten festgelegten Zeitfensters dargestellt werden.

am Von-Anschlussknoten abfährt

Ist die Option ausgewählt, werden nur die Kanten einer Fahrplanfahrt dargestellt, die am Von-Anschlussknoten abfahren.

zwischen Von-Anschlussknoten und Nach-Anschlussknoten unterwegs ist

Ist die Option ausgewählt, werden nur die Kanten einer Fahrplanfahrt dargestellt, deren Zeitintervall (Abfahrtszeit am Von-Anschlussknoten, Ankunftszeit am Nach-Anschlussknoten) sich mit dem festgelegten Zeitfenster überschneidet.

Hilfsgitter anzeigen

Sie können optional die Anzeige eines Hilfsgitters festlegen.

Wählen Sie im Menü Schematischer Liniennetzplan den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator die Seite Hilfsgitter.

Element

Beschreibung

Hilfsgitter

Hier können Sie folgende Einstellungen für das Hilfsgitter vornehmen.

Achsen zeichnen

Ist die Option ausgewählt, werden Achsen gezeichnet.

Gitterpunkte zeichnen

Ist die Option ausgewählt, werden Gitterpunkte gezeichnet. Sie können folgende Einstellungen für die Gitterpunkte vornehmen.

  • Punktgröße

Punktgröße in mm

  • Farbe

Über die Auswahlliste können Sie eine Farbe für das Gitter auswählen.

Gitterlinien zeichnen

Ist die Option ausgewählt, werden Gitterlinien gezeichnet. Sie können folgende Einstellungen für die Gitterlinien vornehmen.

  • Abstand

Abstand zwischen den Gitterlinien in Rastereinheiten

  • Linienstil

Hier können Sie festlegen, in welchem Stil die Gitterlinien gezeichnet werden (Strichdarstellung einstellen).

Die Einstellungen werden übernommen.

Darstellung und Klassengrenzen einer Punktobjektklasse bearbeiten

Fügen Sie Klassen für die Darstellung ein (Klassen und Klassengrenzen für eine Klassifizierung einfügen und Einzelne Klasse einfügen und löschen).

Wählen Sie bei Bedarf die Option Layer-Reihenfolge verwenden.

Ist die Option ausgewählt, können Sie festlegen, in welcher Reihenfolge die Netzobjekte der Klassen gezeichnet werden. Die Reihenfolge legen Sie über die Ziffern in der Spalte Layer fest. Die Spalte wird angezeigt, wenn die Option ausgewählt ist.

Standardmäßig werden die Netzobjekte der Klassen in der Reihenfolge gezeichnet, in der die Klassen in der Tabelle stehen.

Nehmen Sie bei Bedarf in der Tabelle die gewünschten Änderungen vor.

Sie können die Werte in den weiß hinterlegten Zellen ändern, indem Sie in die betreffende Zelle doppelklicken.

Eine Quickinfo zeigt die jeweiligen Werte an.

Über das Symbol  Attributauswahl können Sie festlegen können, welche Attribute als Spalten in der Tabelle angezeigt werden (Fenster zur Auswahl mehrerer Attribute). Mit Rechtsklick auf einen Spaltennamen können Sie über das Kontextmenü einfach einzelne Spalten entfernen oder hinzufügen.

Über das Symbol  Auf Standard-Attributauswahl zurücksetzen können Sie die in der Tabelle als Spalten angezeigten Attribute auf die Standardeinstellungen zurücksetzen.

Element

Beschreibung

Obergrenze

Die Spalte zeigt bei numerischen Attributen den oberen Grenzwert der Klasse an. Sie können die Werte direkt ändern (Einheiten: Stunden und Kilometer).

Zeichnen

Ist die Option für die jeweilige Klasse ausgewählt, wird die Klasse mit den eingestellten Optionen dargestellt. Wenn Sie eine Klasse nicht darstellen möchten, deaktivieren Sie das jeweilige Kästchen.

Vorschau

In diesem Feld wird die Darstellung der Klasse gemäß den aktuellen Einstellungen angezeigt.

Wenn Sie in dieses Feld klicken, werden die Darstellungseigenschaften der Klasse in einem neuen Fenster angezeigt. Hier können Sie sie ändern und über die Schaltfläche Nächste Klasse zur nächsten Klassen springen und diese bearbeiten.

Klassenname

Standardmäßig wird für Attribute die Klassengrenze angezeigt. Sie können die Werte ändern.

Hinweis

Sie können die Klassennamen wieder auf die Klassengrenzen zurücksetzen. Klicken Sie dazu mit der rechten Maustaste auf den Spaltenkopf und wählen Sie den Eintrag Alle Klassennamen zurücksetzen.

Klassennamen werden beispielsweise in der Legende angezeigt.

Layer

Die Ziffern bestimmen die Reihenfolge, in der die Klassen in der Netzdarstellung gezeichnet werden.

Hinweise

Die Spalte wird nur angezeigt, wenn die Option Layer-Reihenfolge verwenden ausgewählt ist.

Sie können die Reihenfolge auch für die Anzeige der Klassen in der Legende verwenden (Legende einfügen).

Punkt Typ

Hier wird der ausgewählte Typ angezeigt.

Wenn Sie in das Feld klicken, können Sie in der Auswahlliste einen anderen Typ auswählen.

Hinweis

Je nach ausgewähltem Typ sind weitere Attributfelder ausgegraut.

Form

Hier wird, je nach gewähltem Punkttyp, das ausgewählte Textattribut oder das ausgewählte Symbol angezeigt.

Wenn Sie beim ausgewählten Punkttyp Text in das Feld klicken, können Sie ein anderes Attribut auswählen.

Wenn Sie beim ausgewählten Punkttyp Symbol in das Feld klicken, können Sie in der Auswahlliste ein anderes Symbol auswählen.

Hinweis

Beim Punkttyp Bild ist das Feld ausgegraut.

Größe

Hier wird, je nach gewähltem Punkttyp, die Größe des Textes bzw. Symbols oder der Bildhöhe angezeigt. Sie können die Werte ändern.

Darstellung

Hier wird die Farbe und ggf. der Strichtyp des dargestellten Punktobjekts angezeigt.

Wenn Sie beim ausgewählten Punkttyp Text in das Feld klicken, können Sie eine andere Farbe für den dargestellten Text auswählen.

Wenn Sie beim ausgewählten Punkttyp Symbol in das Feld klicken, können Sie, je nach ausgewähltem Symbol, eine andere Füllfarbe und einen anderen Strichtyp festlegen.

Hinweise

Bei manchen Symbolen ist das Feld ausgegraut.

Wenn Sie beim Punkttyp Bild in das Feld klicken, wird das Symbol angezeigt. Über das Symbol können Sie eine Bilddatei auswählen.

Polygon Zeichnen

Ist die Option für die jeweilige Klasse ausgewählt, wird ein Polygon gezeichnet. Wenn Sie für die Klasse kein Polygon darstellen möchten, deaktivieren Sie das jeweilige Kästchen.

Polygon

Vorschau der Darstellung des Polygons

Hinweise

Ein Polygon wird nur gezeichnet, wenn die Option Polygon Zeichnen in der entsprechenden Zeile ausgewählt ist.

Wird ein Polygon gezeichnet, können Sie die Füllung und Umrandung in einem separaten Fenster festlegen, indem Sie in das Feld klicken (siehe „Eigenschaften eines geschlossenen Polygons bearbeiten“ auf Seite 1).

Klicken Sie in der Tabelle in die Zeile der gewünschten Klasse.

Klicken Sie in der Symbolleiste auf das Symbol  Grafikparameter bearbeiten.

Alternativ können Sie in der Tabelle in die Zeile der gewünschten Klasse doppelklicken.

Das Fenster Grafikparameter von Klasse <Nummer> bearbeiten öffnet sich.

Nehmen Sie die gewünschten Änderungen vor.

Element

Beschreibung

Vorschau

Hier wird die Darstellung der Klasse gemäß den aktuellen Einstellungen angezeigt.

Objekt zeichnen

Ist die Option ausgewählt, wird die ausgewählte Klasse im Netz gezeichnet.

Zeichnen bis Maßstab

Ist die Option ausgewählt, werden die Objekte beim Rauszoomen bis zum eingestellten Maßstab gezeichnet (Netzmaßstab ändern).

Ist die Option nicht ausgewählt, werden die Objekte immer dargestellt, unabhängig vom Netzmaßstab.

Tabelle zeichnen

Ist die Option ausgewählt, werden die Objekte gemäß den Einstellungen auf der Seite Tabelle mit einer Tabelle beschriftet (Netzobjekte mit Tabellen beschriften).

Diagramm zeichnen

Ist die Option ausgewählt, werden die Objekte gemäß den Einstellungen auf der Seite Diagramm beschriftet mit einem Diagramm (Netzobjekte mit Säulendiagrammen beschriften).

Polygon zeichnen

Ist die Option ausgewählt, werden vorhandene Umrandungen und Flächen von flächigen Punktobjekten gezeichnet. Sie können die Schraffur, Farbe und einen Strichtyp in einem separaten Fenster festlegen, indem Sie in das Feld klicken (Eigenschaften eines geschlossenen Polygons bearbeiten).

Punktobjekttyp

Hier wählen Sie aus, ob das Punktobjekt als Text, Symbol oder Bild dargestellt wird.

Text/Symbol/ Bild

Je nach gewählter Option wird das Punktobjekt als Text, Symbol oder Bild dargestellt. Die Einstellungen ändern sich entsprechend (Aktive Punktobjekte darstellen).

Vorige Klasse/ Nächste Klasse

Über die Schaltflächen können Sie zu der vorigen oder der nächsten Klasse wechseln und diese direkt bearbeiten.

Die Änderungen werden übernommen.

Alternativ können Sie die Bearbeitungsfunktionen auch über das Kontextmenü der Tabelle aufrufen.

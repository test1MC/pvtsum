Matrixspinnen anzeigen und Grundeinstellungen vornehmen

Sie legen fest, ob Matrixspinnen für Bezirke oder Oberbezirke angezeigt werden, und nehmen Grundeinstellungen vor. Matrixspinnen können aus folgenden Elementen bestehen:

  • Matrixspinnenbalken
  • Beschriftungen von Matrixspinnenbalken
  • Matrixspinnenstrecken

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Alternativ können Sie das Fenster öffnen, indem Sie gleichzeitig die Tasten Strg und G drücken.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag Matrixspinnen Bezirke.

Wenn Sie Matrixspinnen für Oberbezirke anzeigen möchten, wählen Sie im Navigator den Eintrag Matrixspinnen Oberbezirke. Die Parameter entsprechen sich.

Wählen Sie das Register Layersteuerung.

Element

Beschreibung

Layer Matrixspinnen Bezirke zeichnen

Ist die Option gewählt, können Matrixspinnen angezeigt werden.

Balken zeichnen

Ist die Option gewählt, können Matrixbalken angezeigt werden.

Balkenbeschriftungen zeichnen

Ist die Option gewählt, können Balkenbeschriftungen angezeigt werden.

Matrixspinnenstrecken zeichnen

Ist die Option gewählt, wird jede Beziehung durch eine Linie angezeigt.

Wählen Sie das Register Allgemeine Einstellungen.

Element

Beschreibung

Querschnittswerte verwenden

Ist die Option ausgewählt, wird der Mittelwert aus der Hin- und Rückrichtung ermittelt.

Ist die Option nicht ausgewählt, werden die Werte für Hin- und Rückrichtung getrennt angezeigt.

Hinweis

Wenn ein darzustellender Querschnittswert gemittelt wird, wird intern durch zwei geteilt, so dass als Skalierungsobergrenze die Hälfte des Querschnittswertes gilt. Beispielsweise wird bei einem Skalierungsbereich von 1-1000 und einer Balkenbreite von 10 mm ein Querschnittswert von 2000 auf beiden Seiten jeweils mit einer Belastung von 1000 und einer Balkenbreite von 10 mm dargestellt.

Abstand zwischen Balken

Abstand zwischen den Matrixspinnenbalken in mm

Strichdarstellung

Sie können die Strichdarstellung in einem separaten Fenster ändern, indem Sie in das Feld klicken (Strichdarstellung einstellen).

Textpositionierung vertikal

außerhalb Balken

Ist die Option ausgewählt, wird die Beschriftung außerhalb des Balkens angezeigt.

innerhalb Balken

Ist die Option ausgewählt, wird die Beschriftung innerhalb des Balkens angezeigt.

Textabstand

Abstand der Beschriftung von der Matrixspinnenstrecke (wenn die Option innerhalb Balken ausgewählt ist) bzw. Abstand der Beschriftung vom Balken (wenn die Option außerhalb Balken ausgewählt ist).

Wählen Sie das Register Textformat.

Element

Beschreibung

Schriftart und Farbe

Schriftart

Sie können in einem separaten Fenster eine Schriftart und einen Schriftschnitt auswählen, indem Sie in das Feld klicken.

Schriftgröße

Textgröße in mm

Hinweis

Bei einer Textgröße von 0.0 mm wird keine Balkenbeschriftung ausgegeben.

Schriftfarbe wie Balkenfüllfarbe

Ist die Option ausgewählt, wird die Schrift in der gleichen Farbe angezeigt, wie der Balken.

Tipp

Wir empfehlen diese Option, wenn mehrere Balken angezeigt werden.

Ist die Option nicht ausgewählt, können Sie über die Auswahlliste die Farbe auswählen.

Textfeld

Transparent

Ist die Option ausgewählt, wird die Balkenbeschriftung nur als Schrift ohne weißen Hintergrund gezeichnet.

Rahmen zeichnen

Ist die Option ausgewählt, wird ein Rahmen um die Beschriftung gezeichnet. Über die Auswahlliste können Sie die Farbe auswählen.

Ausrichtung

Über die Auswahlliste können Sie die Anordnung und Anzeige mehrerer Balkenbeschriftungen festlegen:

  • Horizontal

Die Beschriftungen stehen hintereinander.

  • Vertikal

Die Beschriftungen stehen untereinander.

  • Vertikal mit Summe

Alle Beschriftungen stehen untereinander, darunter wird die Summe der Werte angezeigt.

  • Nur Summe

Nur die Summe der Werte wird angezeigt.

Zeilenabstand

Zeilenabstand in Prozent der aktuellen Textgröße

Wählen Sie das Register Beziehungsauswahl.

Element

Beschreibung

Beziehungen werden dargestellt

Maximale Anzahl der Beziehungen, die gezeichnet werden

Es werden die Beziehungen mit den größten Attributwerten im Skalierungsattribut gezeichnet (Darstellungseigenschaften eines Balkens festlegen).

Nur aktive Beziehungen zeichnen

Ist die Option ausgewählt, werden nur aktive Beziehungen gezeichnet.

Alle Beziehungen berücksichtigen

Ist die Option ausgewählt, werden alle Beziehungen gezeichnet.

Nur ausgewählte Beziehungen zeichnen

Ist die Option ausgewählt, können Sie Beziehungen mit bestimmten Attributwerten anzeigen. Über die Schaltfläche unter der Option wählen Sie das gewünschte Attribut.

Über die Schaltfläche Bearbeiten erstellen Sie die Klassifizierung (Objekte anhand von Attributwerten klassifiziert darstellen). Über das Kontrollkästchen in der Spalte Zeichnen legen Sie fest, ob die Beziehungen der jeweiligen Klasse gezeichnet werden.

Layer-Reihenfolge verwenden

Ist die Option ausgewählt, können Sie festlegen, in welcher Reihenfolge die Klassen gezeichnet werden. Die Reihenfolge legen Sie über die Ziffern in der Spalte Layer fest. Die Spalte wird angezeigt, wenn die Option ausgewählt ist. Standardmäßig werden die Netzobjekte der Klassen in der Reihenfolge gezeichnet, in der die Klassen in der Liste stehen.

Matrixspinnen werden grundsätzlich angezeigt und die grundlegenden Einstellungen sind festgelegt. Sie müssen nun noch auf der Seite Matrixspinnen Bezirke > Darstellung die einzelnen Balken einfügen und ihre Darstellungseigenschaften festlegen (Einen oder mehrere Balken festlegen und Darstellungseigenschaften eines Balkens festlegen).

Darstellung für Kantenverläufe einstellen

Ein Kantenverlauf im schematischen Liniennetzplan repräsentiert eine Menge von Fahrplanfahrten (siehe Grundlagen: Kap. , Seite 1). Kantenverläufe können sowohl außerhalb als auch innerhalb von Anschlussknoten gezeichnet werden.

Um die Kantenverläufe innerhalb der Anschlussknoten sichtbar zu machen, müssen Sie für Anschlussknoten eine transparente Füllfarbe auswählen (siehe „Darstellung für Anschlussknoten einstellen“ auf Seite 1).
Anzeige der Kantenverläufe einstellen

Sie können die Anzeige der Kantenverläufe folgendermaßen einstellen:

Wählen Sie im Menü Schematischer Liniennetzplan den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator die Seite Kantenverläufe.

Element

Beschreibung

Einheitliche Darstellung

Ist die Option ausgewählt, werden die Kantenverläufe einheitlich dargestellt.

Im Bereich Darstellung können Sie folgende Einstellungen für die Anzeige vornehmen.

Zeichnen

Ist die Option ausgewählt, werden Kantenverläufe gezeichnet.

Frequenz

Die Bedienungsfrequenz ist ein ganzzahliger Wert zwischen 1 und 4. Er gibt die Anzahl der Striche an, die je Kantenverlauf gezeichnet werden. Die Teilstriche werden parallel zueinander gezeichnet.

Über die Auswahlliste können Sie den gewünschten Wert wählen.

Kantenverlauf

Wenn Sie in das Feld klicken, öffnet sich ein neues Fenster. Hier können Sie den Linienstil festlegen, mit dem die Kantenverläufe angezeigt werden (siehe „Linienstildarstellung einstellen“ auf Seite 1).

Pfeilspitze

Ist die Option ausgewählt, wird eine Pfeilspitze gezeichnet. Wenn Sie in das Feld klicken, öffnet sich ein neues Fenster. In dem Fenster können Sie festlegen, wie die Pfeilspitze dargestellt wird.

Hinweis

Eine Pfeilspitze wird nur bei Kantenverläufen der Hinrichtung gezeichnet. Werden beide Richtungen in einem Kantenverlauf dargestellt, wird keine Pfeilspitze gezeichnet.

Mittenbeschriftung

Ist die Option ausgewählt, wird eine Beschriftung in der Mitte des Verlaufs gezeichnet.

Hinweis

Eine Mittenbeschriftung wird nicht angezeigt, wenn der Wert des ausgewählten Attributs null oder leer ist.

Wenn Sie in das Feld klicken, öffnet sich das Fenster Grafikparameter Text bearbeiten in dem Sie weitere Einstellungen vornehmen können (siehe „Darstellung für Kantenverläufe einstellen“ auf Seite 1).

Mittenbeschriftung gerichtet

Ist die Option ausgewählt, wird eine gerichtete Mittenbeschriftung, also eine Beschriftung je Richtung, gezeichnet.

Wenn Sie in das Feld klicken, öffnet sich das Fenster Grafikparameter Text bearbeiten in dem Sie weitere Einstellungen vornehmen können (siehe „Beschriftungsformen für Kantenverläufe einstellen“ auf Seite 5)

 

Anfangsbeschriftung

Ist die Option ausgewählt, wird eine Anfangsbeschriftung gezeichnet.

Sie wird am Anfang eines Kantenverlaufs jeweils rechts neben der Kante gezeichnet.

Wenn Sie in das Feld klicken, öffnet sich das Fenster Grafikparameter Text bearbeiten in dem Sie weitere Einstellungen vornehmen können (siehe „Beschriftungsformen für Kantenverläufe einstellen“ auf Seite 5)

Endbeschriftung

Ist die Option ausgewählt, wird eine Endbeschriftung gezeichnet.

Sie wird am Ende eines Kantenverlaufs jeweils rechts neben der Kante gezeichnet.

Wenn Sie in das Feld klicken, öffnet sich das Fenster Grafikparameter Text bearbeiten in dem Sie weitere Einstellungen vornehmen können (siehe „Beschriftungsformen für Kantenverläufe einstellen“ auf Seite 5)

Endknotenbeschriftung

Ist die Option ausgewählt, wird eine Beschriftung innerhalb des Endknotens gezeichnet. Wenn Sie in das Feld klicken, öffnet sich ein neues Fenster.

Wenn Sie in das Feld klicken, öffnet sich das Fenster Grafikparameter Text bearbeiten in dem Sie weitere Einstellungen vornehmen können (siehe „Beschriftungsformen für Kantenverläufe einstellen“ auf Seite 5)

Hinweise

Ein Anschlussknoten ist der Endknoten eines Kantenverlaufs, wenn die Kante an dem Anschlussknoten beginnt oder endet. Eine Kante beginnt oder endet, wenn es keine andere Kante gibt, auf der die Fahrplanfahrt fortgeführt werden kann.

Die Beschriftungen der Kantenverläufe werden nur an äußeren Kantenverläufen gezeichnet, also nicht innerhalb von Anschlussknoten. Die Beschriftung wird außerdem nur gezeichnet, wenn mindestens einer der beiden Anschlussknoten eines Kantenverlaufs gezeichnet wird.

Wenn Sie die Option Einheitliche Darstellung ausgewählt haben, können Sie die Darstellung der Kantenverläufe dennoch im unteren Teil der Seite über die Option Klassifizierte Darstellung klassifizieren.

Sie können bei Bedarf auf die gleiche Weise die Einstellungen für markierte Kantenverläufe im Register Markiert vornehmen.

Klassifizierte Darstellung

Ist die Option ausgewählt, werden die Kantenverläufe nach Klassen differenziert dargestellt (siehe „Objekte anhand von Attributwerten klassifiziert darstellen“ auf Seite 1).

Über eine Schaltfläche können Sie ein Kantenverlaufsattribut auswählen, nach dem die Darstellung klassifiziert werden soll.

Über die Symbolleiste können Sie die gewünschten Klassen und Attribute einfügen (siehe „Einzelne Klasse einfügen und löschen“ auf Seite 1 und „Darstellung und Klassengrenzen einer Klasse bearbeiten“ auf Seite 1). In der Tabelle können Sie die gewünschten Einstellungen für die Klassen vornehmen.

Hinweis

Innerhalb der Bearbeitungsfenster der einzelnen Klassen können Sie eine weitere Klassifizierung vornehmen, beispielsweise um eine Darstellung der Bedienungsfrequenz mit den Standardfarben für bestimmte Linien zu kombinieren.

Alle Klassen neu belegen

Über die Schaltfläche öffnen Sie das Fenster Grafikparameter für Kantenverläufe neu belegen, in dem Sie in drei Schritten die Einstellungen für die Klassifizierung vornehmen können, wenn Sie beispielsweise eine Darstellung der Bedienungsfrequenz mit den Standardfarben für bestimmte Linien kombinieren möchten.

Die Kantenverläufe werden gemäß den Einstellungen dargestellt.

Wählen Sie im Fenster Grafikparameter bearbeiten: Schematischer Liniennetzplan im Navigator die Seite Kantenverläufe.

Das Fenster Grafikparameter für Kantenverläufe neu belegen Schritt 1 von 3 öffnet sich.

Wählen Sie über die Schaltfläche ein Attribut für die äußere Klassifizierung aus.

(siehe „Farben einer Klassifizierung eines Linienobjekts ändern“ auf Seite 1 und „Darstellung für Kantenverläufe einstellen“ auf Seite 1)

Das Fenster Grafikparameter für Kantenverläufe neu belegen Schritt 3 von 3 öffnet sich.

Legen Sie die Einstellungen der Strichelung und Frequenz für die gewünschten Klassen neu fest.

Wenn Sie ausschließlich die Frequenz und Strichelung einiger Klassen in einer bestehenden Klassifizierung ändern möchten, klicken Sie auf die Schaltfläche Frequenz und Strichelung für ausgewählte Klassen ändern.

Die Einstellungen für die klassifizierte Darstellung der Kantenverläufe werden gemäß der Einstellungen geändert.

Die Darstellung des Schematischen Liniennetzplans wird gemäß den neuen Einstellungen geändert.

Beschriftungsformen für Kantenverläufe einstellen

Sie können über Beschriftungsformen einstellen, wie Text in der Beschriftung von Kantenverläufen dargestellt wird.

Wählen Sie im Fenster Grafikparameter bearbeiten: Schematischer Liniennetzplan im Navigator die Seite Kantenverläufe > Beschriftungsformen.

Die standardmäßig angelegten Beschriftungsformen werden in einer Tabelle angezeigt. Alle Attribute der Darstellung werden standardmäßig angezeigt.

Die Beschriftungsform Standard kann als einzige nicht gelöscht werden. Auf diesen Standard werden die 'Beschriftungen zurückgesetzt, falls benutzerdefinierte Beschriftungsformen gelöscht werden.

Element Beschreibung

Fügt eine neue Beschriftungsform mit den Einstellungen von Standard ein.

Dupliziert eine oder mehrere markierte Beschriftungsformen

Löscht die markierte Beschriftungsform

Öffnet das Fenster Attributauswahl, in dem Sie festlegen können, welche Attribute als Spalten in der Tabelle angezeigt werden (siehe „Fenster zur Auswahl mehrerer Attribute“ auf Seite 1).

Setzt die in der Tabelle als Spalten angezeigten Attribute auf die Standardeinstellungen zurück.

Rahmen Zeichnen

Rahmen

Die Kantenverläufe werden gemäß den Einstellungen beschriftet.

Beschriftungsvarianten für Abfahrts- und Ankunftsfahrzeiten einstellen

Beschriftungsvarianten passen die Textformatierung einer Beschriftungsform an. Sie wirken bei der Darstellung von Attributen, die zwei- und vierstündlich angegebene Abfahrts- und Ankunftszeiten darstellen. Es gibt vier Beschriftungsvarianten, deren Werte frei angepasst werden können.

  • Ankunfts- und Abfahrtzeiten (gerade Stunde)
  • Ankunfts- und Abfahrtzeiten (ungerade Stunde)
  • Ankunfts- und Abfahrtzeiten (zweite Stunde)
  • Ankunfts- und Abfahrtzeiten (dritte Stunde)

    • Beschriftungvarianten sind nur für die Beschriftung bei 2- oder 4-Stunden-Takten relevant. Das entsprechende Attribut für die Beschriftung muss ausgewählt sein.

    Wählen Sie im Menü Schematischer Liniennetzplan den Eintrag Grafikparameter bearbeiten.

    Das Fenster Grafikparameter bearbeiten öffnet sich.

    Wählen Sie im Navigator die Seite Kantenverläufe > Beschriftungsvarianten.

    Die Beschriftungsvarianten werden in einer Tabelle angezeigt.

Aktive Linienobjekte darstellen

Sie können festlegen, wie aktive Linienobjekte im Netzfenster dargestellt werden.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten: Netzeditor öffnet sich.

Wählen Sie im Navigator den Eintrag Strecken.

Die Grafikparameter sind für alle Linienobjekte nahezu identisch. Diese Anleitung beschreibt beispielhaft die Einstellungen für das Linienobjekt Strecke.

Element

Beschreibung

Layer Strecken zeichnen

Hier können Sie einstellen, welche Bestandteile des Layers Strecken angezeigt werden.

Ist die Option ausgewählt, werden Strecken in der Netzdarstellung gezeichnet.

Ist die Option nicht ausgewählt, sind die untergeordneten Optionen ausgegraut.

Strecken zeichnen

Ist die Option ausgewählt, werden Strecken gezeichnet.

Beschriftungen zeichnen

Ist die Option ausgewählt, werden die Strecken beschriftet (Linienobjekte beschriften).

Balken zeichnen

Ist die Option ausgewählt, werden Streckenbalken gezeichnet (Eigenschaften über Balken darstellen).

Balkenbeschriftungen zeichnen

Ist die Option ausgewählt, werden die Streckenbalken beschriftet.

Zeichenreihenfolge

Die Liste zeigt die Reihenfolge, in der die Layer gezeichnet werden. Über die Symbole und können Sie die Reihenfolge der gezeichneten Layer ändern.

Wählen Sie im Navigator den Eintrag Strecken > Darstellung > Register Aktiv.

Alternativ können Sie die Seite aufrufen, indem Sie im Fenster Netz mit der rechten Maustaste auf das Symbol des Netzobjekttyps klicken.

Element

Beschreibung

Einheitliche Darstellung

Ist die Option ausgewählt, werden alle aktiven Objekte einheitlich dargestellt.

Klassifizierte Darstellung

Ist die Option ausgewählt, können die aktiven Objekte nach einem Attribut klassifiziert und dargestellt werden (Objekte anhand von Attributwerten klassifiziert darstellen).

Optionen bei einheitlicher Darstellung

In diesem Bereich legen Sie fest, wie die Linienobjekte gezeichnet werden.

Vorschau

Hier sehen Sie, wie eine Strecke gemäß den aktuellen Einstellungen gezeichnet wird.

Objekt zeichnen

Ist die Option ausgewählt, werden aktive Linienobjekte gezeichnet.

Zeichnen bis Maßstab

Ist die Option ausgewählt, werden die Linienobjekte beim Rauszoomen bis zum eingestellten Maßstab gezeichnet (Netzmaßstab ändern).

Ist die Option nicht ausgewählt, werden die Linienobjekte immer dargestellt, unabhängig vom Netzmaßstab.

Linienstil

Hier können Sie festlegen, in welchem Stil der Linienstrich gezeichnet wird (Linienstildarstellung einstellen).

Pfeilspitze zeichnen

Ist die Option ausgewählt, wird eine Pfeilspitze gezeichnet. Für die Pfeilspitze nehmen Sie die folgenden Einstellungen vor.

Über die Auswahlliste wählen Sie das Symbol, das als Pfeilspitze dargestellt wird.

  • Größe

Pfeilspitzengröße in mm

  • Farbe

Sie können eine Farbe für die Pfeilspitze in einem separaten Fenster auswählen, indem Sie in das Feld klicken (Eigenschaften eines geschlossenen Polygons bearbeiten).

  • Zeichen

Wenn Sie Pfeilspitze Kreis oder Quadrat auswählen, können Sie über das Kontrollkästchen ein Eingabefeld aktivieren. In das Eingabefeld können Sie ein Zeichen eingeben, das innerhalb des Kreises oder Quadrates dargestellt wird. Über die Auswahlliste können Sie eine Farbe für das Zeichen auswählen.

Wenn Sie auf die Schaltfläche Vorschau klicken, werden Ihre Änderungen sofort angezeigt, ohne dass sich das Fenster Grafikparameter bearbeiten: Netzeditor schließt.

Die Linienobjekte werden gemäß den Einstellungen dargestellt.

Listenlayout speichern

Sie haben zwei Möglichkeiten Listenlayouts zu speichern.

Erste Möglichkeit

Stellen Sie sicher, dass das Listenfenster der gewünschten Liste geöffnet ist.

Nehmen Sie die gewünschten Listeneinstellungen vor (Listenlayout anpassen).

Wählen Sie im Menü Liste (<Netzobjekttyp>) den Eintrag Layout speichern.

Das Fenster Listen-Layout speichern öffnet sich.

Speichern Sie die Datei am gewünschten Ort ab.

Die Einstellungen werden in eine Datei *.lla gespeichert.

Zweite Möglichkeit

Stellen Sie sicher, dass das Listenfenster der gewünschten Liste geöffnet ist.

Nehmen Sie die gewünschten Listeneinstellungen vor (Listenlayout anpassen).

Geben Sie in das Listenfeld Listenlayout auswählen einen Namen für das Layout ein.

Drücken Sie die Taste Enter.

Die Einstellungen werden unter dem eingegebenen Namen in eine Datei *.lla im für Listen-Layoutdateien eingestellten Projektverzeichnis gespeichert (Projektverzeichnisdatei öffnen und speichern).

Listenlayout als Standardlayout speichern

Intern verwendet für jede Listenart ein Standardlayout. Wenn Sie eine Liste öffnen, entspricht diese dem Standardlayout der Listenart. Sie können das Standardlayout einer Liste ändern, indem Sie ein geändertes Layout als Standardlayout speichern.

Stellen Sie sicher, dass das Listenfenster der gewünschten Liste geöffnet ist.

Nehmen Sie die gewünschten Listeneinstellungen vor.

Wählen Sie im Menü Liste (<Netzobjekttyp>) den Eintrag Als Standardlayout speichern.

Die Einstellungen werden in eine Datei std_*.lla, beispielsweise std_link.lla für Strecken, in Ihrem Benutzerverzeichnis gespeichert.

Das benutzerdefinierte Standardlayout wird für jedes neue Listenfenster der Listenart verwendet.

Wenn Sie wieder das -interne Standardlayout verwenden möchten, müssen Sie das benutzerdefinierte Standardlayout in Ihrem Benutzerverzeichnis löschen.

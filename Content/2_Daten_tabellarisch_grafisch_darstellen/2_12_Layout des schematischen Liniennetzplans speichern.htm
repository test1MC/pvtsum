Layout des schematischen Liniennetzplans speichern

Das Layout des schematischen Liniennetzplans wird in einer Datei *.xml gespeichert.

Nehmen Sie die gewünschten Einstellungen im schematischen Liniennetzplan vor (Layout für den schematischen Liniennetzplan einstellen).

Wählen Sie im Menü Schematischer Liniennetzplan den Eintrag Layout speichern unter.

Das Fenster Layout schematischer Liniennetzplan speichern öffnet sich.

Wählen Sie den gewünschten Pfad und geben Sie den gewünschten Dateinamen ein.

Klicken Sie auf die Schaltfläche Speichern.

Die Layoutparameter des schematischen Liniennetzplans werden gespeichert.

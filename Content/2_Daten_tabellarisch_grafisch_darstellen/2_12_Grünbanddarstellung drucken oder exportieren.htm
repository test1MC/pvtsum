Grünbanddarstellung drucken oder exportieren

Sie können die Grünbanddarstellung drucken und als Grafikdatei exportieren.

Grünbanddarstellung drucken

Nehmen Sie die gewünschten Einstellungen für das Drucken vor.

Einstellungen für den Druckrahmen treffen Sie in den Grafikparametern, auf der Seite Druckrahmen (Grafikparameter für die Grünbanddarstellung einstellen).

Weitere Druck-Einstellungen entsprechen denen beim Drucken der Netzdarstellung (Netzdarstellung drucken).

Stellen Sie sicher, dass das Fenster Grünbanddarstellung geöffnet ist (Grünbanddarstellung öffnen).

Zeigen Sie im Menü Datei auf den Eintrag Drucken.

Wählen Sie den Eintrag Drucken.

Das Fenster Drucken: Grünbanddarstellung öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (Druckausgabe starten).

Die Grünbanddarstellung wird gedruckt.

Grünbanddarstellung als Grafikdatei exportieren

Sie können die Grünbanddarstellung direkt in einer Grafikdatei *.svg oder *.dxf speichern.

Stellen Sie sicher, dass das Fenster Grünbanddarstellung geöffnet ist (Grünbanddarstellung öffnen).

Wählen Sie im Menü Grünbanddarstellung einen der Einträge SVGExport oder DXF Export.

Das Fenster SVG-Datei speichern oder DXF-Datei speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster SVG-Export oder DXF-Export öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (SVG-Export und DXF Export).

Die Grünbanddarstellung wird in einer Datei *.svg oder *.dxf gespeichert.

Sie können die Grünbanddarstellung auch als Grafikdatei exportieren, indem Sie im Menü Grünbanddarstellung den Eintrag JPG Export (Screenshot) wählen oder im Fenster Grünbanddarstellung in der Symbolleiste auf das Symbol  JPG Export (Screenshot) klicken (Screenshot vom Bildschirm exportieren).

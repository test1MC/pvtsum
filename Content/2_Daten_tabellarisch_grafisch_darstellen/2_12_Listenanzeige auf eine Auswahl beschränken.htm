Listenanzeige auf eine Auswahl beschränken

In einigen Listen können Sie die angezeigten Objekte beschränken, indem Sie ein Auswahlkriterium einstellen.

Nur GIS-Objekte oder POIs einer Kategorie anzeigen

In den Listen GIS-Objekte und Points of Interest können Sie eine Kategorie auswählen. Die Liste enthält dann nur die Objekte dieser Kategorie.

Öffnen Sie die gewünschte Liste (siehe „Liste öffnen“ auf Seite 1).

Wählen Sie über die Auswahlliste in der Symbolleiste eine Kategorie.

Die Liste enthält nur noch die Objekte der ausgewählten Kategorie. Bei POIs enthält sie zudem alle Objekte der Unterkategorien der ausgewählten Kategorie.

Anzeige auf ein Nachfragesegment beschränken

In einigen Auswertungslisten können Sie die Anzeige auf ein Nachfragesegment begrenzen.

Öffnen Sie die gewünschte Auswertungsliste (siehe „Liste öffnen“ auf Seite 1).

Die Liste wird angezeigt.

Wählen Sie in der Auswahlliste NSeg ein Nachfragesegment aus.

Die Auswertungen werden ausschließlich für das ausgewählte Nachfragesegment angezeigt.

Anzeige auf eine Wegemenge oder ein Nachfragesegment beschränken

In einigen Auswertungslisten können Sie die Anzeige auf eine Wegemenge oder ein Nachfragesegment beschränken.

Öffnen Sie die gewünschte Auswertungsliste (siehe „Liste öffnen“ auf Seite 1).

Die Liste wird angezeigt.

Wählen Sie in der Auswahlliste NSeg / Wegemenge einen Eintrag aus.

Die Auswertungen werden ausschließlich für die ausgewählte Wegemenge/das Nachfragesegment angezeigt.

Anzeige auf aktive Fahrzeitprofile oder aktive Strecken beschränken

In einigen Auswertungslisten können Sie die Anzeige auf nur aktive Fahrzeitprofile (Liste ÖV-Teilwege) oder Strecken (Liste IV-Wege streckenfein) begrenzen.

Öffnen Sie die gewünschte Auswertungsliste (siehe „Liste öffnen“ auf Seite 1).

Die Liste wird angezeigt.

Wählen Sie die Option nur aktive Fahrzeitprofile oder nur aktive Strecken aus.

Lediglich die aktiven Fahrzeitprofile oder Strecken werden in der Liste angezeigt.

Wegelistenanzeige auf eine Auswahl beschränken

In den Wege- und Umsteigerlisten können Sie die Anzeige auf eine Auswahl begrenzen.

Öffnen Sie die gewünschte Wegeliste (siehe „Liste öffnen“ auf Seite 1).

Die Liste wird angezeigt.

Wählen Sie unter Auswahl eine der folgenden Ansichten aus.

Element

Beschreibung

alle Routen

Alle Wege oder Routen der Umlegung werden angezeigt.

Spinne-Routen

Alle Wege oder Routen der aktuellen Spinnenausgabe werden angezeigt.

Hinweis

Dieser Eintrag steht nur für Wege eines Nachfragesegments zur Verfügung, nicht für Wege einer Wegemenge.

Beziehungsfilter

Alle Wege oder Routen des aktuellen Beziehungsfilters werden angezeigt.

Die Auswahl wird angezeigt.

Wegelistenanzeige auf einen Quellbezirk beschränken

Generell zeigt eine Liste der Wege die Wege des ersten Quellbezirks in der Liste an. Sie können die Auswahl des Quellbezirks ändern oder alle Quellbezirke anzeigen lassen.

Öffnen Sie die Liste ÖV-Wege oder IV-Wege (siehe „Liste öffnen“ auf Seite 1)

Klicken Sie auf die Schaltfläche neben Quellbezirksfilter.

Das Fenster Quellbezirk suchen öffnet sich.

Wählen Sie den gewünschten Quellbezirk aus (siehe „Netzobjekte im Netz suchen“ auf Seite 1).

In der Liste werden die Wege des ausgewählten Quellbezirks oder alle Wege angezeigt.

Fußwege in der ÖV-Teilwegliste ausblenden

Generell zeigt die Liste ÖV-Teilwege alle Teilwege inklusive Fußwege an. In der Liste können Sie bei Bedarf die Fuß-Teilwege (Abgang, Umstieg und Zugang) ausblenden.

Öffnen Sie die Liste ÖV-Teilwege (siehe „Liste öffnen“ auf Seite 1).

Die Liste wird angezeigt.

Wählen Sie die Option nur ÖV-Teilwege aus.

In der Liste werden die Fuß-Teilwege ausgeblendet.

Einsteige-, Aussteige- und Umsteigevorgänge an Haltestellen anzeigen

In der ListeÖV-Auswertungen > Umsteiger Haltestellen/Fahrzeitprofile können Sie gezielt die Quell-Einsteiger, Ziel-Aussteiger und Umsteiger an einer bestimmten Haltestelle anzeigen.

Öffnen Sie die Liste ÖV-Auswertungen > Umsteiger Haltestellen/Fahrzeitprofile (siehe „Liste öffnen“ auf Seite 1).

Wählen Sie aus den Optionen Quell-Einsteiger, Umsteiger und Ziel-Aussteiger die Elemente aus, die angezeigt werden sollen.

Klicken Sie oberhalb der Liste auf die mit der Haltestelle beschriftete Schaltfläche.

Das Fenster Haltestelle auswählen öffnet sich.

Wählen Sie in der Liste die gewünschten Haltestellen aus und klicken Sie auf die Schaltfläche Auswahl übernehmen.

Wenn Sie auf die Schaltfläche Alle klicken, werden die Umsteigevorgänge an allen Haltestellen angezeigt.

In der Liste werden die Einsteige-, Aussteige- und Umsteigevorgänge an den gewünschten Haltestellen angezeigt.

Einsteige-, Aussteige- und Umsteigevorgänge auf Objekte der Linienhierarchie einschränken

Öffnen Sie die Liste ÖV-Auswertungen > Umsteiger Haltestellen/Fahrzeitprofile (siehe „Liste öffnen“ auf Seite 1).

Wählen Sie aus den Optionen Quell-Einsteiger, Umsteiger und Ziel-Aussteiger die Elemente aus, die angezeigt werden sollen.

Klicken Sie auf das schwarze Dreieck der standardmäßig mit Fahrzeitprofile beschrifteten Schaltfläche.

Eine Auswahlliste der Objekte der Linienhierarchie öffnet sich.

Wählen Sie in der Liste das gewünschte Objekt aus.

Die Schaltfläche der Auswahlliste und die weiteren Optionen werden entsprechend des gewählten Objektes beschriftet.

Klicken auf die Schaltfläche neben <Objekt der Linienhierarchie>.

Das Fenster <Objekt der Linienhierarchie> auswählen öffnet sich.

Wählen Sie die gewünschten Objekte aus der Liste aus und klicken Sie auf die Schaltfläche Auswahl übernehmen.

Wählen Sie bei Bedarf die Option Von- und Nach-<Objekt der Linienhierarchie> separat filtern aus.

Klicken Sie auf die Schaltflächen, die mit Von-<Objekt der Linienhierarchie> und Nach-<Objekt der Linienhierarchie> beschriftet sind.

Wählen Sie jeweils die gewünschten Objekte aus der Liste aus und klicken Sie auf Auswahl übernehmen.

Die Liste zeigt die Einsteige-, Aussteige- und Umsteigevorgänge gemäß der eingestellten Einschränkungen an.

Abschnitte in Verlaufslisten anzeigen

In den Verlaufslisten können Sie von der klassischen Ansicht auf eine nach Abschnitten gegliederte Sicht umschalten. Das letzte Verlaufselement wird in dieser Ansicht nicht angezeigt.

Öffnen Sie die gewünschte Verlaufsliste (siehe „Liste öffnen“ auf Seite 1).

Klicken Sie in der Symbolleiste auf das Symbol  Abschnittsansicht.

Die Verlaufsliste wird in der Abschnittsansicht angezeigt.

Wenn Sie das Symbol erneut klicken, kehren Sie in die klassische Ansicht zurück.
Matrizenanzeige auf eine Auswahl beschränken

Sie können die Anzeige der Matrizen in der Liste (Matrizen) einschränken, indem Sie den Matrixfilter verwenden.

Öffnen Sie Liste Nachfrage > Matrizen.

Klicken Sie neben dem Eingabefeld Matrixfilter auf das Symbol .

Das Fenster Matrix auswählen öffnet sich.

Wählen Sie die gewünschten Matrizen aus (siehe „Matrizen auswählen“ auf Seite 1).

Um die Änderungen in den Eingabefeldern zu verwerfen, klicken Sie auf die Schaltfläche Leere Auswahl.

Die gewünschten Matrizen werden in der Liste angezeigt.

Sie können bei Bedarf die Matrixliste auch mehrfach öffnen und unterschiedlich filtern.

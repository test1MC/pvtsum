Das Listenfenster

Wenn Sie eine Listenart ausgewählt haben, öffnet sich ein Listenfenster.

(1) Titelleiste

In der Titelleiste wird der Name der Liste in eckigen Klammern angezeigt. Sie können den Namen der Liste ändern (Fenstertitel von Listen ändern).

(2) Menüleiste

Sobald ein Listenfenster aktiv ist, wird in der Menüleiste ganz rechts das Menü Liste (<Netzobjekttyp>) angezeigt.

(3) Spaltenkopf

Im Spaltenkopf wird, je nach Einstellung, der Kurz- oder Langname des Attributs angezeigt. Dabei werden eventuell ausgewählte Relationen vorangestellt, beispielsweise Min:Eingangsstrecken\Von-Knot-Nr. Wenn Sie den Mauszeiger auf einen Spaltenkopf stellen, erhalten Sie in einer Quickinfo zusätzliche Informationen zum angezeigten Attribut.

Sie können hier die aktuelle Attributauswahl ändern (Attribute anzeigen und ausblenden), das Layout anpassen (Listenlayout anpassen) oder eine Liste sortieren (Liste sortieren).

(4) Daten

Hier werden die Werte der ausgewählten Attribute angezeigt. Über das Kontextmenü stehen Ihnen diverse Bearbeitungsmöglichkeiten zur Verfügung.

In der ersten Spalte einer Liste wird die Zeilennummer angezeigt.
(5) Symbolleiste

In der Symbolleiste stehen Ihnen folgende Funktionen zur Verfügung:

Symbol

Name

Beschreibung

Liste als Attributsdatei speichern

Liste in Textdatei speichern (Liste als Attributsdatei speichern)

Liste in Zwischenablage kopieren

Inhalte in den Zwischenspeicher kopieren (Listeninhalte über die Zwischenablage austauschen)

Liste als Datenbank speichern

Liste in einer Datenbank speichern (Attribute in Datenbank exportieren)

Inhalt aus Zwischenablage einfügen

Inhalte aus dem Zwischenspeicher einfügen (Listeninhalte über die Zwischenablage austauschen)

Attribute auswählen

Auswahl der Spalten, die in der Liste angezeigt werden (Attribute anzeigen und ausblenden)

Nur aktive <Netzobjekte> anzeigen

Ein- oder ausschalten, dass nur aktive Objekte des ausgewählten Typs angezeigt werden (Nur aktive Netzobjekte anzeigen)

Querschnittsansicht an/aus

Ist das Symbol gedrückt, wird pro Netzobjekt nur eine Zeile angezeigt. Die Attributwerte beider Richtungen werden zusammengefasst.

Ist das Symbol nicht gedrückt, wird pro Richtung des Netzobjekts eine Zeile angezeigt. Die Attributwerte je Richtung werden separat angezeigt.

Hinweise

Die Option wird nur bei Netzobjekttypen mit Richtungsbezug angezeigt, beispielsweise für Strecken, Abbieger oder Bezirks-Beziehungen.

Wie die Attributwerte zusammengefasst werden, ist durch eine interne Logik vorgegeben. Bei nicht numerischen Attributen wird der Wert der Hinrichtung angezeigt.

Netzobjekt suchen

Markiert die Zeile mit dem gesuchten Netzobjekt (Objekt in einer Liste finden)

Aufsteigend sortieren

Liste nach einer oder mehreren Spalten aufsteigend sortieren (Liste sortieren)

Absteigend sortieren

Liste nach einer oder mehreren Spalten absteigend sortieren (Liste sortieren)

Auswertungszeile Minimum an/aus

Ist das Symbol gedrückt, zeigt eine zusätzliche Zeile im Spaltenkopf für jede Spalte den Minimalwert an.

Auswertungszeile Maximum an/aus

Ist das Symbol gedrückt, zeigt eine zusätzliche Zeile im Spaltenkopf für jede Spalte den Maximalwert an.

Auswertungszeile Durchschnitt an/aus

Ist das Symbol gedrückt, zeigt eine zusätzliche Zeile im Spaltenkopf für jede Spalte den Durchschnitt an.

Auswertungszeile Summe an/aus

Ist das Symbol gedrückt, zeigt eine zusätzliche Zeile im Spaltenkopf für jede Spalte die Summe an.

Keine Synchronisation mit anderen Fenstern

Die Liste ist nicht mit anderen Fenstern synchronisiert.

Synchronisation mit anderen Fenstern: Markieren

In der Liste markierte Objekte werden in anderen Fenstern markiert und umgekehrt.

Synchronisation mit anderen Fenstern: Markieren mit Ansicht verschieben

In der Liste markierte Objekte werden in anderen Fenstern markiert und umgekehrt. Zusätzlich wird der Netzausschnitt bei Bedarf so verschoben, dass das markierte Objekt/die markierten Objekte vollständig sichtbar sind.

Synchronisation mit anderen Fenstern: Markieren mit Autozoom

In der Liste markierte Objekte werden in anderen Fenstern markiert und umgekehrt. Zusätzlich wird im Netz ein Autozoom auf das markierte Objekt/die markierten Objekte ausgelöst.
(6) Fensterauswahl

Über die Auswahlliste können Sie zwischen allen geöffneten Fenstern wechseln.

Anzeigeeinstellungen für eine POI-Kategorie ändern

Sie können festlegen, wie die POIs einer POI-Kategorie angezeigt werden. Wenn die POI-Kategorie Unterkategorien besitzt, können Sie die Einstellungen für alle Unterkategorien übernehmen.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten: Netzeditor öffnet sich.

Wählen Sie im Navigator den Eintrag Points of Interest.

Klicken Sie im Navigator der POI-Kategorien auf die gewünschte POI-Kategorie.

Klicken Sie auf das Symbol  Bearbeiten.

Das Fenster Grafikparameter für POI-Kategorie bearbeiten öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (Punktobjekte im Netzfenster darstellen).

Bestätigen Sie mit der entsprechenden Schaltfläche.

Element

Beschreibung

Übernehmen für Kategorie

Die Einstellungen gelten nur für die gewählte POI-Kategorie.

Übernehmen für Kategorie und Unterkategorien

Die Einstellungen gelten für die gewählte POI-Kategorie und alle Unterkategorien.

Die POIs der POI-Kategorie werden gemäß den Einstellungen dargestellt.

Für einzelne POIs können Sie ein anderes Bild als Symbol festlegen (Symbol eines einzelnen POIs ändern).

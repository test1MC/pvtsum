Darstellung eines Linienbalkens festlegen

Sie können an Strecken über einen Balken den Wert eines Attributs der jeweiligen Linienroute anzeigen. Beispielsweise können Sie die Belastung der Linie darstellen.

Die Attributwerte mehrerer Linienrouten an einer Strecke können Sie entweder durch mehrere einzelne Balken anzeigen oder nach folgenden Netzobjekten aggregieren:

  • Verkehrssysteme
  • Oberlinien
  • Linien
  • Betreiber

Sie können auch die Linien selbst im Netzfenster anzeigen (Linienwege im Netzfenster darstellen).

Stellen Sie sicher, dass im Fenster Grafikparameter bearbeiten die Seite Strecken > Balken > Darstellung geöffnet ist.

Fügen Sie bei Bedarf eine neue Spalte für einen Balken ein (Einen oder mehrere Balken festlegen).

Klicken Sie auf den Spaltenkopf der Spalte mit dem gewünschten Balken.

Die in der Spalte angezeigten Darstellungseigenschaften können Sie direkt ändern. Über die Funktion Bearbeiten erreichen Sie alle möglichen Darstellungseigenschaften.

Klicken Sie auf das Symbol  Bearbeiten.

Das Fenster Balken <Nr> bearbeiten öffnet sich.

Wählen Sie die Option Diesen Balken zeichnen.

Wählen Sie im Bereich Balkentyp die Option Linienbalken.

Nehmen Sie im Register Skalierung die gewünschten Einstellungen vor (Darstellung eines Standardbalkens festlegen).

Nehmen Sie im Register Beschriftung die gewünschten Einstellungen vor (Darstellung eines Standardbalkens festlegen).

Im Bereich Allgemeine Beschriftungseinstellungen steht Ihnen als Titel die Option Netzobjektname zur Verfügung. Ist diese Option ausgewählt, wird der Name des Netzobjekttyps angezeigt, den Sie im Register Füllung als Aggregationsebene ausgewählt haben.

Der Bereich Beschriftungstyp ist ausgegraut.

Wählen Sie das Register Füllung.

Individuelle Darstellung je <Aggregationsebene>

Ist die Option ausgewählt, werden alle Balken in der gleichen Farbe und mit der gleichen Umrandung dargestellt. Im rechten Bereich legen Sie fest, wie die Balken gezeichnet werden (siehe „Linienbalken einheitlich darstellen“ auf Seite 1515).

Klassifizierte Darstellung

Ist die Option ausgewählt, können Sie Farbe und Umrandung der Balken nach einem Attribut klassifizieren. Über die Schaltfläche wählen Sie das gewünschte Attribut. Im rechten Bereich legen Sie fest, wie die Balken gezeichnet werden (Objekte anhand von Attributwerten klassifiziert darstellen).

Linienbalken einheitlich darstellen

Sie können Linienbalken pro gewähltem Objekt, beispielsweise pro Linie oder pro Verkehrssystem, einheitlich darstellen. Die Liste enthält alle Objekte und ihre Darstellungseigenschaften. Alternativ können Sie Linienbalken nach einem Attribut klassifiziert darstellen (siehe „Linienbalken klassifiziert darstellen“ auf Seite 1517).

Stellen Sie sicher, dass im Fenster Balken <Nr> bearbeiten > Register Füllung die Option Individuelle Darstellung je <Netzobjekt der gewählten Aggregationsebene> ausgewählt ist.

Ändern Sie in der rechten Liste für jedes Objekt die gewünschten Darstellungseigenschaften.

In der Spalte Darstellung werden die Balken zunächst im Standardstil für Linien- und Wegebalken angezeigt. Diesen können Sie auf der Seite Strecken > Balken > Register Linien- und Wegebalken anpassen.

Linienbalken klassifiziert darstellen

Sie können die Füllung der Linienbalken pro gewähltem Objekt, beispielsweise pro Linie oder pro Verkehrssystem, klassifiziert darstellen.

Stellen Sie sicher, dass im Fenster Balken <Nr> bearbeiten > Register Füllung die Option Klassifizierte Darstellung ausgewählt ist.

Wählen Sie über die mit Attributnamen beschriftete Schaltfläche das Attribut der ausgewählten Aggregationsebene aus, nach dem Sie klassifizieren möchten.

Ändern Sie in der rechten Liste für jedes Objekt die gewünschten Darstellungseigenschaften.

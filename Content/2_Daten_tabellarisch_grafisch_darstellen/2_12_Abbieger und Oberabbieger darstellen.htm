Abbieger und Oberabbieger darstellen

Sie können Abbieger, Abbiegeverbote, U-Turns und Hauptströme an Knoten darstellen.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Alternativ können Sie das Fenster öffnen, indem Sie gleichzeitig die Tasten Strg und G drücken.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag Abbieger.

Element

Beschreibung

Layer Abbieger zeichnen

Ist die Option ausgewählt, können Abbieger, Oberabbieger und Knotenströme im Netzfenster dargestellt werden.

Abbieger zeichnen

Ist die Option ausgewählt, können Abbieger und Oberabbieger im Netzfenster dargestellt werden.

Knotenströme zeichnen

Ist die Option ausgewählt, können Knotenströme im Netzfenster dargestellt werden (Grundeinstellungen für die Darstellung der Knotenströme vornehmen).

Balken zeichnen

Ist die Option ausgewählt, können die Balken der Knotenströme im Netzfenster dargestellt werden (Grundeinstellungen für die Darstellung der Knotenströme vornehmen).

Balkenbeschriftungen zeichnen

Ist die Option ausgewählt, können die Balkenbeschriftungen der Knotenströme im Netzfenster dargestellt werden (Grundeinstellungen für die Darstellung der Knotenströme vornehmen).

Der oberste Layer wird zuletzt gezeichnet

Die Liste zeigt die Reihenfolge, in der Abbieger und Knotenströme gezeichnet werden. Über die Symbole und können Sie die Reihenfolge der gezeichneten Layer ändern.

Abbieger/Abbiegeverbote/ U-Turns/Hauptströme

Hier können Sie je Objekt festlegen, ob und wie es im Netz angezeigt wird.

Farbe/Strichtyp

Sie können je Objekt in einem separaten Fenster festlegen, in welcher Farbe und in welchem Strichtyp es gezeichnet wird, indem Sie in das Feld mit dem Strich klicken.

Zeichnen

Ist die Option ausgewählt, wird das jeweilige Objekt im Netz angezeigt.

Pfeilspitze mm

Größe der Pfeilspitze in mm

Verkehrssysteme

Sie können entweder ein Verkehrssystem auswählen oder Alle, wenn Sie alle Abbieger anzeigen möchten, die für alle IV- und ÖV-Verkehrssysteme zugelassen sind.

Hinweis

Für Hauptströme kann kein Verkehrssystem ausgewählt werden.

Das Verkehrssystem ÖVFuß wird nicht dargestellt.

Die Objekte werden gemäß den Einstellungen dargestellt.

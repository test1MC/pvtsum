Größen aller Klassen ändern

Sie können die Größe aller Symbole, Bilder oder Texte eines klassifizierten Punktobjekts oder die Linienbreite eines klassifizierten Linienobjekts folgendermaßen ändern.

Größen der Symbole, Bilder oder Texte eines Punktobjekts ändern

Nachfolgend wird am Beispiel des Netzobjekts Bezirke beschrieben, wie Sie die Größe der Symbole, Bilder oder Texte aller Klassen ändern können.

Wählen Sie im Navigator den Eintrag Bezirke > Darstellung.

Klicken Sie auf die Schaltfläche Größe für alle Klassen ändern.

Das Fenster Klassifizierung bearbeiten: Größen für alle Klassen ändern öffnet sich.

Nehmen Sie im Bereich Klassengrenzen die gewünschten Änderungen vor (Klassen und Klassengrenzen für eine Klassifizierung einfügen).

Klicken Sie auf eines der folgenden Symbole:

Element

Beschreibung

 Konstante Größe

Über ein Eingabefeld können Sie einen Wert für die Größe aller angezeigten Objekte angeben.

 Größenverlauf

Über zwei Eingabefelder können Sie eine Größe für die oberste Klasse und eine Größe für die unterste Klasse eingeben. Die Größe ändert sich für die dazwischenliegenden Klassen kontinuierlich.

 Zweiteiliger Verlauf

Über drei Eingabefelder können Sie eine Größe für die oberste Klasse, eine Größe für die mittlere Klasse und eine Größe für die unterste Klasse eingeben. Die Größe ändert sich für die dazwischenliegenden Klassen kontinuierlich.

Die Einstellungen werden übernommen und in der Tabelle angezeigt.

Breiten eines klassifizierten Linienobjekts ändern

Nachfolgend wird am Beispiel des Netzobjekts Strecken beschrieben, wie Sie die Linienbreiten eines Linienobjekts ändern können.

Wählen Sie im Navigator den Eintrag Strecken > Darstellung.

Klicken Sie auf die Schaltfläche Größe für alle Klassen ändern.

Das Fenster Klassifizierung bearbeiten: Größe ändern öffnet sich.

Nehmen Sie im Bereich Klassengrenzen die gewünschten Änderungen vor (Klassen und Klassengrenzen für eine Klassifizierung einfügen).

Druckbereich festlegen

Sie können die Druckbereichsgrenzen in der Bildschirmdarstellung anzeigen.

Zeigen Sie im Menü Datei auf Drucken.

Wählen Sie den Eintrag Druckbereich.

Das Fenster Druckbereich öffnet sich und der Druckbereich wird in der Netzdarstellung durch ein Rechteck angezeigt.

Sobald das Fenster Druckbereich geöffnet ist, können Sie das angezeigte Rechteck in der Netzdarstellung mit der Maus vergrößern oder verkleinern.

Alternativ können Sie mit der rechten Maustaste ein neues Rechteck aufziehen und so einen neuen Druckbereich festlegen.

Element

Beschreibung

Auswahlliste

Hier können Sie zwischen folgenden Optionen wählen.

  • Maßstab halten

Ist diese Option ausgewählt, verändert sich der Maßstab nicht, wenn der Druckbereich geändert wird.

Hinweis

Wenn Sie den Druckbereich mit der Maus vergrößern, kann die verfügbare Papiergröße überschritten werden. Ist dies der Fall, wird dies durch rote Werte im Bereich Größe angezeigt.

  • Größe halten

Ist diese Option ausgewählt, bleiben die Höhe und die Breite des bedruckbaren Bereichs gleich, wenn sie den Druckbereich mit der Maus ändern. Der Maßstab wird entsprechend angepasst.

Größe

X,Y

Breite und Höhe des Druckbereichs in mm

Hinweis

Sind die Werte rot, überschreitet der Druckbereich die verfügbare Papiergröße.

Auf Seitengröße

Sie können den Druckbereich auf das bei der Druckereinrichtung angegebene Seitenformat vergrößern, indem Sie auf diese Schaltfläche klicken.

Maßstab

Angabe des Maßstabs

Hinweis

Ist die Option Größe halten ausgewählt, wird ein anderer Teil des Netzes im Druckbereich angezeigt, wenn Sie den Maßstab ändern. Ist die Option Maßstab halten ausgewählt, verändern sich die Größenwerte für Breite und Höhe, wenn Sie den Maßstab ändern.

Ausschnitt

Hier können Sie den Druckbereich über die Eckkoordinaten festlegen.

Hinweis

Sie können den Druckbereich auch verschieben, indem Sie auf die Pfeile klicken. Der Druckbereich wird dann um die eigene Größe in die jeweilige Pfeilrichtung verschoben.

Druckbereich

Aus Fenster

Über die Schaltfläche können Sie den Druckbereich auf die Größe des aktuellen -Fensters setzen.

Auf maximalen Maßstab

Über die Schaltfläche können Sie den Maßstab für den Netzausschnitt im festgelegten Druckbereich so maximieren, dass der Druckbereich auf den druckbaren Bereich der Druckseite passt.

Autozoom

Über die Schaltfläche können Sie in den Druckbereich zoomen. Der Druckbereich wird vollständig am Bildschirm angezeigt.

Die Einstellungen des Druckbereichs werden übernommen.

Sie können den Druckbereich als gestricheltes Rechteck auf dem Bildschirm anzeigen, wenn Sie die entsprechende Option im Menü Grafik > Grafikparameter bearbeiten > Seite Druckrahmen auswählen. Der Druckbereich wird jedoch nur dargestellt, wenn im Fenster Druckseite einrichten die Option Druckbereich ausgewählt ist (Druckseite einrichten).

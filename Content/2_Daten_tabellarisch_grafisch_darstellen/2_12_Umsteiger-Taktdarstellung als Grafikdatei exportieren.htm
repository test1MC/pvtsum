Umsteiger-Taktdarstellung als Grafikdatei exportieren

Sie können die Umsteiger-Taktdarstellung direkt in einer Grafikdatei *.svg oder *.dxf speichern.

Stellen Sie sicher, dass das Fenster Umsteiger-Taktdarstellung geöffnet ist (Umsteiger-Taktdarstellung anzeigen).

Wählen Sie im Menü Umsteiger-Taktdarstellung einen der Einträge SVGExport oder DXF Export.

Das Fenster SVG-Datei speichern oder DXF-Datei speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster SVG-Export oder DXF-Export öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (SVG-Export und DXF Export).

Die Umsteiger-Taktdarstellung wird in einer Datei *.svg oder *.dxf gespeichert.

Sie können die Umsteiger-Taktdarstellung auch als Grafikdatei exportieren, indem Sie im Menü Umsteiger-Taktdarstellung den Eintrag JPG Export (Screenshot) wählen oder im Fenster Umsteiger-Taktdarstellung in der Symbolleiste auf das Symbol  JPG Export (Screenshot) klicken (Screenshot vom Bildschirm exportieren).

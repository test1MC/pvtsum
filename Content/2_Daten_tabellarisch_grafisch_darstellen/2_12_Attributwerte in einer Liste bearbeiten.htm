Attributwerte in einer Liste bearbeiten

Sie können in einer Liste Attributwerte bearbeiten. Je nach Typ des Attributs stehen Ihnen ein Eingabefeld, eine Auswahlliste oder eine Schaltfläche zur Verfügung. Pflichteingabeattribute und Ausgabeattribute können Sie nicht ändern. Sie sind grau hinterlegt dargestellt.

Alternativ können Sie Attributwerte eines Objekts folgendermaßen bearbeiten:

Sie können in der Liste mit der rechten Maustaste auf das gewünschte Objekt klicken und den Eintrag Bearbeiten wählen. Dadurch öffnet sich ein Fenster, in dem Sie die Attributwerte des Objekts bearbeiten können. Beispielsweise öffnet sich bei einem Knoten der Knoteneditor.
Attributwerte in einer Liste eingeben

Sie können in einer Liste numerische Werte oder Texte bearbeiten.

Wählen Sie die gewünschten Zellen aus (Listenbereiche auswählen).

Bei der Eingabe von numerischen Attributwerten mit Längen- oder Zeitbezug müssen Sie die Einheiten berücksichtigen:

  • Wenn Sie nur die Zahl eingeben, wird der Wert in dem Format interpretiert, mit dem das Attribut angezeigt wird. Beispiel: Wenn ein Attribut im Format Minuten angezeigt wird, wird der Wert 5 als 5 Minuten interpretiert.

Das Format können Sie im Attributauswahlfenster ändern (Fenster zur Auswahl mehrerer Attribute).

  • Sie können eine Zahl mit Einheit eingeben. In diesem Fall rechnet den Wert so um, dass er entsprechend dem gewählten Format angezeigt wird. Sie können Längenangaben auch wahlweise mit metrischen (m, km) oder imperialen (ft, mi) Einheiten eingeben.

Beispiel: Wenn ein Attribut im Format Minuten angezeigt wird, können Sie 2h eingeben. Nach der Eingabe enthält das Feld den Wert 120.

Drücken Sie die Taste Enter.

Die Werte werden geändert.

Sie können numerische Werte auch über das Kontextmenü Rechenoperationen auf markiertem Bereich > Wert zuweisen ändern.
Attributwerte in einer Liste über eine Auswahlliste ändern

Sie können in einer Liste bestimmte Attributwerte über Auswahllisten bearbeiten. Diese Funktion steht Ihnen nur zur Verfügung, wenn unter dem Menü Bearbeiten > Benutzereinstellungen > Eintrag Benutzeroberfläche > Listen die Option Erweiterte Eingabemöglichkeiten aktiv ist (Optionen von Listen einstellen).

Wählen Sie die gewünschten Zellen aus (Listenbereiche auswählen).

Klicken Sie auf das Symbol mit dem Pfeil .

Wählen Sie den gewünschten Eintrag aus.

Die Werte werden geändert.

Sie können die Werte auch über das Kontextmenü ändern. Die zur Auswahl stehenden Werte stehen im Kontextmenü ganz unten.
Attributwerte in einer Liste über eine Schaltfläche ändern

Sie können in einer Liste bestimmte Attributwerte über Schaltflächen bearbeiten. Diese Funktion steht Ihnen nur zur Verfügung, wenn unter dem Menü Bearbeiten > Benutzereinstellungen > Eintrag Benutzeroberfläche > Listen die Option Erweiterte Eingabemöglichkeiten aktiv ist (Optionen von Listen einstellen).

Klicken Sie auf die gewünschte Zelle (Listenbereiche auswählen).

Wählen Sie die gewünschten Einträge aus.

Die Werte werden geändert.

Sie können die Werte auch über das Kontextmenü ändern. Die zur Auswahl stehenden Werte stehen im Kontextmenü ganz unten.

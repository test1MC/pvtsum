Einen oder mehrere Balken festlegen

Sie können für verschiedene Netzobjekttypen Balken festlegen.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten: Netzeditor öffnet sich.

Nehmen Sie auf der Seite Strecken > Balken die gewünschten Einstellungen vor.

Wählen Sie im Navigator den Eintrag Strecken > Balken > Darstellung.

Im oberen Bereich sehen Sie eine Vorschau der aktuell eingestellten Balken. Die Tabelle darunter enthält die Beschreibung der einzelnen Balken. Jede Spalte beschreibt einen Balken. Die Balken werden in der Reihenfolge der Spalten gezeichnet: Die erste Spalte beschreibt den innersten Balken, die letzte den äußersten. Der aktuell markierte Balken wird in der Vorschau durch zwei Pfeile gekennzeichnet, die zugehörige Beschriftung ist farbig hinterlegt. Die Pfeile sind für aktuell angezeigte Balken gefüllt, ansonsten leer.

Markieren Sie die gewünschte Spalte, indem Sie auf den Spaltenkopf klicken.

Nehmen Sie über die Symbolleiste die gewünschten Einstellungen vor.

Öffnet das Fenster Attributauswahl, in dem Sie festlegen können, welche Attribute als Zeilen in der Tabelle angezeigt werden (Fenster zur Auswahl mehrerer Attribute).

Tipp

Mit Rechtsklick auf einen Zeilennamen können Sie über das Kontextmenü einfach einzelne Zeilen entfernen oder hinzufügen.

Setzt die in der Tabelle als Zeilen angezeigten Attribute auf die Standardauswahl zurück.

Die Funktionen erreichen Sie auch über das Kontextmenü. Hier können Sie auch die Spaltenbreiten verändern.

Die Balken werden entsprechend der aktuellen Einstellungen im Netzfenster angezeigt.

Sie haben festgelegt, wie viele Balken für den Netzobjekttyp dargestellt werden. Die Inhalte in den Spalten der Tabelle zeigen Darstellungseigenschaften der Balken. Der ausgewählte Balkentyp legt fest, welche Eigenschaften Sie direkt in der Tabelle bearbeiten können. Alle Darstellungseigenschaften eines Balkens erreichen Sie über das Symbol  (Darstellungseigenschaften eines Balkens festlegen).

Liste als PDF speichern

Als PDF gespeicherte Listen können Sie komfortabel in Textdokumente einbinden.

Sie können diverse Layouteinstellungen vornehmen, bevor Sie eine Liste als PDF speichern (Layout für Listendruck festlegen).

Klicken Sie in der Symbolleiste auf das Symbol  Drucken.

Das Fenster Drucken: Liste öffnet sich.

Wählen Sie unter Name die PDF-Ausgabemöglichkeit aus.

Das Fenster PDF-Datei speichern unter öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster Druckinformation öffnet sich. Darin wird die Anzahl der Seiten und der Zeilen ausgegeben, die gedruckt werden.

Das PDF wird erstellt und die Datei geöffnet.

Linienwege im Netzfenster darstellen

Sie können festlegen, wie Linienrouten bzw. Linien im Netzfenster dargestellt werden. Sie können beispielsweise jede Linie in einer anderen Farbe darstellen. Für Strecken, auf denen mehrere Linien verkehren, definieren Sie eine spezielle Farbe.

Sie können entlang von Linien zusätzlich Linienbalken darstellen (Darstellung eines Linienbalkens festlegen).

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag Linienweg.

Element

Beschreibung

Layer Linienweg zeichnen

Ist die Option ausgewählt, können Linienwege dargestellt werden.

Auswahlliste

ÖV-Linienwege können auf vier Ebenen dargestellt werden. Hier können Sie die gewünschte Detaillierungsebene auswählen.

  • Verkehrssysteme
  • Oberlinien
  • Linien
  • Linienrouten

Gemeinsam benutzte Strecken zeichnen

Ist die Option ausgewählt, werden Strecken gezeichnet, die von mehreren Linienrouten, Linien, Oberlinien oder Verkehrssystemen genutzt werden.

Ist die Option nicht ausgewählt, werden Linien auf gemeinsam genutzten Strecken nicht dargestellt.

Strichdarstellung gemeinsam benutzter Strecken

Hier können Sie die Darstellung gemeinsam genutzter Strecken einstellen. Sie können die Farbe und den Strichtyp in einem separaten Fenster auswählen, indem Sie in das Farbfeld klicken (Strichdarstellung einstellen).

Zeichnen

Ist die Option ausgewählt, wird der jeweilige Linienweg gezeichnet.

Hinweis

Über die Schaltfläche Bearbeiten können Sie gleichzeitig Einstellungen für alle Objekte vornehmen.

Name

Liste der vorhandenen Objekte der Detaillierungsebene

Darstellung

Sie können die Farbe für jeden Strich einzeln in einem separaten Fenster auswählen, indem Sie in das Farbfeld klicken (Strichdarstellung einstellen).

Hinweis

Über die Schaltfläche Bearbeiten können Sie gleichzeitig Einstellungen für alle Objekte vornehmen.

Klicken Sie bei Bedarf auf die Schaltfläche Bearbeiten.

Über die Schaltfläche können Sie gleichzeitig Einstellungen für alle Objekte vornehmen.

Das Fenster Darstellung <Netzobjekttyp> bearbeiten öffnet sich.

Element

Beschreibung

Sichtbarkeit

Hier legen Sie fest, welche Optionen Sie in der Liste der Objekte in der Spalte Zeichnen auswählen möchten.

Beibehalten

Ist die Option ausgewählt, bleiben die Einstellungen für die einzelnen Objekte erhalten, wenn Sie die Strichdarstellung ändern.

Alle

Ist die Option ausgewählt, legen Sie über die Option zeichnen fest, ob Sie alle Optionen in der Spalte Zeichnen auswählen oder deaktivieren möchten.

Aktive exklusiv an

Ist die Option ausgewählt, legen Sie fest, dass die Optionen in der Spalte Zeichnen für alle aktiven Objekte ausgewählt und für alle passiven deaktiviert werden.

Hinweis

Die Option steht nur für die Netzobjekte zur Verfügung, die passiv sein können.

Die Einstellung der passiven Objekte ändert sich dadurch nicht.

Strichdarstellung

Hier legen Sie fest, welche Farben Sie in der Liste der Objekte in der Spalte Darstellung einstellen möchten.

Beibehalten

Ist die Option ausgewählt, bleiben die Einstellungen für die einzelnen Objekte erhalten, wenn Sie im Bereich Zeichnen die Einstellungen ändern.

Konstant

Ist die Option ausgewählt, wird die ausgewählte Farbe für alle Objekte der Detaillierungsebene übernommen. Wählen Sie die Farbe, indem Sie in das Farbfeld klicken (Eigenschaften eines geschlossenen Polygons bearbeiten).

Qualitative Farbpalette

Ist die Option ausgewählt, erhält jedes Objekt eine andere Farbe.

Die Linienwege werden gemäß den Einstellungen dargestellt.

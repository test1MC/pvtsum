Formelmatrix bearbeiten

Sie können die Werte einer Formelmatrix nicht direkt bearbeiten. Sie können jedoch die Formel ändern.

Formel einer Formelmatrix ändern

Klicken Sie im Fenster Matrizen mit der rechten Maustaste auf die Formelmatrix.

Wählen Sie den Eintrag Formel bearbeiten.

Das Fenster Formel Matrix <Nummer> bearbeiten öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (Aus Formel berechnete Matrix erstellen).

Die Formel wird geändert und die Matrixwerte werden neu berechnet.

Formelmatrix in Datenmatrix umwandeln

Klicken Sie im Fenster Matrizen mit der rechten Maustaste auf die Formelmatrix.

Wählen Sie den Eintrag In Datenmatrix konvertieren.

Die Formelmatrix wird in eine Datenmatrix umgewandelt. Die Formeln werden in Werte umgewandelt.

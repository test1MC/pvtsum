Matrizen einseitig gekoppelt hochrechnen

Sie können Matrizen einseitig gekoppelt hochrechnen. Dabei werden entweder jede Zeile oder jede Spalte der Matrizen mit einem Faktor multipliziert.

Stellen Sie sicher, dass ein Matrixeditorfenster mit der gewünschten Matrix geöffnet ist (Matrizen öffnen).

Klicken Sie in der Symbolleiste des Matrixfensters auf das Symbol  Hochrechnen.

Das Fenster Hochrechnen öffnet sich.

Wenn Sie die Einstellungen für die Hochrechnung in einer Datei gespeichert haben (*.xml, *.cod), können Sie sie über die Schaltfläche Öffnen laden.

Wählen Sie in der Auswahlliste Hochrechnen wird durchgeführt einen der Einträge einseitig gekoppelt bzgl. ....

Element

Beschreibung

einseitig gekoppelt bzgl. Quellaufkommen

Die Matrixwerte werden zeilenweise hochgerechnet.

einseitig gekoppelt bzgl. Zielaufkommen

Die Matrixwerte werden spaltenweise hochgerechnet.

Nehmen Sie im Bereich Allgemeine Parameter die gewünschten Einstellungen vor.

Element

Beschreibung

mit Faktor multiplizieren

Ist die Option ausgewählt, wird jede Zeile bzw. Spalte mit einem Faktor multipliziert.

Hochrechnen auf Sollwert(e)

Ist die Option ausgewählt, werden die Matrixwerte jeder Zeile bzw. Spalte so hochgerechnet, dass die Zeilen- bzw. Spaltensummen den vorgegebenen Sollwerten entsprechen.

Nehmen Sie im Bereich Faktoren verwenden bzw. Sollwerte verwenden die gewünschten Einstellungen vor.

Element

Beschreibung

aus Bezirksattributen

Hinweis

Die Option ist nur für -Matrizen aktiv.

Ist die Option ausgewählt, verwenden Sie für die Hochrechnung Faktoren bzw. Sollwerte, die in einem Bezirksattribut stehen.

Faktor für Zeilen aus Attribut bzw. Faktor für Spalten aus Attribut

Sollwert für Zeilen aus Attribut bzw. Sollwert für Spalten aus Attribut

Über die Schaltfläche wählen Sie das Bezirksattribut mit den Faktoren bzw. Sollwerten für die Hochrechnung.

Hinweis

Nur diejenige Option ist aktiv, die zu den Einstellungen im Bereich Allgemeine Parameter passt.

Kennzeichnung eines Bezirks als „extern“ aus Attribut

Wenn Sie mit Sollwerten hochrechnen, können Sie über dieses Bezirksattribut bestimmte Quelle-Ziel-Beziehungen von der Hochrechnung ausschließen. Die Matrixwerte aller Quelle-Ziel-Beziehungen zwischen zwei als extern gekennzeichneten Bezirken werden auf 0 gesetzt. Die Sollwerte werden auf die restlichen Matrixwerte verteilt.

aus Parametern

Ist die Option ausgewählt, verwenden Sie für die Hochrechnung die in der Tabelle angegebenen Faktoren bzw. Sollwerte.

  • ErzeugungWert bzw. AnziehungWert: Geben Sie hier die gewünschten Faktoren bzw. Sollwerte pro Zeile bzw. Spalte ein.

Hinweis

Wenn Sie für mehrere Zeilen bzw. Spalten den gleichen Faktor benötigen, ist es eventuell sinnvoll, das gebietsweise Hochrechnen zu verwenden (Gebietsweise hochrechnen – einseitig gekoppelt).

  • Extern: Wenn Sie mit Sollwerten hochrechnen, können Sie über diese Kennzeichnung bestimmte Quelle-Ziel-Beziehungen von der Hochrechnung ausschließen. Die Matrixwerte aller Quelle-Ziel-Beziehungen zwischen zwei als extern gekennzeichneten Bezirken werden auf 0 gesetzt. Die Sollwerte werden auf die restlichen Matrixwerte verteilt.

Tipp

Alternativ können Sie die Werte in der Tabelle über die Schaltfläche Öffnen aus einer entsprechenden Parameterdatei einlesen.

Die Matrixwerte werden hochgerechnet.

Langbezeichnung der Aktivität

Heimataktivität

Ist die Option ausgewählt, ist die Aktivität als Heimatstandort der Verkehrsteilnehmer definiert.

Hinweis

Sie können nur eine Aktivität als Heimataktivität definieren.

Strukturgrößen

Über die Schaltfläche können Sie die Größen, deren Werte als Quell- und Zielpotenzial in die Verkehrserzeugung einfließen, für die jeweilige Aktivität in einem separaten Fenster auswählen (Grundlagen: EVA-Datenmodell).

Nachfragemodelle des Typs VISEM

Element

Beschreibung

Code

Kurzbezeichnung der Aktivität

Name

Langbezeichnung der Aktivität

Heimataktivität

Ist die Option ausgewählt, ist die Aktivität als Heimatstandort der Verkehrsteilnehmer definiert.

Hinweis

Sie können nur eine Aktivität als Heimataktivität definieren.

Strukturgröße

Über die Schaltfläche können Sie die Größe, deren Wert als Zielpotenzial in die Verkehrsverteilung einfließt, für die jeweilige Aktivität in einem separaten Fenster auswählen (Grundlagen: VISEM-Datenmodell).

Zielseitige Kopplung

Ist die Option ausgewählt, wird bei der VISEM-Verkehrsverteilung eine Zielkopplung für die jeweilige Aktivität berücksichtigt (Grundlagen: VISEM-Datenmodell).

Hinweis

Für die Heimataktivität wird immer eine Zielkopplung berücksichtigt.

RSF ZMin

Faktor für die untere Grenze des Zielaufkommens (Grundlagen: VISEM-Datenmodell)

Hinweise

Die Option ist nur verfügbar, wenn die Option Zielseitige Kopplung für die Aktivität ausgewählt ist.

Für die Heimataktivität beträgt der Faktor immer 1.0.

RSF ZMax

Faktor für die obere Grenze des Zielaufkommens (Grundlagen: VISEM-Datenmodell)

Hinweise

Die Option ist nur verfügbar, wenn die Option Zielseitige Kopplung für die Aktivität ausgewählt ist.

Für die Heimataktivität beträgt der Faktor immer 1.0.

Die Faktoren RSF ZMin und RSF ZMax definieren eine Randsummenbedingung für die Zielseite. Die Randsummenbedingung auf Quellseite ist immer hart. Je nach Wahl der Randsummenfaktoren auf Zielseite wird dann die (beidseitig gekoppelte) Verteilung für jeden Aktivitätenübergang berechnet (Grundlagen: Kombinierte VISEM-Verkehrsverteilung/Moduswahl).

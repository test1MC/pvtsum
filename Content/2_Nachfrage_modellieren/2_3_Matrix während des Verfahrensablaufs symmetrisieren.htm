Matrix während des Verfahrensablaufs symmetrisieren

Klicken Sie in der Symbolleiste auf das Symbol Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Matrixoperation aus der Kategorie Matrizen ein (siehe „Verfahrensablauf festlegen und starten“ auf Seite 1).

Klicken Sie in der Zeile des neu eingefügten Verfahrens auf die Schaltfläche in der Spalte Bezugsobjekt(e).

Das Fenster Matrix auswählen öffnet sich.

Wählen Sie in der Liste die Matrix, für die Sie den Mittelwert des unteren und oberen Dreiecks berechnen möchten. Dafür stehen Ihnen zwei Methoden zur Verfügung (siehe „Fenster Matrizen auswählen“ auf Seite 1).

Element Beschreibung

Matrix auswählen nach Nummer

Wählen Sie aus der Liste aller Matrizen anhand der Nummer die gewünschte Matrix aus.

Matrix auswählen nach Eigenschaften

Wählen Sie anhand bestimmter Attribute und deren Werte die Matrix aus.

Die ausgewählte Matrix wird in die Spalte Bezugsobjekt(e) übernommen.

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist, und klicken Sie im Bereich Aktionen auf die Schaltfläche Bearbeiten.

Das Fenster Parameter Matrix-Operationen öffnet sich.

Wählen Sie die Option Symmetrisieren.

Führen Sie das Verfahren aus (siehe „Verfahren im Verfahrensablauf ausführen“ auf Seite 1).

Die Mittelwerte der Matrixwerte werden berechnet und in die Matrix geschrieben.

Die Berechnung des Mittelwerts können Sie auch bei der Kombination von Matrizen und Vektoren als Transformationsfunktion auswählen und ausführen (siehe „Matrizen und Attributsvektoren im Matrixfenster kombinieren“ auf Seite 1).

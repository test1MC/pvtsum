Oberbezirksmatrix aus Bezirksmatrix berechnen

Sie können aus einer Bezirksmatrix eine Oberbezirksmatrix berechnen. Dabei addieren Sie die Matrixwerte von Bezirken, die zum gleichen Oberbezirk gehören, und schreiben diese in eine vorhandene Oberbezirksmatrix (siehe Grundlagen: Kap. 0.0.1 , Seite 1).

Klicken Sie in der Symbolleiste auf das Symbol  Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Matrixaggregation aus der Kategorie Matrizen ein (siehe „Verfahrensablauf festlegen und starten“ auf Seite 1).

Klicken Sie in der Zeile des neu eingefügten Verfahrens auf die Schaltfläche in der Spalte Bezugsobjekt(e).

Das Fenster Matrix auswählen öffnet sich.

Wählen Sie in der Liste die Oberbezirksmatrix, in die Sie die aggregierten Werte schreiben möchten. Dafür stehen Ihnen zwei Methoden zur Verfügung (siehe „Fenster Matrizen auswählen“ auf Seite 1).

Element Beschreibung

Matrix auswählen nach Nummer

Wählen Sie aus der Liste aller Matrizen anhand der Nummer die gewünschte Matrix aus.

Matrix auswählen nach Eigenschaften

Wählen Sie anhand bestimmter Attribute und deren Werte die Matrix aus.

Die ausgewählte Matrix wird in die Spalte Bezugsobjekt(e) übernommen.

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist, und klicken Sie im Bereich Aktionen auf die Schaltfläche Bearbeiten.

Das Fenster Matrix auswählen öffnet sich.

Wählen Sie in der Liste die Bezirksmatrix, die Sie aggregieren möchten. Dafür stehen Ihnen zwei Methoden zur Verfügung.

Element Beschreibung

Matrix auswählen nach Nummer

Wählen Sie aus der Liste aller Matrizen anhand der Nummer die gewünschte Matrix aus.

Matrix auswählen nach Eigenschaften

Wählen Sie anhand bestimmter Attribute und deren Werte die Matrix aus.

Führen Sie das Verfahren aus (siehe „Verfahren im Verfahrensablauf ausführen“ auf Seite 1).

Die Werte der Bezirksmatrix werden addiert und das Ergebnis wird in die ausgewählte Oberbezirksmatrix geschrieben.

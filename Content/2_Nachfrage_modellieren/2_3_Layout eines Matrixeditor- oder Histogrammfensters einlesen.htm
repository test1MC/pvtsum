Layout eines Matrixeditor- oder Histogrammfensters einlesen

Stellen Sie sicher, dass ein Matrixeditor- oder Histogrammfenster mit den gewünschten Matrizen geöffnet ist (Matrizen öffnen und Matrizen als Histogramm anzeigen).

Wählen Sie im Menü Matrixeditor oder Matrix-Histogramm den Eintrag Layout öffnen.

Das Fenster Matrixeditor-Layout öffnen öffnet sich.

Wählen Sie die gewünschte Datei.

Sie können in ein Matrixeditorfenster nur eine Layoutdatei einlesen, die von einem Matrixeditorfenster aus gespeichert wurde, und in ein Histogrammfenster nur eine Layoutdatei, die von einem Histogrammfenster aus gespeichert wurde.

Alternativ können Sie Pfad und Dateiname manuell eingeben.

Klicken Sie auf die Schaltfläche Öffnen.

Das Fenster öffnet sich.

Element

Beschreibung

Ja

Neben dem Layout wird auch die Information über die angezeigten Matrizen aus der Layoutdatei eingelesen.

Nein

Aus der Layoutdatei wird nur das Layout eingelesen. Welche Matrizen angezeigt werden ändert sich durch das Einlesen nicht.

Das Layout wird eingelesen.

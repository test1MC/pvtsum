Matrixwerte in -Matrizen einlesen

Klicken Sie im Fenster Matrizen im Navigator mit der rechten Maustaste auf die gewünschte -Matrix.

Wählen Sie den Eintrag Öffnen von Datei.

Das Fenster Matrix öffnen öffnet sich.

Falls der Matrix ein Dateiname zugewiesen ist, wird dieser vorgeschlagen.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Die Matrixwerte werden überschrieben. Der Name der gewählten Datei wird mit der Matrix gespeichert.

Alternativ können Sie Werte aus einer Datei in eine -Matrix einlesen, indem Sie die -Matrix öffnen und in der Symbolleiste des Matrixfensters auf das Symbol  Matrixinhalt von Datei laden klicken oder das Menü Matrixeditor > Öffnen von Datei wählen. Einen entsprechenden Eintrag im Kontextmenü finden Sie auch in der Liste Matrizen (Liste öffnen).

Die Anzahl der Bezirke und die Bezirksnummern in der eingelesenen Matrixdatei müssen nicht mit denen der gewählten -Matrix identisch sein. Die Rechenoperation wird aber nur mit Matrixwerten durchgeführt, deren Quelle-Ziel-Beziehungen übereinstimmen. Falls die Matrizen nicht die gleichen Quelle-Ziel-Beziehungen besitzen, gilt:

  • Falls in der Matrixdatei eine Quelle-Ziel-Beziehung nicht vorhanden ist, wird für diese der Matrixwert null verwendet.
  • Quelle-Ziel-Beziehungen, die nicht in der -Datei vorhanden sind, werden ignoriert.

Nachfragematrizen für die Standard-4-Stufen-Verkehrsverteilung erzeugen

Sind für Nachfrageschichten noch keine Nachfragematrizen ausgewählt, in die das Ergebnis der Verkehrsverteilung pro Nachfrageschicht gespeichert wird (Attribute und Optionen von Nachfrageschichten), können Sie neue Nachfragematrizen erzeugen und den Nachfrageschichten für die Verkehrsverteilung automatisch zuordnen.

Wählen Sie im Menü Nachfrage den Eintrag Nachfragemodelle.

Das Fenster Nachfragemodelle öffnet sich.

Wählen Sie das Register Basis.

Wählen Sie das gewünschte Nachfragemodell.

Klicken Sie auf die Schaltfläche Verfahrensablauf erzeugen.

Das Fenster Verfahrensablauf erzeugen öffnet sich.

Klicken Sie auf die Schaltfläche Nachfragematrizen.

Das Fenster Optionen zur Erzeugung und Zuordnung von Nachfragematrizen öffnet sich.

Element

Beschreibung

Verteilungsmatrizen erzeugen

erzeugt Verteilungsmatrizen

Moduswahlmatrizen erzeugen

eine Matrix je Modus

erzeugt eine Matrix je Modus des Nachfragemodells

separate Matrizen für Nachfrageschichten

erzeugt pro Nachfrageschicht und Modus eine eigene Matrix

Unten im Fenster erhalten Sie eine Information darüber, wie viele Matrizen aufgrund Ihrer Einstellungen erzeugt werden.

Für alle Objekte, denen noch keine Nachfragematrix zugeordnet ist, werden neue Nachfragematrizen erzeugt und zugeordnet.

VISEM-Nachfragemodelle aus VISEM-Modelldateien (*.vhd) einlesen

Im Nachfrageberechnungsprogramm VISEM 8.x sind die Modelldaten in VISEM-Eingabedatendateien *.vhd enthalten. In können Sie VISEM-Nachfragemodelle, alle zugehörigen Nachfrageobjekte und die Verfahrensparameter zum Verfahren VISEM-Verkehrserzeugung aus einer VISEM-Eingabedatendatei *.vhd einlesen. VISEM-Eingabematrizen werden als Kenngrößenmatrizen eingelesen.

Sie können eine VISEM-Eingabedatendatei *.vhd in nur in ein bestehendes Netz einlesen.

Wählen Sie im Menü Nachfrage den Eintrag Nachfragemodelle.

Das Fenster Nachfragemodelle öffnet sich.

Wählen Sie das Register Basis.

Klicken Sie auf die Schaltfläche Modellvorlage laden.

Das Fenster Modellvorlage öffnen öffnet sich.

Wählen Sie die gewünschte Datei *.vhd aus.

Klicken Sie auf die Schaltfläche Öffnen.

Eine Abfrage öffnet sich.

Ein Nachfragemodell vom Typ VISEM und die zugehörigen Nachfrageobjekte werden eingefügt. Alle Modi im bestehenden Netz werden als relevante Modi ausgewählt. Für jede VISEM-Eingabematrix wird eine neue Kenngrößenmatrix eingefügt. Der Schritt VISEM-Verkehrserzeugung wird automatisch am Ende des Verfahrensablaufs eingefügt und die Parameter werden automatisch eingestellt.

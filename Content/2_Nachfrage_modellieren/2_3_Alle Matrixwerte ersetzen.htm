Alle Matrixwerte ersetzen

Stellen Sie sicher, dass ein Matrixeditorfenster mit den gewünschten Matrizen geöffnet ist (Matrizen öffnen).

Klicken Sie auf das Symbol  Wert(e) zuweisen.

Das Fenster Operand für ‚Wert(e) zuweisen...‘ auf Gesamtmatrix auswählen öffnet sich.

Element

Beschreibung

Matrix auswählen

Ist die Option ausgewählt, können Sie die Matrixwerte durch die Werte einer Matrix ersetzen, die im Navigator enthalten ist.

Tipp

Der Navigator enthält die gleichen Matrizen wie das Fenster Matrizen (Matrizen im Fenster Matrizen anzeigen).

Bezirksattribut

Ist die Option ausgewählt, ersetzen Sie die Matrixwerte durch Werte eines Bezirksattributs.

Quelle

Ist der Eintrag ausgewählt, wird der Matrixwert jeder Quelle-Ziel-Beziehung ij durch den Attributwert des Quellbezirks i ersetzt.

Ziel

Ist der Eintrag ausgewählt, wird der Matrixwert jeder Quelle-Ziel-Beziehung ij durch den Attributwert des Zielbezirks j ersetzt.

Hinweis

Diese Funktion steht nur für -Matrizen zur Verfügung.

Matrix von Datei laden

Ist die Option ausgewählt, können Sie die Matrixwerte durch die Werte einer externen Matrix ersetzen. Über die Schaltfläche wählen Sie die gewünschte Matrixdatei aus.

Konstante

Ist die Option ausgewählt, können Sie die Matrixwerte durch eine Konstante ersetzen. Den Wert geben Sie in das Eingabefeld ein (siehe „Alle Matrixwerte ersetzen“ auf Seite 909).

Die Matrixwerte werden ersetzt.

Über das Symbol  Initialisieren in der Symbolleiste des Matrixfensters oder das Menü Matrixeditor > Initialisieren können Sie alle Matrixwerte auf null setzen.

Wenn Sie eine oder mehrere Matrizen als Operanden auswählen, müssen die Anzahl der Bezirke und die Bezirksnummern der Matrizen nicht mit denen der Matrizen im Matrixfenster identisch sein. Die Rechenoperation wird aber nur mit Matrixwerten durchgeführt, deren Quelle-Ziel-Beziehungen übereinstimmen. Falls die Matrizen nicht die gleichen Quelle-Ziel-Beziehungen besitzen, gilt:

  • Falls in einer als Operand ausgewählten Matrix eine Quelle-Ziel-Beziehung nicht vorhanden ist, wird für diese der Matrixwert null verwendet.
  • Quelle-Ziel-Beziehungen, die nicht im Matrixfenster vorhanden sind, werden ignoriert.

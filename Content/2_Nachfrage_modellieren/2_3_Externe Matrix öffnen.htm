Externe Matrix öffnen

Sie können eine Matrixdatei in öffnen.

Klicken Sie in der Symbolleiste des Fensters Matrizen auf das Symbol  Externe Matrix öffnen.

Das Fenster Externe Matrizen öffnen öffnet sich.

Alternativ können Sie im Menü Bearbeiten den Eintrag Matrixeditor > Externe Matrix öffnen wählen.

Alternativ können Sie im Fenster Matrizen mit der rechten Maustaste neben den Navigator klicken und im Kontextmenü den Eintrag Externe Matrix öffnen wählen.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Alternativ können Sie Pfad und Dateiname manuell eingeben.

Ein Matrixeditorfenster mit der Matrix öffnet sich (Das Matrixeditorfenster).

Sie können eine externe Matrix direkt beim Öffnen in das Netzmodell einlesen (Externe Matrix ins Netzmodell einlesen).

Über das Symbol  Matrizen auswählen können Sie die in diesem Fenster angezeigten Matrizen verwalten (Im Matrix- oder Histogrammfenster angezeigte Matrizen verwalten).

Wenn Sie eine externe Matrix öffnen, wird sie im Navigator des Fensters Matrizen angezeigt. Sie bleibt auch angezeigt, wenn Sie die Matrix schließen. Anschließend können Sie sie einfach mit einem Doppelklick öffnen.

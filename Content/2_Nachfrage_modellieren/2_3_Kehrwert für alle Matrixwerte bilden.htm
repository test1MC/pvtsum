Kehrwert für alle Matrixwerte bilden

Stellen Sie sicher, dass ein Matrixeditorfenster mit der gewünschten Matrix geöffnet ist (Matrizen öffnen).

Prüfen Sie bei Bedarf die Einstellungen für eine Division durch null (Behandlung von Division durch null festlegen).

Klicken Sie in der Symbolleiste des Matrixfensters auf das Symbol  Kehrwert bilden.

Alternativ können Sie im Menü Matrixeditor den Eintrag Rechenoperationen > Kehrwert bilden wählen.

Für jede Zelle wird der Kehrwert des Matrixwerts gebildet und in der Matrix übernommen. Bei einer Division durch null wird der festgelegte Wert übernommen.

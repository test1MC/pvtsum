Zwei Matrizen in einem Streudiagramm vergleichen

Sie können die Matrixwerte von zwei Matrizen mit gleichen Quelle-Ziel-Beziehungen in einem Streudiagramm vergleichen.

Drücken Sie Taste Strg und halten Sie sie gedrückt.

Wählen Sie im Fenster Matrizen im Navigator die gewünschten Matrizen aus.

Klicken Sie mit der rechten Maustaste auf eine der gewählten Matrizen.

Wählen Sie den Eintrag Paarweise vergleichen.

Das Fenster Matrixvergleich öffnet sich. Die Matrixwerte der beiden Matrizen werden als Streudiagramm angezeigt. Die zuerst ausgewählte Matrix wird auf der X-Achse dargestellt, die andere auf der Y-Achse. Eine blaue Linie kennzeichnet die Diagonale, eine rote Linie die Regressionsgerade. Rechts unten werden statistische Werte angezeigt: die Gleichung der Regressionsgeraden in der Form a • x + b, das Bestimmtheitsmaß r2 und die Anzahl der Matrixwerte n.

Wenn die Matrizen sehr groß sind, werden die Punkte nicht mehr einzeln, sondern in einer gerasterten Darstellung angezeigt. Eine zusätzliche Legende erklärt die Bedeutung der Graustufen.

Im Fenster Matrixvergleich können Sie die Ansicht vergrößern, verkleinern, verschieben und wieder auf die ursprüngliche Ansicht zurücksetzen. Diese Funktionen entsprechen denen im Fenster Netzeditor und können über das Menü Ansicht, über Symbole in der Symbolleiste oder mit dem Scrollrad durchgeführt werden (Fenster Netzeditor).

Sie können den Matrixvergleich als Grafikdatei exportieren, indem Sie im Menü Matrixvergleich einen der folgenden Einträge wählen:

  • JPG Export (Screenshot) ( Screenshot vom Bildschirm exportieren ) ,
  • SVG Export ( SVG-Export ) ,
  • DXF Export ( DXF Export ) wählen.

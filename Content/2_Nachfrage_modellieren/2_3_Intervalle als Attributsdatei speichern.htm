Intervalle als Attributsdatei speichern

Stellen Sie sicher, dass Sie im Histogramm Intervalle definiert haben (Histogramm erstellen).

Klicken Sie auf das Symbol  Intervalle speichern.

Das Fenster Intervalle speichern öffnet sich.

Alternativ können Sie im Menü Matrixhistogramm den Eintrag Intervalle speichern wählen.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster Matrix wählen öffnet sich.

Wählen Sie in der Auswahlliste die Matrizen aus, deren berechnete Attribute sie speichern möchten.

Das Fenster Attribute schreiben (Verteilungsintervalle) öffnet sich.

Wählen Sie in der Auswahlliste Trennzeichen den gewünschten Eintrag.

Wenn Sie die Attributsdatei wieder einlesen möchten, müssen Sie als Trennzeichen entweder das Semikolon oder den Tabulator wählen.

Die Attributsdatei wird gespeichert.

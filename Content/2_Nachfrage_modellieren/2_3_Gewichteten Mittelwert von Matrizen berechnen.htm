Gewichteten Mittelwert von Matrizen berechnen

Sie können den Mittelwert von zwei Matrizen (Nachfragematrizen oder Kenngrößenmatrizen) gemäß MSA (Method of Successive Averages) ermitteln (siehe Grundlagen: Kap. 0.0.0.1 , Seite 1). Dabei wird in mehreren Iterationen jeweils der gewichtete Mittelwert ermittelt zwischen der gemittelten Matrix aus der letzten Iteration und der Matrix aus der aktuellen Iteration.

Klicken Sie in der Symbolleiste auf das Symbol  Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Mittelwertbildung (MSA) von Matrizen aus der Kategorie Matrizen ein (siehe „Verfahrensablauf festlegen und starten“ auf Seite 1).

Klicken Sie in der Zeile des neu eingefügten Verfahrens auf die Schaltfläche in der Spalte Bezugsobjekt(e).

Das Fenster Matrix auswählen öffnet sich.

Wählen Sie die Matrix, in die Sie die Mittelwerte schreiben möchten. Dafür stehen Ihnen zwei Methoden zur Verfügung (siehe „Fenster Matrizen auswählen“ auf Seite 1).

Element Beschreibung

Matrix auswählen nach Nummer

Wählen Sie aus der Liste aller Matrizen anhand der Nummer die gewünschte Matrix aus.

Matrix auswählen nach Eigenschaften

Wählen Sie anhand bestimmter Attribute und deren Werte die Matrix aus.

Die ausgewählte Matrix wird in die Spalte Bezugsobjekt(e) übernommen.

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist, und klicken Sie im Bereich Aktionen auf die Schaltfläche Bearbeiten.

Das Fenster Parameter Mittelwertbildung (MSA) von Matrizen öffnet sich.

Element

Beschreibung

Ergebnismatrix mit 0 initialisieren

Ist die Option ausgewählt, werden die Werte der Ergebnismatrix, die in der Spalte Bezugsobjekt(e) ausgewählt ist, vor Beginn des Verfahrensablaufs auf 0 gesetzt.

Schaltflächen

Über die obere Schaltfläche wählen Sie die Ergebnismatrix aus. Sie entspricht automatisch der ersten Matrix auf der rechten Seite der Gleichung (siehe „Fenster Matrizen auswählen“ auf Seite 1).

Über die untere Schaltfläche wählen Sie die Matrix aus der aktuellen Iteration aus.

Auswahlliste

Über die Auswahlliste wählen Sie den Gewichtungsfaktor, mit dem die Matrixwerte der Ergebnismatrix multipliziert werden.

i/(i+1)

Die Ergebnismatrix wird elementweise mit dem Iterationszähler i multipliziert und durch (i+1) dividiert. Der Gewichtungsfaktor der anderen Matrix ändert sich automatisch zu 1/(i+1). Dies ist die Standardeinstellung zur Berechnung von MSA.

1/(i+1)

Die Ergebnismatrix wird elementweise mit 1 multipliziert und durch (i+1) dividiert. Der Gewichtungsfaktor der anderen Matrix ändert sich automatisch zu i/(i+1).

Bedingter Rücksprung

Hier wird angezeigt, in welcher Zeile des Verfahrensablaufs der bedingte Rücksprung festgelegt ist, der für die iterative Berechnung zuständig ist.

Iteration

Hier wird angezeigt, in welchem Iterationsschritt sich das Verfahren aktuell befindet.

Führen Sie das Verfahren aus (siehe „Verfahren im Verfahrensablauf ausführen“ auf Seite 1).

Die ausgewählten Matrizen werden gemittelt.

Aufkommen von VISEM-Nachfrageschichten ändern

Sie können den Wert der in der Stufe VISEM-Verkehrserzeugung berechneten Heimataufkommen für eine Nachfrageschicht ändern (VISEM-Verkehrserzeugung berechnen).

Wählen Sie im Menü Nachfrage den Eintrag Nachfragemodelle.

Das Fenster Nachfragemodelle öffnet sich.

Wählen Sie das Register Nachfrageschichten.

Wählen Sie in der Auswahlliste Nachfragemodell auswählen das gewünschte Nachfragemodell.

Wählen Sie in der Liste die Nachfrageschicht aus, deren Heimataufkommen Sie ändern möchten.

Klicken Sie auf die Schaltfläche Heimataufkommen.

Das Fenster Heimataufkommen festlegen öffnet sich.

Nehmen Sie die gewünschten Änderungen vor.

Element

Beschreibung

Heimataufkommen

Anzahl Fahrten der Nachfrageschicht, die im jeweiligen Bezirk beginnen und enden

Der Wert wird in der Stufe VISEM-Verkehrserzeugung berechnet (VISEM-Verkehrserzeugung berechnen).

Die geänderten Werte werden für die Nachfrageschicht übernommen.

Alternativ können Sie das Heimataufkommen von Nachfrageschichten in Listen anzeigen und ändern (Aufkommen von VISEM-Nachfrageschichten in Listen anzeigen).

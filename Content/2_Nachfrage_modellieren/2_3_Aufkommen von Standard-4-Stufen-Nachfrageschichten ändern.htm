Aufkommen von Standard-4-Stufen-Nachfrageschichten ändern

Sie können den Wert der in der Stufe Verkehrserzeugung berechneten Quell- und Zielaufkommen für eine Nachfrageschicht ändern (Iterative Wiederholung vornehmen).

Wählen Sie im Menü Nachfrage den Eintrag Nachfragemodelle.

Das Fenster Nachfragemodelle öffnet sich.

Wählen Sie das Register Nachfrageschichten.

Wählen Sie in der Auswahlliste Nachfragemodell auswählen das gewünschte Nachfragemodell.

Wählen Sie in der Liste die Nachfrageschicht aus, deren Quell- und Zielaufkommen Sie ändern möchten.

Klicken Sie auf die Schaltfläche Quell/Zielaufkommen.

Das Fenster Quell-/Zielaufkommen festlegen öffnet sich.

Nehmen Sie die gewünschten Änderungen vor.

Element

Beschreibung

Quellaufkommen

Anzahl Fahrten der Nachfrageschicht, die im jeweiligen Bezirk beginnen

Der Wert wird in der Stufe Verkehrserzeugung berechnet.

Zielaufkommen

Anzahl Fahrten der Nachfrageschicht, die im jeweiligen Bezirk enden

Der Wert wird in der Stufe Verkehrserzeugung berechnet.

Die geänderten Werte werden für die Nachfrageschicht übernommen.

Alternativ können Sie das Quell- und Zielaufkommen von Nachfrageschichten für einen bestimmten Bezirk ändern (Bezirke: Register NSchichten).

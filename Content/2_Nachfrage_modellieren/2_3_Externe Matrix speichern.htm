Externe Matrix speichern

Sie können eine externe, geöffnete Matrix speichern.

Stellen Sie sicher, dass die gewünschte Matrix geöffnet ist (Matrizen öffnen).

Klicken Sie in der Symbolleiste des Matrixeditorfensters auf das Symbol  Speichern unter.

Das Fenster Matrix speichern öffnet sich.

Alternativ können Sie im Navigator des Registers Matrizen mit der rechten Maustaste auf eine externe Matrix klicken und im Kontextmenü den Eintrag Speichern unter wählen.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Der Dateiname der externen Matrix wird automatisch vorgeschlagen.

Wählen Sie bei Bedarf den Dateityp aus.

Die verfügbaren Dateitypen können Sie über das Menü Datei > Projektverzeichnisse >Projektverzeichnisse bearbeiten definieren und verwalten (Speicherort von Dateien ändern).

Wenn Sie den Dateityp Matrix (*.*) wählen, müssen Sie zusätzlich eine Erweiterung für die Matrixdatei eingeben.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster Matrix <Matrixname> schreiben öffnet sich.

Element

Beschreibung

Format

Über die Auswahlliste können Sie das Ausgabeformat für die Matrix wählen.

Weitere Informationen zu den Formaten finden Sie im Anhang (Matrixformate).

Matrix

Wenn Sie eine externe Matrix zum Speichern ausgewählt haben, enthält die Auswahlliste nur den Eintrag Externe Matrix.

Random Round

Über die Option können Sie einstellen, wie Werte beim Speichern gerundet werden sollen, wenn die als Matrixattribut festgelegte Anzahl Dezimalstellen kleiner ist als die tatsächliche Anzahl Dezimalstellen (Attributwerte von Matrizen anzeigen und bearbeiten).

Ist die Option ausgewählt, werden die Werte zufällig gerundet, d. h. in Abhängigkeit vom zu rundenden Anteil. Die Summe der Matrixwerte bleibt dabei konstant.

Beispiel

0,3 wird mit 30% Wahrscheinlichkeit auf 1 aufgerundet und mit 70% Wahrscheinlichkeit auf 0 abgerundet.

Ist die Option nicht ausgewählt, werden die Werte arithmetisch gerundet, d. h. mit einem Dezimalwert bis inklusive 4 wird abgerundet und mit einem Dezimalwert ab 5 aufgerundet. Dies gilt auch für negative Zahlen.

Beispiel

1,2 wird auf 1 abgerundet; 4,5 wird auf 5 und 5,5 auf 6 aufgerundet.

Die Matrix wird in der angegebenen Matrixdatei gespeichert. Im Matrixeditorfenster ist die neue Matrixdatei geöffnet.

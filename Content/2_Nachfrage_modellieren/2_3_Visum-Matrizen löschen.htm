-Matrizen löschen

Sie können -Matrizen löschen.

Wählen Sie im Fenster Matrizen im Navigator die gewünschten Matrizen.

Klicken Sie mit der rechten Maustaste auf eine der gewählten Matrizen.

Wählen Sie den Eintrag Löschen.

Alternativ können Sie die Matrizen auswählen und in der Symbolleiste des Fensters auf das Symbol  Löschen klicken. Beide Möglichkeiten finden Sie auch in der Liste Matrizen (Liste öffnen).

Eine Sicherheitsabfrage öffnet sich.

Wenn Sie eine -Matrix löschen, die noch einen Verweis zu einem anderen Nachfrageobjekt besitzt, löscht diesen Verweis.

Bestätigen Sie die Sicherheitsabfrage mit Ja.

Die Matrizen werden gelöscht.

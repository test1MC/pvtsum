Vordefinierte Nachfragemodelle *.dmd einlesen

Sie können eine Versionsdatei schnell und einfach mit Daten versorgen, indem Sie vordefinierte Nachfragemodelle und deren zugehörige Daten aus einer Nachfragedatendatei *.dmd einlesen.

Mit dem Programm ausgelieferte Nachfragemodell-Vorlagen finden Sie unter ...\Programme\PTV Vision\\Data\Demand_Templates.

Wählen Sie im Menü Nachfrage den Eintrag Nachfragemodelle.

Das Fenster Nachfragemodelle öffnet sich.

Wählen Sie das Register Basis.

Klicken Sie auf die Schaltfläche Modellvorlage laden.

Das Fenster Modellvorlage öffnen öffnet sich.

Wählen Sie die gewünschte Nachfragedatendatei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Eine Abfrage öffnet sich.

Das Nachfragemodell und die zugehörigen Nachfrageobjekte werden additiv hinzugelesen.

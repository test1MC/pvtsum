Matrixwerte aus einer Datei während des Verfahrensablaufs addieren

Klicken Sie in der Symbolleiste auf das Symbol  Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Matrix öffnen aus der Kategorie Matrizen ein (siehe „Verfahrensablauf festlegen und starten“ auf Seite 1).

Klicken Sie in der Zeile des neu eingefügten Verfahrens auf die Schaltfläche in der Spalte Bezugsobjekt(e).

Das Fenster Matrix auswählen öffnet sich.

Wählen Sie die -Matrix, zu der Sie Werte addieren möchten. Dafür stehen Ihnen zwei Methoden zur Verfügung (siehe „Fenster Matrizen auswählen“ auf Seite 1).

Element Beschreibung

Matrix auswählen nach Nummer

Wählen Sie aus der Liste aller Matrizen anhand der Nummer die gewünschte Matrix aus.

Matrix auswählen nach Eigenschaften

Wählen Sie anhand bestimmter Attribute und deren Werte die Matrix aus.

Die ausgewählte Matrix wird in die Spalte Bezugsobjekt(e) übernommen.

Klicken Sie in der Zeile des eingefügten Verfahrens in die Spalte Variante/Datei.

Das Symbol wird angezeigt.

Klicken Sie auf das Symbol .

Das Fenster Matrix öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Sie können auch Matrizen auswählen, die in einem Fremdformat vorliegen, beispielsweise EMME/2 oder TModel (siehe „EMME/2-Import“ auf Seite 1 und „TModel-Import“ auf Seite 1).

Der Dateiname wird in die Spalte Variante/Datei übernommen.

Alternativ können Sie in das Feld in der Spalte Variante/Datei doppelklicken und Pfad und Dateinamen manuell eingeben.

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist und klicken Sie im Bereich Aktionen auf die Schaltfläche Bearbeiten.

Das Fenster Additiv lesen öffnet sich.

Aktivieren Sie die Option Additiv lesen.

Führen Sie das Verfahren aus (siehe „Verfahren im Verfahrensablauf ausführen“ auf Seite 1).

Die Matrix wird mit Werten aus der ausgewählten Matrixdatei belegt.

Die Anzahl der Bezirke und die Bezirksnummern in der eingelesenen Matrixdatei müssen nicht mit denen der gewählten -Matrix identisch sein. Die Rechenoperation wird aber nur mit Matrixwerten durchgeführt, deren Quelle-Ziel-Beziehungen übereinstimmen. Falls die Matrizen nicht die gleichen Quelle-Ziel-Beziehungen besitzen, gilt:

  • Falls in der Matrixdatei eine Quelle-Ziel-Beziehung nicht vorhanden ist, wird für diese der Matrixwert null verwendet.
  • Quelle-Ziel-Beziehungen, die nicht in der -Datei vorhanden sind, werden ignoriert.

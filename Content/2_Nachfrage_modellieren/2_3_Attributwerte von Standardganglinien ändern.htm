Attributwerte von Standardganglinien ändern

Die folgende Schrittanleitung beschreibt, wie Sie die Werte der Attribute Nr, Name und Typ ändern können. Die Werte der Attribute von und bis ändern Sie, indem Sie die für die Standardganglinie eingefügten Zeitintervalle ändern (Zeitintervalle von Standardganglinien ändern).

Wählen Sie im Menü Nachfrage den Eintrag Nachfragedaten.

Das Fenster Nachfragedaten öffnet sich.

Wählen Sie das Register Standardganglinien.

Nehmen Sie die gewünschten Einstellungen vor (Attribute und Optionen von Standardganglinien).

Die Änderungen werden übernommen.

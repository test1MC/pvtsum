Zeilen bzw. Spalten in Gruppen einteilen

Sie haben mehrere Möglichkeiten, die Bezirke in Gruppen einzuteilen:

  • Gebietszuordnung aus einer entsprechenden Parameterdatei einlesen, die Sie beim Aggregieren erzeugt haben (Zeilen und Spalten für die Aggregierung auswählen)
  • Bezirke entsprechend der Attributwerte eines Attributs in Gruppen einteilen

Diese Funktion steht nur für -Matrizen zur Verfügung.

  • Einteilung manuell festlegen
Gebietszuordnung aus einer Datei einlesen

Wenn Sie zuvor eine Gebietszuordnung als Datei *.xml oder *.cod gespeichert haben, beispielsweise beim Aggregieren, können Sie diese einlesen (Zeilen und Spalten für die Aggregierung auswählen).

Stellen Sie sicher, dass das Fenster Gebietsbezogenes Hochrechnen geöffnet ist.

Klicken Sie auf die Schaltfläche Parameter-Datei öffnen.

Das Fenster Parameter für Matrixoperationen öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Bestätigen Sie mit Öffnen.

Die Gebietszuordnungen werden angezeigt.

Gebietszuordnung aus einem Attribut erzeugen

Für -Matrizen können Sie die Bezirke entsprechend der Werte eines bestimmten Attributs in Gebiete einteilen. Beispielsweise können Sie das Attribut Oberbezirks-Nummer verwenden, um die Bezirke entsprechend ihrer Zugehörigkeit zu Oberbezirken in Gruppen einzuteilen.

Wenn Sie eine Gebietszuordnung aus einem Attribut erzeugen, werden alle anderen Einträge in der Liste gelöscht.

Stellen Sie sicher, dass das Fenster Gebietsbezogenes Hochrechnen geöffnet ist.

Klicken Sie auf die Schaltfläche Aus Attribut erzeugen.

Das Fenster Attribute öffnet sich.

Wählen Sie das gewünschte Attribut aus.

Die Gebietszuordnungen werden erzeugt. Bezirke mit gleichem Attributwert werden gruppiert.

Gebiet manuell definieren

Stellen Sie sicher, dass das Fenster Gebietsbezogenes Hochrechnen geöffnet ist.

Klicken Sie auf die Schaltfläche Einfügen.

Das Fenster Auswahl der Bezirke öffnet sich.

Wählen Sie in der Liste die Objekte aus, die Sie in eine Gruppe einteilen möchten.

Über die Schaltflächen Alle an und Alle aus können Sie alle Objekte auswählen bzw. nicht auswählen. Über die Schaltfläche Invertieren können Sie die aktuelle Auswahl umkehren.

Die Gebietszuordnung wird als neue Zeile in der Liste angezeigt.

Gebietseinteilung ändern
  • Über die Schaltfläche Bearbeiten ändern Sie die Gebietszuordnung des aktuell ausgewählten Gebiets.
  • Über die Schaltfläche Löschen löschen Sie die aktuell ausgewählten Gebietszuordnungen.

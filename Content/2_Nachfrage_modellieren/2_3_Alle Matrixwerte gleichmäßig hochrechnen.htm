Alle Matrixwerte gleichmäßig hochrechnen

Stellen Sie sicher, dass ein Matrixeditorfenster mit der gewünschten Matrix geöffnet ist (Matrizen öffnen).

Klicken Sie in der Symbolleiste des Matrixfensters auf das Symbol  Hochrechnen.

Alternativ können Sie im Menü Matrixeditor den Eintrag Spezielle Matrixoperationen > Hochrechnen wählen.

Das Fenster Hochrechnen öffnet sich.

Wenn Sie die Einstellungen für die Hochrechnung in einer Datei gespeichert haben (*.xml, *.cod), können Sie sie über die Schaltfläche Öffnen laden.

Wählen Sie in der Auswahlliste Hochrechnen wird durchgeführt den Eintrag für gesamte Matrix.

Element

Beschreibung

Allgemeine Parameter

mit Faktor multiplizieren

Ist die Option ausgewählt, wird jeder Matrixwert mit einem Faktor multipliziert.

Hochrechnen auf Sollwert(e)

Ist die Option ausgewählt, wird jeder Matrixwert so hochgerechnet, dass die Summe aller Matrixwerte einem vorgegebenen Sollwert entspricht.

Parameter für Bezugstyp ‚Gesamte Matrix‘

Der Inhalt dieses Eingabefelds hängt von der Option ab, die im Bereich Allgemeine Parameter ausgewählt ist.

Faktor

Faktor, mit dem die Matrixwerte multipliziert werden

Sollwert

Sollwert, auf den die Matrixsumme hochgerechnet wird

Die Matrixwerte werden hochgerechnet.

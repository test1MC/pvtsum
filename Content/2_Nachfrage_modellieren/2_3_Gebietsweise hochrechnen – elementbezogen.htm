Gebietsweise hochrechnen – elementbezogen

Stellen Sie sicher, dass ein Matrixeditorfenster mit der gewünschten Matrix geöffnet ist (Matrizen öffnen).

Klicken Sie in der Symbolleiste des Matrixfensters auf das Symbol  Gebietsweise hochrechnen.

Alternativ können Sie im Menü Matrixeditor den Eintrag Spezielle Matrixoperationen > Gebietsweise hochrechnen wählen.

Das Fenster Gebietsbezogenes Hochrechnen öffnet sich.

Wählen Sie im Bereich Hochrechnen erfolgt die Option elementbezogen.

Legen Sie im Bereich Gebietsdefinition die Gruppen fest (Zeilen bzw. Spalten in Gruppen einteilen).

Im Bereich Elementbezogenes Hochrechnen wird eine Matrix angezeigt, die die festgelegten Gruppen (=Gebiete) als Zeilen und Spalten enthält.

Nehmen Sie im Bereich Elementbezogenes Hochrechnen die gewünschten Einstellungen vor.

Element

Beschreibung

Matrixwerte

In der Matrix können Sie pro Beziehung zwischen zwei Gebieten einen Faktor eingeben.

Matrix öffnen

Über die Schaltfläche lesen Sie die Hochrechnungsfaktoren aus einer Matrixdatei ein.

Hinweis

Die Gebietsnummern in der Matrixdatei müssen mit denen in der Gebietsdefinition übereinstimmen. Sie können bei Bedarf im Bereich Gebietsdefinition die Gebietsnummern ändern.

Matrix speichern

Über die Schaltfläche speichern Sie die in der Matrix definierten Hochrechnungsfaktoren.

Die Matrix wird gebietsweise hochgerechnet.

Zeitintervalle von Standardganglinien löschen

Wählen Sie im Menü Nachfrage den Eintrag Nachfragedaten.

Das Fenster Nachfragedaten öffnet sich.

Wählen Sie das Register Standardganglinien.

Wählen Sie in der Liste die Standardganglinie aus, deren Zeitintervalle Sie löschen möchten.

Klicken Sie auf die Schaltfläche Bearbeiten.

Das Fenster Ganglinie bearbeiten öffnet sich.

Wählen Sie in der Liste das Zeitintervall aus, das Sie löschen möchten.

Klicken Sie auf die Schaltfläche Löschen.

Das Zeitintervall wird gelöscht.

Löschen Sie bei Bedarf weitere Zeitintervalle.

Die Änderungen werden für die Standardganglinie übernommen.

Attributwerte von Matrizen anzeigen und bearbeiten

Sie können Attributwerte von Matrizen im Fenster Schnellansicht und in der Liste bearbeiten.

Attributwerte im Fenster Schnellansicht bearbeiten

Stellen Sie sicher, dass das Fenster Schnellansicht geöffnet ist (Fenster Schnellansicht).

Wählen Sie die gewünschten Attribute für das Fenster Schnellansicht aus (Attribute für die Schnellansicht auswählen).

Klicken Sie im Fenster Matrizen auf die gewünschte Matrix.

Ändern Sie im Fenster Schnellansicht die bearbeitbaren Attributwerte.

Attributwerte externer Matrizen werden nur angezeigt, wenn die Matrizen geöffnet sind.

Element

Beschreibung

Nummer

Frei wählbare Nummer der Matrix

Code

Kurzbezeichnung der Matrix

Name

Langbezeichnung der Matrix

Matrix-Typ

Nachfragematrix oder Kenngrößenmatrix

Bezugstyp

Beziehungen in Matrizen können sowohl zwischen Bezirken als auch zwischen Oberbezirken bestehen.

Anzahl Zeilen

Anzahl der Zeilen in der Matrix (nicht bearbeitbar)

Anzahl Spalten

Anzahl der Spalten in der Matrix (nicht bearbeitbar)

Summe

Summe aller Matrixwerte (nicht bearbeitbar)

Diagonal-Summe

Summe der Matrixdiagonale (nicht bearbeitbar)

Hinweis

Die Diagonale einer Matrix verläuft von oben links nach unten rechts (Von-Bezirksnummer = Nach-Bezirksnummer). In Nachfragematrizen entspricht die Diagonale dem Zellbinnenverkehr.

NSeg-Code

Für Kenngrößenmatrizen können Sie über die Auswahlliste ein Nachfragesegment wählen, auf das die Matrix verweist.

NSeg-Set

Für Nachfragematrizen werden die Codes der Nachfragesegmente, die mit der Matrix verknüpft sind, angezeigt (Nachfragematrizen mit Nachfragesegmenten verknüpfen).

NSchicht-Set

Nachfrageschichten, für die das Ergebnis der Verkehrsverteilung in die Matrix gespeichert wird (Attribute und Optionen von Nachfrageschichten).

Dateiname

Bei externen Matrizen Name der Matrixdatei

Bei -Matrizen Name der Datei, aus der die Matrixwerte zuletzt eingelesen wurden oder in die die Matrix zuletzt ausgegeben wurde.

Zufälliges Runden

Einstellung für das Runden, die beim letzten Speichern verwendet wurde

Dezimalstellen

Anzahl Dezimalstellen, mit denen die Matrixwerte in eine Datei gespeichert werden

Datenquellen Typ

Über die Auswahlliste wählen Sie aus, ob die Daten aus einer Datei stammen oder über eine Formel berechnet werden.

Hinweis

Sie können vom Eintrag Formel zu Daten wechseln, aber nicht umgekehrt.

Formel

Anzeige der verwendeten Formel

Die Attributwerte werden direkt geändert.

Attributwerte in der Liste bearbeiten

Öffnen Sie die Liste Matrizen (Liste öffnen).

Zeigen Sie die gewünschten Attribute an (Attribute anzeigen und ausblenden).

Ändern Sie die gewünschten Attributwerte (Attributwerte in einer Liste bearbeiten).

Die Attributwerte werden direkt geändert.

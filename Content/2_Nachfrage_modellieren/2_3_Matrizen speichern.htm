Matrizen speichern

Änderungen an -Matrizen bleiben zunächst erhalten, wenn Sie das Matrixeditorfenster schließen. Um die Änderungen dauerhaft zu speichern, müssen Sie die Versionsdatei speichern. -Matrizen können Sie bei Bedarf in eine Datei speichern.

Änderungen an externen Matrizen müssen Sie explizit in eine Datei speichern, damit sie erhalten bleiben, wenn Sie das Matrixeditorfenster schließen.

Sie können Matrizen entweder binär oder in einem von vier Textformaten speichern (-spezifische Matrixdateiformate).

Darüber hinaus können Sie für Nachfragematrizen die Belastung gefundener Routen je Beziehung als Teilmatrizen speichern, die jeweils nur die aktiven Fahrten oder nur die verbleibenden Fahrten enthalten.

Alternativ können Sie Kenngrößenmatrizen beim Berechnen als Datei speichern (Kenngrößen des IV berechnen und Kenngrößen des ÖV berechnen).

Alternativ können Sie Nachfragematrizen auch zusammen mit der Nachfragebeschreibung speichern (Nachfragedaten speichern).

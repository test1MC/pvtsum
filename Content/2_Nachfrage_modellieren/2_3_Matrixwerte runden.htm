Matrixwerte runden

Sie können Matrixwerte während des Verfahrensablaufs kaufmännisch auf einen bestimmten Wert runden. Die Matrixwerte werden dabei kaufmännisch so auf- oder abgerundet, dass der neue Wert ein Vielfaches des für die Rundung festgelegten Wertes ist.

Stellen Sie sicher, dass ein Matrixeditorfenster mit den gewünschten Matrizen geöffnet ist (Matrizen öffnen).

Klicken Sie auf das Symbol  Runden.

Das Fenster Operand für ‚Runden...‘ auf Gesamtmatrix öffnet sich.

Alternativ können Sie im Menü Matrixeditor den Eintrag Rechenoperationen > Runden wählen.

Geben Sie den Wert ein, auf den Sie runden möchten.

Die Matrixwerte werden gerundet.

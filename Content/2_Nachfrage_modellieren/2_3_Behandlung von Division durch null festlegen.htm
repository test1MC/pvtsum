Behandlung von Division durch null festlegen

Wählen Sie im Menü Bearbeiten den Eintrag Benutzereinstellungen.

Das Fenster Benutzereinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag Benutzeroberfläche > Matrixeditor.

Nehmen Sie im Bereich Verhalten von Rechenoperationen die gewünschten Einstellungen vor.

Element

Beschreibung

0 / 0

Standardwert, der gesetzt wird, wenn der Matrixwert 0 durch 0 geteilt wird.

Pos / 0

Standardwert, der gesetzt wird, wenn ein positiver Matrixwert durch 0 geteilt wird.

Neg / 0

Standardwert, der gesetzt wird, wenn ein negativer Matrixwert durch 0 geteilt wird.

Die Einstellungen werden übernommen.

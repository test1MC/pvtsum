Matrixdaten filtern

Sie können Matrixdaten von -Matrizen entsprechend bestimmter Filterkriterien ein- und ausblenden.

Stellen Sie sicher, dass ein Matrixeditorfenster mit der gewünschten Matrix geöffnet ist (Matrizen öffnen).

Wenn Sie zuvor schon Filterkriterien festgelegt haben, können Sie die Filterkriterien über das Symbol Anzeige der Matrixwerte filtern oder über das Menü Matrixeditor > Anzeige der Matrixwerte filtern anwenden.

Klicken Sie in der Symbolleiste des Matrixeditorfensters auf das Symbol Ansichtsoptionen.

Das Fenster Ansichtsoptionen Matrixeditor öffnet sich.

Alternativ können Sie im Menü Matrixeditor den Eintrag Ansichtsoptionen wählen.

Wählen Sie im Bereich Modus die Ansicht aus, für die Sie Einstellungen vornehmen möchten.

Bereich

Beschreibung
Matrixansicht

Ist die Option ausgewählt, wirken die Einstellungen in der Matrixansicht.

Listenansicht

Ist die Option ausgewählt, wirken die Einstellungen in der Listenansicht (Matrizen in Listenansicht öffnen).

Nehmen Sie im Bereich Filterung die gewünschten Einstellungen vor.

Bereich

Beschreibung

Filterung gemäß aktiver Bezirke

Ist die Option ausgewählt, werden nur die Matrixdaten aktiver Bezirke angezeigt (Netzobjekte über Filter aktiv und passiv setzen).

Hinweis

Wenn eine Oberbezirksmatrix geöffnet ist, ändert sich die Beschriftung entsprechend.

Filterung gemäß aktiver Bezirksbeziehungen

Ist die Option ausgewählt, werden nur die Matrixdaten aktiver Bezirksbeziehungen angezeigt (Sonderfall: Beziehungsfilter).

Hinweis

Wenn eine Oberbezirksmatrix geöffnet ist, ändert sich die Beschriftung entsprechend.

Filterung gemäß Matrixwerten

Ist die Option ausgewählt, werden nur die Matrixwerte angezeigt, die dem festgelegten Filterkriterium entsprechen.

von

unterster Wert des Intervalls

bis

oberster Wert des Intervalls

Komplement (Filterlogik umkehren)

Ist die Option ausgewählt, werden nur die Matrixwerte außerhalb dieses Intervalls angezeigt.

Ist die Option nicht ausgewählt, werden nur die Matrixwerte innerhalb dieses Intervalls angezeigt.

Diese Filterkriterien können kombiniert werden. Die Filter werden mit UND verknüpft, das heißt es werden nur Matrixdaten angezeigt, auf die alle Kriterien zutreffen.

Die Einstellungen werden für das Matrixeditorfenster übernommen. Das Symbol Anzeige der Matrixwerte filtern in der Symbolleiste wird gedrückt dargestellt.

Es werden nur ganze Zeilen und Spalten ausgeblendet. Sobald eine Matrixzelle gemäß der Filterkriterien angezeigt werden muss, werden die Zeile und die Spalte, in der sich die Matrixzelle befindet, mit angezeigt. Eine komplette Zeile oder Spalte wird in der Matrixansicht also erst ausgeblendet, wenn alle Zellen mindestens einer Matrix die Bedingungen der Filterung verletzen. In der Listenansicht wird eine Beziehung (Zeile) nur angezeigt, wenn alle ihre Matrixzellen die Bedingungen der Filterung erfüllen.

Sie können den Filter über das Symbol Anzeige der Matrixwerte filtern oder über das Menü Matrixeditor > Anzeige der Matrixwerte filtern wieder ausschalten.

Die gefilterte Matrix können Sie so speichern, dass nur die aktuell sichtbaren Zellen gespeichert werden (Teilmatrix speichern).

Externe Matrix erstellen

Klicken Sie im Fenster Matrizen mit der rechten Maustaste neben den Navigator.

Wählen Sie den Eintrag Externe Matrizen erzeugen.

Das Fenster Externe Matrizen erzeugen öffnet sich.

Element

Beschreibung

Anzahl erzeugter Matrizen

Anzahl der Matrizen, die Sie erzeugen möchten.

Hinweis

Fügen Sie mehr als eine Matrix ein, wird die Matrixnummer automatisch fortlaufend erhöht. Alle eingefügten Matrizen besitzen die gleichen Eigenschaften.

Nummer

Frei wählbare Nummer der Matrix (nur für Visum-Matrizen)

Code

Kurzbezeichnung der Matrix

Name

Langbezeichnung der Matrix

Matrix-Typ

Matrix-Typ der -Matrizen

  • Nachfragematrix
  • Kenngrößenmatrix

Matrizen mit Bezirksdimension

Ist die Option ausgewählt, wird eine externe Matrix mit der Dimension entsprechend der Bezirke in der Version angelegt. Sie können über die entsprechende Option die Bezirksnummern und -namen übernehmen.

Matrizen mit Oberbezirksdimension

Ist die Option ausgewählt, wird eine externe Matrix mit der Dimension entsprechend der Oberbezirke in der Version angelegt. Sie können über die entsprechende Option die

Oberbezirksnummern und -namen übernehmen.

Matrizen mit Haltestellenbereichsdimension

Ist die Option ausgewählt, wird eine externe Matrix mit der Dimension entsprechend der Haltestellenbereiche in der Version angelegt. Sie können über die entsprechende Option die Haltestellenbereichsnummern und -namen übernehmen..

Matrizen mit beliebiger Dimension

Ist die Option ausgewählt, können Sie eine externe Matrix mit beliebiger Dimension erzeugen. In die Eingabefelder geben Sie die Anzahl der Zeilen und Spalten ein.

Die neue Matrix wird erstellt und in einem Matrixeditorfenster geöffnet. Im Fenster Matrizen wird ein neuer Eintrag für die Matrix angezeigt.

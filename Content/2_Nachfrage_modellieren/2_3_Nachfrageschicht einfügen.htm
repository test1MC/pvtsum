Nachfrageschicht einfügen

Sie haben mehrere Möglichkeiten, Nachfrageschichten einzufügen.

  • Nachfrageschichten einzeln einfügen
  • Nachfrageschichten automatisch für markierte Aktivitätenpaare einfügen
  • Nachfrageschichten automatisch für markierte Personengruppen einfügen
  • Nachfrageschichten automatisch für markierte Aktivitätenketten einfügen
Nachfrageschichten einzeln einfügen

Wählen Sie im Menü Nachfrage den Eintrag Nachfragemodelle.

Das Fenster Nachfragemodelle öffnet sich.

Wählen Sie das Register Nachfrageschichten.

Wählen Sie in der Auswahlliste Nachfragemodell auswählen das gewünschte Nachfragemodell.

Klicken Sie auf die Schaltfläche Einfügen.

Eine neue Nachfrageschicht wird eingefügt.

Nehmen Sie die gewünschten Einstellungen vor (Attribute und Optionen von Nachfrageschichten).

Die Nachfrageschicht wird gespeichert.

Nachfrageschichten automatisch für markierte Aktivitätenpaare einfügen

Wenn Sie eine große Anzahl von Nachfrageschichten einfügen möchten, können Sie für Aktivitätenpaare von Nachfragemodellen des Typs Standard-4-Stufen oder EVA-P die zugehörigen Nachfrageschichten automatisch einfügen.

Wählen Sie im Menü Nachfrage den Eintrag Nachfragemodelle.

Das Fenster Nachfragemodelle öffnet sich.

Wählen Sie das Register Aktivitätenpaare.

Wählen Sie in der Auswahlliste Nachfragemodell auswählen das gewünschte Nachfragemodell.

Wählen Sie in der Liste die Aktivitätenpaare aus, für die Sie die zugehörigen Nachfrageschichten einfügen möchten.

Sie können mehrere Aktivitätenpaare auswählen, indem Sie die Taste Strg drücken, diese gedrückt halten und auf die gewünschten Einträge klicken.

Klicken Sie auf die Schaltfläche Nachfrageschichten einfügen.

Für die markierten Aktivitätenpaare werden so viele Nachfrageschichten-Einträge eingefügt, wie Personengruppen definiert sind (Personengruppen verwalten).

Nehmen Sie die gewünschten Einstellungen vor (Attribute und Optionen von Nachfrageschichten).

Die Nachfrageschichten werden gespeichert.

Wenn Sie eine große Anzahl von Nachfrageschichten einfügen möchten, können Sie für Personengruppen die zugehörigen Nachfrageschichten automatisch einfügen.

Das Fenster Nachfragemodelle öffnet sich.

Wählen Sie das Register Personengruppen.

Wählen Sie in der Auswahlliste Nachfragemodell auswählen das gewünschte Nachfragemodell.

Wählen Sie in der Liste die Personengruppen aus, für die Sie die zugehörigen Nachfrageschichten einfügen möchten.

Sie können mehrere Personengruppen auswählen, indem Sie die Taste Strg drücken, diese gedrückt halten und auf die gewünschten Einträge klicken.

Klicken Sie auf die Schaltfläche Nachfrageschichten einfügen.

Für die markierten Personengruppen werden so viele Nachfrageschichten-Einträge eingefügt, wie Aktivitätenpaare (Standard-4-Stufen und EVA-P) oder Aktivitätenketten (VISEM) definiert sind (Aktivitäten, Aktivitätenpaare, Aktivitätenketten verwalten).

Nehmen Sie die gewünschten Einstellungen vor (Attribute und Optionen von Nachfrageschichten).

Die Nachfrageschichten werden gespeichert.

Matrizen im Fenster Matrizen anzeigen

Das Fenster Matrizen enthält einen Navigator mit den in der Version vorhandenen Matrizen, unterteilt nach dem Matrixtyp (Nachfragematrix oder Kenngrößenmatrix). Unterhalb dieser Einteilung gibt es eine Einteilung nach dem Bezugstyp (Bezirks- oder Oberbezirksmatrix) und darunter Daten- und Formelmatrizen. Bei -Matrizen wird die Nummer und der Code angezeigt.

Außerdem werden auch alle externen Matrizen angezeigt, die Sie von dieser Version aus während der aktuellen Sitzung geöffnet haben. Bei externen nicht gespeicherten Matrizen wird der Code angezeigt, bei gespeicherten Matrizen der Dateiname. Externe Matrizen werden nicht in der Versionsdatei gespeichert.

Aktuell geöffnete Matrizen werden fett angezeigt. Wenn Sie den Mauszeiger auf den Namen einer Matrix stellen, öffnet sich eine Quickinfo mit Informationen zu der Matrix.

Von hier aus können Sie über die Symbolleiste und das Kontextmenü Matrizen zur Bearbeitung öffnen, neue Matrizen anlegen, Matrizen löschen und einfache Rechenoperationen an Matrizen durchführen, ohne sie zuvor zu öffnen.

Eine Liste aller -Matrizen können Sie öffnen, indem Sie im Fenster Matrizen mit der rechten Maustaste neben den Navigator klicken und im Kontextmenü den Eintrag Liste wählen oder indem Sie die Liste über das Menü Listen auswählen (Liste öffnen). In der Liste können Sie nach Matrizen suchen und Matrixattributwerte ändern (Mit Listen arbeiten und Attributwerte von Matrizen anzeigen und bearbeiten).

Wenn Sie im Fenster Matrizen eine Matrix auswählen, werden im Fenster Schnellansicht ausgewählte Attribute der Matrix angezeigt (Attribute für die Schnellansicht auswählen). Hier können Sie auch Attributwerte ändern (Attributwerte von Matrizen anzeigen und bearbeiten). Attributwerte externer Matrizen werden nur angezeigt, wenn die Matrizen geöffnet sind.

Die Einträge von aktuell nicht geöffneten externen Matrizen können Sie aus dem Navigator entfernen, indem Sie mit der rechten Maustaste neben den Navigator klicken und den Eintrag Nicht mehr geöffnete externe Matrizen aus der Ansicht entfernen wählen.

Layout eines Matrixeditor- oder Histogrammfensters speichern

Stellen Sie sicher, dass ein Matrixeditor- oder Histogrammfenster mit den gewünschten Matrizen geöffnet ist (Matrizen öffnen und Matrizen als Histogramm anzeigen).

Wählen Sie im Menü Matrixeditor oder Matrix-Histogramm den Eintrag Layout speichern.

Das Fenster Matrixeditor-Layout speichern öffnet sich.

Wählen Sie die gewünschte Datei aus.

Alternativ können Sie Pfad und Dateiname manuell eingeben.

Klicken Sie auf die Schaltfläche Speichern.

Das Layout des Matrixeditorfensters wird gespeichert.

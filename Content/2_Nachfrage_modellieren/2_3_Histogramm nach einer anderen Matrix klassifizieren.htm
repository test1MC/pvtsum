Histogramm nach einer anderen Matrix klassifizieren

Sie können die Klassen eines Histogramms entsprechend der Werte einer anderen Matrix klassifizieren. Beispielsweise können Sie damit darstellen, wie sich die Fahrten einer Nachfragematrix auf die Reiseweiten verteilen. Dazu stellen Sie die Daten der Nachfragematrix in einem Histogramm dar, indem Sie sie nach einer Kenngrößenmatrix mit den Reiseweiten klassifizieren.

Stellen Sie sicher, dass ein Histogramm geöffnet ist (Histogramm erstellen).

Klicken Sie auf das Symbol  Ansichtsoptionen.

Das Fenster Ansichtsoptionen Matrixhistogramm öffnet sich.

Wählen Sie die Option Klassifikationsmatrix verwenden.

Wählen Sie im Navigator die gewünschte Matrix für die Klassifizierung.

Legen Sie die gewünschten Intervalle fest.

Klicken Sie auf das Symbol  Klassifikation neu berechnen.

Alternativ können Sie im Menü Matrixhistogramm den Eintrag Klassifikation neu berechnen wählen.

Das Histogramm wird im oberen Bereich des Fensters angezeigt.

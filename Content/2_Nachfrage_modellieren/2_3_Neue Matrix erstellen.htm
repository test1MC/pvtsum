Neue Matrix erstellen

Sie können eine neue Matrix erstellen:

  • eine -Matrix, die für alle Beziehungen den Wert 0 enthält
  • eine externe Matrix
  • eine -Matrix, die automatisch aus den Werten anderer -Matrizen oder aus Attributwerten berechnet und später ständig aktualisiert wird.

Wenn Sie im Verfahrensablauf Ergebnismatrizen über Eigenschaften definieren, werden neue Matrizen erstellt, wenn entsprechende Matrizen nicht vorhanden sind.

Aktivitätenpaare mit Standardganglinien verknüpfen

Mit einem VISEM-Nachfragemodell können Sie die Nachfrage auch stundenfein berechnen, indem Sie prozentuale Standardganglinien mit Aktivitätenpaaren und Personengruppen verknüpfen (Attribute und Optionen von Standardganglinien).

Wählen Sie im Menü Nachfrage den Eintrag Nachfragemodelle.

Das Fenster Nachfragemodelle öffnet sich.

Wählen Sie das Register Aktivitätenpaare.

Wählen Sie in der Auswahlliste Nachfragemodell auswählen das gewünschte Nachfragemodell.

Wählen Sie in der Liste das Aktivitätenpaar, das Sie mit Standardganglinien und Personengruppen verknüpfen möchten.

Klicken Sie in der Spalte Ganglinie(n) auf die jeweilige Schaltfläche.

Das Fenster Ganglinie(n) auswählen öffnet sich.

Element

Beschreibung

Für alle Personengruppen die gleiche Ganglinie

Ist die Option ausgewählt, wird allen Personengruppen die gleiche Standardganglinie zugeordnet. Über die Auswahlliste können Sie die Standardganglinie auswählen.

Pro Personengruppe eine eigene Ganglinie

Ist die Option ausgewählt, können Sie jeder Personengruppe eine eigene Standardganglinie zuordnen, indem Sie die gewünschte Personengruppe in der Liste markieren und in der Spalte Ganglinie über die jeweilige Auswahlliste die Standardganglinie wählen.

Die Zuordnung wird übernommen.

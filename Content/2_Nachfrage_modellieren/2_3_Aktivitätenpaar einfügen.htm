Aktivitätenpaar einfügen

Sie haben zwei Möglichkeiten, Aktivitätenpaare einzufügen.

  • einzeln
  • automatisch für markierte Aktivitäten

Für Nachfragemodelle des Typs VISEM werden Aktivitätenpaare automatisch eingefügt, wenn Sie die Aktivitäten-Sequenz einer Aktivitätenkette definieren (Attribute und Optionen von Aktivitätenketten). Sie können Aktivitätenpaare aber auch einzeln oder automatisch für ausgewählte Aktivitäten einfügen.
Aktivitätenpaare einzeln einfügen

Wählen Sie im Menü Nachfrage den Eintrag Nachfragemodelle.

Das Fenster Nachfragemodelle öffnet sich.

Wählen Sie das Register Aktivitätenpaare.

Wählen Sie in der Auswahlliste Nachfragemodell auswählen das gewünschte Nachfragemodell.

Klicken Sie auf die Schaltfläche Einfügen.

Ein neues Aktivitätenpaar wird eingefügt.

Nehmen Sie die gewünschten Einstellungen vor (Attribute und Optionen von Aktivitätenpaaren).

Das Aktivitätenpaar wird gespeichert.

Aktivitätenpaare automatisch für ausgewählte Aktivitäten einfügen

Sie können für ausgewählte Aktivitäten die zugehörigen Aktivitätenpaare automatisch einfügen.

Wählen Sie im Menü Nachfrage den Eintrag Nachfragemodelle.

Das Fenster Nachfragemodelle öffnet sich.

Wählen Sie das Register Aktivitäten.

Wählen Sie in der Auswahlliste Nachfragemodell auswählen das gewünschte Nachfragemodell.

Wählen Sie in der Liste die Aktivitäten aus, für die Sie die zugehörigen Aktivitätenpaare einfügen möchten.

Sie können mehrere Aktivitäten auswählen, indem Sie die Taste Strg drücken, diese gedrückt halten und auf die gewünschten Einträge klicken.

Klicken Sie auf die Schaltfläche Aktivitätenpaare einfügen.

Für jede Kombination aller ausgewählten Aktivitäten mit allen anderen Aktivitäten in der Liste werden zwei Aktivitätenpaare angelegt – in jede Richtung eine. Außerdem wird für jede ausgewählte Aktivität ein Aktivitätenpaar angelegt, bei dem die ausgewählte Aktivität sowohl Quelle als auch Ziel ist.

Wenn Sie mehrere Aktivitäten ausgewählt haben, erhalten Sie eine Meldung über bereits vorhandene Aktivitätenpaare.

Bestätigen Sie die Meldung gegebenenfalls mit OK für alle.

Nehmen Sie die gewünschten Einstellungen vor (Attribute und Optionen von Aktivitätenpaaren).

Die Aktivitätenpaare werden gespeichert.

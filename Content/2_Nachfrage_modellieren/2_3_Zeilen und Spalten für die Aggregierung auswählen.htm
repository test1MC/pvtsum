Zeilen und Spalten für die Aggregierung auswählen

Für die Funktion Aggregieren wählen Sie aus, welche Zeilen bzw. Spalten Sie aggregieren, umnummerieren oder löschen möchten.

Zeilen bzw. Spalten zum Aggregieren auswählen

Stellen Sie sicher, dass das Fenster Aggregation geöffnet ist.

Klicken Sie auf die Schaltfläche Einfügen.

Das Fenster Auswahl aggregierter Bezirke öffnet sich.

Wählen Sie die Nummern der Zeilen bzw. Spalten aus, die Sie zu einer Zeile bzw. Spalte aggregieren möchten.

Über die Schaltflächen können Sie Einträge einfach auswählen:

  • Alle an: Auswählen aller Einträge
  • Alle aus: Auswahl aufheben
  • Invertieren: Auswählen aller aktuell nicht ausgewählten Einträge

In der Liste wird ein neuer Eintrag angelegt. Er enthält eine Spalte mit der neuen Zeilen- bzw. Spaltennummer und eine Spalte mit den entsprechenden bisherigen Zeilen- bzw. Spaltennummern.

Ändern Sie bei Bedarf in der Spalte NeuBezNr die Nummer der neuen Zeile bzw. Spalte.

Fügen Sie bei Bedarf auf die gleiche Weise weitere Einträge zur Liste hinzu.

Zeile bzw. Spalte zum Umnummerieren auswählen

Stellen Sie sicher, dass das Fenster Aggregation geöffnet ist.

Klicken Sie auf die Schaltfläche Einfügen.

Das Fenster Auswahl aggregierter Bezirke öffnet sich.

Wählen Sie die Nummer der Zeile bzw. Spalte aus, die Sie umnummerieren möchten.

In der Liste wird ein neuer Eintrag angelegt.

Geben Sie in der Spalte NeuBezNr die neue Nummer ein.

Fügen Sie bei Bedarf auf die gleiche Weise weitere Einträge zur Liste hinzu.

Zeilen bzw. Spalten zum Löschen auswählen

Stellen Sie sicher, dass das Fenster Aggregation geöffnet ist.

Klicken Sie auf die Schaltfläche Einfügen.

Das Fenster Auswahl aggregierter Bezirke öffnet sich.

Wählen Sie die Nummern der Zeilen bzw. Spalten aus, die Sie löschen möchten.

In der Liste wird ein neuer Eintrag angelegt.

Wählen Sie in der Spalte Aktion den Eintrag Löschen.

Fügen Sie bei Bedarf auf die gleiche Weise weitere Einträge zur Liste hinzu.

Sie können eine einzelne Zeile der Liste bearbeiten, indem Sie sie auswählen und auf die Schaltfläche Bearbeiten klicken.

Sie können Zeilen aus der Liste löschen, indem Sie sie auswählen und auf die Schaltfläche Löschen klicken.

Sie können die Liste in einer Parameterdatei (*.xml) speichern (Einstellungen für spezielle Matrixoperationen speichern).

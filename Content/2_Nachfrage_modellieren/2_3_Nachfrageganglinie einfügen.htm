Nachfrageganglinie einfügen

Für die Umlegung muss die zeitliche Verteilung der Fahrtwünsche mit der Nachfrage verknüpft werden. Die Nachfrage liegt als Nachfragematrix pro Nachfragesegment vor Siehe "Matrizen". Zur Verknüpfung stellen Sie eine Standardganglinie als Nachfrageganglinie bereit und weisen diese einem Nachfragesegment zu (Nachfrageganglinie mit Nachfragesegment verknüpfen).

Sie können für Quelle-Ziel-Beziehungen zwischen bestimmten Bezirkstypen die Standardganglinie ändern, indem Sie den gewünschten Beziehungen andere Standardganglinien zuweisen (Beziehungen zwischen bestimmten Bezirkstypen andere Standardganglinie zuweisen). Dies ist nur für prozentuale Standardganglinien möglich.

Wählen Sie im Menü Nachfrage den Eintrag Nachfragedaten.

Das Fenster Nachfragedaten öffnet sich.

Wählen Sie das Register Nachfrageganglinien.

Klicken Sie auf die Schaltfläche Einfügen.

Eine neue Nachfrageganglinie wird eingefügt.

Über die Auswahlliste können Sie der Nachfrageganglinie eine Standardganglinie zuweisen.

IV-Matrix kalibrieren

Mit dem Verfahren Nachfragematrix-Kalibration können Sie eine IV-Nachfragematrix anhand von Zählwerten hochrechnen (Grundlagen: IV-Matrix kalibrieren).

Sie können das Verfahren Nachfragematrix-Kalibration nicht mit der Gleichgewichtsumlegung LUCE oder der Tribut-Gleichgewichtsumlegung kombinieren.

Stellen Sie sicher, dass die Zählbelastungen als Strecken-ZWerte vorliegen (Strecken: Register Basis).

Stellen Sie sicher, dass das Netz ein IV-Umlegungsergebnis (Wege, die die Strecke benutzen) enthält.

Stellen Sie sicher, dass die vorhandene Nachfragematrix eine bestmöglich ermittelte Matrix ist, die aus früheren Zählungen oder Berechnungen entstanden sein kann und deren innere Struktur die Widerstandssituation im Netz richtig abbildet.

Dabei können durch repräsentative Zählquerschnitte auch signifikante Veränderungen (wie zum Beispiel Schließung einer Firma mit ehemals zahlenmäßig starker Belegschaft oder ein neuerbautes Wohngebiet) abgeglichen werden.

Stellen Sie sicher, dass das Netz bezüglich der Routenwahl bestmöglich geeicht ist.

Klicken Sie in der Symbolleiste auf das Symbol  Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Nachfragematrix-Kalibration aus der Kategorie Matrizen ein (Verfahrensablauf festlegen und starten).

Klicken Sie in der Zeile des neu eingefügten Verfahrens auf die Schaltfläche in der Spalte Bezugsobjekt(e).

Das Fenster Nachfragesegment Auswahl öffnet sich.

Wählen Sie in der Auswahlliste das IV-Nachfragesegment, deren Matrix Sie kalibrieren möchten.

Das ausgewählte Nachfragesegment wird in die Spalte Bezugsobjekt(e) übernommen.

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist, und klicken Sie im Bereich Aktionen auf die Schaltfläche Bearbeiten.

Das Fenster Parameter Matrixkalibrierung Lohse öffnet sich.

Element

Beschreibung

Zählwerte aus

Über die Auswahlliste können Sie den Strecken-ZWert wählen, der die Zählwerte enthält.

Max. Anzahl Iterationen

Der Iterationsprozess wird abgebrochen, wenn die eingegebene Anzahl Iterationen gerechnet wurde, bevor die Berechnung konvergiert.

Genauigkeitsschranke

Abbruchkriterium für das Multi-Verfahren (Grundlagen: Hochrechnen)

Hinweis

Je höher der Wert, desto höher sind auch die Rechenzeit und die Genauigkeit der Berechnungen.

Führen Sie das Verfahren aus (Verfahren im Verfahrensablauf ausführen).

Die Nachfragematrix des ausgewählten IV-Nachfragesegments wird kalibriert.

Sie können die Kalibrierung durch eine grafische oder tabellarische Darstellung der Berechnungsergebnisse überprüfen.

  • Differenz zwischen umgelegter Streckenbelastung und Zählwerten über das Menü Grafik > Grafikparameter bearbeiten > Eintrag Strecken > Balken > Darstellung > Balkentyp Differenzbalken (Darstellung eines Differenzbalkens festlegen).
  • Statistische Abweichungen über das Menü Rechnen > Umlegungsanalyse > Liste oder Diagramm (Umlegungsanalyse durchführen).

Nachfragematrizen mit Nachfragesegmenten verknüpfen

Für die Umlegung müssen Sie die Nachfrage, die als Nachfragematrix pro Nachfragesegment vorliegt, mit dem gewünschten Nachfragesegment verknüpfen. Bei der Umlegung wird für jedes gewünschte Nachfragesegment die Nachfrage der jeweils zugewiesenen Matrix berücksichtigt.

Sie können die gleiche Nachfragematrix mehreren Nachfragesegmenten zuweisen.

Wählen Sie im Menü Nachfrage den Eintrag Nachfragedaten.

Das Fenster Nachfragedaten öffnet sich.

Wählen Sie das Register Nachfragesegmente.

Klicken Sie in der Zeile des gewünschten Nachfragesegments in der ersten Spalte Matrix auf die Schaltfläche.

Das Fenster Matrix auswählen öffnet sich.

Wählen Sie die Matrix, deren Werte Sie auf Wege verteilen möchten. Dafür stehen Ihnen zwei Methoden zur Verfügung (siehe „Fenster Matrizen auswählen“ auf Seite 1).

Element Beschreibung

Matrix auswählen nach Nummer

Wählen Sie aus der Liste aller Matrizen anhand der Nummer die gewünschte Matrix aus.

Matrix auswählen nach Eigenschaften

Wählen Sie anhand bestimmter Attribute und deren Werte die Matrix aus.

Wählen Sie die gewünschte Nachfragematrix.

Die Nachfragematrix wird mit dem Nachfragesegment verknüpft. In der zweiten Spalte Matrix wird ein Verweis auf die Nachfragematrix angezeigt. Es ist eine Kombination aus Matrixnummer und Matrixname, wenn eine Matrix nach Nummer ausgewählt wurde. Bei Auswahl über Referenzen bleibt das Feld leer.

Wenn Sie mit der rechten Maustaste auf die gewählte Matrix klicken, können Sie über die Einträge Bearbeiten und Histogramm erstellen ein Matrixeditorfenster mit der Matrix bzw. den Matrixdaten öffnen (siehe „Matrizen verwalten, anzeigen und analysieren“ auf Seite 1).

Bestätigen Sie nochmals mit OK.

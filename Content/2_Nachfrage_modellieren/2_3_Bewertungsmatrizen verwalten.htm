Bewertungsmatrizen verwalten

Bewertungsmatrizen sind spezielle Matrizen, die als Berechnungsergebnis der Stufe EVA-Bewertung je Nachfrageschicht gespeichert werden (EVA-Bewertung berechnen).

Sie enthalten für jede Nachfrageschicht die Zahl der Fahrten von Verkehrszelle i nach Verkehrszelle j je Modus, d. h. sie sind dreidimensional.

Sie setzen sich aus den einzelnen Kenngrößenmatrizen einer Nachfrageschicht zusammen, die beim Einfügen der Nachfrageschicht je Modus automatisch erzeugt (Nachfrageschicht einfügen). Sie können Bewertungsmatrizen ähnlich wie Nachfrage- und Kenngrößenmatrizen bearbeiten (Matrizen verwalten, anzeigen und analysieren).

Bewertungsmatrizen werden nicht mit einer Nachfragedatendatei *.dmd gespeichert.
Bewertungsmatrix in speichern

Sie können eine Bewertungsmatrix für eine Nachfrageschicht als Bewertungsmatrixdatei *.wmt speichern.

Zeigen Sie im Menü Datei auf den Eintrag Speichern unter.

Wählen Sie den Eintrag EVA-Bewertungsmatrix.

Das Fenster EVA-Bewertungsmatrix speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster Nachfrageschichten Auswahl öffnet sich.

Wählen Sie die Nachfrageschicht, für die Sie die Bewertungsmatrix speichern möchten.

Die Bewertungsmatrix wird gespeichert.

Bewertungsmatrix in einlesen

Klicken Sie auf das Symbol  Öffnen.

Das Fenster Dateiart-Auswahl öffnet sich.

Alternativ können Sie das Fenster über das Menü Datei > Öffnen > EVA-Bewertungsmatrix aufrufen.

Wählen Sie den Eintrag EVA-Bewertungsmatrix.

Das Fenster EVA-Bewertungsmatrix öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Das Fenster Nachfrageschichten Auswahl öffnet sich.

Die Bewertungsmatrix wird eingelesen.

Bewertungsmatrix ändern

Wählen Sie im Menü Nachfrage den Eintrag EVA-Bewertungsmatrizen.

Das Fenster EVA-Bewertungsmatrizen öffnet sich.

Wählen Sie in der Auswahlliste Nachfragemodell auswählen das Nachfragemodell, zu dem die Bewertungsmatrizen gehören.

Alle Nachfrageschichten des ausgewählten Nachfragemodells werden angezeigt. Jeder Nachfrageschicht entspricht eine Bewertungsmatrix.

Wählen Sie in der Liste die Bewertungsmatrix aus, die Sie ändern möchten.

Klicken Sie auf die Schaltfläche Bearbeiten.

Je Modus wird ein Matrixeditorfenster geöffnet, das die modusspezifische Kenngrößenmatrix der gewählten Nachfrageschicht enthält.

Alternativ können Sie die einzelnen, modusspezifischen Kenngrößenmatrizen im Matrixeditor öffnen (Matrizen öffnen).

Nehmen Sie die gewünschten Änderungen je Modus vor (Matrizen verwalten, anzeigen und analysieren).

Sie können mit Hilfe der Auswahlliste in der Symbolleiste Fenster zwischen den einzelnen Fenstern des Matrixeditors wechseln (Zwischen Fenstern wechseln).
Werte von Bewertungsmatrizen auf null setzen

Wählen Sie im Menü Nachfrage den Eintrag EVA-Bewertungsmatrizen.

Das Fenster EVA-Bewertungsmatrizen öffnet sich.

Wählen Sie in der Auswahlliste Nachfragemodell auswählen das Nachfragemodell, zu dem die Bewertungsmatrizen gehören.

Alle Nachfrageschichten des ausgewählten Nachfragemodells werden angezeigt. Jeder Nachfrageschicht entspricht eine Bewertungsmatrix.

Wählen Sie in der Liste die Bewertungsmatrizen aus, deren Werte Sie unwiderruflich auf 0 setzen möchten.

Klicken Sie auf die Schaltfläche Initialisieren.

Eine Sicherheitsabfrage öffnet sich.

Bestätigen Sie die Sicherheitsabfrage mit Ja.

Die Werte der Bewertungsmatrizen werden unwiderruflich auf 0 gesetzt.

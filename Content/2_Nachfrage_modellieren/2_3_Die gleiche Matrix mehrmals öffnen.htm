Die gleiche Matrix mehrmals öffnen

Stellen Sie sicher, dass die gewünschte Matrix geöffnet ist (-Matrix öffnen und Externe Matrix öffnen).

Wählen Sie im Fenster Matrizen im Navigator die gewünschte Matrix aus.

Klicken Sie mit der rechten Maustaste auf die Matrix.

Wählen Sie den Eintrag In neuem Fenster bearbeiten.

Die Matrix öffnet sich in einem weiteren Matrixeditorfenster.

Über das Symbol  Matrizen auswählen können Sie die in diesem Fenster angezeigten Matrizen verwalten (Im Matrix- oder Histogrammfenster angezeigte Matrizen verwalten).

Grafikparameter für aktive Beziehungen einstellen

Stellen Sie sicher, dass ein Matrixeditorfenster mit der gewünschten Matrix geöffnet ist (Matrizen öffnen).

Wählen Sie im Menü Matrixeditor den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag Aktive Beziehungen > Darstellung.

Element

Beschreibung

Einheitliche Darstellung

Ist die Option ausgewählt, werden alle aktiven Beziehungen einheitlich dargestellt. Wählen Sie im Bereich Darstellung die gewünschte Farbe für den Hintergrund und die Schrift aus.

Klassifizierte Darstellung

Ist die Option ausgewählt, können die aktiven Beziehungen nach einem Attribut klassifiziert und dargestellt werden (Objekte anhand von Attributwerten klassifiziert darstellen).

Tipp

Über die Schaltflächen auf der Seite Aktive Beziehungen können Sie zuvor definierte Klassen für eine klassifizierte grafische Darstellung einlesen (siehe „Klassen für die Klassifizierung von Datei einlesen“ auf Seite 895).

Wenn Sie auf die Schaltfläche Vorschau klicken, werden Ihre Änderungen sofort angezeigt, ohne dass sich das Fenster Grafikparameter bearbeiten schließt.

Die entsprechenden Matrixwerte werden in den eingestellten Farben dargestellt.

Klassen für die Klassifizierung von Datei einlesen

Stellen Sie sicher, dass ein Matrixeditorfenster mit der gewünschten Matrix geöffnet ist (Matrizen öffnen).

Wählen Sie im Menü Matrixeditor den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag Aktive Beziehungen.

Klicken Sie auf die gewünschte Schaltfläche.

Element

Beschreibung

Klassifizierung aus der Zwischenablage lesen

Über die Schaltfläche lesen Sie Klassen ein, die Sie zuvor in die Zwischenablage kopiert haben, beispielsweise aus den Histogrammeinstellungen (Matrizen als Histogramm anzeigen).

Klassifizierung von Datei lesen

Über die Schaltfläche lesen Sie Klassen ein, die Sie zuvor als Attributsdatei gespeichert haben (Intervalle als Attributsdatei speichern).

Die Klassen werden eingelesen.

Wählen Sie im Navigator den Eintrag Aktive Beziehungen > Darstellung.

Nehmen Sie bei Bedarf weitere Einstellungen vor.

Einstellung für das Einlesen von Matrizen im $O-Format vornehmen

Bei Matrizen im $O-Format ($O-Format) können (Ober-)-Bezirksbeziehungen mehrfach vorkommen. Somit gibt es für solche (Ober-)-Bezirksbeziehungen mehrere Matrixwerte. Sie können einstellen, wie diese Matrixwerte beim Einlesen in die Nachfragebeschreibung behandelt werden sollen.

Wählen Sie im Menü Bearbeiten den Eintrag Benutzereinstellungen.

Das Fenster Benutzereinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag Dateien & Protokolle > Matrizen.

Stellen Sie ein, wie mehrfach vorkommende (Ober-)Bezirksbeziehungen in Matrizen im $O-Format behandelt werden sollen.

Element

Beschreibung

alle Werte einer (Ober-)Bezirksbeziehung addieren

Ist die Option ausgewählt, werden alle Werte der gleichen (Ober-)-Bezirksbeziehung zu einem Matrixwert addiert.

letzten Wert einer (Ober-)Bezirksbeziehung verwenden

Ist die Option ausgewählt, wird der Wert, der als letztes für die (Ober-)-Bezirksbeziehung gelesen wird, als Matrixwert verwendet.

Die Einstellung wird übernommen.

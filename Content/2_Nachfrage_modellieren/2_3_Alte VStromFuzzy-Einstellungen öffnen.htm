Alte VStromFuzzy-Einstellungen öffnen

Sie können wie für alle anderen Verfahren die Einstellungen der Verfahrensparameter in einer Verfahrensparameterdatei speichern und Einstellungen aus einer bestehenden Verfahrensparameterdatei einlesen (Verfahrensparameter in einer Datei speichern und aus Datei einlesen).

Zusätzlich können Sie VStromFuzzy-Einstellungen aus Vorgängerversionen von aus einer VStromFuzzy-Datei *.vsf einlesen.

Stellen Sie sicher, dass das Fenster VStromFuzzy geöffnet ist (Matrixkorrektur mit VStromFuzzy durchführen).

Klicken Sie auf die Schaltfläche Alte Einstellungen öffnen.

Das Fenster Öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Die Einstellungen werden eingelesen.

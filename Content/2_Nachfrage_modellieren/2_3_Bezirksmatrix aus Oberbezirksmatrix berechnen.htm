Bezirksmatrix aus Oberbezirksmatrix berechnen

Beim Erzeugen einer Bezirksmatrix aus einer Oberbezirksmatrix teilen Sie die Matrixwerte jedes Oberbezirks auf die Bezirke auf, die zu diesem Oberbezirk gehören, und schreiben diese in eine vorhandene Bezirksmatrix. Sie können die Werte gleichmäßig oder gewichtet aufteilen (siehe Grundlagen: Kap. 0.0.1 , Seite 1).

Klicken Sie in der Symbolleiste auf das Symbol  Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Matrixdisaggregation aus der Kategorie Matrizen ein (siehe „Verfahrensablauf festlegen und starten“ auf Seite 1).

Klicken Sie in der Zeile des neu eingefügten Verfahrens auf die Schaltfläche in der Spalte Bezugsobjekt(e).

Das Fenster Matrix auswählen öffnet sich.

Wählen Sie in der Liste die Bezirksmatrix, in die Sie die disaggregierten Werte schreiben möchten. Dafür stehen Ihnen zwei Methoden zur Verfügung (siehe „Fenster Matrizen auswählen“ auf Seite 1).

Element Beschreibung

Matrix auswählen nach Nummer

Wählen Sie aus der Liste aller Matrizen anhand der Nummer die gewünschte Matrix aus.

Matrix auswählen nach Eigenschaften

Wählen Sie anhand bestimmter Attribute und deren Werte die Matrix aus.

Die ausgewählte Matrix wird in die Spalte Bezugsobjekt(e) übernommen.

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist, und klicken Sie im Bereich Aktionen auf die Schaltfläche Bearbeiten.

Das Fenster Matrix disaggregieren öffnet sich.

Element

Beschreibung

Matrix wählen

Über die Schaltfläche wählen Sie die Oberbezirksmatrix, die disaggregiert werden soll. Dafür stehen Ihnen zwei Methoden zur Verfügung (siehe Schritt 4)

Disaggregationsgewichte

Zwei Quelle-Ziel-Beziehungseigenschaften, die bei der Disaggregation als Gewichtungsfaktoren wirken. Sie können zum Beispiel eine Matrix oder ein Quell- oder Zielbezirksattribut wählen.

Hinweise

Wenn Sie für beide Gewichtungen den Faktor 1.0 verwenden, werden die Werte gleichmäßig auf die Bezirke des Oberbezirks aufgeteilt.

Wenn der Nenner der Formel null wird, werden die Werte ebenfalls gleichmäßig aufgeteilt.

Führen Sie das Verfahren aus (siehe „Verfahren im Verfahrensablauf ausführen“ auf Seite 1).

Die Werte der Oberbezirksmatrix werden aufgeteilt und das Ergebnis wird in die ausgewählte Bezirksmatrix geschrieben.

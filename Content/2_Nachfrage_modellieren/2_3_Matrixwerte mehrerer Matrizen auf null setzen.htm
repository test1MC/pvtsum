Matrixwerte mehrerer Matrizen auf null setzen

Wählen Sie im Fenster Matrizen im Navigator die gewünschten Matrizen.

Klicken Sie mit der rechten Maustaste auf eine der gewählten Matrizen.

Wählen Sie den Eintrag Initialisieren.

Über das Symbol  Initialisieren in der Symbolleiste des Matrixfensters können Sie alle Matrixwerte im Matrixfenster auf null setzen.

Die Matrixwerte der gewählten Matrizen werden auf null gesetzt.

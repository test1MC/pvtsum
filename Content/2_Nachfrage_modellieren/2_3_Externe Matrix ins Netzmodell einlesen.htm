Externe Matrix ins Netzmodell einlesen

Sie können eine Matrix, die als Datei vorliegt, in das -Netzmodell einlesen. Dabei werden nur die Matrixwerte eingelesen, deren Quelle-Ziel-Beziehungen im Netz vorhanden sind.

Klicken Sie im Fenster Matrizen mit der rechten Maustaste neben den Navigator.

Wählen Sie den Eintrag Externe Matrix ins Netzmodell einfügen.

Das Fenster Externe Matrix ins Netzmodell einfügen öffnet sich.

Element

Beschreibung

Nummer

Frei wählbare Nummer der Matrix

Code

Kurzbezeichnung der Matrix

Name

Langbezeichnung der Matrix

Nachfragematrix für Bezirke

Über diese Option lesen Sie eine Nachfragematrix für Bezirke ein.

Kenngrößenmatrix für Bezirke

Über diese Option lesen Sie eine Kenngrößenmatrix für Bezirke ein.

Nachfragematrix für Oberbezirke

Über diese Option lesen Sie eine Nachfragematrix für Oberbezirke ein.

Kenngrößenmatrix für Oberbezirke

Über diese Option lesen Sie eine Kenngrößenmatrix für Oberbezirke ein.

Datei

Über das Symbol wählen Sie die Matrixdatei aus.

Hinweis

Alternativ können Sie Dateiname und Pfad in das Feld eingeben.

Falls die Matrixdatei Quelle-Ziel-Beziehungen enthält, die nicht im Netz vorhanden sind, öffnet sich eine entsprechende Meldung.

Bestätigen Sie gegebenenfalls mit OK.

Die Matrixdaten werden in eine neue -Matrix eingelesen. Quelle-Ziel-Beziehungen, die in der eingelesenen Matrixdatei nicht vorhanden sind, enthalten den Wert 0. Die Matrix wird geöffnet.

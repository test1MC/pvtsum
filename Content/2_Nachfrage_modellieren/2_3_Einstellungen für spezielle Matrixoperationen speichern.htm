Einstellungen für spezielle Matrixoperationen speichern

Bei den folgenden Matrixoperationen können Sie die aktuellen Einstellungen als Parameterdatei (*.xml) speichern:

  • Hochrechnen (Matrixwerte hochrechnen)
  • Aggregieren (Spalten und Zeilen einer Matrix aggregieren oder löschen)
  • Splitten (Spalten und Zeilen splitten)

Dabei können Sie entscheiden, dass die in der Liste vorgenommenen Einstellungen in eine separate Textdatei *.att ausgelagert werden. Die Parameterdateien können Sie für entsprechende Operationen wieder einlesen.

Stellen Sie sicher, dass das Parameterfenster der gewünschten Operation geöffnet ist.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster Speichern der Parameter für Matrixoperation ... öffnet sich.

Element

Beschreibung

Parameter für Bezirke in separater Attribute-Datei speichern

Ist die Option ausgewählt, werden die in der Liste vorgenommenen Zuordnungen in einer separaten Textdatei (*.att) gespeichert. Die Datei *.xml enthält in diesem Fall einen Verweis auf die Datei *.att.

Ist die Option nicht ausgewählt, werden alle Einstellungen direkt in der Datei *.xml gespeichert.

Dateiname

Falls die Option ausgewählt ist, wählen Sie hier die Datei *.att, in die die Einstellungen gespeichert werden sollen.

Klicken Sie auf die Schaltfläche Weiter.

Das Fenster Parameter für Matrixoperationen speichern öffnet sich.

Wählen Sie die gewünschte Datei aus.

Alternativ können Sie Pfad und Dateiname manuell eingeben.

Klicken Sie auf die Schaltfläche Speichern.

Falls die Option Parameter für Bezirke in separater Attribute-Datei speichern ausgewählt wurde, öffnet sich das Fenster Attribute schreiben.

Wählen Sie in der Auswahlliste Trennzeichen den gewünschten Eintrag.

Sie können Attributsdateien wieder einlesen, wenn Sie als Trennzeichen entweder Semikolon oder Tabulator auswählen.

Die Einstellungen werden gespeichert.

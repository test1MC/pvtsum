Nachfragedaten speichern

Beim Speichern von Nachfragedaten werden Attributwerte mit einer bestimmten Anzahl an Nachkommastellen gespeichert (Einstellungen für Dezimalstellen und Aggregationsfunktionen).

Klicken Sie auf das Symbol  Speichern unter.

Das Fenster Dateiart-Auswahl öffnet sich.

Alternativ zeigen Sie im Menü Datei auf Speichern unter und wählen den Eintrag Nachfragedaten.

Wählen Sie den Eintrag Nachfragedaten.

Das Fenster Nachfragedaten speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster Nachfragedatei schreiben (Benutzerdefinition) öffnet sich.

Schränken Sie bei Bedarf die Menge der zu speichernden Daten ein.

Element

Beschreibung

Berechnete Attribute ausblenden

Ist die Option ausgewählt, werden berechnete Attribute ausgeblendet und nicht gespeichert.

Ist die Option nicht ausgewählt, werden auch berechnete Attribute gespeichert.

Tipp

Das Speichern von berechneten Attributen eignet sich für eine weitere Auswertung, beispielsweise in einer Datenbank.

Hinweis

Berechnete Attribute werden beim Einlesen ignoriert.

Attribute mit Defaultbelegung ausblenden

Ist die Option ausgewählt, werden nur ausgewählte belegte Attribute und deren Werte gespeichert.

Tipp

Mit der Option können Sie die Dateigröße reduzieren.

Ist die Option nicht ausgewählt, werden alle ausgewählten Attribute und deren Werte gespeichert, unabhängig davon, ob der Attributwert dem Standardwert entspricht oder davon abweicht.

Hinweis

Die Standardwerte der Attribute können Sie der Datei Attribute.xls unter ...\Programme\PTV Vision\\Doc entnehmen.

Leere Tabellen ausblenden

Ist die Option ausgewählt, werden nur Tabellen gespeichert, die Einträge enthalten.

Wählen Sie die Datentabellen, die Sie speichern möchten.

Element

Beschreibung

Die Tabelle ist ausgewählt und wird gespeichert.

Die Tabelle ist nicht ausgewählt und wird nicht gespeichert.

Alle Tabellen an

Über die Schaltfläche können Sie alle Tabellen auswählen.

Hinweis

Der Zustand der Attribute in den einzelnen Tabellen bleibt dabei erhalten.

Alle Tabellen aus

Über die Schaltfläche können Sie alle Tabellen bis auf $VERSION ausschalten.

Hinweis

Der Zustand der Attribute in den einzelnen Tabellen bleibt dabei erhalten.

Klicken Sie auf einen Tabellennamen.

Die Attribute der Tabelle werden angezeigt.

Sie können die Attribute mehrerer Tabellen gleichzeitig anzeigen, indem Sie die Taste Strg drücken, diese gedrückt halten und nacheinander auf die gewünschten Tabellennamen klicken.

In Abhängigkeit von den Einstellungen im oberen Bereich des Fensters werden unter Umständen nicht alle Attribute einer Tabelle angezeigt.

Wählen Sie die Attribute, die Sie speichern möchten.

Element

Beschreibung

Das Attribut ist ausgewählt und wird gespeichert.

Das Attribut ist nicht ausgewählt und wird nicht gespeichert.

Das Attribut ist ein Schlüsselattribut und kann nicht ausgeschaltet werden.

Schlüsselattribute sind Attribute, die zur Identifikation der Nachfrageobjekte dienen.

Das Attribut ist ein Pflichtattribut und kann nicht ausgeschaltet werden.

Attributgruppen

Über die Schaltfläche können Sie vordefinierte Gruppen von Attributen in einem separaten Fenster auswählen oder ausschalten (siehe „Gruppen von Attributen für das Speichern auswählen oder ausschalten“ auf Seite 984).

Standard

Alle lesbaren Attribute auswählen

Alternativ können Sie ausgewählte Tabellen und Attribute aus einer Nachfragedatendatei *.dmd einlesen (siehe „Layout aus einer Nachfragedatendatei einlesen“ auf Seite 984)

Das Fenster Speicherort für Matrizen öffnet sich.

Das Fenster öffnet sich nur, wenn Sie die Datentabelle Matrixeinträge gewählt haben.

Wählen Sie für jede Matrix, ob die Matrix in der Nachfragedatendatei oder als separate Matrixdatei gespeichert werden soll.

Element

Beschreibung

Format

(in DMD)

Ist die Option ausgewählt, wird die jeweilige Matrix in der Nachfragedatendatei gespeichert.

Binär, V-Format, O-Format, E-Format, S-Format

Ist eine dieser Optionen ausgewählt, wird die jeweilige Matrix als separate Matrixdatei im ausgewählten Format gespeichert (-spezifische Matrixdateiformate). In der Nachfragedatendatei wird die Matrix nur als Referenz gespeichert.

Dateiname

Dateiname und bei Bedarf Pfad der als separate Matrixdatei zu speichernden Matrix

Über die Schaltfläche rufen Sie das Fenster Matrixdatei speichern auf.

Hinweise

Wenn Sie für eine Matrix ein anderes Format als (in DMD) gewählt haben, müssen Sie einen Dateinamen angeben.

Die Angabe des Pfads ist nur erforderlich, wenn das Verzeichnis, in dem die Datei gespeichert werden soll, von dem für die Dateiart eingestellten Verzeichnis abweicht (Speicherort von Dateien ändern).

Tipp

Dateinamen können Sie kopieren und einfügen.

Die Nachfragedaten werden gemäß der Einstellungen gespeichert.

Gruppen von Attributen für das Speichern auswählen oder ausschalten

Stellen Sie sicher, dass das Fenster Nachfragedatei schreiben (Benutzerdefinition) offen ist (siehe „Nachfragedaten speichern“ auf Seite 981).

Klicken Sie auf die Schaltfläche Attributgruppen.

Das Fenster Gruppenauswahl benutzerdefiniertes Schreiben öffnet sich.

Wählen Sie in der Auswahlliste die vordefinierte Gruppe von Attributen, deren Attribute Sie für das Speichern auswählen oder ausschalten möchten.

Klicken Sie auf die gewünschte Schaltfläche.

Element

Beschreibung

Einschalten

Alle Attribute der ausgewählten Gruppe werden ausgewählt.

Abschalten

Alle Attribute der ausgewählten Gruppe außer Schlüssel- und Pflichtattribute werden für das Speichern ausgeschaltet.

Exklusiv einschalten

Alle Attribute der ausgewählten Gruppe werden ausgewählt und alle Attribute anderer Gruppen außer Schlüssel- und Pflichtattribute werden ausgeschaltet.

Klicken Sie auf die Schaltfläche Schließen.

Die Attribute werden gemäß der Einstellungen ausgewählt oder ausgeschaltet.

Ausgewählte Nachfragedaten als Layout speichern

Sie können die aktuell ausgewählten Tabellen und Attribute als Nachfragedatendatei *.dmd ohne Inhalt speichern und später als Vorgabe wieder einlesen.

Stellen Sie sicher, dass das Fenster Nachfragedatei schreiben (Benutzerdefinition) offen ist (siehe „Nachfragedaten speichern“ auf Seite 981).

Klicken Sie auf die Schaltfläche Layout speichern.

Das Fenster Nachfragedaten speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Die ausgewählten Tabellen und Attribute werden als Datenblöcke ohne Inhalt gespeichert.

Layout aus einer Nachfragedatendatei einlesen

Sie können eine Auswahl von Tabellen und Attributen aus einer Nachfragedatendatei oder aus einer als Layout gespeicherten Nachfragedatendatei ohne Inhalt einlesen.

Stellen Sie sicher, dass das Fenster Nachfragedatei schreiben (Benutzerdefinition) offen ist (siehe „Nachfragedaten speichern“ auf Seite 981).

Klicken Sie auf die Schaltfläche Layout öffnen.

Das Fenster Nachfragedaten öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Die in der Nachfragedatendatei enthaltenen Tabellen und Attribute werden übernommen.

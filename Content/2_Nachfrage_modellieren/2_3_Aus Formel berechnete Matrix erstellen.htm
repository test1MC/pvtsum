Aus Formel berechnete Matrix erstellen

Sie können eine -Matrix erstellen, die aus Werten anderer -Matrizen oder Bezirks-, Oberbezirks- oder Haltestellenbereichsattributen berechnet wird. Eine solche Formelmatrix wird automatisch neu berechnet, sobald sich an den Operanden etwas ändert.

In der formalen Erstellung der Formel werden Sie von unterstützt. Die Syntax wird bei der Eingabe überprüft und Fehler farblich hervorgehoben.

Werte in Formelmatrizen können Sie nicht ändern.

Falls Sie Matrizen oder Attribute löschen, die zur Berechnung einer Formelmatrix verwendet werden, wird die Formelmatrix ebenfalls gelöscht.

Klicken Sie in der Symbolleiste des Fensters Matrizen auf das Symbol  Matrizen einfügen.

Das Fenster Matrizen einfügen öffnet sich.

Alternativ können Sie im Fenster Matrizen mit der rechten Maustaste neben den Navigator klicken und im Kontextmenü den Eintrag Matrizen einfügen wählen. Das Symbol und den Eintrag im Kontextmenü finden Sie auch in der Liste Matrizen (Liste öffnen).

Wählen Sie das Register Matrixattribute.

Wählen Sie im Bereich Datenquellen-Typ die Option Formelmatrix.

Klicken Sie auf die Schaltfläche Formel bearbeiten.

Das Fenster Formel für Matrix <Nummer> bearbeiten öffnet sich.

Geben Sie in das Textfeld die gewünschte Formel für die Berechnung ein oder erstellen Sie die Formel über die Schaltflächen neben dem Textfeld.

Sie können auch die Formel einer anderen Formelmatrix durch Kopieren und Einfügen verwenden.
Achten Sie darauf, dass Ihre Formel keinen direkten oder indirekten Zirkelbezug enthält. Auch Bezüge zwischen Ausgabewerten und Eingabewerten des gleichen Verfahrensschritts im Verfahrensablauf sollten Sie vermeiden, da sie zu unerwünschten Berechnungsergebnissen führen können.

Über die Schaltfläche können Sie einen Operator für die Formel wählen. Die verfügbare Auswahl ist abhängig vom Typ des Operanden, dessen Werte in die Formel einfließen.

Verwendbare Operatoren und ihre Priorität:

Oberste Priorität hat der Klammeroperator, den Sie über das Symbol einfügen können danach kommen in absteigender Priorität die folgenden Gruppen:

1. unärer Operator:

  • ! Negation eines nachfolgenden Terms

2. multiplikative binäre Operatoren:

  • * bildet das Produkt zweier Terme
  • / bildet den Quotienten zweier Terme

3. additive binäre Operatoren:

  • + bildet die Summe zweier Terme
  • - bildet die Differenz zweier Terme

4. relationale binäre Operatoren:

  • > vergleicht zwei Terme bezüglich „größer‟
  • >= vergleicht zwei Terme bezüglich „größer‟ oder „gleich‟
  • <= vergleicht zwei Terme bezüglich „kleiner‟ oder „gleich‟
  • < vergleicht zwei Terme bezüglich „kleiner‟

5. binäre Gleichheitsoperatoren:

  • = prüft die Gleichheit zweier Terme
  • != prüft die Ungleichheit zweier Terme
  • & verknüpft zweier Terme durch logisches Und
  • | verknüpft zwei Terme durch logisches Oder

Binäre Operatoren, die die gleiche Priorität besitzen, werden von links nach rechts (linksassoziativ) ausgewertet.

Beispiel 4/2/2 wird als (4/2)/2 = 1 ausgewertet, nicht als 4/(2/2) = 4

Über die Schaltfläche können Sie eine Funktion für die Formel auswählen (Mit Matrixwerten rechnen).

  • Abs(x) Absolutbetrag von x
  • Abschneiden(x) rundet x auf die nächstkleinere Ganzzahl ab, es wird Floor() an der Cursorposition eingefügt.
  • Exponential(x) bildet die Exponentialfunktion von x,
  • Kehrwert(x) bildet den Kehrwert von x, also 1/x, es wird Reciprocal() an der Cursorposition eingefügt
  • Logarithmus(x) bildet den natürlichen Logarithmus von x
  • Max(x,y) bildet das Maximums von x und y.
  • Min(x,y) bildet das Minimum von x und y
  • NurAktiveBeziehungen(x,y) wirkt sich auf aktive Beziehungen mit dem Wert x aus, für inaktive Beziehungen mit dem Wert y, es wird OnlyActiveRelations(,) an der Cursorposition eingefügt
  • OberesDreieckspiegeln(x) spiegelt das obere Dreieck der Matrix x, es wird Mirror_Upper() an der Cursorposition eingefügt
  • Potenz(x,y) entspricht xy, es wird Pow(,) an der Cursorposition eingefügt
  • Prozent(x,y) entspricht 100*x/y, es wird Percent(,)an der Cursorposition eingefügt
  • Runden(x,p) rundet x auf p Nachkommastellen, es wird der Text Round(,) an der Cursorposition eingefügt
  • Symmetrisieren() bildet in Matrix x aus den Werten unterhalb und oberhalb der Matrixdiagonalen den Mittelwert, es wird Symmetrize() an der Cursorposition eingefügt
  • Transponieren(x) vertauscht die Zeilen und Spaltenwerte einer quadratischen Matrix, es wird Transpose() an der Cursorposition eingefügt
  • UnteresDreieckspiegeln(x) spiegelt das untere Dreieck der Matrix x, es wird Mirror_Lower() an der Cursorposition eingefügt
  • Wenn(b;w;f) bildet eine bedingte Verzweigung, die den Wert w ausgibt, wenn die Bedingung b wahr ist oder den Wert f ausgibt, wenn die Bedingung b falsch ist. Es wird der Text If(,,) an der Cursorposition eingefügt.
  • Wurzel(x) bildet die Quadratwurzel von x, es wird der Text Sqrt() an der Cursorposition eingefügt.

Hinweis

Als Trennzeichen zwischen den Elementen wird das Komma verwendet, wenn Sie in den Benutzereinstellungen als Dezimaltrennzeichen den Punkt eingestellt haben, sonst wird das Semikolon verwendet (siehe „Dezimaltrennzeichen wählen“ auf Seite 1).

Über die Schaltfläche fügen Sie einen Klammerausdruck ohne Inhalt in die Formel ein.

Wählen Sie das Register Matrixmenge.

Element

Beschreibung

Feste Anzahl von Matrizen einfügen

Ist die Option ausgewählt, legen Sie im entsprechenden Feld die Anzahl der Matrizen fest, die Sie einfügen möchten.

Hinweis

Fügen Sie mehr als eine Matrix ein, wird die Matrixnummer automatisch fortlaufend erhöht. Alle eingefügten Matrizen besitzen die gleichen Eigenschaften.

Matrizen über einen Matrixbezug einfügen

Ist die Option ausgewählt, können Sie eine oder mehrere Matrizen erstellen. Für neue Matrizen können Sie die Werte der Matrixattribute über Bezüge, die Sie in als Code eingeben, festlegen. (Fenster Matrizen auswählen).

Neue Matrizen nach dem Einfügen bearbeiten

Ist die Option ausgewählt, werden die eingefügten Matrizen im Matrixeditor geöffnet und Sie können sie bearbeiten.

Bestätigen Sie nochmals mit OK.

Die neue Formelmatrix wird erstellt und berechnet. Sie wird im Fenster Matrizen unter dem Eintrag Formelmatrizen angezeigt.

Wenn Sie eine Formelmatrix als Datei speichern, werden in der Datei nur die aktuellen Werte gespeichert, nicht die Formel.

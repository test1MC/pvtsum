Alle Matrixwerte durch ein Maximum ersetzen

Stellen Sie sicher, dass ein Matrixeditorfenster mit der gewünschten Matrix geöffnet ist (Matrizen öffnen).

Klicken Sie in der Symbolleiste des Matrixfensters auf das Symbol  Maximum bilden.

Das Fenster Operand für ‚Maximum bilden...‘ auf Gesamtmatrix auswählen öffnet sich.

Alternativ können Sie im Menü Matrixeditor den Eintrag Rechenoperationen > Maximum bilden wählen.

Element

Beschreibung

Matrizen auswählen

Ist die Option ausgewählt, können Sie die Matrixwerte mit den Werten einer oder mehrerer anderer Matrizen vergleichen, die im angezeigten Navigator enthalten sind.

Tipp

Der Navigator enthält die gleichen Matrizen wie das Fenster Matrizen (Matrizen im Fenster Matrizen anzeigen).

Matrix auswählen nach Eigenschaften

Ist die Option ausgewählt, können Sie die Matrizen nach bestimmten Eigenschaften von Attributen suchen (siehe „Fenster Matrizen auswählen“ auf Seite 1).

Netzattribut

Ist die Option ausgewählt, können Sie über eine Schaltfläche ein Netzattribut auswählen.

Bezirksattribut

Ist die Option ausgewählt, vergleichen Sie die Matrixwerte mit Bezirksattributwerten.

Quelle

Ist der Eintrag ausgewählt, vergleichen Sie den Matrixwert jeder Quelle-Ziel-Beziehung ij mit dem Attributwert des Quellbezirks i.

Ziel

Ist der Eintrag ausgewählt, vergleichen Sie den Matrixwert jeder Quelle-Ziel-Beziehung ij mit dem Attributwert des Zielbezirks j.

Hinweis

Diese Funktion steht nur für -Matrizen zur Verfügung.

Matrix von Datei laden

Ist die Option ausgewählt, können Sie die Matrixwerte mit den Werten einer externen Matrix vergleichen. Über die Schaltfläche wählen Sie die gewünschte Matrixdatei aus.

Konstante

Ist die Option ausgewählt, können Sie die Matrixwerte mit einer Konstante vergleichen. Den Wert geben Sie in das Eingabefeld ein.

Wenn Sie eine oder mehrere Matrizen für die Rechenoperation auswählen, müssen die Anzahl der Bezirke und die Bezirksnummern der Matrizen nicht mit denen der Matrizen im Matrixfenster identisch sein. Die Rechenoperation wird aber nur mit Matrixwerten durchgeführt, deren Quelle-Ziel-Beziehungen übereinstimmen. Falls die Matrizen nicht die gleichen Quelle-Ziel-Beziehungen besitzen, gilt:

  • Falls in einer für die Rechenoperation ausgewählten Matrix eine Quelle-Ziel-Beziehung nicht vorhanden ist, wird für diese der Matrixwert null verwendet.
  • Ist im Matrixfenster mindestens eine -Matrix geöffnet, werden Quelle-Ziel-Beziehungen, die nicht im Matrixfenster vorhanden sind, ignoriert.
  • Sind im Matrixfenster nur externe Matrizen geöffnet und die für die Rechenoperation ausgewählten Matrizen enthalten Quelle-Ziel-Beziehungen, die im Matrixfenster nicht vorhanden sind, öffnet sich eine Abfrage. Sie können entscheiden, ob die zusätzlichen Quelle-Ziel-Beziehungen zu den Original-Matrizen hinzugefügt werden sollen. Bei einer solchen Matrixaufweitung erhält die Matrix entsprechend neue Spalten und/oder Zeilen (Externe Matrizen während einer Rechenoperation aufweiten).

Das Maximum der verglichenen Matrixwerte wird übernommen.

Histogramm erstellen

Wählen Sie im Fenster Matrizen im Navigator die gewünschten Matrizen.

Klicken Sie mit der rechten Maustaste auf eine der gewählten Matrizen.

Wählen Sie den Eintrag Histogramm erstellen.

Alternativ können Sie in der Symbolleiste des Registers auf das Symbol  Histogramm erstellen klicken. Beide Möglichkeiten finden Sie auch in der Liste Matrizen (Liste öffnen).

Das Fenster Matrixhistogramm (Matrix <Matrixname>) öffnet sich.

Falls Sie mehrere Matrizen ausgewählt haben, wird statt des Matrixnamens eine Information über die Anzahl der ausgewählten Matrizen angezeigt.

Klicken Sie auf das Symbol  Neue Intervalle einfügen.

Alternativ können Sie im Menü Matrixhistogramm den Eintrag Neue Intervalle einfügen wählen.

Das Fenster Intervalle hinzufügen öffnet sich.

Element

Beschreibung

Untere Grenze

Niedrigster Wert des untersten Intervalls

Hinweis

Die untere Grenze ist nicht im Intervall enthalten.

Obere Grenze

Höchster Wert des obersten Intervalls

Hinweis

Die obere Grenze ist im Intervall enthalten.

Zahl Intervalle

Anzahl der Intervalle

Standardmäßig werden die Quelle-Ziel-Beziehungen der gewählten Matrizen entsprechend ihrer eigenen Matrixwerte klassifiziert. Sie können die Klassifizierung jedoch auch nach den Matrixwerten einer anderen Matrix durchführen. Verwenden Sie hierfür das Symbol  Ansichtsoptionen oder das Menü Matrixhistogramm > Ansichtsoptionen (Histogramm nach einer anderen Matrix klassifizieren).

Die Intervalle werden im unteren Teil des Fensters eingefügt.

Alternativ können Sie Intervalle über das Symbol  Intervalle öffnen aus einer Attributsdatei *.att einlesen (Listeninhalte ausgeben und einlesen).

Alternativ können Sie Intervalle aus einem anderen Histogrammfenster übernehmen. Kopieren Sie sie dazu in dem anderen Histogrammfenster über das Symbol  Intervalle in Zwischenablage kopieren und fügen Sie sie über das Symbol  Intervalle aus Zwischenablage einfügen ein.

Bearbeiten Sie die Intervalle bei Bedarf.

Sie können die Intervallgrenzen per Doppelklick direkt bearbeiten. Die Untergrenze eines Intervalls ist nicht Teil des Intervalls, die Obergrenze ist Teil des Intervalls (a < x ≤ b).

Über das Symbol  Markierte Intervalle löschen in der Symbolleiste des Histogrammfensters können Sie markierte Intervalle löschen.

Solange die Klassifikation noch nicht neu berechnet wurde, werden geänderte Intervalle in brauner Schrift angezeigt.

Klicken Sie auf das Symbol  Klassifikation neu berechnen.

Alternativ können Sie im Menü Matrixhistogramm den Eintrag Klassifikation neu berechnen wählen.

Das Histogramm wird im oberen Bereich des Fensters angezeigt. In der Liste werden die berechneten Werte angezeigt. Die Spalte LetzterAnteil enthält die Werte aus der letzten Berechnung.

Über das Symbol  Matrizen auswählen können Sie die in diesem Fenster angezeigten Matrizen verwalten (Im Matrix- oder Histogrammfenster angezeigte Matrizen verwalten).

Die Einstellungen des Histogramms können Sie als Layoutdatei speichern (Layout eines Matrixeditor- oder Histogrammfensters speichern und einlesen). Die Intervalle können Sie als Datei *.att speichern (Intervalle als Attributsdatei speichern).

Wenn Sie in den ausgewählten Matrizen Matrixwerte ändern, wird das Histogramm automatisch angepasst.

Wenn Sie nach dem Berechnen des Histogramms Intervalle verändern, wird die Anzeige nicht automatisch angepasst. Die Säulen werden grau angezeigt, die berechneten Werte braun. Zum Aktualisieren des Histogramms müssen Sie wieder auf das Symbol  Klassifikation neu berechnen klicken.

Nachfragemodell einfügen

Wählen Sie im Menü Nachfrage den Eintrag Nachfragemodelle.

Das Fenster Nachfragemodelle öffnet sich.

Wählen Sie das Register Basis.

Klicken Sie auf die Schaltfläche Einfügen.

Ein neues Nachfragemodell wird eingefügt.

Nehmen Sie die gewünschten Einstellungen vor (Attribute und Optionen von Nachfragemodellen).

Alternativ können Sie ein vordefiniertes Nachfragemodell und die zugehörigen Daten aus einer Nachfragedatendatei einlesen (Vordefinierte Nachfragemodelle *.dmd einlesen).

Das Nachfragemodell wird gespeichert.

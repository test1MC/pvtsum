Kombinierte VISEM-Verkehrsverteilung und Moduswahl berechnen

Bei der kombinierten VISEM-Verkehrsverteilung und Moduswahl werden für jeden Übergang einer Aktivitätenkette simultan eine Gesamtnachfragematrix und eine Nachfragematrix pro Modus berechnet (siehe Grundlagen: Kap. , Seite 1).

Stellen Sie sicher, dass das Aufkommen der einzelnen Bezirke pro Nachfrageschicht als Bezirksattribut Heimataufkommen vorliegt (siehe „VISEM-Verkehrserzeugung berechnen“ auf Seite 1 und „Aufkommen von VISEM-Nachfrageschichten ändern“ auf Seite 1).

Stellen Sie sicher, dass jeder Personengruppe, für die die VISEM-Verkehrsverteilung und Moduswahl berechnet werden soll, eine Nachfragematrix zugeordnet ist, in die die Ergebnisse gespeichert werden (siehe „Nachfragematrizen für die VISEM-Verkehrsverteilung und Moduswahl erzeugen“ auf Seite 1).

Klicken Sie in der Symbolleiste auf das Symbol Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren VISEM-kombinierte Verkehrsverteilung/Moduswahl aus der Kategorie Nachfragemodell ein (siehe „Verfahrensablauf festlegen und starten“ auf Seite 1).

Klicken Sie in der Zeile des neu eingefügten Verfahrens auf die Schaltfläche in der Spalte Bezugsobjekt(e).

Das Fenster Nachfrageschichten Auswahl VISEM öffnet sich.

Wählen Sie in der Auswahlliste Nachfragemodell auswählen das Nachfragemodell, zu dem die Nachfrageschichten gehören.

Wählen Sie in der Auswahlliste Nachfrageschichten die Nachfrageschichten, für die Sie die kombinierte VISEM-Verkehrsverteilung und Moduswahl rechnen möchten.

Sie können mehrere Nachfrageschichten auswählen, indem Sie die Taste Strg drücken, diese gedrückt halten und auf die gewünschten Einträge klicken.

Alternativ können Sie für bestimmte Personengruppen und Aktivitätenpaare die zugehörigen Nachfrageschichten vorauswählen. Wählen Sie dazu in den Auswahllisten Personengruppen und Aktivitätenpaare die gewünschten Nachfrageobjekte und klicken Sie auf die Schaltfläche Vorauswahl übernehmen.

Die ausgewählten Nachfrageschichten werden im Eingabefeld NSchichten und in die Spalte Bezugsobjekt(e) übernommen.

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist, und klicken Sie im Bereich Aktionen auf die Schaltfläche Bearbeiten.

Das Fenster Parameter kombinierte VISEM-Verkehrsverteilung/Moduswahl öffnet sich.

Fehlen innerhalb einer Zeile Eingaben, die für die Berechnung notwendig sind, markiert die Felder rot. Wählen Sie beispielsweise aus, dass eine Nutzenmatrix gespeichert werden soll, müssen Sie eine Ausgabematrix angeben. Andernfalls wird das Feld in der Spalte Ausgabematrix Nutzen rot markiert.

Über diese Schaltfläche können Sie die Tabelle mit allen gewünschten Ausgabematrizen erstellen. Markieren Sie die gewünschten Zeilen in der Tabelle und klicken Sie auf das Symbol. Wählen Sie aus den folgenden Optionen:

Getrennte Matrizen pro Personengruppe

Für jede Personengruppe wird eine eigene Zeile erstellt, die Einträge in den Spalten Modi, Quellaktivitäten und Zielaktivitäten werden übernommen.

Getrennte Matrizen pro Modus

Für jeden Modus wird eine eigene Zeile erstellt, die Einträge in den Spalten Personengruppe, Quellaktivitäten und Zielaktivitäten werden übernommen.

Getrennte Matrizen pro Quellaktivität

Für jede Quellaktivität wird eine eigene Zeile erstellt, die Einträge in den Spalten Modus, Personengruppe und Zielaktivitäten werden übernommen.

Getrennte Matrizen pro Zielaktivität

Für jede Zielaktivität wird eine eigene Zeile erstellt, die Einträge in den Spalten Modi, Personengruppe und Quellaktivitäten werden übernommen.

Tipp

Sie können beispielsweise die Tabelle sehr komfortabel aufbauen, indem Sie in der leeren Tabelle auf das Symbol klicken und eine Standardzeile als Ausgangspunkt einfügen. In dieser Standardzeile sind alle Personengruppen, Modi, Quellaktivitäten und Zielaktivitäten vorhanden. Diese Standardzeile können Sie mit den genannten Optionen nach Bedarf zu der benötigten Tabelle ausbauen.

Fehlende Matrizen für markierte Ausgabeelemente einfügen

Führen Sie das Verfahren aus (siehe „Verfahren im Verfahrensablauf ausführen“ auf Seite 1).

Die kombinierte VISEM-Verteilung und Moduswahl wird für das eingestellte Zeitintervall berechnet. Die Berechnungsergebnisse der Verkehrsverteilung werden in je einer Nachfragematrix pro Personengruppe und Zielaktivität gespeichert und die der Moduswahl in je einer Nachfragematrix pro Personengruppe und Modus (siehe „Nachfragematrizen für die VISEM-Verkehrsverteilung und Moduswahl erzeugen“ auf Seite 1).

Sie können die Nachfragematrizen anzeigen und ändern (siehe „Matrizen verwalten, anzeigen und analysieren“ auf Seite 1).

Wenn Sie die Nachfragematrizen für mehrere Zeitintervalle berechnen möchten, müssen Sie die kombinierte VISEM-Verkehrsverteilung und Moduswahl wiederholt ausführen und verschiedene Nachfragematrizen für das Speichern der Berechnungsergebnisse je Zeitintervall einfügen.

Teilmatrix speichern

Sie können eine Matrix filtern oder dynamisch aggregieren und anschließend nur die Werte speichern, die aktuell angezeigt werden.

Filtern Sie die Matrix oder stellen Sie die Werte aggregiert dar.

  • Filtern (Matrixdaten filtern)
  • Dynamisch aggregieren (Matrixwerte aggregiert darstellen)

Klicken Sie in der Symbolleiste des Matrixeditorfensters auf das Symbol  Speichern unter.

Eine Meldung öffnet sich. Sie entscheiden, ob Sie nur die aktuell angezeigte Teilmatrix oder die gesamte Matrix speichern möchten.

Klicken Sie auf die Schaltfläche Ansicht.

Das Fenster Matrix speichern öffnet sich.

Fahren Sie fort wie beim normalen Speichern einer Matrix.

Die Teilmatrix wird gespeichert. Bei aggregierten Matrizen werden Name und Werte des Aggregationsattributs mit ausgegeben.

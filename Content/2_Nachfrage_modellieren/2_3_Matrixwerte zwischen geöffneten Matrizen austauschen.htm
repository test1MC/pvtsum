Matrixwerte zwischen geöffneten Matrizen austauschen

Sie können Einzelwerte, Matrixbereiche, Spalten oder Zeilen in die Zwischenablage kopieren und in eine andere Matrix oder an eine andere Stelle der gleichen Matrix einfügen.

Stellen Sie sicher, dass ein Matrixeditorfenster mit den gewünschten Matrizen geöffnet ist (Matrizen öffnen).

Markieren Sie die gewünschten Matrixwerte (Matrixwerte markieren).

Die markierten Zellen müssen direkt aneinander angrenzen.

Drücken Sie die Tasten Strg + C.

Alternativ können Sie im Kontextmenü den Eintrag Zellen kopieren wählen.

Die markierten Werte werden in die Zwischenablage kopiert.

Öffnen Sie bei Bedarf die Matrix, in die Sie die kopierten Werte einfügen möchten (Matrizen öffnen).

Markieren Sie die gewünschte Einfügeposition in der Matrix.

Die relative Lage der kopierten Matrixwerte in Zeilen oder Spalten wird beim Einfügen übernommen, d. h. Sie können beim Kopieren mehrerer Zellen einfach die linke obere Zelle des gewünschten Einfügebereichs markieren.

Drücken Sie die Tasten Strg + V.

Alternativ können Sie im Kontextmenü den Eintrag Zellen einfügen wählen.

Die kopierten Werte werden in die entsprechenden Zellen übernommen, unabhängig von der Quelle-Ziel-Beziehung der Zellen.

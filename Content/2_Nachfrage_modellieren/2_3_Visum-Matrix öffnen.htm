-Matrix öffnen

Sie können eine -Matrix, also eine Matrix, die in der Version enthalten ist, zur Bearbeitung öffnen.

  • Doppelklicken Sie im Fenster Matrizen im Navigator auf die gewünschte Matrix.

Alternativ können Sie die Matrix auswählen und in der Symbolleiste des Fensters auf das Symbol  Bearbeiten klicken oder im Kontextmenü den Eintrag Bearbeiten wählen. Beide Möglichkeiten finden Sie auch in der Liste Matrizen (Liste öffnen).

Ein Matrixeditorfenster mit der Matrix öffnet sich (Das Matrixeditorfenster).

Über das Symbol  Matrizen auswählen können Sie die in diesem Fenster angezeigten Matrizen verwalten (Im Matrix- oder Histogrammfenster angezeigte Matrizen verwalten).

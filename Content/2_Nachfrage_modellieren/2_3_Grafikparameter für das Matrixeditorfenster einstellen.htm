Grafikparameter für das Matrixeditorfenster einstellen

Ähnlich wie in der Netzdarstellung, können Sie für jedes Matrixeditorfenster Grafikparameter einstellen, um es nach Ihren Wünschen zu gestalten. Beispielsweise können Sie Zellen klassifiziert darstellen oder die Diagonale hervorheben.

Im Matrixeditorfenster können Sie folgende Einstellungen vornehmen:

  • Schriftfarbe der Matrixwerte festlegen
  • Hintergrundfarbe der Zellen festlegen

Im Wesentlichen stehen Ihnen folgende Möglichkeiten zur Verfügung:

  • Sie können aktive Quelle-Ziel-Beziehungen klassifiziert darstellen. Für die Klassifikation von -Matrizen stehen Ihnen alle (Ober-)Bezirksbeziehungsattribute zur Verfügung. Für externe Matrizen können Sie nur nach dem Matrixwert klassifizieren.

Die Intervalle für die Klassifikation können Sie über die Zwischenablage einfach mit den Intervallen in einem Histogrammfenster austauschen.
  • Sie können für -Matrizen passive Quelle-Ziel-Beziehungen in einer anderen Farbe darstellen.
  • Sie können die Diagonale in einer anderen Farbe darstellen.

Die Grafikparameter können Sie in einer Datei *.gpm speichern und wieder verwenden (Grafikparameter des Matrixfensters speichern und lesen).

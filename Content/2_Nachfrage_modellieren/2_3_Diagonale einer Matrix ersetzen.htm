Diagonale einer Matrix ersetzen

Sie können die Werte der Diagonale einer Matrix mit anderen Werten belegen. Die Diagonale umfasst alle Zellen, bei denen die Zeilennummer der Spaltennummer entspricht. Sie können die Werte entweder durch eine Konstante ersetzen oder durch die Werte einer anderen Matrix oder eines Bezirksattributs. Die Matrixwerte aller anderen Zellen bleiben unverändert Siehe "Diagonale kopieren, einfügen und setzen".

Mit den Tasten Alt + Bild auf / Bild ab können Sie diagonal durch die Matrixwerte scrollen.

Stellen Sie sicher, dass die gewünschte Matrix geöffnet ist (Matrizen öffnen).

Klicken Sie in der Symbolleiste des Matrixfensters auf das Symbol  Diagonale setzen.

Das Fenster Operand für ‚Diagonale setzen...‘ auf Gesamtmatrix auswählen öffnet sich.

Alternativ können Sie im Menü Matrixeditor den Eintrag Rechenoperationen > Diagonale setzen wählen.

Element

Beschreibung

Matrix auswählen

Ist die Option ausgewählt, können Sie die Matrixwerte in der Diagonalen durch die Werte einer Matrix ersetzen, die im Navigator enthalten ist.

Tipp

Der Navigator enthält die gleichen Matrizen wie das Fenster Matrizen (Matrizen im Fenster Matrizen anzeigen).

Bezirksattribut

Ist die Option ausgewählt, ersetzen Sie die Matrixwerte durch Bezirksattributwerte.

Hinweis

Diese Funktion steht nur für -Matrizen zur Verfügung.

Matrix von Datei laden

Ist die Option ausgewählt, ersetzen Sie die Matrixwerte durch die Werte einer externen Matrix. Über die Schaltfläche wählen Sie die gewünschte Matrixdatei aus.

Konstante

Ist die Option ausgewählt, ersetzen Sie die Matrixwerte durch eine Konstante. Den Wert geben Sie in das Eingabefeld ein.

Hinweis

Wenn Sie als konstanten Wert für die Diagonale null eingeben, entspricht die Ergebnismatrix der Ausgangsmatrix, nur ohne Binnenverkehr innerhalb der Bezirke.

Die Matrixwerte in der Diagonale werden ersetzt.

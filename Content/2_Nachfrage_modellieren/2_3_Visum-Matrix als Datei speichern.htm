-Matrix als Datei speichern

Wenn Sie eine Formelmatrix als Datei speichern, werden dort nur die aktuellen Matrixwerte gespeichert, nicht die Formel.

Klicken Sie im Fenster Matrizen mit der rechten Maustaste auf die gewünschte Matrix.

Wählen Sie den Eintrag Schreiben auf Datei.

Das Fenster Matrix speichern öffnet sich.

Alternativ können Sie die Matrix öffnen und in der Symbolleiste des Matrixeditorfensters auf das Symbol  Schreiben auf Datei klicken oder das Menü Matrixeditor > Schreiben auf Datei wählen. Einen entsprechenden Eintrag im Kontextmenü finden Sie auch in der Liste Matrizen (siehe „Liste öffnen“ auf Seite 1).

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Wählen Sie bei Bedarf den Dateityp aus.

Die verfügbaren Dateitypen können Sie über das Menü Datei > Projektverzeichnisse > Projektverzeichnisse bearbeiten definieren und verwalten (siehe „Speicherort von Dateien ändern“ auf Seite 1).

Wenn Sie den Dateityp Matrix (*.*) wählen, müssen Sie zusätzlich eine Erweiterung für die Matrixdatei eingeben.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster Matrix <Matrixname> schreiben öffnet sich.

Beschreibung

Format

Über die Auswahlliste können Sie das Ausgabeformat für die Matrix wählen.

Weitere Informationen zu den Formaten finden Sie im Anhang (siehe „Matrixformate“ auf Seite 1).

Matrix

Wenn Sie eine Kenngrößenmatrix zum Speichern ausgewählt haben, enthält die Auswahlliste nur den Eintrag Kenngrößenmatrix.

Wenn Sie eine Nachfragematrix zum Speichern ausgewählt haben, können Sie über die Auswahlliste wählen, ob Sie eine Gesamt- oder eine Teilmatrix speichern möchten (siehe „Beispiel für Nachfragematrix mit Spinnen- und Restmatrix“ auf Seite 1).

Nachfragematrix

Gesamtnachfrage je Nachfragesegment, bei Bedarf mittels Beziehungsfilter auf eine Teilmatrix reduziert

Spinnenmatrix (Teilmatrix)

Gefilterte Fahrten

Hinweis

Dieser Eintrag steht nur bei aktiver Spinnenauswertung zur Verfügung (siehe „Spinnen anzeigen“ auf Seite 1).

Umlegungsmatrix (Teilmatrix)

Umgelegte Fahrten (≤ Gesamtnachfrage) je Nachfragesegment

Hinweis

Dieser Eintrag steht nur zur Verfügung, wenn Sie eine Umlegung gerechnet haben.

Tipp

Alternativ können Sie Teilmatrizen während des Verfahrensablaufs speichern, wenn zuvor eine Umlegung gerechnet wurde (siehe „Teilmatrizen während des Verfahrensablaufs speichern“ auf Seite 1).

Beziehungsfilter berücksichtigen

Ist die Option ausgewählt, werden nur die aktiven Beziehungen in die Matrix gespeichert (siehe „Sonderfall: Beziehungsfilter“ auf Seite 1).

Ist die Option nicht ausgewählt, werden alle Beziehungen gespeichert.

Hinweise

Für Spinnenmatrizen wird der Beziehungsfilter immer berücksichtigt, die Option ist daher nicht verfügbar.

Für andere Matrixarten ist die Option nur verfügbar, wenn der Beziehungsfilter gesetzt ist.

Random Round

Über die Option können Sie einstellen, wie Werte beim Speichern gerundet werden sollen, wenn die als Matrixattribut festgelegte Anzahl Dezimalstellen kleiner ist als die tatsächliche Anzahl Dezimalstellen (siehe „Attributwerte von Matrizen anzeigen und bearbeiten“ auf Seite 1).

 Ist die Option ausgewählt, werden die Werte zufällig gerundet, d. h. in Abhängigkeit vom zu rundenden Anteil. Die Summe der Matrixwerte bleibt dabei konstant.

Beispiel

0,3 wird mit 30% Wahrscheinlichkeit auf 1 aufgerundet und mit 70% Wahrscheinlichkeit auf 0 abgerundet.

Ist die Option nicht ausgewählt, werden die Werte arithmetisch gerundet, d. h. mit einem Dezimalwert bis inklusive 4 wird abgerundet und mit einem Dezimalwert ab 5 aufgerundet. Dies gilt auch für negative Zahlen.

Beispiel

1,2 wird auf 1 abgerundet; 4,5 wird auf 5 und 5,5 auf 6 aufgerundet.

Restmatrix

Ist die Option ausgewählt, werden die verbleibenden Nachfragewerte gespeichert, die nicht in der aktuell eingestellten Teilmatrix enthalten sind.

Ist die Option nicht ausgewählt, werden die Nachfragewerte für die aktuell eingestellte Teilmatrix gespeichert.

Hinweise

Die Option ist nur verfügbar, wenn Sie in der Auswahlliste Matrix eine Teilmatrix gewählt haben.

Nach einer Gleichgewichtsumlegung LUCE ist der Wert der Restmatrix immer null.

Die Matrixwerte werden in der angegebenen Matrixdatei gespeichert. Im Matrixeditorfenster bleibt weiterhin die -Matrix geöffnet.

Beziehungen zwischen bestimmten Bezirkstypen andere Standardganglinie zuweisen

Sie können die Standardganglinie nur ändern, wenn der Nachfrageganglinie eine prozentuale Standardganglinie zugewiesen ist (Attribute und Optionen von Standardganglinien).

Wählen Sie im Menü Nachfrage den Eintrag Nachfragedaten.

Das Fenster Nachfragedaten öffnet sich.

Wählen Sie das Register Nachfrageganglinien.

Wählen Sie in der Liste die Nachfrageganglinie aus, deren Standardganglinie Sie zwischen bestimmten Bezirkstypen ändern möchten.

Klicken Sie auf die Schaltfläche Bearbeiten.

Das Fenster Ganglinien pro Bezirkstyprelation bearbeiten öffnet sich.

Klicken Sie auf die Schaltfläche Einfügen.

Das Fenster Bezirkstypganglinie einfügen öffnet sich.

Element

Beschreibung

Von Typ

Bezirkstyp des Quellbezirks

Nach Typ

Bezirkstyp des Zielbezirks

Standardganglinie

Über die Auswahlliste können Sie für die jeweilige Beziehung eine andere als die ursprünglich gewählte prozentuale Standardganglinie wählen.

In der Liste wird eine neue Zeile eingefügt. Sie zeigt die gewünschte Beziehung und die ihr zugeordnete Standardganglinie.

Fügen Sie bei Bedarf weitere Beziehungen ein.

Ändern Sie bei Bedarf die Standardganglinie einer in der Liste vorhandenen Beziehung über die jeweilige Auswahlliste Standardganglinie.

Wählen Sie bei Bedarf in der Liste die Zeile aus, die Sie löschen möchten.

Klicken Sie auf die Schaltfläche Löschen.

Die Zeile wird gelöscht.

Die Änderungen werden übernommen.

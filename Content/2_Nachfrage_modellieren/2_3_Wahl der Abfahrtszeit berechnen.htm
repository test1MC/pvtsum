Wahl der Abfahrtszeit berechnen

Die Wahl der Abfahrtszeit ist eine weitere optionale Wahlstufe im Anschluss an die Moduswahl. Je Nachfragesegment wird die Gesamtnachfrage gemäß zeitintervall-spezifischer Nutzenmatrizen über die Zeit verteilt. Hieraus resultiert eine Nachfragematrix mit einem Nachfragezeitprofil, das Sie sofort in ÖV-Umlegungen oder in dynamischen IV-Umlegungen verwenden können (siehe Grundlagen: Kap. , Seite 1).

Nachfrageschichten sind für dieses Verfahren nicht erforderlich, d. h. Sie können es auch unabhängig von den anderen Stufen des Standard-4-Stufen-Modells ausführen.

Klicken Sie in der Symbolleiste auf das Symbol  Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Wahl der Abfahrtszeit aus der Kategorie Nachfragemodell ein (siehe „Verfahrensablauf festlegen und starten“ auf Seite 1).

Klicken Sie in der Zeile des eingefügten Verfahrens auf die Schaltfläche in der Spalte Bezugsobjekt(e).

Das Fenster Matrix auswählen öffnet sich.

Wählen Sie die Matrix, deren Werte Sie zum Berechnen der Nachfrage verwenden möchten. Dafür stehen Ihnen zwei Methoden zur Verfügung (siehe „Fenster Matrizen auswählen“ auf Seite 1).

Element Beschreibung

Matrix auswählen nach Nummer

Wählen Sie aus der Liste aller Matrizen anhand der Nummer die gewünschte Matrix aus.

Matrix auswählen nach Eigenschaften

Wählen Sie anhand bestimmter Attribute und deren Werte die Matrix aus.

Wählen Sie die gewünschte Nachfragematrix.

Die ausgewählte Matrix wird in die Spalte Bezugsobjekt(e) übernommen.

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist. Klicken Sie im Bereich Aktionen auf die Schaltfläche Bearbeiten.

Das Fenster Parameter für die Wahl der Abfahrtszeit öffnet sich.

Element

Beschreibung

Alle Intervall-Werte vor der Berechnung auf 0 setzen

Ist die Option ausgewählt, werden alle Werte der Zeitintervalle, die der Standardganglinie zugeordnet sind, vor der Berechnung auf 0 gesetzt.

Ist die Option nicht ausgewählt, werden folgende Werte zum Gewicht des ausgewählten Zeitintervalls addiert (siehe „Ganglinien verwalten“ auf Seite 1).

  • Bei prozentualen Ganglinien wird die Summe der Nachfragematrizen je Zeitintervall zum bestehenden Gewicht des Zeitintervalls addiert.
  • Bei Ganglinien von Matrixnummern wird die unter Verwendung des Zeitwahlmodells berechnete Nachfragematrix zu der Matrix addiert, die dem jeweiligen Zeitintervall zugeordnet ist.

Zeitwahlmodell

Über die Schaltfläche können Sie eine Nutzenfunktion in einem separaten Fenster einstellen, mit der die Werte, die sich aus der Kombination ergeben, transformiert werden (siehe „Parameter der Nutzenfunktion für die Wahl der Abfahrtszeit einstellen“ auf Seite 3).

Auswahl einer Standardganglinie

Über die Auswahlliste können Sie die Standardganglinie wählen, auf deren Zeitintervalle die als Bezugsobjekt ausgewählte Nachfragematrix aufgeteilt wird (siehe „Ganglinien verwalten“ auf Seite 1).

  • Für prozentuale Ganglinien werden die Intervallgewichte (Matrixsummen) neu berechnet. Vorläufige Nachfragematrizen werden für jedes Zeitintervall mit Hilfe des Zeitwahlmodells berechnet. Die Matrixsumme einer solchen Nachfragematrix ergibt die Gewichtung für das jeweilige Zeitintervall.
  • Für Ganglinien von Matrixnummern werden die Werte der Intervall-Nachfragematrizen je Zeitintervall neu berechnet.

Zeitintervall

Über die Auswahlliste können Sie das gewünschte Zeitintervall der Standardganglinie auswählen (siehe „Zeitintervalle für Standardganglinien einfügen“ auf Seite 1).

Ergebnis-Nachfragematrix

Verweis auf die Nachfragematrix, die dem jeweiligen Standardganglinien-Zeitintervall zugeordnet ist (siehe „Zeitintervalle für Standardganglinien einfügen“ auf Seite 1).

Hinweis

Die zugeordnete Ergebnismatrix wird nur für Ganglinien von Matrixnummern angezeigt.

Nutzen U

Linearkombination von Matrizen, Attributsvektoren und/oder Konstanten je Zeitintervall (siehe „Matrizen und Attributsvektoren im Matrixfenster kombinieren“ auf Seite 1)

Hinweis

Das Ergebnis der Kombination wird als Nutzenmatrix interpretiert.

Einfügen

Löschen

Erste Spalte

Wenn die Liste mehr als einen Eintrag enthält, wird durch ein Plus-Zeichen angezeigt, dass die Einträge zeilenweise addiert werden.

Matrix / Attribut /Konstante

Über die Schaltfläche öffnet sich das Fenster Operand auswählen, in dem Sie eine -Kenngrößenmatrix, ein Bezirksattribut oder eine Konstante wählen können. Die Werte des gewählten Objekts können Sie mit einer Funktion transformieren (Spalte Transformation). Außerdem können Sie sie mit einem Faktor multiplizieren (Spalte Koeffizient).

Transformation

Über die Auswahlliste können Sie eine Funktion wählen, mit der die in der Spalte Matrix / Attribut /Konstante gewählten Werte umgerechnet werden (siehe „Matrixwerte bearbeiten“ auf Seite 1).

Koeffizient

Faktor, mit dem die Werte elementweise multipliziert werden

Führen Sie das Verfahren aus (siehe „Verfahren im Verfahrensablauf ausführen“ auf Seite 1).

Die Wahl der Abfahrtszeit wird berechnet, die Berechnungsergebnisse werden für jedes Zeitintervall in die Ergebnismatrix (Ganglinie von Matrixnummern) oder in die Spalte Gewicht des Fensters Ganglinie bearbeiten (Prozentuale Ganglinie) gespeichert (siehe „Zeitintervalle für Standardganglinien einfügen“ auf Seite 1).

Hintergrundkarte von Kartendiensten im Internet statisch einbinden

Stellen Sie sicher, dass für das geöffnete Netz ein Koordinatensystem eingestellt ist (Koordinatensystem wählen).

Stellen Sie sicher, dass der Teil des Netzes im Fenster Netzeditor angezeigt wird, für den Sie Internetkarten herunterladen möchten.

Die Internetkarten werden nur für den im Fenster Netzeditor sichtbaren Ausschnitt des Netzes heruntergeladen.

Klicken Sie im Fenster Netz auf das Symbol  Modus Einfügen.

Klicken Sie auf die Schaltfläche Hintergründe.

Klicken Sie in die Netzdarstellung.

Wenn für das Netz keine Projektion eingestellt ist, wird ein Hinweis angezeigt, dass keine Hintergründe aus Online-Quellen eingefügt werden können. Sie können in diesem Fall nur Hintergründe aus Datei einlesen (Hintergrund von Kartendiensten aus dem Internet einfügen).

Das Fenster Hintergrund einfügen öffnet sich.

Wählen Sie die Option Internetkarten aus.

Wählen Sie in der Auswahlliste das gewünschte Kartenmaterial aus.

Hier werden die Einträge angezeigt, die in den Voreinstellungen definiert sind (Voreinstellungen für das Einfügen von Hintergrundkarten vornehmen). Zusätzliche, benutzerdefinierte Kartendienste sind möglich.

Element

Beschreibung

OpenStreetMap (Mapnik)

Ist die Option ausgewählt, laden Sie OpenStreetMap-Karten herunter, die mit Mapnik gezeichnet wurden.

OpenStreetMap (CycleMap)

Ist die Option ausgewählt, laden Sie OpenStreetMap-Karten herunter, in denen zum Beispiel Fahrradwege und andere für Fahrradfahrer interessante Punkte markiert sind.

Bing Maps (Luftbild)

Ist die Option ausgewählt, laden Sie Luftbilder von Bing Maps herunter.

Stellen Sie mit dem Schieberegler die gewünschte Auflösung ein.

Mit der Auflösung legen Sie den Detaillierungsgrad des Kartenmaterials fest, das vom Kartendienst abgerufen werden soll. Je detaillierter Sie den Hintergrund haben möchten, desto mehr Kacheln werden heruntergeladen. Wenn Sie auf die Schaltfläche Vorschau klicken, werden die Karten angezeigt, ohne dass sich das Fenster Hintergrund einfügen schließt.

Bestätigen Sie den Hinweis auf die Lizenzbestimmungen des Kartendienstes, wenn Sie frei zugängliche Karten herunterladen möchten.

Die Internetkarten werden heruntergeladen und in die Netzdarstellung als Hintergrund eingefügt.

Die Hintergrundkarten werden für jeden benutzerdefinierten Kartendienst separat in einem Ordner im Verzeichnis ..\Benutzer\<Benutzer>\Eigene Dokumente\InternetMaps abgespeichert.

Grafikparameter für GIS-Objekte einstellen

Wenn Sie mit einer Personal Geodatabase verbunden sind, können Sie folgende Grafikeinstellungen für die Darstellung und Beschriftung der GIS-Objekte im Netz vornehmen.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag GIS-Objekte.

Element

Beschreibung

Layer GIS zeichnen

Ist die Option ausgewählt, werden GIS-Objekte im Netz dargestellt.

GIS-Kategorie

Hier werden alle im Netz vorhandenen GIS-Kategorien aufgelistet.

Tipp

Sie können mit Hilfe der Schaltflächen und die Reihenfolge der gezeichneten Layer ändern, indem Sie den gewünschten Layer in der Liste markieren. Der oberste Layer wird zuerst gezeichnet.

Zeichnen

Sie können die Darstellung der Kategorien einzeln ein- oder ausschalten, indem Sie das jeweilige Element über das Kontrollkästchen auswählen.

Nehmen Sie auf den einzelnen Seiten der gewünschten GIS-Kategorien weitere Einstellungen vor.

Die Einstellungen sind identisch mit jenen für aktive Punktobjekte (Punktobjekte im Netzfenster darstellen).

Die GIS-Objekte werden gemäß den Einstellungen dargestellt.

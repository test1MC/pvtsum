Grafikparameter für GPS-Signal einstellen

Sie können die grafische Darstellung des GPS-Signals wie folgt festlegen.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag GPS-Tracking.

Element

Beschreibung

GPS-Tracking zeichnen

Ist die Option ausgewählt, wird ein GPS-Signal gezeichnet.

 

Hier können Sie festlegen, wie das GPS-Signal dargestellt wird.

In der Auswahlliste können Sie eines der folgenden Symbole auswählen:

  • Kreis
  • Quadrat
  • Dreieck
  • offener Pfeil
  • Kreuz
  • Haltestelle deutsch
  • Haltestelle englisch
  • POI
  • LSA
  • Stoppschild
  • geregelte Vorfahrt (Stopp oder Vorfahrt achten)
  • Kreisverkehr

Größe

Größe in mm

Farbe

Sie können die Farbe für das GPS-Signal in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

Zeichen

Wenn Sie Kreis oder Quadrat auswählen, können Sie über das Kontrollkästchen ein Eingabefeld aktivieren. In das Eingabefeld können Sie ein Zeichen eingeben, das innerhalb des Kreises oder Quadrates dargestellt wird. Sie können eine Farbe für das Zeichen in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

Das GPS-Signal wird gemäß den Einstellungen dargestellt.

Hintergrundkarte von Kartendiensten aus dem Internet dynamisch einbinden

Der Grafiklayer Hintergrundkarte bindet Karten- oder Bildmaterial von Kartendiensten im Internet dynamisch in den Hintergrund Ihres Netzes ein. Das Kartenmaterial wird immer wieder neu geladen, wenn Sie den jeweiligen Netzausschnitt verändern oder das Netz erneut öffnen. verwendet immer das neuste zur Verfügung stehende Kartenmaterial. Es wird dafür eine dauerhafte Internetverbindung benötigt, denn die Kartendaten werden nicht gespeichert. Das hat verschiedene Vorteile:

  • Es stehen immer die aktuellsten Karten des Anbieters zur Verfügung.
  • Sie können Projekte zwischen Computern austauschen, ohne dass Pfadangaben für gespeicherte Karten berücksichtigt werden müssen.
  • Sie benötigen keinen Speicherplatz zum Vorhalten von Karten auf Ihrem Computer.

Stellen Sie sicher, dass für das geöffnete Netz ein Koordinatensystem eingestellt ist (Koordinatensystem wählen).

Klicken Sie im Menü Grafik auf den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag Hintergrundkarte.

Element

Beschreibung

Layer Hintergrundkarte zeichnen

Ist die Option ausgewählt, wird ein eigener Layer für die Hintergrundkarte erstellt und angezeigt. Der Layer wird nach dem Start von automatisch angezeigt.

Tipp

Alternativ können Sie den Layer ein- oder ausschalten, indem Sie in der Symbolleiste Grafikparameter des Fensters Netzeditor auf das Symbol Hintergrundkarte an/aus klicken.

Hinweise

Falls trotz ausgewählter Option anstatt der Karte ein weißer Hintergrund angezeigt wird, klicken Sie auf das Symbol  Vollausschnitt setzen.

Das Laden von Hintergrundkarten aus dem Internet können Sie in den Benutzereinstellungen deaktivieren. Erlauben Sie gegebenenfalls das Laden von Hintergrundkarten aus dem Internet.

Kartendienst

Über die Auswahlliste wählen Sie den Internetkartendienst aus, von dem Sie das Kartenmaterial für den Layer Hintergrundkarte herunterladen.

Hinweise

In der Auswahlliste werden die Kartendienste angezeigt, die standardmäßig in vorhanden sind und die Kartendienste, die Sie unter Menü Bearbeiten > Benutzereinstellungen > Benutzeroberfläche > Hintergrundkarte angelegt haben (Voreinstellungen für das Einfügen von Hintergrundkarten vornehmen).

Der Eintrag Standardkartendienst <Kartendienstname> wird zusammen mit Bing Maps (Luftbild) und OpenStreetMap (Mapnik) verwendet. Wenn Sie keine Lizenz für Bing Maps haben, wird automatisch OpenStreetMap verwendet.

Farbeinstellungen

Farbe

Ist die Option ausgewählt, wird die Hintergrundkarte farbig angezeigt.

Graustufen

Ist die Option ausgewählt, wird die Hintergrundkarte in Graustufen angezeigt.

Intensität

Mit dem Schieberegler können Sie die Farbintensität der Hintergrundkarte stufenlos einstellen. Das ermöglicht Ihnen die Illustration Ihres Netzes zu variieren.

Wenn für das Netz kein Koordinatensystem eingestellt ist, wird ein Hinweis angezeigt, dass keine Hintergründe aus Online-Quellen eingefügt werden können. Sie können in diesem Fall nur Grafikdateien als Hintergründe einlesen (Hintergrund von Kartendiensten aus dem Internet einfügen).

Die Internetkarten werden heruntergeladen und in der Netzdarstellung als eigener Grafiklayer im Hintergrund angezeigt.

Der Kartendienst wird über den Namen angesprochen, der in den Grundeinstellungen angelegt worden ist. Wenn Sie eine Versionsdatei weiter geben, bei der ein benutzerdefinierter Kartendienst verwendet wurde, müssen Sie sicherstellen, dass auf den anderen Rechnern die Kartendienste mit dem gleichen Namen vorhanden sind. Das können Sie durch den Austausch der Benutzereinstellungen als Datei sicherstellen (Benutzereinstellungen speichern, öffnen oder zurücksetzen).

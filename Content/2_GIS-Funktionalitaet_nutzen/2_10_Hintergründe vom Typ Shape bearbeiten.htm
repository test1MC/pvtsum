Hintergründe vom Typ Shape bearbeiten

Wenn Sie Hintergründe vom Typ Shapefile eingefügt haben, können Sie diese wie folgt bearbeiten.

Alle Hintergründe vom Typ Shape werden im Fenster Hintergründe verwalten in einer Zeile angezeigt.

Die separaten Shapefile-Layer ermöglichen eine transparente Hintergrundanzeige.

Klicken Sie im Fenster Netz auf das Symbol  Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Hintergründe.

Klicken Sie in die Netzdarstellung.

Das Fenster Hintergründe verwalten öffnet sich.

Wählen Sie in der Liste die Zeile Shapefile Layer aus.

Klicken Sie auf die Schaltfläche Layer.

Das Fenster Zusätzliche Layer öffnet sich. Alle geöffneten Shapefile-Hintergründe werden darin angezeigt.

Element

Beschreibung

Zeichnen

Ist die Option ausgewählt, wird der jeweilige Layer gezeichnet.

Dateiname

Name des Shapefiles

Strichbreite

Strichbreite in mm

Hinweis

Sie können die Strichbreite direkt im Fenster ändern.

Farbe Strich

Sie können die Farbe für die Strichdarstellung des ausgewählten Layers in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

Farbe Fläche

Sie können die Farbe für die Fläche des ausgewählten Layers in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

 

Sie können die Zeichenreihenfolge der Layer ändern. Wählen Sie den gewünschten Layer aus und verschieben Sie ihn über die Symbole um jeweils eine Position nach oben oder unten. Der oberste Layer wird zuerst gezeichnet.

Löschen

Über die Schaltfläche können Sie das ausgewählte Shapefile löschen.

Hinzufügen

Über die Schaltfläche können Sie weitere Hintergründe vom Typ Shape einlesen.

Die Layer werden wie gewünscht angezeigt.

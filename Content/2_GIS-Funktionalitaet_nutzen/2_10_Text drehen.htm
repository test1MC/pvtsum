Text drehen

Klicken Sie im Fenster Netz auf das Symbol  Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Texte.

Klicken Sie auf den Text, den Sie drehen möchten.

Der Text wird farbig markiert.

Klicken Sie nochmal auf den Text, den Sie drehen möchten.

Um den farbig markierten Text werden vier Doppelpfeile angezeigt.

Klicken Sie möglichst präzise auf eine der Pfeilspitzen, halten Sie die Maustaste gedrückt und drehen Sie den Text in die gewünschte Lage.

Sie können auch einen exakten Drehwinkel angeben (Textparameter).

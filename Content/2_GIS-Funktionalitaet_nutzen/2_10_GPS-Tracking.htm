GPS-Tracking

Wenn Sie einen GPS-Empfänger an Ihren PC angeschlossen haben, können Sie die aktuelle Position in der Netzgrafik anzeigen Siehe "GPS-Tracking".

Wählen Sie im Menü Bearbeiten den Eintrag Benutzereinstellungen.

Das Fenster Benutzereinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag Arbeitsumgebung > GPS-Tracking.

Element

Beschreibung

GPS Tracking aktiv

Ist die Option ausgewählt, wird die Funktionalität eingeschaltet.

Nummer des COM-Ports an dem das GPS-Signal empfangen wird

Angabe der Nummer des COM-Ports

Tipp

Die Nummer finden Sie unter Windows unter Start > Systemsteuerung > System > Hardware > Geräte-Manager.

Baudrate

Angabe der Baudrate der GPS-Maus

Hinweis

Die Standardeinstellung ist 4800.

Zeitintervall in Sekunden, nach dem die geographische Position aktualisiert wird

Angabe des gewünschten Zeitintervalls

Das GPS-Tracking wird aktiviert.

Sie können die Netzansicht bei konstantem Zoom-Level auf die aktuelle GPS-Position (Bildschirmmitte) zentrieren, indem Sie auf das Symbol  GPS-Signal zentrieren klicken oder die Funktion über das Menü Ansicht > GPS-Signal zentrieren aufrufen.

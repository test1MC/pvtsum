Räumliche Auswahl von Netzobjekten speichern

Sie können die Netzobjekte, die aktuell in der räumlichen Auswahl enthalten sind, in einer Datei speichern. Die aktuelle räumliche Auswahl wird mit der Versionsdatei gespeichert.

Klicken Sie im Fenster Netz auf das Symbol Modus Räumliche Auswahl.

Die Symbolleiste Räumliche Auswahl wird angezeigt.

Klicken Sie in der Symbolleiste Räumliche Auswahl auf das Symbol Räumliche Auswahl als Datei speichern.

Das Fenster Aktive Netzelemente speichern öffnet sich.

Sie können das Fenster Aktive Netzelemente speichern auch über Menü Datei > Speichern unter > Eintrag Aktive Netzelemente aufrufen.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Die aktuelle räumliche Auswahl wird gespeichert.

Sie können sich über das Menü Netz > Netzstatistik einen Überblick darüber verschaffen, wie viele Netzobjekte je Netzobjekttyp aktuell in der räumlichen Auswahl enthalten, wie viele gefiltert und wie viele aktiv sind (Zustand der Netzobjekte anzeigen).

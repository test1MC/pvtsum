Mehrere Filter gleichzeitig ausschalten

Wählen Sie im Menü Filter den Eintrag Passiv setzen.

Das Fenster Filter passiv setzen öffnet sich.

Wählen Sie die Optionen für die Netzobjektfilter, die Sie ausschalten möchten.

Hier stehen nur Netzobjekttypen zur Auswahl, bei denen aktuell ein Filter eingeschaltet ist.

Über die Schaltfläche Alle an können Sie alle Netzobjektfilter auswählen und über die Schaltfläche Alle aus alle Netzobjektfilter ausschalten.

Die ausgewählten Filter werden ausgeschaltet.

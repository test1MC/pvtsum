Verkehrstag anlegen

Verkehrstage können Sie nur einfügen, wenn Sie das Zusatzmodul Kalender / Verkehrstage aktiviert haben (Zusatzmodule ein- und ausschalten).

Sie können weitere Verkehrstage nur anlegen, wenn Sie einen Wochen- oder Jahreskalender verwenden (Kalender auswählen).

Der Verkehrstag täglich ist standardmäßig vorgegeben.

Wählen Sie im Menü Netz den Eintrag ÖV-Verkehrstage.

Das Fenster Verkehrstage öffnet sich.

Stellen Sie sicher, dass das Register Verkehrstage ausgewählt ist.

Klicken Sie auf die Schaltfläche  Einfügen.

Das Fenster Verkehrstag einfügen öffnet sich.

Geben Sie die gewünschten Daten ein (Attribute und Optionen von Verkehrstagen).

Der Verkehrstag wird angelegt.

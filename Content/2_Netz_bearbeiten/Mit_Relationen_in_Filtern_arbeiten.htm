Mit Relationen in Filtern arbeiten

Sie können Filterbedingungen für indirekte Attribute definieren, die 1:n-Relationen benutzen, mit denen Netzobjekte daraufhin gefiltert werden, ob sie eine oder alle Relationen erfüllen. Dazu wählen Sie in jeder Filterbedingung eine Relation, ein Attribut des Netzobjekts am anderen Ende der Relation, eine Mengenoperation und einen Operator aus.

Beispielsweise können Sie alle Linien filtern, die eine bestimmte Strecke nicht berühren, oder Sie können alle Bezirke filtern, die an einen Knoten angebunden sind.

Stellen Sie sicher, dass das Filterfenster des gewünschten Netzobjekttyps geöffnet ist (Filterfenster öffnen).

Wählen Sie bei Bedarf das gewünschte Register (hierarchische Filter) oder in der Auswahlliste die gewünschte Kategorie (POI-Filter und GIS-Objekte-Filter).

Standardmäßig ist eine Zeile zur Definition einer Filterbedingung angelegt.

Wählen Sie bei Bedarf die Option Kompl. in der Zeile aus, in der Sie die Filterbedingungen definieren.

Wenn sie gewählt ist, werden genau diejenigen Netzobjekte aktiv, die sonst durch die Filterbedingung passiv würden und umgekehrt.

Wenn Sie die Option Komplement zusammen mit einer 1:n-Relation verwenden, beachten Sie dabei, dass sich Kompl. auf die gesamte Bedingung inklusive der Mengenoperation bezieht. Auf eine 1:n Relation angewendet bedeutet das, dass aus einer Bedingung Alle x =Wert A die Bedingung wird Eines x ≠ Wert A.

Klicken Sie auf die Schaltfläche (<Netzobjekttyp>) unter der Spaltenüberschrift Relation.

Das Fenster Relationen (<Netzobjekttyp>) öffnet sich.

Wählen Sie die gewünschte Relation.

Die gewünschte Relation wird angezeigt.

Standardmäßig ist der erste Eintrag (<Netzobjekttyp>) der Auswahlliste ausgewählt, was keiner Relation entspricht.

Klicken Sie auf die leere Schaltfläche unter der Spaltenüberschrift Attribut.

Das Fenster Attribute (<Netzobjekttyp>) öffnet sich.

Im Fenster werden nicht die Attribute des aktuellen Netzobjekttyps angezeigt, sondern alle Attribute für das Ziel der ausgewählten Relation.

Wählen Sie das gewünschte Attribut.

Das gewählte Attribut wird angezeigt.

Wählen Sie in der Spalte Mengenoperation in der Auswahlliste die gewünschte Mengenoperation aus.

Das Feld ist leer, wenn Sie in der Spalte Relation keine 1:n-Relation ausgewählt haben.

Legen Sie den gewünschten Wertebereich fest.

Je nach ausgewähltem Attribut stehen unterschiedliche Eingabemöglichkeiten zur Verfügung.

Der Filter wird angewendet.

Beispiel 1

Finde alle Linien, die Strecke „A“ nicht berühren

  • Netzobjekt ist Linie
  • Relation ist Linienrouten\Linienroutenelemente
  • Attribut ist Eingangsstrecke\Name
  • Mengenoperation ist Alle
  • Operator ist ≠ Wert
  • Wert ist A
Beispiel 2

Finde alle Bezirke, die an Knoten 4711 angebunden sind

  • Netzobjekt ist Bezirk
  • Relation ist Quell-Anbindungen
  • Attribut ist Knoten\Nummer
  • Mengenoperation ist Eines
  • Operator ist = Wert
  • Wert ist 4711

Wollen Sie alle Bezirke finden, die ausschließlich an Knoten 4711 angebunden sind, müssen sie den Mengenoperator Alle auswählen.

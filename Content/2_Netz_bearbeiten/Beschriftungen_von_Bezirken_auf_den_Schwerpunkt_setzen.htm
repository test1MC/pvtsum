Beschriftungen von Bezirken auf den Schwerpunkt setzen

Sie können die Position der Beschriftung für alle oder alle aktiven Bezirke automatisch auf den geometrischen Schwerpunkt ihrer Bezirkspolygone setzen.

Setzen Sie bei Bedarf die Bezirke aktiv, deren Beschriftung Sie an die Position des geometrischen Schwerpunkts setzen möchten, über

  • die räumliche Auswahl (Netzobjekte über die räumliche Auswahl aktiv und passiv setzen) und/oder
  • Filter (Netzobjekte über Filter aktiv und passiv setzen).

Die ausgewählten Bezirke sind aktiv.

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Bezirke.

Das Kontextmenü Bezirke öffnet sich.

Wählen Sie den Eintrag Schwerpunkte neu berechnen.

Das Fenster Mehrfach-Bearbeiten Bezirke öffnet sich.

Ist der Netzobjekttyp Bezirke ausgewählt, können Sie das Fenster Mehrfach-Bearbeiten Bezirke auch über das Menü Bearbeiten > Schwerpunkte neu berechnen aufrufen.

Stellen Sie sicher, dass das Register Sonderfunktionen ausgewählt ist.

Wählen Sie aus, ob die Beschriftung für alle oder alle aktiven Bezirke an die Position des geometrischen Schwerpunkts gesetzt werden soll.

Element

Beschreibung

Nur aktive

Ist die Option ausgewählt, werden nur aktive Bezirke berücksichtigt.

Klicken Sie auf die Schaltfläche Schwerpunkte aus Flächen neu berechnen.

Eine Abfrage öffnet sich.

Die Beschriftungen werden an die Positionen der geometrischen Schwerpunkte gesetzt.

Klicken Sie auf die Schaltfläche Schließen.

Gebiet einfügen

Klicken Sie im Fenster Netz auf das Symbol Modus Einfügen.

Klicken Sie auf die Schaltfläche Gebiete.

Klicken Sie an die Position im Netz, an der Sie den Schwerpunkt des Gebiets einfügen möchten.

Das Fenster Gebiet einfügen öffnet sich.

Geben Sie die gewünschten Daten ein (Attribute und Optionen von Gebieten).

Der Schwerpunkt des Gebiets wird angelegt. Sie können nun eine Umrandung einfügen.

Die Umrandung ist optional und dient zur schnittscharfen Berechnung gebietsbezogener Attribute.

Sie haben folgende Möglichkeiten, fortzufahren:

  • Wenn Sie keine Umrandung für das Gebiet definieren möchten, drücken Sie die Taste Esc. Das Gebiet wird dann als Schwerpunkt ohne Umrandung eingefügt. Die Umrandung können Sie zu einem späteren Zeitpunkt definieren (Umrandung einfügen).
  • Wenn Sie eine Umrandung für das Gebiet definieren möchten, fahren Sie mit dem nächsten Schritt fort.

Definieren Sie die Umrandung des Gebiets, indem Sie mit der Maus mindestens drei Polygonpunkte gegen dem Uhrzeigersinn an den gewünschten Positionen einfügen.

Die Kanten des Umrandungpolygons werden als Gummiband dargestellt. Schraffur und Pfeile geben die Richtung der Fläche an Siehe "Mehrteilige Flächen".

Sie können die Polygonpunkte auch im Uhrzeigersinn einfügen, dadurch entsteht aber eine sogenannte negative Teilfläche (=Loch), die nicht als Umrandung übernommen wird, da noch keine Fläche existiert, aus der sie ausgeschnitten werden kann Siehe "Das Flächenmodell in ".

Bestätigen Sie die Definition der Umrandung mit der rechten Maustaste oder indem Sie die Taste Enter drücken.

Die Umrandung wird angelegt und die Polygonpunkte werden hervorgehoben. Das Fenster Digitalisieren öffnet sich.

Wenn Sie die Definition der Umrandung mit der rechten Maustaste bestätigen, werden die Koordinaten des Klickpunkts nicht als Polygonpunkt übernommen.

Bearbeiten Sie bei Bedarf die Umrandung erneut (Polygonpunkte bearbeiten).

Das Gebiet wird im Netz eingefügt.

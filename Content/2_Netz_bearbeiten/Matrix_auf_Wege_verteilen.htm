Matrix auf Wege verteilen

Mit diesem Verfahren können Sie die Werte einer Nachfragematrix auf Wege verteilen (siehe Grundlagen: Kap. , Seite 1).

Existierende Belastungen werden bei der Verteilung überschrieben.

Matrizen werden nur auf Wege verteilt, die an passenden Bezirken starten und enden.

Über das Attribut AnteilAnWegemengeSoll können Sie zuvor festlegen, welcher Anteil auf welchen Weg entfällt (siehe „Attribute und Optionen von IV-Wegen“ auf Seite 1).

Klicken Sie in der Symbolleiste auf das Symbol  Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Nachfragematrix auf Wege verteilen aus der Kategorie Matrizen ein (siehe „Verfahrensablauf festlegen und starten“ auf Seite 1).

Klicken Sie in der Zeile des neu eingefügten Verfahrens auf die Schaltfläche in der Spalte Bezugsobjekt(e).

Das Fenster Matrix auswählen öffnet sich.

Wählen Sie die Matrix, die auf Wege verteilt werden soll. Dafür stehen Ihnen zwei Methoden zur Verfügung (siehe „Fenster Matrizen auswählen“ auf Seite 1).

Element Beschreibung

Matrix auswählen nach Nummer

Wählen Sie aus der Liste aller Matrizen anhand der Nummer die gewünschte Matrix aus.

Matrix auswählen nach Eigenschaften

Wählen Sie anhand bestimmter Attribute und deren Werte die Matrix aus.

Die ausgewählte Matrix wird in der Spalte Bezugsobjekt(e) übernommen.

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist und klicken Sie im Bereich Aktionen auf die Schaltfläche Bearbeiten.

Das Fenster Matrix auf Wege verteilen öffnet sich.

Element

Beschreibung

Nur aktive Wegemengen

Ist die Option ausgewählt, werden nur die Belastungen aktiver Wegemengen verteilt.

Beziehungsfilter berücksichtigen

Ist die Option ausgewählt, wird der Beziehungsfilter berücksichtigt (siehe „Sonderfall: Beziehungsfilter“ auf Seite 1).

Führen Sie das Verfahren aus (siehe „Verfahren im Verfahrensablauf ausführen“ auf Seite 1).

Die Matrixwerte werden auf die Wege verteilt.

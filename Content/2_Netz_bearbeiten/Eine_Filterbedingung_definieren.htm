Eine Filterbedingung definieren

Stellen Sie sicher, dass das Filterfenster des gewünschten Netzobjekttyps geöffnet ist (Filterfenster öffnen).

Wählen Sie bei Bedarf das gewünschte Register (hierarchische Filter) oder in der Auswahlliste die gewünschte Kategorie (POI-Filter und GIS-Objekte-Filter).

Standardmäßig ist eine Zeile zur Definition einer Filterbedingung angelegt.

Klicken Sie auf die leere Schaltfläche unter der Spaltenüberschrift Attribut.

Das Fenster Attribute (<Netzobjekttyp>) öffnet sich.

Wählen Sie das gewünschte Attribut (Fenster zur Auswahl eines Attributs).

Das gewählte Attribut wird angezeigt.

Wählen Sie für jede Zeile in der Spalte Operation den gewünschten Operator aus.

Je nach Attributtyp stehen unterschiedliche Operationen zur Auswahl.

Legen Sie den gewünschten Wertebereich fest.

Je nach ausgewähltem Attribut stehen unterschiedliche Eingabemöglichkeiten zur Verfügung.

Die Anzahl der angezeigten Nachkommastellen bei einem Attributwert ist abhängig von der Einheit des Attributs. Sie können die Anzahl der Nachkommastellen ändern (siehe „Einstellungen für Dezimalstellen und Aggregationsfunktionen“ auf Seite 1).

Eingabe des Wertebereichs

Zwei Eingabefelder

Wenn Sie ein numerisches Attribut ausgewählt haben, stehen Ihnen zwei Eingabefelder zur Verfügung.

Geben Sie bei Bedarf die untere Grenze des Wertebereichs in das linke Eingabefeld und die obere Grenze des Wertebereichs in das rechte Eingabefeld ein.

Hinweise

Die untere und die obere Grenze sind jeweils im Wertebereich enthalten.

Leereinträge sind erlaubt, d. h. wenn Sie keine obere Grenze angeben, sind alle Werte größer/gleich der unteren Grenze im Wertebereich enthalten. Wenn Sie keine untere Grenze angeben, sind alle Werte kleiner gleich der oberen Grenze im Wertebereich enthalten.

Wenn Sie weder eine untere noch eine obere Grenze angeben, werden nur Netzobjekte ohne Attributwert aktiv.

Ein Eingabefeld

Wenn Sie ein alphanumerisches Attribut ausgewählt haben, steht Ihnen ein Eingabefeld zur Verfügung. Geben Sie hier die Zeichenfolge ein, nach der Sie filtern möchten.

Sie haben zwei Eingabemöglichkeiten:

  • vollständige Zeichenfolge, zum Beispiel Sandweg

Alle Netzobjekte mit dem Attributwert Sandweg werden aktiv.

  • Teil der Zeichenfolge in Kombination mit Platzhaltern (*), zum Beispiel Sand* oder *weg

Alle Netzobjekte, deren Attributwert mit Sand anfängt oder mit weg endet, werden aktiv.

Hinweise

Sie können mehrere Platzhalter in einer Zeichenfolge angeben (* und ?). Ein Fragezeichen ersetzt genau ein Zeichen. Platzhalter können an jeder Stelle der Zeichenfolge stehen.

Die Groß- und Kleinschreibung wird berücksichtigt.

Wenn Sie nichts eingeben, werden nur Netzobjekte ohne Attributwert aktiv.

Feld mit Schaltfläche

Wenn das Attribut vorbelegte Attributwerte besitzt, steht Ihnen statt der Eingabefelder eine Schaltfläche zur Verfügung. Wenn Sie darauf klicken, öffnet sich ein Fenster, in dem Sie die gewünschten Einträge in einer Auswahlliste auswählen können.

Tipps

Sie können mehrere nicht aufeinanderfolgende Einträge auswählen, indem Sie die Taste Strg drücken, diese gedrückt halten und auf die gewünschten Einträge klicken.

Sie können alle Einträge auswählen, indem Sie auf die Schaltfläche Alles wählen klicken. Sie können Ihre Auswahl aufheben, indem Sie auf die Schaltfläche Nichts wählen klicken.

Wählen Sie bei Bedarf die Option Kompl. in der Zeile aus, in der Sie die Filterbedingungen definieren.

Diese Option kehrt die Filterbedingung in dieser Zeile um. Wenn sie gewählt ist, werden genau diejenigen Netzobjekte aktiv, die sonst durch die Filterbedingung passiv würden und umgekehrt.
Beispiel 1

Der Streckenfilter filtert alle Strecken des Verkehrssystems Car (Pkw) und der ausgewählten Streckentypen, die eine Auslastung über 100 % aufweisen.

Beispiel 2

Strecken werden nach ihrer Streckennummer gefiltert.

Wenn Sie eine Filterbedingung löschen oder duplizieren möchten, markieren Sie die gewünschte Zeile und klicken Sie auf die Schaltfläche Löschen oder Duplizieren.

Wenn Sie mehrere Filterbedingungen löschen oder duplizieren möchten, halten Sie die Taste Strg gedrückt, markieren Sie die gewünschten Zeilen und klicken dann auf die Schaltfläche Löschen oder Duplizieren.

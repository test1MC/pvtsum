Fahrplanfahrt einfügen

Sie können eine einfache Fahrplanfahrt oder mehrere Taktfahrten einfügen (Taktfahrten einfügen). Wenn Sie eine einzelne Fahrplanfahrt einfügen möchten, gehen Sie wie folgt vor.

Stellen Sie sicher, dass die gewünschten Linien im Fahrplan angezeigt werden (Linie(n) für die Anzeige im Fahrplaneditor auswählen).

Klicken Sie in der Fahrplan-Symbolleiste auf das Symbol Fahrplanfahrt(en) einfügen.

Alternativ können Sie die Funktion über das Menü Fahrplan (grafisch> Fahrplanfahrt(en) einfügen oder über das Kontextmenü in der Tabelle der Fahrplanfahrten aufrufen.

Das Fenster Fahrplanfahrt(en) einfügen öffnet sich.

Wählen Sie das Register Basis.

StartHPunkt

Haltepunkt der ausgewählten Linienroute, an dem die Fahrplanfahrt startet, und Angabe einer Abfahrtszeit am ausgewählten Start-Haltepunkt.

Hinweise

Der Name des Start-Haltepunkts setzt sich aus den Haltepunkt-Attributen Nummer und Code zusammen oder Nummer und Name, falls kein Code angegeben ist.

Wird die Linienroute oder das Fahrzeitprofil einer Fahrplanfahrt geändert, wird der bisherige Starthaltepunkt wenn möglich beibehalten.

EndHPunkt

Haltepunkt der ausgewählten Linienroute, an dem die Fahrplanfahrt endet

Hinweise

Die Ankunftszeit am End-Haltepunkt wird automatisch berechnet.

Der Name des End-Haltepunkts setzt sich aus den Haltepunkt-Attributen Nummer und Code zusammen oder Nummer und Name, falls kein Code angegeben ist.

Wird die Linienroute oder das Fahrzeitprofil einer Fahrplanfahrt geändert, wird der bisherige Endhaltepunkt wenn möglich beibehalten.

ReferenzHPunkt

Haltepunkt der ausgewählten Linienroute zur Fixierung der Abfahrtszeit (Referenzhaltepunkt setzen)

Hinweis

Der Referenzhaltepunkt ist standardmäßig der Start-Haltepunkt. Geben Sie die Abfahrtszeit am ausgewählten Referenzhaltepunkt an, wenn dieser vom Start-Haltepunkt abweicht. Die Abfahrtszeit am Start-Haltepunkt wird dann aktualisiert.

Die Fahrplanfahrt wird im Fahrplan eingefügt.

Wenn Sie eine Fahrplanfahrt einfügen, die in einem gekoppelten Bereich liegt, wird immer automatisch eine synchron liegende Fahrplanfahrt auf dem mitgekoppelten Fahrzeitprofil eingefügt. Nach Bestätigen einer Abfrage öffnet sich das Fenster Gekoppelte Fahrt bearbeiten, in dem Sie anschließend direkt die Attribute der neuen Fahrplanfahrt auf dem mitgekoppelten Fahrzeitprofil bearbeiten können.

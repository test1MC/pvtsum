Haltepunkte: Register Basis

Element

Beschreibung

ZWert1 - ZWert3

Freie Attribute, die Sie für die Eingabe von zusätzlichen Werten verwenden können

Tipp

Um einem Netzobjekt zusätzliche Informationen zuzuordnen, können Sie auch benutzerdefinierte Attribute anlegen (Benutzerdefinierte Attribute verwalten).

Standard-Haltezeit

Die Standardhaltezeit am Haltepunkt wird durch Fahrzeitprofile überschrieben.

Bereich Position auf Strecke <Nummer>

Absolute Position vor/nach Bezugsknoten

Anzeige der Entfernung zum aktuellen Bezugsknoten

Relative Position (0...1)

Sie können die relative Position des Haltepunkts ändern. Die markierte Position wird als Anteil an der Streckenlänge Von-Knoten - Nach-Knoten eingetragen. Geben Sie den Anteil in Form einer Kommazahl zwischen 0 und 1 ein. Dabei gilt 0.0 = Von-Knoten, 0.5 = Streckenmitte.

Entfernung von Knoten <Nummer Bezugsknoten>

Alternativ können Sie die Entfernung des Haltepunkts zum Bezugspunkt mit dem Schieberegler festlegen.

Hinweis

Die Attribute Bezugsknoten und Absolute Position werden automatisch aktualisiert, ebenso die grafische oder numerische Anzeige der relativen Position.

Alternativ können Sie einen Haltepunkt im Fenster Netzeditor verschieben (Haltepunkte verschieben oder verschmelzen).

Auswahlmöglichkeit der POIs auf eine Kategorie einschränken

Sie haben die Möglichkeit, die für die Auswahl im Fenster Netzeditor zugelassenen POIs auf eine Kategorie oder auf eine Kategorie und deren Unterkategorien einzuschränken.

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche POIs.

Klicken Sie auf das schwarze Dreieck auf der Schaltfläche POIs .

Eine Auswahlliste klappt aus, in der werden die im Netz vorhandenen Kategorien und Unterkategorien von POIs angezeigt.

Klicken Sie auf die Kategorie, die Sie für die Auswahl im Fenster Netzeditor zulassen möchten.

Falls gewünscht, wählen Sie die Option Unterkategorien einschließen aus.

Die Kategorie wird ausgewählt und auf der Schaltfläche POIs wird die Beschriftung um die ausgewählte Kategorie und gegebenenfalls und Unterkategorien erweitert.

Wenn Sie mit der Maus auf die Schaltfläche zeigen, wird in der Quickinfo die Anzahl der bearbeitbaren, vorhandenen und aktiven POIs angezeigt.

Ist eine Einschränkung auf eine Kategorie oder eine Kategorie mit ihren Unterkategorien aktiv, dann beziehen sich die Aktionen Mehrfach-Bearbeiten und Mehrfach-Löschen bei POIs, sowie das Einfügen benutzerdefinierter Attribute und das Umnummerieren immer auf genau diese Kategorie (mit ihren Unterkategorien).

Fügen Sie einen neuen POI ein, stehen für diesen POI nur die ausgewählte Kategorie (mit Unterkategorien) zur Auswahl zur Verfügung.

Haben Sie auf eine Kategorie ohne ihre Unterkategorien eingeschränkt, ist die Kategorie eines neu eingefügten POIs automatisch festgelegt. Das Fenster zur Vergabe der Kategorie öffnet sich in diesem Fall nicht. Der POI wird ohne Fläche eingefügt.

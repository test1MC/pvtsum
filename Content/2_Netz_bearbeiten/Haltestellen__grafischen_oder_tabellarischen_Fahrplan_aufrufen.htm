Haltestellen: grafischen oder tabellarischen Fahrplan aufrufen

Mit dieser Funktion öffnen Sie den grafischen oder tabellarischen Fahrplan (Fahrplaneditor verwenden).

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Haltestellen.

Wählen Sie eine oder mehrere Haltestellen, die von den gewünschten Linienrouten bedient werden (Haltestellen markieren).

Die ausgewählten Haltestellen sind markiert.

Klicken Sie mit der rechten Maustaste auf eine markierte Haltestelle.

Das Kontextmenü Haltestellen öffnet sich.

Wählen Sie den Eintrag Fahrplan (grafisch), Fahrplan (tabellarisch) oder Beide Fahrpläne.

Wenn die gewünschte Fahrplanansicht bereits geöffnet ist, öffnet sich eine Abfrage (Fahrplaneditor öffnen).

Wenn der Netzobjekttyp Haltestellen ausgewählt ist, können Sie die Fahrplanansichten auch über das Menü Bearbeiten aufrufen.

Wenn Sie eine Haltestelle markiert haben, die sich auf einer Linienroute befindet, die in der Linienauswahl enthalten ist, öffnet sich der ausgewählte Fahrplan. Wenn Sie eine Haltestelle markiert haben, die sich auf einer Linienroute befindet, die nicht ausgewählt ist, öffnet sich eine Abfrage.

Element Beschreibung
Linienauswahl ändern

Die bestehende Linienauswahl im Netz soll geändert werden. Das Fenster Linienauswahl ändern öffnet sich (siehe „Linienauswahl ändern“ auf Seite 1). In dem Fenster können Sie Einstellungen für die Änderung der Linienauswahl vornehmen.

Linienauswahl beibehalten

Die bestehende Linienauswahl des Netzes wird beibehalten und der Fahrplan öffnet sich. Die Fahrplanfahrten, deren Verläufe über die gewählten Strecken führen, sind nicht markiert.

Klicken Sie auf die Schaltfläche Linienauswahl ändern.

(siehe „Linienauswahl ändern“ auf Seite 1).

Der Fahrplan öffnet sich und die überfahrenen Linienrouten werden je nach gewählten Optionen angezeigt.

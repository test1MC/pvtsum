Attribut mit einem konstanten Wert belegen oder addieren

Stellen Sie sicher, dass das Fenster Mehrfach-Bearbeiten <Netzobjekttyp> offen ist und dass das Register Formel ausgewählt ist (Attributwerte von Netzobjekten bearbeiten).

Alle Eingabeattribute des ausgewählten Netzobjekttyps stehen zur Auswahl.

Wählen Sie das Attribut, das Sie mit einem konstanten Wert belegen möchten, und ein Subattribut, sofern vorhanden.

Klicken Sie auf die Schaltfläche Konstant.

Das Fenster Attribut konstant belegen öffnet sich. Abhängig vom jeweils ausgewählten Netzobjekttyp können Sie einen Wert auswählen oder eingeben.

Geben Sie den gewünschten Wert in das Eingabefeld ein.

Bei der Eingabe von numerischen Attributwerten mit Längen- oder Zeitbezug müssen Sie die Einheiten berücksichtigen:

  • Wenn Sie nur den Wert eingeben, wird dieser im Standard-Format interpretiert.
  • Sie können einen Wert mit Einheit eingeben. In diesem Fall rechnet den Wert um, damit er im festgelegten Format angezeigt wird. Beispiel: Für ein Attribut ist das Format Minuten festgelegt. Sie geben nun 2h ein. Das Feld erhält dann den Wert 120.

Nehmen Sie die gewünschte Einstellung vor.

Element

Beschreibung

Wert addieren

Ist die Option ausgewählt, wird der eingegebene Wert zum bestehenden Attributwert addiert.

Ist die Option nicht ausgewählt, wird der bestehende Attributwert durch den eingegebenen Wert ersetzt.

Der Attributwert wird geändert.

Alternativ können Sie ein Attribut während des Verfahrensablaufs mit einem konstanten Wert belegen (Attributwerte während des Verfahrensablaufs ändern).

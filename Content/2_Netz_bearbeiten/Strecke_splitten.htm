Strecke splitten

Sie können eine Strecke teilen, indem Sie einen Zwischenknoten einfügen oder indem Sie einen Zwischenknoten mit dem exakten Abstand zu den bestehenden Knoten einfügen.

Strecke durch einen Zwischenknoten splitten

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Strecken.

Klicken Sie im Netz neben die Strecke in gewünschter Richtung, die Sie splitten möchten.

Drücken Sie die Taste F8.

Fügen Sie einen Zwischenknoten ein, indem Sie ins Netz klicken.

Das Fenster Knoten <Knotennummer> einfügen öffnet sich.

Der Zwischenknoten muss nicht auf der zu splittenden Strecke liegen. Als Zwischenknoten können Sie

  • einen Knoten im Netz markieren (außer Von-Knoten und Nach-Knoten der zu splittenden Strecke) oder
  • einen „isolierten” Knoten ins Netz einfügen (siehe „Knoten einfügen“ auf Seite 1). Eine Vorschau zeigt Ihnen direkt wo der neue Knoten im Netz oder auf der Strecke eingefügt wird. Durch ein Lassosymbol wird Ihnen angezeigt, dass der Modus „Schnappen‟ aktiv ist und der Knoten auf dem bestehenden Streckenverlauf eingefügt wird. Um den Schnappen-Modus auszuschalten halten Sie bitte die Taste Umschalt gedrückt.

Nehmen Sie die gewünschten Änderungen vor (siehe „Attribute und Optionen von Knoten“ auf Seite 1).

Das Fenster Strecke splitten öffnet sich.

Element

Beschreibung

Neue Streckennummern

Die nächsten freien Streckennummern im Netz werden vorgeschlagen. Sie können die Streckennummern ändern.

Die Strecke wird gesplittet. Zwei resultierende Strecken, gegebenenfalls ein neuer Zwischenknoten und neue Abbieger werden eingefügt.

Strecke durch Längenangabe splitten

Stellen Sie sicher, dass eine Strecke ausgewählt ist, die durch eine Längenangabe gesplittet werden soll (siehe „Strecke durch einen Zwischenknoten splitten“ auf Seite 1).

Klicken Sie mit der rechten Maustaste auf die ausgewählte Strecke.

Wählen Sie den Eintrag Splitten nach Längenangabe.

Das Fenster Strecke splitten nach Längenangabe öffnet sich.

Element

Beschreibung

Einfügeposition des neuen Zwischenknotens

Sie legen in diesem Bereich des Fensters fest, von welchem Knoten der Abstand zum Zwischenknoten gemessen wird.

Ausgehend von:

Knoten <Knotennummer> (Von-Knoten)

Ist die Option gewählt, wird der Abstand des Zwischenknotens zum Von-Knoten der Strecke gemessen.

Knoten <Knotennummer> Nach-Knoten)

Ist die Option gewählt, wird der Abstand des Zwischenknotens zum Nach-Knoten der Strecke gemessen.

Im Abstand von:

Mit dem Schieberegler können Sie den Abstand des Zwischenknotens zum Ausgangsknoten variabel festlegen. Sie können eine genaue Entfernung in das Eingabefeld eingeben.

Neue Streckennummern

Die nächsten freien Streckennummern im Netz werden vorgeschlagen. Sie können die Streckennummern ändern.

Das Fenster Knoten <Knotennummer> einfügen öffnet sich.

Nehmen Sie die gewünschten Änderungen vor (siehe „Attribute und Optionen von Knoten“ auf Seite 1).

Die Strecke wird gesplittet. Zwei resultierende Strecken und ein neuer Zwischenknoten werden eingefügt.

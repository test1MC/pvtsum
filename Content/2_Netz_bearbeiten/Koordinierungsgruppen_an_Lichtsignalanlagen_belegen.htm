Koordinierungsgruppen an Lichtsignalanlagen belegen

Mit dieser Funktion können Sie die Lichtsignalanlagen ausgewählter Knoten mit Koordinierungsgruppen-Nummern belegen.

Wählen Sie die Knoten aus, deren Lichtsignalanlagen Sie belegen möchten, über

  • die räumliche Auswahl (Netzobjekte über die räumliche Auswahl aktiv und passiv setzen) und/oder
  • Filterkriterien (Netzobjekte über Filter aktiv und passiv setzen).

Die ausgewählten Knoten sind aktiv.

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Knoten.

Das Kontextmenü Knoten öffnet sich.

Wählen Sie den Eintrag Koordinierungsgruppen an LSA belegen.

Das Fenster Mehrfach-Bearbeiten Knoten öffnet sich.

Ist der Netzobjekttyp Knoten ausgewählt, können Sie das Fenster Mehrfach-Bearbeiten Knoten auch über das Menü Bearbeiten > Koordinierungsgruppen an LSA belegen aufrufen.

Stellen Sie sicher, dass das Register Sonderfunktionen aktiv und in der Auswahlliste der Eintrag Koordinierungsgruppen an LSA belegen gewählt sind.

Wählen Sie aus, ob die Lichtsignalanlagen aller oder aller aktiven Knoten belegt werden sollen.

Element

Beschreibung

Nur aktive

Ist die Option ausgewählt, werden nur aktive Knoten berücksichtigt.

Nehmen Sie die gewünschte Einstellung vor.

Element

Beschreibung

Koordinierungsgruppe

Ist die Option ausgewählt, können Sie den Lichtsignalanlagen aller ausgewählten Knoten eine Koordinierungsgruppe zuordnen. Wählen Sie in der Auswahlliste die gewünschte Koordinierungsgruppe aus.

Tipp

Signalkoordinierungsgruppen können Sie im Menü Netz > Signalkoordinierungsgruppen einfügen und bearbeiten (Signalkoordinierungsgruppen verwalten).

Knotenattribut

Ist die Option ausgewählt, werden alle Lichtsignalkoordinierungsgruppen-Nummern mit dem Wert des ausgewählten Attributs belegt.

Tipp

Zu diesem Zweck können Sie benutzerdefinierte Attribute definieren (Benutzerdefinierte Attribute verwalten).

Klicken Sie auf die Schaltfläche Belegen.

Alle ausgewählten Signalanlagen erhalten die ausgewählte Koordinierungsgruppen-Nummer oder den Wert des ausgewählten Attributs.

Klicken Sie auf die Schaltfläche Schließen.

Sie können das Attribut Koordinierungsgruppen-Nummer auch in der Liste Signalisierung > Signalanlagen für die gewünschten Signalanlagen bearbeiten.

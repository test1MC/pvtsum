Fahrplanfahrten löschen

Das Löschen von Fahrplanfahrten kann Auswirkungen auf gekoppelte Fahrzeitprofile haben (siehe „Mehrere Fahrzeitprofile koppeln“ auf Seite 1).

Sie haben verschiedene Möglichkeiten Fahrplanfahrten zu löschen.

Markierte Fahrplanfahrten löschen

Markieren Sie im Fahrplan die Fahrplanfahrten, die Sie löschen möchten (siehe „Fahrplanfahrten markieren“ auf Seite 1).

Drücken Sie die Taste Entf.

Bestätigen Sie gegebenenfalls die Sicherheitsabfrage.

Alle aktiven Fahrplanfahrten löschen

Setzen Sie die Fahrplanfahrten, die Sie löschen möchten, über Filterkriterien aktiv. Filterkriterien für Fahrplanfahrten definieren Sie über den hierarchischen Filter auf Linien (siehe „Mit hierarchischen Filtern arbeiten“ auf Seite 1).

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Linien.

Das Kontextmenü Linien öffnet sich.

Wählen Sie den Eintrag Mehrfach-Löschen > Fahrplanfahrten.

Eine Sicherheitsabfrage öffnet sich.

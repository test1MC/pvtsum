Systemrouten: Register Fahrzeugkombinationen

Sie können die Zeitattribute der Systemroute für jede Fahrzeugkombination des Verkehrssystems überschreiben und diese für Fahrzeitprofile von Linienrouten dieser Fahrzeugkombination verwenden.

Wenn Sie Zeiten von Systemrouten für Linienrouten verwenden möchten, wählen Sie im Fenster Linienroute bearbeiten > Register Verlauf und Fahrzeitprofile > Symbol Neue Zeiten für das aktuelle Fahrzeitprofil setzen die Option aus Systemrouten (Zeiten für ein Fahrzeitprofil setzen).

Element

Beschreibung

Einfügen

Über die Schaltfläche können Sie eine neue Fahrzeit einfügen. Geben Sie dazu im separaten Fenster die gewünschten Daten ein (Attribute und Optionen von Systemrouten) und bestätigen Sie mit OK.

Hinweis

Je Systemroute können Sie nur eine Durchfahrzeit je Fahrzeugkombination anlegen.

Löschen

Entfernt die im Fenster ausgewählte Zeile

Alle löschen

Entfernt alle Fahrzeiten der Systemroute

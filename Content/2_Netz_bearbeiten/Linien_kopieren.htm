Linien kopieren

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Linien.

Das Fenster Linien öffnet sich.

Stellen Sie sicher, dass die Option Linien ausgewählt ist.

Markieren Sie im Fenster Linien die Linie, die Sie kopieren möchten.

Klicken Sie mit der rechten Maustaste in die Netzdarstellung.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Linie kopieren.

Das Fenster Linie kopieren öffnet sich.

Element

Beschreibung

Linie

Geben Sie hier den Namen der neuen Linie ein. Der Name muss eindeutig sein.

Alle Fahrplanfahrten mitkopieren

Wählen Sie diese Option, wenn alle Fahrplanfahrten mitkopiert werden sollen.

Geh- und Wartezeiten an Haltestellen mitkopieren

Wählen Sie diese Option, wenn Geh- und Wartezeiten an Haltestellen mitkopiert werden sollen.

Die Linie wird einschließlich ihrer Attribute und aller zugeordneten Linienrouten kopiert. Die neue Linie wird im Fenster Linien aufgelistet.

Auf der Originallinie bestehende Kopplungen werden nicht mitkopiert.

Fahrzeugkombinationen auf Fahrplanfahrtabschnitten ersetzen

Diese Funktion ist nützlich, wenn Sie eine bestimmte Fahrzeugkombination durch eine oder mehrere Fahrzeugkombinationen ersetzen möchten. Wenn Sie beispielsweise auf einem Fahrplanfahrtabschnitt die Fahrzeugkombination Train durch die beiden Fahrzeugkombinationen TrainA und TrainB ersetzen, können Sie diese in der Umlaufbildung getrennt verwenden.

Beachten Sie, dass sich durch diesen Vorgang die Kostenangaben ändern können, weil beispielsweise die Kosten für die Kombination aus TrainA und TrainB nicht notwendigerweise den Kosten für Train entsprechen.

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Linien.

Das Kontextmenü Linien öffnet sich.

Wählen Sie den Eintrag Fahrzeugkombinationen auf Fahrplanfahrtabschnitten ersetzen.

Das Fenster Mehrfach-Bearbeiten Fahrplanfahrtabschnitte öffnet sich. Das Register Sonderfunktionen wird angezeigt. In der Auswahlliste ist der Eintrag Fahrzeugkombinationen auf Fahrplanfahrtabschnitten ersetzen ausgewählt.

Element

Beschreibung

nur aktive

Ist die Option ausgewählt, werden nur aktive Fahrplanfahrtabschnitte berücksichtigt.

Fahrzeugkombination

Hier wählen Sie die Fahrzeugkombination, die Sie ersetzen möchten.

Keine FzgKomb

Wählen Sie diese Option, wenn Sie für alle Fahrplanfahrtabschnitte, für die noch keine Fahrzeugkombination (Fahrzeugkombinationsnummer 0) festgelegt ist, eine Fahrzeugkombination angeben möchten.

<Nummer und Code der Fahrzeugkombination>

Wählen Sie die Fahrzeugkombination, die Sie ersetzen möchten.

Beispiel:

Sie möchten die Fahrzeugkombination 1 Train durch die beiden Fahrzeugkombinationen 2 TrainA und 3 TrainB ersetzen. Geben Sie hier 1 Train ein.

ersetzen durch

<n> x <Nummer und Code der Fahrzeugkombination>

oder

<n> x <keine FzgKomb>

Hier wählen Sie eine oder mehrere Fahrzeugkombinationen, durch die Sie die ursprüngliche Fahrzeugkombination ersetzen möchten. Wenn Sie eine Fahrzeugkombination durch mehrere ersetzen, wird für jede zusätzliche Fahrzeugkombination eine Kopie des ursprünglichen Fahrplanfahrtabschnitts angelegt.

Beispiel (Fortsetzung von oben):

Legen Sie unter ersetzen durch Folgendes fest: 1 x 2 TrainA und 1 x 3 TrainB. Dann ersetzt die Fahrzeugkombination 1 Train durch 2 TrainA und 3 TrainB. Da Sie eine Fahrzeugkombination durch zwei ersetzt haben, legt einen zusätzlichen Fahrplanfahrtabschnitt an.

Klicken Sie auf die Schaltfläche Fahrzeugkombination ersetzen.

Die angegebene Fahrzeugkombination wird gemäß Ihren Einstellungen ersetzt. Wenn Sie eine Fahrzeugkombination durch mehrere ersetzt haben, legt zusätzliche Fahrplanfahrtabschnitte an.

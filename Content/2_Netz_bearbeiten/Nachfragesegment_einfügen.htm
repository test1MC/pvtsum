Nachfragesegment einfügen

Wählen Sie im Menü Nachfrage den Eintrag VSys/Modi/NSeg.

Das Fenster VSys/Modi/NSegs öffnet sich.

Wählen Sie das Register Nachfragesegmente.

Klicken Sie auf die Schaltfläche Einfügen.

Das Fenster Nachfragesegment einfügen öffnet sich.

Geben Sie die gewünschten Daten ein (Attribute und Optionen von Nachfragesegmenten).

Das Nachfragesegment wird angelegt.

Nachfragesegmente können Sie beim Anlegen von Verkehrssystemen automatisch mit anlegen (Verkehrssystem einfügen).

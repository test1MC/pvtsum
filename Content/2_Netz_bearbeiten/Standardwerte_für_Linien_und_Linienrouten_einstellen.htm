Standardwerte für Linien und Linienrouten einstellen

Bevor Sie eine Linie, Linienroute oder ein Fahrzeitprofil einfügen, können Sie folgende Standardwerte festlegen.

Wählen Sie im Menü Bearbeiten den Eintrag Benutzereinstellungen.

Das Fenster Benutzereinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag Netz >Linien.

Element

Beschreibung

Linienroute digitalisieren – Fahrzeit zwischen Profilpunkten

Runden auf

Hier können Sie angeben, wie die Fahrzeit zwischen zwei Profilpunkten beim Digitalisieren der Linienroute gerundet werden soll.

Minimal

Hier können Sie einen Mindestwert für die Fahrzeit zwischen zwei Profilpunkten wählen.

Automatische Vergabe von Fahrplanfahrt-Nummern

Hier können Sie angeben, wie die Fahrplanfahrten einer Linienroute beim Einfügen, Kopieren oder Koppeln von Fahrplanfahrten nummeriert werden. Folgende Optionen stehen zur Auswahl:

  • Nächste freie Nummer
  • Hinrichtung gerade, Rückrichtung ungerade
  • Rückrichtung gerade, Hinrichtung ungerade

Default-Parameter beim Digitalisieren

Hier können Sie Standard-Einstellungen für die Wegesuche vornehmen, auf die beim Digitalisieren von Linienrouten zurückgegriffen wird (Linienroute einfügen).

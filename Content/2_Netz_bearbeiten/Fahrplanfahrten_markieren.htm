Fahrplanfahrten markieren

Sie haben folgende Möglichkeiten Fahrplanfahrten zu markieren, um sie anschließend zu bearbeiten.

Markierte Fahrplanfahrten werden grafisch hervorgehoben.

Im tabellarischen Fahrplan markierte Fahrplanfahrten sind auch im grafischen Fahrplan markiert und umgekehrt, wenn Sie in beiden Fenstern die Synchronisation mit anderen Fenstern eingeschaltet haben.

Im grafischen oder tabellarischen Fahrplan markierte Fahrplanfahrten sind auch im Netzeditor und in der Liste Fahrplanfahrten markiert und umgekehrt, wenn die Synchronisation mit anderen Fenstern in den Fenstern eingeschaltet ist (siehe „Grafischen und tabellarischen Fahrplan mit anderen Fenstern synchronisieren“ auf Seite 1).

Sind Fahrplanfahrten im Fahrplan markiert, so sind die zugehörigen Einträge im Linienauswahlfenster fett dargestellt.

Markierung

Beschreibung

Eine Fahrplanfahrt markieren

Tabellarischer Fahrplan

Sie markieren eine Fahrplanfahrt, indem Sie auf den Spaltenkopf der gewünschten Fahrplanfahrt klicken.

Tipp

Wenn Sie in die Spalte klicken, wird die Markierung aufgehoben.

Grafischer Fahrplan

Sie markieren eine Fahrplanfahrt, indem Sie direkt auf den Fahrtverlauf klicken.

Mehrere Fahrplanfahrten markieren

Tabellarischer Fahrplan

Sie markieren mehrere Fahrplanfahrten, indem Sie die Taste Strg gedrückt halten und nacheinander in mehrere Spaltenköpfe klicken. Alle Spalten werden markiert angezeigt. Bei der zuletzt markierten wird die Kopfzeile nicht hinterlegt.

Tipp

Wenn Sie in eine markierte Spalte klicken, wird die Markierung aufgehoben.

Grafischer Fahrplan

Sie markieren mehrere Fahrplanfahrten, indem Sie die Taste Strg gedrückt halten und nacheinander auf die gewünschten Fahrtverläufe klicken.

Tipp

Alternativ können Sie im Grafischer Fahrplan mit der Maus ein Rechteck aufziehen. Die darin enthaltenen Fahrplanfahrten sind dann markiert.

Mehrere nebeneinander stehende Fahrplanfahrten markieren

Tabellarischer Fahrplan

Sie markieren eine Reihe nebeneinander stehende Fahrplanfahrten, indem Sie in den ersten Spaltenkopf klicken, die Taste Umschalt gedrückt halten und dann in den letzten Spaltenkopf klicken.

Gekoppelte Fahrplanfahrten markieren

Sie können alle Fahrplanfahrten markieren, die mit der aktuell markierten Fahrplanfahrt gekoppelt sind, indem Sie den Eintrag Gekoppelte Fahrplanfahrt suchen im Kontextmenü wählen.

Hinweise

Wird eine gekoppelte Fahrplanfahrt bisher nicht im Fahrplan angezeigt, erscheint eine Abfrage. Wenn Sie die Abfrage bestätigen, wird die Fahrplanfahrt angezeigt und markiert. Auch indirekt gekoppelte Fahrplanfahrten werden markiert, also Fahrplanfahrten, die nicht miteinander gekoppelt sind, sondern mit einer gemeinsamen anderen Fahrplanfahrt.

Tipp

Alternativ können Sie die Funktion mit Strg + F3 aufrufen.

Fahrplanfahrten bestimmter Linien, Linienrouten, Fahrzeitprofile markieren

Tabellarischer Fahrplan und Grafischer Fahrplan

Sie markieren die Fahrplanfahrten bestimmter Objekte der Linienhierarchie, indem Sie auf das Symbol  Fahrplanfahrten markieren klicken und im gleichnamigen Fenster die gewünschten Objekte der Linienhierarchie auswählen.

Fahrplanfahrtmarkierungen umkehren

Tabellarischer Fahrplan und Grafischer Fahrplan

Sie kehren die aktuellen Markierungen von Fahrplanfahrten um, indem Sie auf das Symbol  Fahrplanfahrtmarkierung umkehren klicken.

Tipp

Alternativ können Sie im Tabellarischer Fahrplan die Markierung auch im Menü Fahrplan (tabellarisch) Eintrag Fahrplanfahrtmarkierung umkehren umkehren.

Markierung(en) aufheben

Tabellarischer Fahrplan und Grafischer Fahrplan

Sie heben die aktuellen Markierungen von Fahrplanfahrten auf, indem Sie auf das Symbol  Fahrplanfahrtmarkierungen leeren klicken.
Fahrplanfahrten bestimmter Objekte der Linienhierarchie markieren

Sie können die Fahrplanfahrten bestimmter Linien, Linienrouten, Fahrzeitprofile markieren.

Klicken Sie in der Symbolleiste des grafischen oder tabellarischen Fahrplans auf das Symbol  Fahrplanfahrten markieren.

Das Fenster Fahrplanfahrten markieren öffnet sich, und die im Fenster Fahrplan-Linien in der Linienauswahl ausgewählten Linien werden angezeigt.

Wählen Sie die Objekte aus, deren Fahrplanfahrten Sie im Fahrplan markieren möchten.

Element

Beschreibung

Alle Fahrplanfahrten der angezeigten Linien und untergeordneter Netzobjekte im grafischen und tabellarischen Fahrplan markieren

Keine Fahrplanfahrten markieren

Bisherige Auswahl umkehren

Die ausgewählten Fahrplanfahrten werden im fahrplan angezeigt.

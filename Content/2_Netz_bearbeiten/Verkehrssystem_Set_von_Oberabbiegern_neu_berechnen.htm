Verkehrssystem-Set von Oberabbiegern neu berechnen

Sie können für alle oder alle aktiven Oberabbieger im Netz die IV-Verkehrssysteme neu ermitteln. Dabei werden jedem Oberabbieger, auch den U-Turns, die Verkehrssysteme zugeordnet, die sowohl für die Von-Strecken als auch für die Nach-Strecken zugelassen sind. Dies entspricht der Standardeinstellung, die beim Einfügen neuer Oberabbieger verwendet. Diese Funktion können Sie beispielsweise durchführen, nachdem Sie die Verkehrssysteme von Strecken geändert haben.

Wählen Sie die gewünschten Oberabbieger aus, durch:

  • die räumliche Auswahl (Netzobjekte über die räumliche Auswahl aktiv und passiv setzen) und/oder
  • Filterkriterien (Netzobjekte über Filter aktiv und passiv setzen).

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Oberabbieger.

Das Kontextmenü Oberabbieger öffnet sich.

Wählen Sie im Fenster Netz den Eintrag Mehrfach-Bearbeiten.

Das Fenster Mehrfach-Bearbeiten Oberabbieger öffnet sich.

Wählen Sie das Register Sonderfunktionen.

Wählen Sie aus, ob Sie für alle oder für alle aktiven Oberabbieger die Verkehrssysteme neu berechnen möchten.

Element

Beschreibung

nur aktive

Ist die Option ausgewählt, werden nur aktive Oberabbieger berücksichtigt.

Wählen Sie aus der Auswahlliste den Eintrag VSysSet setzen.

Klicken Sie auf die Schaltfläche IV-VSysSet aus IV-VSysSet der Strecken berechnen.

Diese Funktion steht nicht für ÖV-Verkehrssysteme zur Verfügung.

Die IV-Verkehrssysteme werden den Oberabbiegern zugeordnet.

Attributwerte aller oder aller aktiven POIs ändern

Sind keine passiven POIs im Netz, werden alle POIs geändert. Markierungen von POIs werden ignoriert.

Setzen Sie bei Bedarf die POIs aktiv, deren Attributwerte Sie ändern möchten. Verwenden Sie dazu:

  • die räumliche Auswahl (Netzobjekte über die räumliche Auswahl aktiv und passiv setzen) und/oder
  • Filterkriterien (Netzobjekte über Filter aktiv und passiv setzen).

Die ausgewählten POIs sind aktiv.

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche POIs.

Das Kontextmenü Points of Interest öffnet sich.

Wählen Sie den Eintrag Mehrfach-Bearbeiten und dann die gewünschte Kategorie.

Sie können entweder alle POIs ändern oder eine Kategorie wählen. Alle angelegten Kategorien sowie alle Kategorien mitsamt Unterkategorien werden zur Auswahl angeboten. Bei mehr als zwölf Kategorien wählen Sie die Kategorie über den Eintrag Kategorie wählen in einem separaten Fenster aus. In dem Fenster können Sie festlegen, ob auch alle POIs der Unterkategorien geändert werden sollen.
Sie können keine Kategorie auswählen, wenn Sie die POIs bereits zuvor auf eine Kategorie eingeschränkt haben, die Änderungen beziehen sich alle nur auf POIs der ausgewählten Kategorie (siehe „Auswahlmöglichkeit der POIs auf eine Kategorie einschränken“ auf Seite 1)

Das Fenster Mehrfach-Bearbeiten Points of Interest <Nr.>: <Kategorie> öffnet sich.

Wählen Sie aus, ob Sie die Attributwerte aller oder aller aktiven POIs ändern möchten.

Element

Beschreibung

nur aktive

Ist die Option ausgewählt, werden nur aktive POIs berücksichtigt.

Wählen Sie das Register Formel.

Ändern Sie die Werte der gewünschten Attribute (Attribute und Optionen von POIs und Attributwerte von Netzobjekten bearbeiten).

Die Attributwerte der gewählten Kategorie(n) werden geändert.

Klicken Sie auf Schließen.

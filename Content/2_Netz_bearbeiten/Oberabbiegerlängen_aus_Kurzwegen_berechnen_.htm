Oberabbiegerlängen aus Kurzwegen berechnen

Wenn Sie Oberabbieger aus Kurzwegen belegen, können Sie die Längen der Kurzwege in einem Zielattribut speichern.

Wählen Sie die Oberabbieger aus, deren Längen Sie berechnen möchten, über

  • die räumliche Auswahl (Netzobjekte über die räumliche Auswahl aktiv und passiv setzen) und/oder
  • Filterkriterien (Netzobjekte über Filter aktiv und passiv setzen).

Die ausgewählten Oberabbieger sind aktiv.

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Oberabbieger.

Das Kontextmenü Oberabbieger öffnet sich.

Wählen Sie den Eintrag Länge aus Kurzwegen berechnen.

Das Fenster Mehrfach-Bearbeiten Oberabbieger öffnet sich.

Ist der Netzobjekttyp Oberabbieger ausgewählt, können Sie das Fenster Mehrfach-Bearbeiten Oberabbieger auch über das Menü Bearbeiten > Länge aus Kurzwegen berechnen aufrufen.

Stellen Sie sicher, dass das Register Sonderfunktionen gewählt ist.

Wählen Sie aus, ob Sie die Längen aller oder aller aktiven Oberabbieger aus Kurzwegen berechnen möchten.

Element

Beschreibung

Nur aktive

Ist die Option ausgewählt, werden nur aktive Oberabbieger berücksichtigt.

Wählen Sie über die Schaltfläche ein Attribut aus, in das die Längen aus Kurzwegen geschrieben werden.

Klicken Sie auf die Schaltfläche Berechnen.

Die berechneten Längen werden in das ausgewählte Attribut geschrieben.

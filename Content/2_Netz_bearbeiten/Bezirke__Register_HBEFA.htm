Bezirke: Register HBEFA

Im Register HBEFA können Sie Einstellungen für Berechnung der Kaltstart-Emission nach HBEFA vornehmen Siehe "Berechnungsgrundlagen der Kaltstartzuschläge".

Element

Beschreibung

Kaltstartzuschläge flächig berechnen

Ist die Option ausgewählt, wird die Kaltstart-Emission flächig berechnet.

Kaltstartzuschläge auf Wegen berechnen

Ist die Option ausgewählt, wird die Kaltstart-Emission auf der Basis von Wegen berechnet.

Anteil Kaltstart

Über dieses Attribut legen Sie fest, wie hoch der Anteil des Kaltstarts für diesen Bezirk sein soll. Sie können dem Attribut Werte zwischen 0 und 1 zuweisen.

Tipp

Handelt es sich um einen Kordonbezirk, ist es beispielsweise sinnvoll, dem Bezirk einen Wert 0 zuzuordnen, da Fahrzeuge normalerweise bereits warm sind, wenn sie einen Kordonbezirk passieren.

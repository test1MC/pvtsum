Angezeigte Attribute auswählen

Sie können für die Bereiche des tabellarischen Fahrplans auswählen, welche Attribute Sie anzeigen möchten.

Sie können Ihre Auswahl angezeigter Attribute und weitere Einstellungen verwerfen, indem Sie im Menü Fahrplan (tabellarisch) den Eintrag Ansicht zurücksetzen wählen. Die intern gespeicherten Layouteinstellungen des Benutzers werden dann verworfen und das Standardlayout wird geladen.
Attribute der Fahrplanfahrten und Fahrplanfahrtabschnitte anzeigen

Im Bereich (2) des tabellarischen Fahrplans werden Attribute der Fahrplanfahrten angezeigt (Das Fahrplaneditor-Fenster). Im Bereich (3) werden die Attribute der Fahrplanfahrtabschnitte angezeigt. Diese Ansicht können Sie ein- und ausschalten. Im Bereich (4) werden Attribute der Haltefolge, des Verlaufs der Fahrplanfahrten oder der Fahrplanfahrtabschnitte angezeigt. Sie können die Auswahl der angezeigten Attribute anpassen.

Klicken Sie mit der rechten Maustaste in dem gewünschten Bereich in die Spalte der Attribute und klicken Sie auf den Eintrag Attributauswahl bearbeiten.

Das Fenster Attributauswahl (<Objektname>) öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (Fenster zur Auswahl eines Attributs).

Die im tabellarischen Fahrplan angezeigten Attribute werden im Auswahlfenster zum Teil anders benannt:

  • Start-Haltepunkt (Nummer und Code des Haltepunkts): entspricht im Auswahlfenster dem Attribut Von-Fahrzeitprofilelement-Index
  • End-Haltepunkt (Nummer und Code des Haltepunkts): entspricht im Auswahlfenster dem Attribut Nach-Fahrzeitprofilelement-Index

Die gewünschten Attribute werden im tabellarischen Fahrplan angezeigt.

Anzeige der Haltefolgeattribute

Sie können die Attribute der Haltefolge auswählen, die im linken Teil von Bereich (4) angezeigt werden.

Klicken Sie mit der rechten Maustaste in den ersten Spaltenkopf im Bereich (4).

Wählen Sie den Eintrag Attributauswahl Haltefolge bearbeiten.

Das Fenster Attributauswahl (Haltefolgeelement) öffnet sich.

( Fenster zur Auswahl eines Attributs ) .

Die gewünschten Attribute werden im tabellarischen Fahrplan angezeigt.

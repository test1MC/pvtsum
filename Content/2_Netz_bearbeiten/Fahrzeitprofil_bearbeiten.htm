Fahrzeitprofil bearbeiten

Ein angelegtes Fahrzeitprofil können Sie im Fenster Linienroute bearbeiten im Register Verlauf und Fahrzeitprofile bearbeiten.

Öffnen Sie das Fenster Linienrouten (siehe „Fenster Linienrouten öffnen“ auf Seite 1).

Doppelklicken Sie auf die gewünschte Linienroute in der Liste.

Das Fenster Linienroute bearbeiten öffnet sich.

Wählen Sie das Register Verlauf und Fahrzeitprofile.

Das Register enthält alle Fahrzeitprofile der ausgewählten Linienroute.

Sie können die Werte in weiß hinterlegten Feldern direkt bearbeiten, indem Sie in das gewünschte Feld doppelklicken.

Sie können das Register Verlauf und Fahrzeitprofile auch im tabellarischen Fahrplan aufrufen, indem Sie die entsprechende Fahrt markieren und im Kontextmenü den Eintrag Fahrzeitprofil bearbeiten auswählen. Im Fahrplan darf dabei nur eine Fahrt markiert sein.
Name oder Fahrzeugkombination eines Fahrzeitprofils ändern

Sie können den Namen oder die Fahrzeugkombination eines Fahrzeitprofils ändern.

Stellen Sie sicher, dass im Fenster Linienroute bearbeiten das Register Verlauf und Fahrzeitprofile ausgewählt ist.

Klicken Sie auf das Fahrzeitprofil, das Sie bearbeiten möchten.

Klicken Sie auf das Symbol  Aktuelles Fahrzeitprofil bearbeiten.

Das Fenster Fahrzeitprofil <Name> bearbeiten öffnet sich.

Ändern Sie den Namen und/oder die Fahrzeugkombination des Fahrzeitprofils.

Das Fahrzeitprofil wird geändert.

Zeiten für ein Fahrzeitprofil setzen

Sie können die Fahr- und Haltezeiten eines Fahrzeitprofils neu berechnen.

Stellen Sie sicher, dass im Fenster Linienroute bearbeiten das Register Verlauf und Fahrzeitprofile ausgewählt ist.

Klicken Sie auf das Fahrzeitprofil, das Sie bearbeiten möchten.

Klicken Sie auf das Symbol  Neue Zeiten für das aktuelle Fahrzeitprofil setzen.

Das Fenster Fahr- und Haltezeiten für Fahrzeitprofil <Name> setzen öffnet sich.

Element

Beschreibung

Nur aktive Fahrzeitprofilelemente

Ist die Option ausgewählt, werden passive Fahrzeitprofilelemente nicht aktualisiert.

Werte addieren

Ist die Option ausgewählt, werden neue Werte zu bestehenden Werten addiert.

Ist die Option nicht ausgewählt, werden bestehende Werte durch neue Werte ersetzt.

Fahrzeitprofilelemente ohne Fahrten

Sie haben folgende Möglichkeiten, Fahrzeitprofilelemente zu behandeln, die nicht von Fahrplanfahrten genutzt werden.

Zeiten für alle (aktiven) Fahrzeitprofilelemente setzen

Ist diese Option ausgewählt, werden die neuen Fahr- und Haltezeiten für alle (aktiven) Fahrzeitprofilelemente gesetzt.

Fahr- und Haltezeit für Fahrzeitprofilelemente ohne Fahrten auf 0 setzen

Ist diese Option ausgewählt, wird die Fahr-und Haltezeit von Fahrzeitprofilelementen ohne Fahrten auf 0 gesetzt.

Fahrzeitprofilelemente ohne Fahrten wenn möglich löschen, sonst Zeiten auf 0 setzen

Ist diese Option ausgewählt, werden Fahrzeitprofilelemente ohne Fahrten gelöscht. Ist dies nicht möglich, werden die Fahr- und Haltezeiten auf 0 gesetzt. Hinweis

Dies ist nicht möglich am ersten oder letzten Element eines Fahrzeitprofils, wenn an diesem Element eine Fahrplanfahrt oder ein Fahrplanfahrtabschnitt beginnt oder endet, oder wenn an diesem Element eine Koppel- oder Koordinierungsgruppe beginnt oder endet.

Fahrzeiten

aktualisieren

Ist die Option ausgewählt, werden die entsprechenden Fahrzeiten des aktuellen Fahrzeitprofils gemäß den aktuellen Einstellungen neu berechnet und ausgegeben.

aus Streckenfahrzeit

Ist die Option ausgewählt, werden die Fahrzeiten aus den Streckenfahrzeiten des jeweiligen ÖVSys verwendet.

aus Systemrouten

Ist die Option ausgewählt, werden die Zeiten aus Systemrouten verwendet.

aus Streckenattribut

Ist die Option ausgewählt, können Sie einen Gewichtungsfaktor sowie ein Streckenattribut oder Subattribut festlegen, aus dem die Fahrzeit berechnet wird.

aus Fahrzeitprofilverlaufs-Attribut

Ist die Option ausgewählt, können Sie einen Gewichtungsfaktor sowie ein Fahrzeitprofil-Verlaufsattribut oder Subattribut festlegen, aus dem die Fahrzeit berechnet wird.

konstant

Ist die Option ausgewählt, können Sie eine konstante Fahrzeit eingeben.

 

Runden auf

Über die Auswahlliste können Sie die Anzahl Sekunden wählen, auf die gerundet werden soll.

Zeiten nur setzen, wenn FzgKomb passt

Ist die Option ausgewählt, werden statt der Standardzeit der Systemroute die spezifischen Fahrzeugkombinations-Zeiten verwendet.

Hinweis

Die Option ist nur verfügbar, wenn die Option aus Systemrouten gewählt wurde.

nur aktive Strecken berücksichtigen

Ist die Option ausgewählt, werden passive Strecken nicht berücksichtigt.

Hinweis

Die Option ist nur verfügbar, wenn die Option aus Streckenfahrzeit oder aus Streckenattribut gewählt wurde.

(Ober-)Abbieger berücksichtigen

Ist die Option ausgewählt, können Sie einen Gewichtungsfaktor sowie ein Abbieger-/Oberabbiegerattribut des IV festlegen, das bei der Berechnung der Fahrzeiten berücksichtigt wird.

Hinweise

Die Option ist nur verfügbar, wenn die Option aus Streckenfahrzeit oder aus Streckenattribut gewählt wurde.

Bei der Auswahl des Abbiegerattributs überprüft , ob ein Oberabbiegerattribut mit der gleiche Attribut-ID vorhanden ist. Ist das der Fall, wird das Oberabbiegerattribut übernommen.

Haltezeiten

aktualisieren

Ist die Option ausgewählt, werden die entsprechenden Haltezeiten des aktuellen Fahrzeitprofils gemäß den aktuellen Einstellungen neu berechnet und ausgegeben.

Hinweise

Werden die Haltezeiten aktualisiert, kann das folgende Auswirkungen auf das Runden von Fahrzeiten haben:

Werden die Haltezeiten aktualisiert, bleibt die aktuelle Fahrzeit trotz Runden als Summe der Fahrzeiten Von-Haltepunkt Nach-Haltepunkt erhalten; einem Abschnitt kann bei Bedarf eine Fahrzeit = 0 zugeordnet werden.

Werden die Haltezeiten nicht aktualisiert, können gerundete Zeiten von bisherigen Fahrzeiten abweichen.

aus Haltepunktzeit

Ist die Option ausgewählt, können Standard-Haltezeiten einzelner Haltepunkte verwendet werden.

aus Haltepunktattribut

Ist die Option ausgewählt, können Sie einen Gewichtungsfaktor sowie ein Haltepunktattribut oder Subattribut festlegen, aus dem die Haltezeit berechnet wird.

aus Fahrzeitprofilverlaufsattribut

Ist die Option ausgewählt, können Sie einen Gewichtungsfaktor sowie ein Fahrzeitprofil-Verlaufsattribut oder Subattribut festlegen, aus dem die Haltezeit berechnet wird.

konstant

Ist die Option ausgewählt, können Sie eine konstante Haltezeit eingeben.

Runden auf

Über die Auswahlliste können Sie die Anzahl Sekunden wählen, auf die gerundet werden soll.

Die ausgewählten Zeiten werden aktualisiert.

Sie können die Ankunft und Abfahrt nachträglich im Fenster Linienroute bearbeiten editieren, außer der Ankunft und Abfahrt am ersten Element sowie der Abfahrt am letzten Element.
Im Verfahrensablauf Fahr- und Haltezeiten setzen

Klicken Sie in der Symbolleiste auf das Symbol  Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Fahr- und Haltezeiten setzen aus der Kategorie Sonstiges ein (siehe „Verfahrensablauf festlegen und starten“ auf Seite 1).

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist und klicken Sie im Bereich Aktionen auf die Schaltfläche Bearbeiten.

Das Fenster Fahr- und Haltezeiten setzen öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (siehe „Zeiten für ein Fahrzeitprofil setzen“ auf Seite 1).

Führen Sie das Verfahren aus (siehe „Verfahren im Verfahrensablauf ausführen“ auf Seite 1).

Die Zeiten für die Fahrzeitprofile werden gesetzt.

Zeiten für Fahrzeitprofil-Verläufe setzen

Sie können Fahr- und Haltezeiten für Fahrzeitprofil-Verläufe erzeugen.

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Linien.

Das Kontextmenü Linien öffnet sich.

Wählen Sie den Eintrag Fahr- und Haltezeiten setzen.

Das Fenster Mehrfach-Bearbeiten Fahrzeitprofil-Verläufe öffnet sich.

Ist der Netzobjekttyp Linien ausgewählt, können Sie das Fenster Mehrfach-Bearbeiten Fahrzeitprofil-Verläufe auch über das Menü Bearbeiten > Fahr- und Haltezeiten setzen aufrufen.

Stellen Sie sicher, dass das Register Sonderfunktionen aktiv ist.

Wählen Sie aus, ob Sie Zeiten für alle oder alle aktiven Fahrzeitprofil-Verläufe setzen möchten.

Element

Beschreibung

Nur aktive

Ist die Option ausgewählt, werden nur aktive Fahrzeitprofil-Verläufe berücksichtigt.

Klicken Sie auf die Schaltfläche Fahr- und Haltezeiten setzen.

Das Fenster Zeiten für Fahrzeitprofile setzen öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (siehe „Zeiten für ein Fahrzeitprofil setzen“ auf Seite 1).

Für alle oder alle aktiven Fahrzeitprofil-Verläufe werden Fahr- und Haltezeiten erzeugt.

Klicken Sie auf die Schaltfläche Schließen.

Abfahrtszeiten der Fahrplanfahrten verschieben

Sie können alle Fahrplanfahrten eines ausgewählten Fahrzeitprofils verschieben.

Stellen Sie sicher, dass im Fenster Linienroute bearbeiten das Register Verlauf und Fahrzeitprofile ausgewählt ist.

Klicken Sie auf das Fahrzeitprofil, das Sie bearbeiten möchten.

Klicken Sie auf das Symbol  Fahrplanfahrten des aktuellen Fahrzeitprofils zeitlich verschieben.

Das Fenster Fahrplanfahrt(en) verschieben öffnet sich.

Element

Beschreibung

Fahrplanfahrt(en) verschieben um

Geben Sie die gewünschte Zeit ein, um die die Fahrplanfahrten verschoben werden sollen.

Hinweise

Wenn Sie keine Einheit angeben, wird der eingegebene Wert als Minuten interpretiert.

Sie können eine Fahrplanfahrt nicht auf einen Zeitpunkt vor 00:00 Uhr verschieben. Wenn Sie das versuchen, wird eine Warnung angezeigt und die bisherige Zeitangabe bleibt unverändert.

früher/später

Abhängig von der gewählten Option werden die Fahrplanfahrten in die gewünschte Richtung verschoben.

Die Fahrplanfahrten werden verschoben.

Referenzhaltepunkt setzen

Alle Fahrplanfahrten enthalten einen Referenzhaltepunkt, an dem die Abfahrtszeit oder Ankunftszeit sämtlicher Fahrplanfahrten bei Veränderungen von Fahr- und Haltezeiten fixiert wird.

Wird kein Referenzhaltepunkt gesetzt, so ist der erste Haltepunkt standardmäßig der Referenzhaltepunkt.

Stellen Sie sicher, dass im Fenster Linienroute bearbeiten das Register Verlauf und Fahrzeitprofile ausgewählt ist.

Klicken Sie auf das Fahrzeitprofil, das Sie bearbeiten möchten.

Klicken Sie im rechten Teilfenster mit der rechten Maustaste in die Zeile, die sie fixieren möchten.

Ein Kontextmenü öffnet sich.

Fixieren Sie entweder die Abfahrtszeit oder die Ankunftszeit, indem Sie Eintrag Abfahrtszeit fixieren oder Ankunftszeit fixieren wählen.

Die ausgewählte Zeile wird grafisch hervorgehoben und in der Spalte Profilpunkt wird das Symbol  angezeigt.

Sie heben die Fixierung des Referenzhaltepunkts auf, indem Sie im Kontextmenü die Option Fahrplanfahrt-Fixierung aufheben auswählen.

Der Referenzhaltepunkt entspricht dem Attribut Referenz-Element-Index, auf das Sie in Fahrzeitprofil-Listen zugreifen können (siehe „Mit Listen arbeiten“ auf Seite 1).

Änderungen des Referenzhaltepunkts wirken sich nicht unmittelbar aus, sondern erst bei nachfolgenden Operationen.

Verschiebungen von Fahrplanfahrten berücksichtigen den Referenzhaltepunkt nicht. Das gilt sowohl für explizites Verschieben, beispielsweise im grafischen oder tabellarischen Fahrplan (siehe „Grafischen und tabellarischen Fahrplan verwenden“ auf Seite 1), als auch für durch Kopplungen bedingte Anpassverschiebungen von Fahrplanfahrten auf anderen Fahrzeitprofilen (siehe „Mehrere Fahrzeitprofile koppeln“ auf Seite 1).

Fahr- und Haltezeiten eines Abschnitts bearbeiten

Die Schaltflächen, die Sie in Spalte <> (Abgleich von Fahr- und Haltezeiten) finden, geben Auskunft über gleiche und abweichende Fahrzeiten auf einem gemeinsamen Abschnitt (Von-Haltepunkt – Nach-Haltepunkt) und dienen der Anpassung der Fahr- und Haltezeiten auf diesem Abschnitt. Über die Schaltflächen können Sie Halte- und Fahrzeiten einzelner Abschnitte bearbeiten. Sie haben folgende Bedeutungen:

Element

Beschreibung

Mehrere Linienrouten mit gleicher Fahrzeit befahren diesen Abschnitt.

Mehrere Linienrouten, deren Fahrzeiten unterschiedlich sind, befahren diesen Abschnitt.

Der Abschnitt ist nur im aktuellen Fahrzeitprofil enthalten.

Stellen Sie sicher, dass im Fenster Linienroute bearbeiten das Register Verlauf und Fahrzeitprofile ausgewählt ist.

Klicken Sie auf das Register des Fahrzeitprofils, das Sie bearbeiten möchten.

Klicken Sie in der Spalte <> auf die gewünschte Schaltfläche oder .

Das Fenster Vergleich von Fahr- und Haltezeiten zwischen HP <Name> und HP <Name> öffnet sich.

Das Fenster öffnet sich nicht, wenn Sie auf die Schaltfläche klicken. Sie können die Zeiten auf diesem Abschnitt nicht im separaten Fenster bearbeiten, da der Abschnitt ausschließlich im aktuellen Fahrzeitprofil enthalten ist.

Im Fenster werden alle Fahrzeitprofile aufgelistet, die auf dem Abschnitt verkehren. Wenn es Systemrouten gibt, die auf dem Abschnitt verkehren, so werden auch diese aufgelistet. Für jede Fahrzeugkombination erfolgt ein Systemrouteneintrag, auch für die Standardeinstellung. Systemrouteneinträge sind farbig hervorgehoben.

Sie können im Fenster auch die Fahrzeiten von Systemrouten ändern.

Wenn Sie keine Einheit angeben, wird Ihre Eingabe als Minuten interpretiert.

Element

Beschreibung

Kontrollkästchen

Über die Kontrollkästchen am Zeilenbeginn können Sie jedes Fahrzeitprofil und jede Systemroute einzeln für die Bearbeitung auswählen und die Fahr- und Haltezeiten bearbeiten.

Tipp

Sie können einzelne Zellen auch manuell bearbeiten, indem Sie in die Zelle doppelklicken.

Alle an / Alle aus

Über die Schaltflächen können Sie alle Fahrzeitprofile und Systemrouten gemeinsam auswählen oder aus der Auswahl entfernen.

nur aktive Fahrzeitprofile anzeigen

Ist die Option ausgewählt, werden nur die aktiven Fahrzeitprofile im Fenster aufgelistet.

als Haltezeit für alle

Sie können in das Eingabefeld unten links eine Haltezeit eingeben und diese Zeit für alle ausgewählten Fahrzeitprofile übernehmen, indem Sie auf die Schaltfläche als Haltezeit für alle klicken. Die Haltezeit wird dann für alle ausgewählten Zeilen übernommen.

Hinweis

Für Systemrouten können Sie keine Haltezeiten definieren.

als Fahrzeit für alle

Sie können in das Eingabefeld unten links eine Fahrzeit eingeben und diese Zeit für alle ausgewählten Fahrzeitprofile und Systemrouten übernehmen, indem Sie auf die Schaltfläche als Fahrzeit für alle klicken. Die Fahrzeit wird dann für alle ausgewählten Zeilen übernommen.

Für alle

Sie können eine in der Spalte Fahrzeit angegebene Zeit in alle anderen Zeilen übernehmen, indem Sie in der betreffenden Zeile auf die Schaltfläche Für alle klicken.

Die geänderten Zeiten werden übernommen.

Tarifpunkte für Fahrzeitprofile festlegen

Mit Hilfe von Tarifpunkten können die Fahrpreise abgebildet werden. Sie gehen in die Widerstandsfunktion ein und werden zur Erlösberechnung verwendet.

Stellen Sie sicher, dass im Fenster Linienroute bearbeiten das Register Verlauf und Fahrzeitprofile ausgewählt ist.

Klicken Sie auf das Register des Fahrzeitprofils, das Sie bearbeiten möchten.

Wählen Sie über das Symbol  Attribute wählen für rechte Tabelle die folgenden Attribute aus und legen Sie die gewünschten Werte fest.

Element

Beschreibung

Anzahl Tarifpunkte

Anzahl Tarifpunkte, die bei Benutzung des Fahrzeitprofilelements vergeben werden

Anzahl Tarifpunkte für Ausstieg

Anzahl Tarifpunkte, die beim Ausstieg an diesem Fahrzeitprofilelement vergeben werden

Anzahl Tarifpunkte für Einstieg

Anzahl Tarifpunkte, die beim Einstieg an diesem Fahrzeitprofilelement vergeben werden

Anzahl Tarifpunkte für Durchfahrt

Anzahl Tarifpunkte, die bei der Durchfahrt an diesem Fahrzeitprofilelement vergeben werden

Die Werte dürfen auch negativ sein.

Bei der Umlegung werden alle Tarifpunkte eines Fahrzeitprofils addiert. Dabei wird für jedes Fahrzeitprofil geprüft, ob die Summe größer oder gleich null ist.

In der Spalte Anzahl Strecken-Tarifpunkte können Sie die Anzahl Tarifpunkte anzeigen, die sich am Fahrzeitprofilelement durch Aufsummieren der Tarifpunkte aller Strecken ergeben (siehe „Strecken: Register ÖV-VSys“ auf Seite 1).

Sie können die Werte auch beispielsweise über die Funktion Mehrfach-Bearbeiten bearbeiten (siehe „Attributwerte von Netzobjekten bearbeiten“ auf Seite 1).
Unbenutzte Profilpunkte löschen

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Linien.

Wählen Sie den Eintrag Unbenutzte Fahrzeitprofilpunkte löschen.

Das Fenster Mehrfach-Bearbeiten Fahrzeitprofil-Verläufe öffnet sich.

Ist der Netzobjekttyp Linien ausgewählt, können Sie das Fenster Mehrfach-Bearbeiten Fahrzeitprofil-Verläufe auch über das Menü Bearbeiten > Unbenutzte Fahrzeitprofilpunkte löschen aufrufen.

Stellen Sie sicher, dass das Register Sonderfunktionen aktiv ist.

Wählen Sie aus, ob Sie die unbenutzten Profilpunkte aller oder aller aktiven Fahrzeitprofil-Verläufe löschen möchten.

Element

Beschreibung

Nur aktive

Ist die Option ausgewählt, werden nur aktive Fahrzeitprofil-Verläufe berücksichtigt.

Klicken Sie auf die Schaltfläche Unbenutzte Profilpunkte löschen.

Die unbenutzten Profilpunkte aller oder aller aktiven Fahrzeitprofil-Verläufe werden gelöscht.

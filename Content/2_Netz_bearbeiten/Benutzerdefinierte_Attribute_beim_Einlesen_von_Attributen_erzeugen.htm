Benutzerdefinierte Attribute beim Einlesen von Attributen erzeugen

Sie können benutzerdefinierte Attribute bei Bedarf direkt beim Einlesen einer Attributdatei als Zielattribute anlegen. Die angelegten Attribute übernehmen beim Einlesen die Werte der Quellattribute.

Lesen Sie eine Attributdatei ein (Attributsdatei lesen).

Das Fenster Attribute lesen öffnet sich.

Klicken Sie auf die Schaltfläche Erzeugen.

Das Fenster Typ-Auswahl öffnet sich.

Wenn Sie nur ein benutzerdefiniertes Attribut einfügen möchten, klicken Sie in der gewünschten Zeile auf die Schaltfläche Neu und legen ein benutzerdefiniertes Attribut an (Benutzerdefiniertes Attribut einfügen).

Wählen Sie in der Auswahlliste einen Datentyp aus (Benutzerdefiniertes Attribut einfügen).

Für alle nicht zugeordneten Quellattribute werden benutzerdefinierte Attribute vom gewählten Typ erzeugt.

Prüfergebnis anzeigen und Fehler beheben

Stellen Sie sicher, dass der Prüfvorgang durchgeführt und das Fenster Netz prüfen offen ist (Netz prüfen).

Die Ergebnisse sind im Fenster für jede Option auf einer eigenen Schaltfläche angezeigt.

Element

Beschreibung

ok

Keine Fehler gefunden

!

Ein Fehler liegt vor.

Klicken Sie auf die Schaltfläche ! der gewünschten Option.

Es enthält eine Beschreibung des Fehlers und bei manchen Prüfoptionen Vorschläge zur direkten Fehlerbehebung.

Wenn Sie für eine Option, bei der keine Fehler gefunden wurden, auf die Schaltfläche ok klicken, öffnet sich ein Hinweis, dass kein Fehler vorliegt.

Klicken Sie auf die gewünschte Schaltfläche.

Element

Beschreibung

OK

Schließt das Fenster. Den angezeigten Fehler müssen Sie nach Verlassen des Fensters Netz prüfen manuell im Netz korrigieren.

Aktiv setzen

Über die Schaltfläche setzen Sie alle Netzobjekte aktiv, für die ein Fehler entdeckt wurde. Alle anderen Netzobjekte dieser Netzobjekttypen werden passiv gesetzt. So können Sie im Netz leicht erkennen, welche Netzobjekte fehlerhaft sind und sie anschließend bearbeiten, um die Fehler zu beheben.

Direkte Fehlerbehebung

Über die entsprechende Schaltfläche werden die Fehler direkt im Netz behoben.

Löschen

Die fehlerhaften Knoten werden aus dem Netz gelöscht.

Sperren

Die fehlerhaften Abbieger oder Oberabbieger werden für die getroffene Verkehrssystemauswahl gesperrt.

Neu belegen

Die Abbiegetypen an den fehlerhaften Knoten werden mit korrekten Werten belegt.

Öffnen

Für die fehlerhaften Strecken werden U-Turns eingefügt oder die Gegenrichtung geöffnet.

Zeiten berechnen

Für die fehlerhaften Strecken werden die Zeiten aus der jeweiligen Standardgeschwindigkeit berechnet.

Beheben Sie bei Bedarf weitere Fehler.

Führen Sie bei Bedarf den Prüfvorgang erneut durch, bis alle Fehler behoben sind.

Klicken Sie auf die Schaltfläche Schließen.

Ein Hinweis öffnet sich. Ein detailliertes Fehlerprotokoll wird in der Meldungsdatei Messages.TXT gespeichert.

Sie haben folgende Möglichkeiten, fortzufahren:

  • Wenn Sie sich das Fehlerprotokoll ansehen möchten, klicken Sie auf die Schaltfläche Öffnen. Die Meldungsdatei wird dann in dem für Textdateien registrierten Editor geöffnet.
  • Wenn Sie sich das Fehlerprotokoll nicht ansehen möchten, klicken Sie auf die Schaltfläche OK. Sie können die Meldungsdatei zu einem späteren Zeitpunkt aufrufen (Mit Protokolldateien arbeiten).

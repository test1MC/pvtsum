Attributwerte in der Schnellansicht bearbeiten

Stellen Sie sicher, dass im Netz die Netzobjekte ausgewählt sind, deren Attributwerte Sie ändern möchten.

Markieren Sie in der Schnellansicht ein oder mehrere Felder (Fenster Schnellansicht).

Wenn Sie eine Strecke markieren, zeigt die Schnellansicht nur die Attributwerte für die markierte Richtung an. Sie ändern daher auch nur die Attributwerte einer Streckenrichtung.

Sind mehrere Netzobjekte mit unterschiedlichen Attributwerten markiert, wird statt des Attributwerts ein Sternchen * angezeigt.

Geben Sie den gewünschten Wert in ein markiertes Feld ein und drücken Sie die Taste Enter.

Bei der Eingabe von numerischen Attributwerten mit Längen- oder Zeitbezug müssen Sie die Einheiten berücksichtigen:

  • Wenn Sie nur die Zahl eingeben, wird der Wert in dem Format interpretiert, mit dem das Attribut angezeigt wird. Beispiel: Wenn ein Attribut im Format Minuten angezeigt wird, wird der Wert 5 als 5 Minuten interpretiert.

Das Format können Sie im Attributauswahlfenster ändern (Fenster zur Auswahl mehrerer Attribute).

  • Sie können eine Zahl mit Einheit eingeben. In diesem Fall rechnet den Wert so um, dass er entsprechend dem gewählten Format angezeigt wird. Sie können Längeneinheiten wahlweise mit metrischen (m, km) oder imperialen (ft, mi) Einheiten eingeben.

Beispiel: Wenn ein Attribut im Format Minuten angezeigt wird, können Sie 2h eingeben. Nach der Eingabe enthält das Feld den Wert 120.

Der neue Attributwert wird in allen markierten Feldern angezeigt.

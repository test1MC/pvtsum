Fahrstreifendaten von Knoten zurücksetzen

Die Fahrstreifen einzelner Knoten können Sie im Knoteneditor bearbeiten (Knoteneditor verwenden).

Setzen Sie bei Bedarf die Knoten aktiv, deren Fahrstreifendaten Sie auf die Standardwerte zurücksetzen möchten, über

  • die räumliche Auswahl (Netzobjekte über die räumliche Auswahl aktiv und passiv setzen) und/oder
  • Filterkriterien (Netzobjekte über Filter aktiv und passiv setzen).

Die ausgewählten Knoten sind aktiv.

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Knoten.

Das Kontextmenü Knoten öffnet sich.

Wählen Sie den Eintrag Fahrstreifendefinition.

Das Fenster Mehrfach-Bearbeiten Knoten öffnet sich.

Ist der Netzobjekttyp Knoten ausgewählt, können Sie das Fenster Mehrfach-Bearbeiten Knoten auch über das Menü Bearbeiten > Fahrstreifendefinition aufrufen.

Stellen Sie sicher, dass das Register Sonderfunktionen aktiv und in der Auswahlliste der Eintrag Fahrstreifendefinition gewählt sind.

Wählen Sie aus, ob die Fahrstreifendaten für alle oder alle aktiven Knoten zurückgesetzt werden sollen.

Element

Beschreibung

Nur aktive

Ist die Option ausgewählt, werden nur aktive Knoten berücksichtigt.

Klicken Sie auf die Schaltfläche Fahrstreifendaten zurücksetzen.

Eine Sicherheitsabfrage öffnet sich.

Bestätigen Sie die Sicherheitsabfrage mit OK.

Die Fahrstreifendaten werden zurückgesetzt.

Klicken Sie auf die Schaltfläche Schließen.

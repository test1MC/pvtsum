Geometrie in der schematischen Ansicht bearbeiten

In der schematischen Ansicht können Sie beispielsweise Fahrstreifenaufweitungen einfügen oder entfernen.

Doppelklicken Sie im Netz auf den Knoten oder Oberknoten, den Sie bearbeiten möchten.

Der Knoteneditor öffnet sich und zeigt den ausgewählten Knoten oder Oberknoten an.

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Geometrie ausgewählt ist.

Der Knoten wird in der Geometrieansicht dargestellt.

Nehmen Sie die gewünschten Änderungen vor.

Die Optionen Signalgruppen und Phasen werden in der schematischen Ansicht nur angezeigt, wenn Signalgruppen oder Phasen eingefügt wurden.
Arme markieren

Sie können die einzelnen Arme in der schematischen Ansicht über die farbigen Auswahlkästchen neben den Armen markieren.

Klicken Sie auf das gewünschte Auswahlkästchen.

Der ausgewählte Arm wird in der Tabellenansicht markiert.

Sie können die Attribute des markierten Arms direkt in der Tabelle bearbeiten.

Wählen Sie bei Bedarf weitere Arme aus.

Aufweitungen einfügen

Sie können einzelnen Armen in der schematischen Ansicht links oder rechts Aufweitungen hinzufügen.

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Geometrie ausgewählt ist.

Stellen Sie sicher, dass in der schematischen Ansicht die Option Fahrstreifen ausgewählt ist.

Klicken Sie auf das Auswahlkästchen des Armes, dem Sie eine Aufweitung hinzufügen möchten.

Am ausgewählten Arm werden die Kästchen zum Einfügen der Aufweitungen angezeigt. Diese sind mit einem Pluszeichen gekennzeichnet.

Klicken Sie auf das Kästchen zum Einfügen der gewünschten Aufweitung.

Die Aufweitung wird eingefügt.

Sie können die Aufweitung wieder entfernen, indem Sie auf das Kästchen neben der neu eingefügten Aufweitung klicken. Das Kästchen ist mit einem x gekennzeichnet.
Fahrstreifen und Fahrstreifenabbieger markieren

Sobald Sie einen Arm markiert haben, können Sie einzelne Fahrstreifen markieren.

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Geometrie ausgewählt ist.

Stellen Sie sicher, dass in der schematischen Ansicht die Option Fahrstreifen ausgewählt ist.

Markieren Sie den gewünschten Arm, indem Sie mit der Maus auf das jeweilige Kästchen klicken.

Die Kästchen in der schematischen Ansicht verfügen über eine Quickinfo, die jeweils die Aktion beschreibt, die ein Klick auf das Kästchen auslöst.

Auf den Fahrstreifen werden weitere Auswahlkästchen zum Markieren der Fahrstreifen angezeigt.

Markieren Sie den gewünschten Fahrstreifen, indem Sie auf das Auswahlkästchen Fahrstreifen markieren klicken.

Die Fahrstreifenabbieger und weitere Auswahlkästchen zum Markieren der Fahrstreifenabbieger werden angezeigt, wenn der Fahrstreifen über Fahrstreifenabbieger verfügt.

Markieren Sie bei Bedarf den gewünschten Fahrstreifenabbieger, indem Sie auf das Auswahlkästchen Fahrstreifenabbieger markieren klicken.

Weitere Kästchen zum Entfernen der Fahrstreifenabbieger werden angezeigt.

Über die Kästchen Fahrstreifenabbieger löschen und Fahrstreifenabbieger einfügen können Sie Fahrstreifenabbieger löschen oder hinzufügen.

Die Attribute markierter Fahrstreifen können Sie direkt in der Tabellenansicht bearbeiten. Ein Fahrstreifen, der in der schematischen Ansicht markiert ist, ist auch in der Tabellenansicht markiert.
Furten einfügen

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Geometrie ausgewählt ist.

Wählen Sie in der schematischen Ansicht die Option Furten.

Die möglichen Furten werden durch gestrichelte Linien angezeigt.

Fügen Sie eine Furt ein, indem Sie auf das gewünschte Kästchen Furt einfügen mit dem + klicken.

Die Furt wird eingefügt.

Fügen Sie bei Bedarf weitere Furten ein.

Sie können eine Furt entfernen, indem Sie auf das Kästchen Furt löschen mit dem x klicken.

Wenn Sie eine Signalanlage einfügen, werden alle Furten am ausgewählten Knoten oder Oberknoten automatisch eingefügt sofern noch keine vorhanden sind.
Detektoren einfügen

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Geometrie ausgewählt ist.

Wählen Sie in der schematischen Ansicht die Option Detektoren.

Fahrstreifen, an denen Sie Detektoren einfügen können, sind mit einem + markiert.

Fügen Sie einen Detektor ein, indem Sie auf das gewünschte Kästchen Detektor für diesen Fahrstreifen einfügen mit dem + klicken.

Das Fenster Detektor einfügen wird geöffnet.

Geben Sie die gewünschten Daten ein (Attribute und Optionen von Detektoren).

Der Detektor wird eingefügt und in der Darstellung als kleines Quadrat am Knoten angezeigt.

Fügen Sie bei Bedarf weitere Detektoren ein.

Sie können einen Detektor entfernen, indem Sie ihn in der Tabellenansicht markieren und die Taste entf drücken.

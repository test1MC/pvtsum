Suchergebnisse bearbeiten

Sie können die Suchergebnisse im Fenster <Netzobjekttyp> suchen über das Kontextmenü oder das Fenster Schnellansicht weiter bearbeiten. Je nach Netzobjekttyp können sich die verfügbaren Aktionen unterscheiden.

Attribute im Fenster Schnellansicht bearbeiten

Sie können die Attribute der Netzobjekte aus der Liste im Fenster Schnellansicht direkt bearbeiten. Das Fenster <Netzobjekttyp> suchen kann dabei geöffnet bleiben.

Stellen Sie sicher, dass das Fenster <Netzobjekttyp> suchen geöffnet ist und die gewünschte Suche durchgeführt wurde (Netzobjekte im Netz suchen).

Stellen Sie sicher, dass in der Symbolleiste des Fensters <Netzobjekttyp> suchen das Symbol ausgewählt ist. Dadurch werden in der Liste markierte Netzobjekte auch in anderen Fenstern markiert.

Markieren Sie in der Suchergebnisliste das Netzobjekt, das Sie bearbeiten möchten.

Das Netzobjekt wird im Fenster Schnellansicht angezeigt.

Ändern Sie die gewünschten Attributwerte des Netzobjekts (Attributwerte in der Schnellansicht ändern).

Suchergebnisse über das Kontextmenü bearbeiten

Sie können verschiedene Funktionen über das Kontextmenü aufrufen, mit denen Sie die Suchergebnisse bearbeiten können. Je nach Netzobjekttyp können sich die verfügbaren Funktionen unterscheiden.

Stellen Sie sicher, dass das Fenster <Netzobjekttyp> suchen geöffnet ist und die gewünschte Suche durchgeführt wurde (Netzobjekte im Netz suchen).

Markieren Sie in der Suchergebnisliste das Netzobjekt, das Sie bearbeiten möchten.

Klicken Sie mit der rechten Maustaste auf die markierte Zeile.

Ein Kontextmenü öffnet sich.

Wählen Sie den gewünschten Eintrag.

Element

Beschreibung

Autozoom

Der Netzausschnitt wird vergrößert und das Netzobjekt wird in der Mitte des Netzausschnitts angezeigt. Es können auch mehrere Netzobjekte markiert sein. Möglich für alle Netzobjekte.

Bearbeiten

Öffnet das Fenster zum Bearbeiten des Netzobjekts

Tipp

Alternativ können Sie auf die gewünschte Zeile doppelklicken.

Linienroute bearbeiten

Öffnet das Fenster Linienroute bearbeiten (Attributwerte von Linienrouten ändern)

Fahrplan (tabellarisch)

Öffnet das Fenster Fahrplan (tabellarisch) (Fahrplaneditor verwenden). Möglich für alle Netzobjekte der Linienhierarchie. Es können auch mehrere Netzobjekte markiert sein.

Fahrplan (grafisch)

Öffnet das Fenster Fahrplan (grafisch) (Fahrplaneditor verwenden). Möglich für alle Netzobjekte der Linienhierarchie. Es können auch mehrere Netzobjekte markiert sein.

Beide Fahrpläne

Öffnet beide Fahrplanfenster

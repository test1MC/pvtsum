Wege einer Wegemenge oder eines Nachfragesegments konvertieren

Wählen Sie die Wege aus, die Sie konvertieren möchten (Wege auswählen).

Klicken Sie mit der rechten Maustaste in die Netzdarstellung.

Das Kontextmenü IV-Wege öffnet sich.

Wählen Sie den Eintrag Alle konvertieren.

Das Fenster Parameter: Wege konvertieren öffnet sich.

Sie können das Verfahren auch über das Menü Rechnen > Verfahrensablauf > Verfahren Wege konvertieren aufrufen.

Die Elemente im Fenster unterscheiden sich, abhängig davon, ob Sie das Fenster über das Kontextmenü öffnen oder im Fenster Verfahrensablauf.

Element

Beschreibung

Quelle

In der Auswahlliste können Sie die Wegemenge oder das Nachfragesegment auswählen, dessen zugeordnete Wege Sie konvertieren möchten.

Hinweis

Wenn Sie das Verfahren über das Kontextmenü aufrufen, besteht diese Auswahlmöglichkeit nicht.

Zeitintervall

Hier können Sie ein Zeitintervall auswählen. Konvertiert werden dann die Belastungen des ausgewählten Zeitintervalls.

Auswahl

Hier können Sie auswählen, ob alle Wege, jene aus den Spinne-Routen oder jene des Beziehungsfilters konvertiert werden sollen.

Faktor

Hier können Sie einen Faktor eingeben, mit dem die Belastungen der Quellwege multipliziert werden.

Ziel

In der Auswahlliste können Sie die Wegemenge oder das Nachfragesegment auswählen, welche(s) die Wege aufnimmt.

Zeitintervall

Hier können Sie ein Zeitintervall auswählen, das die Belastungen aufnimmt.

Konfliktbehandlung

Wenn ein Quellweg und ein Zielweg identisch sind, bestehen folgende Optionen:

  • Alle ersetzen

Alle existierenden Zielwege werden vor der Konvertierung gelöscht.

  • Summe

Die Belastung der Quellwege und die Belastung existierender Zielwege werden addiert.

  • Überschreiben

Identische Zielwege werden durch die Quellwege überschrieben.

Die Wege werden konvertiert.

Wenn Sie Wege aus Umlegungen konvertieren, entstehen nach der Konvertierung immer Wege, die an einem Bezirk starten und enden.

Signalzeiten phasenbasierter Steuerungen bearbeiten

Bei phasenbasierten Signalsteuerungen wird die Geometrie durch eine schematische Darstellung der Phasenfolgen ersetzt. Die Balkenansicht zeigt einen Balken je Phase an.

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Signalzeiten ausgewählt ist.

Je Phase wird im oberen Bereich ein Teilfenster und im unteren Bereich ein zugehöriger Balken angezeigt.

Nehmen Sie die gewünschten Änderungen vor.

Einige Einträge im Kontextmenü entsprechen jenen für signalgruppenbasierte Steuerungen. Sie werden dort erklärt (Signalzeiten signalgruppenbasierter Steuerungen bearbeiten).
Zuordnung von Signalgruppen zu Phasen manuell ändern

Jedes Quadrat im oberen Bereich bildet eine Phase ab. Die Pfeile stellen Signalgruppen dar. Die Zuordnung von Signalgruppen zu Phasen können Sie durch Klicken dieser Pfeile steuern. Alle Signalgruppen, die einer Phase zugeordnet sind, haben grün.

Markieren Sie im oberen Bereich das Quadrat der Phase, deren Signalgruppenzuordnung Sie ändern möchten.

Wenn Sie mit der Maus auf einen Pfeil zeigen, wird dieser Pfeil und ebenso alle Pfeile, die die gleiche Signalgruppe repräsentieren, in einer anderen Farbe angezeigt.

Ändern Sie die Farbe der gewünschten Pfeile von Rot zu Grün oder von Grün zu Rot, indem Sie auf den gewünschten Pfeil klicken.

Die Pfeile ändern ihre Farbe und die Zuordnung zur Phase.

Phasen einfügen

Für Knoten mit phasenbasierten Steuerungen können Sie Phasen einfügen.

Alternativ können Sie Phasenschablonen verwenden (Phasen aus Schablone erzeugen).

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Signalzeiten ausgewählt ist.

Klicken Sie in der Symbolleiste des Knoteneditors auf das Symbol  Phase einfügen.

Alternativ können Sie über den Eintrag im Kontextmenü Phase hinzufügen eine weitere Phase einfügen.

Das Fenster Phase einfügen öffnet sich.

Element

Beschreibung

Nummer

Eindeutige Nummer der Phase

Name

Langbezeichner der Phase

Grünzeitanfang

Angabe des Grünzeitanfangs der Phase

Grünzeitende

Angabe des Grünzeitendes der Phase

Mindestgrünzeit

Angabe der Mindestgrünzeit in Sekunden

Hinweis

Der Wert wird bei der Grünzeitoptimierung wirksam (LSA-Umlauf- und Grünzeiten für eine einzelne LSA optimieren).

Die Phase wird eingefügt und in der Balkenanzeige als ein Balken angezeigt.

Phasen bearbeiten

Sie können die Phasen manuell in der Ansicht Signalzeiten bearbeiten, indem Sie die Balken mit der Maus auf die gewünschte Breite ziehen. Der Unterschied zur Signalgruppenansicht ist, dass das Verschieben von Grünzeiten sich auf alle Signalgruppen einer Phase auswirkt (Signalzeiten signalgruppenbasierter Steuerungen bearbeiten).

Außerdem können Sie die Attribute von Phasen in der Tabelle der Ansicht Geometrie oder in der Liste Phasen bearbeiten (Liste öffnen).
Phasen löschen

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Signalzeiten ausgewählt ist.

Klicken Sie im unteren Teilfenster des Knoteneditors mit der rechten Maustaste auf den Balken der Phase, die Sie entfernen möchten.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Phase löschen.

Die Phase wird entfernt.

Alternativ können Sie Phasen löschen, indem Sie in der Symbolleiste auf das Symbol  Phase löschen klicken.
Phasenschablonen einlesen

Statt Phasen und Signalgruppen manuell zu erstellen, können Sie vordefinierte Phasenschablonen verwenden (Phasen aus Schablone erzeugen). Diese können Sie als Dateien im Format *.net additiv einlesen.

Zeigen Sie im Menü Datei auf Öffnen.

Wählen Sie den Eintrag Netz.

Das Fenster Netz öffnen öffnet sich.

Die vordefinierten Phasenschablonen sind im Installationsverzeichnis Ihrer -Installation im Unterverzeichnis ...\\Data\Node_Templates enthalten.

Die Phasenschablonen-Dateinamen lauten Signaltemplateslefthandtraffic.net und Signaltemplatesrighthandtraffic.net.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Das Fenster Netz lesen öffnet sich.

Lesen Sie das Netz additiv ein (Netzdaten additiv einlesen).

Die Netzdaten werden eingelesen.

Sie können die eingelesenen Schablonendaten im Menü Netz > Phasenschablonen bearbeiten.

Sie können die Namen der Phasenschablonen bearbeiten, indem Sie im Fenster Schablonen in die gewünschte Zeile doppelklicken.

Phasenschablonen oder Phasenschablonensets löschen

Wählen Sie im Menü Netz den Eintrag Phasenschablonen.

Das Fenster Schablonen öffnet sich. Alle im Netz vorhandenen Phasenschablonen und Phasenschablonensets werden angezeigt.

Wählen Sie in der Liste den gewünschten Eintrag aus.

Sie können mehrere Listeneinträge auswählen, indem Sie die Taste Strg drücken, diese gedrückt halten und auf die gewünschten Einträge klicken.

Klicken Sie auf die Schaltfläche  Löschen.

Die ausgewählten Einträge werden entfernt.

Phasen aus Schablone erzeugen

Statt Phasen und Signalgruppen manuell zu erstellen, können Sie vordefinierte Phasenschablonen verwenden. Vordefinierte Phasenschablonen ermöglichen eine schnelle Versorgung eines signalgesteuerten Knotens oder Oberknotens (Grundlagen: Signalisierung).

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Signalzeiten ausgewählt ist.

Klicken Sie mit der rechten Maustaste in das untere Teilfenster des Knoteneditors.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Phasenschablonen anwenden.

Alternativ können Sie das Fenster über das Symbol  Phasenschablonen anwenden in der Symbolleiste aufrufen.

Das Fenster Phasen aus Schablone erzeugen öffnet sich.

Anzahl Phasen

Hinweis

Wenn Sie beispielsweise eine 4 eingeben, werden nur 4-phasige Phasen angezeigt.

Schablone

Phasenreihenfolge

Über die Auswahlliste können Sie die Reihenfolge der Phasen festlegen.

Die Phasen werden anhand der ausgewählten Schablone erzeugt.

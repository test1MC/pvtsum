Eine Systemroute markieren

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Systemrouten.

Das Fenster Systemrouten öffnet sich.

Schränken Sie bei Bedarf die Menge der Systemrouten ein, die angezeigt wird.

Element

Beschreibung

nur aktive

Ist die Option ausgewählt, werden nur aktive Systemrouten angezeigt.

Wählen Sie eine Systemroute in der Liste aus.

Sie können die gesuchte Systemroute auch über das Eingabefeld auswählen. Geben Sie hierzu den Namen der Systemroute in das Feld ein. Die Systemroute wird in der Liste ausgewählt.

Der Verlauf der Systemroute wird im Netz markiert.

Klicken Sie bei Bedarf auf die Schaltfläche Autozoom.

Der Netzausschnitt wird vergrößert und die markierte Systemroute wird in der Mitte des Ausschnitts platziert.

Nutzen Sie die Schaltfläche Initialisieren, wenn Sie die bisher markierten Systemrouten aus der Auswahl entfernen möchten.

Sie können Systemroutenverläufe auch markieren, indem Sie im Netz auf einen Haltepunkt klicken über den eine Systemroute verläuft.

Alternativ können Sie eine Systemroute markieren, indem Sie die entsprechende Zeile in der Liste der Systemrouten auswählen (Listenbereiche auswählen).

Verkehrstage löschen

Sie können den Standard-Verkehrstag täglich nicht löschen.

Wählen Sie im Menü Netz den Eintrag ÖV-Verkehrstage.

Das Fenster Verkehrstage öffnet sich.

Stellen Sie sicher, dass das Register Verkehrstage ausgewählt ist.

Wählen Sie in der Liste den gewünschten Verkehrstag aus.

Sie können mehrere Listeneinträge auswählen, indem Sie die Taste Strg drücken, diese gedrückt halten und auf die gewünschten Einträge klicken.

Klicken Sie auf die Schaltfläche  Löschen.

Der Verkehrstag wird gelöscht.

Wenn Sie einen Verkehrstag löschen, werden alle Fahrplanfahrten, die diesen Verkehrstag verwenden, auf den Standard-Verkehrstag täglich gesetzt.

Richtung der Linienrouten setzen

Mit dieser Funktion können Sie allen oder allen aktiven Linienrouten eine Richtung zuweisen.

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Linien.

Das Kontextmenü Linien öffnet sich.

Wählen Sie den Eintrag Mehrfach-Bearbeiten > Linienrouten.

Das Fenster Mehrfach-Bearbeiten Linienrouten öffnet sich.

Ist der Netzobjekttyp Linien ausgewählt, können Sie das Fenster Mehrfach-Bearbeiten Linienrouten auch über das Menü Bearbeiten > Richtung für Linienrouten setzen aufrufen.

Stellen Sie sicher, dass das Register Sonderfunktionen aktiv ist.

Wählen Sie aus, ob Sie allen oder allen aktiven Linienrouten eine Richtung zuweisen möchten.

Element

Beschreibung

Nur aktive

Ist die Option ausgewählt, werden nur aktive Linienrouten berücksichtigt.

Klicken Sie auf die Schaltfläche Richtung setzen.

Das Fenster Richtung setzen öffnet sich.

Nehmen Sie die gewünschte Einstellung vor.

Element

Beschreibung

Richtung setzen auf

Ist die Option ausgewählt, können Sie allen Linienrouten über die Auswahlliste die Hin- oder Rückrichtung zuweisen.

Richtung umkehren

Ist die Option ausgewählt, wird den Linienrouten die Gegenrichtung ihrer aktuellen Richtung zugewiesen.

Allen oder allen aktiven Linienrouten wird die gewünschte Richtung zugewiesen.

Klicken Sie auf die Schaltfläche Schließen.

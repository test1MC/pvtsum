Filter lesen

Wählen Sie im Menü Filter den Eintrag Filterdatei öffnen.

Das Fenster Filter öffnen öffnet sich.

Sie können das Fenster Filter öffnen auch über Menü Datei > Öffnen > Eintrag Filter aufrufen.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Das Fenster Filter lesen öffnet sich.

Wählen Sie die Netzobjektfilter, die Sie lesen möchten.

Element

Beschreibung

Bestehende Filter vorher initialisieren

Ist die Option ausgewählt, werden alle aktuell gesetzten Filter, die nicht in der Filterdatei enthalten sind, beim Filter lesen zurückgesetzt.

Alle aus

Über die Schaltfläche können Sie alle Netzobjektfilter ausschalten.

Alle an

Über die Schaltfläche können Sie alle Netzobjektfilter auswählen.

Die Netzobjektfilter sind ausgegraut, wenn sie nicht in der Filterdatei enthalten sind.

Die ausgewählten Netzobjektfilter werden gelesen.

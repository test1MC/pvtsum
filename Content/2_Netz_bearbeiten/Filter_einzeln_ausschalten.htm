Filter einzeln ausschalten

Klicken Sie auf das Symbol Filter ein-/ausschalten des gewünschten Netzobjekttyps.

Der Filter wird ausgeschaltet, alle Netzobjekte, die in der räumlichen Auswahl enthalten sind, werden aktiv. Die farbige Hervorhebung des Symbols Filter ein-/ausschalten wird ausgeschaltet.

Alternativ können Sie einen Netzobjektfilter ausschalten, indem Sie die Option Filter verwenden im jeweiligen Filterfenster ausschalten (Filterbedingungen definieren und Filter anwenden), bei hierarchischen Filtern in Kombination mit der Option Filter wirksam für (Mit hierarchischen Filtern arbeiten).

Externe Steuerung einer Signalanlage bearbeiten

Wenn eine Signalanlage den Signalisierungstyp Vissig besitzt, bearbeiten Sie die Steuerungsdaten in einem externen Programm.

Externe Steuerungsdaten von der Übersicht aus öffnen

Wählen Sie im Menü Netz den Eintrag LSA.

Das Fenster LSA öffnet sich.

Wählen Sie die gewünschte Signalanlage.

Klicken Sie auf die Schaltfläche Externe Daten bearbeiten.

Eine Sicherheitsabfrage öffnet sich.

Bestätigen Sie die Sicherheitsabfrage mit Fortsetzen.

Das externe Programm öffnet sich, in dem Sie die Steuerung bearbeiten können (LSA-Steuerung in Vissig bearbeiten).

Externe Steuerungsdaten vom Knoteneditor aus öffnen

Öffnen Sie den gewünschten (Ober-)Knoten im Knoteneditor (Ein Netzobjekt im Knoteneditor anzeigen).

Klicken Sie in der Symbolleiste des Knoteneditors auf das Symbol   Vissig-Steuerung bearbeiten.

Das externe Programm öffnet sich, in dem Sie die Steuerung bearbeiten können (LSA-Steuerung in Vissig bearbeiten).

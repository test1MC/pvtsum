Fahrzeitprofil löschen

Öffnen Sie das Fenster Linienrouten (Fenster Linienrouten öffnen).

Doppelklicken Sie auf die gewünschte Linienroute in der Liste.

Das Fenster Linienroute bearbeiten öffnet sich.

Wählen Sie das Register Verlauf und Fahrzeitprofile.

Das Register enthält alle Fahrzeitprofile der ausgewählten Linienroute.

Klicken Sie auf das Fahrzeitprofil, das Sie löschen möchten.

Klicken Sie auf das Symbol Aktuelles Fahrzeitprofil löschen.

Eine Abfrage öffnet sich.

Bestätigen Sie die Abfrage.

Das ausgewählte Fahrzeitprofil wird gelöscht.

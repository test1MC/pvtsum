Linienrouten: grafischen oder tabellarischen Fahrplan aufrufen

Sie können Linienrouten im Netz auswählen und diese im grafischen oder tabellarischen Fahrplan anzeigen (siehe „Grafischen und tabellarischen Fahrplan verwenden“ auf Seite 1).

Grafischen oder tabellarischen Fahrplan mit ausgewählten Linienrouten aufrufen

Öffnen Sie das Fenster Linienrouten (siehe „Fenster Linienrouten öffnen“ auf Seite 1).

Drücken Sie die Taste Strg und halten Sie sie gedrückt.

Klicken Sie nacheinander auf die Linienrouten, die Sie markieren möchten.

Klicken Sie auf die Schaltfläche Fahrplan.

Wenn Sie eine Linienroute markiert haben, die sich auf einer Linie befindet, die in der Linienauswahl enthalten ist, öffnet sich der ausgewählte Fahrplan. Wenn Sie eine Linienroute markiert haben, die sich auf einer Linie befindet, die nicht ausgewählt ist, öffnet sich eine Abfrage.

Element Beschreibung
Linienauswahl ändern

Die bestehende Linienauswahl im Netz soll geändert werden. Das Fenster Linienauswahl ändern öffnet sich (siehe „Linienauswahl ändern“ auf Seite 1). In dem Fenster können Sie Einstellungen für die Änderung der Linienauswahl vornehmen.

Linienauswahl beibehalten

Die bestehende Linienauswahl des Netzes wird beibehalten und der Fahrplan öffnet sich. Die Fahrplanfahrten, deren Verläufe über die gewählten Strecken führen, sind nicht markiert.

Klicken Sie auf die Schaltfläche Linienauswahl ändern.

(siehe „Linienauswahl ändern“ auf Seite 1).

Der Fahrplan öffnet sich und die überfahrenen Linienrouten werden je nach gewählten Optionen angezeigt.

Grafischen oder tabellarischen Fahrplan über die Suche nach einer Fahrplanfahrt aufrufen

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Linien

Klicken Sie im Kontextmenü auf den Eintrag Fahrplanfahrt suchen.

Das Fenster Fahrplanfahrten suchen öffnet sich.

Wählen Sie in der Auswahlliste das Attribut, nach dem Sie suchen möchten.

Die Suche nach Name ist nur möglich, wenn für mindestens eine Fahrplanfahrt ein Name angegeben ist.

Geben Sie in das Eingabefeld den Wert oder Text ein, nach dem Sie suchen möchten.

Während des Eingebens wird in der Liste automatisch der erste Attributwert angezeigt und markiert, der dem eingegebenen Wert oder Text entspricht.

Wenn Sie in der Liste eine Fahrplanfahrt anklicken, wird die zugehörige Linie im Netz markiert.

Klicken Sie mit rechten Maustaste auf das gewünschte Suchergebnis.

Element

Beschreibung

Autozoom

Der Netzausschnitt wird vergrößert und die markierte Fahrplanfahrt wird in der Mitte des Ausschnitts platziert

Bearbeiten

Über den Eintrag öffnen Sie das Fenster zum Bearbeiten mehrerer Fahrplanfahrten (siehe „Attributwerte von Netzobjekten bearbeiten“ auf Seite 1).

Linienroute bearbeiten

Über den Eintrag öffnen Sie das Fenster zum Bearbeiten von Linienrouten und Fahrzeitprofilen (siehe „Attribute und Optionen von Linienrouten und Fahrzeitprofilen“ auf Seite 1).

Fahrplan (grafisch)/ Fahrplan (tabellarisch)/ Beide Fahrpläne

Über die Auswahlliste öffnen Sie den grafischen, tabellarischen Fahrplan oder beide Fahrpläne. Die entsprechende Fahrplanfahrt ist dort markiert.

Hinweis

Sie können auch mehrere Fahrplanfahrten im Suchergebnis markieren und einen Fahrplan öffnen.

Der ausgewählte Fahrplan öffnet sich. Alle Linien werden angezeigt, die eine Fahrplanfahrt mit dem ausgewählten Attributwert enthalten. Alle Fahrplanfahrten, die den ausgewählten Attributwert besitzen, sind markiert.

Grafischen oder tabellarischen Fahrplan vom Fenster Linienroute bearbeiten aus aufrufen

Stellen Sie sicher, dass im Fenster Linienroute bearbeiten das Register Verlauf und Fahrzeitprofile ausgewählt ist (siehe „Fahrzeitprofile verwalten“ auf Seite 1).

Wählen Sie bei Bedarf ein oder mehrere Fahrzeitprofile aus.

Klicken Sie auf die Schaltfläche Fahrplan.

Das Fenster wird geschlossen, gegebenenfalls werden Änderungen gespeichert. Der ausgewählte Fahrplan öffnet sich. Im Register Linien wird genau die Linienroute angezeigt, die im Fenster Linienroute bearbeiten geöffnet war. Die ausgewählten Fahrzeitprofile werden angezeigt. Falls eine Bezugsfahrt ausgewählt ist, ist diese markiert.

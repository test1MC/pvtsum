Screenlines: Register Streckenattribute

Element

Beschreibung

Zeilen

Über die Schaltfläche können Sie das Fenster Attributauswahl öffnen. Im Fenster Attributauswahl können Sie indirekte Attribute auswählen, die dann im Register Streckenattribute gelistet werden.

Hinweise

Das Register enthält die jeweilige Anzahl Strecken für die drei folgenden Möglichkeiten:

  • Alle Strecken
  • Strecken in Screenline-Orientierung
  • Strecken gegen Screenline-Orientierung

Über diese drei Möglichkeiten und ihre Aggregationsfunktionen können Sie beispielsweise die Belastung ermitteln, die sich unmittelbar aus den Belastungswerten der Strecken ergeben, die die Screenline schneiden.

Auch wenn eine Screenline eine Strecke mehrfach schneidet, wird die Strecke nur einmal ausgewertet.

Verlauf einer Linienroute digitalisieren

Öffnen Sie das Fenster Linienrouten  (Fenster Linienrouten öffnen).

Wählen Sie die Linienroute aus, die Sie digitalisieren möchten.

Die Linienroute wird markiert.

Klicken Sie mit der rechten Maustaste in die Netzdarstellung.

Wählen Sie den Eintrag Digitalisieren.

Das Fenster Linienroute digitalisieren öffnet sich.

Legen Sie die Parameter für die Wegesuche fest (Linienrouten: Parameter für die Wegesuche einstellen).

Ändern Sie den bisherigen Verlauf.

Sie können zuerst Elemente im Verlauf fixieren, um sie zu erhalten. Anschließend ändern Sie den Verlauf zwischen zwei fixierten Elementen, indem Sie die gewünschten Punkte mit gedrückter Maustaste an neue Positionen ziehen (Grundregeln zum Ändern eines Routenverlaufs).

Falls für den geänderten Verlauf gekoppelte Fahrzeitprofile existieren, öffnet sich ein Fenster mit weiteren Informationen. Hier können Sie entscheiden, welche Kopplungen Sie erhalten möchten (Auswirkungen auf gekoppelte Fahrzeitprofile).

Der neue Verlauf der Linienroute wird gespeichert. Die Linienrouten der gekoppelten Fahrzeitprofile werden je nach Einstellung ebenfalls geändert.

Sie können den Verlauf einer einzelnen Linienroute auch tabellarisch bearbeiten (Verlauf einer Linienroute tabellarisch bearbeiten).

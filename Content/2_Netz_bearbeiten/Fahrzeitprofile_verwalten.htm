Fahrzeitprofile verwalten

Fahrzeitprofile beschreiben die Länge der Fahrzeiten zwischen den Haltepunkten einer Linienroute und geben an, ob an den Haltepunkten der Linienroute ein- oder ausgestiegen werden darf. Sie werden im Fenster Linienroute bearbeiten im rechten Bereich des Registers Verlauf und Fahrzeitprofile angezeigt (Linienrouten: Register Verlauf und Fahrzeitprofile).

Die Fahrzeitprofile einer Linienroute werden in separaten Registern verwaltet, zwischen denen Sie wechseln können (vergleichbar mit Excel).

Wenn Sie den Verlauf einer Linienroute verändern, hat dies Auswirkungen auf die Fahrplanfahrten und Fahrplanfahrtabschnitte der Linienroute (Linienroutenverlauf digitalisieren und Auswirkungen auf Fahrplanfahrten und Fahrplanfahrtabschnitte).
Fahrzeitprofile aufrufen

Zu jeder Linienroute gibt es ein oder mehrere Fahrzeitprofile. Ein (Fahrzeit-)profil beschreibt den zeitlichen Verlauf der Linie entlang der Linienroute, ohne jedoch konkrete Abfahrtszeitpunkte und somit einzelne Fahrplanfahrten vorzugeben. Die angelegten Fahrzeitprofile einer Linienroute werden auf verschiedenen Registern ausgegeben. Die Namen passiv gesetzter Fahrzeitprofile werden hier kursiv angezeigt. Wenn die ausgewählte Linienroute über mehr als ein Fahrzeitprofil verfügt, wird ein Register Mehrere angezeigt (Mehrfachansicht von Fahrzeitprofilen).

  • Klicken Sie im Fenster Linienroute bearbeiten unten im Register Verlauf und Fahrzeitprofile auf den Namen des gewünschten Fahrzeitprofils, beispielsweise 2.

Das gewünschte Fahrzeitprofil wird angezeigt.

Fahrzeitprofil-Verläufe bearbeiten

Folgende Symbole stehen Ihnen auf der rechten Seite des Registers Verlauf und Fahrzeitprofile zur Bearbeitung der Fahrzeitprofil-Verläufe zur Verfügung:

Element

Beschreibung

Über das Symbol Neues Fahrzeitprofil einfügen können Sie ein neues Fahrzeitprofil einfügen (Fahrzeitprofil einfügen).

Über das Symbol Aktuelles Fahrzeitprofil kopieren können Sie ein bestehendes Fahrzeitprofil kopieren (Fahrzeitprofil kopieren).

Über das Symbol Aktuelles Fahrzeitprofil bearbeiten können Sie ein bestehendes Fahrzeitprofil bearbeiten (Fahrzeitprofil bearbeiten).

Über das Symbol Aktuelles Fahrzeitprofil löschen können Sie ein bestehendes Fahrzeitprofil löschen (Fahrzeitprofil löschen).

Über das Symbol Neue Zeiten für das aktuelle Fahrzeitprofil setzen öffnet sich das Fenster Zeiten für Fahrzeitprofil <Nummer> setzen, in dem Sie die Fahr- und Haltezeiten eines Fahrzeitprofils neu berechnen können (Fahrzeitprofil bearbeiten).

Über das Symbol Fahrplanfahrten des aktuellen Fahrzeitprofils zeitlich verschieben können Sie alle Fahrplanfahrten um eine gewünschte Zeit verschieben (Abfahrtszeiten der Fahrplanfahrten verschieben).

Über das Symbol Ausgewählten Bereich mit anderen Fahrzeitprofilen koppeln können Sie verschiedene Fahrzeitprofile auf gemeinsamen Abschnitten verknüpfen (Mehrere Fahrzeitprofile koppeln).

Über das Symbol Kopplung des ausgewählten Bereichs aufheben können Sie die Kopplung verknüpfter Fahrzeitprofile entfernen (Mehrere Fahrzeitprofile koppeln).

Über das Symbol Festlegen, welche Fahrzeitprofile in der Mehrfachansicht angezeigt werden können Sie mehrere Fahrzeitprofile nebeneinander anzeigen, um ihre Attribute direkt zu vergleichen (Mehrfachansicht von Fahrzeitprofilen).

Über das Symbol Attribute wählen für rechte Tabelle öffnet sich das Fenster Attributauswahl, in dem Sie die gewünschten Fahrzeitprofil-Verlaufsattribute zur Ansicht auswählen können (Fenster zur Auswahl eines Attributs).
Fahrzeitprofilverläufe löschen

Sie können einzelne oder alle Fahrzeitprofilverläufe löschen.

Setzen Sie bei Bedarf die Fahrzeitprofilverläufe aktiv, die Sie löschen möchten,

  • Über den Linienfilter (Netzobjekte über Filter aktiv und passiv setzen).

Die räumliche Auswahl von Netzobjekten der Linienhierarchie (Linien, Linienrouten, Linienroutenverläufe, Fahrzeitprofile, Fahrzeitprofilverläufe, Fahrplanfahrten, Fahrplanfahrtabschnitte) ist zwar nicht möglich, aber Sie haben verschiedene Möglichkeiten, die Netzobjekte dieser Netzobjekttypen zur Auswahl hinzuzufügen oder aus der Auswahl zu entfernen (Linienrouten aktiv/passiv setzen).

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Linien.

Das Kontextmenü Linien öffnet sich.

Wählen Sie den Eintrag Mehrfach-Löschen > Fahrzeitprofilverläufe.

Eine Sicherheitsabfrage öffnet sich.

Bestätigen Sie die Abfrage mit OK oder OK für alle.

Alle Fahrzeitprofilverläufe werden gelöscht.

Ist der Netzobjekttyp Linien ausgewählt, können Sie alle aktiven Fahrzeitprofilverläufe auch über das Menü Bearbeiten > Mehrfach-Löschen löschen.

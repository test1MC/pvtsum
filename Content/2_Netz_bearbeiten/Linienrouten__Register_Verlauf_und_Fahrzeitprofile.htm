Linienrouten: Register Verlauf und Fahrzeitprofile

In dem Register können Sie links die Linienrouten-Verläufe und rechts die zugeordneten Fahrzeitprofile bearbeiten. Die Bereiche korrespondieren miteinander.

Sie können die Aufteilung der Bereiche anpassen, indem Sie mit der Maus an der Trennleiste ziehen.
Linienrouten-Verläufe bearbeiten

Folgende Symbole stehen Ihnen auf der linken Seite zur Bearbeitung der Linienrouten-Verläufe zur Verfügung:

Element

Beschreibung

Über das Symbol Längen neu berechnen und setzen können Sie nach Neudefinition der Routenpunkte die Längen zwischen den aktuell als IstRoutenPunkt ausgewählten Netzobjekten berechnen.

Über das Symbol Routenpunkt-Eigenschaft für eine oder mehrere Zeilen neu setzen öffnet sich das Fenster Routenpunkte setzen. In dem Fenster wählen Sie über die Optionen aus, für welche Haltepunkte Sie Routenpunkte setzen möchten (siehe „Routenpunkte setzen“ auf Seite 495).

Über das Symbol Menge der sichtbaren Zeilen auswählen öffnet sich das Fenster Sichtbare Zeilen auswählen. In dem Fenster können Sie festlegen, welche Zeilen im Register Verläufe und Fahrzeitprofile sichtbar sind. Es werden alle Zeilen angezeigt, die das ausgewählte Kriterium erfüllen.

  • alle

Ist die Option ausgewählt, werden alle Haltepunkte und Knoten im Linienweg, also alle Linienroutenelemente angezeigt. Es gilt also als einschränkendes Kriterium.

  • alle Routenpunkte
  • alle Profilpunkte

Die Option bezieht sich auf die Fahrzeitprofile auf der rechten Fensterseite. Ein Linienroutenelement erfüllt diese Filterbedingung genau dann, wenn es in mindestens einem Fahrzeitprofil der bearbeiteten Linienroute einen Profilpunkt über diesem Linienroutenelement gibt. Das gilt unabhängig davon, ob dieses Fahrzeitprofil in der rechten Hälfte des Dialogs aktuell sichtbar ist. Gibt es kein Fahrzeitprofil auf der Linienroute, ist die Bedingung nicht erfüllt.

  • alle Haltepunkte
  • alle Haltepunkte des VSys
  • alle aktiven Haltepunkte

Über das Symbol Attribute wählen für die linke Tabelle öffnet sich das Fenster Attributauswahl, in dem Sie die gewünschten Linienrouten-Verlaufsattribute zur Ansicht auswählen können (Fenster zur Auswahl eines Attributs).

Hinweise

Die Spalte Index gibt Auskunft über die Linienrouten-Elemente. Die Nummer 1 ist dem Start-Haltepunkt zugeordnet, die höchste Nummer dem End-Haltepunkt.

Die Anzahl der entsprechend der aktuellen Einstellung sichtbaren Linienrouten-Elemente wird im Fenster unten links angezeigt.

Das Streckenattribut Anzahl berührender Linienrouten gibt die Anzahl der Linienrouten an, die einen beliebig kleinen Teil einer Strecke überfahren. Sie können es beispielsweise in der Streckenliste oder zum Filtern von Strecken nutzen.
Routenpunkte setzen

Der Verlauf einer Linienroute wird als geordnete Folge von Routenpunkten angegeben. Ein Routenpunkt ist dabei ein Bezug auf einen Knoten oder Haltepunkt im Netz. Legen Sie wie folgt fest, für welche Haltepunkte Sie Routenpunkte setzen möchten.

Wenn Sie lediglich einzelne Haltepunkte als Routenpunkte auswählen möchten, markieren Sie in der Liste der Routenpunkte in der Spalte Ist Routenpunkt das jeweilige Kontrollkästchen.

Klicken Sie im Register Verlauf und Fahrzeitprofile auf das Symbol  Routenpunkt-Eigenschaft für eine oder mehrere Zeilen neu setzen.

Das Fenster Routenpunkte setzen öffnet sich.

Element

Beschreibung

Routenpunkte setzen

Wählen Sie die gewünschte Option aus:

  • für alle
  • für alle Haltepunkte
  • für alle Haltepunkte des VSys
  • für alle aktiven Haltepunkte

Routenpunkt-Eigenschaft setzen auf

Ist die Option ausgewählt, wird die Eigenschaft für die ausgewählte Gruppe gesetzt.

Ist die Option nicht ausgewählt, wird die Eigenschaft nur für den Start- und End-Haltepunkt gesetzt.

Setzen

Setzt die Routenpunkte der Auswahl zusätzlich zur bisherigen Auswahl und schließt das Fenster Routenpunkte setzen.

Exklusiv setzen

Setzt ausschließlich die Routenpunkte der Auswahl und schließt das Fenster Routenpunkte setzen.

Die gewünschte Auswahl wird als Routenpunkt gesetzt.

Werden nicht alle gewünschten Elemente angezeigt, können Sie die Anzeige über das Symbol Menge der sichtbare Zeilen auswählen ändern.
Fahrzeitprofile automatisch anpassen

Sie können festlegen, dass auf der rechten Seite für alle Fahrzeitprofile der Linienroute automatisch ein Profilpunkt eingefügt wird, wenn Sie auf der linken Seite einen Routenpunkt einfügen.

  • Wählen Sie im Register Verlauf und Fahrzeitprofile die Option Fahrzeitprofile automatisch anpassen.

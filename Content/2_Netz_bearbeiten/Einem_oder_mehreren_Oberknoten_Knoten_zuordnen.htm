Einem oder mehreren Oberknoten Knoten zuordnen

Sie können einem Oberknoten beim Einfügen Knoten zuordnen, indem Sie eine Umrandung einfügen (Oberknoten einfügen).

Sie können die zugeordneten Knoten nachträglich ändern, indem Sie sie über eine Liste zuordnen oder indem Sie Umrandungen ändern und jeweils wieder alle innerhalb der Umrandung liegenden Knoten dem Oberknoten zuordnen.

Einem Oberknoten Knoten zuordnen

Stellen Sie sicher, dass der gewünschte Oberknoten im Knoteneditor angezeigt ist (Oberknotenattribute in der Tabellenansicht bearbeiten).

Klicken Sie auf die Schaltfläche des Attributs Teilknoten.

Das Fenster Teilknoten von Oberknoten bearbeiten öffnet sich. Die linke Liste zeigt die Knoten, die dem Oberknoten zugeordnet sind, die rechte Liste zeigt alle Knoten, die keinem Oberknoten zugeordnet sind.

Ändern Sie die Zuordnungen über die Pfeil-Symbole.

Die Zuordnungen werden geändert.

Teilknoten anhand der Umrandungen neu zuordnen

Sie können für alle oder alle aktiven Oberknoten angeben, dass die Knoten, die innerhalb der jeweiligen Umrandung liegen, dem Oberknoten zugeordnet werden.

Liegt ein Knoten innerhalb mehrerer Umrandungen und ist keinem Oberknoten zugeordnet, wird er der ersten gefundenen Umrandung zugeordnet.

Liegt der Knoten innerhalb der Umrandung des Oberknotens, dem er auch zugeordnet ist, wird diese Zuordnung nicht verändert.

Setzen Sie bei Bedarf die Oberknoten aktiv, denen Sie Knoten gemäß der Oberknotenumrandungen zuordnen möchten, über

  • die räumliche Auswahl (Netzobjekte über die räumliche Auswahl aktiv und passiv setzen) und/oder
  • Filter (Netzobjekte über Filter aktiv und passiv setzen).

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Oberknoten.

Das Kontextmenü Oberknoten öffnet sich.

Wählen Sie den Eintrag Knotenzuordnung aus Polygonen neu berechnen.

Das Fenster Mehrfach-Bearbeiten Oberknoten öffnet sich.

Ist der Netzobjekttyp Oberknoten ausgewählt, können Sie das Fenster Mehrfach-Bearbeiten Oberknoten auch über das Menü Bearbeiten > Knotenzuordnung aus Polygonen neu berechnen aufrufen.

Stellen Sie sicher, dass das Register Sonderfunktionen und in der Auswahlliste der Eintrag Knotenzuordnung aus Polygonen neu berechnen ausgewählt sind.

Wählen Sie aus, ob die Zuordnung der Teilknoten für alle oder alle aktiven Oberknoten neu berechnet werden soll.

Element

Beschreibung

Nur aktive

Ist die Option ausgewählt, werden nur aktive Oberknoten berücksichtigt.

Klicken Sie auf die Schaltfläche Knotenzuordnung aus Polygonen neu berechnen.

Eine Sicherheitsabfrage öffnet sich.

Die Knoten werden gemäß den Umrandungen zugeordnet.

Klicken Sie auf die Schaltfläche Schließen.

Wenn Sie die Zuordnung der Teilknoten ändern, wird ein existierendes IV-Umlegungsergebnis verworfen.

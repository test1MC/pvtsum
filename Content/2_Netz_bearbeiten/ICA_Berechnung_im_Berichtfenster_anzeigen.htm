ICA-Berechnung im Berichtfenster anzeigen

Sie können die ICA-Berechnungsergebnisse eines Knotens in Excel anzeigen. Die Anzeige darin wird aktualisiert, sobald Sie eine neue ICA-Berechnung vornehmen.

Stellen Sie sicher, dass der gewünschte Knoten oder Oberknoten im Knoteneditor geöffnet ist.

Klicken Sie in der Symbolleiste des Knoteneditors auf das Symbol  ICA-Berichtsfenster öffnen/schließen.

Das ICA-Berichtfenster öffnet sich in Excel.

Wenn keine ICA-Berechnung durchgeführt wurde, ist das Fenster leer.

Wenn Sie zu einem anderen Knoten oder Oberknoten wechseln, wird das Berichtfenster nicht aktualisiert. Sie können die Anzeige aktualisieren, indem Sie eine neue Berechnung starten (ICA-Berechnung starten).

Sie können das ICA-Berichtfenster schließen, indem Sie erneut auf das Symbol  ICA-Berichtsfenster öffnen/schließen klicken.

Das ICA-Berichtfenster wird automatisch geschlossen, wenn Sie den Knoteneditor oder das -Fenster schließen.

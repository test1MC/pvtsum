ÖV-Richtungen verwalten

In gibt es eine ÖV-Hinrichtung und eine ÖV-Rückrichtung. Den zwei Richtungen können Sie einen Code und einen Namen geben.

Wählen Sie im Menü Netz den Eintrag ÖV-Richtungen.

Das Fenster ÖV-Richtungen öffnet sich.

Geben Sie in den Bereichen Hinrichtung und Rückrichtung die gewünschten Daten ein.

Wenn Sie Änderungen in diesem Fenster vornehmen, werden sie für vorhandene Linienrouten übernommen (Linienrouten und Fahrzeitprofile verwalten).

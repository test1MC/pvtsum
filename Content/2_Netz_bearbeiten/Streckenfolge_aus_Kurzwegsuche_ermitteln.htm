Streckenfolge aus Kurzwegsuche ermitteln

Mit dieser Funktion können Sie die Streckenfolge zwischen mindestens zwei markierten Knoten für ein IV-Verkehrssystem oder mindestens zwei markierten Haltepunkten für ein ÖV-Verkehrssystem gemäß der aktuell eingestellten Parameter ermitteln.

Sie können anschließend beispielsweise folgende Arbeitsschritte an der markierten Streckenfolge vornehmen:

  • Die gleichzeitige Bearbeitung der Attribute der markierten Strecken im Fenster Schnellansicht.
  • Die Bearbeitungsfunktionen für mehrere Objekte im Kontextmenü oder im Menü Bearbeiten.
  • Die Bearbeitung von Linienrouten im tabellarischen Fahrplan (Fahrplaneditor verwenden), welche die markierten Strecken überfahren.

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Strecken.

Das Kontextmenü Strecken öffnet sich.

Wählen Sie den Eintrag Streckenfolge aus Kurzwegsuche.

Das Fenster Streckenfolge digitalisieren öffnet sich.

Ist der Netzobjekttyp Strecken ausgewählt, können Sie das Fenster Streckenfolge digitalisieren auch über das Menü Bearbeiten > Streckenfolge aus Kurzwegsuche aufrufen.

Element

Beschreibung

Autozoom

Über die Schaltfläche können Sie den Netzausschnitt verschieben. Markierte Objekte werden in die Mitte verschoben.

Parameter

Über diese Schaltfläche können Sie die Parameter für die Kurzwegsuche einstellen.

Im Fenster Parameter für die Wegesuche können Sie folgende Einstellungen vornehmen:

Verkehrssystem

In der Auswahlliste können Sie ein Verkehrssystem auswählen. Zur Auswahl stehen alle im Netz angelegten Verkehrssysteme.

auch über gesperrte Strecken routen

Ist die Option ausgewählt, werden auch Strecken berücksichtigt, die für das Verkehrssystem gesperrt sind.

auch über gesperrte Abbieger routen

Ist die Option ausgewählt, werden auch Abbieger berücksichtigt, die für das Verkehrssystem gesperrt sind.

Suchkriterium

In der Auswahlliste können Sie ein Suchkriterium auswählen (Systemroute einfügen).

Klicken Sie im Netz mit der linken Maustaste auf den ersten Knoten oder Haltepunkt und halten Sie die Maustaste gedrückt.

Ziehen Sie den Mauszeiger zum nächsten gewünschten Knoten oder Haltepunkt und lassen Sie die Maustaste los.

Ziehen Sie den Mauszeiger bei Bedarf zu weiteren Knoten oder Haltepunkten.

Im Netz wird die anhand der aktuellen Parametereinstellungen gefundene Streckenfolge für die weitere Bearbeitung markiert.

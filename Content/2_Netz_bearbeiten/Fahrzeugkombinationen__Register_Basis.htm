Fahrzeugkombinationen: Register Basis

Element

Beschreibung

Fahrzeugeinheiten

Hier können Sie

  • Fahrzeugeinheiten für die Fahrzeugkombination einfügen (Fahrzeugeinheit zu Fahrzeugkombination hinzufügen) und entfernen (Fahrzeugeinheit aus Fahrzeugkombination entfernen)
  • die Anzahl der zugeordneten Fahrzeugeinheiten bearbeiten (Anzahl Fahrzeugeinheiten ändern).

Tipp

Sie können eine Fahrzeugkombination aus mehreren gleichen oder unterschiedlichen Fahrzeugeinheiten zusammenstellen. So können Sie beispielsweise Züge besser abbilden. Die Fahrzeugkombination Tram-Kurzzug besteht beispielsweise aus 1 Fahrzeugeinheit Tramwagen und ein Tram-Vollzug aus 2 Fahrzeugeinheiten Tramwagen.

Verkehrssystem-Schnittmenge

Zeigt die Verkehrssysteme an, die in allen der Fahrzeugkombination zugeordneten Fahrzeugeinheiten enthalten sind.

Fahrzeugkombinations-Set

Hier können Sie ein Fahrzeugkombinations-Set als Attribut der Fahrzeugkombination festlegen (Fahrzeugkombinations-Set festlegen). Diese Funktion ist nur anwendbar, wenn Sie das Zusatzmodul Detaillierte Umlaufbildung aktiviert haben.

Fahrzeugeinheit zu Fahrzeugkombination hinzufügen

Stellen Sie sicher, dass das Register Basis geöffnet ist.

Klicken Sie auf die Schaltfläche Einfügen.

Das Fenster Zugeordnete Fahrzeugeinheit öffnet sich.

Wählen Sie die gewünschte Fahrzeugeinheit aus.

Geben Sie die gewünschte Anzahl Fahrzeugeinheiten an.

Die Fahrzeugeinheit wird eingefügt.

Anzahl Fahrzeugeinheiten ändern

Stellen Sie sicher, dass das Register Basis geöffnet ist.

Wählen Sie die gewünschte Fahrzeugeinheit aus.

Klicken Sie auf die Schaltfläche Bearbeiten.

Das Fenster Zugeordnete Fahrzeugeinheit öffnet sich.

Ändern Sie die Anzahl der Fahrzeugeinheiten.

Die Anzahl wird geändert.

Fahrzeugeinheit aus Fahrzeugkombination entfernen

Stellen Sie sicher, dass das Register Basis geöffnet ist.

Wählen Sie die gewünschte Fahrzeugeinheit aus.

Klicken Sie auf die Schaltfläche Löschen.

Die Fahrzeugeinheit wird entfernt.

Fahrzeugkombinations-Set festlegen

Mit dieser Funktion können Sie ein Fahrzeugkombinations-Set als Attribut der Fahrzeugkombination festlegen.

Klicken Sie auf die Schaltfläche neben Fahrzeugkombinations-Set.

Das Fenster Fahrzeugkombinations-Set öffnet sich.

Wählen Sie die Fahrzeugkombinationen aus, die das Fahrzeugkombinations-Set bilden sollen.

Klicken Sie auf OK.

Die Nummern der ausgewählten Fahrzeugkombinationen werden auf der Schaltfläche Fahrzeugkombinations-Set angezeigt und als Attribut der Fahrzeugkombination hinterlegt.

Fahrzeugkombinations-Sets sind nützlich, wenn Sie über viele Fahrplanfahrtabschnitte verfügen, denen Sie mit der Funktion Mehrfach-Bearbeiten schnell Fahrzeugkombinations-Sets zuordnen möchten (Attribut den Wert eines anderen Attributs zuweisen). Sie ersparen sich damit die manuelle Eingabe der Fahrzeugkombinations-Sets. verwendet das einem Fahrplanfahrtabschnitt zugeordnete Fahrzeugkombinations-Set bei der Umlaufbildung mit Fahrzeugtausch (ÖV-Umlaufbildung durchführen).

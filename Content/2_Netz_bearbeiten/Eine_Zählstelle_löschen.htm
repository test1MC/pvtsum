Eine Zählstelle löschen

Wenn Sie eine Zählstelle löschen, der ein Detektor zugeordnet ist, wird der Detektor nicht automatisch ebenfalls gelöscht. Er verliert lediglich seine Zuordnung.

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Zählstellen.

Klicken Sie im Netz auf die Zählstelle, die Sie löschen möchten.

Drücken Sie die Taste Entf.

Die Zählstelle wird gelöscht.

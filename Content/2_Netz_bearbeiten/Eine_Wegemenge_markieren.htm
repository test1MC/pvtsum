Eine Wegemenge markieren

Sie können eine Wegemenge im Netz nur markieren, wenn der Wegemenge mindestens ein Weg und daher ein Verlauf zugeordnet ist.

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche IV-Wege.

Das Fenster Wege öffnet sich.

Abhängig von der gewählten Option, ändert sich der Name des Fensters Wege/Wegemengen.

Stellen Sie sicher, dass die Option Wegemengen ausgewählt ist.

Wählen Sie eine Wegemenge in der Liste aus.

Sie können die gesuchte Wegemenge auch über das Eingabefeld auswählen. Geben Sie hierzu den Namen der Wegemenge in das Feld ein. Die Wegemenge wird in der Liste ausgewählt.

Alle Wegeverläufe der Wegemenge werden im Netz markiert.

Klicken Sie bei Bedarf auf die Schaltfläche Autozoom.

Der Netzausschnitt wird vergrößert und die markierte Wegemenge wird in der Mitte des Ausschnitts platziert.

Wenn Sie die bisher markierten Wegemengen aus der Auswahl entfernen möchten, klicken Sie auf die Schaltfläche Initialisieren.

Sie können Wegeverläufe einer Wegemenge auch markieren, indem Sie im Netz auf einen Knoten klicken über den ein Weg der Wegemenge verläuft.

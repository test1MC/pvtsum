Verlauf mehrerer Linienrouten gleichzeitig digitalisieren

Sie können die Verläufe mehrerer Linienrouten gleichzeitig ändern. Jede Änderung hat dann den gleichen Effekt auf die ausgewählten Linienrouten, den die Änderung bei einer separaten Bearbeitung der Linienrouten hätte.

Öffnen Sie das Fenster Linienrouten (Fenster Linienrouten öffnen).

Markieren Sie die Linienrouten, die Sie digitalisieren möchten.

Sie können beispielsweise mehrere Linienrouten markieren, indem Sie einen oder mehrere Haltepunkte im Netz markieren. Dann sind alle Linienrouten markiert, die diese Haltepunkte überfahren.

Alternativ können Sie mehrere Linienrouten im Fenster Linienrouten oder nacheinander im Netz markieren (Mehrere Linienrouten markieren).

Die Linienrouten sind markiert.

Klicken Sie mit der rechten Maustaste in die Netzdarstellung.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Digitalisieren.

Das Fenster Linienroute digitalisieren öffnet sich.

Oben im Fenster wird die <Anzahl> Linienrouten angezeigt, die markiert ist.

Legen Sie die Parameter für die Wegesuche fest (Linienrouten: Parameter für die Wegesuche einstellen).

Klicken Sie bei Bedarf im Fenster Linienroute digitalisieren auf die Schaltfläche Autozoom, wenn Sie den Ausschnitt mit den Verläufen aller markierten Linienrouten vergrößern möchten.

Ändern Sie die bisherigen Verläufe.

Sie können zuerst Elemente im Verlauf fixieren, um sie zu erhalten. Anschließend ändern Sie den Verlauf zwischen zwei fixierten Elementen, indem Sie die gewünschten Punkte mit gedrückter Maustaste an neue Positionen ziehen (Grundregeln zum Ändern eines Routenverlaufs).

Der Verlauf aller Linienrouten, die die Verlaufspunkte überfahren, wird geändert.

Falls für den geänderten Verlauf gekoppelte Fahrzeitprofile existieren, öffnet sich ein Fenster mit weiteren Informationen. Hier können Sie entscheiden, ob und welche Kopplungen Sie erhalten möchten (Auswirkungen auf gekoppelte Fahrzeitprofile).

Der neue Verlauf der Linienrouten wird gespeichert.

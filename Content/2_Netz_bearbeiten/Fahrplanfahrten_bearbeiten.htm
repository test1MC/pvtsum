Fahrplanfahrten bearbeiten

Sie können die Attributwerte mehrerer Fahrplanfahrten gleichzeitig ändern. Das gilt auch für benutzerdefinierte Attribute, sofern vorhanden.

Die Attributwerte einer Fahrplanfahrt können Sie auch direkt in der Liste bearbeiten.

Wenn Sie nur einzelne Fahrplanfahrten bearbeiten möchten, können Sie das Bearbeitungsfenster mit einem Doppelklick auf die entsprechende Fahrplanfahrt öffnen. Die weiß hinterlegten Attributwerte einer Fahrplanfahrt können Sie auch direkt im tabellarischen Fahrplan bearbeiten.

Markieren Sie die Fahrplanfahrt(en), die Sie bearbeiten möchten, indem Sie auf den Kopf der jeweiligen Spalten klicken (Fahrplanfahrten markieren).

Klicken Sie mit der rechten Maustaste auf eine der markierten Fahrplanfahrten.

Klicken Sie auf den Eintrag Fahrplanfahrt(en) bearbeiten.

Das Fenster Fahrplanfahrt(en) bearbeiten öffnet sich.

Die Attribute im Fenster Fahrplanfahrt(en) bearbeiten und im Fenster Fahrplanfahrt(en) einfügen sind identisch.

( Fahrplanfahrt einfügen ) .

Wenn Attribute markierter Fahrplanfahrten voneinander abweichen, wird ein * angezeigt.

Wenn eine einzelne Fahrplanfahrt markiert ist, können Sie nachträglich Taktfahrten einfügen (Taktfahrten einfügen).

Wenn Sie auf diese Weise die Linie oder Linienroute einer Fahrt ändern, wird die Fahrt in das erste Fahrzeitprofil der Ziel-Linie bzw. der Ziel-Linienroute verschoben. Dabei kann sich der Verlauf der Fahrt mit ändern. Sie können Fahrten auch mit ihren Linienrouten und Fahrzeitprofilen verschieben (Fahrplanfahrten in eine andere Linie verschieben).

Die Änderungen werden übernommen.

Im Fahrplan können Sie außerdem das Bearbeitungsfenster für Fahrzeitprofile einer Fahrt aufrufen und diese direkt bearbeiten. Der Eintrag Fahrzeitprofil bearbeiten im Kontextmenü steht zur Verfügung, wenn genau eine Fahrplanfahrt oder Fahrtabschnitte einer Fahrplanfahrt im Fahrplan markiert sind.

Einen Weg markieren

Öffnen Sie das Fenster Wege (Fenster Wege öffnen).

Schränken Sie bei Bedarf die Menge der Wege ein, die angezeigt wird.

Element

Beschreibung

Auswahlliste Wegemengen/ Nachfragesegmente

Auswahl einer Wegemenge oder eines Nachfragesegments

Hinweise

Angezeigt werden die Wege, die der Wegemenge/dem Nachfragesegment zugeordnet sind.

Manuell eingefügte Wege werden je Wegemenge ausgegeben. In Umlegungen erzeugte Wege werden je Nachfragesegment ausgegeben.

Auswahlliste

Sie können die Anzeige auf folgende Ansichten einschränken:

alle Routen

Alle Wege werden angezeigt.

Spinne-Routen

Alle Wege der aktuellen Spinnenausgabe werden angezeigt.

Beziehungsfilter

Alle Wege des aktuellen Beziehungsfilters werden angezeigt.

Wählen Sie per Linksklick einen Weg in der Liste aus.

Sie können den gesuchten Weg auch über das Eingabefeld auswählen. Geben Sie hierzu den Namen des Weges in das Feld ein. Der Weg wird in der Liste ausgewählt.

Der Verlauf des Weges wird im Netz markiert.

Klicken Sie bei Bedarf auf die Schaltfläche Autozoom.

Der Netzausschnitt wird vergrößert und der markierte Weg wird in der Mitte des Ausschnitts platziert.

Nutzen Sie die Schaltfläche Initialisieren, wenn Sie die bisher markierten Wege aus der Auswahl entfernen möchten.

Sie können die Wegeverläufe einer Wegemenge auch markieren, indem Sie im Netz auf einen Knoten oder Bezirk klicken über den ein Weg verläuft.

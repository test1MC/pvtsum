Fahrplanfahrtabschnitte löschen

Das Löschen von Fahrplanfahrtabschnitten kann Auswirkungen auf gekoppelte Fahrzeitprofile haben (Mehrere Fahrzeitprofile koppeln).

Sie haben verschiedene Möglichkeiten Fahrplanfahrtabschnitte zu löschen.

Einen Fahrplanfahrtabschnitt löschen

Markieren Sie im Fahrplan den Fahrplanfahrtabschnitt, den Sie löschen möchten.

Drücken Sie die Taste Entf.

Der Fahrplanfahrtabschnitt wird gelöscht.

Fahrplanfahrtabschnitte markierter Fahrplanfahrten zurücksetzen

Markieren Sie die Fahrplanfahrten, deren Fahrplanfahrtabschnitte Sie zurücksetzen möchten (siehe „Fahrplanfahrten markieren“ auf Seite 1).

Klicken Sie mit der rechten Maustaste in eine markierte Fahrplanfahrt.

Wählen Sie den Eintrag Fahrplanfahrtabschnitt zurücksetzen.

Eine Sicherheitsabfrage öffnet sich.

Die Fahrplanfahrtabschnitte werden auf einen Fahrplanfahrtabschnitt zurückgesetzt.

Alle aktiven Fahrplanfahrtabschnitte löschen

Wählen Sie die Fahrplanfahrtabschnitte, die Sie löschen möchten, über Filterkriterien aus (Netzobjekte über Filter aktiv und passiv setzen).

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Linien.

Das Kontextmenü Linien öffnet sich.

Wählen Sie den Eintrag Mehrfach-Löschen > Fahrplanfahrtabschnitte.

Eine Sicherheitsabfrage öffnet sich.

Attributwerte aller oder aller aktiven Fahrplanfahrtabschnitte ändern

Sie können nur einzelne oder alle/alle aktiven Fahrplanfahrtabschnitte bearbeiten. Sind keine passiven Fahrplanfahrtabschnitte im Netz, werden alle Fahrplanfahrtabschnitte geändert. Markierungen von Fahrplanfahrtabschnitten werden ignoriert.

Setzen Sie bei Bedarf im Linienfilter die Fahrplanfahrtabschnitte aktiv (Netzobjekte über Filter aktiv und passiv setzen), deren Attributwerte Sie ändern möchten.

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Linien.

Das Kontextmenü Linien öffnet sich.

Wählen Sie den Eintrag Mehrfach-Bearbeiten > Fahrplanfahrtabschnitte.

Das Fenster Mehrfach-Bearbeiten Fahrplanfahrtabschnitte öffnet sich.

Wählen Sie aus, ob Sie alle oder alle aktiven Fahrplanfahrtabschnitte ändern möchten.

Element

Beschreibung

nur aktive

Ist die Option ausgewählt, werden nur aktive Fahrplanfahrtabschnitte berücksichtigt.

Wählen Sie das Register Formel.

Ändern Sie die Werte der gewünschten Attribute (Attributwerte von Netzobjekten bearbeiten).

Die Attributwerte werden geändert.

Klicken Sie auf die Schaltfläche Schließen.

Sie können im gleichen Fenster im Register Sonderfunktionen außerdem alle oder alle aktiven Fahrplanfahrtabschnitte normalisieren (Fahrplanfahrtabschnitte normalisieren), Fahrzeugkombinationen auf Fahrplanfahrtabschnitten ersetzen oder Zwangsbindungen löschen (Zwangsbindungen löschen).

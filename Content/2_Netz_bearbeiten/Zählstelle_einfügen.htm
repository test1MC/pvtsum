Zählstelle einfügen

Sie können Zählstellen nur auf Strecken einfügen.

Klicken Sie im Fenster Netz auf das Symbol Modus Einfügen.

Klicken Sie auf die Schaltfläche Zählstellen.

Klicken Sie an die Position auf der gewünschten Strecke in der gewünschten Richtung, an der Sie die Zählstelle einfügen möchten.

An der Position wird ein rotes Dreieck angezeigt und das Fenster Zählstelle einfügen öffnet sich.

Sie können die Position der Zählstelle später ändern.

Geben Sie die gewünschten Daten ein (Attribute und Optionen von Zählstellen).

Die Zählstelle wird im Netz eingefügt.

Fahrzeitprofil kopieren

Sie können ein neues Fahrzeitprofil auch einfügen, indem Sie das aktuell ausgewählte Fahrzeitprofil kopieren.

Öffnen Sie das Fenster Linienrouten (Fenster Linienrouten öffnen).

Doppelklicken Sie auf die gewünschte Linienroute in der Liste.

Das Fenster Linienroute bearbeiten öffnet sich.

Wählen Sie das Register Verlauf und Fahrzeitprofile.

Das Register enthält alle Fahrzeitprofile der ausgewählten Linienroute.

Klicken Sie auf das Fahrzeitprofil, das Sie kopieren möchten.

Klicken Sie auf das Symbol  Aktuelles Fahrzeitprofil kopieren.

Das Fenster Fahrzeitprofil <Name> kopieren öffnet sich.

Element

Beschreibung

Linie

Anzeige der Linie, der das Fahrzeitprofil zugeordnet ist

Linienroute

Anzeige der Linienroute, der das Fahrzeitprofil zugeordnet ist

Richtung

Anzeige der zugeordneten Richtung des Fahrzeitprofils (ÖV-Richtungen verwalten)

Name

Geben Sie in das Eingabefeld einen Namen für das neue Fahrzeitprofil ein.

FzgKomb.

Wählen Sie in der Auswahlliste die gewünschte Fahrzeugkombination aus.

Alle Fahrplanfahrten mit kopieren

Ist die Option ausgewählt, werden mit dem Fahrzeitprofil auch alle Fahrplanfahrten kopiert.

Hinweis

Wenn Sie die Fahrzeugkombination auf eine von einer Fahrplanfahrt abweichende Fahrzeugkombination ändern, übernimmt die neue Fahrplanfahrt dennoch stets die Fahrzeugkombination der Fahrplanfahrt, die kopiert wird.

Geh- und Wartezeiten an Haltestellen mitkopieren

Ist die Option ausgewählt, werden mit dem Fahrzeitprofil auch die Geh- und Wartezeiten an Haltestellen kopiert.

Das Fahrzeitprofil wird kopiert und eingefügt.

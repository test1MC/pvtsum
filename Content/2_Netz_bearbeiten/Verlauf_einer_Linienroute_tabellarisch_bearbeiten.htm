Verlauf einer Linienroute tabellarisch bearbeiten

Sie können Linienroutenverläufe nur tabellarisch bearbeiten, wenn bereits Systemroutenverläufe existieren (Systemrouten verwalten).

Alternativ zum manuellen Bearbeiten oder Einfügen von Linienrouten können Sie den Verlauf einzelner Linienrouten tabellarisch verlängern (Linienroute tabellarisch verlängern) oder umrouten (Linienroute tabellarisch umrouten). Dies geschieht anhand von im Netz vorhandenen Systemroutenverläufen.

Wenn Sie eine neue Linienroute tabellarisch einfügen, werden die Fahrzeiten der Systemroute als Fahrzeiten für das neue Fahrzeitprofil übernommen.

Summe = Anfahrzuschlag + Durchfahrzeit + Abbremszuschlag

Systemroutenverläufe wiederum können aus Fahrzeitprofilen erzeugt werden (Systemrouten verwalten).

Sie können Systemroutenverläufe am Start- und am End-Haltepunkt einer Linienroute anfügen, um die Linienroute zu verlängern. Sie können Systemroutenverläufe außerdem nutzen, um den Verlauf einer Linienroute abschnittweise umzurouten. Folgende Punkte sind dabei zu beachten:

  • Da Systemrouten zwingend an Haltepunkten starten und enden, kann nur dann tabellarisch verlängert werden, wenn Start- oder Endpunkt der Linienroute auf Haltepunkten liegen. Ist nur einer dieser Punkte ein Haltepunkt, so kann nur von diesem ausgehend tabellarisch verlängert werden.
  • Falls das Verkehrssystem der Systemroute vom Verkehrssystem der Linienroute abweicht, kann es erforderlich sein, Strecken, Abbieger und Haltepunkte für das Linienrouten-Verkehrssystem zu öffnen.
  • Auf dem neuen Verlaufsabschnitt werden (abgesehen von Start- und End-Haltepunkt) keine Routenpunkte gesetzt. Die Länge des neuen Abschnitts wird aus der Systemroute übernommen.
  • Wird eine bestehende Linienroute bearbeitet, so werden die Fahrzeiten aller Fahrzeitprofile der Linienroute angepasst. Standardmäßig werden die Systemroutenzeiten verwendet.
Linienroute tabellarisch verlängern

Öffnen Sie das Fenster Linienrouten (Fenster Linienrouten öffnen).

Markieren Sie die Linienroute, die Sie bearbeiten möchten.

Klicken Sie mit der rechten Maustaste in die Netzdarstellung.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Digitalisieren.

Das Fenster Linienroute digitalisieren öffnet sich.

Klicken Sie auf die Schaltfläche tabellarisch bearbeiten.

Wenn es im Netz keine geeigneten Systemrouten gibt, ist die Schaltfläche nicht verfügbar.

Das Fenster Tabellarische Bearbeitung der Linienroute öffnet sich.

Für jede Systemroute werden die Spalten von VonHP, NachHP, Name, VSys, FzgKomb und Fahrzeit angezeigt.

Unter Fahrzeit ist die Summe aus Durchfahrzeit, Anfahr- und Abbremszuschlägen zu verstehen.

Element

Beschreibung

am Start verlängern/

am Ende verlängern

Das Fenster enthält - abhängig von der gewählten Option - eine Liste aller möglichen Verlängerungen des Linienroutenverlaufs am Start oder am Ende. Sie können den Verlauf verlängern, indem Sie eine der folgenden Optionen und einen Systemroutenabschnitt in der Liste auswählen:

am Start verlängern

Ist die Option ausgewählt, wird der in der Liste ausgewählte Systemroutenabschnitt am Beginn des bisherigen Verlaufs angefügt.

am Ende verlängern

Ist die Option ausgewählt, wird der in der Liste ausgewählte Systemroutenabschnitt am Ende des bisherigen Verlaufs angefügt.

Umrouten

Über diese Schaltfläche können Sie die Linienroute umrouten (Linienroute tabellarisch umrouten).

Nur aktive Systemrouten anzeigen

Ist die Option ausgewählt, werden nur aktive Systemrouten aufgelistet.

Auch Schleifen vorschlagen

Ist die Option ausgewählt, werden auch Systemrouten vorgeschlagen, die eine Rückfahrt zum letzten Halt bedeuten.

Filter

Über die Filter können Sie die Liste der Systemroutenabschnitte einschränken.

Hinweis

Wird eine Einschränkung vorgenommen, so ist die jeweilige Beschriftung anschließend rot.

VonHP/NachHP

Über die Schaltfläche öffnet sich das Fenster Haltepunkt suchen. Sie können nur einen Haltepunkt auswählen.

Hinweis

Abhängig von der gewählten Option am Start verlängern/am Ende verlängern heißt die Schaltfläche VonHP oder NachHP.

VSys

Über die Schaltfläche öffnet sich das Fenster Auswahl Verkehrssysteme. Wählen Sie die zugelassenen Verkehrssysteme aus.

Hinweise

Sie können mehrere Verkehrssysteme auswählen, wenn Sie die Taste Strg gedrückt halten.

Wenn Sie die Auswahl nicht einschränken, werden alle Verkehrssysteme der Linie angezeigt.

FzgKomb

Über die Schaltfläche öffnet sich das Fenster Auswahl Fahrzeugkombinationen. Wählen Sie die zugelassenen Fahrzeugkombinationen aus.

Hinweise

Sie können mehrere Fahrzeugkombinationen auswählen, wenn Sie die Taste Strg gedrückt halten.

Wenn Sie die Auswahl nicht einschränken, werden alle Fahrzeugkombinationen angezeigt.

Undo

Über die Schaltfläche können Sie den letzten Schritt rückgängig machen und den zuletzt hinzugefügten Abschnitt entfernen.

Autozoom

Über die Schaltfläche können Sie sich den vollständigen Routenverlauf der markierten Linienroute anzeigen lassen.

Verlängern

Über diese Schaltfläche bestätigen Sie den verlängerten Verlauf über den in der Liste markierten Abschnitt.

Tipp

Sie können der Linienroute auch einen Abschnitt hinzufügen, indem Sie auf die entsprechende Zeile doppelklicken.

Je nach Einstellung der Parameter für die Wegesuche werden zugehörige Fahrplanfahrten automatisch verlängert, wenn eine Linienroute verlängert wird (Linienrouten: Parameter für die Wegesuche einstellen). Die aktuelle Einstellung entnehmen Sie einem Hinweis im Fenster Tabellarische Bearbeitung der Linienroute.

Klicken Sie auf die Schaltfläche Verlängern.

Die Linienroute wird verlängert und die Systemroutenauswahl entsprechend aktualisiert.

Sie können die Linienroute um Abschnitte verlängern, ohne das Fenster Tabellarische Bearbeitung der Linienroute zwischenzeitlich zu schließen.

Bestätigen Sie mit Schließen.

Der Digitalisiermodus wird erneut aktiviert.

Sie können beliebig oft zwischen Digitalisiermodus und Fenster Tabellarische Bearbeitung der Linienroute wechseln. Schließen Sie die Verlaufsbearbeitung ab, indem Sie im Fenster Linienroute digitalisieren auf OK klicken.
Beispiel

Der Verlauf der ausgewählten Linienroute ist im Netz markiert.

Die dunkle Markierung kennzeichnet die mögliche Verlängerung, die der ausgewählten Zeile im Fenster Tabellarische Bearbeitung der Linienroute entspricht, die helle Markierung den bisherigen Linienroutenverlauf. Wird der obige Verlauf per Doppelklick auf die entsprechende Zeile oder über die Schaltfläche Verlängern übernommen (hier Tannenstr - Weinbergweg), so werden anschließend weitere Verläufe zur Verlängerung ab Haltepunkt Weinbergweg (= dem neuen Von Haltepunkt) vorgeschlagen.

Diese werden wiederum im Fenster Tabellarische Bearbeitung der Linienroute aufgelistet.

Linienroute tabellarisch umrouten

Wenn Sie einen Abschnitt der ausgewählten Linienroute umrouten möchten, gehen Sie wie folgt vor.

Klicken Sie im Fenster Tabellarische Bearbeitung der Linienroute auf die Schaltfläche Umrouten.

Das Fenster Tabellarisches Umrouten öffnet sich.

Legen Sie den Abschnitt fest, den Sie umrouten möchten.

Element

Beschreibung

Von Haltepunkt /Nach Haltepunkt

Über die Auswahllisten können Sie den Von-Haltepunkt und den Nach-Haltepunkt des umzuroutenden Bereichs wählen.

Hinweise

Zur Auswahl stehen alle Routenpunkte auf denen Haltepunkte liegen.

Da zwei existierende Routenpunkte der Linienroute ausgewählt werden müssen, können Sie weder den Start- noch den Endhalt verändern. Der Beginn und das Ende können daher nicht umgeroutet werden.

Das Fenster Tabellarische Bearbeitung der Linienroute öffnet sich.

Der umzuroutende Abschnitt wird im Fenster rot hinterlegt ausgegeben. In der Netzdarstellung wird der im Fenster ausgewählte Vorschlag für den Verlauf der Linienroute in der für Markierung 2 eingestellten Farbe gekennzeichnet.

Wählen Sie in der Liste den gewünschten Abschnitt aus.

Klicken Sie auf die Schaltfläche Anfügen.

Der Vorschlag wird übernommen und die Linienroute umgeroutet.

Routen Sie bei Bedarf weitere Abschnitte um.

Wird der gewählte Nach-Haltepunkt erreicht, ist das Umrouten automatisch abgeschlossen.

Bestätigen Sie Ihre Änderung mit einem Klick auf die Schaltfläche Schließen.

Strecke einfügen

Klicken Sie im Fenster Netz auf das Symbol Modus Einfügen.

Klicken Sie auf die Schaltfläche Strecken.

Klicken Sie auf den Knoten, von dem die neue Strecke ausgehen soll (=Von-Knoten).

Klicken Sie auf den Knoten, an dem die neue Strecke enden soll (=Nach-Knoten).

Das Fenster Strecke einfügen öffnet sich.

Wenn Sie den Verlauf der Strecke bereits während des Einfügens digitalisieren möchten, können Sie ihn nach dem Mausklick auf den Von-Knoten mit darauf folgenden Mausklicks festlegen. Den Abschluss bildet ein Mausklick auf den Nach-Knoten.

Element

Beschreibung

Nummer

Eindeutige Nummer der Strecke

Strecken werden automatisch fortlaufend nummeriert. Die vorgegebene Nummer kann mit einer Nummer überschrieben werden, die im Netz noch nicht vergeben ist.

Hinweis

Beide Richtungen einer Strecke haben die gleiche Nummer.

Von Knoten

Nach Knoten

Anzeige der Nummern der Knoten, an denen die Strecke startet und endet

Typ

Über die Auswahlliste können Sie der Strecke einen Streckentyp zuordnen (Streckentypen festlegen).

Gegenrichtung

Ist die Option ausgewählt, wird eine Gegenrichtung mit identischer Streckennummer eingefügt.

Ist die Option nicht ausgewählt, wird eine Strecke mit identischer Nummer in Gegenrichtung eingefügt, diese ist jedoch für alle Verkehrssysteme gesperrt. Die eingefügte Strecke ist dann eine Einbahnstraße.

Hinweise

Einbahnstraßen werden in Bezug auf zugelassene Verkehrssysteme ausgewertet. Die grafische Hervorhebung von Einbahnstraßen können Sie im Menü Grafik > Grafikparameter bearbeiten > Strecken einstellen. Für die Attribute IstEinbahnStraße und IstEinbahnStraßefürVSys (verkehrssystemfein) können Sie gesondert Markierungen einstellen (Grundeinstellungen für das Netzfenster vornehmen).

Wenn Sie IstEinbahnStraße in der Attributauswahl wählen, werden alle Strecken, die explizit als Einbahnstraße (Gegenrichtung für kein Verkehrssystem zugelassen) eingefügt wurden, grafisch hervorgehoben.

Diese bisher nicht existierende Richtung der Strecke kann bei der Vereinigungsnetzberechnung nach dem Arbeitsschritt VSys für die Gegenrichtung zulassen nicht berücksichtigt werden, da Differenzen lediglich zwischen zwei Attributwerten einer existierenden Strecke ermittelt werden. Bei einer nachfolgenden Vereinigungsnetzberechnung sollten Sie deshalb statt einer „wirklichen“ Einbahnstraße für die Gegenrichtung die Kapazität = 0 setzen (Spinnen anzeigen).

Typ

Über die Auswahlliste können Sie der Gegenrichtung den gewünschten Streckentyp zuordnen. Der kann vom Typ der Hinrichtung abweichen.

Details

Über die Schaltfläche öffnet sich das Fenster Strecke bearbeiten. Sie können weitere Einstellungen der Strecke vornehmen (Attributwerte von Strecken ändern).

Die Strecke wird eingefügt.

Beim Einfügen einer Strecke werden alle theoretisch möglichen Abbieger an beiden Knoten der Strecke erzeugt. Dafür werden die Standardwerte aus benutzerdefinierten Abbiegerstandards verwendet (Abbiegerstandards verwalten).

Knotenstrom als Grafik exportieren

Bevor Sie einen oder mehrere Knotenströme als Grafik exportieren, können Sie verschiedene Einstellungen für den Export vornehmen.

Wählen Sie im Menü Knoteneditor den Eintrag Knotenstrom als Grafik exportieren.

Das Fenster Exportierte Knotenströme speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Wählen Sie den gewünschten Dateityp aus.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster Parameter für den Export von Knotenströmen öffnet sich.

Nehmen Sie im Register Größe die gewünschten Einstellungen vor.

Element

Beschreibung

Parameter

Hier können Sie die Größenverhältnisse für den Export festlegen. Alle Änderungen sind sofort in der Vorschau sichtbar.

Breite

Angabe der Ausgabebreite der Gesamtgrafik in der gewünschten Einheit

Höhe

Angabe der Ausgabehöhe der Gesamtgrafik in der gewünschten Einheit

Rand

Angabe der Breite des Rands

Kreisgröße

Angabe des Verhältnisses von Beschriftung und Kreis in Prozent

Hochformat/Querformat

Auswahl des Exportformats des Knotenstroms

Vorschau

Grafische Vorschau auf die Größe und Seitenverhältnisse des Knotenstromkreises

Nehmen Sie im Register Datei die gewünschten Einstellungen vor.

Element

Beschreibung

Dateityp

Hier wählen Sie die Dateiart, in dem die Grafik gespeichert werden soll.

Auflösung

Hier wählen Sie die gewünschte Auflösung.

Hinweis

Der dpi-Wert ist gemäß der eingestellten Auflösung des eigenen Bildschirms und einem Vielfachen davon vorbelegt. Die Option ist nicht verfügbar, wenn Sie die Dateiart *.svg ausgewählt haben.

Qualität

Wenn Sie die Dateiart *.jpg ausgewählt haben, können Sie die gewünschte Qualität der exportierten Grafik festlegen.

Druckrahmen

Wenn Sie auf diese Schaltfläche klicken, öffnet sich das Fenster Druckrahmenparameter (Einstellungen für den Druckrahmen von Knotenströmen).

Klicken Sie auf die Schaltfläche Export starten.

Die Knotenstromansicht wird im ausgewählten Format gespeichert.

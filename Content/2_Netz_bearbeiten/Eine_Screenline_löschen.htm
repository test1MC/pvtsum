Eine Screenline löschen

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Screenlines.

Klicken Sie im Netz auf die Screenline, die Sie löschen möchten.

Drücken Sie die Taste Entf.

Alternativ können Sie die markierte Screenline über den Eintrag Löschen im Kontextmenü löschen.

Die Screenline wird gelöscht.

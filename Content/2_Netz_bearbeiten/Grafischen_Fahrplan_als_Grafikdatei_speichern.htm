Grafischen Fahrplan als Grafikdatei speichern

Sie können die aktuelle Ansicht des grafischen Fahrplans direkt in einer Grafikdatei .*.svg oder *.dxf speichern.

Wählen Sie im Menü Fahrplan (grafisch) einen der Einträge SVG Export oder DXF Export.

Das Fenster SVG-Datei speichern oder DXF-Datei speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster SVG-Export oder DXF-Export öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (siehe „SVG-Export“ auf Seite 1 und „DXF Export“ auf Seite 1).

Der grafische Fahrplan wird in einer Datei *.svg oder *.dxf gespeichert.

Sie können die Ansicht auch speichern, indem Sie im Menü Fahrplan (grafisch) den Eintrag JPG Export (Screenshot) wählen oder im Fenster Fahrplan (grafisch) in der Symbolleiste auf das Symbol  JPG Export (Screenshot) klicken (siehe „Screenshot vom Bildschirm exportieren“ auf Seite 1).

Sie können den grafischen Fahrplan auch ausdrucken (siehe „Grafischen Fahrplan drucken“ auf Seite 1).

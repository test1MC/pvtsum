Haltestellenbereiche: Allgemeine Attribute

Element

Beschreibung

Nummer

Eindeutige Nummer des Haltestellenbereichs

Haltestellenbereiche werden automatisch fortlaufend nummeriert. Sie können die vorgegebene Nummer mit einer Nummer überschreiben, die im Netz noch nicht vergeben ist.

Hinweis

Wenn Sie die Nummer des Haltestellenbereichs ändern, werden existierende Umlegungsergebnisse für alle ÖV-Nachfragesegmente gelöscht.

Code

Kurzbezeichnung des Haltestellenbereichs

Name

Langbezeichnung des Haltestellenbereichs

Typ

Der Typ (0...99) dient zur Kategorisierung des Haltestellenbereichs.

Haltestelle

Anzeige der Nummer der Haltestelle, der der Haltestellenbereich zugeordnet ist.

Standardmäßig wird die Nummer der nächstgelegenen Haltestelle verwendet. Über die Schaltfläche können Sie eine andere Haltestelle zuordnen.

Zugangsknoten

Anzeige der Nummer des Knotens, dem der Haltestellenbereich zugeordnet ist.

Sie können die Zuordnung über die Schaltfläche treffen oder ändern.

Tipp

Wählen Sie einen Anbindungsknoten für eine Bezirksanbindung an den Haltestellenbereich.

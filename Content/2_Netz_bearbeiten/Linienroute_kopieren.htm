Linienroute kopieren

Öffnen Sie das Fenster Linienrouten (Fenster Linienrouten öffnen).

Klicken Sie in der Liste auf die Linienroute, die Sie kopieren möchten.

Klicken Sie mit der rechten Maustaste in die Netzdarstellung.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Linienroute kopieren.

Das Fenster Linienroute kopieren öffnet sich.

Element

Beschreibung

Linie

Über die Auswahlliste können Sie eine Linie auswählen, der die Linienroute zugeordnet werden soll.

Name

Geben Sie in das Eingabefeld einen Namen für die neue Linienroute ein.

Hinweis

Standardmäßig wird der Name der kopierten Linienroute angezeigt.

Der Name der neuen Linienroute darf im Netz in der gleichen Richtung noch nicht vergeben sein. Wenn Sie die Gegenrichtung auswählen, können Sie der Linienroute den gleichen Namen geben wie der Hinrichtung.

Richtung

Über die Auswahlliste können Sie eine Richtung für die neue Linienroute wählen.

Alle Fahrplanfahrten mitkopieren

Ist die Option ausgewählt, werden mit der Linienroute alle Fahrplanfahrten und -fahrtabschnitte kopiert.

Geh- und Wartezeiten an Haltestellen mitkopieren

Ist die Option ausgewählt, werden mit der Linienroute alle Geh- und Wartezeiten an Haltestellen kopiert.

Verlauf einschränken

Durch die Auswahl eines Von- oder Nach-Haltepunkts kann die Linienroute verkürzt werden.

Von Haltepunkt

Über die Auswahlliste können Sie einen Von-Haltepunkt für die neue Linienroute auswählen.

Nach Haltepunkt

Über die Auswahlliste können Sie einen Nach-Haltepunkt für die neue Linienroute auswählen.

Nur Routenpunkte vorschlagen

Ist die Option ausgewählt, können Sie nur Routenpunkte mit Haltepunkt auswählen.

Die Linienroute wird einschließlich ihrer Attribute kopiert. In den Linienrouten vorhandene Zuordnungen zu Halten der Haltefolge werden mitkopiert. Die neue Linienroute wird im Fenster Linienrouten aufgelistet.

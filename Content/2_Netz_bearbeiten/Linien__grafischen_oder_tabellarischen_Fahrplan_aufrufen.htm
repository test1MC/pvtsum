Linien: grafischen oder tabellarischen Fahrplan aufrufen

Sie können Linien im Netzeditor auswählen und diese im grafischen oder tabellarischen Fahrplan anzeigen (Fahrplaneditor verwenden).

Grafischen oder tabellarischen Fahrplan mit ausgewählten Linien aufrufen

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Linien.

Das Fenster Linien öffnet sich.

Stellen Sie sicher, dass die Option Linien ausgewählt ist.

Drücken Sie die Taste Strg und halten Sie sie gedrückt.

Klicken Sie in der Liste nacheinander auf die Linien, die Sie anzeigen möchten.

Klicken Sie auf die Schaltfläche Fahrplan.

Wählen Sie in den gewünschten Fahrplan aus.

Wenn Sie eine Linie markiert haben, die in der Linienauswahl enthalten ist, öffnet sich der ausgewählte Fahrplan. Wenn Sie eine Linie markiert haben, die nicht ausgewählt ist, öffnet sich das Fenster Linienauswahl ändern öffnet sich (siehe „Linienauswahl ändern“ auf Seite 1). In dem Fenster können Sie gegebenenfalls Einstellungen für die Änderung der Linienauswahl vornehmen.

Der Fahrplan öffnet sich und die überfahrenen Linien werden je nach gewählten Optionen angezeigt.

Grafischen oder tabellarischen Fahrplan über die Suche nach einer Fahrplanfahrt aufrufen

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Linien.

Das Fenster Linien öffnet sich.

Drücken Sie die Tastenkombination Umschalt + F3.

Das Fenster Fahrplanfahrt suchen öffnet sich.

Alternativ können Sie im Kontextmenü der Schaltfläche Linien auf den Eintrag Fahrplanfahrt suchen klicken.

Wählen Sie in der Auswahlliste das Attribut, nach dem Sie suchen möchten.

Die Suche nach Name ist nur möglich, wenn für mindestens eine Fahrplanfahrt ein Name angegeben ist.

Geben Sie in das Eingabefeld den Wert oder Text ein, nach dem Sie suchen möchten.

Während des Eingebens wird in der Liste automatisch der erste Attributwert angezeigt und markiert, der dem eingegebenen Wert oder Text entspricht.

Wenn Sie in der Liste eine Fahrplanfahrt anklicken, wird die zugehörige Linie im Netz markiert.

Klicken Sie mit der rechten Maustaste auf das gewünschte Suchergebnis.

Wählen Sie im Kontextmenü den gewünschten Fahrplan aus.

Der ausgewählte Fahrplan öffnet sich. Alle Linien werden angezeigt, die eine Fahrplanfahrt mit dem ausgewählten Attributwert enthalten. Alle Fahrplanfahrten, die den ausgewählten Attributwert besitzen, sind markiert.

Wenn der Fahrplan bereits geöffnet ist, wird die Ansicht dem Suchergebnis entsprechend geändert.

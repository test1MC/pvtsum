Haltepunkt einfügen

Sie können Haltepunkte auf einer Strecke oder auf einem Knoten im Netz einfügen.

Klicken Sie im Fenster Netz auf das Symbol Modus Einfügen.

Klicken Sie auf die Schaltfläche Haltepunkte.

Das Fenster Haltepunkt einfügen (Optionen) öffnet sich.

Wählen Sie die gewünschten Optionen.

Element

Beschreibung

auf Knoten

Ist die Option ausgewählt, wird der Haltepunkt auf dem markierten Knoten platziert.

auf Strecke

Ist die Option ausgewählt, wird der Haltepunkt an der markierten Position im Streckenverlauf platziert. Die Position kann entlang der Strecke verschoben werden. Als Bezugsknoten wird automatisch der nächstgelegene Knoten der Strecke übernommen.

gerichtet

Ist die Option ausgewählt, wird der Haltepunkt neben der Strecke in Fahrtrichtung angezeigt und kann auch nur in dieser Richtung von ÖV-Linien bedient werden.

Ist die Option nicht ausgewählt, wird der Haltepunkt auf der Strecke ohne Richtungsbezug angezeigt und kann in beiden Richtungen von ÖV-Linien bedient werden.

Hinweise

Die Option wird nur angezeigt, wenn die Option auf Strecke ausgewählt ist.

Sie können die Eigenschaft gerichtet/ungerichtet eines Streckenhaltepunkts später nicht ändern.

Haltestelle und Haltestellenbereich automatisch erzeugen

Ist die Option ausgewählt, werden mit dem Haltepunkt gleichzeitig ein Haltestellenbereich und eine Haltestelle mit der gleichen Nummer erzeugt, sofern kein bereits existierender Haltestellenbereich ausgewählt wird.

Tipp

Wir empfehlen diese Einstellung, wenn Sie je Haltestelle nur einen Haltepunkt einfügen möchten.

Ist die Option nicht ausgewählt, werden die nächste Haltestelle und der zugehörige Haltestellenbereich übernommen für den neuen Haltepunkt.

Klicken Sie auf die Position im Netz, an der Sie den Haltepunkt einfügen möchten.

Das Fenster Haltepunkt einfügen öffnet sich.

Geben Sie die gewünschten Daten ein (Attribute und Optionen von Haltepunkten).

Der Haltepunkt wird im Netz eingefügt.

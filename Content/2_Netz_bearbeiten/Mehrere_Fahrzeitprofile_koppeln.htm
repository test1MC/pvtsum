Mehrere Fahrzeitprofile koppeln

Sie können Fahrzeitprofile mehrerer Linienrouten auf beliebig langen gemeinsam bedienten Abschnitten koppeln Siehe "Kopplung von Fahrzeitprofilen".

Alternativ können Sie im tabellarischen Fahrplan Fahrplanfahrten paarweise koppeln (Fahrplanfahrten paarweise koppeln).

Öffnen Sie das Fenster Linienrouten (Fenster Linienrouten öffnen).

Doppelklicken Sie auf die gewünschte Linienroute in der Liste.

Das Fenster Linienroute bearbeiten öffnet sich.

Wählen Sie das Register Verlauf und Fahrzeitprofile.

Das Register enthält alle Abschnitte der ausgewählten Linienroute.

Ändern Sie bei Bedarf die Anzeige der Attribute über das Symbol  Attribute wählen für rechte Tabelle  (Fenster zur Auswahl eines Attributs).

Wählen Sie die zu koppelnden Abschnitte aus, indem Sie die gewünschten Zeilen markieren.

Klicken Sie auf das Symbol  Ausgewählten Bereich mit anderen Fahrzeitprofilen koppeln.

Das Fenster Zu koppelnde Fahrzeitprofile auswählen öffnet sich. Alle Fahrzeitprofile (auch anderer Verkehrssysteme), für die in diesem Fall Kopplungen möglich sind, werden angezeigt.

Sie können die Liste der angezeigten Fahrzeitprofile durch Auswahl einer oder mehrerer der folgenden Optionen einschränken:

Element

Beschreibung

Gleiches Verkehrssystem

Ist die Option ausgewählt, werden nur Fahrzeitprofile gleicher Verkehrssysteme vorgeschlagen.

Gleicher Betreiber

Ist die Option ausgewählt, werden nur Fahrzeitprofile gleicher Betreiber vorgeschlagen.

Gleiche Fahr- und Haltezeiten

Ist die Option ausgewählt, werden nur Fahrzeitprofile vorgeschlagen, die über die gleichen Fahr- und Haltezeiten verfügen.

Gleiche Fahrten im Koppelabschnitt

Ist die Option ausgewählt, werden nur Fahrzeitprofile vorgeschlagen, die über äquivalente Fahrplanfahrten und Fahrplanfahrtabschnitte verfügen. Koppeln Sie die vorgeschlagenen Fahrzeitprofile, werden keine Fahrplanfahrten oder Fahrplanfahrtabschnitte eingefügt oder gelöscht.

Hinweis

Diese Option kann zusätzlich zu den anderen Optionen ausgewählt werden.

Wählen Sie in der Liste geeigneter Fahrzeitprofile ein oder mehrere Fahrzeitprofile aus.

Falls die Fahrzeitprofile über eine ungleiche Anzahl Fahrplanfahrten verfügen, öffnet sich eine Abfrage.

Bestätigen Sie die Abfrage.

Die Fahrzeitprofile werden gekoppelt und eventuell fehlende Fahrplanfahrten erzeugt.

Sie können im Register Verlauf und Fahrzeitprofile am Attribut Gekoppelte Fahrzeitprofile erkennen, welche Profile gekoppelt sind. Wählen Sie dazu das Attribut über das Symbol  Attribute wählen für die rechte Tabelle aus.

Wenn Sie Fahrzeitprofile abkoppeln möchten, wählen Sie die gewünschten Profilpunkte aus und klicken Sie auf das Symbol  Kopplung des ausgewählten Bereichs aufheben.

Wenn Sie den Verlauf von Linienrouten ändern, bleiben Kopplungen soweit möglich bestehen (Linienroutenverlauf digitalisieren).

Innerhalb von Kopplungen existierende Profilpunkte können Sie entfernen, indem Sie das entsprechende Kontrollkästchen deaktivieren.

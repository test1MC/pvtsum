Attribute und Optionen von Linien

Im Fenster Line <Name> bearbeiten können Sie Attribute von Linien anzeigen und ändern. Beispielsweise können Sie hier das Verkehrssystem einer Linie ändern. Der obere Teil des Fensters enthält die allgemeinen Attribute der Linie. Der untere Teil enthält spezielle Attribute.

Linien: Allgemeine Attribute

Element

Beschreibung

Name

Langbezeichnung der Linie

Verkehrssystem

In der Auswahlliste können Sie das Verkehrssystem eine Linie ändern. Zur Auswahl stehen alle im Netz angelegten ÖV-Verkehrssysteme. Eine Änderung hat diese Folgen:

Im Netz:

  • öffnet die betroffenen Strecken und Abbieger für das neue Verkehrssystem.
  • öffnet die betroffenen Haltepunkte, wenn das Ein- oder Aussteigen am Haltepunkt laut Fahrzeitprofil erlaubt ist.
  • verwirft Umlegungen und Ergebnisse der Linienleistungs- und -erfolgsrechnung (sofern vorhanden).

Innerhalb der Linienhierarchie:

  • prüft die Fahrzeugkombination. Ist die zugeordnete Fahrzeugkombination nicht für das neue Verkehrssystem zulässig, löscht die vorherige Fahrzeugkombination. Betroffen hiervon sind die Linie, das Fahrzeitprofil und der Fahrplanfahrtabschnitt. Durch eine Änderung der Fahrzeugkombination am Fahrplanfahrtabschnitt erhalten Umläufe den Status „Fahrzeugfehler”.

Oberlinie

Anzeige der Oberlinie, falls die Linie einer Oberlinie zugeordnet wurde

Tipp

Eine Linie können Sie einer Oberlinie über das Menü Netz > ÖV-Oberlinien zuordnen (Oberlinien verwalten).

Standard-FzgKomb

In der Auswahlliste können Sie eine Standardfahrzeugkombination für die Linie auswählen. Beim Einfügen neuer Fahrplanfahrten wird diese dann als Standardbelegung vorgeschlagen.

Tipp

Fahrzeugkombinationen können Sie über das Menü Netz > ÖV-Fahrzeuge anlegen (Fahrzeugkombination anlegen).

Standard-Betreiber

In der Auswahlliste können Sie einen Standardbetreiber für die Linie auswählen.

Hinweise

Beim Einfügen neuer Fahrplanfahrten wird dieser dann als Standardbelegung vorgeschlagen.

Wenn Sie den Standardbetreiber einer Linie nachträglich ändern, wird der Betreiber für zugehörige, bereits existierende Fahrplanfahrten nicht angepasst.

Tipp

Betreiber können Sie über das Menü Netz > ÖV-Betreiber anlegen (Betreiber anlegen).

Tarifsysteme

Hier können Sie ein Tarifsystem zuordnen oder ändern (Tarifzonen definieren).

Status der ICA-Berechnung prüfen

Sie können prüfen, ob für einen Knoten oder Oberknoten eine ICA-Berechnung durchgeführt werden kann. Eine ICA-Berechnung ist nur dann möglich, wenn dem Knoten oder Oberknoten eine Lichtsignalanlage zugeordnet ist.

Stellen Sie sicher, dass der gewünschte Knoten oder Oberknoten im Knoteneditor geöffnet ist.

Klicken Sie in der Symbolleiste des Knoteneditors auf das Symbol  Status ICA-Berechnung.

Eine Meldung zeigt an, ob die Berechnung für den angezeigten Knoten oder Oberknoten möglich ist oder nicht. Falls die Berechnung nicht möglich ist, erhalten Sie Informationen über die Gründe. Falls sie möglich ist, erhalten Sie zusätzliche Hinweise zu den relevanten Daten.

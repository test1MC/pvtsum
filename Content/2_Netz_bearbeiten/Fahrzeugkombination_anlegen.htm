Fahrzeugkombination anlegen

Sie können Fahrzeugkombinationen anlegen, um sie später einer Fahrplanfahrt oder einem Fahrplanfahrtabschnitt zuzuordnen.

Sie können eine Fahrzeugkombination automatisch erzeugen, wenn Sie eine Fahrzeugeinheit anlegen (Fahrzeugeinheit anlegen).

Wählen Sie im Menü Netz den Eintrag ÖV-Fahrzeuge.

Das Fenster Fahrzeugeinheiten und -kombinationen öffnet sich.

Stellen Sie sicher, dass das Register Fahrzeugkombinationen geöffnet ist.

Klicken Sie auf die Schaltfläche  Einfügen.

Das Fenster Fahrzeugkombination einfügen öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (Attribute und Optionen von Fahrzeugkombinationen).

Die Fahrzeugkombination wird eingefügt.

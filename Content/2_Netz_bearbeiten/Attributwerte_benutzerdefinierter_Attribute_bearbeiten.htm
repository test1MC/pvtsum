Attributwerte benutzerdefinierter Attribute bearbeiten

Für viele Netzobjekttypen wird beim Einfügen oder Bearbeiten eines Netzobjekts ein Fenster angezeigt, in dem Sie Attributwerte festlegen können. Für diese Netzobjekttypen können Sie wie folgt die Attributwerte benutzerdefinierter Attribute bearbeiten:

Benutzerdefinierte Attribute vom Typ Netz finden Sie über das Menü Netz > Netzeinstellungen im Register Benutzerdef. Attr.

Die Attributwerte der benutzerdefinierten Attribute vom Typ Knoten bearbeiten Sie im Knoteneditor (Knoten im Knoteneditor bearbeiten). Dort können Sie entsprechend auch die benutzerdefinierten Attributwerte von Abbiegern, Fahrstreifen usw. bearbeiten.

Fügen Sie ein benutzerdefiniertes Attribut für den gewünschten Netzobjekttyp ein (Benutzerdefiniertes Attribut einfügen).

Fügen Sie ein Netzobjekt des Netzobjekttyps ein oder bearbeiten Sie ein entsprechendes Netzobjekt.

Das Fenster <Netzobjekttyp> einfügen oder <Netzobjekttyp> bearbeiten öffnet sich.

Wählen Sie das Register Benutzerdef. Attr.

Das Register wird nur angezeigt, wenn es für den Netzobjekttyp ein benutzerdefiniertes Attribut gibt. Es enthält eine Liste aller benutzerdefinierten Attribute des Netzobjekttyps.

Wählen Sie bei Bedarf über die Schaltfläche aus, welche Attribute Sie anzeigen möchten.

Element

Beschreibung

Alle sichtbar

Über die Schaltfläche zeigen Sie alle benutzerdefinierten Attribute an.

Zeilen

Über die Schaltfläche wählen Sie in einem separaten Fenster die benutzerdefinierten Attribute, die angezeigt werden sollen (Fenster zur Auswahl mehrerer Attribute).

Legen Sie in der Zeile des gewünschten Attributs in der Spalte Wert den gewünschten Wert fest.

  • Attribut vom Typ Datei: Wählen Sie über die Schaltfläche die gewünschte Datei.
  • Attribut vom Typ Bool: Klicken Sie auf das Kontrollkästchen
  • Andere Typen: Klicken Sie in die Zelle und geben Sie die Zahl oder den Text ein.

Bei richtungsbezogenen Netzobjekttypen (Strecken, Anbindungen) wird ein Attributwert mit gelbem Hintergrund angezeigt, wenn die Attributwerte der Hin- und Rückrichtung unterschiedlich sind.

Der Attributwert wird geändert.

Außerdem können Sie Attributwerte von benutzerdefinierten Attributen bearbeiten wie andere Attributwerte (Attributwerte von Netzobjekten bearbeiten und Attributwerte in einer Liste bearbeiten).

Ausnahme: Die Attributwerte von benutzerdefinierten Formelattributen können Sie nicht ändern. Deshalb werden benutzerdefinierte Formelattribute in den Fenstern <Netzobjettyp> bearbeiten Register Benutzerdef. Attr. und Mehrfach-Bearbeiten <Netzobjettyp> nicht angezeigt.

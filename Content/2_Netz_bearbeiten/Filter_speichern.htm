Filter speichern

Sie können die aktuell definierten Filterbedingungen aller Netzobjekttypen in einer Datei speichern.

Definierte Filterbedingungen und eingeschaltete Filter werden auch mit der Versionsdatei gespeichert (Versionsdatei öffnen und speichern).

Wählen Sie im Menü Filter den Eintrag Filterdatei speichern.

Das Fenster Filter speichern öffnet sich.

Sie können das Fenster Filter speichern auch über Menü Datei > Speichern unter > Eintrag Filter aufrufen.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Die Filterbedingungen aller Netzobjektfilter werden gespeichert.

Wenn Sie nur den Filter eines bestimmten Netzobjekttyps speichern möchten, definieren Sie nur für diesen Netzobjekttyp Filterbedingungen und speichern sie diese. Wir empfehlen, beim Dateinamen kenntlich zu machen, für welchen Netzobjekttyp der Filter gilt, zum Beispiel K_Bsp.fil für eine Knotenfilterdatei.

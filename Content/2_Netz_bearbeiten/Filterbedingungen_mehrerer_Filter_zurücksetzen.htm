Filterbedingungen mehrerer Filter zurücksetzen

Wählen Sie im Menü Filter den Eintrag Initialisieren.

Das Fenster Filter initialisieren öffnet sich.

Wählen Sie die Netzobjektfilter, deren Filterbedingungen Sie zurücksetzen möchten.

Element

Beschreibung

Alle aus

Über die Schaltfläche können Sie alle Netzobjektfilter ausschalten.

Alle an

Über die Schaltfläche können Sie alle Netzobjektfilter auswählen.

Hier stehen nur diejenigen Netzobjekttypen zur Auswahl, bei denen aktuell ein Filter definiert ist.

Für die ausgewählten Netzobjektfilter werden die Filterbedingungen zurückgesetzt.

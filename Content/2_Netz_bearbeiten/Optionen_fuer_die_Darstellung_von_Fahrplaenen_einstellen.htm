Optionen für die Darstellung von Fahrplänen einstellen

Sie können einstellen, welche Schriftart für den tabellarische Fahrplan verwendet wird und welcher Ausrichtung auf den grafische Fahrplan angewendet wird.

Wählen Sie im Menü Bearbeiten den Eintrag Benutzereinstellungen.

Das Fenster Benutzereinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag Benutzeroberfläche > Fahrpläne.

Element

Beschreibung

Schrift für tabellarischen Fahrplan

Schriftart wählen

Über die Schaltfläche können Sie in einem separaten Fenster eine Schrift, einen Schriftschnitt und -grad für den tabellarischen Fahrplan festlegen.

Sie können die Schriftart wieder auf die -Standardeinstellung zurücksetzen (siehe „Schriftarten zurücksetzen“ auf Seite 1).

Orientierung des grafischen Fahrplans

Horizontal

Ist die Option ausgewählt, wird der grafische Fahrplan horizontal von links nach rechts ausgerichtet angezeigt.

Vertikal

Ist die Option ausgewählt, wird der grafische Fahrplan vertikal von oben nach unten ausgerichtet angezeigt.

Die Einstellungen werden übernommen.

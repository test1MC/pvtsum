Haltestelle einfügen

Sie können Haltestellen überall im Netz einfügen.

Klicken Sie im Fenster Netz auf das Symbol  Modus Einfügen.

Klicken Sie auf die Schaltfläche Haltestellen.

Klicken Sie an die Position im Netz, an der Sie die Haltestelle einfügen möchten.

Das Fenster Haltestelle einfügen öffnet sich.

Geben Sie die gewünschten Daten ein (Attribute und Optionen von Haltestellen).

Die Haltestelle wird im Netz eingefügt.

Alternativ können Sie eine Haltestelle, einen Haltestellenbereich und einen Haltepunkt mit identischer Nummer anlegen, indem Sie einen Haltepunkt einfügen mit aktivierter Option Haltestelle und Haltestellenbereich automatisch erzeugen (Haltepunkt einfügen).

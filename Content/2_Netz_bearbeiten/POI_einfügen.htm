POI einfügen

Bevor Sie einen POI einfügen, müssen Sie eine POI-Kategorie anlegen (POI-Kategorie anlegen).

Klicken Sie im Fenster Netz auf das Symbol Modus Einfügen.

Klicken Sie auf die Schaltfläche POIs.

Klicken Sie auf die Position im Netz, an der Sie den POI einfügen möchten.

Das Fenster POI einfügen öffnet sich.

Sie können die Position des POI später ändern.

Element

Beschreibung

Kategorie

Über die Auswahlliste wählen Sie die dem POI zugeordnete Kategorie.

Fläche

Ist die Option ausgewählt, können Sie ein Polygon für den POI definieren.

Kategorien verwalten

Klicken Sie auf diese Schaltfläche, wenn Sie das Fenster POI-Kategorien bearbeiten öffnen möchten, um die POI-Kategorien zu bearbeiten (POI-Kategorie ändern).

Tipp

Sie können die Kategorie für den POI auch über das Fenster Kategorien verwalten auswählen.

Das Fenster POI einfügen öffnet sich.

Geben Sie die gewünschten Daten ein (Attribute und Optionen von POIs).

Wenn Sie die Option Fläche ausgewählt haben, sind Sie jetzt im Einfügemodus und können die Umrandung des Polygons einfügen (Umrandung einfügen).

Der POI wird eingefügt.

Die Anzeige von POIs steuern Sie über das Menü Grafik > Grafikparameter bearbeiten > Eintrag Points of Interest (POIs anzeigen).

01Seitenränder anzeigen und bearbeiten

Sie können im grafischen Fahrplan bis zu vier Ränder anzeigen.

Klicken Sie mit der rechten Maustaste in den grafischen Fahrplan.

Element

Beschreibung

Oberer Rand sichtbar

Am oberen Rand wird eine Achse ausgegeben.

Unterer Rand sichtbar

Am unteren Rand wird eine Achse ausgegeben.

Linker Rand sichtbar

Am linken Rand wird eine Achse ausgegeben.

Rechter Rand sichtbar

Am rechten Rand wird eine Achse ausgegeben.

Rufen Sie das Kontextmenü erneut auf, wenn Sie weitere Ränder anzeigen möchten.

Die Ränder werden angezeigt.

Sie können Ränder entfernen, indem Sie den jeweiligen Eintrag erneut im Kontextmenü aufrufen.
Elemente verschieben

Sie können die Elemente auf der jeweiligen Achse verschieben.

Klicken Sie auf den gewünschten Zeitpunkt oder Halt und halten Sie die Maustaste gedrückt.

Ziehen Sie das Element an den gewünschten Ort und lassen Sie die Maustaste los.

Die Beschriftung wird verschoben.

Sie können ein Element nicht vor ein anderes verschieben.

Wenn Sie einen Zeitpunkt verschieben, werden die Abstände aller Zeiten proportional angepasst.

Wenn Sie einen Halt verschieben, bleibt der Abstand des verschobenen Elements zu allen nachfolgenden konstant. Sie können einen Halt jedoch einzeln verschieben, indem Sie die Taste Strg gedrückt halten während Sie den gewünschten Halt verschieben.

Gegenrichtung einer Systemroute einfügen

Mit dieser Funktion können Sie die Gegenrichtung einer Systemroute einfügen, die als neue Systemroute mit entsprechendem Verlauf und identischen Attributen erzeugt wird.

Wenn Sie der neuen Systemroute einen neuen Namen geben und ihr die gleiche Richtung wie die der ausgewählten Systemroute zuweisen, entsteht eine Kopie der Systemroute.

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Systemrouten.

Das Fenster Systemrouten öffnet sich.

Klicken Sie in der Liste auf die Systemroute, der Sie eine Gegenrichtung hinzufügen möchten.

Klicken Sie mit der rechten Maustaste in die Netzdarstellung.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Gegenrichtung einfügen.

Das Fenster Gegenrichtung einfügen öffnet sich.

Geben Sie in das Eingabefeld einen Namen für die neue Systemroute ein.

Die Systemroute in Gegenrichtung wird eingefügt.

Startet oder endet die ursprüngliche Systemroute an einem gerichteten Streckenhaltepunkt (Haltepunkte verwalten), wird dieser auch für die Gegenrichtung eingefügt, wenn Sie eine Abfrage bestätigen. Gibt es weitere Streckenhaltepunkte im Verlauf, können Sie auch diese für die Gegenrichtung übernehmen.

Räumliche Auswahl von Netzobjekten lesen

Sie können Netzobjekte, die in einer räumlichen Auswahl enthalten sind, von Datei einlesen.

Nur Netzobjekte, die den aktuellen Filterbedingungen entsprechen, können aktiv gesetzt werden (Netzobjekte über Filter aktiv und passiv setzen).

Klicken Sie im Fenster Netz auf das Symbol Modus Räumliche Auswahl.

Die Symbolleiste Räumliche Auswahl wird angezeigt.

Klicken Sie in der Symbolleiste Räumliche Auswahl auf das Symbol Räumliche Auswahl von Datei lesen.

Das Fenster Aktive Netzelemente öffnen öffnet sich.

Sie können das Fenster Aktive Netzelemente öffnen auch über Menü Datei > Öffnen > Eintrag Aktive Netzelemente aufrufen.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Das Fenster Aktive Netzelemente lesen öffnet sich.

Wählen Sie die Netzobjekttypen, für die Sie die räumliche Auswahl von Datei lesen möchten.

Element

Beschreibung

hinzulesen

Ist die Option ausgewählt, bleiben die aktuell aktiven Netzobjekte aktiv. Beim Lesen werden eventuell weitere Netzobjekte aktiv.

Ist die Option nicht ausgewählt oder steht sie nicht zur Verfügung, werden nur diejenigen Netzobjekte aktiv, die in der Datei als aktiv gespeichert sind.

Hinweis

Die Option steht nur zur Verfügung, wenn im Netz mindestens ein Netzobjekt passiv ist.

Alle an

Über die Schaltfläche können Sie alle Optionen auswählen.

Alle aus

Über die Schaltfläche können Sie alle Optionen ausschalten.

Die entsprechenden Netzobjekte werden aktiv oder passiv gesetzt.

Sie können sich über das Menü Netz > Netzstatistik einen Überblick darüber verschaffen, wie viele Netzobjekte je Netzobjekttyp aktuell in der räumlichen Auswahl enthalten, wie viele gefiltert und wie viele aktiv sind (Überblick über die Zustände aller Netzobjekte).

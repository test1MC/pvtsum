Fahrplanfahrtabschnitt bearbeiten

Stellen Sie sicher, dass im Menü Fahrplan (tabellarisch) die Option Fahrplanfahrtabschnitte anzeigen ausgewählt ist. Im Taktfahrtenmodus werden keine Fahrplanfahrtabschnitte angezeigt.

Sie können einen Fahrplanfahrtabschnitt entweder direkt im tabellarischen Fahrplan oder im Fenster Fahrplanfahrtabschnitt <Nummer Fahrplanfahrt (Bezeichnung Fahrplanfahrtabschnitt)> bearbeiten bearbeiten. Dieses Fenster wird hier beschrieben.

Stellen Sie sicher, dass der tabellarische Fahrplan geöffnet ist und Fahrplanfahrtabschnitte vorhanden sind.

Klicken Sie im tabellarischen Fahrplan mit der rechten Maustaste in die Spalte des Fahrplanfahrtabschnitts, den Sie bearbeiten möchten.

Das Kontextmenü öffnet sich.

Wählen Sie im Kontextmenü den Eintrag Fahrplanfahrtabschnitt bearbeiten.

Das Fenster Fahrplanfahrtabschnitt <Nummer Fahrplanfahrt (Bezeichnung Fahrplanfahrtabschnitt)> bearbeiten öffnet sich. Im oberen Fensterbereich werden allgemeine Informationen zur Fahrplanfahrt und zum Fahrplanfahrtabschnitt angezeigt, die Sie nicht ändern können.

Sie können auch mehrere markierte Fahrplanfahrtabschnitte bearbeiten. Über das Kontextmenü öffnet sich dann das Fenster Mehrfach-Bearbeiten Fahrplanfahrtabschnitte (siehe „Attributwerte von Netzobjekten bearbeiten“ auf Seite 1)

Legen Sie im unteren Bereich des Fensters im Register Basis die gewünschten Einstellungen fest.

Element

Beschreibung

Fahrzeugkombination

In der Auswahlliste können Sie eine andere Fahrzeugkombination für den Fahrplanfahrtabschnitt auswählen.

Fahrzeugkombinations-Set

Klicken Sie auf die Schaltfläche, um das Fahrzeugkombinations-Set zu ändern.

Verkehrstag

In der Auswahlliste können Sie eine andere Einstellung für den Verkehrstag festlegen, wenn der Fahrplanfahrtabschnitt einen abweichenden Verkehrstag hat.

Vorbereitungszeit

Hier können Sie die für den Fahrplanfahrtabschnitt spezifische Vorbereitungszeit für die Umlaufbildung festlegen. Die Umlaufbildung erfolgt auf der Basis von Fahrplanfahrtabschnitten.

Hinweis

Eine spezifische Vorbereitungszeit pro Fahrzeugkombination können Sie über das Register Spezifische Wendezeiten festlegen.

Nachbereitungszeit

Hier können Sie die für den Fahrplanfahrtabschnitt spezifische Nachbereitungszeit für die Umlaufbildung festlegen. Die Umlaufbildung erfolgt auf der Basis von Fahrplanfahrtabschnitten.

Hinweis

Eine spezifische Nachbereitungszeit pro Fahrzeugkombination können Sie über das Register Spezifische Wendezeiten festlegen.

StartHPunkt

Wählen Sie den Start-Haltepunkt für den Fahrplanfahrtabschnitt aus.

EndHPunkt

Wählen Sie den End-Haltepunkt für den Fahrplanfahrtabschnitt aus.

Wählen Sie das Register Spezifische Wendezeiten.

Element

Beschreibung

Spezifische Vorbereitungszeit verwenden

Ist die Option ausgewählt, wird in der Liste für jede Fahrzeugkombination eine Zeile für die Vorbereitungszeit angezeigt. In der Spalte Wert können Sie jeweils eine spezifische Vorbereitungszeit eingeben.

Ist die Option nicht ausgewählt, wird für alle Fahrzeugkombinationen die Vorbereitungszeit aus dem Register Basis verwendet.

Spezifische Nachbereitungszeit verwenden

Ist die Option ausgewählt, wird in der Liste für jede Fahrzeugkombination eine Zeile für die Nachbereitungszeit angezeigt. In der Spalte Wert können Sie jeweils eine spezifische Nachbereitungszeit eingeben.

Ist die Option nicht ausgewählt, wird für alle Fahrzeugkombinationen die Nachbereitungszeit aus dem Register Basis verwendet.

An der Spalte Im Fahrzeugkombinations-Set können Sie erkennen, ob die Fahrzeugkombination für den Fahrplanfahrtabschnitt zugelassen ist. Die Zeilen sind entsprechend grau hinterlegt. Tatsächlich zugeordnete Fahrzeugkombinationen erkennen Sie über die Spalte Zugeordnete Fahrzeugkombination. Sie werden fett gedruckt angezeigt. Sie können die Einträge nach den Spalten Im Fahrzeugkombinations-Set und Zugeordnete Fahrzeugkombination sortieren.

Der Fahrplanfahrtabschnitt wird entsprechend Ihren Angaben geändert.

Linien markieren

Sie können eine Linie im Netz nur dann markieren, wenn der Linie mindestens eine Linienroute und daher ein Verlauf zugeordnet ist.

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Linien.

Das Fenster Linien öffnet sich.

Abhängig von der gewählten Option, ändert sich der Name des Fensters Linien/Linienrouten.

Stellen Sie sicher, dass die Option Linien ausgewählt ist.

Schränken Sie bei Bedarf die Menge der Linien ein, die angezeigt wird.

Element

Beschreibung

nur aktive

Ist die Option ausgewählt, werden nur aktive Linien angezeigt.

Ist die Option nicht ausgewählt, werden alle Linien angezeigt.

Klicken Sie im Fenster Linien auf eine Linie in der Liste. Wenn Sie mehrere Linien markieren möchten, drücken Sie die Taste Strg und halten Sie sie gedrückt. Klicken Sie nacheinander auf die Linien, die Sie markieren möchten.

Sie können die gesuchte Linie auch über das Eingabefeld markieren. Geben Sie hierzu den Namen der Linie in das Feld ein. Die Linie wird in der Liste markiert.

Die Linienroutenverläufe der markierten Linien werden im Netz hervorgehoben. Unter der Liste wird angezeigt (im Beispiel: 1(11)), wie viele Linien Sie markiert haben.

Klicken Sie bei Bedarf auf die Schaltfläche Autozoom.

Der Netzausschnitt wird optimiert, sodass die markierten Linien in der Mitte des Ausschnitts angezeigt werden.

Wenn Sie alle markierten Linien aus der Auswahl entfernen möchten, klicken Sie auf die Schaltfläche Initialisieren.

Sie können Linienroutenverläufe einer Linie auch markieren, indem Sie im Netz auf einen Haltepunkt klicken, über den eine Linienroute der Linie verläuft.

Alternativ können Sie eine Linie markieren, indem Sie die entsprechende Zeile in der Liste der Linien auswählen (Listenbereiche auswählen).

Netzobjekte grafisch zuordnen

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche POIs.

Klicken Sie auf den POI, den Sie zuordnen möchten.

Klicken Sie mit der rechten Maustaste auf den POI.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Zuordnen.

Das Fenster POI Zuordnen öffnet sich.

Wählen Sie in der Auswahlliste Ziel-Netzobjektart den gewünschten Netzobjekttyp aus.

Wählen Sie eine der folgenden Optionen aus:

Element

Beschreibung

Einfügen

Ist die Option ausgewählt, können Sie ein Netzobjekt zuordnen.

Hinweis

Bei Strecken und Knoten öffnet sich ein Fenster, in dem Sie weitere Einstellungen vornehmen können (Einstellungen für POI-Zuordnungen vornehmen).

Bearbeiten

Ist die Option ausgewählt, können Sie für eine Zuordnung zu einer Strecke oder einem Knoten die Einstellungen ändern (Einstellungen für POI-Zuordnungen vornehmen).

Löschen

Ist die Option ausgewählt, können Sie eine Zuordnung löschen.

Nehmen Sie für die Option Details bearbeiten die gewünschte Einstellung vor.

Die Option ist nur beim Zuordnen von Strecken aktiv.

Ist die Option ausgewählt, öffnet sich direkt nach dem Zuordnen ein Fenster für zusätzliche Einstellungen (Einstellungen für POI-Zuordnungen vornehmen).

Klicken Sie auf das gewünschte Netzobjekt.

Bei Strecken und Knoten öffnet sich ein Fenster, in dem Sie weitere Einstellungen vornehmen können (Einstellungen für POI-Zuordnungen vornehmen).

Wenn Sie eine Strecke zuordnen, wird die Position des Klickpunkts in die Einstellungen übernommen.

Die Zuordnung wird eingefügt, geändert oder gelöscht.

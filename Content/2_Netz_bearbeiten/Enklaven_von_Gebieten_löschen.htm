Enklaven von Gebieten löschen

Sie können Enklaven, also „Löcher” in den Flächen von Gebieten löschen.

Setzen Sie bei Bedarf die Gebiete aktiv, deren Enklaven Sie löschen möchten, über

  • die räumliche Auswahl (Netzobjekte über die räumliche Auswahl aktiv und passiv setzen) und/oder
  • Filter (Netzobjekte über Filter aktiv und passiv setzen).

Die ausgewählten Oberbezirke sind aktiv.

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Gebiete.

Das Kontextmenü Gebiete öffnet sich.

Wählen Sie den Eintrag Enklaven löschen.

Das Fenster Mehrfach-Bearbeiten Gebiete öffnet sich.

Ist der Netzobjekttyp Gebiete ausgewählt, können Sie das Fenster Mehrfach-Bearbeiten Gebiete auch über das Menü Bearbeiten > Enklaven löschen aufrufen.

Stellen Sie sicher, dass das Register Sonderfunktionen ausgewählt ist.

Wählen Sie aus, ob die Enklaven aller oder aller aktiven Gebiete gelöscht werden soll.

Element

Beschreibung

nur aktive

Ist die Option ausgewählt, werden nur aktive Gebiete berücksichtigt.

Klicken Sie auf die Schaltfläche Enklaven löschen.

Eine Abfrage öffnet sich.

Die Enklaven werden gelöscht.

Klicken Sie auf die Schaltfläche Schließen.

Filterbedingungen für Beziehungen und ÖV-Wege definieren

Sie können mit dem Beziehungsfilter Bedingungen für Bezirksbeziehungen und ÖV-Wege definieren.

Filterbedingungen für Bezirksbeziehungen definieren

Sie können Beziehungen zwischen Bezirken anhand folgender Bedingungen filtern:

  • Luftlinienentfernung zwischen den Bezirksschwerpunkten
  • Werte von direkten und indirekten Bezirksattributen
  • Werte einer Nachfragematrix

Stellen Sie sicher, dass der Beziehungsfilter geöffnet ist (Beziehungsfilter öffnen).

Klicken Sie auf die Schaltfläche unterhalb des Spaltenkopfes Relation.

Das Fenster Relationen (Bezirks-Beziehungen) öffnet sich.

Wählen Sie die gewünschte Relation.

Die gewählte Relation wird angezeigt.

Klicken Sie auf die Schaltfläche unterhalb des Spaltenkopfes Attribut.

Das Fenster Attribute (Bezirks-Beziehungen) öffnet sich.

Wählen Sie das gewünschte Attribut (Fenster zur Auswahl eines Attributs).

Das gewählte Attribut wird angezeigt.

Wählen Sie in der Spalte Operation den gewünschten Operator aus (Eine Filterbedingung definieren).

Legen Sie den gewünschten Wertebereich fest.

Wählen Sie bei Bedarf die Option Kompl..

Diese Option kehrt die Filterbedingung in dieser Zeile um. Wenn sie gewählt ist, werden genau die Netzobjekte aktiv, die sonst durch die Filterbedingung passiv würden und umgekehrt.

Wählen Sie bei Bedarf die Option Netzbelastungen filtern.

Ist die Option nicht ausgewählt, werden die Netzbelastungen nicht neu berechnet für den Filter.

Klicken Sie bei Bedarf auf die Schaltfläche Einfügen.

Eine weitere Filterbedingung wird eingefügt. In der Spalte Verknüpfung ist automatisch Und eingestellt.

Verknüpfung

Beschreibung

Und

Die mit Und verknüpften Filterbedingungen wirken additiv, d. h. eine Beziehung wird nur aktiv, wenn sie alle mit Und verknüpften Filterbedingungen gleichzeitig erfüllt.

Oder

Eine Beziehung wird aktiv, wenn sie mindestens eine der mit Oder verknüpften Filterbedingungen erfüllt.

Die Eintragungslogik folgt dabei dem Satz von de Morgan A oder B = Nicht (Nicht A und nicht B).

Klicken Sie bei Bedarf auf die Schaltfläche Und.

In der Spalte Verknüpfung wird Oder eingestellt.

Definieren Sie eine weitere Filterbedingung.

Fügen Sie bei Bedarf weitere Filterbedingungen hinzu und legen Sie dabei die Art der Verknüpfung fest.

Wenn Sie mehr als drei Zeilen einfügen, können Sie im Bereich Filterausdruck über die Schaltfläche Klammern setzen Klammern im Filterausdruck setzen (Verknüpfte Filterbedingungen definieren).

Wenn Sie eine Filterbedingung löschen möchten, markieren Sie die gewünschte Zeile und klicken Sie auf die Schaltfläche Löschen.

Wenn Sie mehrere Filterbedingungen löschen möchten, drücken Sie die Taste Strg und halten Sie sie gedrückt. Markieren Sie die gewünschten Zeilen und klicken Sie auf die Schaltfläche Löschen.
Beispiel

Der Beziehungsfilter filtert alle Beziehungen, die folgende Bedingungen erfüllen:

  • Die Quelle ist ein Bezirk, der Teil eines Oberbezirks mit ZWert1 ≥ 1000 ist.
  • Die Luftlinienweite zwischen den Bezirksschwerpunkten ist ≤ 2 Kilometer.
  • Die Von-Bezirks-Nummer ist ungleich 100.
Filterbedingungen für ÖV-Wege definieren

Sie können zusätzlich zu den Beziehungsfiltern Filterbedingungen für ÖV-Wege angeben, um festzulegen, welche ÖV-Wege in die ÖV-Netzbelastungen einfließen.

Stellen Sie sicher, dass der Beziehungsfilter geöffnet ist (Beziehungsfilter öffnen).

Wählen Sie das Register ÖV-Wege.

Klicken Sie auf die Schaltfläche unterhalb des Spaltenkopfes Relation.

Das Fenster Relationen (ÖV-Wege) öffnet sich.

Wählen Sie die gewünschte Relation.

Die gewählte Relation wird angezeigt.

Klicken Sie auf die Schaltfläche unterhalb des Spaltenkopfes Attribut.

Das Fenster Attribute (ÖV-Wege) öffnet sich.

Wählen Sie das gewünschte Attribut (Fenster zur Auswahl eines Attributs).

Sie können nach dem Nachfragesegment des ÖV-Weges filtern, indem Sie das Attribut NSeg-Code auswählen. Ein Weg ist genau dann einem Nachfragesegment zugeordnet, wenn er für das Nachfragesegment eine Belastung größer null hat.

Das gewählte Attribut wird angezeigt.

Wählen Sie in der Spalte Mengenoperation in der Auswahlliste die gewünschte Mengenoperation aus.

Element

Beschreibung

Eines

Die Operation wird für ein ausgewähltes Objekt durchgeführt.

Alle

Die Operation wird für alle Objekte durchgeführt.

Das Feld ist leer, wenn Sie in der Spalte Relation keine 1:n-Relation ausgewählt haben.

Wählen Sie in der Spalte Operation den gewünschten Operator (Eine Filterbedingung definieren).

Legen Sie den gewünschten Wertebereich fest.

Wählen Sie bei Bedarf die Option Kompl..

Diese Option kehrt die Filterbedingung in dieser Zeile um. Wenn sie gewählt ist, werden genau die ÖV-Wege aktiv, die sonst durch die Filterbedingung passiv würden.

Klicken Sie bei Bedarf auf die Schaltfläche Einfügen.

Eine weitere Filterbedingung wird eingefügt. In der Spalte Verknüpfung ist automatisch Und eingestellt.

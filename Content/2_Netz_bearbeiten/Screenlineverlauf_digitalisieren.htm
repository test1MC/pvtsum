Screenlineverlauf digitalisieren

Sie können den konkreten Verlauf bereits beim Einfügen einer Screenline digitalisieren (Screenline einfügen).

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Screenlines.

Klicken Sie im Netz auf die Screenline, die Sie digitalisieren möchten.

Die Screenline wird markiert.

Klicken Sie mit der rechten Maustaste in die Netzdarstellung.

Das Kontextmenü Screenlines öffnet sich.

Wählen Sie den Eintrag Digitalisieren.

Das Fenster Screenline digitalisieren öffnet sich.

Drücken Sie alternativ die Taste F5.

Klicken Sie im Netz auf den Polygonpunkt, den Sie neu positionieren möchten, und halten Sie die Maustaste gedrückt.

Ziehen Sie den Polygonpunkt an die gewünschte Position und lassen Sie die Maustaste los.

Ziehen Sie bei Bedarf weitere Polygonpunkte an andere Positionen.

Löschen Sie bei Bedarf vorhandene Polygonpunkte indem Sie die Taste Strg drücken und auf den Polygonpunkt klicken.

Fügen Sie bei Bedarf weitere Polygonpunkte ein, indem Sie die Taste Strg drücken und auf den Verlauf klicken.

Klicken Sie alternativ in die Netzdarstellung, um den Verlauf zu bestätigen.

Der Verlauf wird geändert.

Haltestellenbereiche verwalten

Haltestellenbereiche unterteilen eine Haltestelle in Bereiche. Ein Bereich kann dabei ein Bus- oder Bahnsteig, eine Kreuzung mit Haltepunkten, einen Park-and-Ride-Parkplatz, eine Bahnhofshalle etc. darstellen. Ein Haltestellenbereich ist genau einer Haltestelle zugeordnet und kann mehrere Haltepunkte umfassen. Ein Bereich kann einem Netzknoten zugeordnet werden, so dass über den Bereich die Verknüpfung mit dem Streckennetz via Anbindungen möglich ist. Die Haltestellenbereiche einer Haltestelle können mittels einer Matrix der Übergangsgehzeiten Von-Bereich-Nach-Bereich miteinander verbunden werden, die die Gehzeiten für jedes Verkehrssystem vom Typ ÖVFuß enthält (Haltestellenbereiche).

Haltestellenbereiche und damit Haltepunkte können Sie einer Haltestelle nur zuordnen, indem Sie den Haltestellenbereich bearbeiten (Attributwerte eines Haltestellenbereichs ändern und Haltestellenbereich einfügen) oder indem Sie einen Haltepunkt mit aktivierter Option Haltestelle und Haltestellenbereich automatisch erzeugen einfügen (Haltepunkt einfügen).

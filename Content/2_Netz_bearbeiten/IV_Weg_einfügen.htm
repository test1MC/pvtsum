IV-Weg einfügen

Wege werden automatisch je berechnetem Nachfragesegment eingefügt, wenn Sie eine IV-Umlegung berechnen. Wenn Sie die Umlegung initialisieren, werden die so erzeugten Wege wieder gelöscht.

Sie können Wege auch manuell einfügen und diese Nachfragesegmenten oder Wegemengen zuordnen. Wege, die Wegemengen zugeordnet sind, bleiben erhalten, wenn Sie Umlegungen initialisieren. Wenn Sie Wege manuell einfügen, die Nachfragesegmenten zugeordnet sind, weichen die Attribute ab (Attribute und Optionen von IV-Wegen). Diese Wege werden auch gelöscht, wenn eine Umlegung für das Nachfragesegment initialisiert wird.

Klicken Sie im Fenster Netz auf das Symbol > Modus Einfügen.

Klicken Sie auf die Schaltfläche IV-Wege.

Das Fenster Einfügen öffnet sich.

Wählen Sie den Eintrag Weg.

Klicken Sie in die Netzdarstellung.

Das Fenster Auswahl der Wegemenge öffnet sich.

Wählen Sie in der Auswahlliste die gewünschte Wegemenge aus, für die Sie einen Weg einfügen möchten.

Sie können Nachfragesegmente oder Wegemengen auswählen. Wenn Sie einen Weg für ein Nachfragesegment einfügen (Attribute und Optionen von IV-Wegen), muss der Weg an einem Bezirk starten und auch enden.

Das Fenster Weg digitalisieren öffnet sich.

Klicken Sie auf den Knoten oder Bezirk, an dem der Weg starten soll und halten Sie die Maustaste gedrückt.

Ein Weg muss an einem Knoten oder Bezirk starten und enden. Startet der Weg an einem Knoten, muss er auch an einem Knoten enden. Startet er an einem Bezirk, muss er an einem Bezirk enden.

Auch Oberknoten können Bestandteile von Wegen sein. Sie können jedoch nur über die Teilknoten ausgewählt werden.

Ziehen Sie die Maus an den nächsten Bezirk oder Knoten, über den der Weg verlaufen soll und lassen Sie die Maustaste los.

Klicken Sie bei Bedarf im Fenster Weg digitalisieren auf die Schaltfläche Autozoom, wenn Sie den Ausschnitt mit dem Verlauf vergrößern möchten.

Fügen Sie bei Bedarf auf die gleiche Weise weitere Teilverläufe ein.

Klicken Sie bei Bedarf im Fenster Weg digitalisieren auf die Schaltfläche Undo, wenn Sie das letzte Teilstück aus dem Wegverlauf entfernen möchten.

Ziehen Sie die Maus auf den gewünschten Endpunkt des Weges und lassen Sie die Maustaste los.

Sie können die Maus auch direkt vom ersten Bezirk oder Knoten zum gewünschten Endpunkt ziehen. Das Programm schlägt dann den besten Weg vor entsprechend dem eingestellten Suchkriterium (siehe „Wege: Parameter für die Wegesuche einstellen“ auf Seite 408).

Klicken Sie im Fenster Weg digitalisieren auf OK.

Das Fenster Weg einfügen öffnet sich.

Geben Sie die gewünschten Daten ein (Attribute und Optionen von IV-Wegen).

Der Weg wird eingefügt.

Sie können den Verlauf eines Weges nachträglich ändern (Wegeverlauf digitalisieren).
Wege: Parameter für die Wegesuche einstellen

Klicken Sie im Fenster Weg digitalisieren auf die Schaltfläche Parameter.

Das Fenster Parameter für die Wegesuche öffnet sich.

Element

Beschreibung

Bezugs-Verkehrssystem

Auswahl des gewünschten Verkehrssystem

auch über gesperrte Strecken routen

Ist die Option ausgewählt, werden für das Verkehrssystem gesperrte Strecken berücksichtigt, als ob sie für das Verkehrssystem geöffnet wären.

auch über gesperrte Abbieger routen

Ist die Option ausgewählt, werden für das Verkehrssystem gesperrte Abbieger berücksichtigt, als ob sie für das Verkehrssystem geöffnet wären.

Suchkriterium

In der Auswahlliste können Sie ein Kriterium für die Wegesuche auswählen. Gesucht wird der jeweils beste Weg zwischen zwei markierten (fixierten) Punkten.

Streckenlänge-Luft

Luftlinien-Streckenlänge vom Start- zum End-Haltepunkt

Streckenfahrzeit

Wird aus der Verkehrssystem-Geschwindigkeit der Strecken berechnet

Zeit aus Streckentypgeschwindigkeit

Wird aus der für den betreffenden Streckentyp definierten Verkehrssystem-Geschwindigkeit berechnet

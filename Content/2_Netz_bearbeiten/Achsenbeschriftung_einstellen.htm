Achsenbeschriftung einstellen

Für die Halte-Achse und die Zeit-Achse können Sie abweichende Achsenbeschriftungen einstellen.

Anzahl und Position der Halte-und Zeit-Achsen einstellen

Sie können einstellen, ob die Achsen für Halte und die Zeit jeweils einmal angezeigt wird oder auf der gegenüber liegenden Seite ein zweites Mal.

Klicken Sie in der Symbolleiste des Fahrplans auf das Symbol  Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch)öffnet sich.

Wählen Sie im Navigator die Seite Achsen.

Element

Beschreibung

Primäre (obere / linke) Halte-Achse anzeigen

Ist die Option ausgewählt, wird die Beschriftung für die Halte-Achse über oder links neben dem grafischen Fahrplan angezeigt.

Hinweis

Die Position der primären Halte- und Zeit-Achse ist abhängig von der Ausrichtung des Fahrplans. Sie können die Ausrichtung in den Benutzereinstellungen einstellen (siehe „Optionen für die Darstellung von Fahrplänen einstellen“ auf Seite 1).

Sekundäre (untere / rechte) Halte-Achse anzeigen

Ist die Option ausgewählt, wird die Beschriftung für die Halte-Achse unter oder rechts neben dem grafischen Fahrplan angezeigt.

Primäre (obere / linke) Zeit-Achse anzeigen

Ist die Option ausgewählt, wird die Beschriftung für die Zeit-Achse über oder links neben dem grafischen Fahrplan angezeigt.

Sekundäre (untere / rechte) Zeit-Achse anzeigen

Ist die Option ausgewählt, wird die Beschriftung für die Zeit-Achse unter oder rechts neben dem grafischen Fahrplan angezeigt.

Wenn Sie auf die Schaltfläche Vorschau klicken, werden Ihre Änderungen im Fahrplan sofort angezeigt, ohne dass sich das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch) schließt.

Die Achsen werden gemäß den Einstellungen angezeigt.

Achsenbeschriftung für die Halte-Achse einstellen

Klicken Sie in der Symbolleiste des Fahrplans auf das Symbol  Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch)öffnet sich.

Wählen Sie im Navigator die Seite Achsen > Halte-Achse > Beschriftung.

Element

Beschreibung

Einheitliche Darstellung

Ist die Option ausgewählt, werden alle Achsenbeschriftungen einheitlich dargestellt. Wählen Sie im Bereich Darstellung die Option Text zeichnen aus und nehmen Sie die weiteren Einstellungen vor.

Klassifizierte Darstellung

Ist die Option ausgewählt, kann die Achsenbeschriftung nach einem Attribut klassifiziert und dargestellt werden (Objekte anhand von Attributwerten klassifiziert darstellen).

Bereich Darstellung (Beschriftung)

Text zeichnen

Ist die Option ausgewählt, wird eine Achsenbeschriftung gezeichnet.

Inhalt

Über die Schaltfläche können Sie das Attribut auswählen, mit welchem die Achse beschriftet wird (Fenster zur Auswahl eines Attributs).

Größe

Textgröße in mm

Font

Wenn Sie in das Feld klicken, können Sie eine Schriftart für die Beschriftung auswählen.

Transparent

Ist die Option ausgewählt, wird die Beschriftung nur als Schrift ohne weißen Hintergrund gezeichnet.

Farbe

Wenn Sie in das Feld klicken, können Sie eine Farbe für die Achsenbeschriftung auswählen.

Rahmen

Wenn Sie die Option auswählen, wird die Achsenbeschriftung umrahmt. Wenn Sie in das Feld klicken, können Sie eine Rahmenfarbe auswählen.

Runden

Rundungsfaktor für die Attributwerte

Nachkommastellen

Angabe der Anzahl Dezimalstellen für die Darstellung der Attributwerte

Hinweis

Die Option ist nur für numerische Attribute vom Typ Fließkommazahl verfügbar.

Wenn Sie auf die Schaltfläche Vorschau klicken, werden Ihre Änderungen sofort angezeigt, ohne dass sich das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch) schließt.

Die Halte-Achse wird gemäß den Einstellungen beschriftet.

Strichdarstellung der Halte-Achse einstellen

Sie können den grafischen Fahrplan optisch strukturieren, indem Sie an jedem Halt einen Strich parallel zur Zeit-Achse zeichnen lassen. Sie können einstellen, wie die Striche dargestellt werden.

Klicken Sie in der Symbolleiste des Fahrplans auf das Symbol  Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch)öffnet sich.

Wählen Sie im Navigator die Seite Achsen > Halte-Achse > Strichdarstellung.

Element

Beschreibung

Einheitliche Darstellung

Ist die Option ausgewählt, werden alle Striche einheitlich dargestellt. Wählen Sie im die Option Objekt zeichnen aus und nehmen Sie die weiteren Einstellungen vor.

Klassifizierte Darstellung

Ist die Option ausgewählt, kann die Strichdarstellung nach einem Attribut klassifiziert und dargestellt werden (Objekte anhand von Attributwerten klassifiziert darstellen).

Vorschau

Vorschau der aktuell gewählten Einstellungen

Objekt zeichnen

Ist die Option ausgewählt, wird eine Strichdarstellung gezeichnet.

Zeichnen bis Maßstab

Ist die Option ausgewählt, wird die Strichdarstellung beim Rauszoomen bis zum eingestellten Maßstab gezeichnet (Netzmaßstab ändern).

Ist die Option nicht ausgewählt, wird die Strichdarstellung immer dargestellt, unabhängig vom Netzmaßstab.

Linienstil

Sie können die Darstellung des Linienstils in einem separaten Fenster einstellen, indem Sie in das Feld klicken (siehe „Linienstil einstellen“ auf Seite 751).

Linienfarbe

Auswahl einer Farbe für die Linie in einem separaten Fenster

Linienbreite

Angabe der Linienbreite in mm

Pfeilspitze zeichnen

Ist die Option ausgewählt, können Sie folgende Einstellungen vornehmen.

In der Auswahlliste können Sie ein Symbol auswählen, das als Pfeilspitze dargestellt wird.

Größe

Pfeilspitzengröße in mm

Farbe

Sie können eine Farbe für die Pfeilspitze in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

Zeichen

Wenn Sie für die Pfeilspitze Kreis oder Quadrat auswählen, können Sie über das Kontrollkästchen ein Eingabefeld aktivieren. In das Eingabefeld können Sie ein Zeichen eingeben, das innerhalb des Kreises oder Quadrates dargestellt wird. Sie können eine Farbe für das Zeichen in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

Wenn Sie auf die Schaltfläche Vorschau klicken, werden Ihre Änderungen im Fahrplan sofort angezeigt, ohne dass sich das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch) schließt.
Darstellung der Zeit-Achse einstellen

Klicken Sie in der Symbolleiste des Fahrplans auf das Symbol  Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch)öffnet sich.

Wählen Sie im Navigator die Seite Achsen > Zeit-Achse.

Element

Beschreibung

Einheitliche Darstellung

Ist die Option ausgewählt, werden Linien auf der Zeit-Achse einheitlich dargestellt. Wählen Sie die Option Objekt zeichnen aus und nehmen Sie die weiteren Einstellungen vor.

Klassifizierte Darstellung

Ist die Option ausgewählt, können die Linien auf der Zeit-Achse nach einem Attribut klassifiziert dargestellt werden (Objekte anhand von Attributwerten klassifiziert darstellen).

Objekt zeichnen

Ist die Option ausgewählt, werden die Linien auf der Zeit-Achse gezeichnet.

Ist die Option ausgewählt, wird die Achse beim Rauszoomen bis zum eingestellten Maßstab gezeichnet (Netzmaßstab ändern).

Zeichnen bis Maßstab

Ist die Option ausgewählt, werden die Linien der Achse beim Rauszoomen bis zum eingestellten Maßstab gezeichnet (Netzmaßstab ändern).

Ist die Option nicht ausgewählt, werden die Linien der Achse immer dargestellt, unabhängig vom Netzmaßstab.
Linienstil

Sie können die Darstellung des Linienstils in einem separaten Fenster einstellen, indem Sie in das Feld klicken (siehe „Linienstil einstellen“ auf Seite 751).
Linienfarbe

Auswahl einer Farbe für die Linie in einem separaten Fenster

Pfeilspitze zeichnen

Ist die Option ausgewählt, können Sie folgende Einstellungen vornehmen.

In der Auswahlliste können Sie ein Symbol auswählen, das als Pfeilspitze dargestellt wird.

Größe

Pfeilspitzengröße in mm

Farbe

Sie können eine Farbe und einen Strichtyp für die Pfeilspitze in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

Zeichen

Wenn Sie für die Pfeilspitze Kreis oder Quadrat auswählen, können Sie über das Kontrollkästchen ein Eingabefeld aktivieren. In das Eingabefeld können Sie ein Zeichen eingeben, das innerhalb des Kreises oder Quadrates dargestellt wird. Sie können eine Farbe für das Zeichen in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

Wenn Sie auf die Schaltfläche Vorschau klicken, werden Ihre Änderungen im Fahrplan sofort angezeigt, ohne dass sich das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch) schließt.

Die Zeit-Achse wird gemäß den Einstellungen dargestellt.

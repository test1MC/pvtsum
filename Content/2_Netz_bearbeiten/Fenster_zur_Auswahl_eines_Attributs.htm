Fenster zur Auswahl eines Attributs

Wenn Sie an einer Stelle im Programm nur ein Attribut auswählen können (Beispiel: beim Bearbeiten von Grafikparametern ein Attribut für das klassifizierte Zeichnen festlegen), öffnet sich das Fenster Attribute(<Netzobjekttyp>).

Wenn Sie den Mauszeiger auf ein Attribut setzen, wird eine Quickinfo angezeigt. Diese gibt den Langnamen an, bei indirekten Attributen zusätzlich den zugehörigen Netzobjekttyp und gegebenenfalls die Aggregationsfunktion des Attributs an.

Bereich

Name

Beschreibung

Adresszeile

 

Die Adresszeile ist insbesondere bei indirekten Attributen hilfreich. Sie gibt den vollständigen Pfad des markierten Attributs an. Bei Attributen mit zeitlichem Bezug wird der Attributname durch eine Kurzform des aktuell ausgewählten Analysezeitraums ergänzt.

 

Über dieses Symbol können Sie ein Attribut in der Liste suchen. Geben Sie dazu den Namen des gesuchten Attributs in die Adresszeile ein und klicken Sie auf das Symbol. Das Attribut wird in der Liste markiert.

Tipp

Sie können auch indirekte Attribute suchen. Dazu geben Sie beispielsweise ein: Von-Knoten\Nummer

C, A, H

Wenn Sie auf das Symbol klicken, wechseln Sie zwischen den drei möglichen Sortierungen:

  • C = die klassische Sortierung (Standardeinstellung). Die am häufigsten benötigten Attribute stehen am Anfang der Liste, indirekte Attribute am Ende.
  • A = die alphanumerische Sortierung. Bei dieser Sortierung werden zuerst Spezialeinträge (sofern vorhanden, wie 1.0 oder keine Auswahl) gelistet, dann in alphabetischer Reihenfolge die direkten Attribute und anschließend die indirekten Attribute. Attribute auf tieferen Ebenen werden ebenfalls alphabetisch gelistet.
  • H = die hierarchische Sortierung. Bei dieser Sortierung werden zuerst die Spezialeinträge gelistet. Dann wird je ein Ordner für jede vorhandene Attributkategorie gelistet (gekennzeichnet durch ein c). Darin sind die direkten Attribute der jeweiligen Kategorie in alphabetischer Reihenfolge enthalten. Anschließend folgen die indirekten Attribute in alphabetischer Reihenfolge.

Hinweis

Die Standardeinstellung können Sie ändern (Einstellungen für die Attributauswahlfenster vornehmen).

Alias einfügen

Über das Symbol können Sie einen benutzerdefinierten Alias für einen Attributnamen festlegen, um schneller darauf zuzugreifen. Der Alias wird zusätzlich zum vorgegebenen Kurz- oder Langnamen angezeigt (Alias für Attributnamen festlegen).

Symbole der Auswahlliste

Form

Punkt

Kennzeichnet vorliegende Attributwerte, d h. Eingabeattribute und berechnete Ausgabeattribute

Ring

Kennzeichnet noch nicht berechnete Ausgabeattribute

c (=category; nur in der hierarchischen Sortierungsansicht)

Kennzeichnet Attributkategorien, die die direkten Attribute der jeweiligen Kategorie in alphabetischer Reihenfolge enthalten.

Symbole der Auswahlliste

Farbe

Grün

Kennzeichnet Attribute ohne Subattribute

Kennzeichnet Attribute, für die Sie ein Subattribut auswählen müssen

Die Liste der Subattribute öffnen Sie mit einem Klick auf das Pluszeichen.

Grau

Kennzeichnet Attribute, für die das zugehörige Zusatzmodul nicht aktiviert ist oder Matrizen, die nicht vorliegen

Blau

Kennzeichnet Aliase, also vom Benutzer angelegte Attributnamen

Schwarz

Kennzeichnet benutzerdefinierte Attribute

Vorbelegung des Analysezeitraums

 

In der Auswahlliste Vorbelegung des Analysezeitraums können Sie vor der Auswahl eines Attributs einen Analysezeitraum festlegen. Dieser ist relevant für Attribute mit zeitlichem Bezug, z. B. Belastung. Wenn Sie dann ein Attribut mit zeitlichem Bezug auswählen, wird es für den hier festgelegten Analysezeitraum ausgewählt (Vorbelegung des Analysezeitraums).

Hinweis

Die Standardeinstellung können Sie ändern (Einstellungen für die Attributauswahlfenster vornehmen).

Schaltfläche

Neu

Wenn Sie auf die Schaltfläche Neu klicken, öffnet sich das Fenster Benutzerdefiniertes Attribut <(Netzobjekttyp)> einfügen. Hier können Sie ein benutzerdefiniertes Attribut einfügen (Benutzerdefinierte Attribute verwalten).

Haltestellenbereichs-Übergangsgehzeiten setzen

Die aktuellen Matrixwerte im Register GehzeitenHstBer können Sie für alle oder alle aktiven Haltestellen und Haltestellenbereiche überschreiben (Haltestellen: Register Gehzeiten HstBer).

Setzen Sie bei Bedarf die Haltestellen aktiv, deren Gehzeiten Sie aktualisieren möchten, über

  • die räumliche Auswahl (Netzobjekte über die räumliche Auswahl aktiv und passiv setzen) und/oder
  • Filterkriterien (Netzobjekte über Filter aktiv und passiv setzen).

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Haltestellen.

Das Kontextmenü Haltestellen öffnet sich.

Wählen Sie den Eintrag HstBer-Übergangsgehzeiten setzen.

Das Fenster Mehrfach-Bearbeiten Haltestellen öffnet sich.

Wenn der Netzobjekttyp Haltestellen ausgewählt ist, können Sie das Fenster Mehrfach-Bearbeiten Haltestellen auch über das Menü Bearbeiten > HstBer-Übergangsgehzeiten setzen aufrufen.

Stellen Sie sicher, dass das Register Sonderfunktionen aktiv und in der Auswahlliste der Eintrag HstBer-Übergangsgehzeiten setzen ausgewählt sind.

Element

Beschreibung

Nur aktive

Ist die Option ausgewählt, werden nur aktive Haltestellen berücksichtigt.

Ist die Option nicht ausgewählt, werden alle Haltestellen berücksichtigt.

Fuß-Verkehrssystem

Über die Auswahlliste können Sie das Verkehrssystem wählen, für das die Gehzeiten zwischen Haltestellenbereichen bearbeitet werden sollen.

Übergang innerhalb desselben Haltestellenbereichs

Für Übergänge innerhalb eines Haltestellenbereichs haben Sie mehrere Optionen.

setzen auf

Ist die Option gewählt, setzen Sie die Übergangsgehzeiten auf den eingegebenen Wert.

sperren

Ist die Option gewählt, werden die Übergänge gesperrt.

beibehalten

Ist die Option gewählt, bleiben die Übergangsgehzeiten unverändert.

Übergang zwischen verschiedenen Haltestellenbereichen

Für Übergänge zwischen verschiedenen Haltestellenbereichen haben Sie mehrere Optionen.

setzen auf

Ist die Option gewählt, setzen Sie die Übergangsgehzeiten auf den eingegebenen Wert.

sperren

Ist die Option gewählt, werden die Übergänge gesperrt.

beibehalten

Ist die Option gewählt, bleiben die Übergangsgehzeiten unverändert.

Wert nur für aktive Haltestellenbereiche ändern

Ist die Option ausgewählt, werden nur aktive Haltestellenbereiche berücksichtigt.

Klicken Sie auf die Schaltfläche Ausführen.

An den ausgewählten Haltestellen wird die Matrix der Haltestellenbereichs-Übergangsgehzeiten für das angegebene Verkehrssystem aktualisiert.

Klicken Sie auf die Schaltfläche Schließen.

Eine Übersicht über alle Übergangsgehzeiten zwischen Haltestellenbereichen erhalten Sie in der Liste Umsteiger und Haltestellenbereichs-Gehzeiten in Haltestelle.

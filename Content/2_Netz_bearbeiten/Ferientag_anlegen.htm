Ferientag anlegen

Ferientage können Sie nur einfügen, wenn Sie das Zusatzmodul Kalender / Verkehrstage aktiviert haben (Zusatzmodule ein- und ausschalten).

Sie können Ferientage nur anlegen, wenn Sie einen Jahreskalender verwenden.

Sie können Ferientage auch außerhalb der Kalenderperiode anlegen.

Wählen Sie im Menü Netz den Eintrag ÖV-Verkehrstage.

Das Fenster Verkehrstage öffnet sich.

Wählen Sie das Register Ferientage.

Klicken Sie auf die Schaltfläche  Einfügen.

Das Fenster Ferien-/Feiertag einfügen öffnet sich.

Geben Sie die gewünschten Daten ein (Attribute und Optionen von Ferientagen).

Der Ferientag wird angelegt.

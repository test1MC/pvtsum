Werte für benutzerdefinierte Fahrplanfahrtattribute ändern

Auch für Fahrplanfahrten können Sie benutzerdefinierte Attribute anlegen (Benutzerdefinierte Attribute verwalten). Wenn benutzerdefinierte Fahrplanfahrtattribute angelegt wurden, können Sie den Attributen beim Einfügen neuer Fahrplanfahrten Werte zuweisen und diese nachträglich ändern.

Markieren Sie im Fahrplan die Fahrplanfahrten, deren Attributwerte Sie ändern möchten (Fahrplanfahrten markieren).

Klicken Sie mit der rechten Maustaste auf eine der markierten Fahrplanfahrten.

Klicken Sie auf den Eintrag Fahrplanfahrt(en) bearbeiten.

Das Fenster Fahrplanfahrt(en) bearbeiten öffnet sich.

Wählen Sie das Register Benutzerdefinierte Attr.

Wählen Sie bei Bedarf die Attribute aus, deren Werte Sie ändern möchten, indem Sie auf die Schaltfläche Attributauswahl klicken (Fenster zur Auswahl eines Attributs).

Geben Sie die gewünschten Werte für die benutzerdefinierten Attribute direkt in die Liste ein.

Die neuen Werte werden übernommen.

Sie können Werte für benutzerdefinierte Fahrplanfahrtattribute auch direkt beim Einfügen einer Fahrplanfahrt eingeben (Fahrplanfahrt einfügen).

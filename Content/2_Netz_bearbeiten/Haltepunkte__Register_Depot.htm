Haltepunkte: Register Depot

Ein Depot ist ein Haltepunkt, an dem ein Aufenthalt für mindestens eine Fahrzeugkombination zulässig ist. In diesem Register sind alle Fahrzeugkombinationen enthalten, die Sie als Depot zulassen können Siehe "Aufenthalte in Depots und an Haltepunkten".

Element

Beschreibung

Zulässig

Ist die Option ausgewählt, ist die jeweilige Fahrzeugkombination als Depot zulässig.

Hinweis

Die erste Zeile enthält die Standardwerte.

Code

Kurzbezeichnung der Fahrzeugkombination

Kapazität

Angabe einer Kapazität pro zulässiger Fahrzeugkombination

Hinweis

Die Kapazität gibt die Anzahl Fahrzeugkombinationen an, die sich gleichzeitig am Depot aufhalten dürfen. Die Angabe unbegrenzt ist möglich, indem der Wert 0 eingegeben wird. Ein solches zulässiges Depot kann dann immer aufgesucht werden.

Min. Aufenthalt

Angabe einer minimalen Aufenthaltszeit im Depot je Fahrzeugkombination

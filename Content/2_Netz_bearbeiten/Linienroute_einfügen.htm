Linienroute einfügen

Beachten Sie die Regeln zum Ändern eines Verlaufs (Grundregeln zum Ändern eines Routenverlaufs).

Klicken Sie im Fenster Netz auf das Symbol Modus Einfügen.

Klicken Sie auf die Schaltfläche Linien.

Das Fenster Einfügen öffnet sich.

Wählen Sie den Eintrag Linienroute aus.

Klicken Sie in die Netzdarstellung.

Das Fenster Linienroute einfügen öffnet sich.

Element

Beschreibung

Linie

In der Auswahlliste können Sie eine Linie auswählen, der die Linienroute zugeordnet wird.

nur aktive Linien

Ist die Option ausgewählt, stehen nur aktive Linien zur Auswahl.

Name

Langbezeichnung der Linienroute angeben

Hinweis

Die Auswahlliste enthält die Namen aller Linienrouten im Netz. Der Name der neuen Linienroute muss davon abweichen.

Richtung

In der Auswahlliste können Sie eine Richtung auswählen (ÖV-Richtungen verwalten).

Verkehrssystem

Anzeige des Verkehrssystems der ausgewählten Linie

Standardbetreiber

Anzeige des Standardbetreibers der ausgewählten Linie

Oberlinie

Anzeige der Oberlinie, falls die ausgewählte Linie einer Oberlinie zugeordnet ist (Oberlinien verwalten)

Das Fenster Linienroute digitalisieren öffnet sich.

Klicken Sie auf den Haltepunkt, an dem die Linienroute starten soll und halten Sie die Maustaste gedrückt.

Eine Linienroute muss an einem Haltepunkt starten und enden.

Ziehen Sie die Maus an den nächsten Haltepunkt oder Knoten, über den die Linienroute verlaufen soll und lassen Sie die Maustaste los.

Die Auswahl von Knoten ist vorzuziehen, da Sie so vermeiden, versehentlich einen Haltepunkt in Gegenrichtung auszuwählen.

Klicken Sie bei Bedarf im Fenster Linienroute digitalisieren auf die Schaltfläche Autozoom, wenn Sie den Ausschnitt mit dem Verlauf vergrößern möchten.

Fügen Sie bei Bedarf auf die gleiche Weise weitere Teilverläufe ein.

Klicken Sie bei Bedarf im Fenster Linienroute digitalisieren auf die Schaltfläche Undo, wenn Sie das letzte Teilstück aus dem Linienroutenverlauf entfernen möchten.

Wird ein Haltepunkt oder Knoten mehrfach überfahren, öffnet sich das Fenster Linienroute: digitalisieren. Wählen Sie in dem Fenster die Überfahrt aus, die Sie übernehmen möchten.

Ziehen Sie die Maustaste auf den gewünschten End-Haltepunkt der Linienroute und lassen Sie die Maustaste los.

Sie können die Maus auch direkt vom ersten Haltepunkt zum gewünschten End-Haltepunkt ziehen. Das Programm schlägt dann den besten Weg vor entsprechend dem eingestellten Suchkriterium (siehe „Linienrouten: Parameter für die Wegesuche einstellen“ auf Seite 490).

Klicken Sie im Fenster Linienroute digitalisieren auf OK.

Die Linienroute wird eingefügt.

Die Gegenrichtung einer Linienroute können Sie über das Kontextmenü einfügen (Gegenrichtung einer Linienroute einfügen).

Wenn im Netz Systemrouten angelegt sind und Sie den ersten Haltepunkt festgelegt haben, können Sie über die Schaltfläche tabellarisch bearbeiten den Verlauf einer Linienroute anhand von Systemroutenverläufen einfügen (Verlauf einer Linienroute tabellarisch bearbeiten).

Sie können den Verlauf einer Linienroute nachträglich ändern (Linienroutenverlauf digitalisieren).
Linienrouten: Parameter für die Wegesuche einstellen

Im Menü Bearbeiten > Benutzereinstellungen können Sie vorab Standardwerte für die Wegesuche einstellen (Standardwerte für Linien und Linienrouten einstellen).

Stellen Sie sicher, dass das Fenster Linienroute digitalisieren geöffnet ist (Verlauf einer Linienroute digitalisieren).

Klicken Sie im Fenster Linienroute digitalisieren auf die Schaltfläche Parameter.

Das Fenster Parameter für die Wegesuche der Linienroute öffnet sich.

Element

Beschreibung

auch über gesperrte Strecken routen

Ist die Option ausgewählt, werden für das Verkehrssystem gesperrte Strecken berücksichtigt, als ob sie für das Verkehrssystem geöffnet wären.

auch über gesperrte Abbieger routen

Ist die Option ausgewählt, werden für das Verkehrssystem gesperrte Abbieger berücksichtigt, als ob sie für das Verkehrssystem geöffnet wären.

Haltepunkte des neuen Verlaufs bedienen

Ist die Option ausgewählt, werden auch die Haltepunkte des neuen Verlaufs bedient.

Ist die Option nicht ausgewählt, bedient die neue Linienroute nur die Haltepunkte, die bereits vor dem Umrouten bedient wurden.

nur aktive Haltepunkte bedienen

Ist die Option ausgewählt, werden passive Haltepunkte nicht berücksichtigt.

Hinweis

Die Option ist nur verfügbar, wenn Sie die Option Haltepunkte des neuen Verlaufs bedienen ausgewählt haben.

Fahrzeit im umgerouteten Bereich aus Streckennetz über-nehmen

Hinweis

Diese Option ist nur beim Ändern eines Linienroutenverlaufs verfügbar. Beim Einfügen eines neuen Verlaufs wird immer die verkehrssystemspezifische Streckenfahrzeit verwendet.

Ist die Option ausgewählt, wird die verkehrssystemspezifische Streckenfahrzeit für das Verkehrssystem der Linie und für alle überfahrenen Strecken verwendet.

Ist die Option nicht ausgewählt, werden die alten Fahrzeiten des Fahrzeugprofils beibehalten. Diese werden längenproportional im neuen Verlauf verteilt.

Fahrplanfahrten automatisch verlängern

Ist die Option ausgewählt, werden auch die Fahrplanfahrten der Linienroute verlängert, wenn die Linienroute verlängert wird.

Ist die Option nicht ausgewählt, ändert sich die Ausdehnung der Fahrplanfahrten nicht.

Suchkriterium

In der Auswahlliste können Sie ein Kriterium für die Wegesuche auswählen. Gesucht wird der jeweils beste Weg zwischen zwei markierten (fixierten) Haltepunkten.

Streckenlänge-Luft

Luftlinien-Streckenlänge vom Start- zum End-Haltepunkt

Streckenfahrzeit

Wird aus der Verkehrssystem-Geschwindigkeit der Strecken berechnet

Zeit aus Streckentypgeschwindigkeit

Wird aus der für den betreffenden Streckentyp definierten Verkehrssystem-Geschwindigkeit berechnet

Streckenlänge

Berechnete Streckenlänge, wenn nicht vom Benutzer manuell angepasst

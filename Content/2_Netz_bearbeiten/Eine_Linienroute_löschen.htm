Eine Linienroute löschen

Öffnen Sie das Fenster Linienrouten (Fenster Linienrouten öffnen).

Wählen Sie die Linienroute aus, die Sie löschen möchten.

Drücken Sie die Taste Entf.

Das Fenster Linienrouten löschen öffnet sich.

Die Linienroute wird gelöscht.

Sie können die ausgewählte Linienroute auch über das Kontextmenü > Eintrag Löschen löschen.

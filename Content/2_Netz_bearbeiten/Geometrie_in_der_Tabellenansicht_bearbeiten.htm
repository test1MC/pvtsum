Geometrie in der Tabellenansicht bearbeiten

Sie können die Breite der Zeilen und Spalten ändern, indem Sie sie mit der Maus in die gewünschte Richtung ziehen.

Sie können die Attributauswahl der Tabelle ändern (Attributauswahl im Knoteneditor anpassen).

Das Kontextmenü in der Tabelle entspricht weitestgehend dem Kontextmenü der Listen (Spaltenbreite an Inhalt anpassen).

Armattribute anzeigen

In der Tabellenansicht beschreibt jede Spalte einen Arm und jede Zeile ein Attribut dieses Objekts.

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Geometrie ausgewählt ist.

Wählen Sie in der schematischen Ansicht die Option Arme.

Die Tabellenansicht zeigt die Arme des ausgewählten Knotens an.

Nehmen Sie die gewünschten Änderungen vor.

Signalgruppen Fahrstreifenabbiegern oder Furten zuordnen

Sie müssen Signalgruppen entweder Fahrstreifenabbiegern oder Furten zuordnen, um den Signalgruppen einen räumlichen Bezug zu geben.

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Geometrie ausgewählt ist.

Wählen Sie in der schematischen Ansicht die Option Fahrstreifenabbieger oder Furten.

Klicken Sie in der Tabellenansicht unter Signalgruppen in das gewünschte Feld.

Das Fenster Signalgruppen öffnet sich.

Wählen Sie die gewünschte Signalgruppe aus.

Die Signalgruppe wird dem Fahrstreifenabbieger oder der Furt zugewiesen.

Sie können die Zuordnungen einer Signalgruppe überprüfen, indem Sie in der Ansicht Geometrie die Option Signalgruppen wählen und unter Fahrstreifenabbieger und Furten für die gewünschte Signalgruppe das jeweilige Auswahlfenster öffnen (Signalgruppenattribute ändern).
Signalgruppen Fahrstreifen oder Armen zuordnen

Sie können Signalgruppen auch Fahrstreifenabbiegern zuordnen, indem Sie sie in einem Schritt allen Fahrstreifenabbiegern eines Arms oder Fahrstreifens zuordnen.

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Geometrie ausgewählt ist.

Wählen Sie in der schematischen Ansicht die Option Fahrstreifen oder Arme.

Klicken Sie in der Tabellenansicht unter Signalgruppen in das gewünschte Feld.

Das Fenster Signalgruppen öffnet sich.

Wählen Sie die gewünschte Signalgruppe aus.

Die Signalgruppe wird allen Fahrstreifenabbiegern des ausgewählten Fahrstreifens oder Arms zugeordnet.

Armschablone einfügen

Sie können Armschablonen für den ausgewählten Knoten definieren. Bei Anwendung einer Armschablone wird eine vordefinierte Menge von Fahrstreifen, Fahrstreifenabbiegern und Furten an einem Arm erzeugt.

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Geometrie ausgewählt ist.

Wählen Sie in der schematischen Ansicht die Option Arme.

Klicken Sie mit der rechten Maustaste in der Tabellenansicht in die Spalte des Arms, mit dessen Hilfe Sie eine Armschablone definieren möchten.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Armschablone definieren.

Das Fenster Schablone einfügen öffnet sich.

Element

Beschreibung

Nummer

Nummer der Armschablone

Name

Langbezeichner der Armschablone

Kommentar

Angabe des Arms, aus dem die Schablone erzeugt wurde

Die Armschablone wird eingefügt.

Sie können die eingefügte Armschablone anschließend einem Arm zuordnen, wenn Sie in der schematischen Ansicht die Option Arme auswählen und die Armschablone in der Tabellenansicht dem gewünschten Arm über Armschablone anwenden zuordnen.
Geometrieschablone einfügen

Geometrieschablonen werden im Gegensatz zu Armschablonen auf alle Arme des ausgewählten Knotens angewendet (Grundlagen: Geometrien).

Stellen Sie sicher, dass im Fenster Ansichten die Ansicht Geometrie ausgewählt ist.

Wählen Sie in der schematischen Ansicht die Option Arme.

Klicken Sie mit der rechten Maustaste in der Tabellenansicht in die Spalte des Arms, mit dessen Hilfe Sie eine Geometrieschablone definieren möchten.

Der ausgewählte Arm wird der Bezugsarm der Geometrieschablone.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Geometrieschablone definieren.

Das Fenster Schablone einfügen öffnet sich.

Element

Beschreibung

Name

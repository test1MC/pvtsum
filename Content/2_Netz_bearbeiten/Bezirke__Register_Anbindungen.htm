Bezirke: Register Anbindungen

Bereich IV

Element

Beschreibung

Quell-Anbindungen

nach Anteilen

Über die Option legen Sie fest, dass der IV-Quellverkehr prozentual auf die IV-Quellanbindungen aufgeteilt wird (Anbindungen: Register Basis).

absolut

Über die Option legen Sie fest, dass der IV-Quellverkehr absolut auf die IV-Quellanbindungen aufgeteilt wird.

Ziel-Anbindungen

nach Anteilen

Über die Option legen Sie fest, dass der IV-Zielverkehr prozentual auf die IV-Zielanbindungen aufgeteilt wird (Anbindungen: Register Basis).

absolut

Über die Option legen Sie fest, dass der IV-Zielverkehr absolut auf die IV-Zielanbindungen aufgeteilt wird.

Methode für Anbindungsanteile

Über die Auswahlliste können Sie für den Bezirk eine vom Standard abweichende Methode für die Berechnung der Anteile von prozentualen Anbindungen wählen.

Tipps

Die Standardmethode können Sie über das Menü Rechnen > Allgemeine Verfahrenseinstellungen > Eintrag IV-Einstellungen > Auswahlliste Anbindungsgewichte anwenden auf ändern (Prozentuale Aufteilung der Nachfrage).

Sie können die Methode auch über das Attribut Methode für Anbindungsanteile für mehrere Bezirke ändern (Attributwerte von Netzobjekten bearbeiten).

Für IV-Anbindungen mit Aufteilung nach Anteilen werden Widerstände bei der Umlegung gemäß der eingestellten Widerstandsfunktion berechnet (Widerstandsfunktionen definieren und verwenden).

Bereich ÖV

Element

Beschreibung

Anbindungen

nach Anteilen

Über die Option legen Sie fest, dass der ÖV-Quell- und -Zielverkehr prozentual auf die ÖV-Quell- und -Zielanbindungen aufgeteilt wird (Anbindungen: Register Basis).

Hinweis

Für die Aufteilung nach Anteilen muss im Menü Rechnen > Verfahrensablauf > das Verfahren ÖV-Umlegung aus der Kategorie Umlegungen > mit der Einstellung Fahrplanfein oder Taktfein und dem Parameter Anbindungsanteile verwenden ausgewählt sein (Fahrplanfeine Umlegung: Register Basis und Taktfeine Umlegung: Register Basis).

absolut

Über die Option können Sie festlegen, dass der ÖV-Quell- und -Zielverkehr absolut auf die ÖV-Quell- und -Zielanbindungen aufgeteilt wird.

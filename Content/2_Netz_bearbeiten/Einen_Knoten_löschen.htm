Einen Knoten löschen

Liegt an dem zu löschenden Knoten ein Haltepunkt, wird dieser ebenfalls gelöscht.

Wird der Haltepunkt von einer ÖV-Linie bedient, werden diese Linie sowie ihre Linienrouten, Fahrzeitprofile und Fahrplanfahrten ebenfalls gelöscht. Eine Ausnahme bilden zweiarmige Knoten (Sonderfall: Zweiarmigen Knoten ohne Haltepunkt löschen und Strecken verbinden).

Klicken Sie im Fenster Netz auf das Symbol  Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Knoten.

Klicken Sie im Netz auf den Knoten, den Sie löschen möchten.

Drücken Sie die Taste Entf.

Sie können den markierten Knoten auch über das Kontextmenü > Eintrag Löschen löschen.

Sie können zugehörige Haltestellenbereiche ebenfalls entfernen, indem Sie die Option Haltestelle(nbereich) mit löschen auswählen. Die zugehörige Haltestelle wird nur gelöscht, wenn sie über keine weiteren Haltestellenbereiche verfügt.

Eine Sicherheitsabfrage öffnet sich.

Der Knoten wird gelöscht.

Sonderfall: Zweiarmigen Knoten ohne Haltepunkt löschen und Strecken verbinden

Wenn Sie einen zweiarmigen Knoten ohne Haltepunkt löschen, können Sie angeben, dass die beiden Strecken des Knotens verbunden werden.

Sie können die Strecke nicht verbinden, wenn zwischen den Nachbarknoten des zu löschenden Knotens bereits eine Strecke vorhanden ist. In diesem Fall würde die verbundene Strecke mit der Strecke der Nachbarknoten zusammenfallen, in dürfen jedoch keine identischen Strecken entstehen.

Klicken Sie im Fenster Netz auf das Symbol  Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Knoten.

Klicken Sie im Netz auf den Knoten, den Sie löschen möchten.

Der Knoten ist markiert.

Drücken Sie die Taste Entf.

Eine Abfrage öffnet sich.

Sie können den markierten Knoten auch über das Kontextmenü > Eintrag Löschen löschen.

Klicken Sie auf die Schaltfläche Verbinden.

Das Fenster Strecken verbinden öffnet sich.

Geben Sie in das Eingabefeld bei Bedarf eine neue Streckennummer ein.

Wählen Sie die Strecke, von der alle weiteren Attribute übernommen werden sollen.

Der Knoten wird gelöscht und die zugehörigen Strecken werden verbunden.

Für die neue Strecke übernimmt die Verkehrssysteme, die auf den beiden verbundenen Strecken zulässig waren.
Sonderfall: Zweiarmigen Knoten mit Haltepunkt löschen

Wenn Sie zweiarmige Knoten mit Haltepunkt löschen, öffnet sich ein Fenster, in dem Sie angeben können, ob und wie die beiden Strecken des Knotens verbunden werden.

Klicken Sie im Fenster Netz auf das Symbol  Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Knoten.

Klicken Sie im Netz auf den Knoten, den Sie löschen möchten.

Der Knoten ist markiert.

Drücken Sie die Taste Entf.

Wählen Sie die gewünschten Optionen.

Element

Beschreibung

Strecken löschen

Ist die Option ausgewählt, werden die Strecken gelöscht.

Strecken verbinden, Haltepunkt löschen

Ist die Option ausgewählt, werden die Strecken verbunden und der Haltepunkt gelöscht.

Strecken verbinden, Haltepunkt auf Strecke setzen

Ist die Option ausgewählt, werden die Strecken verbunden und der Haltepunkt auf die Strecke gesetzt.

Haltestelle(nbereich) mit löschen

Ist die Option ausgewählt, werden zugehörige Haltestellenbereiche ebenfalls entfernt.

Hinweis

Die zugehörige Haltestelle wird nur gelöscht, wenn sie über keine weiteren Haltestellenbereiche verfügt.

Eine Sicherheitsabfrage öffnet sich.

Wenn Sie die Strecken verbinden, öffnet sich das Fenster Strecken verbinden.

Geben Sie in das Eingabefeld bei Bedarf eine neue Streckennummer ein.

Wählen Sie die Strecke, von der alle weiteren Attribute übernommen werden sollen.

Der Knoten wird gelöscht. Die zugehörigen Strecken werden abhängig von der ausgewählten Option verbunden oder gelöscht.

Screenlineverlauf zurücksetzen

Sie können einen soeben geänderten Verlauf einer Screenline zurücksetzen.

Stellen Sie sicher, dass das Fenster Screenline digitalisieren geöffnet ist.

Klicken Sie auf die Schaltfläche Initialisieren.

Der Screenlineverlauf wird auf den bisherigen Verlauf zurückgesetzt.

Alternativ können Sie den Verlauf mit F2 zurücksetzen.

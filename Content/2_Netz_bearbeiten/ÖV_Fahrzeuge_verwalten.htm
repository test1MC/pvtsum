ÖV-Fahrzeuge verwalten

Sie können Fahrzeugeinheiten (Fahrzeugeinheit anlegen) und Fahrzeugkombinationen (Fahrzeugkombination anlegen) anlegen. Fahrzeugeinheiten definieren Sie für die ÖV-Verkehrssysteme des Netzes. Eine Fahrzeugkombination besteht aus einer oder mehreren Fahrzeugeinheiten. So können Sie beispielsweise eine Fahrzeugkombination „Intercity“ aus mehreren Fahrzeugeinheiten „Wagen“ zusammensetzen. Für jede können Sie die Anzahl Sitzplätze und Gesamtplätze angeben Siehe "ÖV-Fahrzeuge: Fahrzeugeinheiten und Fahrzeugkombinationen".

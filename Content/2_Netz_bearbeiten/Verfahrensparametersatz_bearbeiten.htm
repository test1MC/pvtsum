Verfahrensparametersatz bearbeiten

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Wählen Sie im Fenster Projekt bearbeiten das Register Verfahrensparametersätze.

Wählen Sie in der Liste den gewünschten Verfahrensparametersatz aus.

Klicken Sie auf das Symbol   Verfahrensparametersatz bearbeiten.

Ein Szenario, das den Verfahrensparametersatz enthält, wird geöffnet. Gleichzeitig öffnet sich das Fenster Verfahrensparametersatz bearbeiten.

Falls es kein Szenario gibt, das den Verfahrensparametersatz verwendet, wird die Basisversion verwendet. Dabei kann es zu Fehlermeldungen kommen, wenn Bezüge nicht vorhanden sind.

Alternativ können Sie die Funktion über das gleiche Symbol im Register Szenarien aufrufen.

Nehmen Sie in der Versionsdatei die gewünschten Änderungen an den Verfahrensparametern vor (Menü Rechnen > Verfahrensablauf und Rechnen > Allgemeine Verfahrenseinstellungen).

Falls Sie im Verfahrensablauf Dateien öffnen oder speichern, sollten Sie die vorgeschlagenen Pfade verwenden. Die Dateien werden automatisch entweder im gemeinsamen Verzeichnis oder pro Szenario verwaltet, je nach Ihren Einstellungen (Grundeinstellungen eines Projekts bearbeiten).

Falls während des Verfahrensablaufs Daten eingelesen werden sollen, müssen Sie darauf achten, dass diese eingelesenen Daten nicht von anderen Modifikationen verändert werden. Ansonsten besteht die Gefahr, dass sich Szenarien gegenseitig die Daten überschreiben.

Falls Sie als Verfahren ein Skript verwenden, wird das Skript automatisch in dem Editor geöffnet, den Sie im Menü Bearbeiten > Benutzereinstellungen > Eintrag Arbeitsumgebung > Szenariomanagement eingestellt haben.

Klicken Sie im Fenster Verfahrensparametersatz bearbeiten auf die Schaltfläche Fertigstellen.

Die Änderungen werden gespeichert. Das Fenster Projekt bearbeiten öffnet sich wieder. Sie können jetzt weitere Änderungen am Projekt vornehmen.

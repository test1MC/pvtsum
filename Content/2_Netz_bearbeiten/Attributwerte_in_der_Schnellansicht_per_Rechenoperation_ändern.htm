Attributwerte in der Schnellansicht per Rechenoperation ändern

Stellen Sie sicher, dass im Netz die Netzobjekte ausgewählt sind, deren Attributwerte Sie ändern möchten.

Markieren Sie in der Schnellansicht ein oder mehrere numerische Felder (Fenster Schnellansicht).

Geben Sie in ein markiertes Feld eine der folgenden Rechenoperationen ein:

Operation

Syntax

Addition

=+2

Subtraktion

=-2

Multiplikation

=*2

Division

=/2

Potenzieren

=^2

Minimum bilden

=min2

Maximum bilden

=max2

Exponentialfunktion anwenden

=exp

Logarithmieren

=log

Kehrwert bilden

=1/

Die Operation wird für alle markierten Felder mit numerischem Wert ausgeführt. Das Ergebnis wird als Attributwert eingefügt.

Falls Sie ein alphanumerisches Feld ausgewählt haben, wird der Text der Rechenoperation in das Feld eingetragen.

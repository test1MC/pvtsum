Attribute und Optionen von POIs

Der obere Teil des Fensters POI einfügen oder POI bearbeiten enthält die allgemeinen Attribute des POIs. Im unteren Teil können Sie in verschiedenen Registern spezielle Attribute bearbeiten.

POIs: Allgemeine Attribute

Element

Beschreibung

Nummer

Die POIs werden je Kategorie automatisch fortlaufend nummeriert.

Kategorie

Anzeige der Kategorie, der der POI zugeordnet ist

Code

Kurzbezeichnung des POIs

Langbezeichnung des POIs

POIs: Register Basis

Element

Beschreibung

Kommentar

Freie Texteingabe

Position

Sie können die Position des POIs im Netz ändern (POI verschieben).

Darstellung

Sie können eine Bilddatei zur Darstellung eines POIs verwenden.

Eigenes Bild verwenden

Ist die Option ausgewählt, können Sie über die Schaltfläche eine Bilddatei (*.bmp, *.jpg etc.) zur Darstellung dieses POIs einlesen. Danach müssen Sie noch die Bildanzeige in den Grafikparametern aktivieren (Points of Interest darstellen).

Ist die Option nicht ausgewählt, werden alle POIs einer POI-Kategorie mit dem gleichen Bild dargestellt (Anzeigeeinstellungen für eine POI-Kategorie ändern).

Eigene Höhe und Winkel verwenden

Ist die Option ausgewählt, können Sie eine Standardhöhe und einen Standardwinkel für die Darstellung des POIs einstellen.

Hinweis

POIs werden nur angezeigt, wenn die entsprechenden Grafikparameter festgelegt sind (Points of Interest darstellen).

Vor dem Einfügen von Bildern empfehlen wir Ihnen, alle Bilddateien in einem Verzeichnis zu speichern und dieses in der Projektverzeichnisdatei anzugeben (Projektverzeichnisdatei öffnen und speichern). Eine eingefügte Bilddatei wird nicht mit der Versionsdatei gespeichert. Wenn beim nächsten Öffnen der Versionsdatei den Pfad zur Bilddatei nicht findet, erhalten Sie eine entsprechende Meldung und die Grafik wird nicht angezeigt.

POIs: Register Zuordnungen

Sie können POIs verschiedenen Netzobjekten zuordnen (Einem POI Netzobjekte zuordnen). Beispielsweise können Sie einem POI, der einen Parkplatz in einem Innenstadtbereich darstellt, Strecken zuordnen, um zu veranschaulichen, auf welche Strecken die Ausfahrten führen.

Sie können die Verbindungen zu zugeordneten Netzobjekten grafisch darstellen (Verbindungen an POIs anzeigen).

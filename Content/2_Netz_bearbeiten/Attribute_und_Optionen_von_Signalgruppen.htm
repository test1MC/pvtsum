Attribute und Optionen von Signalgruppen

Element

Beschreibung

Nummer

Eindeutige Nummer der Signalgruppe

Signalgruppen werden automatisch fortlaufend nummeriert. Die vorgegebene Nummer kann mit einer Nummer überschrieben werden, die im Netz noch nicht vergeben ist.

Name

Langbezeichnung der Signalgruppe

Grünzeitanfang

Wenn Sie eine Signalgruppe für eine signalgruppenbasierte LSA einfügen, geben Sie hier den Grünzeitanfang in Sekunden ein.

Wenn Sie eine Signalgruppe für eine phasenbasierte LSA einfügen, ist dieses Feld ausgegraut. Der Grünzeitanfang wird aus der gewählten Phase übernommen.

Grünzeitende

Wenn Sie eine Signalgruppe für eine signalgruppenbasierte LSA einfügen, geben Sie hier das Grünzeitende in Sekunden ein.

Wenn Sie eine Signalgruppe für eine phasenbasierte LSA einfügen, ist dieses Feld ausgegraut. Das Grünzeitende wird aus der gewählten Phase übernommen.

Mindestgrünzeit

Angabe der Mindestgrünzeit in Sekunden

Hinweis

Der Wert wird bei der Grünzeitoptimierung wirksam (LSA-Umlauf- und Grünzeiten für eine einzelne LSA optimieren).

Gelb

Angabe der Gelbzeit in Sekunden

Allred

Angabe der Allred-Zeit in Sekunden

Hinweis

Zeit nach der Gelbzeit einer Signalgruppe, in der keine andere Signalgruppe Grün erhalten darf.

Phasen

Wenn Sie eine Signalgruppe für eine phasenbasierte LSA einfügen, müssen Sie die Signalgruppe einer oder mehreren Phasen zuordnen.

Hinweis

Die Signalgruppe muss mindestens einer Phase zugeordnet werden.

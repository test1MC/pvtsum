POI-Kategorie anlegen

Wählen Sie im Menü Netz den Eintrag POI-Kategorien.

Das Fenster POI-Kategorien bearbeiten öffnet sich.

Klicken Sie auf die Schaltfläche  Einfügen.

Das Fenster POI-Kategorie einfügen öffnet sich.

Element

Beschreibung

Nr

Kategorien werden automatisch fortlaufend nummeriert

Code

Kurzbezeichnung der Kategorie

Name

Langbezeichnung der Kategorie

Kommentar

Kommentar für die Kategorie (freie Texteingabe)

Übergeordnete Kategorie

Über die Auswahlliste können Sie eine übergeordnete Kategorie wählen (sofern vorhanden).

Ist keine übergeordnete Kategorie ausgewählt, wird die eingefügte Kategorie die übergeordnete Kategorie.

Die POI-Kategorie wird angelegt.

Es werden ausschließlich POI-Kategorien mit aktiven POIs in die Netzdatei geschrieben.

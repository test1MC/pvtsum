Benutzerdefinierte Attribute für POI-Kategorien verwenden

Wenn Sie einer POI-Kategorie ein benutzerdefiniertes Attribut zuweisen (Benutzerdefiniertes Attribut einfügen), wird es automatisch von allen Unterkategorien dieser übergeordneten Kategorie übernommen. Diese Funktion ist nützlich, wenn Sie ein gemeinsames Merkmal aller POIs mehrerer Kategorien und Unterkategorien verfolgen oder auswerten möchten.

Beispiel: Sie legen die POI-Kategorie „Ausbildungsstätten“ mit dem benutzerdefinierten Attribut „Schülerzahlen“ an. Dann können Sie für alle Ausbildungsstätten, für die Sie Unterkategorien und POIs in dieser Kategorie anlegen, die Schülerzahlen verfolgen und bei Bedarf für Berechnungen nutzen.

Ein benutzerdefiniertes Attribut, das Sie für eine POI-Kategorie definiert haben, können Sie nur auf der Ebene bearbeiten, auf der Sie es angelegt haben.

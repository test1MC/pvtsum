Detektor für eine Zählstelle einfügen

Möchten Sie einen Detektor einfügen, der einer Zählstelle zugeordnet wird, gehen Sie folgendermaßen vor:

Klicken Sie im Fenster Netz auf das Symbol Modus Einfügen.

Klicken Sie auf die Schaltfläche Detektoren.

Klicken Sie im Netz an die Position, an der Sie den Detektor einfügen möchten.

Das Fenster Detektor einfügen öffnet sich.

Sie können die Position des Detektors später ändern.

Geben Sie die gewünschten Daten ein (Attribute und Optionen von Detektoren).

Der Detektor wird eingefügt.

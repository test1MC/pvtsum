Fahrplanfahrtabschnitte einfügen

Normalerweise gibt es genau einen Fahrplanfahrtabschnitt pro Fahrplanfahrt. Dieser wird beim Einfügen einer Fahrplanfahrt automatisch angelegt. Es kann aber sinnvoll sein, eine Fahrplanfahrt in mehrere Fahrplanfahrtabschnitte zu unterteilen Siehe "Netzobjekte der Linienhierarchie".

Sie sehen die einzelnen Fahrplanfahrtabschnitte einer Fahrplanfahrt nur, wenn die Ansicht Fahrplanfahrtabschnitte ausgewählt ist (Fahrplanfahrten und Fahrplanfahrtabschnitte verwalten).

Das Einfügen von Fahrplanfahrtabschnitten kann Auswirkungen auf gekoppelte Fahrzeitprofile haben (Mehrere Fahrzeitprofile koppeln).

Stellen Sie sicher, dass die gewünschten Linien im tabellarischen Fahrplan angezeigt werden (Linie(n) für die Anzeige im Fahrplaneditor auswählen).

Markieren Sie die Fahrplanfahrt, der Sie einen Fahrplanfahrtabschnitt hinzufügen möchten, indem Sie auf den Kopf der Spalte klicken.

Klicken Sie mit der rechten Maustaste auf die markierten Fahrplanfahrt.

Wählen Sie den Eintrag Fahrplanfahrtabschnitt einfügen.

Unterhalb der ausgewählten Fahrplanfahrt wird im Bereich Fahrtabschnitte eine weiter Spalte und somit ein weiterer Fahrplanfahrtabschnitt hinzugefügt.

Bearbeiten Sie die gewünschten Attribute des neu eingefügten Fahrplanfahrtabschnitts, indem Sie in die gewünschten Zellen (doppel)klicken (Fahrplanfahrtabschnitt bearbeiten).

Ein neuer Fahrplanfahrtabschnitt wird als Kopie der ursprünglichen Fahrplanfahrt eingefügt und unterscheidet sich erst dann von ihr, wenn ein oder mehrere Attributwerte geändert werden.

Fügen Sie bei Bedarf weitere Fahrplanfahrtabschnitte ein.

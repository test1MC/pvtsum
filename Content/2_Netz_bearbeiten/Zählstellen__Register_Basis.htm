Zählstellen: Register Basis

Element

Beschreibung

ZWert1 - ZWert5

Freie Attribute, denen Sie Werte zuordnen können

Tipp

Um einem Netzobjekt zusätzliche Informationen zuzuordnen, können Sie benutzerdefinierte Attribute anlegen. Der Vorteil besteht darin, dass Sie benutzerdefinierten Attributen eine passende Bezeichnung geben können.

Bereich Position auf Strecke <Nummer>

Relative Position. (0...1)

Sie können die relative Position der Zählstelle ändern. Die markierte Position wird als Anteil an der Streckenlänge Von-Knoten - Nach-Knoten eingetragen. Geben Sie den Anteil in Form einer Kommazahl zwischen 0 und 1 ein. Dabei gilt 0.0 = Von-Knoten, 0.5 = Streckenmitte.

Entfernung von Knoten <Nummer>

Alternativ können Sie die Entfernung der Zählstelle zum Von-Knoten mit dem Schieberegler festlegen (Zählstelle verschieben).

Anzeige-Eigenschaften von Attributen ändern

Im oberen Fensterbereich legen Sie fest, wie die ausgewählten Attribute bei der Ausgabe angezeigt werden. Von der Art des Attributs hängt ab, welche Einstellungen Sie vornehmen können. Im Fenster Attributauswahl (<Netzobjekttyp>) der Listenansicht und der Schnellansicht werden zusätzliche Funktionen zum Definieren einer aggregierten Ansicht angezeigt.

Spalte

Beschreibung

Attribut

Langname des ausgewählten Attributs

Den Namen des Attributs können Sie nicht ändern.

Gruppieren

(nur in Listenansicht)

Sie können Netzobjekte mit gleichem Attributwert in einer Zeile zusammenfassen. Wenn Sie diese Option für ein Attribut aktivieren, müssen Sie für alle anderen Attribute (oder Spalten) angeben, wie deren Attributwerte aggregiert werden sollen. Eine Gruppierung können Sie für eine oder mehrere Spalten festlegen.

Beispiel: Sie möchten die durchschnittliche Länge von Strecken des Typs 20 ermitteln. Dazu gruppieren Sie die Liste der Strecken nach dem Attribut TypNr. Damit werden alle Strecken von Typ 20 in einer Zeile zusammengefasst. Für das Attribut Länge legen Sie die Aggregationsfunktion Durchschnitt fest. Dann ermittelt die durchschnittliche Länge der Strecken des Typs 20 und zeigt den Wert in der Spalte Avg(Länge) an.

Aggregationsfunktion

(nur in Listenansicht und Schnellansicht)

Hier legen Sie fest, welche Aggregationsfunktion verwendet werden soll (Aggregationsfunktion für eine Spalte einer gruppierten Liste auswählen).

Gewicht

(nur in Listenansicht und Schnellansicht)

Wenn Sie Gewichtetes Mittel als Aggregationsfunktion ausgewählt haben, ist zunächst die Gewichtung 1.0 festgelegt. Klicken Sie auf das Feld öffnet sich das Fenster Attribut (<Netzobjekttyp>) in dem Sie das Attribut auswählen, nach dem gewichtet werden soll (Aggregationsfunktion für eine Spalte einer gruppierten Liste auswählen).

Diese Option ist nur für numerische Attribute verfügbar.

Dezimalstellen

Anzahl der angezeigten Dezimalstellen

Über die Schaltflächen können Sie die Anzahl Dezimalstellen erhöhen oder reduzieren.

Die Option ist nur für Attribute vom Typ Gleitkommazahl verfügbar.

Den Standardwert können Sie ändern (Einstellungen für Dezimalstellen und Aggregationsfunktionen).

Einheiten

Ist die Option ausgewählt, wird die in der Spalte Format festgelegte Ausgabeeinheit mit dem Attributwert angezeigt.

Diese Option ist nur für Zeiten und Längen verfügbar.

Ausrichtung

Die Ausrichtung des Attributwerts in der Spalte der Ausgabeliste

Für numerische Attribute empfehlen wir die Ausrichtung rechtsbündig, für alphanumerische die Ausrichtung linksbündig.

Format

Über die Auswahlliste können Sie für einige Arten von Attributen das Format festlegen, in dem die Attributwerte ausgegeben werden. Das Format wirkt sich auch auf die Dateneingabe aus (Attributwerte in einer Liste bearbeiten):

  • wissenschaftlich: Einige numerische Attribute können in wissenschaftlicher Notation ausgegeben werden. Die Zahlen werden in Exponentialschreibweise ausgegeben, wodurch große Zahlenwerte verkürzt angezeigt werden. Beispiel: 0.000234 ist in wissenschaftlicher Notation 2.34e-004.
  • Zeiten können in Sekunden (s), Minuten (m) oder Stunden (h) oder in den Formaten hh:mm und hh:mm:ss angezeigt werden.
  • Längen können in verschiedenen Einheiten angezeigt werden (z. B. Kilometer oder Meter).

Ob die Daten in metrischen oder imperialen Einheiten angezeigt werden, legen Sie in den Grundeinstellungen fest (Standardeinstellungen für Längen und Geschwindigkeiten wählen).

  • Belastungen werden zum Beispiel standardmäßig in ihrer jeweiligen Einheit angezeigt (beispielsweise Fahrzeuge, Pkw-Einheiten, Fahrgäste), können aber auch als prozentualer Anteil an der Gesamtbelastung oder im wissenschaftlichen Format angezeigt werden.

Mit dem Pfeil können Sie ein oder mehrere Attribute in der Liste nach oben schieben. Markieren Sie dazu die Attribute und klicken Sie auf den Pfeil.

Mit dem Pfeil können Sie ein oder mehrere Attribute in der Liste nach unten schieben. Markieren Sie dazu die Attribute und klicken Sie auf den Pfeil.

Analysezeiträume für Attribute mit zeitlichem Bezug festlegen

Bei Attributen, die sich auf einen bestimmten Zeitraum beziehen, zum Beispiel die Belastung, wählen Sie zuerst den Analysezeitraum, für den Sie einen Attributwert anzeigen möchten, und anschließend das Attribut. Für jeden gewählten Analysezeitraum wird dann ein Eintrag hinzugefügt. Zur Unterscheidung wird dem Attributnamen in Klammern der Zeitraum hinzufügt.

Optional können Sie die gewählten Analysezeiträume auch auf schon ausgewählte Attribute anwenden Siehe "Zeitliche und räumliche Zusammenhänge in ".

Stellen Sie sicher, dass das Fenster Attributauswahl (<Netzobjekttyp>) geöffnet ist.

Klicken Sie auf die Schaltfläche neben der Beschriftung Vorbelegung der Analysezeiträume.

 

Die Schaltfläche ist mit den aktuell ausgewählten Analysezeiträumen beschriftet. Wenn kein Analysezeitraum ausgewählt ist, ist die Schaltfläche mit ... beschriftet.

Das Fenster Analysezeiträume vorbelegen mit öffnet sich.

Wählen Sie in der Auswahlliste einen oder mehrere Einträge. Zur Auswahl mehrerer Einträge halten Sie die Taste Strg gedrückt und klicken auf die gewünschten Einträge.

Über die Schaltfläche heben Sie eine zuvor getroffene Auswahl von Einträgen auf.

Alles wählen

Über die Schaltfläche wählen Sie alle Einträge gleichzeitig aus.

 

Wenn Sie die Vorbelegung eines oder mehrerer Analysezeiträume festlegen, werden Attribute mit zeitlichem Bezug im Navigator als Einträge ohne Untereinträge dargestellt. Wenn Sie in diesem Fall ein zeitliches Attribut der Liste im oberen Bereich hinzufügen, wird das Attribut für jeden zutreffenden Analysezeitraum hinzugefügt.

Wenn Sie keinen Analysezeitraum auswählen, werden Attribute mit zeitlichem Bezug im Navigator mit Untereinträgen dargestellt. Die Untereinträge entsprechen den vorhandenen Analysezeiträumen. In diesem Fall können Sie das Attribut im Navigator direkt mit dem gewünschten Analysezeitraum auswählen.

Eine Abfrage öffnet sich.

Fahrplanfahrten zeitlich verschieben

Sie können Fahrplanfahrten im grafischen oder tabellarischen Fahrplan verschieben.

Markieren Sie die Fahrplanfahrten, die Sie bearbeiten möchten (Fahrplanfahrten markieren).

Klicken Sie mit der rechten Maustaste auf eine der markierten Fahrplanfahrten.

Wählen Sie den Eintrag Fahrplanfahrten zeitlich verschieben.

Alternativ können Sie die Funktion auch über den Eintrag Markierte Fahrplanfahrten verschieben im Kontextmenü aufrufen.

Das Fenster Fahrplanfahrt(en) verschieben öffnet sich.

Element

Beschreibung

Fahrplanfahrt(en) verschieben um

Geben Sie die gewünschte Zeit ein, um die die markierten Fahrplanfahrten verschoben werden sollen. Sie können die Zeitangaben als Stunden (h), Minuten (min) und Sekunden (s) eingeben.

Beispiel: 1h1min1s.

Tipp

Wenn Sie keine Einheit angeben, wird der eingegebene Wert als Minuten interpretiert.

Hinweis

Sie können eine Fahrplanfahrt nicht auf einen Zeitpunkt vor dem gültigen Zeitintervall verschieben (Ansicht des grafischen Fahrplans anpassen). Wenn Sie das versuchen, wird eine Warnung angezeigt und die bisherige Zeitangabe bleibt unverändert.

früher/später

Abhängig von der gewählten Option werden die ausgewählten Fahrplanfahrten auf der Zeitachse in die gewünschte Richtung verschoben.

Die Fahrplanfahrten werden verschoben.

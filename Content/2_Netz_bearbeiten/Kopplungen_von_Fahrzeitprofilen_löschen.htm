Kopplungen von Fahrzeitprofilen löschen

Sie können einzelne Kopplungen oder alle Kopplungen von Fahrzeitprofilen löschen.

Einzelne Kopplungen von Fahrzeitprofilen löschen

Öffnen Sie die Liste Koppelabschnitte (Liste öffnen).

Wählen Sie die gewünschten Kopplungen aus.

Klicken Sie mit der rechten Maustaste.

Wählen Sie den Eintrag Löschen.

Die ausgewählten Kopplungen werden gelöscht.

Alle Kopplungen von Fahrzeitprofilen löschen

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Linien.

Das Kontextmenü Linien öffnet sich.

Wählen Sie den Eintrag Mehrfach-Löschen > Koppelabschnitte.

Eine Abfrage öffnet sich.

Bestätigen Sie die Abfrage.

Alle Kopplungen werden gelöscht.

Es werden nur Kopplungen aktiver Fahrzeitprofile gelöscht. Befinden sich die Kopplungen an passiven Fahrzeitprofilen können Sie diese einzeln in der Liste der Koppelabschnitte löschen.

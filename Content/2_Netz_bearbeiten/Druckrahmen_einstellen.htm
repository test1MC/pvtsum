Druckrahmen einstellen

Wenn Sie den grafischen Fahrplan drucken möchten, können Sie für den Druckrahmen folgende Einstellungen vornehmen.

Klicken Sie in der Symbolleiste auf das Symbol  Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch)öffnet sich.

Wählen Sie im Navigator die Seite Druckrahmen.

Element

Beschreibung

Zeichnen

Ist die Option ausgewählt, wird ein Druckrahmen gezeichnet.

Hinweis

Ist die Option ausgewählt, wird eine einfache Umrandung gezeichnet, wenn die Option Zeichnen auf den Seiten Überschrift und Fußzeile nicht ausgewählt wurde.

Umrandung

Sie können die Darstellung der Umrandung in einem separaten Fenster einstellen, indem Sie in das Feld klicken (Darstellung des Umrandungsstrichs einstellen).

Wählen Sie die Seite Druckrahmen > Überschrift, wenn Sie eine Überschrift einfügen möchten.

Die Überschrift besteht aus einer Zeile, die sich über die gesamte Breite des Druckrahmens erstreckt.

Element

Beschreibung

Zeichnen

Ist die Option ausgewählt, wird eine Überschrift angezeigt.

Schriftart

Sie können die Schriftart und den Schriftschnitt der Überschrift in einem separaten Fenster einstellen, indem Sie in das Feld klicken.

Textgröße

Textgröße in mm

Farbe

Sie können eine Farbe für die Überschrift in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

Hinweis

Sie können die Überschrift linksbündig, zentriert oder rechtsbündig positionieren, indem Sie auf das entsprechende Symbol auf der rechten Seite klicken.

Mindestzeilenhöhe

Ist die Option ausgewählt, können Sie eine Mindesthöhe für die Zeile angeben. Geben Sie eine Höhe in mm ein.

Hinweis

Die Zeilenhöhe sollte größer als die Textgröße sein, da sonst als Zeilenhöhe die doppelte Textgröße verwendet wird.

Hilfe

Über die Schaltfläche öffnet sich ein Fenster mit sämtlichen verfügbaren Autowerten.

Hinweis

Die Einträge in eckigen Klammern werden automatisch mit vordefinierten Werten belegt (Autowerte). Sie werden beim Zeichnen des Druckrahmens mit den aktuellen Werten befüllt. Autowerte sind sprachabhängig und können auch in benutzerdefinierten Einträgen verwendet werden.

Wählen Sie die Seite Druckrahmen > Fußzeile, wenn Sie eine Fußzeile einfügen möchten.

Die Fußzeile besteht aus zwei Zeilen mit jeweils drei Spalten, die am unteren Rand des bedruckbaren Bereichs dargestellt wird.

Sie können die Spalteninhalte jeweils linksbündig, zentriert oder rechtsbündig positionieren, indem Sie auf das jeweilige Symbol unterhalb der Eingabefelder klicken.

Starten Sie bei Bedarf den Druckvorgang (Netzdarstellung drucken).

Der Druckrahmen wird gemäß den Einstellungen gedruckt.

Die Einstellungen des Druckrahmens werden mit den anderen Fahrplangrafikparametereinstellungen in der Grafikparameterdatei *.gpgt gespeichert (Fahrplangrafikparameter speichern).
Darstellung des Umrandungsstrichs einstellen

Sie können die Strichdarstellung der Umrandung des Druckrahmens festlegen, wenn Sie in das Feld Umrandung klicken.

Klicken Sie auf der Seite Druckrahmen in das Feld Umrandung.

Das Fenster Strich Grafikparameter öffnet sich.

Element

Beschreibung

Vorschau

Hier sehen Sie, wie der Umrandungsstrich im Druck gezeichnet wird.

Breite

Strichstärke in mm

Farbe

Sie können eine Farbe für den Umrandungsstrich in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

Strichelung

Das Strichelungsmuster setzt sich aus einzelnen quadratischen Einheiten zusammen, die die Länge der Punkte oder Striche und deren Abstand festlegen. Sie können ein benutzerdefiniertes Strichelungsmuster einstellen, indem Sie den Schieber verschieben und auf einzelne Einheiten klicken.

Alternativ können Sie eine Strichelungsvorlage auswählen.

Hinweise

Aktive Einheiten (schwarz) werden als Punkt oder Strich in der ausgewählten Farbe und Strichstärke dargestellt, inaktive Einheiten (weiß) werden transparent dargestellt und bilden den Abstand zwischen zwei Punkten oder Strichen ab.

Die Länge der einzelnen Striche oder Abstände ergibt sich aus der Anzahl Einheiten pro Strich oder pro Abstand • Breite des Elements. Ein Strich, der beispielsweise aus vier Einheiten besteht, wird in der Netzdarstellung mit einer Länge von 4 mm dargestellt, wenn die Breite eines Elements 1 mm ist.

Die Höhe jeder Einheit entspricht der eingestellten Strichstärke (=Breite).

Breite eines Elements

Länge jeder Einheit in mm

Hinweis

Der Schieber legt die Gesamtlänge des Musters fest (Anzahl Einheiten • Breite des Elements).

Strichelung aus Vorlage laden

Über diese Schaltfläche öffnet sich das Fenster Strichelungsvorlagen. In dem Fenster können Sie eine Vorlage auswählen, in der die Umrandung gezeichnet werden soll.

Hinweis

Wenn Sie eine Vorlage auswählen, wird das benutzerdefinierte Strichelungsmuster überschrieben. Die Strichstärke und Farbe bleiben unverändert.

Der Umrandungsstrich wird gemäß den Einstellungen dargestellt.

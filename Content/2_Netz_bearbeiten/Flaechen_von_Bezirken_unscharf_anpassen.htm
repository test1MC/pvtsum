Flächen von Bezirken unscharf anpassen

Sie können Flächen von Bezirken im Netz unscharf anpassen. Dabei werden die Punkte einer Fläche innerhalb einer definierten Toleranz in Richtung einer Referenzfläche verschoben. Ungewünschte Überlappungen oder Lücken zwischen Flächen können so entfernt werden (Grundlagen: Flächen unscharf anpassen).

Beim Einlesen von Shapefiles können Sie festlegen, ob Flächen unscharf angepasst werden (Shapefiles importieren).

Setzen Sie bei Bedarf die Bezirke aktiv, deren Flächen Sie unscharf anpassen möchten, über

  • die räumliche Auswahl (Netzobjekte über die räumliche Auswahl aktiv und passiv setzen) und/oder
  • Filter (Netzobjekte über Filter aktiv und passiv setzen).

Die ausgewählten Bezirke sind aktiv.

Klicken Sie im Fenster Netz auf das Symbol  Modus Bearbeiten.

Klicken Sie mit der rechten Maustaste auf die Schaltfläche Bezirke.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Flächen unscharf anpassen

Das Fenster Flächen unscharf anpassen öffnet sich.

Element

Beschreibung

Spalte Flächenmenge

Auswahl der Flächen, die bearbeitet werden (Flächenmenge bearbeiten).

Spalte Toleranz

Geben Sie einen Wert für den Toleranzbereich ein. Der Toleranzbereich legt fest, in welchem Radius Punkte der Fläche verschoben werden können. Vergeben Sie bei einer Flächenmenge den Wert 0, wird sie zur Referenzmenge und wird nicht verschoben.

Fügt eine zusätzliche Flächenmenge ein.

Löscht die markierte Flächenmenge.

Hinweis

Sie können mehrere Flächenmengen auswählen, indem Sie die Taste Strg drücken, diese gedrückt halten und auf die gewünschten Einträge klicken

Die gewünschten Flächen im Netz werden angepasst..

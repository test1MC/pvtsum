Knoten einfügen

Sie können Knoten an einer gewünschten Position im Netz oder auf einer Strecke einfügen.

Wenn Sie einen Knoten auf einer Strecke einfügen möchten, müssen Sie die Strecke vorher splitten (Strecke splitten).

Klicken Sie im Fenster Netz auf das Symbol Modus Einfügen.

Klicken Sie auf die Schaltfläche Knoten.

Klicken Sie an die Position im Netz, an der Sie den Knoten einfügen möchten.

Das Fenster Knoten einfügen öffnet sich.

Geben Sie die gewünschten Daten im oberen Teil des Fensters Knoten einfügen ein.

Element

Beschreibung

Nummer

Eindeutige Nummer des Knotens

Knoten werden automatisch fortlaufend nummeriert. Sie können die vorgegebene Nummer mit einer Nummer überschreiben, die im Netz noch nicht vergeben ist.

Hinweis

Wenn Sie die Knotennummer ändern, werden existierende Umlegungsergebnisse für alle Nachfragesegmente gelöscht.

Typ

Der Knotentyp (0...99) dient zur Kategorisierung des Knotens für die Definition von Abbiegerstandards (Abbieger verwalten).

Hinweis

Der Standardtyp ist 0.

Code

Kurzbezeichnung des Knotens

Name

Langbezeichnung des Knotens

Geben Sie im Register Basis die gewünschten Daten ein.

Element

Beschreibung

ZWert1 - ZWert3

Freie Attribute, die Sie für die Eingabe von zusätzlichen Werten verwenden können.

Tipp

Um einem Netzobjekt zusätzliche Informationen zuzuordnen, können Sie auch benutzerdefinierte Attribute anlegen (Benutzerdefinierte Attribute verwalten).

Kapazität IV

Zulässige stündliche IV-Auslastung des Knotens

t0 IV

Abbiegezeit im unbelasteten Netz für den IV

Oberknoten-Nummer

Anzeige der Oberknotennummer, falls der Knoten einem Oberknoten zugeordnet ist

Sie können die Zuordnung über die Schaltfläche ändern (Oberknoten verwalten).

Position

Koordinaten des Knotens im Netz

Die Koordinaten werden automatisch gemäß dem eingestellten Koordinatensystem vergeben (Koordinatensystem wählen).

Hinweis

Z-Koordinaten (Höhe) benötigen Sie für die 3D-Darstellung in Vissim. Die Höhendaten werden beim ANM-Export mit exportiert.

Steuerung

Über die Auswahlliste können Sie den Steuerungstyp am Knoten wählen:

  • unbekannt
  • rechts vor links
  • geregelte Vorfahrt (Stopp)
  • geregelte Vorfahrt (Vorfahrt achten)
  • signalisiert
  • All Way Stop
  • Kreisverkehr

Methode für Widerstände am Knoten

Über die Option legen Sie fest, dass für diesen Knoten nicht die Standardmethode für die Berechnung des Widerstands am Knoten verwendet wird, sondern die Methode, die direkt für den Knoten eingestellt ist. Wenn die Option ausgewählt ist, können Sie über die Auswahlliste die gewünschte Methode wählen.

Die Standardmethode können Sie im Menü Rechnen > Allgemeine Verfahrenseinstellungen > Eintrag IV-Einstellungen > Widerstände am Knoten einstellen (Widerstandsfunktionen definieren und verwenden und Knotenwiderstände nach der Methode Knotenwiderstandsberechnung (ICA) berechnen).

Nehmen Sie die gewünschten Einstellungen im Register Befristete Attr. vor (Befristungsdaten für befristete Attribute einfügen und bearbeiten).

Der Knoten wird im Netz eingefügt.

Nur aktive Netzobjekte markieren

Sie können einstellen, dass im Netz nur aktive Netzobjekte angeklickt werden können. Dies ist zum Beispiel hilfreich, wenn Sie Netzobjekte markieren möchten, die in der Netzdarstellung dicht neben anderen Netzobjekten des gleichen Typs liegen, die nicht markiert werden sollen. Passive Netzobjekte werden in diesem Fall ignoriert.

Klicken Sie in der Symbolleiste auf das Symbol Nur aktive Objekte anklicken.

Das Symbol wird gedrückt dargestellt. Sie können nicht auf passive Netzobjekte klicken.

Klicken Sie im Netz auf die gewünschten Netzobjekte (Netzobjekte im Netz markieren).

Nur aktive Netzobjekte werden markiert.

Wenn Sie wieder auf alle Netzobjekte klicken möchten, klicken Sie in der Symbolleiste auf das Symbol Nur aktive Objekte anklicken.

Alle Netzobjekte können angeklickt werden.

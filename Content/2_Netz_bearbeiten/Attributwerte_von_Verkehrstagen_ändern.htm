Attributwerte von Verkehrstagen ändern

Wählen Sie im Menü Netz den Eintrag ÖV-Verkehrstage.

Das Fenster Verkehrstage öffnet sich.

Stellen Sie sicher, dass das Register Verkehrstage ausgewählt ist.

Wählen Sie in der Liste den gewünschten Verkehrstag aus.

Klicken Sie auf die Schaltfläche  Bearbeiten.

Das Fenster Verkehrstag bearbeiten öffnet sich.

Sie können das Fenster auch im tabellarischen Fahrplan oder im Fenster Umlaufeditor öffnen, indem Sie mit der rechten Maustaste auf eine Fahrplanfahrt bzw. einen Umlauf klicken und den Eintrag Verkehrstag bearbeiten wählen (Das Fahrplaneditor-Fenster und Blockdarstellung öffnen).

Ändern Sie die Werte der gewünschten Attribute (Attribute und Optionen von Verkehrstagen).

Sie können für den Standard-Verkehrstag täglich nur den Namen, den Code und die Hochrechnungsfaktoren ändern.

Die Attributwerte werden geändert.

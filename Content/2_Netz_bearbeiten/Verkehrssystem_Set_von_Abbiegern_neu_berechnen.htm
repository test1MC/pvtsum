Verkehrssystem-Set von Abbiegern neu berechnen

Sie können für alle oder alle aktiven Abbieger im Netz die Verkehrssysteme neu ermitteln. Dabei werden jedem Abbieger, auch den U-Turns, die Verkehrssysteme zugeordnet, die sowohl für die Von-Strecke als auch für die Nach-Strecke zugelassen sind. Dies entspricht der Standardeinstellung, die beim Einfügen neuer Abbieger verwendet. Diese Funktion können Sie beispielsweise durchführen, nachdem Sie die Verkehrssysteme von Strecken geändert haben.

Setzen Sie bei Bedarf die Abbieger aktiv, deren Verkehrssysteme Sie neu berechnen möchten, über

  • die räumliche Auswahl (Netzobjekte über die räumliche Auswahl aktiv und passiv setzen) und/oder
  • Filterkriterien (Netzobjekte über Filter aktiv und passiv setzen).

Klicken Sie im Fenster Netz mit der rechten Maustaste auf die Schaltfläche Abbieger.

Das Kontextmenü Abbieger öffnet sich.

Wählen Sie den Eintrag Mehrfach-Bearbeiten.

Das Fenster Mehrfach-Bearbeiten Abbieger öffnet sich.

Wählen Sie das Register Sonderfunktionen.

Wählen Sie aus, ob Sie für alle oder für alle aktiven Abbieger die Verkehrssysteme neu berechnen möchten.

Element

Beschreibung

nur aktive

Ist die Option ausgewählt, werden nur aktive Abbieger berücksichtigt.

Klicken Sie auf die gewünschte Schaltfläche.

Element

Beschreibung

IV-VSysSet aus IV-VSysSet der Strecken berechnen

Über die Schaltfläche ermitteln Sie die IV-Verkehrssysteme für die gewählten Abbieger neu.

ÖV-VSysSet aus ÖV-VSysSet der Strecken berechnen

Über die Schaltfläche ermitteln Sie die ÖV-Verkehrssysteme für die gewählten Abbieger neu.

Die Verkehrssysteme werden den Abbiegern zugeordnet.

U-Turns werden entsprechend der Einstellung im Menü Bearbeiten > Benutzereinstellungen > Netz > Abbieger und Oberabbieger behandelt (Abbieger einfügen).

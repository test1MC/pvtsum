Signalanlage löschen

Signalanlagen in der Übersicht löschen

Wählen Sie im Menü Netz den Eintrag LSA.

Das Fenster LSA öffnet sich.

Wählen Sie die gewünschten Signalanlagen.

Klicken Sie auf die Schaltfläche   Löschen.

Die Signalanlagen werden aus der Liste entfernt.

Die Signalanlagen werden gelöscht.

Alternativ können Sie in der Liste der Signalanlagen im Kontextmenü auf den Eintrag Löschenklicken.
Signalanlage im Knoteneditor löschen

Eine Signalanlage, die einem (Ober-)Knoten zugeordnet ist, können Sie im Knoteneditor löschen.

Öffnen Sie den gewünschten (Ober-)Knoten im Knoteneditor (Ein Netzobjekt im Knoteneditor anzeigen).

Klicken Sie auf das Symbol  LSA löschen.

Eine Sicherheitsabfrage öffnet sich.

Bestätigen Sie die Sicherheitsabfrage mit OK.

Die Signalanlage wird gelöscht.

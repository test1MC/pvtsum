Darstellung von Haltefolgen-Abschnitten einstellen

Sie können die Darstellung der Beschriftung und des Hintergrunds von Haltefolgen-Abschnitten einstellen. Ein Haltefolgen-Abschnitt ist der Bereich zwischen zwei Halten der Haltefolge.

Beschriftung der Haltefolgen-Abschnitte einstellen

Klicken Sie in der Symbolleiste des Fahrplans auf das Symbol  Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch)öffnet sich.

Wählen Sie im Navigator die Seite Haltefolgen-Abschnitte > Beschriftung.

Element

Beschreibung

Beschriftung zeichnen

Ist die Option ausgewählt, können Sie die Beschriftung der Haltefolgen-Abschnitte einstellen. Die Haltefolgen-Abschnitte werden auf der Halte-Achse beschriftet.

Bereich Beschriftung

Inhalt

Über die Schaltfläche können Sie das Attribut auswählen, mit welchem die Haltefolgen-Abschnitte beschriftet werden (Fenster zur Auswahl eines Attributs).

Größe

Textgröße in mm

Font

Wenn Sie in das Feld klicken, können Sie eine Schriftart für die Beschriftung auswählen.

Transparent

Ist die Option ausgewählt, wird die Beschriftung nur als Schrift ohne weißen Hintergrund gezeichnet.

Farbe

Wenn Sie in das Feld klicken, können Sie eine Farbe für die Beschriftung auswählen.

Rahmen

Wenn Sie die Option auswählen, wird die Beschriftung umrahmt. Wenn Sie in das Feld klicken, können Sie eine Rahmenfarbe auswählen.

Runden

Rundungsfaktor für die Attributwerte

Nachkommastellen

Angabe der Anzahl Dezimalstellen für die Darstellung der Attributwerte

Hinweis

Die Option ist nur für numerische Attribute vom Typ Fließkommazahl verfügbar.

Wenn Sie auf die Schaltfläche Vorschau klicken, werden Ihre Änderungen im Fahrplan sofort angezeigt, ohne dass sich das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch) schließt.

Die Haltefolgen-Abschnitte werden gemäß den Einstellungen beschriftet.

Hintergrund der Haltefolgen-Abschnitte einstellen

Klicken Sie in der Symbolleiste des Fahrplans auf das Symbol  Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch)öffnet sich.

Wählen Sie im Navigator die Seite Haltefolgen-Abschnitte > Hintergrund.

Element

Beschreibung

Einheitliche Darstellung

Ist die Option ausgewählt, wird der Hintergrund für alle Haltefolgen-Abschnitte einheitlich dargestellt. Wählen Sie im Bereich Darstellung (Hintergrund) die Option Zeichnen aus und nehmen Sie die weiteren Einstellungen vor.

Klassifizierte Darstellung

Ist die Option ausgewählt, kann der Hintergrund der Haltefolgen-Abschnitte nach einem Attribut klassifiziert und dargestellt werden (Objekte anhand von Attributwerten klassifiziert darstellen).

Bereich Darstellung (Hintergrund)

Zeichnen

Ist die Option ausgewählt, wird ein Hintergrund für die Haltefolgen-Abschnitte gezeichnet.

Füllung und Randfarbe

Sie können die Füllung und Umrandung des Hintergrund in einem separaten Fenster festlegen, indem Sie in das Feld klicken (siehe „Eigenschaften eines geschlossenen Polygons bearbeiten“ auf Seite 1).

Wenn Sie auf die Schaltfläche Vorschau klicken, werden Ihre Änderungen sofort angezeigt, ohne dass sich das Fenster Grafikparameter bearbeiten: Fahrplan (grafisch) schließt.

Der Hintergrund für die Haltefolgen-Abschnitte wird gemäß den Einstellungen dargestellt.

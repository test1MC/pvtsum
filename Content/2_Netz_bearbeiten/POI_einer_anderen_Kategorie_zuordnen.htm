POI einer anderen Kategorie zuordnen

Sie können POIs einer anderen POI-Kategorie zuordnen.

Falls ein POI ein benutzerdefiniertes Attribut besitzt, das in der neuen Kategorie nicht vorhanden ist, geht es verloren. Legen Sie daher zuvor die entsprechenden Attribute an.

Beim Ändern der Kategorie erhält ein POI eine neue Nummer, falls die Nummer schon vergeben ist.

Falls ein POI eine Zuordnung zu der Kategorie besitzt, zu der er verschoben wird, wird die Zuordnung auf die Kategorie verschoben, aus der er stammt (Einem POI Netzobjekte zuordnen).
Einzelnen POI einer anderen Kategorie zuordnen

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie auf die Schaltfläche POIs.

Klicken Sie mit der rechten Maustaste auf den gewünschten POI.

Wählen Sie den Eintrag Kategorie ändern.

Das Fenster Auswahl der POI-Kategorie öffnet sich.

Alternativ können Sie das Fenster von der Liste der POIs aus über das Kontextmenü > Kategorie ändern öffnen.

Wählen Sie in der Auswahlliste die gewünschte Kategorie.

Der POI wird der anderen Kategorie zugeordnet.

Mehrere POIs einer anderen Kategorie zuordnen

Sie können mehrere POIs einer anderen POI-Kategorie zuordnen. Dabei können Sie die POIs zum einen auswählen, indem Sie sie vorher aktiv setzen. Zum anderen können Sie sie nach Kategorie auswählen.

Markieren Sie bei Bedarf die gewünschten POIs (POIs auswählen).

Klicken Sie im Fenster Netz auf das Symbol Modus Bearbeiten.

Klicken Sie mit der rechten Maustaste auf die Schaltfläche POIs.

Das Kontextmenü Points of Interest öffnet sich.

Wählen Sie den Eintrag Mehrfach-Bearbeiten und dann die gewünschte Kategorie unter dem gewünschten Eintrag.

Element

Beschreibung

alle POIs...

Ordnet allen oder allen aktiven POIs eine neue Kategorie zu

Kategorie wählen...

Öffnet das Fenster Auswahl der POI-Kategorie. In der Auswahlliste wählen Sie die Kategorie aus, deren POIs eine neue Kategorie zugeordnet werden.

Über die Option Auch POIs der Unterkategorien bearbeiten legen Sie fest, ob auch die POIs der Unterkategorien bearbeitet werden sollen.

Hinweis

Die Option ist verfügbar, wenn mehr als zwölf Kategorien vorhanden sind.

Kategorie POI <Name>

Ordnet allen oder allen aktiven POIs der gewählten Kategorie eine neue Kategorie zu. POIs in Unterkategorien werden nicht geändert.

Kategorie POI <Name> und Unterkategorien

Ordnet allen bzw. allen aktiven POIs der gewählten Kategorie und aller Unterkategorien eine neue Kategorie zu.

Das Fenster Mehrfach-Bearbeiten öffnet sich.

Wählen Sie das Register Sonderfunktionen.

Wählen Sie aus, ob die Kategorie für alle oder alle aktiven POIs geändert werden soll.

Element

Beschreibung

nur aktive

Ist die Option ausgewählt, werden nur aktive POIs berücksichtigt.

Wählen Sie in der Auswahlliste die gewünschte Kategorie.

Bestätigen Sie mit Kategorie ändern.

Die POIs werden der neuen Kategorie zugeordnet.

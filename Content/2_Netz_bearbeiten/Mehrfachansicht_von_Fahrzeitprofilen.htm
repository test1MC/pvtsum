Mehrfachansicht von Fahrzeitprofilen

Sie können die Attribute ausgewählter Fahrzeitprofile in der Liste der Profile nebeneinander anzeigen und diese direkt vergleichen. Wenn die ausgewählte Linienroute über mehr als ein Fahrzeitprofil verfügt, wird ein Register Mehrere angezeigt. Sie können die Auswahl der angezeigten Fahrzeitprofile im Register Mehrere wie folgt einschränken.

Öffnen Sie das Fenster Linienrouten (Fenster Linienrouten öffnen).

Doppelklicken Sie auf die gewünschte Linienroute in der Liste.

Das Fenster Linienroute bearbeiten öffnet sich.

Wählen Sie das Register Verlauf und Fahrzeitprofile.

Das Register enthält alle Profile der ausgewählten Linienroute.

Klicken Sie auf das Symbol  Festlegen, welche Fahrzeitprofile in der Mehrfachansicht angezeigt werden.

Das Fenster Fahrzeitprofile für Mehrfachansicht auswählen öffnet sich.

Wählen Sie die Fahrzeitprofile aus, deren Attribute Sie direkt vergleichen möchten.

Im Register Mehrere wird jedes angezeigte Attribut mehrmals nebeneinander angezeigt, abhängig von der Anzahl der ausgewählten Profile.

Wenn Sie die Profile 1, 2 und 3 für die Mehrfachansicht ausgewählt haben, werden im Register Mehrere beispielsweise die Attribute Abfahrt 1, Abfahrt 2 und Abfahrt 3 angezeigt.

In der Mehrfachansicht sind nicht alle Funktionen verfügbar, die die Einzelansicht der Profile bietet.

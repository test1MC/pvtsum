Befristungsdaten für befristete Attribute einfügen und bearbeiten

Wenn Sie für das Attribut eines Netzobjekts festgelegt haben, dass es befristet ist, müssen Sie noch die Daten für die Befristung angeben. Dabei geben Sie an, in welchem Zeitraum die Befristung gilt und welcher Wert während der Befristung für das Attribut verwendet wird.

Befristete Attribute von Strecken bearbeiten Sie im Fenster Netzeditor. Befristete Attribute von Knoten, Oberknoten, Abbiegern und Oberabbiegern bearbeiten Sie im Knoteneditor. Dazu klicken Sie auf das Symbol  Befristete Attribute bearbeiten. Unabhängig von der Art des Zugriffs öffnet sich das Fenster Befristete Attribute bearbeiten. Im Folgenden wird der Vorgang am Beispiel von Strecken beschrieben.

Klicken Sie im Fenster Netz auf das Symbol  Modus Bearbeiten.

Klicken Sie auf die Schaltfläche Strecken.

Doppelklicken Sie im Netz auf eine Strecke, für die Sie die Daten der Befristung eingeben möchten.

Das Fenster Strecke bearbeiten öffnet sich.

Wählen Sie das Register Befristete Attr.

In dem Register werden alle Attribute aufgelistet, die Sie als befristete Attribute eingefügt haben (Befristetes Attribut einfügen).

Bei Strecken und Abbiegern müssen Sie die Attribute für jede Richtung separat befristen.

Klicken Sie auf die Schaltfläche Bearbeiten.

Das Fenster Befristete Attribute bearbeiten öffnet sich.

Klicken Sie auf die Schaltfläche Einfügen.

Das Fenster Befristetes Attribut einfügen an <Netzobjekt> öffnet sich.

Sie können im Fenster folgende Einstellungen vornehmen:

Element

Beschreibung

Attribut

Der Name des befristeten Attributs

Periodisch

Ist die Option ausgewählt, besitzt das Attribut an jedem Tag des gewählten Verkehrstages im angegebenen Uhrzeitbereich den abweichenden Wert.

Verkehrstag

Über die Auswahlliste können Sie einen Verkehrstag auswählen.

Die bestehende Auswahl hängt von den Kalendereinstellungen ab (Kalender und Verkehrstage einstellen).

von

bis

Angabe von Beginn und Ende der Befristung

Als Zeitraum von - bis können Sie ein Zeitintervall zwischen zwei Zeitpunkten (bei Verwendung eines Kalenders auch über Tagesgrenzen hinweg) festlegen. Mit der Option periodisch geben Sie eine Kombination aus Verkehrstag und Zeitintervall (innerhalb eines Tages) an.

Wert

Angabe eines Attributwerts

Dieser abweichende Attributwert gilt zur angegebenen Zeit.

Die Befristungsdaten für das befristete Attribut werden eingefügt.

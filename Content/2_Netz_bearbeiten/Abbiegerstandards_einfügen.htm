Abbiegerstandards einfügen

Sie können neue Abbiegerstandards einfügen.

Wählen Sie im Menü Netz den Eintrag Abbiegerstandards.

Das Fenster Abbiegerstandards öffnet sich. Alle vorhandenen Abbiegerstandards werden angezeigt.

Sie können den ausgewählten Eintrag in der Liste verschieben, indem Sie auf die Schaltflächen und klicken. Die IDs der verschobenen Objekte ändern sich automatisch. Der Abbiegestandard mit der höchsten ID hat die höchste Priorität. Sobald ein Abbieger eingefügt wird, prüft , beginnend mit der höchsten ID, welcher Abbiegestandard zutrifft. Die Werte dieses Abbiegestandards werden dem neuen Abbieger zugewiesen.

Klicken Sie auf die Schaltfläche  Einfügen.

Das Fenster Abbiegerstandard einfügen öffnet sich.

Element

Beschreibung

ID

Angabe einer eindeutigen ID

Hinweis

Abbiegerstandards werden automatisch fortlaufend nummeriert. Die vorgegebene Nummer kann mit einer Nummer überschrieben werden, die im Netz noch nicht vergeben ist.

Abbiegetyp

Abbiegetyp der Abbieger, für die die Attributwerte verwendet werden sollen

Hinweise

Der Wertebereich ist 0-9 oder ?. Die Nummern haben folgende Bedeutung:

0=unbelegt

1=rechts

2=geradeaus

3=links

4=U-Turn

5-9=frei für die Abbildung von Sonderfällen

?=beliebiger Abbiegetyp

Stromhierarchie

Stromhierarchie der Abbieger, für die die Attributwerte verwendet werden sollen

Hinweise

Der Wertebereich ist ++, +-, --, -+. Die Zeichen haben folgende Bedeutung:

++ = Hauptstrom in Hauptstrom

+- = Hauptstrom in Nebenstrom

-+ = Nebenstrom in Hauptstrom

-- = Nebenstrom in Nebenstrom

Das ? kann als Platzhalter für ein beliebiges Zeichen verwendet werden.

Knotentyp

Knotentypnummer der Abbieger, für die die Attributwerte verwendet werden sollen

Hinweis

Der Wertebereich ist 0 bis 99.

Das ? kann als Platzhalter für ein beliebiges Zeichen verwendet werden.

Zeitzuschlag

Zeitzuschlag für einen Abbiegevorgang in Sekunden

Standardwert, der automatisch für neue Abbieger verwendet wird, die die oben festgelegten Eigenschaften besitzen.

Kapazität IV

IV- Kapazität

Standardwert, der automatisch für neue Abbieger verwendet wird, die die oben festgelegten Eigenschaften besitzen.

Ist Fahrtrichtungswechsel

Ist die Option ausgewählt, stellt der Abbieger, der aus diesem Abbiegerstandard belegt wird, einen Fahrtrichtungswechsel dar.

Standardwert, der automatisch für neue Abbieger verwendet wird, die die oben festgelegten Eigenschaften besitzen.

Der Abbiegerstandard wird eingefügt.

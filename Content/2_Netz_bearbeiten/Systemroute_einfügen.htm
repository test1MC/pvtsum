Systemroute einfügen

Beachten Sie die Regeln zum Ändern eines Verlaufs (Grundregeln zum Ändern eines Routenverlaufs).

Klicken Sie im Fenster Netz auf das Symbol Modus Einfügen.

Klicken Sie auf die Schaltfläche Systemrouten.

Klicken Sie in die Netzdarstellung.

Das Fenster Systemroute einfügen öffnet sich.

Element

Beschreibung

Name

Langbezeichnung der Systemroute angeben

Verkehrssystem

Verkehrssystem für die Systemroute auswählen

Das Fenster Systemroute digitalisieren öffnet sich.

Klicken Sie auf den Haltepunkt, an dem die Systemroute starten soll und halten Sie die Maustaste gedrückt.

Eine Systemroute muss an einem Haltepunkt starten und enden.

Ziehen Sie die Maus an den nächsten Haltepunkt oder Knoten, über den die Systemroute verlaufen soll und lassen Sie die Maustaste los.

Die Systemroute wird farbig hervorgehoben.

Klicken Sie bei Bedarf im Fenster Systemroute digitalisieren auf die Schaltfläche Autozoom, wenn Sie den Ausschnitt mit dem Verlauf vergrößern möchten.

Fügen Sie bei Bedarf auf die gleiche Weise weitere Teilverläufe ein.

Klicken Sie bei Bedarf im Fenster Systemroute digitalisieren auf die Schaltfläche Undo, wenn Sie das letzte Teilstück aus dem Systemroutenverlauf entfernen möchten.

Wird ein Haltepunkt oder Knoten mehrfach überfahren, öffnet sich das Fenster Systemroute: digitalisieren. Wählen Sie in dem Fenster die Passage aus, die Sie übernehmen möchten.

Ziehen Sie die Maustaste auf den gewünschten End-Haltepunkt der Systemroute und lassen Sie die Maustaste los.

Sie können die Maus auch direkt vom ersten Haltepunkt zum gewünschten End-Haltepunkt ziehen. Das Programm schlägt dann den besten Weg vor entsprechend dem eingestellten Suchkriterium (Linienrouten: Parameter für die Wegesuche einstellen).

Klicken Sie im Fenster Systemroute digitalisieren auf OK.

Die Systemroute wird eingefügt.

Die Gegenrichtung einer Systemroute können Sie über das Kontextmenü einfügen (Gegenrichtung einer Systemroute einfügen).

Sie können den Verlauf einer Systemroute nachträglich ändern (Systemroutenverlauf digitalisieren).
Systemrouten: Parameter für die Wegesuche einstellen

Klicken Sie im Fenster Systemroute digitalisieren auf die Schaltfläche Parameter.

Das Fenster Parameter für die Wegesuche der Systemroute öffnet sich.

Element

Beschreibung

auch über gesperrte Strecken routen

Ist die Option ausgewählt, werden für das Verkehrssystem gesperrte Strecken berücksichtigt, als ob sie für das Verkehrssystem geöffnet wären.

auch über gesperrte Abbieger routen

Ist die Option ausgewählt, werden für das Verkehrssystem gesperrte Abbieger berücksichtigt, als ob sie für das Verkehrssystem geöffnet wären.

Suchkriterium

In der Auswahlliste können Sie ein Kriterium für die Wegesuche auswählen. Gesucht wird der jeweils beste Weg zwischen zwei markierten (fixierten) Haltepunkten.

Streckenlänge-Luft

Luftlinien-Streckenlänge vom Start- zum End-Haltepunkt

Streckenfahrzeit

Wird aus der Verkehrssystem-Geschwindigkeit der Strecken berechnet

Zeit aus Streckentypgeschwindigkeit

Wird aus der für den betreffenden Streckentyp definierten Verkehrssystem-Geschwindigkeit berechnet

Streckenlänge

Berechnete Streckenlänge, wenn nicht vom Benutzer manuell angepasst

Eine Linienroute markieren

Öffnen Sie das Fenster Linienrouten (Fenster Linienrouten öffnen).

Schränken Sie bei Bedarf die Menge der Linienrouten ein, die angezeigt wird.

Element

Beschreibung

nur aktive

Ist die Option ausgewählt, werden nur aktive Linienrouten angezeigt.

Ist die Option nicht ausgewählt, werden alle Linienrouten angezeigt.

Wählen Sie per Linksklick eine Linienroute in der Liste aus.

Sie können die gesuchte Linienroute auch über das Eingabefeld auswählen. Geben Sie hierzu den Namen der Linienroute in das Feld ein. Die Linienroute wird in der Liste ausgewählt.

Der Verlauf der Linienroute wird im Netz markiert.

Klicken Sie bei Bedarf auf die Schaltfläche Autozoom.

Der Netzausschnitt wird vergrößert und die markierte Linienroute wird in der Mitte des Ausschnitts platziert.

Nutzen Sie die Schaltfläche Initialisieren, wenn Sie die bisher markierten Linienrouten aus der Auswahl entfernen möchten.

Sie können die Linienroutenverläufe einer Linie auch markieren, indem Sie im Netz auf einen Haltepunkt klicken über den eine Linienroute verläuft.

Alternativ können Sie eine Linienroute markieren, indem Sie die entsprechende Zeile in der Liste der Linienrouten auswählen (Listenbereiche auswählen).

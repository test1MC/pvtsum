Haltepunkte

Da sich die Fahrzeuge nur innerhalb des modellierten Netzes bewegen können, muss ein Haltepunkt Verbindung zum Netz haben. Dies wird erreicht, indem ein Haltepunkt entweder auf einer Strecke oder auf einem Knoten eingefügt wird. Befindet sich ein Haltepunkt auf einer Strecke, so wird er als Streckenhaltepunkt bezeichnet. Ein Haltepunkt auf einem Knoten kann von allen Linien bedient werden, die über diesen Knoten fahren. Ein Haltepunkt auf einer Strecke kann nur von den Linien bedient werden, die über diese Strecke fahren. Damit ist optional eine detaillierte richtungsfeine Modellierung auf der Basis von Masten möglich. Streckenhaltepunkte können jedoch auch ungerichtet eingefügt werden, so dass sie für beide Richtungen der Strecke befahren werden können.

Möglichkeiten der Modellierung von Haltepunkten

Die Differenzierung zwischen Haltepunkten auf Knoten und Strecken ermöglicht es, mit Netzmodelle unterschiedlicher Detaillierung aufzubauen:

  • Für die strategische Planung genügen Haltepunkte auf Knoten, da hier die genaue Lage des Haltepunktes – vor oder hinter der Kreuzung – in der Regel nicht interessiert. Haltestellenbereich und Haltestelle werden im Hintergrund automatisch erzeugt, bleiben dem Anwender aber auf Wunsch weitgehend verborgen.
  • Für die Betriebsplanung und die Versorgung von Rechnergestützten Betriebsleitsystemen (RBL) ist eine Modellierung der Haltepunkte auf Strecken sinnvoll, da so der notwendige Detaillierungsgrad erreicht werden kann.

Grundsätzlich ist es in auch möglich, beide Formen zu mischen, indem man zum Beispiel im Innerortsbereich die genauere streckenbezogene Modellierung und im Außerortsbereich die knotenbezogene Modellierung anwendet.

Ein Haltepunkt kann für jedes vorhandene Verkehrssystem zugelassen oder gesperrt werden. Nur die Fahrplanfahrten der Linienrouten, deren Verkehrssystem zugelassen ist, können dort halten.

Wir empfehlen, Linienrouten nur an Haltepunkten, die auf Knoten liegen, beginnen oder enden zu lassen, da im Fall von Linienrouten, die an Streckenhaltepunkten beginnen oder enden, ungenaue Ergebnisse entstehen können, beispielsweise bei der Berechnung ÖV-betrieblicher Kennzahlen oder im Fall von ÖV-Belastungen, welche streckenfein dargestellt werden.

Da Halte von Fahrplanfahrten stets an einem Haltepunkt stattfinden, muss jede Haltestelle mindestens einen Haltepunkt haben.

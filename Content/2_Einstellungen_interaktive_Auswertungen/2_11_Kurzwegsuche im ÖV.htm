Kurzwegsuche im ÖV

Sie können im ÖV eine fahrplanfeine oder verkehrssystemfeine Kurzwegsuche vornehmen Siehe "Kurzwegsuche".

Fahrplanfeine Suche

Wählen Sie im Menü Grafik den Eintrag Kurzwegsuche.

Das Fenster Grafikwerkzeuge (Kurzwegsuche) öffnet sich.

Wählen Sie das Register ÖV.

Element

Beschreibung

Auswahlliste

Hier können Sie das gewünschte Netzobjekt auswählen. Zur Auswahl stehen Bezirk und Haltestellenbereich.

Abfahrtszeit/ Ankunftszeit

Abfahrtszeit

Ist diese Option ausgewählt, können Sie die gewünschte Abfahrtszeit für die Kurzwegsuche in das Feld eingeben.

Hinweise

Bestimmt wird die erste mögliche Verbindung nach dem Abfahrtszeitpunkt. Hier werden auch Wege mit einem Fußweg vor der Einstiegs- oder nach der Ausstiegshaltestelle gefunden.

Wenn Sie einen Wochen- oder Jahreskalender verwenden, können Sie einen Tag auswählen, der im aktuellen Kalender enthalten ist.

Ankunftszeit

Ist diese Option ausgewählt, können Sie gewünschte Ankunftszeit für die Kurzwegsuche in das Feld eingeben.

Hinweise

Bestimmt wird die Verbindung mit dem geringsten Widerstand vor oder gleich dem angegebenen Ankunftszeitpunkt am Zielbezirk oder -haltestellenbereich (Rückwärtssuche). Gibt es mehrere Verbindungen mit dem gleichen Widerstand, so wird die Verbindung mit den wenigsten Umstiegen gewählt.

Wird eine schon berechnete Kurzwegsuche interaktiv verlängert, bezieht sich die Ankunftszeit auf das neue Ziel.

Über die Schaltfläche starten Sie die Kurzwegsuche.

Nachlauf

Eingabe des Werts für den Nachlauf

Hinweis

Der Nachlauf wird an das Suchintervall angeschlossen. Es umfasst den kompletten Tag des gewählten Suchzeitpunkts. Der Nachlauf schließt bei der Abfahrtssuche immer an 24:00 Uhr des Suchtages an. Ihn auf einen Wert > 0 zu setzen ist also umso wichtiger, je später der Suchzeitpunkt selbst liegt.

Bei der Rückwärtssuche, die auf die Ankunft bezogen ist, wird der Nachlauf vor 0:00 Uhr des Suchtags eingefügt.

Modus

Auswahlliste für den ÖV-Modus

Suchparameter

Über die Schaltfläche können Sie Einstellungen für die Kurzwegsuche nach Suchparametern auf der Basis der fahrplanfeinen Verbindungssuche vornehmen (Parameter der fahrplanfeinen Umlegung).

Hinweis

In dem Fenster können Sie beispielsweise festlegen, dass nur aktive Fahrplanfahrtabschnitte berücksichtigt werden.

Zielattribut Routenindex

Über die Schaltfläche können Sie in einem separaten Fenster ein Streckenattribut auswählen, das für die Strecken auf dem Kurzweg je Teilweg die Nummer der aktuellen Position auf dem Kurzweg erhält und auf null gesetzt wird für alle Strecken, die nicht auf dem Kurzweg liegen.

Hinweis

Wenn Sie anschließend die Streckenliste öffnen und die Liste nach dem Attribut Index sortieren, können Sie alle Attribute der Strecken entlang des Kurzwegs anzeigen.

Tipp

Zu diesen Zweck können Sie auch ein benutzerdefiniertes Attribut einfügen (Benutzerdefinierte Attribute verwalten).

Klicken Sie im Netz auf das gewünschte Startobjekt.

Klicken Sie im Netz auf das gewünschte Zielobjekt.

Das Startobjekt und das Zielobjekt müssen nicht vom gleichen Netzobjekttyp sein. Sie können den Typ ändern, indem Sie zwischen der Auswahl im Netz den gewünschten Typ in der Auswahlliste im Fenster Grafikwerkzeuge (Kurzwegsuche) wechseln.

Bezirke können nur Start- und Zielobjekte eines Weges sein. Sind Bezirke Start und/oder Ziel eines Weges, sind die jeweiligen Quell- und/oder Zielanbindungszeiten in den resultierenden Zeiten enthalten.

Der Kurzweg wird gemäß den Einstellungen angezeigt.

Schließen Sie das Fenster Grafikwerkzeuge (Kurzwegsuche) über das Menü Ansicht Eintrag Grafikwerkzeuge.

Wenn Sie das Fenster über das Menü Ansicht > Grafikwerkzeuge verlassen, wird die Kurzwegdarstellung weiterhin angezeigt.

Sie können die Darstellung beenden, indem Sie auf die Schaltfläche Initialisieren klicken. Die Ergebnisse aller Kurzwegsuchen werden dann gelöscht.

Wenn die Kurzwege nicht wie gewünscht angezeigt werden, können Sie die gewünschten Grafikparameter einstellen (Grafikparameter für Kurzwege einstellen).

Sie können die Kurzwegsuche auch in Listen anzeigen (Kurzwegsuche in Listen anzeigen).
Verkehrssystemfeine Suche

Wenn Sie eine verkehrssystemfeine Kurzwegsuche vornehmen möchten, gehen Sie wie folgt vor.

Wählen Sie im Menü Grafik den Eintrag Kurzwegsuche.

Das Fenster Grafikwerkzeuge (Kurzwegsuche) öffnet sich.

Wählen Sie das Register ÖV VSys.

Grafikparameter für die Darstellung von Spinnen einstellen

Sie können folgende Grafikparameter für Spinnen festlegen:

  • Sie legen fest, wie die für die Spinnen ausgewählten Netzobjekte dargestellt werden.
  • Sie legen fest, wie die Spinnenbalken dargestellt werden.
Darstellung ausgewählter Netzobjekte einstellen

Sie können eine Farbe für die Netzobjekte einstellen, die im Netz für die Spinne ausgewählt wurden.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag Spinnen.

Element

Beschreibung

Darstellung der ausgewählten Spinnenobjekte

Farbe Spinnenobjekte

Sie können eine Farbe für die ausgewählten Netzobjekte in einem separaten Fenster auswählen, indem Sie in das Farbfeld klicken.

Objektfüllfarbe für Beschriftung verwenden

Ist die Option ausgewählt, wird für die Beschriftung der ausgewählten Objekte die gleiche Farbe verwendet, wie für die Balken.

Ist die Option nicht ausgewählt, können Sie unter Eigene Textfarbe verwenden eine Farbe für die Beschriftung der selektierten Objekte auswählen.

Eigene Textfarbe verwenden

Über die Schaltfläche können Sie eine Farbe für die Beschriftung der im Netz ausgewählten Objekte auswählen.

Die für die Spinne ausgewählten Objekte werden gemäß den Einstellungen angezeigt.

Darstellung von Spinnenbalken einstellen

Sie können Spinnenbelastungen in Form von Balken anzeigen. Diese werden auf (nicht neben) die zugehörigen Standardbelastungsbalken gezeichnet.

Sie können Spinnenbalken manuell einfügen oder automatisch während der Berechnung der Spinne (Spinne einfügen).

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag Strecken > Balken > Darstellung.

Fügen Sie einen Balken ein.

Wählen Sie in der Zeile SkalierungAttrID das Attribut Belastung Spinne IV/ÖV aus.

Spinnenbalken werden gemeinsam mit ihrem Bezugsbalken gezeichnet. Als Bezugsbalken gelten folgende Balken:

  • Bei Belastung Spinne IV - Belastung IV [Fzg] (AP)
  • Bei Belastung Spinne ÖV - Belastung ÖV [Pers] (AP)
  • Bei Belastung Spinne VSys - Belastung VSys [Pers] (bei ÖV-VSys) oder Belastung VSys [Fzg] (bei IV-VSys)

Wenn ein passender Bezugsbalken existiert, öffnet sich eine Abfrage.

Bestimmen Sie in diesem Fall, ob die Grafikparameter des Spinnenbalkens automatisch an die Einstellungen des Bezugsbalkens angepasst werden sollen.

Die neue Spalte für den Spinnenbalken wird eingefügt. Die neue Spalte und der Bezugsbalken enthalten gegebenenfalls in der Zeile Bezugsbalken einen Verweis auf den jeweils anderen Balken. Der Spinnenbalken wird immer direkt rechts von seinem Bezugsbalken angezeigt.

Nehmen Sie weitere Einstellungen vor.

Die Einstellungen sind identisch mit jenen für Streckenbalken (Eigenschaften über Balken darstellen).

Sie sollten die gleiche Skalierung für den Spinnenbalken und den zugehörigen Bezugsbalken einstellen.

In der Vorschau im oberen Teil des Fensters wird der Spinnenbalken nur mit halber Breite angezeigt, damit der darunter liegende Bezugsbalken sichtbar ist.

Die Spinnenbalken werden gemäß den Einstellungen dargestellt.

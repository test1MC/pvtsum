Grafikparameter für Kurzwege einstellen

Sie können folgende Einstellungen für die Darstellung der IV- und ÖV-Kurzwege vornehmen.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag Kurzwegsuche.

Element

Beschreibung

Ausgewählte Netzobjekte

Sie können eine Farbe für die im Netz für die Kurzwegsuche ausgewählten Objekte in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

IV-Wege

Sie können eine Farbe für den IV-Wegeverlauf im Netz in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

ÖV-Wege

Sie können eine Farbe für den ÖV-Wegeverlauf im Netz in einem separaten Fenster auswählen, indem Sie in das Feld klicken.

Die Wegerichtung und Umsteigehaltestellen werden in der Netzdarstellung durch einen Pfeil gekennzeichnet.

Die Kurzwege werden gemäß den Einstellungen dargestellt.

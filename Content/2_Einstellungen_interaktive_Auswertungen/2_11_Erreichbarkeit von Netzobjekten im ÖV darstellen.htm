Erreichbarkeit von Netzobjekten im ÖV darstellen

Sie können die Erreichbarkeit von Knoten, Oberknoten, Haltestellen, Haltestellenbereichen, Haltepunkten oder Bezirken darstellen, indem Sie den gewünschten Netzobjekttyp nach dem Attribut Isochronenzeit ÖV oder Isochronen-Umsteigehäufigkeit ÖV klassifizieren.

Die Einstellungen sind für alle zur Verfügung stehenden Netzobjekttypen nahezu identisch. Sie sind hier am Beispiel von Haltestellen beschrieben.

Sie können mehrere Netzobjekttypen gleichzeitig klassifiziert darstellen.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag Haltestellen > Darstellung > Register Aktiv.

Wählen Sie die Option Klassifizierte Darstellung.

Wählen Sie über die Schaltfläche das Attribut Isochronen-Zeit ÖV aus.

Sie können das Netzobjekt auch nach dem Attribut Isochronen-Umsteigehäufigkeit ÖV klassifizieren.

Fügen Sie die gewünschten Klassen ein (Objekte anhand von Attributwerten klassifiziert darstellen).

Wählen Sie im Menü Grafik den Eintrag Isochronen.

Das Fenster Grafikwerkzeuge (Isochronen) öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (ÖV-Isochronen anzeigen).

Klicken Sie im Netz auf den gewünschten Bezirk, Haltestellenbereich oder Knoten.

Die Haltestellen werden klassifiziert angezeigt. Die Darstellung beschreibt die Entfernungen vom ausgewählten Netzobjekt zu allen Haltestellen im Netz.

Klicken Sie bei Bedarf auf weitere Netzobjekte.

Die klassifizierte Darstellung zeigt die Entfernungen zum jeweils nächstgelegenen ausgewählten Netzobjekt.

Schließen Sie das Fenster Grafikwerkzeuge (Isochronen) über das Menü Ansicht Eintrag Grafikwerkzeuge.

Wenn Sie das Fenster über das Menü Ansicht > Grafikwerkzeuge verlassen, wird die Isochronendarstellung weiterhin angezeigt.

Sie können die Darstellung beenden, indem Sie vor dem Schließen auf die Schaltfläche Initialisieren klicken. Die Ergebnisse aller Isochronensuchvorgänge werden dann gelöscht.

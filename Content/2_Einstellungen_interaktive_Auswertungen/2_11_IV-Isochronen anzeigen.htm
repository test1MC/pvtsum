IV-Isochronen anzeigen

Nehmen Sie im Fenster Grafikparameter bearbeiten auf der Seite 2D-Darstellung die gewünschten Einstellungen vor (Grafikparameter für die Darstellung von Isochronen einstellen).

Damit die IV-Isochronen angezeigt werden, müssen Sie im Bereich 2D Darstellung die Option Zeichnen auswählen (Grafikparameter für die Darstellung von Isochronen einstellen).

Wählen Sie im Menü Grafik den Eintrag Isochronen.

Das Fenster Grafikobjekte (Isochronen) öffnet sich.

Wählen Sie das Register IV.

Nehmen Sie weitere Einstellungen vor.

Auswahlliste Netzobjekttyp

Hier wählen Sie den Netzobjekttyp aus, von dem aus Sie die Entfernungen messen möchten.

  • Knoten
  • Oberknoten
  • Bezirk

Klicken Sie im Netz entsprechend des gewählten Netzobjekttyps auf den gewünschten Knoten, Oberknoten oder Bezirk.

Wenn in der Symbolleiste das Symbol  Nur aktive Objekte anklicken gedrückt dargestellt ist, können Sie passive Netzobjekte nicht mit der Maus markieren (Nur aktive Netzobjekte markieren).

Die Entfernungen zwischen dem ausgewählten Netzobjekt und allen Punkten im Netz werden farbig dargestellt.

Die Farben entsprechen den Grafikparameter-Einstellungen im Bereich Klassifizierung auf der Seite 2D-Darstellung (Grafikparameter für die Darstellung von Isochronen einstellen).

Klicken Sie bei Bedarf auf weitere Knoten, Oberknoten oder Bezirke.

Sie können Ihre Auswahl rückgängig machen, indem Sie erneut auf ein ausgewähltes Netzobjekt klicken.

Sie können Netzobjekte unterschiedlichen Netzobjekttyps gleichzeitig auswählen, indem Sie zwischendurch im Fenster Grafikobjekte (Isochronen) den Eintrag in der Auswahlliste Netzobjekttyp ändern.

Die Farben stellen die Entfernungen zum jeweils nächstgelegenen ausgewählten Netzobjekt dar.

Sie können im Isochronenmodus die Funktionen der Symbolleisten des Fensters Netzeditor verwenden ohne den Isochronen zu verlassen.

Sie können die Darstellung auch beenden, indem Sie auf die Schaltfläche Initialisieren klicken.

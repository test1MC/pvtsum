ÖV-Isochronen anzeigen

Nehmen Sie im Fenster Grafikparameter bearbeiten auf der Seite 2D-Darstellung die gewünschten Einstellungen vor (Grafikparameter für die Darstellung von Isochronen einstellen).

Damit die ÖV-Isochronen angezeigt werden, müssen Sie im Bereich 2D Darstellung die Option Zeichnen auswählen (Grafikparameter für die Darstellung von Isochronen einstellen).

Wählen Sie im Menü Grafik den Eintrag Isochronen.

Das Fenster Grafikwerkzeuge (Isochronen) öffnet sich.

Wählen Sie das Register ÖV.

Nehmen Sie bei Bedarf weitere Einstellungen vor.

Element

Beschreibung

Modus

Hier können Sie einen ÖV-Modus auswählen.

Abfahrtszeit

Ist die Option gewählt, suchen Sie Verbindungen, die im gewählten Zeitraum beginnen und vor Ende der Nachlaufzeit enden.

Ankunftszeit

Ist die Option gewählt, suchen Sie Verbindungen, die nach Beginn der Vorlaufzeit beginnen und im gewählten Zeitraum am Ziel ankommen.

Zeitraum von - bis

Zeitraum für die gewünschte Ankunft bzw. Abfahrt der gesuchten Verbindungen

Hinweise

Wird ein Wochen- oder Tageskalender verwendet, können Sie einen Tag auswählen, der im aktuellen Kalender enthalten ist.

Für Bezirke muss die Zugangszeit berücksichtigt werden.

Nachlauf

Zeit nach dem angegebenen Zeitraum, innerhalb der Verbindungen das Fahrziel erreichen müssen

Hinweis

Der Nachlauf wird an das Suchintervall angehängt.

Vorlauf

Zeit vor dem angegebenen Zeitraum, innerhalb der Verbindungen beginnen müssen

Hinweis

Der Wert für den Vorlauf wird dem Suchintervall vorangestellt.

Auswahlliste ÖV-Verfahren

Hier können Sie ein ÖV-Verfahren auswählen.

Hinweis

Derzeit können Sie nur das fahrplanfeine Verfahren auswählen.

Auswahlliste Netzobjekttyp

Hier wählen Sie den Netzobjekttyp aus, für den Sie die Isochronen anzeigen möchten.

Hinweis

Zur Auswahl stehen Bezirke, Haltestellenbereiche oder Knoten.

Parameter

Über die Schaltfläche können Sie Einstellungen für die Verbindungssuche vornehmen (Parameter der fahrplanfeinen Umlegung).

Hinweis

In dem Fenster können Sie beispielsweise festlegen, dass nur aktive Fahrplanfahrtabschnitte berücksichtigt werden.

Klicken Sie im Netz entsprechend des gewählten Netzobjekttyps auf den gewünschten Bezirk, Haltestellenbereich oder Knoten.

Wenn in der Symbolleiste das Symbol  Nur aktive Objekte anklicken gedrückt dargestellt ist, können Sie passive Netzobjekte nicht mit der Maus markieren (Nur aktive Netzobjekte markieren).

Klicken Sie auf die Schaltfläche Ausführen.

Die Entfernungen zwischen dem ausgewählten Netzobjekt und allen Punkten im Netz werden berechnet und als Isochronen dargestellt.

Die Farben entsprechen den Grafikparameter-Einstellungen im Bereich Klassifizierung auf der Seite 2D-Darstellung (Grafikparameter für die Darstellung von Isochronen einstellen).

Klicken Sie bei Bedarf auf weitere Bezirke, Haltestellenbereiche oder Knoten.

Sie können Ihre Auswahl rückgängig machen, indem Sie erneut auf ein ausgewähltes Netzobjekt klicken.

Sie können Netzobjekte unterschiedlichen Netzobjekttyps gleichzeitig auswählen, indem Sie zwischendurch im Fenster Isochronen den Eintrag in der Auswahlliste Netzobjekttyp ändern.

Klicken Sie auf die Schaltfläche Ausführen.

Die Isochronen für die gewählten Netzobjekte werden berechnet und angezeigt.

Schließen Sie das Fenster Grafikwerkzeuge (Isochronen) über das Menü Ansicht Eintrag Grafikwerkzeuge.

Wenn Sie das Fenster über das Menü Ansicht > Grafikwerkzeuge verlassen, wird die Isochronendarstellung weiterhin angezeigt.

Sie können die Darstellung beenden, indem Sie vor dem Schließen auf die Schaltfläche Initialisieren klicken. Die Ergebnisse aller Isochronensuchvorgängen werden dann gelöscht.

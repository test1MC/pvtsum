Kurzwegsuche im IV

Wählen Sie im Menü Grafik den Eintrag Kurzwegsuche.

Das Fenster Grafikwerkzeuge (Kurzwegsuche) öffnet sich.

Wählen Sie das Register IV.

Element

Beschreibung

Auswahlliste

Hier können Sie das gewünschte Netzobjekt auswählen. Zur Auswahl stehen Bezirk, Knoten und Oberknoten.

Wegewahl-Kriterium

Hier können Sie das Wegewahl-Kriterium auswählen.

tAkt

Fahrzeit im belasteten Netz

t0

Fahrzeit im unbelasteten Netz

Entfernung

Streckenlänge

Widerstand

In den Umlegungsparametern festgelegter Widerstand

ZWert 1...3

Freie Attribute mit benutzerdefinierten Werten

Verkehrssystem

Hier können Sie ein Verkehrssystem des IV auswählen.

Zielattribut Routenindex

Über die Schaltfläche können Sie in einem separaten Fenster ein Streckenattribut auswählen, das für die Strecken auf dem Kurzweg je Teilweg die Nummer der aktuellen Position auf dem Kurzweg erhält und auf null gesetzt wird für alle Strecken, die nicht auf dem Kurzweg liegen.

Hinweis

Wenn Sie anschließend die Streckenliste öffnen und die Liste nach dem Attribut Index sortieren, können Sie alle Attribute der Strecken entlang des Kurzwegs anzeigen.

Tipp

Zu diesem Zweck können Sie auch ein benutzerdefiniertes Attribut einfügen (Benutzerdefinierte Attribute verwalten).

Klicken Sie im Netz auf das gewünschte Startobjekt.

Wenn in der Symbolleiste das Symbol Nur aktive Objekte anklicken gedrückt dargestellt ist, können Sie passive Netzobjekte nicht mit der Maus markieren (Nur aktive Netzobjekte markieren).

Klicken Sie im Netz auf das gewünschte Zielobjekt.

Das Startobjekt und das Zielobjekt müssen nicht vom gleichen Netzobjekttyp sein. Sie können den Typ ändern, indem Sie zwischen der Auswahl im Netz den gewünschten Typ in der Auswahlliste im Fenster Grafikwerkzeuge (Kurzwegsuche) wechseln.

Wenn Sie einen Knoten oder Oberknoten als Zielobjekt auswählen, können Sie weitere Netzobjekte markieren und dem Weg somit weitere Teilwege hinzufügen.

Bezirke können nur Start- und Zielobjekte eines Weges sein. Sind Bezirke Start und/oder Ziel eines Weges, sind die jeweiligen Quell- und/oder Zielanbindungszeiten in den resultierenden Zeiten enthalten.

Der Kurzweg wird gemäß den Einstellungen angezeigt.

Schließen Sie das Fenster Grafikwerkzeuge (Kurzwegsuche) über das Menü Ansicht Eintrag Grafikwerkzeuge.

Wenn Sie das Fenster über das Menü Ansicht > Grafikwerkzeuge verlassen, wird die Kurzwegdarstellung weiterhin angezeigt.

Sie können die Darstellung beenden, indem Sie vor dem Schließen auf die Schaltfläche Initialisieren klicken. Die Ergebnisse aller Kurzwegsuchen werden dann gelöscht.

Wenn die Kurzwege nicht wie gewünscht angezeigt werden, können Sie die gewünschten Grafikparameter einstellen (Grafikparameter für Kurzwege einstellen).

Sie können die Kurzwegsuche auch in Listen anzeigen (Kurzwegsuche in Listen anzeigen).

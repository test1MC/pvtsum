Erreichbarkeit von Netzobjekten im IV klassifiziert darstellen

Sie können darstellen, wie gut Knoten, Oberknoten oder Bezirke von einem gewählten Netzobjekt aus erreichbar sind. Dafür klassifizieren Sie den gewünschten Netzobjekttyp nach dem Attribut Isochronen-Zeit IV.

Die Einstellungen für Bezirke, Knoten und Oberknoten sind nahezu identisch. Sie sind hier am Beispiel von Bezirken beschrieben.

Sie können mehrere Netzobjekttypen gleichzeitig klassifiziert darstellen.

Wählen Sie im Menü Grafik den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Wählen Sie im Navigator den Eintrag Bezirke > Darstellung > Register Aktiv.

Wählen Sie die Option klassifiziert zeichnen.

Wählen Sie über die Schaltfläche das Attribut Isochronen-Zeit IV aus.

Fügen Sie die gewünschten Klassen ein (Objekte anhand von Attributwerten klassifiziert darstellen).

Nehmen Sie bei Bedarf Einstellungen für die Klassen über die Schaltfläche Alle Klassen neu belegen vor.

Im Bereich Text können Sie für jede Klasse einstellen, wie die Netzobjekte beschriftet sein sollen. Hier können Sie beispielsweise die Isochronen-Zeit IV auswählen, so dass bei jedem Bezirk die Zeitspanne angezeigt wird, in der er erreichbar ist.

Wählen Sie im Menü Grafik den Eintrag Isochronen.

Das Fenster Grafikwerkzeuge (Isochronen) öffnet sich.

Wählen Sie das Register IV.

Nehmen Sie weitere Einstellungen vor (Erreichbarkeit von Strecken darstellen).

Klicken Sie in der Netzdarstellung auf das gewünschte Netzobjekt.

Die Bezirke werden klassifiziert angezeigt. Die Darstellung beschreibt die Entfernungen vom ausgewählten Netzobjekt zu allen Bezirken im Netz.

Klicken Sie bei Bedarf auf weitere Netzobjekte.

Die klassifizierte Darstellung zeigt die Entfernungen zum jeweils nächstgelegenen ausgewählten Netzobjekt.

Schließen Sie das Fenster Grafikwerkzeuge (Isochronen) über das Menü Ansicht Eintrag Grafikwerkzeuge.

Wenn Sie das Fenster über das Menü Ansicht > Grafikwerkzeuge verlassen, wird die Isochronendarstellung weiterhin angezeigt.

Sie können die Darstellung beenden, indem Sie vor dem Schließen auf die Schaltfläche Initialisieren klicken. Die Ergebnisse aller Isochronensuchvorgänge werden dann gelöscht.

Erreichbarkeit von Strecken darstellen

Nehmen Sie im Fenster Grafikparameter bearbeiten auf der Seite 2D-Darstellung im Bereich Klassifizierung die gewünschten Einstellungen vor (Grafikparameter für die Darstellung von Isochronen einstellen).

Wenn Sie nur die Erreichbarkeit von Strecken und nicht gleichzeitig IV-Isochronen anzeigen möchten, müssen Sie im Bereich 2D Darstellung die Option Zeichnen deaktivieren (Grafikparameter für die Darstellung von Isochronen einstellen).

Stellen Sie sicher, dass der Layer Isochronen IV aktiv ist (Anzeige und Reihenfolge der Grafiklayer ändern).

Wählen Sie im Menü Grafik den Eintrag Isochronen.

Das Fenster Grafikwerkzeuge (Isochronen) öffnet sich.

Wählen Sie das Register IV.

Nehmen Sie weitere Einstellungen vor.

Auswahlliste Netzobjekttyp

Hier wählen Sie den Netzobjekttyp aus, vom dem aus Sie die Entfernungen messen möchten.

  • Knoten
  • Oberknoten
  • Bezirk

Klicken Sie im Netz entsprechend des gewählten Netzobjekttyps auf den gewünschten Knoten, Oberknoten oder Bezirk.

Wenn in der Symbolleiste das Symbol Nur aktive Objekte anklicken gedrückt dargestellt ist, können Sie passive Netzobjekte nicht mit der Maus markieren (Nur aktive Netzobjekte markieren).

Die Entfernungen zwischen dem ausgewählten Netzobjekt und allen Streckenabschnitten werden im Netz farbig dargestellt.

Die Farben entsprechen den Grafikparameter-Einstellungen im Bereich Klassifizierung auf der Seite 2D-Darstellung (Grafikparameter für die Darstellung von Isochronen einstellen).

Die Strichstärke der Strecken ergibt sich aus den Einstellungen des Linienstils für markierte Strecken (Markierte Linienobjekte darstellen).

Klicken Sie bei Bedarf auf weitere Knoten, Oberknoten oder Bezirke.

Die Farben der Strecken stellen die Entfernungen zum jeweils nächstgelegenen ausgewählten Netzobjekt dar.

Sie können Ihre Auswahl rückgängig machen, indem Sie erneut auf ein ausgewähltes Netzobjekt klicken.

Schließen Sie das Fenster Grafikwerkzeuge (Isochronen) über das Menü Ansicht Eintrag Grafikwerkzeuge.

Wenn Sie das Fenster über das Menü Ansicht > Grafikwerkzeuge verlassen, wird die Isochronendarstellung weiterhin angezeigt.

Sie können die Darstellung beenden, indem Sie vor dem Schließen auf die Schaltfläche Initialisieren klicken. Die Ergebnisse aller Isochronensuchvorgänge werden dann gelöscht.

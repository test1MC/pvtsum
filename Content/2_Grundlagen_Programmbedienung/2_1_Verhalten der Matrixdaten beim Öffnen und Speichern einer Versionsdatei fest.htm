Verhalten der Matrixdaten beim Öffnen und Speichern einer Versionsdatei festlegen

Für jede Versionsdatei können Sie festlegen, wie die in der Version enthaltenen Matrizen beim Öffnen der Versionsdatei eingelesen werden. Außerdem können Sie die Matrixdaten in eine eigene Datei auslagern.

Öffnen Sie die gewünschte Versionsdatei.

Wählen Sie im Menü Datei den Eintrag Datei-Eigenschaften.

Das Fenster Datei-Eigenschaften öffnet sich.

Nehmen Sie die gewünschte Einstellung vor.

Element

Beschreibung

Keine Sonderbehandlung der Matrixdaten

Ist die Option ausgewählt, werden alle Matrixdaten beim Öffnen der Versionsdatei sofort eingelesen.

Matrixdaten bei Bedarf aus Versionsdatei nachladen

Ist die Option ausgewählt, werden alle Matrixdaten beim Öffnen der Versionsdatei nicht sofort eingelesen. Sie werden erst bei Bedarf nachgeladen. So wird die Versionsdatei schneller geöffnet.

Hinweis

Diese Option ist nur dann wirksam, wenn die Matrizenauslagerungsdatei aktiv ist (Einstellungen für die Behandlung von Matrizen vornehmen).

Matrixdaten in separater Matrixdatendatei speichern

Ist diese Option ausgewählt, werden alle Matrixdaten der Version in einer eigenen Datei gespeichert. Sie wird im gleichen Verzeichnis abgelegt und erhält den gleichen Namen wie die Versionsdatei. Die Dateierweiterung lautet *.vmx.

Hinweise

Diese Option ist nur dann wirksam, wenn die Matrizenauslagerungsdatei aktiv ist (Einstellungen für die Behandlung von Matrizen vornehmen).

Die Option sollten Sie aktivieren, wenn Sie mit Versionsdateien arbeiten, die eine große Menge Matrixdaten enthalten. Sie können damit das Schreiben von Versionsdateien beschleunigen, beispielsweise wenn Sie nur Änderungen am Netz vornehmen. Die Matrizen werden nicht geändert und deshalb auch nicht neu gespeichert.

Tipp

Möchten Sie die Matrixdateien wieder in die Versionsdatei integrieren, deaktivieren Sie die Option und speichern Sie die Version.

Speichern Sie die Versionsdatei.

Die gewählte Einstellung wird in die Versionsdatei übernommen. Beim nächsten Öffnen wird sie angewendet.

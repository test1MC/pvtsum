Beim Programmstart eine andere Konfigurationsdatei verwenden

Sie können festlegen, dass beim Programmstart eine andere Konfigurationsdatei verwendet. Außerdem können Sie weitere Parameter festlegen.

Legen Sie auf Ihrem Desktop eine Verknüpfung zur Datei ...\Programme\\\Exe\.exe an.

Klicken Sie mit der rechten Maustaste auf die Verknüpfung.

Wählen Sie den Eintrag Eigenschaften.

Das Fenster Eigenschaften von Visum öffnet sich.

Wählen Sie das Register Verknüpfung.

Löschen Sie gegebenenfalls im Eingabefeld Ziel am Ende den Namen der angegebenen Konfigurationsdatei.

Ergänzen Sie im Eingabefeld Ziel am Ende ein Leerzeichen und anschließend den vollständigen Pfad zur gewünschten Konfigurationsdatei.

Wenn der Pfad Leerzeichen enthält, müssen Sie ihn in Anführungszeichen setzen.

Ergänzen Sie bei Bedarf weitere Parameter (Kommandozeilenparameter zum -Start festlegen).

Die Verknüpfung wird mit Ihren Einstellungen angelegt. Wenn Sie über diese Verknüpfung starten, werden die eingegebenen Parameter berücksichtigt.

Falls Sie im Eingabefeld Ziel über einen Parameter eine andere Einstellung getroffen haben als in der Konfigurationsdatei, wird die Einstellung im Eingabefeld Ziel verwendet.

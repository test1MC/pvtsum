Benutzereinstellungen speichern, öffnen oder zurücksetzen

Benutzereinstellungen speichern

Wählen Sie im Menü Bearbeiten den Eintrag Benutzereinstellungen.

Das Fenster Benutzereinstellungen öffnet sich.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster Benutzereinstellungen speichern öffnet sich.

Wählen Sie den Pfad und geben Sie den Namen der Datei an, die gespeichert werden soll.

Die Grundeinstellungen werden in der angegebenen Datei im XML-Format gespeichert.

Alternativ können Sie die Grundeinstellungen auch über das Menü Datei > Speichern unter > Benutzereinstellungen speichern.
Benutzereinstellungen öffnen

Wählen Sie im Menü Bearbeiten den Eintrag Benutzereinstellungen.

Das Fenster Benutzereinstellungen öffnet sich.

Klicken Sie auf die Schaltfläche Öffnen.

Das Fenster Benutzereinstellungen öffnen öffnet sich.

Wählen Sie den Pfad und den Namen der Datei an, die Sie öffnen möchten.

Die Grundeinstellungen werden geöffnet. Beim Beenden von werden sie dauerhaft in der Windows-Registry gespeichert.

Alternativ können Sie die Grundeinstellungen auch über das Menü Datei > Öffnen > Benutzereinstellungen öffnen.
Benutzereinstellungen auf Standardwerte zurücksetzen

Wählen Sie im Menü Bearbeiten den Eintrag Benutzereinstellungen.

Das Fenster Benutzereinstellungen öffnet sich.

Klicken Sie auf die Schaltfläche Standardwerte.

Alle Grundeinstellungen im Fenster Benutzereinstellungen werden auf die voreingestellten Standardwerte zurückgesetzt.

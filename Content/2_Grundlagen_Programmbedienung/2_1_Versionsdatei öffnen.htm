Versionsdatei öffnen

Die Versionsdateien von sind nicht abwärts kompatibel. Sie können also Versionsdateien, die mit neueren -Versionen gespeichert wurden, nicht mit älteren -Versionen öffnen. Beispielsweise können Sie eine Versionsdatei von 14.01 nicht mit 14.00 öffnen. Die Nummer der Bugfix-Version, die mit einem Bindestrich an die Versionsnummer angehängt wird, ist davon nicht betroffen (Menü Hilfe > Info > Eintrag Version).

Wählen Sie im Menü Datei den Eintrag Version öffnen.

Wenn Sie die Änderungen im aktuellen Netz noch nicht gespeichert haben, öffnet sich eine Sicherheitsabfrage.

Bestätigen Sie die Sicherheitsabfrage.

Das Fenster Version öffnen öffnet sich.

Wählen Sie die gewünschte Datei.

Klicken Sie auf die Schaltfläche Öffnen.

Je nach dem Inhalt der Versionsdatei und Ihren Einstellungen werden diverse Abfragen oder Warnungen angezeigt, die Sie bestätigen müssen.

Die Versionsdatei wird geöffnet und der Name der Versionsdatei wird in der Titelleiste eingeblendet.

Welche Fenster nach dem Öffnen der Versionsdatei offen sind, hängt von Ihren Einstellungen ab (Fenstergrößen und -positionen einstellen).

Für jede Versionsdatei können Sie festlegen, wie die in der Version enthaltenen Matrizen beim Öffnen der Versionsdatei eingelesen werden (Verhalten der Matrixdaten beim Öffnen und Speichern einer Versionsdatei festlegen).

Wenn Sie eine Version öffnen, die mit einer -Version vor 10 gespeichert wurde und Oberknoten enthält, wird ein darin enthaltenes IV-Umlegungsergebnis verworfen.

Wenn Sie eine Version öffnen, die größer ist als Ihre Lizenz erlaubt, wird eine Warnung angezeigt, die darauf hinweist, dass sich im Oversize-Modus befindet. In der Titelleiste wird auf den Oversize-Modus hingewiesen, solange das Netz zu groß ist. Sie können die Version in diesem Modus nicht speichern. Sobald Sie das Netz auf die lizenzgerechte Größe verkleinert haben (Teilnetz erzeugen), können Sie alle Funktionen nutzen und die Version speichern.

Folgende Funktionen können Sie außer dem Teilnetzgenerator im Oversize-Modus durchführen:

  • Netz ändern
  • Screenshot von Netz und Fahrplan erstellen
  • COM-Skripte ausführen

Alternativ können Sie eine Version öffnen, indem Sie im Menü Datei in der Liste der zuletzt geöffneten Dateien auf die gewünschte Datei klicken oder indem Sie die Versionsdatei per Drag&Drop aus dem Windows-Explorer auf die Programmoberfläche ziehen.

Alternativ können Sie eine Versionsdatei automatisch beim Programmstart öffnen (Einstellungen in der Konfigurationsdatei ändern).

Alternativ können Sie eine Version öffnen, indem Sie im Windows-Explorer auf die Versionsdatei doppelklicken. In diesem Fall wird beim Start eine spezielle Projektverzeichnisdatei (aktdrv.pfd) verwendet. Sie bewirkt, dass im gleichen Verzeichnis, in dem die Versionsdatei liegt, nach verknüpften Dateien sucht.

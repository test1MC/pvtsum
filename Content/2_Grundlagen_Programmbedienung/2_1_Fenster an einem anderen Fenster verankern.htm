Fenster an einem anderen Fenster verankern

Sie können ein Fenster mit anderen Fenstern verankern, zu Gruppen zusammenfassen und frei verschieben. unterstützt Sie mit optischen Hilfen. Ein Symbol zeigt Ihnen zusammen mit einer Hintergrundschattierung, wo das Fenster verankert werden kann.

Klicken Sie auf die Titelleiste oder das Register des Fensters und halten Sie die Maustaste gedrückt.

Wenn Sie die Maus bewegen wird ein Symbol wird angezeigt, das Ihnen die Möglichkeiten zur Verankerung anzeigt.

Element

Beschreibung

Sie können das Fenster im Zielfenster an allen vier Fensterrändern verankern. Zusätzlich kann es als Register verankert werden, wenn Sie mit dem Mauszeiger direkt auf die Mitte des Symbols zeigen.

Sie können das Fenster im Zielfenster am oberen Fensterrand verankern.

Sie können das Fenster im Zielfenster am unteren Fensterrand verankern.

Sie können das Fenster im Zielfenster am linken Fensterrand verankern.

Sie können das Fenster im Zielfenster am rechten Fensterrand verankern.

Ziehen Sie den Mauszeiger auf das gewünschte Symbol.

Der gewünschte Zielbereich wird farbig schattiert.

Entscheidend ist, wohin Sie den Mauszeiger ziehen, das Fenster ist dabei nicht relevant.

Das Fenster wird an der gewünschten Position im Zielfenster verankert.

Fenster Meldungen

Das Fenster Meldungen enthält die Inhalte der Meldungsdatei und der Protokolldatei (Log-Dateien anzeigen), die um vorhandene Informationen erweitert werden, die einen direkten Zugriff auf Netzobjekte ermöglichen. Sie können Rückmeldungen von , wie beispielsweise Meldungen, Warnungen oder Sicherheitsabfragen in einem eigenen Fenster auswerten.

Die Listeneinträge werden in verschiedenen Farben angezeigt.

Schriftfarbe

Beschreibung

Grau

Eintrag bezieht sich auf ein nicht mehr vorhandenes Netz, beispielsweise nachdem Sie ein anderes Netz geladen oder neues Netz angelegt haben.

Schwarz

Eintrag bezieht sich auf das aktuell geöffnete Netz, es gibt kein Objekt, das im Fenster Netzeditor angezeigt werden kann,

Blau

Eintrag hat einen Bezug zum aktuell geöffneten Netz. Synchronisierung mit dem Fenster Netzeditor ist möglich und das zum Eintrag gehörende Netzobjekt kann im Netz markiert werden.

Wählen Sie im Menü Ansicht den Eintrag Meldungen aus.

Das Fenster Meldungen öffnet sich.

Nehmen Sie über die Symbole die gewünschten Einstellungen vor.

Symbol

Name

Beschreibung

Anzuzeigende Prioritäten auswählen

Über das Symbol wird das Fenster Prioritäten auswählen geöffnet. Sie können auswählen, welche Arten von Einträgen in der Liste angezeigt werden:

  • Fehler
  • Warnungen
  • Hinweise
  • Systemmeldungen

Suchen...

Suchen

Eingabe eines Suchbegriffs, der in einer Volltextsuche in den angezeigten Meldungen gesucht wird. Der erste Suchvorgang wird durch Drücken der Taste Eingabe gestartet.

Weitersuchen

Sucht den nächsten Treffer des Suchbegriffs aus dem Feld Suchen... in den angezeigten Meldungen.

Markierte Meldungen löschen

Die ausgewählten Listeneinträge werden nicht mehr angezeigt.

Hinweis

Sie können die Aktion nicht rückgängig machen.

Keine Synchronisation mit anderen Fenstern

Das Fenster Meldungen ist nicht mit anderen Fenstern synchronisiert.

Synchronisation mit anderen Fenstern: Markieren

Im Fenster Meldungen markierte Eintrage werden, wenn möglich, in anderen Fenstern markiert und umgekehrt.

Synchronisation mit anderen Fenstern: Markieren mit Ansicht verschieben

Im Fenster Meldungen markierte Eintrage werden, wenn möglich, in anderen Fenstern markiert. Zusätzlich wird der Netzausschnitt bei Bedarf so verschoben, dass das markierte Objekt/die markierten Objekte vollständig sichtbar sind.

Synchronisation mit anderen Fenstern: Markieren mit Autozoom

Im Fenster Meldungen markierte Eintrage werden, wenn möglich, in anderen Fenstern markiert. Zusätzlich wird im Netz ein Autozoom auf das markierte Objekt/die markierten Objekte ausgelöst.

Folgende Daten können Sie sich im Fenster Meldungen ansehen:

Element

Beschreibung

Zeit

Zeitpunkt an dem die Meldung gespeichert wurde im Format hh:mm:ss.

Priorität

Prioritäten der Meldungen, folgende Optionen können ausgewählt werden:

  • Fehler
  • Warnungen,
  • Hinweise,
  • Systemmeldungen

Typ

Netzobjekttyp, der von der Meldung betroffen ist.

ID

Identifikationsnummer des Netzobjekts

Meldungstext

Meldungstext aus der Datei messages.txt oder protocol.txt

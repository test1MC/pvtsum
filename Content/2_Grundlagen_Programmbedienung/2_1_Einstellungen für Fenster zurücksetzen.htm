Einstellungen für Fenster zurücksetzen

Sie können folgende Einstellungen zurücksetzen:

  • Fenstergröße
  • Position auf der Programmoberfläche
  • Layout von Listen in Fenstern
Position der Fenster zurücksetzen

Sie können Fenster auf ihre Standardposition zurücksetzen.

  • Wählen Sie im Menü Ansicht > Benutzeroberfläche den Eintrag Alle Fenster zurücksetzen.

Die Fenster werden auf ihre Standardposition zurückgesetzt und Fenster werden gemäß der -Standardeinstellung eingeblendet oder ausgeblendet.

Position und Größe von Fenstern zurücksetzen

Für manche Fenster können Sie die Position und Größe ändern, zum Beispiel für das Fenster Projektverzeichnisse bearbeiten (Menü Datei > Projektverzeichnisse > Projektverzeichnisse bearbeiten). Sie können diese Fenster auf ihre Standardposition (zentriert auf der Programmoberfläche) und auf ihre Standardgröße zurücksetzen.

  • Wählen Sie im Menü Ansicht > Benutzeroberfläche den Eintrag Dialogpositionen und Dialogeinstellungen zurücksetzen.

Die Fenster werden auf ihre Standardposition und Standardgröße zurückgesetzt.

Layout von Listen in Fenstern zurücksetzen

Manche Fenster in enthalten Listen in Tabellenform, beispielsweise Fenster, die Sie über das Menü Netz öffnen können. In diesen Listen können Sie die Spaltenbreite verändern, um beispielsweise die Lesbarkeit der Zeileninhalte zu verbessern. Die Änderungen werden gespeichert, wenn Sie die Fenster über die Schaltfläche Schließen verlassen. Solche Änderungen können Sie global zurücksetzen, indem Sie die Listen auf ihr Standardlayout zurücksetzen.

Diese Funktion wirkt sich nicht auf Listenfenster aus (Mit Listen arbeiten).
  • Wählen Sie im Menü Ansicht > Benutzeroberfläche den Eintrag Grideinstellungen zurücksetzen.

Die Listen werden auf ihr Standardlayout zurückgesetzt.

Schriftarten zurücksetzen

Sie können die Schriftarten im Fenster Schnellansicht, in Listen und im Knoteneditor auf die -Standardeinstellung zurücksetzen.

Wählen Sie im Menü Bearbeiten den Eintrag Benutzereinstellungen.

Das Fenster Benutzereinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag Benutzeroberfläche > Allgemein.

Klicken Sie auf die Schaltfläche Schriftarten zurücksetzen.

Die Schriftarten werden auf die -Standardeinstellung zurückgesetzt.

Alle Einstellungen gleichzeitig zurücksetzen

Sie können die Einstellungen für Symbolleisten, Größe, Ansicht und Position der Fenster Netz, Schnellansicht, Matrizen, Markierung, Protokoll, Suchen und Smart Map, Schriftarten und das Listenlayout in Fenstern gleichzeitig zurücksetzen.

  • Wählen Sie im Menü Ansicht > Benutzeroberfläche den Eintrag Komplette Konfiguration zurücksetzen.

Alle Symbolleisten, die Fenster Netz, Schnellansicht, Matrizen, Markierung, Protokoll, Suchen und Smart Map und das Listenlayout in Fenstern werden auf ihre Standardeinstellungen zurückgesetzt.

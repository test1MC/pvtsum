Vorhergehende Ansichten anzeigen

Wenn Sie die Ansicht verändert haben, können Sie zur jeweils vorhergehenden Ansicht zurückkehren und diese anzeigen. Wenn Sie momentan eine vorhergehende Ansicht angezeigt haben, können Sie zu der nachfolgenden Ansicht vor blättern und diese wieder anzeigen.

Klicken Sie in der Symbolleiste Ansicht auf das Symbol  Vorhergehender Ausschnitt.

Die vorhergehende Ansicht wird angezeigt.

Alternativ können Sie über das Menü Ansicht > Vorhergehender Ausschnitt oder über die Tastenkombination Alt + Pfeil nach links zur vorhergehenden Ansicht wechseln.

Klicken Sie in der Symbolleiste Ansicht auf das Symbol  Nächster Ausschnitt.

Die nächste Ansicht wird angezeigt.

Alternativ können Sie über das Menü Ansicht > Nächste Ausschnitt oder über die Tastenkombination Alt + Pfeil nach rechts zur nächsten Ansicht wechseln.

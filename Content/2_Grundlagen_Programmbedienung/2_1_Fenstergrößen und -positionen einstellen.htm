Fenstergrößen und -positionen einstellen

Sie legen grundsätzlich fest, wie Fenster angezeigt werden, wenn Sie das Programm neu starten oder eine Datei öffnen. Dabei können Sie beispielsweise aktuelle Einstellungen beibehalten.

Die Einstellungen umfassen die Sichtbarkeit, Größe und Position der Fenster Netzeditor, Listen, Knoteneditor, grafischer und tabellarischer Fahrplan, Schematischer Liniennetzplan, Umsteiger-Taktdarstellung sowie Größe und Position aller anderen Fenster, zum Beispiel dem Matrixeditor.

Sie können die Einstellungen über die Schaltfläche Fensterkonfiguration zurücksetzen auf die Standardeinstellung zurücksetzen.

Wählen Sie im Menü Bearbeiten den Eintrag Benutzereinstellungen.

Das Fenster Benutzereinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag Benutzeroberfläche > Allgemein.

Wählen Sie im Bereich Fensterkonfiguration die gewünschte Option.

Element

Beschreibung

Immer zurücksetzen

Beim Öffnen einer Versions- oder Netzdatei und beim Programmstart ohne Version wird nur das Fenster Netzeditor im Vollbildmodus geöffnet.

Aus Versionsdatei übernehmen, sonst zurücksetzen

Beim Öffnen einer Versionsdatei werden die in der Version zuletzt verwendeten Einstellungen geladen. Beim Programmstart ohne Version oder beim Öffnen einer Netzdatei wird nur das Fenster Netzeditor im Vollbildmodus geöffnet.

Aus Versionsdatei übernehmen, sonst beibehalten

Beim Öffnen einer Versionsdatei werden die in der Version zuletzt verwendeten Einstellungen geladen. Beim Öffnen einer Netzdatei bleibt die aktuelle Einstellung erhalten. Beim Programmstart ohne Version wird die zuletzt verwendete Einstellung geladen.

Immer beibehalten

Beim Öffnen einer Versions- oder Netzdatei bleibt die aktuelle Einstellung erhalten. Beim Programmstart ohne Version wird die zuletzt verwendete Einstellung geladen.

Ältere Versionsdateien enthalten die Informationen zur Fensterkonfiguration nicht. Wenn Sie eine solche Datei öffnen, wird mit der ersten Option nur das Fenster Netzeditor angezeigt. Ist die letzte Option ausgewählt, bleibt die aktuelle Einstellung erhalten.

Die Einstellung wird übernommen.

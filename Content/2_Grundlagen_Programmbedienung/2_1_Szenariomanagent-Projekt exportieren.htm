Szenariomanagement-Projekt exportieren

Sie können ein Projekt wie folgt exportieren.

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (siehe „Grundeinstellungen eines Projekts bearbeiten“ auf Seite 1).

Wählen Sie im Fenster Projekt bearbeiten das Register Mehrbenutzerbetrieb.

Klicken Sie auf die Schaltfläche Exportieren.

Das Fenster Szenariomanagement-Projekt speichern öffnet sich.

Wählen Sie bei Bedarf ein anderes Verzeichnis.

Geben Sie einen Dateinamen ein.

Das Projekt wird in einer Datei *.vpdb gespeichert.

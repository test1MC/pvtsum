Netzvereinigung erstellen

Sie müssen die beiden Modelle, die Sie vergleichen möchten, zunächst als Versionsdateien speichern.

Zeigen Sie im Menü Datei auf den Eintrag Netze vergleichen und überführen.

Wählen Sie den Eintrag Netzvereinigungs-Modus.

Das Fenster Netzvereinigungs-Modus öffnet sich.

Element

Beschreibung

Netz 1

Versionsdatei

Über die Schaltfläche wählen Sie eine Versionsdatei aus, die Sie mit einer anderen vergleichen möchten.

Hinweis

Alternativ können Sie Dateiname und Pfad in das Feld eingeben.

Netz 2

Versionsdatei

Über die Schaltfläche wählen Sie die Versionsdatei aus, die Sie mit der anderen ausgewählten Versionsdatei vergleichen möchten.

Grafikparameter

Datei

Über die Schaltfläche wählen Sie eine Grafikparameter-Datei aus.

Speichern

Sie können die aktuellen Grafikparameter des Netzes in einer Datei speichern, indem Sie auf die Schaltfläche klicken.

Filter

Datei

Über die Schaltfläche wählen Sie eine Filter-Datei aus.

Speichern

Sie können die aktuellen Filtereinstellungen in einer Datei speichern, indem Sie auf die Schaltfläche klicken.

Öffnen

Über die Schaltfläche können Sie eine Datei *.nmp öffnen.

Hinweis

Wenn Sie eine Vereinigungsnetz-Datei öffnen, werden die Daten im Fenster angezeigt. Das Vereinigungsnetz selbst wird nicht geladen.

Speichern

Über die Schaltfläche können Sie die im Fenster angezeigten Daten in einer Datei *.nmp speichern.

Das Vereinigungsnetz wird berechnet und angezeigt. In der Titelzeile des Programmfensters wird angezeigt, dass Sie sich im Netzvereinigungs-Modus befinden.

Sie können die Anzeige des Vereinigungsnetzes beenden, indem Sie eine andere Versions- oder Netzdatei öffnen.

Im Netzvereinigungs-Modus können Sie folgende Operationen ausführen:

  • Zoomen, Scrollen, Verschiebemodus (Hand)
  • Neue benutzerdefinierte Attribute anlegen
  • Mehrfach-Bearbeiten-Auswahl setzen
  • Filter setzen
  • Listenanzeige inklusive Attributschnittstelle (Zwischenablage, Attributdatei, Datenbank)
  • Grafikparameter ändern, speichern und öffnen
  • Netz/Version/(Datenbank, INTREST, TModel) öffnen
  • Netz speichern
  • Datenbankexport
  • mit Personal Geodatabase verbinden
  • Knotenstromdarstellung
  • Netzstatistik anzeigen
  • Drucken

-Version als COM-Server registrieren

Sie können die gleiche -Version mehrmals auf Ihrem Rechner installieren. Wenn Sie installieren, wird die aktuelle installierte Instanz automatisch als COM-Server registriert. Wenn Sie eine andere -Instanz mit der gleichen Versionsnummer als COM-Server nutzen möchten, müssen Sie sie dafür registrieren.

Öffnen Sie die gewünschte -Instanz.

Wählen Sie im Menü Hilfe den Eintrag Als COM-Server registrieren.

Die Instanz wird als COM-Server registriert.

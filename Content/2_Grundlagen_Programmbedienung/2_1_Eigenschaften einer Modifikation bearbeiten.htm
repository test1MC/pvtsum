Eigenschaften einer Modifikation bearbeiten

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Wählen Sie im Fenster Projekt bearbeiten das Register Modifikationen.

Über das Symbol  Attributauswahl können Sie Spalten ein- und ausblenden (Fenster zur Auswahl mehrerer Attribute).

Nehmen Sie in der Liste die gewünschten Einstellungen vor.

Element

Beschreibung

Ladereihenfolge

Nummer, die die Anwendungsreihenfolge der Modifikationen bestimmt. Sie können die Nummern auch über Symbole ändern (Anwendungsreihenfolge der Modifikationen festlegen).

Code

Code der Modifikation

Beschreibung

Beschreibung der Modifikation

Gruppe

Über dieses Feld können Sie einen Text eingeben oder eine vorhandene Gruppe auswählen. Mit Hilfe der Eigenschaft Gruppe können Sie Modifikationen sortieren.

Abhängig von

Eine Modifikation kann von anderen Modifikationen abhängig sein. Beispielsweise erzeugen Sie eine neue ÖV-Linie auf einer Strecke, die Sie in einer anderen Modifikation eingefügt haben. Über die Schaltfläche wählen Sie manuell die Modifikationen aus, von denen eine Modifikation abhängig ist.

Die Einstellung bewirkt Folgendes: Enthält ein Szenario eine abhängige Modifikation, werden automatisch auch die Modifikationen, von denen die Modifikation abhängig ist, angewendet.

Hinweise

Sie können die Abhängigkeit zwischen zwei Modifikationen prüfen und dabei automatisch den entsprechenden Eintrag in der Spalte auswählen (Unabhängigkeit von zwei Modifikationen prüfen).

Falls eine Modifikation von einer Modifikation abhängt, die wiederum abhängig ist, ist die Liste tatsächlich länger als die, die in dieser Spalte angezeigt wird. In den Spalten Abhängig von (vollständig) und Abhängig von (vollständig in Ladereihenfolge) werden alle Modifikation angezeigt, von denen eine Modifikation abhängig ist.

Ausschluss

Eine Modifikation kann sich mit anderen Modifikationen ausschließen. Über die Schaltfläche wählen Sie die gewünschten Modifikationen aus. Die Einstellung bewirkt, dass die sich ausschließenden Modifikationen nicht auf das gleiche Szenario angewendet werden können.

Hinweis

Sie können die Kombinierbarkeit zweier Modifikationen prüfen und dabei automatisch den entsprechenden Eintrag in der Spalte auswählen (Kombinierbarkeit von Modifikationen prüfen).

Falls eine Modifikation von einer Modifikation abhängt, die eine dritte Modifikation ausschließt, gilt dieser Ausschluss auch für die Modifikation selbst. In der Spalte Ausschluss (vollständig) werden alle Modifikationen angezeigt, die ausgeschlossen sind.

Wenn Sie Ausschlüsse und Abhängigkeiten zwischen Modifikationen ändern, führt einige Prüfungen durch:

  • Eine Modifikation kann nicht gleichzeitig von einer anderen Modifikation abhängig sein und diese ausschließen, auch nicht indirekt.
  • Zwei Modifikationen können sich nicht ausschließen, wenn sie im gleichen Szenario vorkommen.
  • Eine abhängige Modifikation muss in der Ladereihenfolge nach der Modifikation stehen, von der sie abhängig ist.

Modifikationen verwalten

Eine Modifikation ist eine Gruppe von Änderungen an Angebot oder Nachfrage, die inhaltlich zusammen gehören.

Modifikationen verwenden Sie zur Erzeugung von Szenarien. Ein Szenario entsteht dadurch, dass sie eine oder mehrere Modifikationen auf eine Basisversion anwenden (Szenarien verwalten).

Projektverzeichnisdatei öffnen

Wenn Sie eine Projektverzeichnisdatei öffnen, legen Sie damit fest, welche Pfade standardmäßig ausgewählt sind, wenn Sie eine Datei öffnen oder speichern.

Zeigen Sie im Menü Datei auf den Eintrag Projektverzeichnisse.

Wählen Sie den Eintrag Projektverzeichnisse öffnen.

Das Fenster Projektverzeichnisse öffnen öffnet sich.

Wählen Sie die gewünschte Datei.

Klicken Sie auf die Schaltfläche Öffnen.

Die Projektverzeichnisdatei wird geöffnet.

Sie können eine Projektverzeichnisdatei auch automatisch beim Programmstart öffnen (Einstellungen in der Konfigurationsdatei ändern).

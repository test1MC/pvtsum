Ansicht in der Smart Map vergrößern oder verkleinern (Zoomen)

Sie können die Darstellung des Netzes in der Smart Map vergrößern oder verkleinern, indem Sie in die Smart Map hineinzoomen oder aus der Smart Map herauszoomen.

Die Ansicht im Fenster Netzeditor ändert sich dadurch nicht.
Darstellung verkleinern (Herauszoomen)

Klicken Sie mit der rechten Maustaste in das Fenster Smart Map.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Verkleinern.

Alternativ können Sie die Taste Bild auf verwenden.

Die Darstellung im Fenster Smart Map wird um den Faktor 2.0 verkleinert.

Darstellung vergrößern (Hineinzoomen)

Klicken Sie mit der rechten Maustaste in das Fenster Smart Map.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Vergrößern.

Alternativ können Sie die Taste Bild ab verwenden.

Die Darstellung im Fenster Smart Map wird um den Faktor 2.0 vergrößert.

Gesamtes Netz in der Smart Map anzeigen

Klicken Sie mit der rechten Maustaste in das Fenster Smart Map.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Ganzes Netz zeigen.

Alternativ können Sie die Taste Pos1 verwenden.

Das gesamte Netz wird im Fenster Smart Map angezeigt, das rote Rechteck wird gemäß dem im Fenster Netzeditor angezeigten Ausschnitt angepasst.

Ausschnitt des Fensters Netzeditor in der Smart Map zentrieren

Klicken Sie mit der rechten Maustaste in das Fenster Smart Map.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Zentrieren.

Die Ansicht im Fenster Smart Map wird so verschoben, dass das rote Rechteck, also der aktuelle Ausschnitt des Fensters Netzeditor, zentral angezeigt wird.

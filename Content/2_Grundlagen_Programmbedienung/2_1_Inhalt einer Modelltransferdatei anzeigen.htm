Inhalt einer Modelltransferdatei anzeigen

Den Inhalt einer Modelltransferdatei können Sie anzeigen. Die Daten können Sie auf zwei verschiedene Arten in einer Baumstruktur anzeigen. Außerdem können Sie die Daten mit dem Netz der gerade geöffneten Versionsdatei vergleichen. Dazu können Sie die Modelltransferdatei mit anderen Fenstern, beispielsweise dem Fenster Netzeditor, synchronisieren, so dass die entsprechenden Netzobjekte in beiden Fenstern gleichzeitig markiert oder angezeigt werden.

Öffnen Sie die gewünschte Versionsdatei.

Wählen Sie im Menü Datei den Eintrag Netze vergleichen und überführen > Modelltransfer-Datei anzeigen.

Das Fenster Modelltransferdatei öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Die Modelltransferdatei öffnet sich. Die Daten werden in einem Navigator angezeigt.

Unter jedem Objekttyp gibt es bis zu vier Einträge:

 Eingefügt: Unter diesem Eintrag stehen die eingefügten Objekte

 Entfernt: Unter diesem Eintrag stehen die gelöschten Objekte

 Geändert: Unter diesem Eintrag stehen die direkt geänderten Objekte. Darunter befindet sich der Untereintrag Alle, der alle geänderten Objekte enthält, und je ein Untereintrag für die geänderten Attribute. Unter jedem Attribut sind die Objekte aufgelistet, bei denen das Attribut geändert ist, und jeweils der neue Attributwert.

 Geändert:<Attributname>: Unter diesem Eintrag stehen alle Objekte, bei denen sich das Attribut geändert hat und der neue Attributwert. gibt es keinen Untereintrag, wurde das Attribut verglichen, aber kein Objekt geändert.

Abhängig geändert: Unter diesem Eintrag stehen Objekte, bei denen sich indirekt Änderungen ergeben. Beispielsweise ändern sich automatisch Fahrzeitprofile und Fahrplanfahrten, wenn sich eine Linienroute ändert.

Ein Punkt vor einem Objekt zeigt an, dass das Objekt im aktuell geöffneten Netz vorhanden ist. Objekte mit einem Ring sind im aktuellen Netz nicht vorhanden.

Nehmen Sie über die Symbole die gewünschten Einstellungen vor.

Alternativ können Sie die Funktionen über das Kontextmenü oder das Menü Modelltransferdatei aufrufen.

Baumansicht > Flach

Ist das Symbol gedrückt, werden bestimmte „abhängige“ Objekttypen im Navigator auf oberster Ebene angezeigt, beispielsweise das Polygon einer Strecke oder der Verlauf einer Linienroute.

Baumansicht > Hierarchisch

Ist das Symbol gedrückt, werden bestimmte „abhängige“ Objekttypen im Navigator unterhalb ihres zugehörigen Objekttyps angezeigt, beispielsweise das Polygon unter Strecken.

Nur geänderte Attribute anzeigen

Ist das Symbol gedrückt, werden nur geänderte Objekttypen im Navigator angezeigt, bei denen Attribute geändert wurden.

Auch verglichene Attribute anzeigen

Ist das Symbol gedrückt, werden alle Objekttypen im Navigator angezeigt, bei denen Attribute geändert wurden und zusätzlich werden alle Attribute angezeigt, die verglichen wurden.

Auch verglichene Attribute und Tabellen anzeigen

Ist das Symbol gedrückt, werden alle Objekttypen im Navigator angezeigt, bei denen Attribute und Tabellen verglichen wurden.
Modelltransferdatei mit anderen Fenstern synchronisieren

Sie können die Modelltransferdatei mit anderen Fenstern synchronisieren, zum Beispiel mit dem Fenster Netzeditor, dem Knoteneditorfenster oder einer Liste. Wenn die Fenster synchronisiert sind, können Sie in einem Fenster ein oder mehrere Netzobjekte markieren, und die Objekte mit dem gleichen Schlüssel werden automatisch in den anderen Fenstern auch markiert. Je nach gewählter Einstellung wird die Ansicht im Fenster Netzeditor automatisch verschoben oder entsprechend vergrößert.

Damit das Synchronisieren funktioniert, müssen Sie die Synchronisierung in dem jeweils anderen Fenster ebenfalls aktivieren:

  • Listenfenster (Das Listenfenster)
  • Knoteneditorfenster (Knoteneditor verwenden)
  • Umsteiger-Taktdarstellung (Umsteiger-Taktdarstellung mit anderen Fenstern synchronisieren)
  • Schematischer Liniennetzplan (Schematischen Liniennetzplan mit anderen Fenstern synchronisieren).

Ordnen Sie das Fenster mit der Modelltransferdatei und die gewünschten anderen Fenster nebeneinander an.

Wählen Sie im Menü Modelltransferdatei > Synchronisation mit anderen Fenstern den gewünschten Eintrag.

Element

Beschreibung

Keine Synchronisation

Die Modelltransferdatei ist nicht mit anderen Fenstern synchronisiert.

Markieren

In der Modelltransferdatei markierte Objekte werden in anderen Fenstern markiert und umgekehrt.

Markieren mit Ansicht verschieben

In der Modelltransferdatei markierte Objekte werden in anderen Fenstern markiert und umgekehrt. Zusätzlich wird der Netzausschnitt des Fensters Netzeditor bei Bedarf so verschoben, dass das markierte Objekt/die markierten Objekte vollständig sichtbar sind.

Hinweis

Der Maßstab des Netzeditors verändert sich nicht.

Markieren mit Autozoom

In der Modelltransferdatei markierte Objekte werden in anderen Fenstern markiert und umgekehrt. Zusätzlich wird im Netz ein Autozoom auf das markierte Objekt/die markierten Objekte ausgelöst.

Klicken Sie in der Modelltransferdatei auf einen Eintrag.

Das Netzobjekt oder die Netzobjekte unterhalb des gewählten Eintrags werden in den anderen Fenstern markiert.

Ein Punkt neben einem Netzobjekt zeigt an, dass das Objekt im aktuell geöffneten Netz vorhanden ist. Netzobjekte mit einem Ring sind nicht im aktuell geöffneten Netz vorhanden und können daher auch nicht markiert werden.

Klicken Sie in einem anderen Fenster auf einen Eintrag für ein eingefügtes, geändertes oder gelöschtes Netzobjekt.

Das Netzobjekt wird in dem Fenster Modelltransferdatei markiert.

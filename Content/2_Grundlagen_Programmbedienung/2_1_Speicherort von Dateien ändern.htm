Speicherort von Dateien ändern

arbeitet mit verschiedenen Dateiarten, in denen Daten gespeichert oder von denen Daten eingelesen werden. Für die meisten dieser Dateiarten können Sie jeweils ein Verzeichnis festlegen, in das diese Dateien standardmäßig gespeichert und von dem sie standardmäßig gelesen werden. In diesem Verzeichnis sucht auch nach verknüpften Dateien der Dateiart, zum Beispiel nach Hintergründen oder POIs.

Zeigen Sie im Menü Datei auf den Eintrag Projektverzeichnisse.

Wählen Sie den Eintrag Projektverzeichnisse bearbeiten.

Das Fenster Projektverzeichnisse bearbeiten öffnet sich.

In diesem Fenster sind alle für relevanten Dateiarten aufgelistet (Dateiarten).

Klicken Sie in der Zeile der gewünschten Dateiart auf die Schaltfläche .

Über die Schaltfläche Alle können Sie das Verzeichnis für fast alle Dateiarten gleichzeitig ändern. Davon ausgenommen sind die Projektverzeichnisdatei (*.pfd), User-VDF-DLLs, Projections (*.prj) und -AddIns.

Wir empfehlen Ihnen, für Ihr Projekt ein Projektverzeichnis einzurichten und darin für die benötigten Dateiarten Unterverzeichnisse. Sie ändern dann den Pfad zunächst für alle Dateiarten auf das Projektverzeichnis (Beispiel: D:\Projekt_XY) und anschließend für die benötigten Dateiarten auf die Unterverzeichnisse (beispielsweise für die Versionen D:\Projekt_XY\ver oder für die Grafikparameter D:\Projekt_XY\gpa).

Wählen Sie das gewünschte Verzeichnis.

In der Spalte Pfad wird der Pfad zum gewählten Verzeichnis angezeigt.

Alternativ können Sie den Pfad ändern, indem Sie auf den Eintrag doppelklicken und den Pfad direkt bearbeiten. Auf diese Weise können Sie auch relative Pfade eingeben.

  • ..\ verweist auf ein Unterverzeichnis des Installationsverzeichnisses Ihrer -Installation (Beispiel: ..\\Doc)
  • %MYDOCUMENTS%\ verweist unter Windows 7 auf C:\Benutzer\<Benutzer>\Documents
  • %APPDATA%\ unter Windows 7 auf C:\Benutzer\<Benutzer>\AppData\Roaming\

Ändern Sie bei Bedarf weitere Verzeichnisse.

Beim nächsten Öffnen oder Speichern verwendet als Vorbelegung die neuen Pfade.

Diese Einstellung ist nur so lange wirksam, bis Sie beenden. Sie können die Einstellung jedoch auch speichern und wieder lesen (Netzdatei speichern).

Sie können hier auch die Dateierweiterungen ändern oder neue Dateierweiterungen hinzufügen. Wir empfehlen Ihnen, diese Einstellung nicht zu verändern.

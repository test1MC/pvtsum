Modifikation anlegen

Sie erzeugen eine Modifikation, indem Sie ein Basis-Szenario auswählen, in diesem Änderungen durchführen und die Änderungen speichern. erzeugt aus den Abweichungen zum Basis-Szenario eine Modelltransfer-Datei.

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Wählen Sie im Fenster Projekt bearbeiten das Register Modifikationen.

Klicken Sie auf das Symbol   Modifikation einfügen.

Alternativ können Sie über das Symbol  Modifikation duplizieren eine vorhandene Modifikation kopieren.

Das Fenster Neue Modifikation anlegen öffnet sich.

Element

Beschreibung

Code

Kurzer Name für die Modifikation

Beschreibung

Beschreibung für die Modifikation

Gruppe

Über dieses Feld können Sie einen Text eingeben oder eine vorhandene Gruppe auswählen. Mit Hilfe der Eigenschaft Gruppe können Sie Modifikationen sortieren.

Sie können diese Felder zunächst leer lassen und sie später im Register Modifikationen direkt eingeben oder ändern.

Das Fenster Basis-Szenario auswählen öffnet sich.

Sie haben hier zwei Möglichkeiten:

  • Entweder wählen Sie über die beiden Schaltflächen die abhängigen und sich ausschließenden Modifikationen. In diesem Fall ist als Basis-Szenario benutzerdefiniert ausgewählt.
  • Oder Sie wählen in der unteren Liste ein Basis-Szenario. In diesem Fall werden die abhängigen und die sich ausschließenden Modifikationen automatisch passend ausgewählt und die Schaltflächen entsprechend beschriftet.

Element

Beschreibung

Abhängig von

Eine Modifikation kann von anderen Modifikationen abhängig sein. Beispielsweise erzeugen Sie eine neue ÖV-Linie auf einer Strecke, die Sie in einer anderen Modifikation eingefügt haben. Über die Schaltfläche wählen Sie die Modifikationen aus, von denen die aktuelle Modifikation abhängt.

Die Einstellung bewirkt Folgendes: Enthält ein Szenario eine abhängige Modifikation, werden automatisch auch die Modifikationen, von denen die Modifikation abhängig ist, angewendet.

Hinweis

Diese Einstellung können Sie nachträglich verändern (Eigenschaften einer Modifikation bearbeiten).

Schließt sich aus mit

Eine Modifikation kann sich mit anderen Modifikationen ausschließen. Über die Schaltfläche wählen Sie die gewünschten Modifikationen aus. Die Einstellung bewirkt, dass die sich ausschließenden Modifikationen nicht auf das gleiche Szenario angewendet werden können.

Hinweis

Diese Einstellung können Sie nachträglich verändern (Eigenschaften einer Modifikation bearbeiten).

Basis-Szenario

Szenario oder Basisversion, das/die Sie als Ausgangspunkt für die Modifikation verwenden. Die abhängigen und die sich ausschließenden Modifikationen werden automatisch ausgewählt und später in die Spalten der Modifikation übernommen.

Hinweis

Das Szenario benutzerdefiniert wird automatisch ausgewählt, wenn Sie über eine der Schaltflächen Modifikationen ausgewählt haben.

Die Versionsdatei des gewählten Basis-Szenarios öffnet sich. Gleichzeitig öffnet sich das Fenster Modifikation bearbeiten.

Nehmen Sie an der Versionsdatei die gewünschten Änderungen vor.

Sie haben hier verschiedene Möglichkeiten.

  • Sie können die Änderungen direkt, beispielsweise im Fenster Netzeditor, vornehmen.
  • Sie können Netz- oder Nachfragedaten additiv einlesen.
  • Sie können eine oder mehrere vorhandene Modelltransferdateien anwenden (Modelltransferdatei anwenden). Auf diese Weise können Sie vorhandene Modelltransferdateien in ein Szenariomanagement integrieren.
  • Sie können eine andere Versionsdatei öffnen. Die Abweichungen zwischen dem gewählten Basis-Szenario und der geöffneten Versionsdatei werden dann anschließend als Modelltransferdatei im Projekt gespeichert. So können Sie zwei Versionsdateien in ein Szenariomanagement integrieren.

Beachten Sie dabei die Hinweise zur Eindeutigkeit der Netzobjekt-IDs (Hinweise zur Eindeutigkeit von IDs in Modifikationen).

Klicken Sie im Fenster Modifikation bearbeiten auf die Schaltfläche Fertigstellen.

Das Fenster Projekt bearbeiten öffnet sich wieder. In der Liste wurde die neue Modifikation angelegt. Die Änderungen werden innerhalb des Projekts als Modelltransferdatei gespeichert. Sie können jetzt weitere Änderungen am Projekt vornehmen.

Die Modelltransferdatei können Sie über das Symbol  Modifikation anzeigen öffnen.

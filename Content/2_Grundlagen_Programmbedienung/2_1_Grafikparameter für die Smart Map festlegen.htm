Grafikparameter für die Smart Map festlegen

Standardmäßig wird das Netz im Fenster Smart Map genauso angezeigt wie im Fenster Netzeditor. Sie können die Grafikparameter jedoch ändern.

Grafikparameter der Smart Map ändern

Klicken Sie mit der rechten Maustaste in das Fenster Smart Map.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Grafikparameter bearbeiten.

Das Fenster Grafikparameter bearbeiten öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (Grundeinstellungen für das Netzfenster vornehmen).

Hier stehen Ihnen die gleichen Grafikparameter zur Verfügung, die auch im Fenster Netzeditor verwendet werden. Einzelne Parameter, zum Beispiel die Farbe markierter Objekte, sind für die Smart Map nicht relevant, da Markierungen hier nicht angezeigt werden.

Die Darstellung in der Smart Map wird entsprechend geändert. Wenn Sie die Version speichern, werden diese Einstellungen mit gespeichert.

Grafikparameter der Smart Map speichern

Sie können die aktuellen Grafikparameter der Smart Map in einer Grafikparameterdatei speichern.

Die Grafikparameter der Smart Map werden als Datei *.gpa oder *.gpax gespeichert.

Klicken Sie mit der rechten Maustaste in das Fenster Smart Map.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Grafikparameter speichern unter.

Das Fenster Grafikparameter speichern öffnet sich.

Wählen Sie das gewünschte Verzeichnis und geben Sie den gewünschten Dateinamen ein.

Klicken Sie auf die Schaltfläche Speichern.

Die aktuellen Grafikparameter werden gespeichert.

Grafikparameter der Smart Map lesen

Sie können Grafikparameter der Smart Map einlesen.

Die Grafikparameter der Smart Map werden als Datei *.gpa oder *.gpaxgespeichert.

Klicken Sie mit der rechten Maustaste in das Fenster Smart Map.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Grafikparameter öffnen.

Das Fenster Grafikparameter öffnen öffnet sich.

Wählen Sie die gewünschte Datei.

Klicken Sie auf die Schaltfläche Öffnen.

Die Grafikparameter der Smart Map werden eingelesen.

Grafikparameter des Netzes verwenden

Sie können die aktuellen Grafikparameter des Fensters Netzeditor für die Smart Map verwenden.

Klicken Sie mit der rechten Maustaste in das Fenster Smart Map.

Ein Kontextmenü öffnet sich.

Wählen Sie den Eintrag Grafikparameter aus Netzansicht.

Die aktuellen Grafikparameter des Fensters Netzeditor werden für die Smart Map übernommen. Wenn Sie anschließend die Grafikparameter des Fensters Netzeditor ändern, ändern sich auch die Grafikparameter der Smart Map entsprechend. Zeichnen Sie die Smart Map bei Bedarf neu (Smart Map neu zeichnen).

Fenster an eine andere Position verschieben

Sie können alle Fenster unabhängig vom Programmfenster auf Ihrem Monitor verschieben. Das Verschieben auf einen zweiten Monitor ist ebenfalls möglich.

Klicken Sie auf die Titelleiste oder das Register des Fensters, halten Sie die Maustaste und die Taste Strg gedrückt und ziehen Sie das Fenster an die gewünschte Position.

Das Fenster wird an die gewünschte Stelle verschoben. Befinden sich innerhalb des Fensters Register mit anderen geöffneten Fensters, werden sie mit verschoben.

Wenn Sie das Fenster verschieben, ohne die Taste Strg zu drücken, wird das Symbol zur Verankerung des Fensters angezeigt (Fenster an einem anderen Fenster verankern).

Fenster über die Symbolleiste anordnen

Die folgenden Funktionen gelten für alle Fenster, die in der Auswahlliste der Symbolleiste Fenster angezeigt werden (Symbolleisten). Sie gelten nicht für die Fenster Netz, Schnellansicht, Matrizen, Markierung, Protokoll, Suchen und Smart Map.

Sie können die aktuell geöffneten Fenster übersichtlich auf der Programmoberfläche anordnen.

  • Klicken Sie in der Symbolleiste Fenster auf das gewünschte Symbol.

Symbol

Name

Beschreibung

Fenster als Register anordnen

Über das Symbol können Sie alle geöffneten Fenster als Register anordnen, ähnlich den Tabellenblättern in Excel. Am unteren Rand des Fensters wird eine Registerleiste angezeigt. Die Fenster erscheinen dabei in der Reihenfolge, in der sie geöffnet wurden. Sobald Sie auf ein Register klicken, erscheint das entsprechende Fenster im Vordergrund.

Fenster horizontal anordnen

Über das Symbol können Sie alle geöffneten Fenster untereinander anzeigen. Die Fenster erscheinen dabei in der Reihenfolge, in der sie geöffnet wurden.

Fenster vertikal anordnen

Über das Symbol können Sie alle geöffneten Fenster nebeneinander anzeigen. Die Fenster erscheinen dabei in der Reihenfolge, in der sie geöffnet wurden.

Die Fenster werden neu angeordnet.

Alternativ können Sie die Anordnung der Fenster auch über das Menü Fenster (oberer Bereich) oder das Menü Ansicht > Fensterkonfiguration ändern.

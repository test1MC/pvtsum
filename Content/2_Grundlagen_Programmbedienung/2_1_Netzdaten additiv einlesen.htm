Netzdaten additiv einlesen

Durch das additive Einlesen von Netzdaten können Sie ein vorhandenes Netz erweitern oder gezielt Daten ändern.

Am Beginn dieses Vorgangs haben Sie ein Netz geöffnet. Dann öffnen Sie die Netzdatei, deren Netzdaten Sie zum bereits geöffneten Netz additiv einlesen möchten. Nun legen Sie fest, welche Tabellen aus der einzulesenden Netzdatei übernommen werden und wie Konflikte behandelt werden sollen. Der gesamte Vorgang gliedert sich in diese Schritte:

  • Einstellungen für das additive Einlesen von Netzdaten festlegen
  • Datentabellen zum Einlesen auswählen
  • Einstellungen für die Konfliktvermeidung und Konfliktbehandlung festlegen

Zusätzlich haben Sie folgende Möglichkeiten:

  • Einstellungen für additives Lesen von Netzdaten speichern
  • Einstellungen für additives Lesen von Netzdaten öffnen

Falls Sie eine Netzdatei mit Koordinateninformationen additiv lesen möchten, sollten Sie darauf achten, dass das aktuelle Netz ein vordefiniertes Koordinatensystem besitzt (Koordinatensystem wählen).

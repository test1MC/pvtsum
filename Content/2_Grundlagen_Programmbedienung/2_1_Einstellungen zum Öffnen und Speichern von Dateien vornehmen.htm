Einstellungen zum Öffnen und Speichern von Dateien vornehmen

Wählen Sie im Menü Bearbeiten den Eintrag Benutzereinstellungen.

Das Fenster Benutzereinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag Benutzeroberfläche > Warnungen.

Wählen Sie die gewünschte Option.

Element

Beschreibung

Warnhinweis beim Speichern anzeigen

Über die Option legen Sie fest, ob beim Speichern einer Version unter dem gleichen Namen eine Sicherheitsabfrage angezeigt wird (Versionsdatei speichern).

Wählen Sie im Navigator den Eintrag Dateien & Protokolle > Allgemein.

Wählen Sie die gewünschten Optionen.

Element

Beschreibung

Dateiname als Dokumentname übernehmen

Über die Option legen Sie eine Voreinstellung für das Speichern einer Netzdatei fest.

Ist die Option ausgewählt, ist nach dem Speichern die gespeicherte Netzdatei geöffnet.

Ist die Option nicht ausgewählt, ist nach dem Speichern weiterhin die vorher geöffnete Datei geöffnet.

Hinweis

Diese Einstellung können Sie beim Speichern einer Netzdatei ändern (Netzdatei speichern).

Liste zuletzt geöffneter Dateien

Über die Auswahlliste legen Sie fest, wie viele Einträge die Liste der zuletzt geöffneten Dateien im Menü Datei enthalten soll.

Zeichencodierung für Textdateien

Über die Auswahlliste legen Sie fest, in welchem Format Textdateien gespeichert werden.

  • ANSI
  • Unicode
  • Unicode big endian
  • UTF-8
  • Automatik: In diesem Modus wird die Datei mit ANSI-Codepage geschrieben, wenn alle Zeichen darin verlustfrei darstellbar sind, ansonsten als UTF-8.

Wählen Sie im Navigator den Eintrag Dateien & Protokolle > Sicherungskopie.

Wählen Sie die gewünschten Optionen.

Element

Beschreibung

Sicherungskopie beim Versionsschreiben

Über die Option legen Sie fest, ob während des Speicherns eine Sicherungskopie der Version angelegt wird.

Hinweis

Die Originaldatei wird nur durch die neue Version ersetzt, wenn diese korrekt und vollständig ist.

Automatische Sicherheitskopie alle ... Minuten

Über die Option legen Sie fest, ob in regelmäßigen Abständen eine Sicherungskopie der aktuellen Arbeit in Form einer Versionsdatei anlegt. Die Sicherungskopie hat den gleichen Namen wie die aktuell geöffnete Datei – inklusive Dateierweiterung, ergänzt um die Dateierweiterung .ves (Beispiel: Testnetz.net.ves). Standardmäßig wird die Sicherungskopie im gleichen Ordner gespeichert wie die aktuell geöffnete Datei, dies können Sie jedoch über die Option Gemeinsames Verzeichnis für Sicherheitskopie ändern. Über das Eingabefeld legen Sie den zeitlichen Abstand zwischen zwei Sicherungskopien fest (Maximalwert: 120 Minuten).

Hinweise

Wenn Sie eine Sicherungskopie öffnen möchten, benennen Sie sie um in *.ver und öffnen Sie sie wie eine Versionsdatei.

kann keine Sicherungskopie erstellen, wenn ein modales Fenster offen ist. In diesem Fall wird die Sicherungskopie erstellt, sobald das Fenster geschlossen wird. So wird sichergestellt, dass nur konsistente Datenzustände gespeichert werden.

Ist aktuell keine Datei geöffnet, beispielsweise direkt nach dem Programmstart, wird die Sicherungskopie im Standardordner für Versionsdateien (Einstellungen in der Konfigurationsdatei ändern) unter dem Namen Tmpn.ves gespeichert.

Gemeinsames Verzeichnis für Sicherungskopie

Über diese Option legen Sie einen Ordner fest, in dem alle Sicherungskopien (siehe Option Automatische Sicherheitskopie alle ... Minuten) gespeichert werden. Solange Sie hier keinen Ordner festlegen, wird jede Sicherungskopie im gleichen Verzeichnis gespeichert wie die aktuell geöffnete Datei.

Hinweise

Gibt es in dem Ordner bereits eine Datei gleichen Namens, wird diese überschrieben.

Wenn eine Sicherungskopie nicht erstellt werden kann, beispielsweise weil kein Schreibrecht für den Zielordner vorliegt, versucht zunächst, die Sicherungskopie im Standardordner für Versionsdateien (Einstellungen in der Konfigurationsdatei ändern) unter dem Namen Tmpn.ves anzulegen. Schlägt auch das fehl, wird ein Hinweis angezeigt.

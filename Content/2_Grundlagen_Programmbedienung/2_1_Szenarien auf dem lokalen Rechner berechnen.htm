Szenarien auf dem lokalen Rechner berechnen

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Nehmen Sie die Grundeinstellungen zur Berechnung von Szenarien vor (Rechenknoten verwalten).

Wählen Sie im Fenster Projekt bearbeiten das Register Szenarien.

Stellen Sie sicher, dass in der Spalte Verfahrensparametersatz die gewünschten Verfahrensparameter ausgewählt sind (Verfahrensparametersatz anlegen).

Wählen Sie in der Liste die gewünschten Szenarien.

Mehrere Szenarien können Sie auswählen, indem Sie die Taste Strg gedrückt halten, während Sie auf die Szenarien klicken.

Klicken Sie auf das Symbol   Szenario berechnen.

Die Szenarien werden neu berechnet. In der Spalte Berechnungsstatus wird der Status Berechnet angezeigt. In der Spalte Ergebnis-Versionsdatei wird der Name der Ergebnisversionsdatei angezeigt, den automatisch vergibt. Falls die Liste zusätzliche Spalten mit den von Ihnen gewählten Kennziffern enthält, werden diese durch die Berechnung mit Werten gefüllt (Grundeinstellungen eines Projekts bearbeiten).

Modelltransferdatei erzeugen

Öffnen Sie die gewünschte Versionsdatei.

Zeigen Sie im Menü Datei auf den Eintrag Netze vergleichen und überführen.

Wählen Sie den Eintrag Modelltransfer-Datei erzeugen.

Das Fenster Modelltransfer-Datei erzeugen öffnet sich.

Element

Beschreibung

Netz 2

Versionsdatei

Über die Schaltfläche wählen Sie die gewünschte Datei aus.

Hinweis

Die beiden Versionsdateien müssen das gleiche Koordinatensystem besitzen.

Ausgabe

Transferdatei überführt Netz 2 in Netz 1

Die erzeugte Datei dient dazu, das gewählte Netz in das aktuell geöffnete zu überführen.

Transferdatei überführt Netz 1 in Netz 2

Die erzeugte Datei dient dazu, das aktuell geöffnete Netz in das gewählte zu überführen.

Modelltransfer-Datei nach dem Erstellen anzeigen

Ist die Option ausgewählt, wird die Modelltransfer-Datei nach dem Erstellen angezeigt.

Das Fenster Modelltransfer-Datei speichern öffnet sich.

Wählen Sie das gewünschte Verzeichnis.

Geben Sie den gewünschten Dateinamen ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster Modelltransferdatei schreiben (Benutzerdefinition) öffnet sich.

Die Netzdatei kann nicht in allen Sprachen geschrieben werden, die für die Programmoberfläche von zur Verfügung stehen. Wenn Sie die Programmoberfläche in einer Sprache (z. B. Griechisch) anzeigen, in der die Netzdatei nicht geschrieben wurde, wird oben im Fenster ein zusätzlicher Bereich zur Auswahl der Anzeigesprache angezeigt. Hier stellen Sie die Anzeigesprache für die Elemente der Tabelle ein.

Schränken Sie bei Bedarf die Menge der zu speichernden Daten ein.

Element

Beschreibung

Berechnete Attribute ausblenden

Ist die Option ausgewählt, werden berechnete Attribute ausgeblendet und nicht gespeichert.

Ist die Option nicht ausgewählt, werden auch berechnete Attribute gespeichert.

Attribute mit Defaultbelegung ausblenden

Ist die Option ausgewählt, werden nur ausgewählte belegte Attribute und deren Werte gespeichert.

Tipp

Mit der Option können Sie die Dateigröße reduzieren.

Ist die Option nicht ausgewählt, werden alle ausgewählten Attribute und deren Werte gespeichert, unabhängig davon, ob der Attributwert dem Standardwert entspricht oder davon abweicht.

Hinweis

Die Standardwerte der Attribute können Sie der Datei Attribute.xls unter ...\Programme\PTV Vision\\Doc entnehmen.

Leere Tabellen ausblenden

Ist die Option ausgewählt, werden nur Tabellen gespeichert, die Objekte enthalten.

Informationen über verglichene Attribute herausschreiben

Ist diese Option ausgewählt, wird in der Transfer-Datei gespeichert welche Attribute und Tabellen verglichen wurden.

Wählen Sie die Datentabellen, die Sie speichern möchten.

Achten Sie bei der Auswahl der Tabellen mit Netzobjekttypen darauf, dass die gewählte Kombination sinnvoll ist. Verwenden Sie wenn möglich die Schaltfläche Häufige Fälle, über die Sie auf eine Anzahl sinnvoller Kombinationen zugreifen können.

Element

Beschreibung

Die Tabelle ist ausgewählt und wird gespeichert.

Die Tabelle ist nicht ausgewählt und wird nicht gespeichert.

Hinweis

Die Tabelle $VERSION kann nicht ausgeschaltet werden.

Alle Tabellen an

Über die Schaltfläche können Sie alle Tabellen auswählen.

Hinweis

Der Zustand der Attribute in den einzelnen Tabellen bleibt dabei erhalten.

Alle Tabellen aus

Über die Schaltfläche können Sie alle Tabellen bis auf $VERSION ausschalten.

Hinweis

Der Zustand der Attribute in den einzelnen Tabellen bleibt dabei erhalten.

Häufige Fälle

Die notwendigen Daten zur Beschreibung eines Netz- oder Nachfrageobjekts können in einer oder in mehreren Tabellen enthalten sein. Beispielsweise sind alle Daten eines Streckenobjekts in der Tabelle $Strecken enthalten, für digitalisierte Strecken ist zusätzlich der Block $Streckenpoly erforderlich.

Über die Schaltfläche können Sie alle Datentabellen, die zur Beschreibung bestimmter Objekttypen nötig sind (sog. Anwendungsfälle), und deren Attribute gleichzeitig auswählen (siehe „Zur Beschreibung eines Objekttyps benötigte Datentabellen für das Speichern auswählen“ auf Seite 114).

Klicken Sie auf einen Tabellennamen.

Die Tabelle wird aufgeklappt und die Attribute der Tabelle werden angezeigt.

Sie können die Attribute mehrerer Tabellen gleichzeitig anzeigen, indem Sie die Taste Strg drücken, diese gedrückt halten und nacheinander auf die gewünschten Tabellennamen klicken.

In Abhängigkeit von den Einstellungen im oberen Bereich des Fensters werden unter Umständen nicht alle Attribute einer Tabelle angezeigt.

Wählen Sie die Attribute, die Sie speichern möchten.

Element

Beschreibung

Das Attribut ist ausgewählt und wird gespeichert.

Das Attribut ist nicht ausgewählt und wird nicht gespeichert.

Das Attribut ist ein Schlüsselattribut und kann nicht ausgeschaltet werden.

Schlüsselattribute sind Attribute, die zur Identifikation der Netzobjekte dienen.

Das Attribut ist ein Pflichtattribut und kann nicht ausgeschaltet werden.

Attributgruppen

Über die Schaltfläche können Sie vordefinierte Gruppen von Attributen in einem separaten Fenster auswählen oder ausschalten (siehe „Gruppen von Attributen für das Speichern auswählen oder ausschalten“ auf Seite 114).

Standard

Über diese Schaltfläche blenden Sie leere Tabellen und berechnete Attribute aus. Alle anderen Attribute sind ausgewählt.

Alternativ können Sie die Auswahl der Tabellen und Attribute aus einer Netzdatei *.net einlesen (siehe „Layout aus einer Netzdatei einlesen“ auf Seite 115).

Die Netz- und Nachfragedaten werden gemäß der Einstellungen gespeichert.

Die Tabellen mit den Netz- und Nachfragedaten in der Modelltransferdatei besitzen folgende Kennzeichnungen:

  • $-<Name des Objekttyps>: zu löschende Daten (aus Sicht der zu Beginn geöffneten Version)
  • $+<Name des Objekttyps>: hinzukommende Daten (aus Sicht der zu Beginn geöffneten Version)
  • $<Name des Objekttyps>: zu ändernde Daten, insbesondere topologisch veränderte Verläufe
  • $*<Name des Objekttyps>: zu ändernde Werte einzelner Attribute
  • $!<Name des Objekttyps>: Daten, die aus internen Gründen geschrieben werden müssen, auch wenn sich die Daten in den beiden Modellen entsprechen
  • $#<Name des Objekttyps>: alle Attribute, die verglichen wurden
Gruppen von Attributen für das Speichern auswählen oder ausschalten

Stellen Sie sicher, dass das Fenster Modelltransferdatei schreiben (Benutzerdefinition) offen ist (siehe „Modelltransferdatei erzeugen“ auf Seite 111).

Klicken Sie auf die Schaltfläche Attributgruppen.

Das Fenster Gruppenauswahl benutzerdefiniertes Schreiben öffnet sich.

Wählen Sie in der Auswahlliste die vordefinierte Gruppe von Attributen, deren Attribute Sie auswählen oder ausschalten möchten.

Klicken Sie auf die gewünschte Schaltfläche.

Element

Beschreibung

Einschalten

Alle Attribute der ausgewählten Gruppe werden ausgewählt.

Abschalten

Alle Attribute der ausgewählten Gruppe außer Schlüssel- und Pflichtattribute werden ausgeschaltet.

Exklusiv einschalten

Alle Attribute der ausgewählten Gruppe werden ausgewählt und alle Attribute anderer Gruppen außer Schlüssel- und Pflichtattribute werden ausgeschaltet.

Klicken Sie auf die Schaltfläche Schließen.

Die Attribute werden gemäß der Einstellungen ausgewählt oder ausgeschaltet.

Zur Beschreibung eines Objekttyps benötigte Datentabellen für das Speichern auswählen

In werden die Daten zur Beschreibung von bestimmten Netz- oder Nachfrageobjekten in mehreren Tabellen gespeichert, zum Beispiel Bezirke und zugehörige Polygone, Fahrplan- oder Umlaufdaten.

Damit Sie beim Speichern solcher Daten in die Modelltransferdatei alle benötigten Tabellen und Attribute auswählen (sog. Anwendungsfall), können Sie eine vordefinierte, auf die zu speichernden Daten abgestimmte Auswahl von Tabellen und Attributen aktivieren.

Stellen Sie sicher, dass das Fenster Modelltransferdatei schreiben (Benutzerdefinition) offen ist (siehe „Modelltransferdatei erzeugen“ auf Seite 111).

Klicken Sie auf die Schaltfläche Häufige Fälle.

Das Fenster Anwendungsfälle für selektives Netz schreiben öffnet sich.

Element

Beschreibung

Polygone schreiben

Ist die Option ausgewählt, werden alle Tabellen ausgewählt, die polygonbezogene Daten enthalten, und Sie können festlegen, ob die Tabellen der zugehörigen Netzobjekttypen (Bezirke, Oberbezirke, Oberknoten, Gebiete, POIs, Mautsysteme) auch gespeichert werden sollen.

Fahrplan schreiben

Ist die Option ausgewählt, werden alle Tabellen ausgewählt, die fahrplanbezogene Daten enthalten, und Sie können festlegen, ob kalenderperiode- und verkehrstagbezogene Daten gespeichert werden sollen.

Kalenderperiode und Verkehrstage schreiben

Ist die Option ausgewählt, werden die Tabellen Kalenderperiode, Verkehrstage und Ferientage gespeichert.

Hinweis

Die Option ist nur verfügbar, wenn das Netz einen Wochen- oder Jahreskalender enthält.

Umläufe schreiben

Ist die Option ausgewählt, werden die Tabellen der Umläufe und ihrer zugehörigen Daten ausgewählt.

Hinweis

Da Umläufe auf Verkehrstage verweisen, wird die Option Kalenderperiode und Verkehrstage schreiben automatisch gesetzt.

Oberknoten schreiben

Ist die Option ausgewählt, werden die Tabellen der Oberknoten und ihrer zugehörigen Daten ausgewählt.

Ausgewählte Netz- und Nachfragedaten als Layout speichern

Sie können die aktuell ausgewählten Tabellen und Attribute als Netzdatei *.net ohne Inhalt speichern und später als Vorgabe wieder einlesen.

Stellen Sie sicher, dass das Fenster Modelltransferdatei schreiben (Benutzerdefinition) offen ist (siehe „Modelltransferdatei erzeugen“ auf Seite 111).

Klicken Sie auf die Schaltfläche Layout speichern.

Das Fenster Netz speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Die ausgewählten Tabellen und Attribute werden als Datenblöcke ohne Inhalt gespeichert.

Layout aus einer Netzdatei einlesen

Sie können eine Auswahl von Tabellen und Attributen aus einer Netzdatei oder aus einer als Layout gespeicherten Netzdatei ohne Inhalt einlesen.

Stellen Sie sicher, dass das Fenster Modelltransferdatei schreiben (Benutzerdefinition) offen ist (siehe „Modelltransferdatei erzeugen“ auf Seite 111).

Klicken Sie auf die Schaltfläche Layout öffnen.

Das Fenster Netz öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Die in der Netzdatei enthaltene Auswahl von Tabellen und Attributen wird übernommen.

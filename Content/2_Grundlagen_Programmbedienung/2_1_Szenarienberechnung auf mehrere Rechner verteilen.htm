Szenarienberechnung auf mehrere Rechner verteilen

Sie haben die Möglichkeit, die Berechnung von Szenarien auf verschiedene Rechner zu verteilen. Dadurch können Sie Rechenkapazitäten bündeln und Ergebnisse in kürzerer Zeit erhalten. Für die Verteilung der Berechnung verwenden Sie die Software PTV Vision Berechnungsserver, die gleichzeitig mit installiert wird. Damit Sie Szenarien verteilt berechnen können, müssen Voraussetzungen gegeben sein (Grundlagen: Voraussetzungen für die Nutzung des verteilten Rechnens).

Berechnungsserver starten

So starten Sie den Berechnungsserver:

  • Klicken Sie in der Programmgruppe auf das Symbol PTV Vision Berechnungs-Server <Versionsnummer> (Hinweise zur Installation und zum Start von ).

Das Symbol PTV Vision Berechnungs-Server wird im Infobereich der Windows-Taskleiste angezeigt und der Server ist aktiv.

Solange der Szenarioberechnungsserver aktiv ist, kann die Software nicht aktualisiert oder neu installiert werden.
Berechnungsserver einstellen

Sie müssen die Berechnungsserver auf jedem Rechner einstellen, den Sie als Rechenknoten in das Berechnungsnetz einbinden möchten.

Klicken Sie auf das Symbol .

Das Fenster PTV Vision Berechnungs-Server öffnet sich.

Berechnungsserver: Register Berechnung
Maximale Anzahl paralleler Visum-Instanzen

Auswahl der maximalen Anzahl Instanzen von mit denen gleichzeitig gerechnet werden kann.
Anzahl der Rechenkerne pro Visum-Instanz

Auswahl der Anzahl von Rechenkernen, die pro -Instanz verwendet wird.

Die mögliche Auswahl in den Listen richtet sich nach der Anzahl der Prozessorkerne des Rechners, auf dem der Server aktiviert ist.

Sie können die Einstellungen auch in im Projekt auf dem Register Verteiltes Rechnen ändern (Rechenknoten verwalten).

Bei der Einstellung der Werte sollten Sie Folgendes beachten:

  • Das Produkt aus den beiden Werten sollte die Anzahl der Prozessorkerne des Rechenknotens nicht übersteigen.
  • Wenn Sie viele Szenarien gleichzeitig rechnen, ist es sinnvoll, die maximale Anzahl an -Instanzen zu wählen. Bei Bedarf können Sie eine Instanz weniger einstellen, so haben Sie noch einen Kern übrig, um während der Szenarioberechnung an dem Rechenknoten gut weiterarbeiten zu können. Gleichzeitig sollten Sie dann nur einen Kern pro Instanz einstellen.
  • Wenn Sie immer nur ein Szenario rechnen, ist es sinnvoll, nur eine -Instanz zu verwenden und dieser alle Kerne zuzuweisen.

Zusätzliche Dateien auf Rechenknoten kopieren

Die Basisversion, Modifikationen und Verfahrensparameter werden automatisch auf jeden Rechenknoten übertragen . Benötigen Sie noch andere Dateien oder möchten Sie nicht benötige Dateien von der Verteilung ausschließen, können Sie das über die Schaltfläche Zusätzliche Input-Dateien festlegen.

Sie müssen diese Einstellungen nur vornehmen, wenn Sie Szenarien auf anderen Rechnern als dem Hauptrechner berechnen möchten.

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Wählen Sie im Fenster Projekt bearbeiten das Register Verteiltes Rechnen.

Klicken Sie auf die Schaltfläche Zusätzliche Input-Dateien.

Das Fenster Zusätzliche Input-Dateien öffnet sich. Es werden zwei Bereiche angezeigt. Im linken Bereich geben Sie an, welche Dateien zusätzlich auf die Rechenknoten kopiert werden sollen, im rechten Bereich geben Sie an, welche Dateien nicht auf die Rechenknoten kopiert werden sollen.

Es wird nur der linke Bereich des Fensters beschrieben, der rechte Bereich funktioniert nach dem gleichen Prinzip, es werden jedoch die festgelegten Dateien nicht kopiert.

Sie können festlegen, welche Ergebnisdateien nach der Berechnung automatisch von den Rechenknoten heruntergeladen werden. Das ist von Bedeutung, wenn Sie beispielsweise ein langsames Netzwerk haben oder mit sehr großen Szenarien arbeiten, bei denen nicht alle Ergebnisdateien benötigt werden.

Sie müssen diese Einstellungen nur vornehmen, wenn Sie Szenarien auf anderen Rechnern als dem Hauptrechner berechnen möchten.

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Die Einstellungen im Fenster Ergebnisdateien werden automatisch angewendet, wenn Sie im Fenster Szenarien berechnen die Option Ergebnisdateien herunterladen ausgewählt haben. Haben Sie die Option nicht ausgewählt, werden die Einstellungen angewendet, wenn Sie im dem Register Szenarien auf das Symbol klicken, um die Ergebnisdateien herunterzuladen.

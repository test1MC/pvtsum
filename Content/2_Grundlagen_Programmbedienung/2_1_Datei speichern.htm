Datei speichern

Sie können bestimmte Daten oder Einstellungen als Datei speichern.

Klicken Sie auf das Symbol  Speichern unter.

Alternativ können Sie das Fenster öffnen, indem Sie gleichzeitig die Tasten Strg und O drücken.

Das Fenster Dateiart-Auswahl öffnet sich.

Wählen Sie die gewünschte Dateiart.

Je nach Dateiart öffnet sich ein weiteres Fenster, in dem Sie Einstellungen vornehmen müssen.

Das Fenster <Dateiart> speichern öffnet sich.

Den Pfad, der hier standardmäßig angezeigt wird, können Sie ändern (Speicherort von Dateien ändern).

Wählen Sie den gewünschten Pfad und geben Sie den gewünschten Dateinamen ein.

Die entsprechenden Daten werden in der gewünschten Dateiart gespeichert.

Wenn Sie Ihre Daten als Version unter einem neuen Namen speichern, ist anschließend automatisch die neue Versionsdatei geöffnet. Wenn Sie eine Version als Netz speichern, hängt es von Ihrer Einstellung in Dateiname als Dokumentname übernehmen ab, ob anschließend das gespeicherte Netz oder weiterhin die Version geöffnet ist (Einstellungen zum Öffnen und Speichern von Dateien vornehmen). Welche Datei geöffnet ist, erkennen Sie in der Titelleiste. Beim Speichern aller anderen Dateiarten ist nach dem Speichern weiterhin Ihre Originaldatei geöffnet.

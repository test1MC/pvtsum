Kommandozeilenparameter zum -Start festlegen

Sie können eine Verknüpfung zum Start von anlegen, in der Sie spezielle Einstellungen festlegen.

Legen Sie auf Ihrem Desktop eine Verknüpfung zur Datei ...\Programme\PTV Vision\\Exe\.exe an.

Klicken Sie mit der rechten Maustaste auf die Verknüpfung.

Wählen Sie den Eintrag Eigenschaften.

Das Fenster Eigenschaften von Visum öffnet sich.

Wählen Sie das Register Verknüpfung.

Ergänzen Sie im Eingabefeld Ziel am Ende ein Leerzeichen und dann den oder die gewünschten Parameter.

Wenn ein Pfad oder Dateiname Leerzeichen enthält, müssen Sie den Pfad in Anführungszeichen setzen.

Mehrere Parameter trennen Sie einfach durch Leerzeichen.

Parameter

Beschreibung

-f <Pfad und Name der Meldungsdatei>

Legt fest, in welche Meldungsdatei Fehlermeldungen ausgegeben werden (Mit Protokolldateien arbeiten)

-g <Pfad und Name der Grafikparameterdatei>

Öffnet die angegebene Grafikparameterdatei (Netzdarstellung drucken)

-h <Pfad und Name der Versionsdatei>

Öffnet die angegebene Versionsdatei (Versionsdatei öffnen)

Beispiel: -h „D:\ProjektA\ProjektA.ver“

-l<dreistelliges Sprachkürzel>

Startet mit der gewünschten Sprache, falls diese vorhanden ist.

Beispiel: Der Parameter -IDEU startet auf Deutsch.

Hinweis

Derzeit verfügbare Sprachen sind: Deutsch (DEU), Englisch (ENG), Französisch (FRA), Italienisch (ITA), Spanisch (ESP), Portugiesisch (POR), Polnisch (POL), Russisch (RUS), Ungarisch (MAG), Griechisch (GRE), Chinesisch (CHI), Japanisch (JAP)

-modoff<Zusatzmodul>

Schaltet ein Zusatzmodul aus

Beispiel: -modoffPUT

Hinweise

Hierfür müssen Sie die definierten Kürzel der Zusatzmodule verwenden (Zusatzmodule).

Manche Zusatzmodule sind nur gemeinsam mit anderen sinnvoll. Daher kann es sein, dass gleichzeitig weitere Zusatzmodule ausgeschaltet werden.

Sie können das Zusatzmodul nach dem Start wieder einschalten (Zusatzmodule ein- und ausschalten).

-modon<Zusatzmodul>

Schaltet ein Zusatzmodul ein, falls vorhanden

Beispiel: -modonPUT

Hinweis

Hierfür müssen Sie die definierten Kürzel der Zusatzmodule verwenden (Zusatzmodule)

-nw

Unterdrückt die Meldung beim Öffnen einer schreibgeschützten Versionsdatei

-o <Pfad und Name der Listenlayoutdatei>

Öffnet die angegebene Listenlayoutdatei (Listenlayout anpassen)

-p <Pfad und Name der Projektverzeichnisdatei>

Öffnet die angegebene Projektverzeichnisdatei (Projektverzeichnisdatei öffnen)

-reg

Registriert als COM-Server. Dabei öffnet sich eine zusätzliche Meldung.

-regs

Registriert ohne weitere Abfrage als COM-Server

-regserver

entspricht -reg

-s

Speichert die Versionsdatei unter gleichem Namen und ohne Rückfrage, beendet anschließend

-t <Pfad und Name der Protokolldatei>

Legt fest, in welche Protokolldatei die Informationen ausgegeben werden (Mit Protokolldateien arbeiten)

-u <Pfad und Name der Verfahrensparameterdatei>

Öffnet die angegebene Verfahrensparameterdatei (Verfahrensparameter in einer Datei speichern und aus Datei einlesen)

-unregserver

Löscht die Registrierung der -Instanz als COM-Server

-vj

Zeigt Fenster in größer an

-x

Führt den aktuellen Verfahrensablauf durch. Warnungen erscheinen dabei nicht, Fehler führen zum Abbruch des Ablaufs.

Die Verknüpfung wird mit Ihren Einstellungen angelegt. Wenn Sie über diese Verknüpfung starten, werden die eingegebenen Parameter berücksichtigt.

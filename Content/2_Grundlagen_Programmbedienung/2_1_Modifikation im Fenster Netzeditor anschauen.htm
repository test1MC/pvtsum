Modifikation im Fenster Netzeditor anschauen

Sie können die Inhalte einer Modifikation anschauen. Dabei öffnet sich die Modelltransferdatei, die die Modifikation enthält. Wenn Sie zuvor die gewünschte Versionsdatei öffnen, können Sie die in der Modelltransferdatei aufgelisteten Netzobjekte im Netz anzeigen.

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Öffnen Sie die Basisversion des Projekts oder eines der Szenarien.

Wählen Sie im Fenster Projekt bearbeiten das Register Modifikationen.

Wählen Sie in der Liste die gewünschten Modifikationen aus.

Klicken Sie auf das Symbol  Modifikation anzeigen.

Die Modifikation wird als Modelltransferdatei geöffnet (Inhalt einer Modelltransferdatei anzeigen). Wenn Sie die Datei mit dem Fenster Netzeditor synchronisieren, können Sie die Abweichungen gleichzeitig in beiden Fenstern anzeigen.

Netzdatei speichern

Beim Speichern einer Netzdatei werden Attributwerte mit einer bestimmten Anzahl an Nachkommastellen gespeichert (siehe „Einstellungen für Dezimalstellen und Aggregationsfunktionen“ auf Seite 1).

Klicken Sie auf das Symbol  Speichern unter.

Das Fenster Dateiart-Auswahl öffnet sich.

Wählen Sie den Eintrag Netz und bestätigen Sie mit OK.

Das Fenster Netz speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster Netz schreiben (Benutzerdefinition) öffnet sich.

Wenn Sie die Programmoberfläche in einer Sprache (beispielsweise Griechisch) anzeigen, in der die Netzdatei nicht geschrieben werden kann, wird oben im Fenster ein zusätzlicher Bereich zur Einstellung der Anzeigesprache angezeigt. Hier stellen Sie die Anzeigesprache für die Tabelle ein. Die Netzdatei wird in Englisch geschrieben.

Schränken Sie bei Bedarf die Menge der zu speichernden Daten ein.

Element

Beschreibung

Berechnete Attribute ausblenden

Ist die Option ausgewählt, werden berechnete Attribute ausgeblendet und nicht gespeichert.

Ist die Option nicht ausgewählt, werden auch berechnete Attribute gespeichert.

Tipp

Das Speichern von berechneten Attributen eignet sich für eine weitere Auswertung, beispielsweise in einer Datenbank.

Hinweis

Berechnete Attribute werden beim Einlesen ignoriert.

Attribute mit Defaultbelegung ausblenden

Ist die Option ausgewählt, werden nur ausgewählte belegte Attribute und deren Werte gespeichert.

Tipp

Mit der Option können Sie die Dateigröße reduzieren.

Ist die Option nicht ausgewählt, werden alle ausgewählten Attribute und deren Werte gespeichert, unabhängig davon, ob der Attributwert dem Standardwert entspricht oder davon abweicht.

Hinweis

Die Standardwerte der Attribute können Sie der Datei attribute.xls unter ...\Programme\PTV Vision\\Doc entnehmen.

Leere Tabellen ausblenden

Ist die Option ausgewählt, werden nur Tabellen gespeichert, die Netzobjekte enthalten.

Nur aktive Netzelemente schreiben

Ist die Option ausgewählt, werden nur die aktiven Netzobjekte der gewählten Tabellen gespeichert.

Hinweis

Knotengeometrien und Signalisierungen werden in diesem Fall nur für aktive Knoten bzw. Oberknoten gespeichert. Ob die beteiligten Strecken aktiv sind, ist nicht relevant.

Stellen Sie ein, welche Datei nach dem Speichern geöffnet sein soll.

Element

Beschreibung

Dateiname als Dokumentname übernehmen

Über die Option können Sie festlegen, welche Datei geöffnet ist, nachdem Sie die Netzdatei gespeichert haben.

Ist die Option ausgewählt, ist nach dem Speichern die gespeicherte Netzdatei geöffnet.

Ist die Option nicht ausgewählt, ist nach dem Speichern weiterhin die vorher geöffnete Datei geöffnet.

Hinweis

Die Voreinstellung dieser Option können Sie ändern (siehe „Einstellungen zum Öffnen und Speichern von Dateien vornehmen“ auf Seite 1).

Wählen Sie die Datentabellen, die Sie speichern möchten.

Element

Beschreibung

Die Tabelle ist ausgewählt und wird gespeichert.

Die Tabelle ist nicht ausgewählt und wird nicht gespeichert.

Hinweis

Die Tabelle Versionsblock kann nicht ausgeschaltet werden.

Alle Tabellen an

Über die Schaltfläche können Sie alle Tabellen auswählen.

Hinweis

Der Zustand der Attribute in den einzelnen Tabellen bleibt dabei erhalten.

Alle Tabellen aus

Über die Schaltfläche können Sie alle Tabellen bis auf Versionsblock ausschalten.

Hinweis

Der Zustand der Attribute in den einzelnen Tabellen bleibt dabei erhalten.

Häufige Fälle

Die notwendigen Daten zur Beschreibung eines Netzobjekts können in einer oder in mehreren Tabellen enthalten sein. Beispielsweise sind die Daten eines Streckenobjekts alle in der Tabelle Strecken enthalten, für digitalisierte Strecken ist zusätzlich der Block Streckenpolygone erforderlich.

Über die Schaltfläche können Sie alle Datentabellen, die zur Beschreibung bestimmter Netzobjekte nötig sind (sog. Anwendungsfälle), und deren Attribute gleichzeitig auswählen (siehe „Zur Beschreibung eines Netzobjekttyps benötigte Datentabellen für das Speichern auswählen“ auf Seite 1).

Klicken Sie auf einen Tabellennamen.

Die Attribute der Tabelle werden angezeigt.

Sie können die Attribute mehrerer Tabellen gleichzeitig anzeigen, indem Sie die Taste Strg drücken, diese gedrückt halten und nacheinander auf die gewünschten Tabellennamen klicken.

In Abhängigkeit von den Einstellungen im oberen Bereich des Fensters werden unter Umständen nicht alle Attribute einer Tabelle angezeigt.

Wählen Sie die Attribute, die Sie speichern möchten.

Element

Beschreibung

Das Attribut ist ausgewählt und wird gespeichert.

Das Attribut ist nicht ausgewählt und wird nicht gespeichert.

Das Attribut ist ein Schlüsselattribut und kann nicht ausgeschaltet werden.

Schlüsselattribute sind Attribute, die zur Identifikation der Netzobjekte dienen.

Das Attribut ist ein Pflichtattribut und kann nicht ausgeschaltet werden.

Attributgruppen

Über die Schaltfläche können Sie vordefinierte Gruppen von Attributen in einem separaten Fenster auswählen oder ausschalten (siehe „Gruppen von Attributen für das Speichern auswählen oder ausschalten“ auf Seite 1).

Standard

Über diese Schaltfläche blenden Sie leere Tabellen und berechnete Attribute aus. Alle anderen Attribute sind ausgewählt. Die Option Nur aktive Netzelemente schreiben bleibt unverändert.

Sie können mit der rechten Maustaste auf einen Tabellenbezeichner klicken und über das Kontextmenü alle aktuell angezeigten Attribute der Tabelle aktivieren oder deaktivieren.

Alternativ können Sie die Auswahl der Tabellen und Attribute aus einer Netzdatei *.net einlesen (siehe „Layout aus einer Netzdatei einlesen“ auf Seite 1).

Die Netzdaten werden gemäß der Einstellungen gespeichert.

Gruppen von Attributen für das Speichern auswählen oder ausschalten

Stellen Sie sicher, dass das Fenster Netz schreiben (Benutzerdefinition) offen ist (siehe „Netzdatei speichern“ auf Seite 1).

Klicken Sie auf die Schaltfläche Attributgruppen.

Das Fenster Gruppenauswahl benutzerdefiniertes Schreiben öffnet sich.

Wählen Sie in der Auswahlliste die vordefinierte Gruppe von Attributen, deren Attribute Sie auswählen oder ausschalten möchten.

Sie können mehrere Attributgruppen auswählen, indem Sie die Taste STRG drücken, diese gedrückt halten und auf die gewünschten Einträge klicken.

Klicken Sie auf die gewünschte Schaltfläche.

Element

Beschreibung

Einschalten

Alle Attribute der ausgewählten Gruppe werden ausgewählt.

Abschalten

Alle Attribute der ausgewählten Gruppe außer Schlüssel- und Pflichtattribute werden ausgeschaltet.

Exklusiv einschalten

Alle Attribute der ausgewählten Gruppe werden ausgewählt und alle Attribute anderer Gruppen außer Schlüssel- und Pflichtattribute werden ausgeschaltet.

Klicken Sie auf die Schaltfläche Schließen.

Die Attribute werden gemäß der Einstellungen ausgewählt oder ausgeschaltet.

Zur Beschreibung eines Netzobjekttyps benötigte Datentabellen für das Speichern auswählen

In werden die Daten zur Beschreibung von bestimmten Netzobjekttypen in mehreren Tabellen gespeichert, zum Beispiel Bezirke und zugehörige Polygone, Fahrplan- oder Umlaufdaten.

Damit Sie beim Speichern solcher Daten in die Netzdatei alle benötigten Tabellen und Attribute auswählen (sog. Anwendungsfall), können Sie eine vordefinierte, auf die zu speichernden Daten abgestimmte Auswahl von Tabellen und Attributen aktivieren.

Stellen Sie sicher, dass das Fenster Netz schreiben (Benutzerdefinition) offen ist (siehe „Netzdatei speichern“ auf Seite 1).

Klicken Sie auf die Schaltfläche Häufige Fälle.

Das Fenster Anwendungsfälle für selektives Netz schreiben öffnet sich.

Element

Beschreibung

Polygone schreiben

Ist die Option ausgewählt, werden alle Tabellen ausgewählt, die polygonbezogene Daten enthalten, und Sie können festlegen, ob die Tabellen der zugehörigen Netzobjekttypen (Bezirke, Oberbezirke, Oberknoten, Gebiete, POIs, Mautsysteme) auch gespeichert werden sollen.

Fahrplan schreiben

Ist die Option ausgewählt, werden alle Tabellen ausgewählt, die fahrplanbezogene Daten enthalten, und Sie können festlegen, ob kalenderperiode- und verkehrstagbezogene Daten gespeichert werden sollen.

Kalenderperiode und Verkehrstage schreiben

Ist die Option ausgewählt, werden die Tabellen Kalenderperiode, Verkehrstage und Ferientage gespeichert.

Hinweis

Die Option ist nur verfügbar, wenn das Netz einen Wochen- oder Jahreskalender enthält.

Umläufe schreiben

Ist die Option ausgewählt, werden die Tabellen der Umläufe und ihrer zugehörigen Daten ausgewählt.

Hinweis

Da Umläufe auf Verkehrstage verweisen, wird die Option Kalenderperiode und Verkehrstage schreiben automatisch gesetzt.

Oberknoten schreiben

Ist die Option ausgewählt, werden die Tabellen der Oberknoten und ihrer zugehörigen Daten ausgewählt.

Ausgewählte Netzdaten als Layout speichern

Sie können die aktuell ausgewählten Tabellen und Attribute als Netzdatei *.net ohne Inhalt speichern und später als Vorgabe wieder einlesen.

Stellen Sie sicher, dass das Fenster Netz schreiben (Benutzerdefinition) offen ist (siehe „Netzdatei speichern“ auf Seite 1).

Klicken Sie auf die Schaltfläche Layout speichern.

Das Fenster Netz speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Die ausgewählten Tabellen und Attribute werden als Datenblöcke ohne Inhalt gespeichert.

Layout aus einer Netzdatei einlesen

Sie können für das Speichern einer Netzdatei eine Auswahl von Tabellen und Attributen aus einer Netzdatei oder aus einer als Layout gespeicherten Netzdatei ohne Inhalt einlesen.

Stellen Sie sicher, dass das Fenster Netz schreiben (Benutzerdefinition) offen ist (siehe „Netzdatei speichern“ auf Seite 1).

Klicken Sie auf die Schaltfläche Layout öffnen.

Das Fenster Netz öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Die in der Netzdatei enthaltenen Tabellen und Attribute werden übernommen.

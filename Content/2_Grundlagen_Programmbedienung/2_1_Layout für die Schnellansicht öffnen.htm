Layout für die Schnellansicht öffnen

Sie können eine Schnellansichtslayout-Datei *.qla in Ihr geöffnetes Netz laden.

Stellen Sie sicher, dass das Fenster Schnellansicht geöffnet ist.

Klicken Sie im Fenster Schnellansicht auf das Symbol  Schnellansichtslayout öffnen.

Das Fenster Schnellansichtslayout öffnen öffnet sich.

Alternativ können Sie im Menü Datei den Eintrag Öffnen > Schnellansichtslayout wählen.

Wählen Sie die gewünschte Datei.

Die Datei wird geöffnet.

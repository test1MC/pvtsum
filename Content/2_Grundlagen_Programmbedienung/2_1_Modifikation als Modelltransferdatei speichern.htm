Modifikation als Modelltransferdatei speichern

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Wählen Sie im Fenster Projekt bearbeiten das Register Modifikationen.

Wählen Sie in der Liste die gewünschte Modifikation aus.

Klicken Sie auf das Symbol  Modifikation speichern unter.

Das Fenster Modelltransferdatei speichern öffnet sich.

Wählen Sie bei Bedarf ein anderes Verzeichnis.

Geben Sie einen Dateinamen ein.

Klicken Sie auf die Schaltfläche Speichern.

Die Modifikation wird gespeichert.

Alternativ können Sie die Modifikation als Modelltransferdatei öffnen und dann unter einem anderen Namen speichern.

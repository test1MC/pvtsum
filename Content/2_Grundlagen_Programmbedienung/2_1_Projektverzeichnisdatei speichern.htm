Projektverzeichnisdatei speichern

Sie können die Einstellungen für die Standard-Pfade der Dateiarten speichern (Speicherort von Dateien ändern).

Zeigen Sie im Menü Datei auf den Eintrag Projektverzeichnisse.

Wählen Sie den Eintrag Projektverzeichnisse speichern.

Das Fenster Projektverzeichnisse speichern öffnet sich.

Wählen Sie das gewünschte Verzeichnis und geben Sie den gewünschten Dateinamen ein.

Hier wird standardmäßig der Pfad angezeigt, den Sie für Projektverzeichnisdateien eingestellt haben (Speicherort von Dateien ändern).

Nur wenn Sie die Pfade direkt in der Standard-Projektverzeichnisdatei C:\<Benutzer>\Anwendungsdaten\Roaming\PTV Vision\\std.pfd speichern (Einstellungen in der Konfigurationsdatei ändern), werden Ihre Änderungen automatisch beim nächsten Programmstart wirksam. Ansonsten werden beim nächsten Öffnen von wieder die Einstellungen aus der Standard-Projektverzeichnisdatei gelesen. Die gespeicherte Datei können Sie bei Bedarf öffnen, um Ihre Einstellungen in einzulesen (Projektverzeichnisdatei öffnen).

Wir empfehlen Ihnen, Ihre benutzerdefinierte Projektverzeichnisdatei mit den Projektdateien in Ihrem Projektverzeichnis (Speicherort von Dateien ändern) zu speichern. So können Sie alle Projektdaten gemeinsam archivieren.

Die Datei wird gespeichert.

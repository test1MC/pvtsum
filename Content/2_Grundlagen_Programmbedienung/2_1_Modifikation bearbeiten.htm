Modifikation bearbeiten

Sie können die Änderungen an Netz- oder Nachfragedaten, die eine Modifikation enthält, bearbeiten.

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Wählen Sie im Fenster Projekt bearbeiten das Register Modifikationen.

Wählen Sie in der Liste die gewünschte Modifikation aus.

Klicken Sie auf das Symbol  Netzänderungen der Modifikation bearbeiten.

Die Versionsdatei wird geöffnet. Gleichzeitig öffnet sich das Fenster Modifikation bearbeiten.

Nehmen Sie an der Version die gewünschten Änderungen vor.

Sie können die Änderungen direkt vornehmen. Sie können aber beispielsweise auch eine vorhandene Modelltransferdatei einlesen (Modelltransferdatei anwenden).

Klicken Sie im Fenster Modifikation bearbeiten auf die Schaltfläche Fertigstellen.

Die Änderungen werden als Modelltransferdatei gespeichert. Das Fenster Projekt bearbeiten öffnet sich wieder. Sie können jetzt weitere Änderungen am Projekt vornehmen.

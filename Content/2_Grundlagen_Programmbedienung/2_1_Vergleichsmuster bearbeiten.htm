Vergleichsmuster bearbeiten

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Wählen Sie im Fenster Projekt bearbeiten das Register Vergleichsmuster.

Über das Symbol  Attributauswahl können Sie Spalten ein- und ausblenden (Fenster zur Auswahl mehrerer Attribute).

Wählen Sie in der Liste das gewünschte Vergleichsmuster aus.

Klicken Sie auf das Symbol   Vergleichsmuster bearbeiten.

Das Fenster Vergleichsmuster bearbeiten öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (Vergleichsmuster anlegen).

Das Vergleichsmuster wird geändert.

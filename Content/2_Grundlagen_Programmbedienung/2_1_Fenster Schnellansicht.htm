Fenster Schnellansicht

Das Fenster Schnellansicht enthält eine Auswahl von Attributen und Attributwerten des oder der aktuell markierten Objekte. Hier können Sie Attributwerte der markierten Objekte direkt ändern (siehe „Attributwerte in der Schnellansicht ändern“ auf Seite 1).

Welche Attribute angezeigt werden, können Sie einstellen (siehe „Attribute für die Schnellansicht auswählen“ auf Seite 1). Sie können außerdem verschiedene Anzeigeeigenschaften ändern (siehe „Einstellungen für die Schnellansicht ändern“ auf Seite 1).

In der Symbolleiste des Fensters Schnellansicht stehen Ihnen folgende Symbole zur Verfügung.

Symbol

Name

Beschreibung

Attributauswahl

Attribute bestimmen, die im Fenster Schnellansicht angezeigt werden sollen (siehe „Attribute für die Schnellansicht auswählen“ auf Seite 1).

Attribute kopieren

Werte der aktuell angezeigten Eingabeattribute (außer Schlüsselattribute) in die Zwischenablage kopieren (siehe „Attributwerte in die Zwischenablage kopieren“ auf Seite 1).

Hinweis

Die Funktion ist nur verfügbar, wenn ein einzelnes Netzobjekt markiert ist.

Attribute einfügen

Attributwerte aus der Zwischenablage einer beliebigen Anzahl Netzobjekte desselben oder eines anderen Netzobjekttyps zuweisen (siehe „Attributwerte aus der Zwischenablage einfügen“ auf Seite 1).

Schnellansichtslayout öffnen

Öffnet eine Schnellansichtslayout-Datei (siehe „Layout für die Schnellansicht öffnen“ auf Seite 1)

Schnellansichtslayout speichern

Speichert eine Schnellansichtslayout-Datei (siehe „Layout für die Schnellansicht speichern“ auf Seite 1)

Der oberste Eintrag <Netzobjekttyp> mit einer Zahl gibt an, wie viele Objekte des genannten Netzobjekttyps aktuell markiert sind.

Sind mehrere Objekte markiert, deren Attributwerte voneinander abweichen, wird in der jeweiligen Zeile ein * angezeigt, wenn die Aggregationsfunktion Vergleichen ausgewählt ist. Wollen Sie in der Schnellansicht beispielsweise die Gesamtlänge aller im Fenster Netzeditor markierten Strecken berechnen, wählen Sie die Aggregationsfunktion Summe (siehe „Anzeige-Eigenschaften von Attributen ändern“ auf Seite 1).

Sie können die Position, die Größe und die Sichtbarkeit des Fensters Schnellansicht anpassen (siehe „Darstellung von Fenstern ändern“ auf Seite 1).

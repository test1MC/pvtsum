ohne Oberfläche starten

Sie können für COM-Anwendungen ohne Oberfläche verwenden.

Legen Sie auf Ihrem Desktop eine Verknüpfung zur Datei ...\Programme\PTV Vision\\Exe\.exe an.

Klicken Sie mit der rechten Maustaste auf die Verknüpfung.

Wählen Sie den Eintrag Eigenschaften.

Das Fenster Eigenschaften von Visum öffnet sich.

Wählen Sie das Register Verknüpfung.

Ergänzen Sie im Eingabefeld Ziel am Ende ein Leerzeichen und anschließend den Parameter -modoffUI.

Die Verknüpfung wird mit Ihren Einstellungen angelegt. Wenn Sie auf diese Verknüpfung doppelklicken, startet ohne Benutzeroberfläche. Auf Ihrer Windows-Oberfläche sehen Sie rechts unten in der Windows-Symbolleiste das -Symbol.

Wenn Sie mit der rechten Maustaste auf das -Symbol in der Windows-Symbolleiste klicken, öffnet sich ein Kontextmenü, über das Sie beenden können.

Die Einstellung, ohne Oberfläche zu starten, wird beim Beenden in der Registry gespeichert. Daher müssen Sie explizit festlegen, wenn Sie wieder mit Oberfläche starten möchten. Dafür gibt es zwei Möglichkeiten:

  • Sie klicken mit der rechten Maustaste auf das -Symbol in der Windows-Symbolleiste und wählen den Eintrag Sichtbar nach Restart.
  • Sie starten mit dem Parameter -modonUI.

Alternativ können Sie auch mit Hilfe von COM ohne Oberfläche starten. Dafür müssen Sie die gewünschte -Instanz als COM-Server registrieren (-Version als COM-Server registrieren).

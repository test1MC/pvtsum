Ansicht je Analysezeitintervall anzeigen

Wenn Sie Analysezeitintervalle definiert haben (Analysezeitintervalle festlegen), können Sie Attributwerte, die sich über die Zeit ändern, pro Analysezeitintervall anzeigen. In der Netzdarstellung entspricht jedem Analysezeitintervall eine Ansicht. Sie können zwischen diesen Ansichten manuell oder automatisch vor- und zurückschalten.

Damit die zeitabhängigen Attributwerte überhaupt grafisch dargestellt werden, müssen Sie die entsprechenden Grafikparameter festlegen (Grundeinstellungen für das Netzfenster vornehmen).

Wählen Sie im Menü Ansicht > Symbolleisten > Analysezeitintervalle die Option Sichtbar oder Automatisch aus.

Wenn die Option Automatisch ausgewählt ist, wird die Symbolleiste Analysezeitintervalle nur dann eingeblendet, wenn Sie Analysezeitintervalle definiert haben.

Wählen Sie in der Symbolleiste Analysezeitintervalle in der Auswahlliste das Analysezeitintervall, das Sie anzeigen möchten.

Die Ansicht für das gewählte Analysezeitintervall wird angezeigt.

Klicken Sie in der Symbolleiste Analysezeitintervalle auf das gewünschte Symbol.

Dateibasiertes Projekt in SQL Serverdatenbank konvertieren

Wenn Sie ein bestehendes dateibasiertes Projekt in eine SQL Serverdatenbank konvertieren möchten, gehen Sie folgendermaßen vor.

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (siehe „Grundeinstellungen eines Projekts bearbeiten“ auf Seite 1).

Wählen Sie im Fenster Projekt bearbeiten das Register Mehrbenutzerbetrieb.

Klicken Sie auf die Schaltfläche Konvertieren.

Das Fenster Datenbankverbindung bearbeiten öffnet sich.

Nehmen Sie die gewünschten Einstellungen für die Datenbankverbindung vor (siehe Projekt für das Szenariomanagement anlegen auf Seite 1).

Das Projekt wird in einer SQL Serverdatenbank gespeichert.

Im Register Mehrbenutzerbetrieb werden alle angemeldeten Benutzer und gesperrten Objekte aufgelistet. Durch den Benutzer gesperrte Objekte sind blau hinterlegt. Benutzer und gesperrte Objekte können über die jeweilige Schaltfläche gelöscht werden.

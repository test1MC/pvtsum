Aktivitätsprotokoll [Versionsname].trc

Pro Operation wird standardmäßig eine Zeile geschrieben. Jede Zeile besteht aus 5 Einträgen, die durch TAB getrennt sind: 

<Zeit> <Satzart> <Benutzer> <Bezugsobjekt> <Beschreibung>
  • <Zeit>

Das Feld <Zeit> enthält den Zeitstempel der Ausführung (lokale Zeit) im Format YYYY-MM-DD hh:mm:ss. Generell wird vor dem Ausführen einer Operation der passende Eintrag geschrieben, damit Abstürze zugeordnet werden können.

  • <Satzart>

Das Feld <Satzart> kann folgende Werte enthalten:

Versionsvergleiche bearbeiten

Sie können einen Versionsvergleich bearbeiten, der Attribute aus Vergleichsnetzen übernimmt. Dabei können Sie Attribute zu dem bestehenden Versionsvergleich hinzufügen oder entfernen.

Zeigen Sie im Menü Datei auf Netze vergleichen und überführen.

Wählen Sie den Eintrag Versionsvergleiche.

Das Fenster Versionsvergleiche öffnet sich.

Wählen Sie den gewünschten Versionsvergleich.

Klicken Sie auf die Schaltfläche  Bearbeiten.

Das Fenster Versionsvergleich erstellen (Benutzerdefinition) öffnet sich.

Ändern Sie die aktuelle Attributauswahl wie gewünscht.

Klicken Sie auf die Schaltfläche OK.

Wenn Sie Attribute hinzugefügt haben, öffnet sich eine Sicherheitsabfrage.

Bestätigen Sie die Sicherheitsabfrage mit Ja.

Die Attribute werden aktualisiert.

Verfahrensparametersatz anlegen

Sie können einen Satz Verfahrensparameter anlegen. Dabei verwenden Sie zunächst die Parameter der aktuell geöffneten Versionsdatei. Sie können aber auch die Parameter aus einer Verfahrensparameterdatei *.par, *.xml oder einer Skriptdatei *.vbs,*.py, *.pys, *.js, *.rb, *.pl und *.tcl verwenden.

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Wählen Sie im Fenster Projekt bearbeiten das Register Verfahrensparametersätze.

Klicken Sie auf das Symbol   Verfahrensparametersatz einfügen.

Alternativ können Sie über das Symbol  Verfahrensparametersatz duplizieren einen vorhandenen Satz kopieren.

Das Fenster Verfahrensparametersatz einfügen öffnet sich.

Element

Beschreibung

Code

Kurzer Name für den Parametersatz

Beschreibung

Beschreibung für den Parametersatz

aktuell geladene Verfahrensparameter verwenden

Ist die Option ausgewählt, enthält der neue Satz die Verfahrensparameter der aktuell geöffneten Versionsdatei.

existierende Verfahrensparameter oder Skriptdatei verwenden

Ist die Option ausgewählt, können Sie über die Schaltfläche eine Datei mit Verfahrensparametern (*.par, *.xml) oder eine Skriptdatei (*.vbs,*.py, *.pys, *.js, *.rb, *.pl und *.tcl) verwenden (Verfahrensparameterdatei speichern).

Hinweis

Ist eine Skriptdatei ausgewählt, in die andere Skripte eingebunden sind, müssen Sie die eingebundenen Skripte manuell in den Ordner des Hauptskripts kopieren.

Der Verfahrensparametersatz wird als neue Zeile eingefügt. Die Verfahrensparameter können Sie anschließend weiter bearbeiten (Verfahrensparametersatz bearbeiten).

Nachdem Sie die Verfahrensparameter festgelegt haben, müssen Sie sie dem gewünschten Szenario zuordnen (Szenario anlegen).

Versionsvergleiche aktualisieren

Sie können einen Versionsvergleich aktualisieren. Dabei werden die Werte der gleichen Attribute aus der gleichen Versionsdatei neu eingelesen.

Zeigen Sie im Menü Datei auf Netze vergleichen und überführen.

Wählen Sie den Eintrag Versionsvergleiche.

Das Fenster Versionsvergleiche öffnet sich.

Wählen Sie die gewünschten Versionsvergleiche.

Klicken Sie auf die Schaltfläche  Aktualisieren.

Das Fenster Versionsvergleich aktualisieren öffnet sich.

Überprüfen Sie im Feld Versionsdatei den angegebenen Pfad und ändern Sie ihn bei Bedarf.

Die Überprüfung des Pfads ist wichtig, wenn Sie die zu vergleichenden Versionen beispielsweise von einer externen Quelle erhalten haben.

Die Versionsvergleiche werden aktualisiert.

Sie können das Attribut IstInAuswahl in der Vergleichsversion nicht aktualisieren. Alle passiven Objekte der Vergleichsversion bleiben passiv, auch wenn in der aktuellen Version Objekte auf IstInAuswahl gesetzt werden. Weicht das Vergleichsnetz vom aktuellen Netz ab, weist Sie darauf hin.

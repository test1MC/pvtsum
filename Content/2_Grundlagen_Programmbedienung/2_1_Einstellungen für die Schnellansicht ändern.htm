Einstellungen für die Schnellansicht ändern

Sie können verschiedene Einstellungen für die Schnellansicht ändern.

Wählen Sie im Menü Bearbeiten den Eintrag Benutzereinstellungen.

Das Fenster Benutzereinstellungen öffnet sich.

Wählen Sie im Navigator den Eintrag Benutzeroberfläche > Schnellansicht.

Element

Beschreibung

Schrift für Schnellansicht

Schriftart wählen

Über die Schaltfläche können Sie in einem separaten Fenster eine Schrift, einen Schriftschnitt und -grad für die Schnellansicht festlegen.

Hinweise

Auf der Schaltfläche ist in Klammern die aktuell gewählte Schriftart angezeigt.

Sie können die Schriftart wieder auf die -Standardeinstellung zurücksetzen (Schriftarten zurücksetzen).

Zeilenbeschriftungen

Hier können Sie auswählen, ob die Attribute als Kurz- oder Langnamen angezeigt werden.

Sonstiges

Erweiterte Eingabemöglichkeiten

Ist die Option ausgewählt, haben Sie folgenden Eingabemöglichkeiten:

  • Logische Werte werden über aktivierte/deaktivierte Kontrollkästchen gesteuert, beispielsweise Streckenorientierung automatisch vergeben.
  • Werte einer Menge können in Auswahllisten ausgewählt werden, beispielsweise Steuerungstyp.
  • Werte einer Menge, die die Auswahl mehrerer Elemente erlaubt, werden als mit der Auswahl beschriftete Schaltflächen dargestellt, über die sich ein Fenster öffnet. Das Fenster ermöglicht eine Mehrfachauswahl, beispielsweise VSysSet.

Werte für benutzerdefinierte Attribute vom Typ Datei werden als Schaltfläche dargestellt, die mit dem Namen der ausgewählten Datei beschriftet ist. Über die Schaltfläche öffnet sich das Fenster Datei öffnen, in dem eine andere Auswahl getroffen werden kann.

Die Schnellansicht wird mit den gewählten Einstellungen angezeigt.

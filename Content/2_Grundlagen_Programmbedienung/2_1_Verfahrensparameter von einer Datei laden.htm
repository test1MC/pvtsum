Verfahrensparameter von einer Datei laden

Sie können einen Verfahrensparametersatz durch die Verfahrensparameter aus einer Datei überschreiben.

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Wählen Sie im Fenster Projekt bearbeiten das Register Verfahrensparametersätze.

Wählen Sie in der Liste den gewünschten Verfahrensparametersatz aus.

Klicken Sie auf das Symbol   Externe Datei für Verfahrensparametersatz auswählen.

Das Fenster Verfahrensparameter öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Die Verfahrensparameter werden in den gewählten Verfahrensparametersatz kopiert.

Einem Szenario Modifikationen zuordnen

Damit sich das Modell eines Szenarios von dem der Basisversion unterscheidet, müssen Sie dem Szenario Modifikationen zuordnen. Sie können einem Szenario nur Modifikationen zuordnen, die kombinierbar sind. Dies können Sie zuvor prüfen (Kombinierbarkeit von Modifikationen prüfen).

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Wählen Sie im Fenster Projekt bearbeiten das Register Szenarien.

Klicken Sie in der Zeile des gewünschten Szenarios in der Spalte Modifikationen auf die Schaltfläche.

Das Fenster Modifikationen öffnet sich.

Wählen Sie die gewünschten Modifikationen aus.

Die gewählten Modifikationen werden dem Szenario zugeordnet. Die Nummern der gewählten Modifikationen werden auf der Schaltfläche aufsteigend angezeigt.

Die Schaltfläche zeigt nur die gewählten Modifikationen an. Falls eine dieser Modifikationen von einer anderen abhängig ist, wird diese automatisch auch dem Szenario zugeordnet. Eine Liste aller zugeordneten Modifikationen finden Sie in der Spalte Modifikationen (vollständig). Die Spalte Modifikationen (vollständig in Ladereihenfolge) zeigt alle Modifikationen des Szenarios in der Reihenfolge an, in der sie geladen werden. Die Anwendungsreihenfolge legen Sie im Register Modifikationen fest (Anwendungsreihenfolge der Modifikationen festlegen).

Projekt aus SQL Serverdatenbank in dateibasiertes Projekt konvertieren

Wenn Sie ein mehrbenutzerfähiges SQL Serverdatenbank-Projekt in ein dateibasiertes Projekt umwandeln möchten, gehen Sie folgendermaßen vor.

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (siehe „Grundeinstellungen eines Projekts bearbeiten“ auf Seite 1).

Wählen Sie im Fenster Projekt bearbeiten das Register Mehrbenutzerbetrieb.

Klicken Sie auf die Schaltfläche Konvertieren.

Das Projekt wird in einer Datei *.vpdb als SQL-CE-Datenbank gespeichert. Sie können das Projekt nun auf einen Rechner kopieren, von dem Sie keinen Zugriff auf die SQL-Server-Instanz haben.

Standardmäßig schlägt das Verzeichnis \Backups Ihres Projekts vor. Außer der Datei *.vpdb speichert auch eine Datei *.vpdbx in das ausgewählte Verzeichnis. Bevor Sie das Projekt erneut über die neue Datei *.vpdbx öffnen, müssen Sie diese Datei in das Verzeichnis kopieren, in dem sich die Basisversion befindet.

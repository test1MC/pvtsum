Mit einer leeren Oberfläche starten

Bevor Sie mit einer leeren Oberfläche starten, sollten Sie Ihre bisherige Arbeit speichern (Dateien öffnen und speichern).

Klicken Sie in der Symbolleiste Funktionen auf das Symbol  Neu.

Sie können eine neue Version auch über das Menü Datei > Neu anlegen oder indem Sie die Tastenkombination Strg+N drücken.

Wenn Sie Änderungen vorgenommen und diese noch nicht gespeichert haben, öffnet sich eine Sicherheitsabfrage.

Bestätigen Sie die Sicherheitsabfrage mit OK.

Eine leere Oberfläche wird geöffnet.

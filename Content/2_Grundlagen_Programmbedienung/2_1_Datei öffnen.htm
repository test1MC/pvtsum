Das Fenster <Dateiart> öffnen öffnet sich.

Wählen Sie den gewünschten Pfad und Dateinamen.

Den Pfad, der hier standardmäßig angezeigt wird, können Sie ändern (Speicherort von Dateien ändern).

Klicken Sie auf die Schaltfläche Öffnen.

Falls Sie eine Version oder ein Netz öffnen und Ihre Daten zuvor nicht gespeichert haben, öffnet sich eine Sicherheitsabfrage.

Abhängig von der gewählten Dateiart können sich an dieser Stelle noch weitere Fenster öffnen, in denen Sie Einstellungen vornehmen. Informationen finden Sie in den jeweiligen Abschnitten.

Bestätigen Sie die Sicherheitsabfrage gegebenenfalls mit OK.

Die Datei wird geöffnet.

Die Auswirkungen hängen von der gewählten Dateiart und Ihren Einstellungen ab.

  • Versionsdatei: Die zuvor geöffnete Datei wird geschlossen und die Version wird geöffnet. Der Name der Version wird in der Titelleiste angezeigt (Versionsdatei öffnen).
  • Netzdatei: Eine Netzdatei können Sie öffnen oder additiv einlesen. Im ersten Fall wird die zuvor geöffnete Datei geschlossen und die Netzdatei geöffnet. Im zweiten Fall wird das Netz zur geöffneten Datei ergänzt (Netzdatei öffnen und speichern und Kommentare ergänzen).
  • Netzvereinigungsparameter: Die zuvor geöffnete Datei wird geschlossen und eine Kombination zweier Versionen wird geöffnet. Die Namen der beiden Versionen werden in der Titelleiste angezeigt (Spinnen anzeigen).
  • Alle anderen Dateiarten: Die Daten werden zur geöffneten Datei eingelesen.

Sie können eine Datei auch öffnen, indem Sie im Menü Datei in der Liste der zuletzt geöffneten Dateien auf die gewünschte Datei klicken. Oder Sie ziehen eine Datei mit der Maus vom Windows-Explorer auf Ihre Programmoberfläche.

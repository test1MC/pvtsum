Vergleichswerte in benutzerdefinierte Attribute kopieren

Sie können die Werte von ausgewählten oder allen verglichenen Attributen des Vergleichsnetzes in neue benutzerdefinierte Attribute speichern. So bleiben die Werte erhalten, nachdem Sie den Versionsvergleich beendet haben.

Sie gehen bei den beiden Möglichkeiten des Versionsvergleichs unterschiedlich vor.

Benutzerdefinierte Attribute bei einem Versionsvergleich mit Übernahme direkter Attribute erstellen

Zeigen Sie im Menü Datei auf Netze vergleichen und überführen.

Wählen Sie den Eintrag Versionsvergleiche.

Das Fenster Versionsvergleiche öffnet sich.

Wählen Sie den gewünschten Versionsvergleich oder die gewünschten Attribute.

Klicken Sie auf die Schaltfläche  In benutzerdef. Attr. kopieren.

Die Versionsvergleiche oder Attribute werden in neue benutzerdefinierte Attribute gespeichert.

Da benutzerdefinierte Attribute keine Subattribute besitzen können, wird für jedes Subattribut ein benutzerdefiniertes Attribut angelegt.

ID und Kurzname der neuen benutzerdefinierten Attribute werden aus dem ursprünglichen Kurznamen des Attributs gebildet, zusätzlich wird der Name des Versionsvergleichs ergänzt. Existiert die erzeugte ID schon, wird eine Fehlermeldung angezeigt und dieses Attribut wird nicht angelegt.
Benutzerdefinierte Attribute bei einem Versionsvergleich mit Vergleichsnetz im Hintergrund erstellen

Zeigen Sie im Menü Datei auf Netze vergleichen und überführen.

Wählen Sie den Eintrag Versionsvergleiche.

Das Fenster Versionsvergleiche öffnet sich.

Wählen Sie den gewünschten Versionsvergleich oder die gewünschten Attribute.

Klicken Sie auf die Schaltfläche  In benutzerdef. Attr. kopieren.

Das Fenster Daten in benutzerdefinierte Attribute kopieren öffnet sich.

Da benutzerdefinierte Attribute keine Subattribute besitzen können, wird für jedes Subattribut ein benutzerdefiniertes Attribut angelegt.

ID und Kurzname der neuen benutzerdefinierten Attribute werden aus dem ursprünglichen Kurznamen des Attributs gebildet, zusätzlich wird der Name des Versionsvergleichs ergänzt. Existiert die erzeugte ID schon, wird eine Fehlermeldung angezeigt und dieses Attribut wird nicht angelegt.

Schränken Sie bei Bedarf die Menge der zu kopierenden Daten ein.

Es werden ausschließlich direkte Attribute zur Auswahl angezeigt.

Element

Beschreibung

Berechnete Attribute ausblenden

Ist die Option ausgewählt, werden zusätzlich zu Schlüsselattributen nur ausgewählte lesbare Attribute und deren Werte kopiert.

Ist die Option nicht ausgewählt, werden sowohl lesbare als auch nicht lesbare Attribute und deren Werte kopiert.

Attribute mit Defaultbelegung ausblenden

Ist die Option ausgewählt, werden nur ausgewählte belegte Attribute und deren Werte verglichen.

Ist die Option nicht ausgewählt, werden alle ausgewählten Attribute und deren Werte kopiert, unabhängig davon, ob der Attributwert dem Standardwert entspricht oder davon abweicht.

Hinweis

Die Standardwerte der Attribute können Sie der Datei Attribute.xls unter ...\Programme\PTV Vision\\Doc entnehmen.

Leere Tabellen ausblenden

Ist die Option ausgewählt, werden nur Tabellen kopiert, die Netzobjekte enthalten.

Wählen Sie die Netzobjekttypen, die Sie in benutzerdefinierte Attribute kopieren möchten.

Element

Beschreibung

Der Netzobjekttyp wird kopiert.

Der Netzobjekttyp wird nicht kopiert.

Layout speichern

Über die Schaltfläche können Sie Ihre Auswahl als Netzdatei *.net speichern (Ausgewählte Netzdaten als Layout speichern).

Layout öffnen

Über die Schaltfläche können Sie eine gespeicherte Auswahl aus einer Netzdatei *.net einlesen (Layout aus einer Netzdatei einlesen).

Alle Tabellen an

Über die Schaltfläche können Sie alle Netzobjekttypen auswählen.

Alle Tabellen aus

Über die Schaltfläche können Sie alle Netzobjekttypen ausschalten, die Attribute bleiben jedoch ausgewählt.

Klicken Sie auf einen Netzobjekttyp.

Die Attribute und gegebenenfalls Subattribute des Netzobjekttyps werden angezeigt.

Sie können die Attribute mehrerer Netzobjekttypen gleichzeitig anzeigen, indem Sie die Taste Strg drücken, diese gedrückt halten und nacheinander auf die gewünschten Einträge klicken.

Abhängig von den Einstellungen im oberen Bereich des Fensters werden unter Umständen nicht alle Attribute eines Netzobjekttyps angezeigt.

Wählen Sie die Attribute und Subattribute, die Sie kopieren möchten.

Element

Beschreibung

Das Attribut ist ausgewählt und wird kopiert.

Das Attribut ist nicht ausgewählt und wird nicht kopiert.

Attributgruppen

Über die Schaltfläche können Sie vordefinierte Gruppen von Attributen in einem separaten Fenster auswählen oder ausschalten (Gruppen von Attributen für den Vergleich auswählen oder ausschalten).

Standard

Über die Schaltfläche wird das Fenster auf die Standardeinstellung zurückgesetzt.

Sie können mit der rechten Maustaste auf einen Netzobjekttyp klicken und über das Kontextmenü alle aktuell angezeigten Attribute des Netzobjekttyps aktivieren oder deaktivieren.

Die Attribute werden in benutzerdefinierte Attribute kopiert.

Versionsdateien vergleichen

Öffnen Sie die gewünschte Versionsdatei.

Zeigen Sie im Menü Datei auf Netze vergleichen und überführen.

Wählen Sie den Eintrag Versionsvergleiche.

Das Fenster Versionsvergleiche öffnet sich.

Klicken Sie auf die Schaltfläche  Hinzufügen.

Das Fenster Versionsvergleich erstellen öffnet sich.

Element

Beschreibung

Art des Versionsvergleichs

Über die Optionen legen Sie fest, welche Variante des Versionsvergleichs Sie durchführen.

Attribute aus Vergleichsnetz übernehmen

Ist diese Option ausgewählt, werden die Attribute des Vergleichsnetzes übernommen und verbleiben beim Speichern in der Version. Im Fenster Versionsvergleiche wird dem Code des Versionsvergleichs das Wort Attribute vorangestellt.

Vergleichsnetz im Hintergrund laden

Ist diese Option ausgewählt, sind alle Attribute über eine Relation zum Vergleichsnetz zugänglich, solange dieses Netz im Hintergrund geladen ist. Die Attribute des Vergleichsnetzes werden nicht in der Version gespeichert, sind jedoch beim erneuten Öffnen wieder sichtbar. Dazu muss das Vergleichsnetz an der Stelle im Verzeichnis gefunden werden, die Sie unter Versionsdatei ausgewählt haben. Den Pfad zum Vergleichsnetz können Sie bearbeiten, wenn Sie den Vergleich aktualisieren (Versionsvergleiche aktualisieren).

Im Fenster Versionsvergleiche wird dem Code des Versionsvergleichs das Wort Vergleichsnetz vorangestellt.

Hinweis

Die Unterschiede zwischen den zwei Arten des Versionsvergleichs werden ausführlich in den Grundlagen beschrieben Siehe "Versionsdateien vergleichen".

Versionsdatei

Über das Symbol wählen Sie die Versionsdatei, mit der Sie die aktuelle Version vergleichen möchten.

Code

Eindeutiger Code für den Versionsvergleich

Hinweise

Der Code wird für die IDs und Attributsnamen verwendet. Er darf keine Leerzeichen enthalten.

Dieselben Vergleichsattribute verwenden wie bei

Ist die Option ausgewählt, können Sie über die Auswahlliste einen schon vorhandenen Versionsvergleich wählen, dessen Vergleichsattribute verwendet werden.

Ist die Option ausgewählt, öffnet sich das nächste Fenster zur Auswahl der Attribute nicht.

Hinweis

Die Option ist nur verfügbar, wenn Sie die Option Attribute aus Vergleichsnetz übernehmen ausgewählt haben.

Matrizen nicht übernehmen

Ist die Option ausgewählt, werden keine Matrizen aus dem Vergleichsnetz übernommen, der benötigte Speicherplatz wird dadurch verringert.

Hinweis

Diese Option wird nur verfügbar, wenn Sie die Option Vergleichsnetz im Hintergrund laden ausgewählt haben.

Streckenbalken automatisch auf Differenzen umstellen

Ist die Option ausgewählt, werden Skalierungs- und Klassifizierungsattribute von Streckenbalken automatisch auf das durch den Versionsvergleich neu erzeugte Attribut mit der Differenz zwischen dem ursprünglichen und dem neu hinzugefügten Attributwert umgestellt.

Das Fenster Versionsvergleich erstellen (Benutzerdefinition) öffnet sich.

Das Fenster wird nicht angezeigt, wenn Sie die Option Dieselben Vergleichsattribute verwenden wie bei oder als Versionsvergleich die Option Vergleichsnetz im Hintergrund laden ausgewählt haben.

Schränken Sie bei Bedarf die Menge der zu vergleichenden Daten ein.

Element

Beschreibung

Berechnete Attribute ausblenden

Ist die Option ausgewählt, werden zusätzlich zu Schlüsselattributen nur ausgewählte lesbare Attribute und deren Werte verglichen.

Ist die Option nicht ausgewählt, werden sowohl lesbare als auch nicht lesbare Attribute und deren Werte verglichen.

Attribute mit Defaultbelegung ausblenden

Ist die Option ausgewählt, werden nur ausgewählte belegte Attribute und deren Werte verglichen.

Ist die Option nicht ausgewählt, werden alle ausgewählten Attribute und deren Werte verglichen, unabhängig davon, ob der Attributwert dem Standardwert entspricht oder davon abweicht.

Hinweis

Die Standardwerte der Attribute können Sie der Datei Attribute.xls unter ...\Programme\PTV Vision\\Doc entnehmen.

Leere Tabellen ausblenden

Ist die Option ausgewählt, werden nur Tabellen verglichen, die Netzobjekte enthalten.

Wählen Sie die Netzobjekttypen, die Sie vergleichen möchten.

Element

Beschreibung

Der Netzobjekttyp wird für den Vergleich verwendet.

Der Netzobjekttyp wird für den Vergleich nicht verwendet.

Alle Tabellen an

Über die Schaltfläche können Sie alle Netzobjekttypen auswählen.

Alle Tabellen aus

Über die Schaltfläche können Sie alle Netzobjekttypen und deren Attribute ausschalten.

Klicken Sie auf einen Netzobjekttyp.

Die Attribute und gegebenenfalls Subattribute des Netzobjekttyps werden angezeigt.

Sie können die Attribute mehrerer Netzobjekttypen gleichzeitig anzeigen, indem Sie die Taste Strg drücken, diese gedrückt halten und nacheinander auf die gewünschten Einträge klicken.

Abhängig von den Einstellungen im oberen Bereich des Fensters werden unter Umständen nicht alle Attribute eines Netzobjekttyps angezeigt.

Wählen Sie die Attribute und Subattribute, die Sie vergleichen möchten.

Element

Beschreibung

Das Attribut ist ausgewählt und wird verglichen.

Das Attribut ist nicht ausgewählt und wird nicht verglichen.

Das Attribut ist ein Schlüsselattribut und kann nicht ausgeschaltet werden.

Das Attribut ist ein Pflichtattribut.

Attributgruppen

Über die Schaltfläche können Sie vordefinierte Gruppen von Attributen in einem separaten Fenster auswählen oder ausschalten (siehe „Gruppen von Attributen für den Vergleich auswählen oder ausschalten“ auf Seite 100).

Standard

Über die Schaltfläche blenden Sie leere Tabellen aus und berechnete Attribute ein. Die Auswahl der Attribute wird auf die Standardeinstellung zurückgesetzt.

Sie können mit der rechten Maustaste auf einen Netzobjekttyp klicken und über das Kontextmenü alle aktuell angezeigten Attribute des Netzobjekttyps aktivieren oder deaktivieren.

Alternativ können Sie die Auswahl der Netzobjekttypen und Attribute aus einer Netzdatei *.net einlesen (siehe „Layout aus einer Netzdatei einlesen“ auf Seite 101).

Der Versionsvergleich wird durchgeführt. Im Fenster Versionsvergleiche wird der Versionsvergleich als Eintrag angezeigt.

Bei Versionsvergleichen der Art Attribute aus Vergleichsnetz übernehmen wird vor dem Eintrag ein Pluszeichen angezeigt. Wenn Sie auf das Pluszeichen klicken, werden die Attribute des Versionsvergleichs angezeigt.

Wenn Sie den Mauszeiger auf den Eintrag stellen, erhalten Sie in einer Quickinfo zusätzliche Informationen zum Zeitpunkt des Versionsvergleichs und zum Datum der zum Vergleich verwendeten Versionsdatei.

Auf die gleiche Weise können Sie weitere Versionsvergleiche hinzufügen.

Die Versionsdatei können Sie einschließlich des Versionsvergleichs speichern.
Gruppen von Attributen für den Vergleich auswählen oder ausschalten

Stellen Sie sicher, dass das Fenster Versionsvergleich erstellen (Benutzerdefinition) offen ist (siehe „Versionsdateien vergleichen“ auf Seite 97).

Klicken Sie auf die Schaltfläche Attributgruppen.

Das Fenster Gruppenauswahl benutzerdefiniertes Schreiben öffnet sich.

Wählen Sie in der Auswahlliste die vordefinierte Gruppe von Attributen, deren Attribute Sie auswählen oder ausschalten möchten.

Klicken Sie auf die gewünschte Schaltfläche.

Element

Beschreibung

Einschalten

Alle Attribute der ausgewählten Gruppe werden ausgewählt.

Abschalten

Alle Attribute der ausgewählten Gruppe außer Schlüssel- und Pflichtattribute werden ausgeschaltet.

Exklusiv einschalten

Alle Attribute der ausgewählten Gruppe werden ausgewählt und alle Attribute anderer Gruppen außer Schlüssel- und Pflichtattribute werden ausgeschaltet.

Klicken Sie auf die Schaltfläche Schließen.

Die Attribute werden gemäß der Einstellungen ausgewählt oder ausgeschaltet.

Ausgewählte Netzdaten als Layout speichern

Sie können die aktuell ausgewählten Tabellen und Attribute als Netzdatei *.net ohne Inhalt speichern und später als Vorgabe wieder einlesen.

Stellen Sie sicher, dass das Fenster Versionsvergleich erstellen (Benutzerdefinition) offen ist (siehe „Versionsdateien vergleichen“ auf Seite 97).

Klicken Sie auf die Schaltfläche Layout speichern.

Das Fenster Netz speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Die ausgewählten Tabellen und Attribute werden als Datenblöcke ohne Inhalt gespeichert.

Layout aus einer Netzdatei einlesen

Sie können für den Versionsvergleich eine vorgegebene Auswahl von Netzobjekttypen und Attributen aus einer Netzdatei oder aus einer als Layout gespeicherten Netzdatei ohne Inhalt einlesen.

Stellen Sie sicher, dass das Fenster Versionsvergleich erstellen (Benutzerdefinition) offen ist.

Klicken Sie auf die Schaltfläche Layout öffnen.

Das Fenster Netz öffnen öffnet sich.

Wählen Sie die gewünschte Netzdatei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Die in der Netzdatei enthaltene Auswahl von Netzobjekttypen und Attributen wird übernommen.

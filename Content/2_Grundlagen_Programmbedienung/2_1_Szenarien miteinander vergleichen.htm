Szenarien miteinander vergleichen

Sie können mehrere Szenarien miteinander vergleichen. Dabei führen Sie einen Versionsvergleich auf Basis eines Vergleichsmusters durch (Versionen vergleichen und Vergleichsmuster anlegen)

Wenn Sie die Basisversion für einen Vergleich verwenden möchten, legen Sie einfach ein neues Szenario an, das lediglich die Basisversion ohne Modifikationen enthält.

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Wählen Sie im Fenster Projekt bearbeiten das Register Szenarien.

Wählen Sie mindestens zwei Szenarien aus.

Wenn Sie Berechnungsergebnisse vergleichen möchten, müssen Sie die Szenarien zuvor berechnen.

Klicken Sie auf das Symbol   Szenarien vergleichen.

Das Fenster Szenarien vergleichen öffnet sich.

Element

Beschreibung

Vergleichsmuster

Vergleichsmuster, das für den Vergleich verwendet wird

Vergleich durchführen

für Berechnungsergebnisse der Szenarien

Ist die Option ausgewählt, öffnen Sie für das Erstellen des Szenarienvergleichs die Versionsdateien von zuvor berechneten Szenarien. Sie sollten diese Option wählen, wenn Sie beispielsweise Berechnungsergebnisse über Streckenbalken darstellen möchten.

für Eingabedaten der Szenarien

Ist die Option ausgewählt, öffnen Sie für das Erstellen des Szenarienvergleichs die Netze der Szenarien. Diese enthalten keine Berechnungsergebnisse. Diese Option eignet sich, wenn Sie die Netze von Szenarien vergleichen möchten.

Führendes Szenario

Über die Auswahlliste wählen Sie das Szenario aus, das geöffnet wird und die Vergleichsattribute der anderen Szenarien enthält.

Hinweis

Wenn Sie die Option für Berechnungsergebnisse der Szenarien gewählt haben, werden in der Auswahlliste nur berechnete Szenarien angeboten.

Szenarien

Liste der ausgewählten Szenarien

Hinweis

Wenn Sie die Option für Berechnungsergebnisse der Szenarien gewählt haben, werden nur berechnete Szenarien angezeigt.

Klicken Sie auf die Schaltfläche Vergleichen.

Der Versionsvergleich wird durchgeführt. Die Versionsdatei des führenden Szenarios wird geöffnet. Sie enthält die Vergleichsattribute der anderen gewählten Szenarien (Versionen vergleichen).

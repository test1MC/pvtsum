Szenario als Versionsdatei speichern

Sie können ein Szenario eines Szenariomanagement-Projekts exportieren, um es außerhalb des Projekts weiterzubearbeiten.

Stellen Sie sicher, dass das gewünschte Projekt geöffnet ist (Grundeinstellungen eines Projekts bearbeiten).

Wählen Sie im Fenster Projekt bearbeiten das Register Szenarien.

Wählen Sie in der Liste das gewünschte Szenario aus.

Klicken Sie auf das Symbol  Szenario speichern unter.

Das Fenster Version speichern öffnet sich.

Wählen Sie bei Bedarf ein anderes Verzeichnis.

Geben Sie einen Dateinamen ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Szenario wird gespeichert.

Alternativ können Sie das Szenario als Versionsdatei öffnen und dann unter einem anderen Namen speichern.

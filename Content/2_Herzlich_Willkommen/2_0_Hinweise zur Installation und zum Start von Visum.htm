Nachdem Sie installiert haben, stehen Ihnen in der -Programmgruppe (standardmäßig Start > Programme > PTV Vision 2014> ) mehrere Symbole zur Verfügung:

Symbol

Beschreibung

 

Startet mit einem leeren Netz.

  Support

Startet Ihren Internet-Browser und ruft die -Supportseite auf.

  Berechnungs-Server

Startet den Server zur verteilten Berechnung auf mehreren Rechnern.

  Berechnungs-Server für mit Engine

Startet den Server zur verteilten Berechnung auf mehreren Rechnern für die Engine Lizenzvariante

Hinweise

Die Engine können Sie zusätzlich zu installieren. Sie dient ausschließlich als Rechenknoten ohne Benutzeroberfläche. Sie können sie bei verteilten Berechnungen als Rechenknoten auswählen und über den Berechnungsserver verwalten.

Beispiele

Ordner Beispiele öffnen: Öffnet ein Windows-Explorer-Fenster. Darin finden Sie in mehreren Ordnern -Versionsdateien mit verschiedenen Beispielnetzen.

Überblick Beispiele: Öffnet eine Datei *.pdf, die die -Beispiele beschreibt.

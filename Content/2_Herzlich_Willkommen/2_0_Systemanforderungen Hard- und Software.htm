Systemanforderungen Hard- und Software

Folgende Voraussetzungen müssen erfüllt sein, um mit arbeiten zu können.

Betriebssysteme
  • Windows Vista (32-bit/64-bit)
  • Windows 7 (32-bit/64-bit)
  • Windows 8 (32-bit/64-bit)

kann ab der Version 14 nicht mehr auf Windows XP installiert werden.
Hardware (Mindestanforderungen)
  • Prozessor mit Unterstützung für SSE2, das heißt mindestens Intel Pentium 4, Intel Core, AMD Athlon 64 oder AMD Opteron.(Quad-Core oder besser empfohlen)
  • Prozessorgeschwindigkeit mindestens 1 GHz, empfohlen 2 GHz
  • Arbeitsspeicher mindestens 512 MB, empfohlen 4 GB, bei großen Netzen bzw. komplexen Modellen entsprechend mehr

Für einen ungestörten Betrieb und optimales Multitasking sollte genügend RAM-Reserve vorhanden sein. Bei außergewöhnlichen RAM-Spitzen kann der virtuelle Speicher (Swap-File) vergrößert werden.
  • Bildschirmauflösung mindestens 1024 x 768 Pixel
  • Grafikspeicher mindestens 32 MB, empfohlen 64 MB
  • Freie Festplattenkapazität mindestens 1,5 GB

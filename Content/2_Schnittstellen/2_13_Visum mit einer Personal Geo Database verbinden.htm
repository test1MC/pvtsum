mit einer Personal Geodatabase verbinden

Sie können GIS-Objekte in Ihrem Netz anzeigen, indem Sie sich mit einer Personal Geodatabase (PGD) verbinden (GIS-Objekte).

Sie benötigen für den Datenaustausch eine ArcView-Lizenz für die Version 8.3 oder höher.
Mit einer Personal Geodatabase verbinden
  • Wählen Sie im Menü Datei den Eintrag Importieren > Mit Personal Geodatabase verbinden ( mit einer Personal Geodatabase verbinden).
Verbindung zu einer Personal Geodatabase trennen
  • Wählen Sie im Menü Datei den Eintrag Importieren > Verbindung zu Personal Geodatabase trennen (Verbindung zu einer Personal Geodatabase trennen).

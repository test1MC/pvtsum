Screenshot vom Bildschirm exportieren

Sie können einen Screenshot des aktuellen Bildschirmausschnitts (ohne Fensterrand) als Grafikdatei mit der Dateierweiterung *.jpg, *.bmp, *.gif, *.tiff, *.png, *.emf oder *.wmf exportieren.

Klicken Sie im Fenster Netzeditor in der Symbolleiste auf das Symbol JPG Export (Screenshot). Das Fenster Screenshot speichern öffnet sich.

Alternativ können Sie im Menü Datei den Eintrag Exportieren > JPG Export (Screenshot) wählen.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Wählen Sie in der Auswahlliste Dateityp die gewünschte Dateiart.

Das Fenster Screenshot Parameter öffnet sich.

Wählen Sie in der Auswahlliste die gewünschte Auflösung.

Der dpi-Wert ist gemäß der eingestellten Auflösung des eigenen Bildschirms und einem Vielfachen davon vorbelegt.

Wenn Sie eine Datei *.jpg speichern, können Sie außerdem die gewünschte Qualität der exportierten Grafik festlegen.

merkt sich die Einstellung für Auflösung und Qualität für die Dauer der Sitzung.

Der aktuelle Bildschirmausschnitt wird als Grafikdatei gespeichert.

Sie können auch andere Ansichten als Screenshot exportieren. Dazu klicken Sie in der Symbolleiste des jeweiligen Fensters Fahrplan (grafisch), Grünbanddarstellung, Umsteiger-Taktdarstellung oder Schematischer Liniennetzplan auf das Symbol JPG Export (Screenshot) und fahren mit Schritt 2 fort.

Sie können die Grafikdatei zur Weiterbearbeitung in einem externen Grafikprogramm öffnen oder in als Hintergrund einlesen (Hintergrund von Kartendiensten aus dem Internet einfügen).

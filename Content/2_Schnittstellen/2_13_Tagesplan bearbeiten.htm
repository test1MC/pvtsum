Tagesplan bearbeiten

Wählen Sie im Navigator die Seite Tagespläne.

Markieren Sie in der Tabelle den Tagesplan, den Sie bearbeiten möchten.

Klicken Sie in der Symbolleiste auf das Symbol Bearbeiten.

Die Bearbeitungsansicht des ausgewählten Tagesplans öffnet sich.

 

Element

Beschreibung

Name

Hier können Sie den Namen des Tagesplans wie gewünscht anpassen.

Uhrzeit

Hier können Sie Uhrzeiten Signalprogramme zuordnen, die während der festgelegten Zeit ablaufen sollen.

Hinweise

Über das Symbol Neu können Sie weitere Uhrzeiten-Zeilen einfügen.

Über das Symbol Löschen können Sie ausgewählte Zeilen entfernen. Die Symbole stehen in der Symbolleiste und im Kontextmenü zur Verfügung. Sie können die Uhrzeiten in der Spalte Uhrzeit sekundengenau anpassen, indem Sie zweimal nacheinander in die gewünschte Zelle klicken.

Signalprogramm

Hier können Sie in der Auswahlliste für die eingestellte Zeitspanne ein Signalprogramm auswählen.

Beschreibung

Hier können Sie eine individuelle Beschreibung des Tagesplans einfügen.

Die Änderungen werden sofort übernommen.

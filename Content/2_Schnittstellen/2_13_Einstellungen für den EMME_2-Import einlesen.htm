Einstellungen für den EMME/2-Import einlesen

Sie können Einstellungen und EMME/2-Datendateien für den EMME/2-Import aus einer Emme-Projektdatei *.emme einlesen.

Stellen Sie sicher, dass das Fenster Import von EMME/2 geöffnet ist (EMME/2-Import).

Klicken Sie auf die Schaltfläche Parameter öffnen.

Das Fenster Emme-Projekt öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Die Einstellungen und die EMME/2-Daten werden gemäß Emme-Projekt eingelesen.

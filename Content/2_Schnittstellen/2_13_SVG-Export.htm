SVG-Export

 

Diese Funktion ist nur verfügbar, wenn Sie das Zusatzmodul SVG Export aktiviert haben (Zusatzmodule ein- und ausschalten).

Das vom World Wide Web Consortium (W3C) entwickelte Format SVG (Scalable Vector Graphics) ist ein XML-basiertes Vektorgrafikformat für das Internet.

Sie können Grafikdateien *.svg in Browsern anzeigen, die das Format *.svg nativ unterstützen oder die über ein SVG-Plug-In verfügen.

Sie können diese Dateien in anderen Grafikprogrammen weiter bearbeiten, beispielsweise mit Adobe Illustrator oder dem Open Source Vektorgrafikeditor Inkscape, erhältlich unter der URL http://www.inkscape.org. Außerdem können Sie die Dateien mit Hilfe eines Texteditors bearbeiten. Das Format *.svg eignet sich gut als Grafikaustauschformat und bietet zahlreiche Vorteile gegenüber anderen Grafikformaten:

  • Skalierbarkeit ohne Qualitätsverlust
  • Schwenkbarkeit und stufenlose Zoombarkeit
  • eingebettete Informationen zu Objekten
  • interaktiver Zugriff auf Objekte
  • Manipulation per Skript
  • Animation

Sie können Grafikdateien im Format *.svg in als Hintergrund einlesen (Hintergrund von Kartendiensten aus dem Internet einfügen).

Sie können auch andere Ansichten als Dateien im Format *.svg exportieren. Dazu wählen Sie im Menü Fahrplan (grafisch), Grünbanddarstellung oder Umsteiger-Taktdarstellung die Option SVG Export aus, im Menü Schematischer Liniennetzplan den Eintrag Exportieren > SVG Export.

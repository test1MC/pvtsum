-Daten nach railML® exportieren

Sie können Ihre -Daten nach railML® exportieren. Dabei wird eine Datei *.xml gemäß dem railML®-Schema 2.1 erzeugt. Darin enthalten sind Informationen zu Haltepunkt- und Fahrplandaten, sowie zu den Fahrzeugen. Sie exportieren Ihre Daten wie folgt.

Wählen Sie im Menü Datei den Eintrag Exportieren > RailML.

Das Fenster Export nach RailML öffnet sich.

Wählen Sie das Register Grundeinstellungen.

Klicken Sie neben Export in Datei auf die Schaltfläche .

Das Fenster RailML-Datei speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Der Dateiname und der Pfad werden im Eingabefeld übernommen.

Nehmen Sie die weiteren gewünschten Einstellungen vor.

 

Element

Beschreibung

Umfang der exportierten Daten

Nur aktive Fahrplanfahrtabschnitte exportieren

Ist die Option ausgewählt, werden nur die aktiven Fahrplanfahrtabschnitte und die zugehörige Infrastruktur exportiert.

Linienroute

Nur Routenpunkte als Laufweg exportieren

Ist die Option ausgewählt, werden nur Routenpunkte als Laufweg exportiert.

Ist die Option nicht ausgewählt, werden alle Haltepunkte als Laufweg exportiert, die auf dem von einem (aktiven) Fahrplanfahrtabschnitt befahrenen Teil einer Linienroute liegen.

Fahrplanperiode

Sie können die Fahrplanperiode einstellen, wenn das exportierte Netz keinen oder einen Wochenkalender hat.

Hinweis

Der Bereich ist nicht verfügbar, wenn ein Jahreskalender eingestellt ist.

Beginn der exportierten Fahrplanperiode

Über die Schaltfläche können Sie einen Beginn für die exportierte Fahrplanperiode angeben.

Ende der Fahrplanperiode

Das Ende der Fahrplanperiode ergibt sich aus dem gewählten Beginn.

Wählen Sie das Register Attribute.

Element

Beschreibung

Attributauswahl für Datenobjekte der Export-Datei

Datenobjekt

Auflistung der Elemente der Export-Datei

Ziel-Attribut

Attribut des Elements der Export-Datei, das die Information speichert

Quell-Attribut

-Attribut, aus dem das Element erzeugt wird

Hinweis

Hier können Sie die Attribute auswählen, die exportiert werden sollen. Es werden nur die Attribute exportiert, für die ein Quell-Attribut ausgewählt ist.

Wenn Sie ein Attribut nicht übernehmen möchten, können Sie im Attributauswahlfenster den Eintrag keine Auswahl wählen.

Die Daten werden in eine Datei *.xml exportiert.

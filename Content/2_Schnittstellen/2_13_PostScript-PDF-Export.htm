PostScript-PDF-Export

PostScript- und PDF-Dateien haben gegenüber der Bereitstellung der Dateien im Originalformat den Vorteil, dass sie plattformunabhängig und ohne Formatierungsverlust zur Verfügung stehen und dass der Nutzer nicht über die Version der Anwendungssoftware, mit der die Dateien erstellt wurden, verfügen muss.

Sie können die exportierten Abbildungen uneingeschränkt ändern und weiterverarbeiten. Somit können Sie Netzdarstellungen aus problemlos für die weitere flexible Verwendung in DTP-, Layout- und Bildbearbeitungsprogrammen bereitstellen. Bei einer Datei *.pdf, die Sie beispielsweise in Macromedia FreeHand 8.0 importieren, können Sie sämtliche Linien, Flächen, Texte usw. beliebig ändern.

Das Portable Document Format (PDF) von Adobe ist darüber hinaus hervorragend geeignet für die Darstellung im Internet (zum Beispiel in Internet Browser) und ist in interaktiven Medien problemlos realisierbar. Es kann Hypertext-Links, Anmerkungen und Bookmarks enthalten und unterstützt verschiedene Komprimierungsverfahren wie JPEG oder LZW. Schließlich können Dokumente im Format *.pdf auch durch Suchdienste wie Excite for Web Servers indiziert werden. Aufgrund dieser großen Vorteile hat sich das Format *.pdf in letzter Zeit zu dem Hauptstandard für den Dokumentenaustausch im Internet entwickelt.

Sie können aus einer -Netzdarstellung eine Datei im Format *.pdf erzeugen, indem Sie sie als PostScript-Datei über PostScript-fähige Windows-Druckertreiber exportieren und anschließend in das Format *.pdf konvertieren (PostScript-Datei in PDF-Datei konvertieren).

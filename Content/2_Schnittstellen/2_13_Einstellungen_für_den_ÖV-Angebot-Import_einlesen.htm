Einstellungen für den ÖV-Angebot-Import einlesen

Sie können die Einstellungen für den ÖV-Angebot-Import aus einem ÖV-Schnittstellen-Projekt *.puti einlesen.

Stellen Sie sicher, dass das Fenster Import von ÖV-Angebot geöffnet ist (ÖV-Angebotsdaten mit Visum importieren).

Klicken Sie auf die Schaltfläche Parameter öffnen.

Das Fenster ÖV-Schnittstellen-Projekt öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Die Einstellungen und die ÖV-Angebotsdaten werden gemäß ÖV-Schnittstellen-Projekt eingelesen.

Netzdarstellung als PostScript-Datei exportieren

Stellen Sie sicher, dass ein PostScript-fähiger Windows-Druckertreiber auf Ihrem System installiert ist.

Wählen Sie in Windows die Druckerverwaltung.

Wählen Sie den gewünschten Druckertreiber aus.

Wählen Sie im Menü Datei den Eintrag Drucken > Druckereigenschaften.

Das Fenster Eigenschaften von <Name des Druckertreibers> öffnet sich.

Wählen Sie das Register Anschlüsse.

Richten Sie für den Druckertreiber den Anschluss FILE ein.

Zeigen Sie im -Menü Datei auf den Eintrag Drucken.

Wählen Sie den Eintrag Drucken.

Das Fenster Drucken: Netzeditor öffnet sich.

Wählen Sie im Bereich Drucker in der Auswahlliste Name den gewünschten Druckertreiber.

Stellen Sie sicher, dass die Option In Grafikdatei drucken nicht ausgewählt ist.

Die Option ist nur für den Export in den Formaten *.svg und *.dxf relevant (DXF Export und SVG-Export).

Klicken Sie auf die Schaltfläche Eigenschaften.

Das Fenster Eigenschaften von <Name des Druckertreibers> öffnet sich.

Schließen Sie das Fenster Eigenschaften von <Name des Druckertreibers> mit OK.

Klicken Sie bei Bedarf auf die Schaltfläche Druckseite einrichten.

Das Fenster Seite einrichten öffnet sich.

(Druckseite einrichten).

Schließen Sie das Fenster Drucken mit OK.

Das Fenster Ausgabe in Datei umleiten öffnet sich.

Geben Sie einen Dateinamen, den Pfad und die Dateierweiterung *.ps oder *.eps ein.

Durch Eingabe der Dateierweiterung *.ps wird die PostScript-Datei direkt lesbar für PDF-Konvertierungsprogramme.

Durch Eingabe der Dateierweiterung *.eps können Sie die PostScript-Datei direkt in vielen Text- und Layoutprogrammen einlesen, zum Beispiel Microsoft Word, wenn Sie sie als eingekapselte PostScript-Datei erzeugt haben.

Die Netzdarstellung wird als PostScript-Datei exportiert.

PostScript-Dateien können erhebliche Ausmaße annehmen. Die anschließend erzeugte Datei *.pdf kann aber um den Faktor 20 kleiner als die Ausgangsdatei sein.

Sie können die PostScript-Datei anschließend in Bildbearbeitungs-, Grafik-, DTP- und Textverarbeitungsprogrammen öffnen und bearbeiten. Folgende Programme werden unterstützt (getestet ab der angegebenen Version):

  • Zeichen- und Grafikprogramme: CorelDraw 7+, Macromedia FreeHand 7.0, Adobe Illustrator 7.0
  • Textverarbeitung/Layout: Microsoft Word 6.0, Adobe PageMaker 6.5, QuarkXPress 4.0
  • Bildbearbeitung: Corel Photo-Paint 7+, Adobe Photoshop 3.0
  • Präsentation: Microsoft PowerPoint 97

In CorelDraw 7+ können Sie die PostScript-Datei direkt als Vektorgrafik öffnen. Zahlen und Text können Sie als String jedoch nicht mehr bearbeiten.

In Adobe Photoshop 4+ können Sie die PostScript-Datei direkt als Rasterbild mit Angabe der Zielparameter wie Auflösung, Farbsystem usw. öffnen. Sie können die bearbeitete PostScript-Datei in den Formaten *.bmp, *.jpg und *.gif speichern.

Wenn Sie die PostScript-Datei als eingekapselter PostScript *.eps gespeichert haben, können Sie sie direkt in vielen Text- und Layoutprogrammen einlesen, zum Beispiel in Microsoft Word. Sie können die Datei jedoch nicht am Bildschirm darstellen und in der Größe verändern, wenn das Programm nicht über eine integrierte Bildvorschau verfügt (beispielsweise *.tif). Wir empfehlen, zunächst über einen Grafikkonverter oder über ein geeignetes Grafikprogramm aus der PostScript-Datei entweder eine Rastergrafik (zum Beispiel *.bmp) oder eine Vektorgrafik (zum Beispiel das von Microsoft Word unterstützte Format *.wmf) zu erzeugen und anschließend einzulesen. Bei Microsoft-Anwendungen können Sie auch die OLE-Funktion verwenden.

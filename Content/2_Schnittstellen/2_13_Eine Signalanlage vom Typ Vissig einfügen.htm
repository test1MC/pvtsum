Eine Signalanlage vom Typ Vissig einfügen

Wenn Sie eine Signalanlage in Vissig bearbeiten möchten, müssen Sie am gewünschten Knoten oder Oberknoten eine Vissig-LSA einfügen oder eine vorhandene LSA in eine Vissig-LSA konvertieren.

Neue Signalanlage vom Typ Vissig einfügen

Wenn dem Knoten noch keine Signalanlage zugeordnet ist, fügen Sie eine Signalanlage vom Typ Vissig ein:

Öffnen Sie den gewünschten Knoten oder Oberknoten im Knoteneditor (Ein Netzobjekt im Knoteneditor anzeigen).

Ordnen Sie dem Knoten über das Symbol LSA einfügen eine Signalanlage zu.

Das Fenster LSA einfügen öffnet sich.

Geben Sie die gewünschten Daten ein und wählen Sie den Signalisierungstyp Vissig aus.

Das Fenster Externe Steuerung speichern öffnet sich.

Geben Sie den Dateinamen und bei Bedarf den Pfad an.

Klicken Sie auf die Schaltfläche Speichern.

Das Programm Vissig öffnet sich zur weiteren Bearbeitung der Signalanlage.

Vorhandene -interne Signalanlage für Vissig konvertieren

Wenn dem Knoten bereits eine Signalanlage vom Signalisierungstyp Signalgruppenbasiert oder Phasenbasiert zugeordnet ist, konvertieren Sie die Signalanlage:

Öffnen Sie den gewünschten Knoten oder Oberknoten im Knoteneditor (Ein Netzobjekt im Knoteneditor anzeigen).

Klicken Sie in der Symbolleiste auf das Symbol LSA in externe Steuerung konvertieren.

Das Fenster Externe Steuerung speichern öffnet sich.

Wählen Sie in der Auswahlliste Dateityp den Eintrag Externe Steuerung (*.sig).

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Eine Steuerungsdatei *.sig wird gespeichert und das Programm Vissig öffnet sich zur weiteren Bearbeitung der Signalanlage.

Netzdarstellung als Grafikdatei *.svg exportieren

Sie haben zwei Möglichkeiten, eine Grafikdatei *.svg zu exportieren.

Erste Möglichkeit, eine Grafikdatei *.svg zu exportieren

Wählen Sie im Menü Datei den Eintrag Exportieren > SVG Export.

Das Fenster SVG-Datei speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster SVG-Export: <Name der gewählten Ansicht> öffnet sich.

Das Fenster enthält eine Schaltfläche, die entweder mit Erweitern oder Reduzieren beschriftet ist. Wenn Sie auf die Schaltfläche Erweitern klicken, werden weitere Export-Optionen im Fenster angezeigt. Wenn Sie auf die Schaltfläche Reduzieren klicken, verkleinert sich das Fenster.

Element

Beschreibung

Ausgabedatei

Über die Option legen Sie fest, ob die exportierte Datei für die Anzeige in einem Browser, als -Hintergrund oder für die Weiterverarbeitung in Adobe Illustrator (Hinweise zu -SVG-Dateien in Adobe Illustrator) oder Inkscape optimiert werden soll.

optimiert für Browserdarstellung

Ist die Option ausgewählt, wird die exportierte Datei für eine Anzeige im Browser optimiert.

optimiert für (Hintergrund)

Ist die Option ausgewählt, wird die exportierte Datei für eine Anzeige als -Hintergrund optimiert.

optimiert für Inkscape

Ist die Option ausgewählt, wird die exportierte Datei für Ink-scape optimiert.

optimiert für Adobe Illustrator

Ist die Option ausgewählt, wird die exportierte Datei für Adobe Illustrator optimiert.

Hinweis

Wird der SVG-Export für die Weiterbearbeitung in Adobe Illustrator oder Inkscape optimiert, wird die Grafik auf eine Ausgabeseite transformiert. Die Einstellungen für die Ausgabeseite nehmen Sie in einem separaten Fenster vor, das sich automatisch öffnet, wenn Sie mit OK bestätigen.

Allgemeine Optionen

In diesem Bereich sind abhängig von der gewählten Option im Bereich Ausgabedatei bereits sinnvolle Einstellungen vorgegeben.

SVG-Datei in Html einbetten

Ist die Option ausgewählt, werden eine Datei *.svg und eine HTML-Seite *.html exportiert, in der die Datei *.svg eingebettet ist. Zusätzlich wird eine Datei *.css exportiert, mit der Sie das Layout der Html-Seite anpassen können.

Ist die Option nicht ausgewählt, wird nur eine Datei *.svg exportiert.

Hinweis

Die Option ist nur für die Anzeige in Browsern relevant.

Tipp

Eine in eine HTML-Seite eingebettete Datei *.svg bietet den Vorteil, dass die HTML-Seite eine Steuerung aller Ebenen enthält über die Sie im Browser die jeweiligen Ebenen ein- oder ausblenden können.

Clipping-Rechteck erzeugen

Ist die Option ausgewählt, wird in der exportierten Datei ein rechteckiger Bereich definiert, in dem alle Objekte eingefügt werden. Der Bereich entspricht dem aktuellen Bildschirmausschnitt. Alle Objekte, die aus dem Rechteck herausragen, werden abgeschnitten.

Strichstärken nicht skalieren

Ist die Option ausgewählt, werden Striche beim Zoomen in der SVG-Grafik mit einer konstanten Breite dargestellt. So kann beispielsweise eine Streckensignatur immer mit einer konstanten Millimeterbreite dargestellt werden.

Ist die Option nicht ausgewählt, wird die Strichbreite beim Zoomen in der SVG-Grafik skaliert.

Hinweis

Die SVG-Eigenschaft, die bei ausgewählter Option verwendet wird, wird von den meisten Browsern nicht richtig interpretiert. Eine Ausnahme bildet Opera ab Version 9. Daher empfehlen wir, diese Option nur auszuwählen, wenn die exportierte SVG-Datei als -Hintergrund verwendet werden soll.

Originalkoordinaten verwenden

Ist die Option ausgewählt, werden in der Datei *.svg für alle Elemente die Originalkoordinaten verwendet und erst dann in das SVG-Koordinatensystem transformiert.

Ist die Option nicht ausgewählt, werden die Koordinaten bereits vor dem Export in transformiert.

Verknüpfungen

Absolute Pfade verwenden

Ist die Option ausgewählt, werden für den Verweis auf in der Version eingebundene Grafikdateien absolute Pfade in der Datei *.svg gespeichert.

Ist die Option nicht ausgewählt, werden für den Verweis auf in der Version eingebundene Grafikdateien relative Pfade in der Datei *.svg gespeichert.

Hinweis

Wir empfehlen, relative Pfade zu verwenden.

Bilder in SVG-Verzeichnis kopieren

Ist die Option ausgewählt, wird eine Kopie der in der Version eingebundenen Grafikdateien (zum Beispiel als POIs, Hintergründe oder Symbole für Punktobjekte) im gleichen Verzeichnis wie die Datei *.svg gespeichert.

Ist die Option nicht ausgewählt, greift die Datei *.svg weiterhin auf die Grafikdateien zu, die in der Version eingebunden sind.

JavaScript

Onclick Event Handler erzeugen

Ist die Option ausgewählt, wird ein Javascript Event Handler erzeugt.

Hinweis

Wenn Sie die SVG-Datei im Browser anzeigen und JavaScript für ihren Browser aktiviert ist, können Sie bei ausgewählter Option interaktiv auf die Objekte in der Datei *.svg zugreifen, indem Sie auf einzelne Objekte klicken. In einem separaten Fenster werden dann Informationen (Schlüsselattribute) zu den Objekten angezeigt sowie zu den Ebenen, die die Objekte enthalten. In die JavaScript-Datei können Sie weitere benutzerdefinierte Funktionen einfügen, zum Beispiel COM-Aufrufe.

Funktion in Datei auslagern

Ist die Option ausgewählt, wird der JavaScript-Code in eine externe Datei ausgelagert.

Ist die Option nicht ausgewählt, wird der JavaScript-Code in die SVG-Datei geschrieben.

Hinweise

Die Option steht nur zur Verfügung, wenn die Option Onclick Event Handler erzeugen ausgewählt wurde.

Eine externe Datei ist nützlich, wenn Sie das Skript anpassen möchten und diese Funktion aus mehreren verschiedenen SVG-Dateien aufrufen wollen.

Wenn Sie die Option optimiert für Browserdarstellung ausgewählt haben, wird die Netzdarstellung direkt exportiert. Wenn Sie die Option optimiert für Adobe Illustrator ausgewählt haben, öffnet sich das Fenster Ausgabeseite für SVG-Export festlegen.

Nehmen Sie die gewünschten Einstellungen für die Ausgabegröße der Grafik vor.

Die Netzdarstellung wird als Grafikdatei *.svg exportiert.

Alle auf dem System installierten Schriftarten können im Format *.svg exportiert werden. Wir empfehlen, für Texte und Beschriftungen gängige Schriftarten zu verwenden, beispielsweise Arial oder Times New Roman, da die Darstellung mancher Schriftarten im SVG-Viewer nicht mit der -Bildschirmdarstellung übereinstimmt.

SVG-Dateien können nur Rastergrafiken in den Formaten *.jpg, *.gif und *.png anzeigen. Wenn in der Version Rastergrafiken in anderen Formaten eingebunden sind, wird von diesen eine Kopie im Format *.png erzeugt und in die SVG-Grafik eingebunden.
Zweite Möglichkeit, eine Grafikdatei *.svg zu exportieren

Zeigen Sie im Menü Datei auf den Eintrag Drucken.

Wählen Sie den Eintrag Drucken.

Das Fenster Drucken öffnet sich.

Wählen Sie im Bereich Dateiausgabe die Option In Grafikdatei drucken aus.

Das Fenster Grafikdatei speichern öffnet sich.

Wählen Sie in der Auswahlliste Dateityp den Eintrag Grafikdatei (*.svg).

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster SVG-Export öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (siehe „Erste Möglichkeit, eine Grafikdatei *.svg zu exportieren“ auf Seite 1748).

Wenn Sie die Option optimiert für Browserdarstellung ausgewählt haben, wird die Netzdarstellung direkt exportiert. Wenn Sie die Option optimiert für Adobe Illustrator ausgewählt haben, öffnet sich das Fenster Ausgabeseite für SVG-Export festlegen.

Nehmen Sie die gewünschten Einstellungen für die Ausgabegröße der Grafik vor (siehe „Erste Möglichkeit, eine Grafikdatei *.svg zu exportieren“ auf Seite 1748).

Die Netzdarstellung wird als Grafikdatei *.svg exportiert.

Beispiel einer im Format *.svg exportierten Netzdarstellung

Der Bildschirmausschnitt ist als eine in HTML eingebettete Datei *.svg gespeichert. Die Ebenen für die Objekte Anbindungen und Haltepunkte sind ausgeschaltet und daher nicht sichtbar.

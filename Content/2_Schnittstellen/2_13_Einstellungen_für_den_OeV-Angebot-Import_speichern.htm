Einstellungen für den ÖV-Angebot-Import speichern

Sie können die Einstellungen für den ÖV-Angebot-Import und die ausgewählten ÖV-Angebotsdaten-Dateien für den Import als Projekt *.puti speichern.

Wählen Sie im Menü Datei > Importieren den Eintrag ÖV-Angebot aus Visum.

Das Fenster Import von ÖV-Angebot öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (ÖV-Angebotsdaten mit Visum importieren).

Klicken Sie auf die Schaltfläche Parameter speichern.

Das Fenster ÖV-Schnittstellen-Projekt speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Die aktuellen Einstellungen und die ausgewählten ÖV-Angebotsdaten werden als ÖV-Schnittstellen-Projekt *.puti gespeichert.

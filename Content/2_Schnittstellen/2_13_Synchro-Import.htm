Synchro-Import

bietet eine Schnittstelle für den Import von Synchro-Daten. importiert alle Elemente aus einer Synchro® UTDF CSV Datei, einschließlich Geometrie, Steuerung, LSA Zeiten und Abbiegerbelastungen.

Der Import von Daten aus Synchro® Version 6 oder älter wird nicht unterstützt.

Wählen Sie im Menü Datei den Eintrag Importieren > Synchro.

Das Fenster Synchro-Netz öffnen öffnet sich.

Wählen Sie die gewünschte Synchro-Datei aus.

Das Synchro-Netz wird importiert und in angezeigt.

Abbiegerbelastungen aus Synchro werden in in benutzerdefinierte Attribute ANMVOLUME_CAR und ANMVOLUME_HGV geschrieben

Beim Import vorgenommene Einstellungen am Verfahrensablauf

Beim Import werden die allgemeinen Verfahrenseinstellungen angepasst:

  • Standard Methode für die Widerstände am Knoten: Abbieger CR-Funktion (siehe „Methode für Knotenwiderstandsberechnung festlegen“ auf Seite 1)
  • Bemessungsverkehrsstärke [PkwE / h]: Vistro-Stundenbelastung (siehe „Bemessungsverkehrsstärke für Knotenwiderstandsberechnung festlegen“ auf Seite 1)
  • Für die Umlegung wird die Grundbelastung fixiert, die detaillierte Berechnung wird eingestellt (siehe „Grundbelastung berücksichtigen“ auf Seite 1):
  • Strecken: Vistro Gesamtbelastung Analyse
  • Anbindungen: ZWert 1
  • Abbieger: Vistro Gesamtbelastung Analyse
  • Oberabbieger: ZWert 1

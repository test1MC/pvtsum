Phasenfolge festlegen und Signalprogramm erstellen

Sie können die Reihenfolge der Phasen festlegen und anschließend aus den vorhandenen Phasen phasenbasierte oder signalgruppenbasierte Signalprogramme erstellen.

Im oberen Bereich der Seite Editieren der Phasenfolge werden alle vorhandenen Phasen angezeigt. Im unteren Bereich wird die erstellte Phasenfolge angezeigt.

Sie können die Höhe der Bereiche ändern, indem Sie an der Trennleiste zwischen Phasen und Phasenfolge ziehen.

Sie können eine Phasenfolge als Grafik exportieren (Phasenfolge exportieren).

Wählen Sie im Navigator die Seite Editieren der Phasenfolge.

Klicken Sie auf die Phase, die die erste Phase der Phasenfolge sein soll.

Halten Sie die Taste Strg gedrückt und klicken Sie auf die zweite gewünschte Phase.

Halten Sie die Taste Strg gedrückt und klicken Sie in der gewünschten Reihenfolge auf alle weiteren Phasen, die Sie der Phasenfolge hinzufügen möchten.

Öffnen Sie mit einem Rechtsklick das Kontextmenü.

 

Element

Beschreibung

Phasenfolge erzeugen

Über diesen Eintrag wird eine Phasenfolge erstellt.

Zur Phasenfolge hinzufügen

Über diesen Eintrag können Sie einer bereits erstellten Phasenfolge weitere Phasen einzeln hinzufügen.

Hinweis

Alternativ können Sie auf die gewünschte Phase doppelklicken.

Die Phasenfolge wird im unteren Bereich der Seite angezeigt.

 

Mit einem Doppelklick auf eine Phase in der Anzeige der Phasenfolge können Sie einzelne Phasen aus der Folge entfernen. Die gesamte Phasenfolge können Sie über den Eintrag Phasenfolge löschen im Kontextmenü löschen.

Geben Sie die gewünschte Umlaufzeit für das Signalprogramm ein.

Klicken Sie auf die gewünschte Schaltfläche.

 

Element

Beschreibung

Erstelle phasenbasiertes Signalprogramm

Über diese Schaltfläche erstellen Sie ein phasenbasiertes Signalprogramm mit vorgegebener Umlaufdauer.

Erstelle signalgruppenbasiertes Signalprogramm

Über diese Schaltfläche erstellen Sie ein signalgruppenbasiertes Signalprogramm mit vorgegebener Umlaufdauer.

 

Bevor Sie ein Signalprogramm erstellen, müssen Sie auf der Seite Phasenzuordnung eine Zwischenzeitmatrix als Standard auswählen (Standardzwischenzeitmatrix auswählen).

Das Signalprogramm wird berechnet. Die Bearbeitungsansicht öffnet sich im Navigator unter Signalprogramme. Für alle Übergänge zwischen verwendeten Phasen werden, falls erforderlich, automatisch Phasenübergänge erstellt.

Phasenübergänge können im Navigator unter Phasenübergänge bearbeitet werden (Phasenübergänge verwalten).

Sie können Phasenübergänge auch manuell einfügen (Phasenübergang manuell einfügen). Bevor Sie Phasenübergänge einfügen, können Sie allgemeine Einstellungen zur Optimierung der Phasenübergänge vornehmen (Phasenübergänge optimieren).

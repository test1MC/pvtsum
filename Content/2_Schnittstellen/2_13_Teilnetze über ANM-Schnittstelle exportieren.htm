Teilnetze über ANM-Schnittstelle exportieren

Sie können mit dem Teilnetzgenerator ein Teilnetz erzeugen und über die ANM-Schnittstelle in importieren. Sie können dabei für das Teilnetz den Verfahrensablauf der geöffneten Version noch einmal ausführen lassen.

Wählen Sie im Menü Rechnen den Eintrag ANM-Export eines Teilnetzes.

Das Fenster ANM-Export Teilnetz öffnet sich.

Element

Beschreibung

Teilnetzgenerator

Parameterdatei verwenden

Ist die Option ausgewählt, können Sie Pfad und Namen der Teilnetzgenerator-Parameterdatei *.xml eingeben oder Sie wählen über das Symbol eine Datei aus.

Parameter für Teilnetzgenerator festlegen

Ist die Option ausgewählt, können Sie über die Schaltfläche Teilnetzgenerator die Einstellungen für den Teilnetzgenerator festlegen (Teilnetz erzeugen)

Bereich Verfahrensablauf

Verfahrensablauf

Kein Verfahrensablauf

Ist die Option ausgewählt, wird kein Verfahrensablauf für das Teilnetz ausgeführt.

Nur IV-Umlegungen wie in geöffneter Version

Ist die Option ausgewählt, werden die gleichen IV-Umlegungen wie in der geöffneten Version ausgeführt.

Hinweis

Für jedes ins Teilnetz exportierte IV-Nachfragesegment mit Nachfragematrix wird eine Umlegung ausgeführt. Die Einstellungen, die zuletzt in der geöffneten Version gerechnet wurden, werden übernommen.

Wie in geöffneter Version durchgeführt

Ist die Option ausgewählt, werden nur die aktiven Verfahren der geöffneten Version ausgeführt.

Hinweis

Es werden nur ins Teilnetz exportierte IV-Nachfragesegmente mit Nachfragematrix berücksichtigt. Die Einstellungen, die zuletzt in der geöffneten Version gerechnet wurden, werden übernommen.

Aus Datei

Ist die Option ausgewählt, können Sie Pfad und Namen einer Verfahrensparameterdatei *.par oder *.xml eingeben oder Sie wählen über das Symbol eine Datei aus (Verfahrensparameterdatei öffnen).
ANM-Export

 

Parameterdatei verwenden

Ist die Option ausgewählt, können Sie Pfad und Namen der ANM-Export-Parameterdatei *.anmp eingeben oder Sie wählen über das Symbol eine Datei aus.

Parameter für ANM-Export festlegen

Ist die Option ausgewählt, können Sie über die Schaltfläche ANM-Export Parameter die Einstellungen für den ANM-Export festlegen (Abstrahiertes Netzmodell speichern)

Dateien in importieren

Ist die Option ausgewählt, wird das Teilnetz automatisch in einer neuen Instanz von geöffnet.

Hinweis

Routen werden als statische Routen nach importiert. Möchten Sie in eine dynamische Umlegung ausführen, dürfen Sie die Option nicht auswählen. Sie müssen in diesem Fall das Teilnetz manuell über die ANM-Schnittstelle in importieren.

Das ANM-Export Teilnetz wird erstellt.

Datentransfer von nach Vissim

Wenn Sie Daten aus in Vissim nutzen möchten, gehen Sie wie folgt vor.

Bearbeiten Sie zunächst ein -Netz.

Wenn Sie den aktuellen Netzzustand nach Vissim exportieren wollen, speichern Sie aus ein abstrahiertes Netzmodell, das nur die Vissim-relevanten Daten enthält, in einer Datei *.anm (Abstrahiertes Netzmodell speichern).

Lesen Sie das abstrahierte Netzmodell (ANM) in Vissim ein.

Vissim erzeugt dazu ein Vissim-Netz (*.inp).

Bearbeiten Sie das Vissim-Netz.

Parallel dazu können Sie das -Netz bearbeiten.

Speichern Sie aus ein weiteres abstrahiertes Netzmodell, wenn das -Netz zu einem späteren Zeitpunkt erneut exportiert werden soll (Abstrahiertes Netzmodell speichern).

Sie können die gleiche Datei *.anm verwenden, wie beim vorherigen Export. Der Inhalt der vorherigen Datei *.anm wird intern vorgehalten.

Lesen Sie das neue abstrahierte Netzmodell in Vissim ein.

Vissim vergleicht dieses Modell mit dem beim vorherigen Export von gelieferten Modell und ermittelt veränderte oder eingefügte und gelöschte Objekte. Abhängig von der Art der Änderung wird das bisherige Vissim-Netz so geändert, dass die -Änderungen abgebildet werden. In nicht geänderte Netzteile und in Vissim durchgeführte Änderungen bleiben unberührt.

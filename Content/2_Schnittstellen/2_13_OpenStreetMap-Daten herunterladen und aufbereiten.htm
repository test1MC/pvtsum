OpenStreetMap-Daten herunterladen und aufbereiten

Für den Import nach müssen die OpenStreetMap-Daten als Dateien mit der Dateierweiterung *.osm, einem von OpenStreetMap definierten XML-Format oder als komprimierte Archive im *bzip Dateiformat vorliegen. Sie können die Rohdaten von verschiedenen Internetseiten herunterladen und anschließend bearbeiten.

Solche Dateien können Sie für kleine Netzausschnitte direkt von der OpenStreetMap-Webseite herunterladen. Für größere Gebiete wie Staaten oder Bundesländer können Sie Dateien von verschiedenen Diensten wie beispielsweise Geofabrik http://www.geofabrik.de/) oder CloudMade (http://cloudmade.com/solutions/portals/) als Zip-Archive beziehen. Sie können aus den Daten ein Teilnetz importieren, wenn Sie in den Importeinstellungen einen Ausschnitt definieren. Mit dem frei verfügbaren Tool osmosis (http://wiki.openstreetmap.org/wiki/Osmosis) können Sie auch Daten anhand inhaltlicher Kriterien vor dem Import filtern.

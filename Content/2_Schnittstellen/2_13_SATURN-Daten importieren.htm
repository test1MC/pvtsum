SATURN-Daten importieren

Wählen Sie im Menü Datei den Eintrag Importieren > SATURN.

Das Fenster SATURN Import öffnet sich.

Element

Beschreibung

SATURN-Datei

Geben Sie in das Eingabefeld Verzeichnis und Name der gewünschten Datei *.dat ein oder wählen Sie ihn über das Symbol .

Hinweis

Falls die Datei *.dat Verweise auf externe Dateien enthält, müssen diese im gleichen Verzeichnis liegen.

Erzeugte Netzdatei (*.net) speichern?

Ist die Option ausgewählt, werden die Daten nach dem Import als Netzdatei gespeichert und die Netzdatei wird geöffnet.

Hinweis

Die initial gespeicherte Netzdatei enthält keine Gehzeiten t‑ÖVSys(PUTW) an Strecken, diese werden vom Importer nach dem automatischen ersten Öffnen der Datei in berechnet. Öffnen Sie zu einem späteren Zeitpunkt die initial gespeicherte Netzdatei in , bedenken Sie bitte, dass in dieser Netzdatei keine Gehzeiten an Strecken enthalten sind.

Ist die Option nicht ausgewählt, werden die Daten beim Import in eine temporäre Netzdatei geschrieben und in geöffnet. Die Datei wird anschließend gelöscht.

Ausgabe-Netzdatei

Geben Sie in das Eingabefeld Verzeichnis und Name der Netzdatei ein, in der Sie die importierten Daten speichern möchten oder wählen Sie ihn über das Symbol .

Klicken Sie auf die Schaltfläche Ausführen.

Die Daten werden importiert. Dabei wird eine temporäre Netzdatei angelegt, die in eingelesen wird (Beschreibung der beim SATURN-Import erzeugten Netzdatei). Aufgrund der verschiedenen Datenstrukturen werden auch Änderungen in der Netztopologie vorgenommen (SATURN-Import: Hinweise zu Änderungen an der Netztopologie).

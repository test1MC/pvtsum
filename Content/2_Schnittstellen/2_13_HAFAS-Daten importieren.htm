HAFAS-Daten importieren

Wählen Sie im Menü Datei den Eintrag Importieren > Hafas.

Das Fenster Import von HAFAS öffnet sich.

Wählen Sie das Register Dateien.

Klicken Sie je HAFAS-Datendateiart, für die Sie Daten importieren möchten, auf die Schaltfläche .

Das Fenster <Dateiart> öffnen öffnet sich.

Für die Dateiart Fahrplandateien können Sie mehrere Dateien auswählen. Wenn Sie in der Zeile Fahrplandatei(en) auf die Schaltfläche klicken, öffnet sich das Fenster Fahrplandateien, in dem Sie über die Schaltflächen Hinzufügen und Löschen mehrere Fahrplandateien zur Auswahl hinzufügen oder aus der Auswahl löschen können.

Falls Sie Kurswagen mit einlesen, müssen Sie darauf achten, dass die Dateien mit den Kurswagendaten nach den Dateien mit den ziehenden Zügen ausgewählt sind.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Der Dateiname und der Pfad werden im jeweiligen Eingabefeld Pfad übernommen.

Sie können die Zuordnung einer zu importierenden Datei löschen, indem Sie auf die jeweilige Schaltfläche Löschen klicken.

Wenn zur Ergänzung einer ausgewählten HAFAS-Datendatei weitere HAFAS-Datendateien einer anderen Dateiart erforderlich sind, werden die benötigten Dateiarten durch einen roten Balken hervorgehoben.

Wählen Sie, ob Warnungen während des Importvorgangs am Bildschirm angezeigt werden sollen.

Element

Beschreibung

Keine Warnungen anzeigen

Ist die Option ausgewählt, werden Warnungen während des Importvorgangs nicht angezeigt. Sie werden jedoch in der Meldungsdatei gespeichert, wenn die Optionen Warnungen und Hinweise in Meldungsdatei schreiben und Neue Datei erzeugen ausgewählt sind (Einstellungen für Protokolldateien vornehmen).

Wählen Sie das Register Verkehrssysteme.

Die Einstellungen im Register Verkehrssysteme sind nur zugänglich, wenn Sie eine HAFAS-Datendatei der Dateiart Verkehrsgattungen ausgewählt haben.

Import

In dieser Spalte können Sie je Verkehrsgattung festlegen, ob sie als -Verkehrssystem importiert werden soll. Ist das Kontrollkästchen inaktiv, wird die Verkehrsgattung nicht importiert.

VSys-Name

Tipp

Hinweis

Für alle ÖV-Verkehrssysteme werden nur ein Modus und ein Nachfragesegment erzeugt.

Wählen Sie das Register Kalender.

Element

Beschreibung

Kalendertyp

Über die Auswahlliste können Sie einen Kalendertyp zur Einstellung der Kalenderperiode wählen.

Hinweis

Die Kalendertypen Wochenkalender und Jahreskalender sind nur verfügbar, wenn Sie das Zusatzmodul Kalender / Verkehrstage aktiviert haben (Zusatzmodule ein- und ausschalten).

Datumsbereich

Kalenderperiode, für die Fahrten importiert werden

Hinweis

Wenn Sie eine Eckdatendatei einlesen, wird die gesamte Fahrplanperiode als Kalenderperiode übernommen.

von, bis

Über die Schaltfläche rufen Sie einen Kalender auf, in dem Sie den Beginn oder das Ende der Kalenderperiode mit Mausklick auf das gewünschte Datum auswählen.

Hinweise

Die Option ist nur verfügbar, wenn der Kalendertyp Wochenkalender oder Jahreskalender ausgewählt ist.

Alle Fahrten, für die mindestens ein Verkehrstag im Datumsbereich liegt, werden importiert, sofern kein Filter definiert ist.

Filter

Über den Filter können Sie die Anzahl der zu importierenden Fahrten auf die Fahrten einschränken, die an von Ihnen festgelegten Tagen verkehren.

Über das Symbol rufen Sie einen Kalender auf, in dem Sie einen Stichtag mit Mausklick auf das gewünschte Datum auswählen.

Über die Schaltfläche Hinzufügen fügen Sie einen Stichtag der Liste hinzu.

Über die Schaltfläche Löschen können Sie die in der Liste markierten Stichtage löschen.

Über die Schaltfläche Alle löschen können Sie alle Stichtage gleichzeitig löschen.

Zeitverschiebungen

Über die Option können Sie eine Verschiebung der Ankunfts- und Abfahrtszeiten von Fahrten an Haltestellen im Eingabefeld resultierende Zeitverschiebung eingeben oder aus einer Zeitverschiebungsdatei einlesen und anhand eines Stichzeitpunkts für jede Haltestelle berechnen.

Über die Schaltfläche rufen Sie einen Kalender auf, in dem Sie einen Tag für den Stichzeitpunkt mit Mausklick auf das gewünschte Datum auswählen. Geben Sie im Eingabefeld den Stichzeitpunkt im Uhrzeitformat ein (hh:mm:ss).

Wählen Sie das Register Namen Fahrplanfahrten.

In diesem Register legen Sie fest, aus welchen Daten die Namen der Linien und Fahrplanfahrten in gebildet werden. Jeweils die linke Liste enthält die Daten, aus denen sich die Namen in der gegebenen Reihenfolge zusammensetzen. Die rechte Liste enthält alle Daten, die für die Namensbildung zur Verfügung stehen.

Wählen Sie im Bereich Linienname in der linken Liste den Eintrag, über dem Sie eine weitere Namenskomponente hinzufügen möchten.

Wählen Sie im Bereich Linienname in der rechten Liste den Eintrag, den Sie zum Liniennamen hinzufügen möchten.

Klicken Sie auf das Symbol .

Der Eintrag wird in die linke Liste über dem aktuell ausgewählten Eintrag eingefügt.

Geben Sie bei Bedarf im Bereich Linienname in das Eingabefeld Trennzeichen ein Zeichen zur Trennung mehrerer Namenskomponenten ein.

Das Semikolon ist als Trennzeichen nicht erlaubt.

Fügen Sie bei Bedarf weitere Einträge hinzu.

Sie können einen Eintrag aus der linken Liste löschen, indem Sie den gewünschten Eintrag auswählen und auf die Schaltfläche klicken.

Legen Sie auf die gleiche Weise im Bereich Fahrtnamen fest, aus welchen Daten die Fahrtnamen gebildet werden sollen.

Wählen Sie das Register Namen Kurswagenfahrten.

In diesem Register legen Sie fest, aus welchen Daten die Namen der Linien und Fahrplanfahrten von Kurswagen in gebildet werden. Zur Unterscheidung von den anderen Linien und Fahrplanfahrten können Sie eine Zeichenfolge festlegen, die an einer bestimmten Stelle im Namen enthalten ist.

Wählen Sie im Bereich Maßgeblicher Zug ist die gewünschte Option.

Element

Beschreibung

erste ziehende Fahrt

Ist diese Option ausgewählt, werden HAFAS-Gattungscode, HAFAS-Linienname, HAFAS-Zugnummer, HAFAS-Verwaltung und -Verkehrssystemcode aus den Daten des Zuges in der ersten Zeile *KWZ ermittelt.

ranghöchste ziehende Fahrt

Ist diese Option ausgewählt, werden die oben genannten Daten aus den Daten des Zuges in der Zeile *KWZ ermittelt, der die höchste Rangnummer besitzt. Falls mehrere Züge mit gleichem Rang vorhanden sind, wird der erste mit diesem Rang verwendet.

Geben Sie in die Eingabefelder Fester Namensbestandteil für Linie und Fester Namensbestandteil für Fahrt jeweils die gewünschte Zeichenfolge ein.

Diese Zeichenfolge steht für die Namensbildung als Eintrag Fester Namensbestandteil zur Verfügung.

Legen Sie auf die gleiche Weise wie im Register Namen Fahrplanfahrten fest, aus welchen Daten die Linien- und Fahrtnamen gebildet werden sollen.

Die ausgewählten HAFAS-Daten werden importiert.

Beim HAFAS-Import wird aus jeder HAFAS-Fahrt eine Linienroute erzeugt. Sie können anschließend Linienrouten aggregieren (Linienrouten aggregieren).

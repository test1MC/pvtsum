PostScript-Datei in PDF-Datei konvertieren

Sie haben verschiedene Möglichkeiten, aus einer -Netzdarstellung eine Datei im Format *.pdf zu erzeugen.

  • Sie können die Netzdarstellung nur direkt als Datei *.pdf exportieren, wenn Sie die kommerzielle Software Acrobat Distiller als Bestandteil von Adobe Acrobat erworben haben. Nach der Installation von Distiller können Sie diesen in wie jeden beliebigen Windows-Druckertreiber verwenden.
  • Sie können die Netzdarstellung als PostScript-Datei über einen PostScript-fähigen Windows-Druckertreiber exportieren (Netzdarstellung als PostScript-Datei exportieren) und anschließend in das Format *.pdf konvertieren.

Zur Konvertierung der PostScript-Datei in eine Datei *.pdf können Sie nicht-kommerzielle Software verwenden, beispielsweise das Freeware-Tool Aladdin Ghostview/Ghostscript, erhältlich unter der URL http://pages.cs.wisc.edu/~ghost/.

Sie können auch im Internet zur Verfügung stehende nicht-kommerzielle Online-Konvertierungsdienste nutzen, zum Beispiel von Universitäten und Fachhochschulen oder erhältlich unter den URLs http://convert.neevia.com/ und http://www.ps2pdf.com/convert.htm.

Das Vissig-Fenster

(1) Titelleiste

Hier werden der Programmname, die Nummer der ausgewählten Lichtsignalanlage und der Name der geladenen Signalsteuerungsdatei *.sig angezeigt.

(2) Menüleiste

Über die Menüs können Sie die Programmfunktionen aufrufen.

(3) Symbolleiste

In der Symbolleiste stehen Ihnen spezifische Funktionen zur Verfügung.

(4) Navigator

Im Navigator wählen Sie das Objekt aus, das Sie bearbeiten möchten.

Sie können den Navigator aus- und einblenden, indem Sie auf das Symbol klicken.
(5) Objektansicht

In diesem Bereich wird das im Navigator ausgewählte Objekt angezeigt.

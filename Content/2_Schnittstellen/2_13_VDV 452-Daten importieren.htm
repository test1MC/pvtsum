-Daten importieren

Wählen Sie im Menü Datei den Eintrag Importieren > .

Das Fenster Import öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor.

Element

Beschreibung

-Pfad

Geben Sie in das Eingabefeld das Verzeichnis mit den gewünschten Dateien *.x10 ein oder wählen Sie es über das Symbol .

Umläufe importieren

Ist die Option ausgewählt, werden auch Umläufe aus den -Daten importiert.

Hinweis

Diese Option ist nur verfügbar, wenn Sie das Zusatzmodul Linienleistungs- und Erfolgsrechnung eingeschaltet haben (Zusatzmodule ein- und ausschalten).

Erzeugte Netzdatei (*.net) speichern?

Ist die Option ausgewählt, werden die Daten nach dem Import dauerhaft als Netzdatei gespeichert und die Netzdatei wird geöffnet.

Ist die Option nicht ausgewählt, werden die Daten beim Import in eine temporäre Netzdatei geschrieben und in geöffnet. Die Datei wird anschließend gelöscht.

Ausgabe-Netzdatei

Geben Sie in das Eingabefeld Verzeichnis und Name der Netzdatei ein, in der Sie die importierten Daten speichern möchten oder wählen Sie ihn über das Symbol .

Hinweis

Der von automatisch vergebene Dateiname spiegelt das Erstellungsdatum mit Uhrzeit wieder: YYYY-MM-DD_HHMMSS.net

In einem Verzeichnis mit VDV-Daten können mehrere Versionen der Daten gespeichert sein. Generell werden diese sogenannten Basisversionen innerhalb eines Datenbestandes über die Tabelle 485 - 'MENGE_BASIS_VERSIONEN' definiert. Bei der Variante der Dateibenennung über Tabellennummern können weiterhin mehrere Datenbestände mit unterschiedlichem Exportdatum im gleichen Verzeichnis gespeichert sein. Beim Import der Daten nach kann jeweils nur eine Basisversion übernommen werden. Gibt es mehrere Basisversionen in dem angegebenen Verzeichnis, öffnet sich ein weiteres Fenster, in dem Sie in einer Auswahlliste die gewünschte Basisversion auswählen können.

Die Daten werden importiert. Dabei wird eine (temporäre) Netzdatei angelegt, die in eingelesen wird (Gegenüberstellung von - und -Datenmodell).

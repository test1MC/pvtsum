DXF Export

Über die DXF-Export-Schnittstelle können Sie die aktuelle Netzdarstellung als Grafikdatei *.dxf speichern.

Sie haben zwei Möglichkeiten, die aktuelle Netzdarstellung als Grafikdatei *.dxf zu speichern.

Erste Möglichkeit des DXF Exports

Wählen Sie im Menü Datei den Eintrag Exportieren > DXF Export.

Das Fenster DXF-Datei speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Das Fenster DXF-Export öffnet sich.

Wählen Sie den Linienstil, mit dem Striche in der Grafikdatei *.dxf dargestellt werden sollen.

Der aktuelle Netzausschnitt wird als Grafikdatei *.dxf gespeichert.

Sie können die Grafikdatei *.dxf zur Weiterbearbeitung in einem externen Grafikprogramm öffnen oder in als Hintergrund einlesen (Hintergrund von Kartendiensten aus dem Internet einfügen).

Sie können auch andere Ansichten als Dateien im Format *.dxf exportieren. Dazu wählen Sie im Menü Fahrplan (grafisch), Grünbanddarstellung oder Umsteiger-Taktdarstellung die Option DXF Export aus, im Menü Schematischer Liniennetzplan den Eintrag Exportieren > DXF Export, und fahren Sie mit Schritt 2 fort.
Zweite Möglichkeit des DXF Exports

Zeigen Sie im Menü Datei auf den Eintrag Drucken.

Wählen Sie den Eintrag Drucken.

Das Fenster Drucken: Netzeditor öffnet sich.

Wählen Sie im Bereich Dateiausgabe die Option In Grafikdatei drucken aus.

Das Fenster Grafikdatei speichern öffnet sich.

Wählen Sie in der Auswahlliste Dateityp den Eintrag Grafikdatei (*.dxf).

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Das Fenster DXF-Export öffnet sich.

Wählen Sie den Linienstil, mit dem Striche in der Grafikdatei *.dxf dargestellt werden sollen.

Der aktuelle Bildschirmausschnitt wird als Grafikdatei *.dxf gespeichert.

Sie können die Grafikdatei *.dxf zur Weiterbearbeitung in einem externen Grafikprogramm öffnen oder in als Hintergrund einlesen (Hintergrund von Kartendiensten aus dem Internet einfügen).

Einstellungen für den Vissim-Export speichern

Sie können die Einstellungen für den Vissim-Export als ANM-Exportparameterdatei *.anmp speichern.

Wählen Sie im Menü Datei den Eintrag Exportieren > Vissim (ANM).

Das Fenster ANM – Export Parameter öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (Abstrahiertes Netzmodell speichern).

Klicken Sie auf die Schaltfläche Parameter speichern.

Das Fenster ANM-Export-Parameter speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Die aktuellen Einstellungen werden als ANM-Exportparameterdatei *.anmp gespeichert.

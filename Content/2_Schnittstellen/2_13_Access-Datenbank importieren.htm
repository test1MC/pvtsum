Access-Datenbank importieren

Sie können vollständige Netze aus einer bestehenden Access-Datenbank oder ausgewählte Netztabellen additiv zum aktuellen Netz einlesen. Daneben können Sie auch Listen aus Datenbanken importieren.

Netz aus Access-Datenbank importieren

Die Datenbank muss mindestens die für die Einlesbarkeit notwendige Tabelle $Version enthalten.

Wählen Sie im Menü Datei den Eintrag Importieren > Datenbank.

Das Fenster Datenbank auswählen öffnet sich.

Wählen Sie den Eintrag MS Access 2003 oder älter oder MS Access 2007 oder neuer.

Das Fenster Access-Datenbank öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Das Fenster Netz lesen öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (Netzdatei öffnen und Netzdaten additiv einlesen).

Der Import der Netzdaten aus der Access-Datenbank wird gestartet. Wenn Sie die Option Netzdatei additiv lesen ausgewählt haben, öffnet sich zusätzlich das Fenster Netzdaten hinzulesen (Netzdaten additiv einlesen). Wenn die eingelesenen Netzdaten unvollständige Linien- oder Systemrouten enthalten, d. h. mindestens eine Strecke oder ein Haltepunkt im Routenweg fehlt oder für den Routenweg gesperrt ist, öffnet sich zusätzlich das Fenster Netz lesen (Linienroutenweg / Systemroutenweg ergänzen) (Netzdatei mit unvollständigen Linien- oder Systemrouten einlesen).

Liste aus Access-Datenbank importieren

Wählen Sie im Menü Datei den Eintrag Importieren > Datenbank.

Das Fenster Datenbank auswählen öffnet sich.

Wählen Sie den Eintrag MS Access 2003 oder älter oder MS Access 2007 oder neuer.

Das Fenster Access-Datenbank öffnen öffnet sich.

Wählen Sie die gewünschte Datei aus.

Klicken Sie auf die Schaltfläche Öffnen.

Das Fenster <Listenname>: Attribute lesen öffnet sich.

( Attributsdatei lesen ) .

Die Daten werden aus der Datenbank in das Netz importiert.

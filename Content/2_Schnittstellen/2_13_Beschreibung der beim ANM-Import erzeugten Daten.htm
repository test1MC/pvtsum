Beschreibung der beim ANM-Import erzeugten Daten

ANM-Dateien können nur in ein leeres Netz eingelesen werden. ÖV-Daten können derzeit nicht eingelesen werden.

Die folgende Beschreibung zeigt, welche Netzdaten in beim Import von ANM-Daten gespeichert werden.

CROSSWALKS

Die Furten in werden aus den zwei ANM-Klassen <CROSSWALK> und <CROSSWALKSIDE> eingelesen. In ist eine Furt definiert durch den Arm, zu dem sie gehört, durch die Richtung, und einen Index von 1 bis 3. Es kann daher maximal sechs Furten an einem Arm geben.

Eine <CROSSWALKSIDE> auf der linken Seite wird in auf eine Furt mit der Richtung „false” abgebildet. Auf der rechten Seite erhält sie den Wert „true”.

Da es in nur eine Furt je Typ geben kann, also insgesamt sechs, werden möglicherweise ANM-Daten zu Furten beim Import verworfen.

Die <CROSSWALKS> sind ein Element der <NODES>. Sie werden wie folgt abgebildet:

Hinweise auf die Richtungen

Hinweis

Der Wert kann nur „true” sein, wenn Arm\Dreiecksinsel-Länge des zugehörigen Arms ungleich null ist.

SCNO

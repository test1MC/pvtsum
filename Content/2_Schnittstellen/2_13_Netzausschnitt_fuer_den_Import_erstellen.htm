Netzausschnitt für den Import erstellen

Sie können die OpenStreetMap-Daten für den Import auf einen Ausschnitt zuschneiden, um den Import schneller und effektiver zu gestalten.

Möchten Sie einen kleineren Ausschnitt eines OpenStreetMap-Datensatzes importieren, gehen Sie idealerweise folgendermaßen vor:

Starten Sie in mit einer leeren Oberfläche (Mit einer leeren Oberfläche starten).

Stellen Sie sicher, dass eine dynamische Hintergrundkarte angezeigt wird (Hintergrundkarte von Kartendiensten aus dem Internet dynamisch einbinden).

Zoomen Sie so weit in die Karte hinein, dass der Ausschnitt in Ihrem Fenster sichtbar ist, für den Sie den Ausschnitt Ihrer OpenStreetMap-Daten importieren möchten.

Wählen Sie im Menü Datei > Importieren den Eintrag OpenStreetMap.

Das Fenster OpenStreetMap Importer öffnet sich.

( Importprozess mit einer vordefinierten Konfiguration durchführen )

Wählen Sie die Option Kartenausschnitt wählen aus.

In dem Bereich neben der Option werden Ihnen die aktuellen WGS84-Koordinaten des ausgewählten Netzausschnitts angezeigt.

Beispiel

Sie können die Koordinaten bei Bedarf auch manuell bearbeiten.

Nehmen Sie weitere Import Einstellungen für den Netzausschnitt vor.

Element Beschreibung

OSM Elemente importieren, die komplett innerhalb des Ausschnitts liegen

Ist die Option ausgewählt, werden nur Elemente importiert, die vollständig im gewählten Ausschnitt liegen. Beispielweise werden Knoten, die außerhalb des Ausschnitts, und Strecken, die aus dem Ausschnitt heraus ragen, nicht importiert.
OSM Elemente importieren, die komplett oder teilweise innerhalb des Ausschnitts liegen

Ist die Option ausgewählt, werden auch Elemente importiert, die nur teilweise im gewählten Ausschnitt liegen. Beispielweise werden Strecken, die aus dem Ausschnitt heraus ragen, vollständig importiert.

Das Netz des festgelegten Ausschnitts aus den OpenStreetMap-Daten wird importiert.

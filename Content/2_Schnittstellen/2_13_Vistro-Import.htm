Vistro-Import

bietet eine Schnittstelle für den Import von -Daten.

Sie können in erstellte Netze in importieren, um beispielsweise mit den vorhandenen Nachfragedaten in Wege zu erzeugen. In stehen Ihnen für die Berechnung verschiedene Kostenfunktionen zur Verfügung. Nach der Umlegung können Sie das Netz wieder in importieren und die Wegemenge als -Wegemenge übernehmen.

Wählen Sie im Menü Datei > Importieren den Eintrag VISTRO.

Das Fenster <VISTRO-Netz öffnen> öffnet sich.

Wählen Sie die gewünschte Datei aus.

Das -Netz wird in importiert. verändert oder erzeugt dabei eine Reihe von Daten.

Nach dem Import können Sie in die angelegten Verfahren anpassen und durchführen (siehe „Verfahren bearbeiten“ auf Seite 1 und „Verfahren im Verfahrensablauf ausführen“ auf Seite 1)

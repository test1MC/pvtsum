-Daten in eine Access-Datenbank exportieren

Stellen Sie sicher, dass das gewünschte Dezimaltrennzeichen ausgewählt ist (Dezimaltrennzeichen wählen).

Wählen Sie im Menü Datei den Eintrag Exportieren > Datenbank.

Das Fenster Datenbank auswählen öffnet sich.

Wählen Sie den Eintrag MS Access 2003 oder älter oder MS Access 2007 oder neuer.

Das Fenster Access-Datenbank speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Alternativ können Sie eine bestehende Access-Datenbank auswählen, die überschrieben wird.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster Netz schreiben (Benutzerdefinition) öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (Netzdatei speichern).

Die ausgewählten Netztabellen und Attribute werden in die Access-Datenbank exportiert.

Beim Datenbankexport nach Access werden auch die Primärschlüssel exportiert.

Bei Attributnamen mit mehr als 64 Zeichen kürzt die Namen beim Export.

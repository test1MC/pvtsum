Leerfahrt in Fahrplanfahrt umwandeln

Öffnen Sie den Umlaufeditor und klicken Sie in der Blockdarstellung mit der rechten Maustaste auf einen Block, der eine Leerfahrt repräsentiert (Blockdarstellung öffnen).

Wählen Sie im Kontextmenü den Eintrag In Fahrplanfahrt umwandeln.

Das Fenster Fahrplanfahrt erzeugen aus Leerfahrt öffnet sich.

Element

Beschreibung

Leerfahrt-VSys des Umlaufs

Ausgabe von Name und Code des Leerfahrt-VSys im Umlauf

Fahrzeugkombination des Umlaufs

Ausgabe der Fahrzeugkombination im Umlauf

Nur Linien mit dem Leerfahrt-Verkehrssystem zeigen

Ist die Option ausgewählt, werden nur die Linien angezeigt, die das VSys der Leerfahrt benutzen.

Linie

Wählen Sie in der Auswahlliste eine geeignete Linie aus.

Linienrouten-Name

Geben Sie den Namen der neuen Linienroute ein.

Richtung

Wählen Sie in der Auswahlliste eine der beiden Optionen.

Fahrzeitprofilname

Geben Sie den FZP-Namen für das neue Fahrzeitprofil ein.

Fahrplanfahrt-Nummer

Die Fahrplanfahrtnummer wird vom System vorgeschlagen und kann überschrieben werden.

Name der Fahrplanfahrt

Geben Sie den Namen für die neue Fahrplanfahrt ein.

Wenn Sie einen Wochenkalender oder Jahreskalender verwenden, öffnet sich ein weiteres Fenster, in dem Sie die relevanten Verkehrstage wählen müssen.

Ihre Einstellungen werden übernommen. Die ausgewählte Leerfahrt wird in eine neue Fahrplanfahrt umgewandelt.

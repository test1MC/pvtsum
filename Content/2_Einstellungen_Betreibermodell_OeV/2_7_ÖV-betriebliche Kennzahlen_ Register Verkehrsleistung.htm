ÖV-betriebliche Kennzahlen: Register Verkehrsleistung

Die Kennzahlen der Verkehrsleistung werden immer mit der Umlegung berechnet Siehe "Messung der Verkehrsleistung". Zusätzlich können Sie sie auch für Gebiete berechnen.

Element

Beschreibung

auch für Gebiete rechnen

Ist die Option ausgewählt, werden die Kennzahlen zusätzlich auch je Gebiet berechnet

berechne Kennzahlen für

Die aktuelle Auswahl wird angezeigt. Änderungen können Sie im Fenster Parameter für ÖV-betriebliche Kennzahlen vornehmen (ÖV-betriebliche Kennzahlen: Register Allgemein)

Belastungen speichern

Die aktuelle Einstellung wird angezeigt. Änderungen können Sie vornehmen über das Menü Rechnen > Allgemeine Verfahrenseinstellungen > ÖV-Einstellungen > Umlegung (Einstellungen zum Speichern der ÖV-Umlegungsergebnisse)

Für Zeitintervalle rechnen

Die aktuelle Einstellung wird angezeigt. Änderungen können Sie vornehmen über das Menü Rechnen > Allgemeine Verfahrenseinstellungen > Analysezeiträume (Analysezeitintervalle festlegen).

Wege speichern

Die aktuelle Einstellung wird angezeigt. Änderungen können Sie vornehmen über das Menü Rechnen > Allgemeine Verfahrenseinstellungen > ÖV-Einstellungen > Umlegung (Einstellungen zum Speichern der ÖV-Umlegungsergebnisse)

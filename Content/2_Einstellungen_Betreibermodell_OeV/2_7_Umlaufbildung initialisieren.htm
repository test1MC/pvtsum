Umlaufbildung initialisieren

Sie können eine oder alle Umlaufversionen initialisieren. Damit verwerfen Sie das Ergebnis der Umlaufbildung für die ausgewählte Umlaufversion.

Klicken Sie in der Symbolleiste auf das Symbol  Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Init Umlaufbildung aus der Kategorie ÖV-Auswertungen ein (Verfahrensablauf festlegen und starten).

Klicken Sie in der Zeile des neu eingefügten Verfahrens auf die Schaltfläche in der Spalte Bezugsobjekt(e).

Das Fenster Initialisierung Umlaufbildung öffnet sich.

Element

Beschreibung

Alle Umlaufversionen initialisieren

Ist die Option ausgewählt, werden alle Umlaufversionen initialisiert, für die eine Umlaufbildung berechnet wurde.

Umlaufversion

Über die Auswahlliste wählen Sie die Umlaufversion, für die das Ergebnis der Umlaufbildung verworfen werden soll.

Führen Sie das Verfahren aus (Verfahren im Verfahrensablauf ausführen).

Das Ergebnis der Umlaufbildung der ausgewählten Umlaufversionen wird verworfen.

Eine initialisierte Umlaufversion wird nicht automatisch gelöscht. Sie können sie neu berechnen, bei Bedarf mit geänderten Eigenschaften oder Verfahrensparametern.

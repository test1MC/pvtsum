Attribute von Umlaufelementtypen

Element

Beschreibung

ID

Nummer des Umlaufelementtyps. Die Nummer wird automatisch aufsteigend vergeben.

Hinweis

Nummer 1, 2, 3 und 4 sind standardmäßig vom System vorbelegt.

Name

Bezeichnung des Umlaufelementtyps

Standard-Dauer

Zeitdauer: Defaultwert für die Dauer von Elementen dieses Typs

Standard-Länge

Lange Länge: Defaultwert für die Länge von Elementen dieses Typs

Grenzwert für Zeit

Zeitdauer: Grenzwert für die Dauer zwischen zwei Elementen dieses Typs in Minuten

Hinweis

Wert 0 = unbegrenzt

Grenzwert für Länge

Lange Länge: Grenzwert für die Länge zwischen zwei Elementen dieses Typs in km

Hinweis

Wert 0 = unbegrenzt

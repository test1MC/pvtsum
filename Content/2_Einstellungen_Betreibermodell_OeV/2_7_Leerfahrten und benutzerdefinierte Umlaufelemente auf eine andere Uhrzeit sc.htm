Leerfahrten und benutzerdefinierte Umlaufelemente auf eine andere Uhrzeit schieben

Sie können Leerfahrten und benutzerdefinierte Umlaufelemente innerhalb eines Umlauftages und Verkehrstages auf eine andere Uhrzeit schieben.

Öffnen Sie den Umlaufeditor (Blockdarstellung öffnen).

Stellen Sie sicher, dass Sie die gewünschte Option für das Verschieben von Umlaufelementen gewählt haben (Verhalten beim Einfügen von Umlaufelementen beeinflussen).

Markieren Sie in der Blockdarstellung die gewünschten Blöcke.

Klicken Sie auf einen der markierten Blöcke und halten Sie die Maustaste gedrückt.

Ziehen Sie die markierten Elemente mit Hilfe der Maus auf die gewünschte Uhrzeit.

fügt die Elemente mit einer Genauigkeit von fünf Minuten ein. Die Uhrzeit der aktuellen Einfügeposition wird in der Quickinfo angezeigt. Wenn Sie die Elemente feiner positionieren möchten, müssen Sie gleichzeitig die Taste Umschalt drücken.

Wenn Sie das Element auf einen vorhandenen Block ziehen, wird es direkt im Anschluss an den vorhandenen Block eingefügt.

Die Umlaufelemente werden auf die gewünschte Uhrzeit verschoben. Alle anderen Attributwerte bleiben erhalten.

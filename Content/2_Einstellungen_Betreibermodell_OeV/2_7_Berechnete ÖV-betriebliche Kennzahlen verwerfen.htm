Berechnete ÖV-betriebliche Kennzahlen verwerfen

Klicken Sie in der Symbolleiste auf das Symbol  Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Init ÖV-betriebliche Kennzahlen aus der Kategorie ÖV-Auswertungen ein (Verfahrensablauf festlegen und starten).

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist, und klicken Sie im Bereich Aktionen auf Bearbeiten.

Das Fenster Init ÖV-betriebliche Kennzahlen öffnet sich.

Element

Beschreibung

Betriebsleistung

Über diese Option können Sie die Kennzahlen der Betriebsleistung löschen. Hinweis

Ist die Option ausgewählt, können Sie optional zusätzlich die Kennzahlen der Betriebsleistung auch für Gebiete löschen

Infrastrukturkosten

Über diese Option können Sie die Kennzahlen der Infrastrukturkostenberechnung löschen.

Hinweis

Ist die Option ausgewählt, können Sie optional zusätzlich die Kennzahlen der Infrastrukturkostenberechnung auch für Gebiete löschen

Erlöse

Über diese Option können Sie die Kennzahlen der Erlösberechnung löschen.

Hinweis

Ist die Option ausgewählt, können Sie optional zusätzlich die Kennzahlen der Erlösberechnung auch für Gebiete löschen

Verkehrsleistung für Gebiete

Über diese Option können Sie die je Gebiet berechnete Verkehrsleistung löschen.

Führen Sie das Verfahren aus (Verfahren im Verfahrensablauf ausführen).

Die ausgewählten ÖV-betrieblichen Kennzahlen werden gelöscht.

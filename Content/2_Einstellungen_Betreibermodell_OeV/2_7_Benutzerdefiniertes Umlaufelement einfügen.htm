Benutzerdefiniertes Umlaufelement einfügen

Sie können ein Umlaufelement mit einem benutzerdefinierten Umlaufelementtyp einfügen.

Öffnen Sie den Umlaufeditor (siehe „Umlaufeditor öffnen“ auf Seite 1).

Stellen Sie sicher, dass Sie einen Umlaufelementtyp eingefügt haben (siehe „Umlaufelementtyp einfügen“ auf Seite 1).

Klicken Sie im Umlaufeditor mit der rechten Maustaste im gewünschten Umlauftag auf die gewünschte Uhrzeit.

Wählen Sie im Kontextmenü den Eintrag Benutzerdefiniertes Umlaufelement einfügen.

Das Fenster Umlaufelement einfügen öffnet sich.

Nehmen Sie die gewünschten Einstellungen vor (siehe „Umlaufelement bearbeiten“ auf Seite 1).

Das Umlaufelement wird eingefügt.

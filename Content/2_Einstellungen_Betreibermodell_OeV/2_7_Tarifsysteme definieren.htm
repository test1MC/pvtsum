Tarifsysteme definieren

Wählen Sie im Menü Netz den Eintrag ÖV-Fahrpreise.

Das Fenster ÖV-Fahrpreise öffnet sich.

Wählen Sie das Register Tarifsysteme.

Die Liste der definierten Tarifsysteme wird ausgegeben.

Klicken Sie auf die Schaltfläche Einfügen.

Eine weitere Zeile wird der Liste der Tarifsysteme hinzugefügt.

Element

Beschreibung

Nummer

Nummer des Tarifsystems

Name

Name des Tarifsystems

Rang

Rang des Tarifsystems. Wenn es für einen Weg mehrere Tarifsysteme gibt, wird für die Fahrpreisermittlung der Fahrpreis des Tarifsystems mit dem höchsten Rang verwendet Siehe "Vorgehen bei mehrdeutigen Tarifsystemen".

Preisgewicht für Auswahl

Gewichtung des Tarifsystems. Wenn es für einen Weg mehrere Tarifsysteme mit gleichem Rang gibt, wird standardmäßig der Weg mit dem niedrigsten Fahrpreis verwendet. Den Fahrpreis können Sie nochmals mit dieser Gewichtung belegen. Damit können Sie beispielsweise abbilden, dass von außerhalb kommende Fahrgäste in der Regel den maximalen Weg mit der Bahn zurücklegen, bevor sie in ein Nahverkehrssystem umsteigen, selbst wenn dies nicht der billigste Fahrpreis ist Siehe "Preisgewichte".

Fahrpreis bezieht sich auf

Über die Auswahlliste legen Sie fest, ob für eine Verbindung mit mehreren Teilwegen eine oder mehrere Fahrkarten gekauft werden müssen Siehe "„Fahrpreis-Bezug“ eines Tarifsystems".

jeweils zusammenhängende Teilwege

Eine Fahrkarte kann für aufeinander folgende Teilwege einer Verbindung genutzt werden.

alle Teilwege zusammen

Eine Fahrkarte kann für alle Teilwege einer Verbindung genutzt werden.

jeden Teilweg einzeln

Für jeden einzelnen Teilweg muss eine Fahrkarte gekauft werden.

Hinweis

Diese Option verwendet automatisch, wenn Sie im Verfahren ÖV-Umlegung Taktfein Fahrpreise für die Berechnung des Widerstands verwenden („Fahrpreis-Bezug“ eines Tarifsystems und Taktfeine Umlegung: Register Widerstand)

Anfangsfahrpreis

Anfangsfahrpreis für die Fahrkartenart Siehe "Anfangsfahrpreis und Umstiegsfahrpreis"

Linien

Über die Schaltfläche können Sie in einem separaten Fenster für jedes Tarifsystem die Linien auswählen, die innerhalb des Tarifsystems verkehren. Für jede Linie werden in der Spalte TarifsystemMenge automatisch die Nummern der Tarifsysteme, zu denen sie gehört, aktualisiert.

Fahrkartenarten

Die Spalte gibt die Nummern der Fahrkartenarten aus, die für das Tarifsystem gelten (Fahrkartenarten definieren).

Das Tarifsystem wird eingefügt.

Tarifsystem bearbeiten

Stellen Sie sicher, dass im Fenster ÖV-Fahrpreise das Register Tarifsysteme geöffnet ist.

Klicken Sie in der Liste der Tarifsysteme direkt in die Zelle, die Sie bearbeiten möchten.

Die Änderungen werden übernommen.

Tarifsystem löschen

Stellen Sie sicher, dass im Fenster ÖV-Fahrpreise das Register Tarifsysteme geöffnet ist.

Wählen Sie in der Liste das gewünschte Tarifsystem aus.

Sie können mehrere Listeneinträge auswählen, indem Sie die Taste Strg drücken, diese gedrückt halten und auf die gewünschten Einträge klicken.

Klicken Sie auf die Schaltfläche Löschen.

Das Tarifsystem wird gelöscht.

Umlaufeditor öffnen

Umläufe können Sie in einer Blockdarstellung anzeigen.

Wählen Sie im Menü Bearbeiten den Eintrag Umlaufeditor.

Die Fenster Umlaufeditor und Umlaufauswahl öffnen sich.

Wählen Sie bei Bedarf über die Auswahlliste in der Symbolleiste des Umlaufeditors einen anderen Verkehrstag-Grundfilter.

Wählen Sie im Navigator der Umlaufauswahl die Umlaufversionen, Fahrzeugkombinationen und Umläufe, die Sie im Umlaufeditor anzeigen möchten.

Sie können im Navigator die untergeordneten Objekte eines Eintrags mit einem Klick auf das jeweilige Pluszeichen anzeigen.

Die ausgewählten Umläufe werden im Fenster Umlaufeditor angezeigt.

Sie können die Ansicht im Fenster Umlaufeditor ähnlich wie die Ansicht im Fenster Netzeditor verändern (siehe „Fenster Netzeditor“ auf Seite 1). Beispielsweise können Sie die Ansicht mit Hilfe von Symbolen in der Symbolleiste vergrößern, verkleinern oder verschieben.

Sie können die Ränder der Blockdarstellung ein- und ausblenden, indem Sie im Kontextmenü einen der Einträge Oberer Rand sichtbar, Unterer Rand sichtbar, Linker Rand sichtbar, Rechter Rand sichtbar wählen.

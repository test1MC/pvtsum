ÖV-betriebliche Kennzahlen: Register benutzerdefinierte Attribute

Sie können Attribute des Fahrplanfahrtverlaufs oder von Fahrplanfahrtabschnitten auf höhere Ebenen der Linienhierarchie bzw. optional auch auf Gebiete hochrechnen (siehe Grundlagen: Kap. , Seite 1).

Element

Beschreibung

berechnen

Ist die Option ausgewählt, werden ausgewählte Attribute von Fahrplanfahrtabschnitten oder Fahrplanfahrtelementen hochgerechnet auf höhere Ebenen der Linienhierarchie.

auch für Gebiete berechnen

Ist die Option ausgewählt, werden die Kennzahlen zusätzlich gebietsfein berechnet gemäß der Detaillierungsebene, die im Register Allgemein ausgewählt ist.

Aktiv

Ist die Option ausgewählt, wird das Attribut unter Berücksichtigung des dafür gewählten Hochrechnungsbezugs hochgerechnet.

Quellobjekt

Wählen Sie über die Auswahlliste das Quellobjekt für die Hochrechnung aus.

Attribut

Wählen Sie ein direktes oder numerisches Attribut des gewählten Quellobjekts aus, auch benutzerdefinierte Attribute sind möglich.

Hochrechnungsbezug

Wählen Sie über die Auswahlliste, ob das Attribut ein Attribut der Länge ist oder zeitbezogen wirkt oder ob es sich auf den gesamten Fahrplanfahrtabschnitt / das gesamte Fahrplanfahrtelement bezieht.

Einfügen

Über die Schaltfläche können Sie eine weitere Zeile einfügen.

Löschen

Über die Schaltfläche können Sie ausgewählte Zeilen löschen.

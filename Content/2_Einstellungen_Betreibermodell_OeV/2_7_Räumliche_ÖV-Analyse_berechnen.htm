Räumliche ÖV-Analyse berechnen

Sie können mit dem Verfahren Räumliche ÖV-Analyse Auswertungen durchführen, in denen Sie Attribute von Linienroutenelementen mit Attributen von Fahrplanfahrtabschnitten kombinieren können (Grundlagen: Räumliche ÖV-Analyse).

Die Kennzahlen der räumlichen ÖV-Analyse werden für die aktuell aktive Umlaufversion berechnet (Umlaufversion aktiv setzen).

Im Verzeichnis ...\Programme\PTV Vision\\Doc\Deu Ihrer -Installation finden Sie weitere Informationen zu den Kennzahlen:

  • Der Datei Kennzahlenherkunft.xls können Sie entnehmen, welche Kennzahlen Sie mit welchem Verfahren ermitteln können.
  • Der Datei Kennzahlenverfuegbarkeit.xls können Sie entnehmen, welche Kennzahlen Sie mit welcher Listenart ausgeben können.

Klicken Sie in der Symbolleiste auf das Symbol  Verfahrensablauf öffnen.

Das Fenster Verfahrensablauf öffnet sich.

Fügen Sie an der gewünschten Stelle im Verfahrensablauf das Verfahren Räumliche ÖV-Analyse aus der Kategorie ÖV-Auswertungen ein (Verfahrensablauf festlegen und starten).

Stellen Sie sicher, dass das eingefügte Verfahren markiert ist, und klicken Sie im Bereich Aktionen auf Bearbeiten.

Das Fenster Parameter für räumliche Analyse öffnet sich.

Element Beschreibung
Nur aktive Fahrplanfahrtabschnitte berücksichtigen

Ist die Option ausgewählt, werden nur aktive Fahrplanfahrtabschnitte für die Berechnung berücksichtigt.
Aktiv

Ist die Option ausgewählt, wird das Attribut berechnet.
AttID

Wählen Sie ein direktes oder indirektes numerisches Fahrplanfahrtelement-Attribut, auch benutzerdefinierte Attribute sind möglich.

Standardmäßig wird das berechnete Attribut Personenkilometer (PersKm (AP)) verwendet. Es ist das Produkt aus der Belastung des jeweiligen Subattributs und der Nach-Länge des Fahrplanfahrtelements.

Wählen Sie ein direktes oder indirektes numerisches Fahrplanfahrtverlaufs- Attribut, auch benutzerdefinierte Attribute sind möglich.
Aufteilungsregel

nach Vorkommen in AP

Aufteilung nach Vorkommen in der Analyseperiode

je Kalendertag in AP

Aufteilung der Auswertung auf die Kalendertage in der Analyseperiode
Gewichtsattribut Numerisches Fahrplanfahrtabschnittsattribut, das zur Gewichtung verwendet wird.

Einfügen

Über die Schaltfläche können Sie eine weitere Zeile einfügen.
Löschen

Über die Schaltfläche können Sie eine ausgewählte Zeile löschen.

Führen Sie das Verfahren aus (siehe „Verfahrensablauf festlegen und starten“ auf Seite 1).

Die räumliche ÖV-Analyse wird berechnet.Die Ergebnisse des Verfahrens werden in der Liste Linienroutenelement-ÖV-Detail ausgegeben.

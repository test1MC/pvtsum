Zeitpunkt für den Tagesübergang ändern

Sie können den Zeitpunkt für den Tagesübergang der Blockdarstellung ändern.

Wählen Sie im Menü Bearbeiten den Eintrag Umlaufeditor.

Wählen Sie im Menü Umlaufeditor den Eintrag Parameter bearbeiten.

Das Fenster Parameter Umlauf-Blockdarstellung öffnet sich.

Element

Beschreibung

Zeitpunkt für Tagesübergang

Diese Einstellung bestimmt die Uhrzeit für den Zeilenumbruch in der Blockdarstellung. Wenn Sie eine Uhrzeit größer 00:00:00 einstellen, werden die Blöcke entsprechend verschoben angezeigt. So können Sie eine vom Kalendertag abweichende Lage der Betriebstage darstellen.

Die Änderungen werden übernommen.

Freie Leistungen als Grafikdatei speichern

Sie können freie Leistungen direkt in einer Grafikdatei *.svg oder *.dxf speichern.

Wählen Sie im Menü Umlaufeditor > Freie Leistungen exportiereneinen der Einträge SVG Export oder DXF Export.

Das Fenster SVG-Datei speichern oder DXF-Datei speichern öffnet sich.

Wählen Sie den Eintrag SVG Export oder DXF Export.

Das Fenster SVG-Datei speichern oder DXF-Datei speichern öffnet sich.

Geben Sie einen Dateinamen und bei Bedarf den Pfad ein.

Klicken Sie auf die Schaltfläche Speichern.

Das Fenster SVG-Export oder DXF-Export öffnet sich.

( SVG-Export und DXF Export ) .

Die freien Leistungen werden in einer Datei *.svg oder *.dxf gespeichert.

Sie können die Ansicht auch speichern, indem Sie im Menü Umlaufeditor > Freie Leistungen exportieren den Eintrag JPG Export (Screenshot) wählen oder im Fenster Umlaufeditor in der Symbolleiste auf das Symbol  JPG Export (Screenshot) klicken (Screenshot vom Bildschirm exportieren).

Sie können freie Leistungen auch ausdrucken (Freie Leistungen drucken).

Attribute und Optionen von Tarifzonen

Der obere Teil des Fensters Tarifzoneeinfügen oder Tarifzone bearbeiten enthält die allgemeinen Attribute der Tarifzone. Im unteren Teil können Sie die Haltestellen zuordnen oder diese aus der Zuordnung entfernen.

Element

Beschreibung

Nummer

Nummer der Tarifzone

Typ

Typ der Tarifzone

Code

Kurzbezeichnung der Tarifzone

Name

Langbezeichnung der Tarifzone

Haltestellen

In diesem Bereich können Sie der Tarifzone eine oder mehrere Haltestellen zuordnen.

Markieren Sie hierzu die gewünschten Haltestellen und verwenden Sie die Schaltflächen und .

Tipp

Sie können mehrere Haltestellen auswählen, indem Sie die Taste Strg drücken, diese gedrückt halten und auf die gewünschten Einträge klicken.

Hinweis

Eine Haltestelle kann mehreren Tarifzonen zugeordnet werden.

alle Haltestellen

Ist die Option ausgewählt, werden alle Haltestellen in der Haltestellenliste aufgelistet.

Tipp

Damit können Sie Haltestellen, die sich genau auf Tarifzonengrenzen befinden, mehreren Tarifzonen zuordnen.

nur freie Haltestellen

Ist die Option ausgewählt, werden nur die Haltestellen in der Haltestellenliste aufgelistet, die noch keiner Tarifzone zugeordnet sind.

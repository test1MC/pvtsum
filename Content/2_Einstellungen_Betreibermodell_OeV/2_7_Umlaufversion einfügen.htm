Umlaufversion einfügen

Wählen Sie im Menü Netz den Eintrag ÖV-Umlaufversionen.

Das Fenster Umlaufversionen öffnet sich.

Die Liste enthält alle vorhandenen Umlaufversionen mit ihren allgemeinen Attributen. Sie können der Liste nicht entnehmen, ob für eine Umlaufversion bereits eine Umlaufbildung gerechnet wurde.

Klicken Sie auf die Schaltfläche  Einfügen.

Das Fenster Umlaufversion einfügen öffnet sich.

Geben Sie die gewünschten Daten ein (Attribute von Umlaufversionen).

Alternativ können Sie eine neue Umlaufversion anlegen, indem Sie eine vorhandene Umlaufversion kopieren (Umläufe oder Umlaufversion kopieren).

Die Umlaufversion ist angelegt und kann berechnet werden (Verfahren Umlaufbildung ausführen).

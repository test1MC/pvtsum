Benutzerdefinierte CR-Funktionen

Anwender können für folgende Anwendungsfälle benutzerdefinierte CR-Funktionen definieren:

  • Zur Berücksichtigung weiterer Attribute für Strecken, Abbieger und Anbindungen in der Berechnung
  • Zur Berechnung der Pkw-Einheiten auf nicht-standardisierte Weise
  • Zur Definition separater CR-Funktionen für verschiedene Verkehrssysteme

CR-Funktionen werden innerhalb der Umlegungsverfahren sehr oft ausgewertet, daher spielt der Rechenzeitaufwand eine große Rolle. Deshalb wird bei benutzerdefinierten CR-Funktionen in eine kompilierte anstatt einer interpretierten Vorgehensweise verwendet. Anwender programmieren ihre Funktionskurven in Form einer Dynamic Link Library (DLL) nach einer vorgegebenen Vorlage. Alle angelegten Dateien *.dll kopieren Sie dann in das folgende, bei der Installation standardmäßig angelegte Verzeichnis, das beim Start durchsucht: %APPDATA%\\\UserVDF-DLLs (Anwendung: Benutzerdefinierte CR-Funktionen verwenden).

Eine Datei *.bmp mit identischem Dateinamen im gleichen Verzeichnis wird bei der Auswahl der CR-Funktion angezeigt.
